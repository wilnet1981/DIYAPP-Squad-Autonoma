# DIYAPP Evolution - V12 Core - Arquitetura de Microsserviços

## 1. ADR-001: Arquitetura de Microsserviços V12

**Data:** 2024-03-20
**Status:** Aceita
**Autores:** Tech Lead, Especialista Infra, SRE Lead

### CONTEXTO:
A V11 do DIYAPP evoluiu para uma arquitetura monolítica complexa que:
1. Dificulta deploy independente de componentes
2. Cria acoplamento técnico entre módulos
3. Limita escalabilidade horizontal específica
4. Aumenta tempo de build/test para pequenas mudanças
5. Representa single point of failure em componentes críticos

Forças em jogo:
- Necessidade de estabilidade 100% autônoma
- Time-to-market para novas features
- Custos operacionais controlados
- Manutenibilidade a longo prazo

### DECISÃO:
Adotar arquitetura de microsserviços com os seguintes princípios:
1. **Deploy independente**: Cada serviço tem seu pipeline CI/CD
2. **Banco de dados por serviço**: Isolamento de dados, schemas proprietários
3. **Comunicação assíncrona por eventos**: Event-driven architecture com Apache Kafka
4. **API Gateway unificado**: Único ponto de entrada para clientes externos
5. **Service Mesh**: Istio para observabilidade e resiliência

### OPÇÕES CONSIDERADAS:

**Opção A: Arquitetura de Microsserviços Completa (ESCOLHIDA)**
- Prós: 
  - Isolamento total de falhas
  - Escalabilidade granular
  - Evolução tecnológica independente
  - Times autônomos por serviço
- Contras:
  - Complexidade operacional aumentada
  - Overhead de comunicação
  - Necessidade de SRE dedicado
  - Custo inicial de migração alto

**Opção B: Arquitetura Híbrida (Monólito + Microsserviços)**
- Prós:
  - Migração gradual
  - Menor risco inicial
  - Reutilização de código existente
- Contras:
  - Complexidade híbrida (pior dos dois mundos)
  - Dificuldade em definir limites claros
  - Acoplamento residual

**Opção C: Refatoração do Monólito (Modular Monolith)**
- Prós:
  - Baixo custo de migração
  - Operação simplificada
  - Comunicação in-process
- Contras:
  - Não resolve problemas fundamentais de escalabilidade
  - Deploy ainda acoplado
  - Limites de tecnologia únicos

### CONSEQUÊNCIAS:

**Positivas:**
- Resiliência: falha em um serviço não derruba o sistema
- Escalabilidade: serviços com alta carga escalam independentemente
- Velocidade: times podem deployar sem coordenação central
- Evolução: adoção de tecnologias específicas por domínio

**Negativas:**
- Latência: comunicação entre serviços adiciona overhead
- Debug: tracing distribuído necessário
- Transações: padrão SAGA necessário para consistência
- Monitoramento: complexidade aumentada

**Riscos:**
1. **Degradação de performance**: Mitigar com caching estratégico e design assíncrono
2. **Consistência de dados**: Implementar SAGA pattern com compensação
3. **Cascata de falhas**: Implementar circuit breakers e retry com backoff exponencial
4. **Complexidade operacional**: Investir em observabilidade (metrics, logs, traces)

**REVISÃO:** 2024-06-20 (3 meses após implementação)

### 2.2 Padrões de Comunicação

```python
# libs/event_schemas/__init__.py
"""
Esquemas de eventos para comunicação entre serviços
"""
from pydantic import BaseModel
from datetime import datetime
from typing import Optional, Dict, Any
from enum import Enum

class EventType(str, Enum):
    USER_CREATED = "user.created"
    USER_UPDATED = "user.updated"
    PROJECT_PUBLISHED = "project.published"
    EXECUTION_COMPLETED = "execution.completed"
    PAYMENT_PROCESSED = "payment.processed"

class BaseEvent(BaseModel):
    event_id: str
    event_type: EventType
    timestamp: datetime
    source_service: str
    correlation_id: str
    payload: Dict[str, Any]
    
    class Config:
        json_encoders = {
            datetime: lambda dt: dt.isoformat()
        }

class UserCreatedEvent(BaseEvent):
    event_type: EventType = EventType.USER_CREATED
    payload: Dict[str, Any]  # User data
    
class ProjectPublishedEvent(BaseEvent):
    event_type: EventType = EventType.PROJECT_PUBLISHED
    payload: Dict[str, Any]  # Project data

# Exemplo de producer
class EventProducer:
    def __init__(self, kafka_brokers: List[str]):
        self.producer = KafkaProducer(
            bootstrap_servers=kafka_brokers,
            value_serializer=lambda v: json.dumps(v).encode('utf-8')
        )
    
    async def publish(self, event: BaseEvent):
        await self.producer.send(
            topic=event.event_type.value,
            value=event.dict()
        )
```

### 3.3 Fase 3: Database Migration (Sprint 7-10)

```sql
-- migration/database_migration.sql
-- Estratégia: Dual Write + Change Data Capture

-- 1. Criar schemas isolados
CREATE SCHEMA IF NOT EXISTS v12_users;
CREATE SCHEMA IF NOT EXISTS v12_projects;

-- 2. Configurar CDC para replicação inicial
CREATE PUBLICATION v11_changes FOR TABLE 
    users, projects, executions;

-- 3. Implementar dual-write pattern
CREATE OR REPLACE FUNCTION dual_write()
RETURNS TRIGGER AS $$
BEGIN
    -- Write to V11 (legacy)
    PERFORM legacy_write(TG_TABLE_NAME, NEW);
    
    -- Write to V12 (new microservice)
    PERFORM v12_write(TG_TABLE_NAME, NEW);
    
    RETURN NEW;
EXCEPTION WHEN OTHERS THEN
    -- Log failure but don't block V11
    INSERT INTO migration_errors VALUES (NOW(), TG_TABLE_NAME, SQLERRM);
    RETURN NEW;
END;
$$ LANGUAGE plpgsql;

-- 4. Plano de verificação de consistência
WITH consistency_check AS (
    SELECT 
        'users' as table_name,
        (SELECT COUNT(*) FROM v11.public.users) as v11_count,
        (SELECT COUNT(*) FROM v12_users.users) as v12_count,
        v11_count - v12_count as diff
    UNION ALL
    SELECT 
        'projects' as table_name,
        (SELECT COUNT(*) FROM v11.public.projects) as v11_count,
        (SELECT COUNT(*) FROM v12_projects.projects) as v12_count,
        v11_count - v12_count as diff
)
SELECT * FROM consistency_check WHERE diff != 0;
```

```typescript
// src/core/orchestrator/orchestrator.service.ts
import { Injectable, Logger } from '@nestjs/common';
import { InjectRepository } from '@nestjs/typeorm';
import { Repository } from 'typeorm';
import { Queue, Worker, Job } from 'bullmq';
import IORedis from 'ioredis';
import { CircuitBreaker } from './circuit-breaker';
import { HealthCheck, HealthStatus } from './health-check.entity';
import { Task, TaskStatus, TaskPriority } from './task.entity';
import { Agent, AgentStatus } from './agent.entity';
import { CreateTaskDto, TaskResponseDto } from './orchestrator.dto';

@Injectable()
export class OrchestratorService {
  private readonly logger = new Logger(OrchestratorService.name);
  private taskQueue: Queue;
  private redisConnection: IORedis;
  private circuitBreakers: Map<string, CircuitBreaker> = new Map();
  private readonly MAX_RETRIES = 3;
  private readonly TIMEOUT_MS = 30000;

  constructor(
    @InjectRepository(Task)
    private taskRepository: Repository<Task>,
    @InjectRepository(Agent)
    private agentRepository: Repository<Agent>,
    @InjectRepository(HealthCheck)
    private healthRepository: Repository<HealthCheck>,
  ) {
    this.initializeRedis();
    this.initializeQueue();
    this.initializeWorkers();
    this.initializeCircuitBreakers();
  }

  private initializeRedis(): void {
    this.redisConnection = new IORedis({
      host: process.env.REDIS_HOST || 'localhost',
      port: parseInt(process.env.REDIS_PORT || '6379'),
      maxRetriesPerRequest: 3,
      retryStrategy: (times) => {
        const delay = Math.min(times * 50, 2000);
        return delay;
      },
    });

    this.redisConnection.on('error', (error) => {
      this.logger.error(`Redis connection error: ${error.message}`, error.stack);
    });
  }

  private initializeQueue(): void {
    this.taskQueue = new Queue('agent-tasks', {
      connection: this.redisConnection,
      defaultJobOptions: {
        attempts: this.MAX_RETRIES,
        backoff: {
          type: 'exponential',
          delay: 1000,
        },
        removeOnComplete: 100,
        removeOnFail: 1000,
      },
    });
  }

  private initializeWorkers(): void {
    // Worker para tarefas de alta prioridade
    new Worker(
      'agent-tasks',
      async (job: Job) => {
        return await this.processTask(job);
      },
      {
        connection: this.redisConnection,
        concurrency: 5,
        limiter: {
          max: 50,
          duration: 1000,
        },
      },
    );

    // Worker para tarefas de baixa prioridade
    new Worker(
      'agent-tasks',
      async (job: Job) => {
        return await this.processTask(job);
      },
      {
        connection: this.redisConnection,
        concurrency: 10,
        limiter: {
          max: 100,
          duration: 1000,
        },
      },
    );
  }

  private initializeCircuitBreakers(): void {
    const services = ['database', 'redis', 'external-api', 'llm-service'];
    services.forEach(service => {
      this.circuitBreakers.set(service, new CircuitBreaker(
        service,
        5, // failure threshold
        30000, // reset timeout
        10000, // half-open timeout
      ));
    });
  }

  async createTask(createTaskDto: CreateTaskDto): Promise<TaskResponseDto> {
    const correlationId = this.generateCorrelationId();
    this.logger.log({
      correlation_id: correlationId,
      action: 'create_task',
      payload: this.maskSensitiveData(createTaskDto),
    });

    try {
      // Validação de input
      this.validateTaskInput(createTaskDto);

      // Circuit breaker para database
      const dbCircuitBreaker = this.circuitBreakers.get('database');
      if (!dbCircuitBreaker?.allowRequest()) {
        throw new Error('Database service unavailable');
      }

      // Cria tarefa no banco
      const task = this.taskRepository.create({
        ...createTaskDto,
        correlationId,
        status: TaskStatus.PENDING,
        createdAt: new Date(),
      });

      const savedTask = await this.taskRepository.save(task);

      // Adiciona à fila com prioridade
      const job = await this.taskQueue.add(
        `task-${savedTask.id}`,
        { taskId: savedTask.id, correlationId },
        {
          priority: this.mapPriorityToBullMQ(createTaskDto.priority),
          jobId: savedTask.id.toString(),
        }
      );

      // Atualiza task com jobId
      savedTask.jobId = job.id;
      await this.taskRepository.save(savedTask);

      return this.mapToTaskResponseDto(savedTask);
    } catch (error) {
      this.logger.error({
        correlation_id: correlationId,
        action: 'create_task_failed',
        error: error.message,
        stack: error.stack,
      });
      
      // Atualiza circuit breaker
      this.circuitBreakers.get('database')?.recordFailure();
      
      throw this.mapErrorToHttp(error);
    }
  }

  async getTask(id: number): Promise<TaskResponseDto> {
    const correlationId = this.generateCorrelationId();
    this.logger.log({
      correlation_id: correlationId,
      action: 'get_task',
      task_id: id,
    });

    try {
      const dbCircuitBreaker = this.circuitBreakers.get('database');
      if (!dbCircuitBreaker?.allowRequest()) {
        throw new Error('Database service unavailable');
      }

      const task = await this.taskRepository.findOne({
        where: { id },
        relations: ['agent'],
      });

      if (!task) {
        throw new Error('Task not found');
      }

      dbCircuitBreaker.recordSuccess();
      return this.mapToTaskResponseDto(task);
    } catch (error) {
      this.logger.error({
        correlation_id: correlationId,
        action: 'get_task_failed',
        task_id: id,
        error: error.message,
      });
      
      this.circuitBreakers.get('database')?.recordFailure();
      throw this.mapErrorToHttp(error);
    }
  }

  async getQueueStats(): Promise<any> {
    const correlationId = this.generateCorrelationId();
    
    try {
      const redisCircuitBreaker = this.circuitBreakers.get('redis');
      if (!redisCircuitBreaker?.allowRequest()) {
        throw new Error('Redis service unavailable');
      }

      const [waiting, active, completed, failed, delayed] = await Promise.all([
        this.taskQueue.getWaitingCount(),
        this.taskQueue.getActiveCount(),
        this.taskQueue.getCompletedCount(),
        this.taskQueue.getFailedCount(),
        this.taskQueue.getDelayedCount(),
      ]);

      redisCircuitBreaker.recordSuccess();

      return {
        waiting,
        active,
        completed,
        failed,
        delayed,
        timestamp: new Date().toISOString(),
        correlation_id: correlationId,
      };
    } catch (error) {
      this.logger.error({
        correlation_id: correlationId,
        action: 'get_queue_stats_failed',
        error: error.message,
      });
      
      this.circuitBreakers.get('redis')?.recordFailure();
      throw this.mapErrorToHttp(error);
    }
  }

  async healthCheck(): Promise<any> {
    const correlationId = this.generateCorrelationId();
    const checks: Array<{ service: string; status: HealthStatus; latency?: number }> = [];
    const startTime = Date.now();

    try {
      // Check database
      const dbStart = Date.now();
      const dbCircuitBreaker = this.circuitBreakers.get('database');
      const dbStatus = dbCircuitBreaker?.getStatus();
      
      if (dbCircuitBreaker?.allowRequest()) {
        await this.taskRepository.query('SELECT 1');
        dbCircuitBreaker.recordSuccess();
        checks.push({
          service: 'database',
          status: HealthStatus.HEALTHY,
          latency: Date.now() - dbStart,
        });
      } else {
        checks.push({
          service: 'database',
          status: HealthStatus.UNHEALTHY,
        });
      }

      // Check Redis
      const redisStart = Date.now();
      const redisCircuitBreaker = this.circuitBreakers.get('redis');
      
      if (redisCircuitBreaker?.allowRequest()) {
        await this.redisConnection.ping();
        redisCircuitBreaker.recordSuccess();
        checks.push({
          service: 'redis',
          status: HealthStatus.HEALTHY,
          latency: Date.now() - redisStart,
        });
      } else {
        checks.push({
          service: 'redis',
          status: HealthStatus.UNHEALTHY,
        });
      }

      // Check queue
      const queueStart = Date.now();
      try {
        const stats = await this.getQueueStats();
        checks.push({
          service: 'queue',
          status: HealthStatus.HEALTHY,
          latency: Date.now() - queueStart,
        });
      } catch {
        checks.push({
          service: 'queue',
          status: HealthStatus.DEGRADED,
        });
      }

      // Salva health check
      const healthCheck = this.healthRepository.create({
        correlationId,
        checks,
        overallStatus: checks.every(c => c.status === HealthStatus.HEALTHY) 
          ? HealthStatus.HEALTHY 
          : HealthStatus.DEGRADED,
        timestamp: new Date(),
        responseTime: Date.now() - startTime,
      });

      await this.healthRepository.save(healthCheck);

      return {
        status: healthCheck.overallStatus,
        checks,
        timestamp: healthCheck.timestamp.toISOString(),
        correlation_id: correlationId,
        response_time: healthCheck.responseTime,
      };
    } catch (error) {
      this.logger.error({
        correlation_id: correlationId,
        action: 'health_check_failed',
        error: error.message,
      });

      return {
        status: HealthStatus.UNHEALTHY,
        checks,
        timestamp: new Date().toISOString(),
        correlation_id: correlationId,
        error: 'Health check failed',
      };
    }
  }

  private async processTask(job: Job): Promise<any> {
    const { taskId, correlationId } = job.data;
    
    this.logger.log({
      correlation_id: correlationId,
      action: 'process_task_start',
      job_id: job.id,
      task_id: taskId,
    });

    try {
      // Busca task
      const task = await this.taskRepository.findOne({
        where: { id: taskId },
        relations: ['agent'],
      });

      if (!task) {
        throw new Error(`Task ${taskId} not found`);
      }

      // Atualiza status
      task.status = TaskStatus.PROCESSING;
      task.startedAt = new Date();
      await this.taskRepository.save(task);

      // Processa com timeout
      const result = await this.executeWithTimeout(
        () => this.executeTask(task),
        this.TIMEOUT_MS,
        correlationId
      );

      // Atualiza task com sucesso
      task.status = TaskStatus.COMPLETED;
      task.completedAt = new Date();
      task.result = result;
      await this.taskRepository.save(task);

      this.logger.log({
        correlation_id: correlationId,
        action: 'process_task_success',
        task_id: taskId,
        duration: Date.now() - task.startedAt.getTime(),
      });

      return result;
    } catch (error) {
      this.logger.error({
        correlation_id: correlationId,
        action: 'process_task_failed',
        task_id: taskId,
        error: error.message,
        stack: error.stack,
        attempt: job.attemptsMade,
      });

      // Atualiza task com falha
      await this.taskRepository.update(taskId, {
        status: TaskStatus.FAILED,
        error: error.message,
        completedAt: new Date(),
      });

      // Se ultrapassou max retries, marca como dead letter
      if (job.attemptsMade >= this.MAX_RETRIES) {
        await this.taskRepository.update(taskId, {
          status: TaskStatus.DEAD_LETTER,
        });
      }

      throw error;
    }
  }

  private async executeTask(task: Task): Promise<any> {
    // Lógica específica do tipo de tarefa
    switch (task.type) {
      case 'data_processing':
        return await this.processDataTask(task);
      case 'llm_inference':
        return await this.processLLMTask(task);
      case 'external_api':
        return await this.processExternalApiTask(task);
      default:
        throw new Error(`Unknown task type: ${task.type}`);
    }
  }

  private async processLLMTask(task: Task): Promise<any> {
    const llmCircuitBreaker = this.circuitBreakers.get('llm-service');
    
    if (!llmCircuitBreaker?.allowRequest()) {
      throw new Error('LLM service circuit breaker open');
    }

    try {
      // Simulação de chamada LLM com timeout específico
      const result = await this.executeWithTimeout(
        async () => {
          // Aqui viria a integração real com o LLM
          // Por enquanto, simulação
          await new Promise(resolve => setTimeout(resolve, 1000));
          return { 
            generated_text: 'Simulated LLM response',
            tokens_used: 150,
            model: 'gpt-4' 
          };
        },
        60000, // Timeout maior para LLMs
        task.correlationId
      );

      llmCircuitBreaker.recordSuccess();
      return result;
    } catch (error) {
      llmCircuitBreaker.recordFailure();
      throw error;
    }
  }

  private async executeWithTimeout<T>(
    fn: () => Promise<T>,
    timeoutMs: number,
    correlationId: string
  ): Promise<T> {
    return new Promise(async (resolve, reject) => {
      const timeout = setTimeout(() => {
        reject(new Error(`Operation timeout after ${timeoutMs}ms`));
      }, timeoutMs);

      try {
        const result = await fn();
        clearTimeout(timeout);
        resolve(result);
      } catch (error) {
        clearTimeout(timeout);
        reject(error);
      }
    });
  }

  private validateTaskInput(dto: CreateTaskDto): void {
    if (!dto.type) {
      throw new Error('Task type is required');
    }

    if (!dto.payload) {
      throw new Error('Task payload is required');
    }

    if (dto.priority && !Object.values(TaskPriority).includes(dto.priority)) {
      throw new Error('Invalid task priority');
    }
  }

  private mapPriorityToBullMQ(priority: TaskPriority): number {
    switch (priority) {
      case TaskPriority.HIGH:
        return 1;
      case TaskPriority.MEDIUM:
        return 5;
      case TaskPriority.LOW:
        return 10;
      default:
        return 5;
    }
  }

  private mapToTaskResponseDto(task: Task): TaskResponseDto {
    return {
      id: task.id,
      type: task.type,
      status: task.status,
      priority: task.priority,
      correlationId: task.correlationId,
      createdAt: task.createdAt.toISOString(),
      startedAt: task.startedAt?.toISOString(),
      completedAt: task.completedAt?.toISOString(),
      result: task.result,
      error: task.error,
      jobId: task.jobId,
    };
  }

  private mapErrorToHttp(error: Error): any {
    const errorMap: Record<string, { status: number; message: string }> = {
      'Task not found': { status: 404, message: 'Task not found' },
      'Database service unavailable': { status: 503, message: 'Service temporarily unavailable' },
      'Redis service unavailable': { status: 503, message: 'Service temporarily unavailable' },
      'LLM service circuit breaker open': { status: 503, message: 'AI service temporarily unavailable' },
    };

    const mapped = errorMap[error.message];
    if (mapped) {
      const httpError = new Error(mapped.message);
      (httpError as any).status = mapped.status;
      return httpError;
    }

    const httpError = new Error('Internal server error');
    (httpError as any).status = 500;
    return httpError;
  }

  private maskSensitiveData(data: any): any {
    const masked = { ...data };
    
    if (masked.payload?.apiKey) {
      masked.payload.apiKey = '***MASKED***';
    }
    
    if (masked.payload?.token) {
      masked.payload.token = '***MASKED***';
    }
    
    if (masked.payload?.password) {
      masked.payload.password = '***MASKED***';
    }

    return masked;
  }

  private generateCorrelationId(): string {
    return `corr_${Date.now()}_${Math.random().toString(36).substr(2, 9)}`;
  }

  async onModuleDestroy(): Promise<void> {
    await this.taskQueue.close();
    await this.redisConnection.quit();
  }
}
```

```sql
-- Função para adicionar partições dinamicamente
CREATE PROCEDURE IF NOT EXISTS diyapp_raw.add_daily_partition(
    IN table_name STRING,
    IN partition_date DATE,
    IN partition_value STRING
)
COMMENT 'Adiciona partição diária para tabelas particionadas'
LANGUAGE SQL
BEGIN
    SET hive.exec.dynamic.partition = true;
    SET hive.exec.dynamic.partition.mode = nonstrict;
    
    EXECUTE IMMEDIATE CONCAT(
        'ALTER TABLE ', table_name, ' ADD IF NOT EXISTS PARTITION (',
        'event_date="', partition_date, '",',
        CASE 
            WHEN table_name LIKE '%execution_logs%' OR table_name LIKE '%colony_heartbeats%' 
            THEN CONCAT('colony_id="', partition_value, '")')
            WHEN table_name LIKE '%performance_metrics%' 
            THEN CONCAT('metric_type="', partition_value, '")')
            WHEN table_name LIKE '%error_events%' 
            THEN CONCAT('error_type="', partition_value, '")')
            ELSE CONCAT('colony_id="', partition_value, '")')
        END
    );
END;

-- Função para compactar partições antigas
CREATE PROCEDURE IF NOT EXISTS diyapp_raw.compress_old_partitions(
    IN table_name STRING,
    IN days_to_keep INT
)
COMMENT 'Compacta partições mais antigas que X dias'
LANGUAGE SQL
BEGIN
    EXECUTE IMMEDIATE CONCAT(
        'ALTER TABLE ', table_name, ' PARTITION (event_date < DATE_SUB(CURRENT_DATE, ', days_to_keep, '))',
        ' SET FILEFORMAT = PARQUET',
        ' SET parquet.compression = "SNAPPY"'
    );
END;
```

```javascript
const jwt = require('jsonwebtoken');
const crypto = require('crypto');

class JWTSecurity {
  constructor() {
    this.secret = process.env.JWT_SECRET || this.generateSecret();
    this.accessTokenExpiry = '15m';
    this.refreshTokenExpiry = '7d';
    this.algorithm = 'HS256';
    this.tokenBlacklist = new Set();
  }

  generateSecret() {
    console.warn('⚠️  Using auto-generated JWT secret. Set JWT_SECRET in production!');
    return crypto.randomBytes(64).toString('hex');
  }

  generateTokens(user) {
    const accessToken = jwt.sign(
      {
        sub: user.id,
        email: user.email,
        role: user.role,
        type: 'access'
      },
      this.secret,
      {
        expiresIn: this.accessTokenExpiry,
        algorithm: this.algorithm,
        issuer: 'diyapp-v12',
        audience: 'diyapp-users'
      }
    );

    const refreshToken = jwt.sign(
      {
        sub: user.id,
        type: 'refresh'
      },
      this.secret,
      {
        expiresIn: this.refreshTokenExpiry,
        algorithm: this.algorithm,
        issuer: 'diyapp-v12',
        audience: 'diyapp-users'
      }
    );

    return { accessToken, refreshToken };
  }

  verifyToken(token, type = 'access') {
    try {
      const decoded = jwt.verify(token, this.secret, {
        algorithms: [this.algorithm],
        issuer: 'diyapp-v12',
        audience: type === 'access' ? 'diyapp-users' : 'diyapp-refresh'
      });

      // Check blacklist
      if (this.tokenBlacklist.has(token)) {
        throw new Error('Token revoked');
      }

      // Check token type
      if (decoded.type !== type) {
        throw new Error(`Invalid token type. Expected: ${type}`);
      }

      return decoded;
    } catch (error) {
      throw new Error(`Token verification failed: ${error.message}`);
    }
  }

  revokeToken(token) {
    try {
      const decoded = jwt.decode(token);
      if (decoded && decoded.exp) {
        const expiresIn = decoded.exp - Math.floor(Date.now() / 1000);
        if (expiresIn > 0) {
          this.tokenBlacklist.add(token);
          // Auto-remove after expiry
          setTimeout(() => {
            this.tokenBlacklist.delete(token);
          }, expiresIn * 1000);
        }
      }
    } catch (error) {
      console.error('Error revoking token:', error);
    }
  }

  rotateRefreshToken(oldRefreshToken, user) {
    this.revokeToken(oldRefreshToken);
    return this.generateTokens(user);
  }
}

module.exports = new JWTSecurity();
```

```javascript
const jwtSecurity = require('./jwt');

const jwtAuth = (req, res, next) => {
  const authHeader = req.headers.authorization;
  
  if (!authHeader || !authHeader.startsWith('Bearer ')) {
    return res.status(401).json({
      error: 'Authentication required',
      code: 'AUTH_REQUIRED'
    });
  }

  const token = authHeader.substring(7);

  try {
    const decoded = jwtSecurity.verifyToken(token, 'access');
    
    // Attach user to request
    req.user = {
      id: decoded.sub,
      email: decoded.email,
      role: decoded.role,
      permissions: getPermissions(decoded.role)
    };

    // Add security context
    req.securityContext = {
      timestamp: new Date().toISOString(),
      authMethod: 'jwt',
      tokenType: 'access'
    };

    next();
  } catch (error) {
    res.status(401).json({
      error: 'Invalid or expired token',
      code: 'TOKEN_INVALID',
      details: process.env.NODE_ENV === 'development' ? error.message : undefined
    });
  }
};

const authorize = (requiredPermissions) => {
  return (req, res, next) => {
    if (!req.user) {
      return res.status(401).json({ error: 'Authentication required' });
    }

    const userPermissions = req.user.permissions;
    const hasPermission = requiredPermissions.every(perm => 
      userPermissions.includes(perm)
    );

    if (!hasPermission) {
      return res.status(403).json({
        error: 'Insufficient permissions',
        code: 'FORBIDDEN',
        required: requiredPermissions,
        has: userPermissions
      });
    }

    next();
  };
};

const checkResourceOwnership = (resourceType, idParam = 'id') => {
  return (req, res, next) => {
    const resourceId = req.params[idParam];
    const userId = req.user.id;
    const userRole = req.user.role;

    // Admins bypass ownership check
    if (userRole === 'admin') {
      return next();
    }

    // For user-specific resources, check ownership
    if (resourceType === 'user' && resourceId !== userId) {
      return res.status(403).json({
        error: 'Cannot access other user resources',
        code: 'RESOURCE_OWNERSHIP_VIOLATION'
      });
    }

    // Add ownership check for other resource types here
    next();
  };
};

function getPermissions(role) {
  const permissions = {
    admin: [
      'users:read',
      'users:write',
      'users:delete',
      'content:read',
      'content:write',
      'content:delete',
      'system:admin'
    ],
    user: [
      'profile:read',
      'profile:write',
      'content:read:own',
      'content:write:own',
      'content:delete:own'
    ],
    viewer: [
      'content:read'
    ]
  };

  return permissions[role] || permissions.viewer;
}
```

```javascript
const rateLimit = require('express-rate-limit');
const Redis = require('ioredis');
const RedisStore = require('rate-limit-redis');

class RateLimiter {
  constructor() {
    this.redisClient = process.env.REDIS_URL 
      ? new Redis(process.env.REDIS_URL)
      : null;

    this.store = this.redisClient 
      ? new RedisStore({
          client: this.redisClient,
          prefix: 'rate-limit:'
        })
      : undefined;
  }

  // Global rate limiter for all requests
  global = rateLimit({
    windowMs: 15 * 60 * 1000, // 15 minutes
    max: 100, // Limit each IP to 100 requests per windowMs
    message: {
      error: 'Too many requests from this IP',
      code: 'RATE_LIMIT_EXCEEDED',
      retryAfter: '15 minutes'
    },
    standardHeaders: true,
    legacyHeaders: false,
    skipSuccessfulRequests: false,
    keyGenerator: (req) => {
      // Use IP + user agent for more accurate identification
      return `${req.ip}-${req.headers['user-agent']}`;
    },
    skip: (req) => {
      // Skip rate limiting for health checks
      return req.path === '/health';
    },
    store: this.store
  });

  // Stricter rate limiter for API endpoints
  api = rateLimit({
    windowMs: 5 * 60 * 1000, // 5 minutes
    max: 50, // 50 requests per 5 minutes
    message: {
      error: 'API rate limit exceeded',
      code: 'API_RATE_LIMIT',
      retryAfter: '5 minutes'
    },
    standardHeaders: true
  });

  // Very strict limiter for authentication endpoints
  auth = rateLimit({
    windowMs: 60 * 60 * 1000, // 1 hour
    max: 5, // 5 failed attempts per hour
    message: {
      error: 'Too many authentication attempts',
      code: 'AUTH_RATE_LIMIT',
      retryAfter: '1 hour'
    },
    skipSuccessfulRequests: true // Only count failed attempts
  });

  // Dynamic rate limiting based on user tier
  dynamicLimiter(tier = 'free') {
    const limits = {
      free: { windowMs: 15 * 60 * 1000, max: 100 },
      pro: { windowMs: 15 * 60 * 1000, max: 1000 },
      enterprise: { windowMs: 15 * 60 * 1000, max: 10000 }
    };

    const limit = limits[tier] || limits.free;

    return rateLimit({
      windowMs: limit.windowMs,
      max: limit.max,
      keyGenerator: (req) => req.user?.id || req.ip,
      message: {
        error: `Rate limit exceeded for ${tier} tier`,
        code: 'TIER_RATE_LIMIT',
        tier: tier,
        limit: limit.max,
        window: `${limit.windowMs / 60000} minutes`
      }
    });
  }

  // Cleanup
  async cleanup() {
    if (this.redisClient) {
      await this.redisClient.quit();
    }
  }
}

module.exports = new RateLimiter();
```

## 2. server.js

```javascript
const express = require('express');
const http = require('http');
const socketIo = require('socket.io');
const path = require('path');
const AutoTuningAgent = require('./src/agent/AutoTuningAgent');
const logger = require('./src/utils/logger');

const app = express();
const server = http.createServer(app);
const io = socketIo(server);

// Middleware
app.use(express.json());
app.use(express.static(path.join(__dirname, 'public')));

// Banco de dados de métricas
const MetricsDB = require('./src/database/metrics-db');
const metricsDB = new MetricsDB();

// API Routes
const metricsAPI = require('./src/api/metrics-api')(metricsDB);
const tuningAPI = require('./src/api/tuning-api')(metricsDB);

app.use('/api/metrics', metricsAPI);
app.use('/api/tuning', tuningAPI);

// Inicializar agente de auto-tuning
const autoTuningAgent = new AutoTuningAgent({
  db: metricsDB,
  io: io,
  config: {
    collectionInterval: 30000, // 30 segundos
    analysisInterval: 60000,   // 1 minuto
    optimizationInterval: 300000 // 5 minutos
  }
});

// Rota principal - Dashboard
app.get('/', (req, res) => {
  res.sendFile(path.join(__dirname, 'index.html'));
});

// WebSocket para atualizações em tempo real
io.on('connection', (socket) => {
  logger.info('Cliente conectado ao dashboard');
  
  socket.on('request-metrics', async () => {
    const metrics = await metricsDB.getRecentMetrics(50);
    socket.emit('metrics-update', metrics);
  });
  
  socket.on('request-optimizations', async () => {
    const optimizations = await metricsDB.getRecentOptimizations(20);
    socket.emit('optimizations-update', optimizations);
  });
  
  socket.on('disconnect', () => {
    logger.info('Cliente desconectado');
  });
});

// Iniciar servidor
const PORT = process.env.PORT || 3000;
server.listen(PORT, async () => {
  logger.info(`Servidor rodando na porta ${PORT}`);
  logger.info(`Dashboard disponível em: http://localhost:${PORT}`);
  
  // Inicializar banco de dados
  await metricsDB.initialize();
  
  // Iniciar agente de auto-tuning
  await autoTuningAgent.start();
});

// Graceful shutdown
process.on('SIGTERM', async () => {
  logger.info('Recebido SIGTERM, encerrando...');
  await autoTuningAgent.stop();
  await metricsDB.close();
  process.exit(0);
});
```

## 2. Anonimizador de Dados (src/compliance/telemetry/anonymizer.js)

```javascript
// src/compliance/telemetry/anonymizer.js
const crypto = require('crypto');
const config = require('../../../config/compliance-config');

class DataAnonymizer {
  constructor() {
    this.salt = config.anonymization.salt;
    this.pepper = config.anonymization.pepper;
    this.methods = config.anonymization.methods;
  }

  /**
   * Anonimiza dados pessoais conforme LGPD/GDPR
   */
  anonymize(data, dataType) {
    if (!config.anonymization.enabled) {
      return data;
    }

    switch (dataType) {
      case 'ip_address':
        return this.anonymizeIpAddress(data);
      case 'email':
        return this.anonymizeEmail(data);
      case 'phone':
        return this.anonymizePhone(data);
      case 'user_id':
        return this.pseudonymizeUserId(data);
      case 'location':
        return this.anonymizeLocation(data);
      case 'name':
        return this.anonymizeName(data);
      default:
        return this.genericAnonymization(data);
    }
  }

  /**
   * Anonimização de IP com hash salt+pepper
   */
  anonymizeIpAddress(ip) {
    if (!ip || ip === '::1' || ip === '127.0.0.1') {
      return 'localhost';
    }

    const method = this.methods.ipAddress;
    if (method === 'hash_salt_pepper') {
      // Mantém apenas os primeiros 2 octetos para análise geográfica agregada
      const parts = ip.split('.');
      if (parts.length === 4) {
        return `${parts[0]}.${parts[1]}.0.0`;
      }
      
      // Para IPv6, simplifica
      if (ip.includes(':')) {
        const hash = crypto
          .createHash('sha256')
          .update(ip + this.salt + this.pepper)
          .digest('hex')
          .substring(0, 16);
        return `ipv6:${hash}`;
      }
    }
    
    return 'anonymized';
  }

  /**
   * Mascaramento parcial de email
   */
  anonymizeEmail(email) {
    if (!email || !email.includes('@')) {
      return 'anonymous@domain.com';
    }

    const [localPart, domain] = email.split('@');
    const method = this.methods.email;
    
    if (method === 'partial_masking') {
      // Mantém primeiro e último caractere, mascara o meio
      if (localPart.length <= 2) {
        return `${localPart.charAt(0)}***@${domain}`;
      }
      
      const maskedLocal = 
        localPart.charAt(0) + 
        '*'.repeat(Math.min(localPart.length - 2, 3)) + 
        localPart.charAt(localPart.length - 1);
      
      return `${maskedLocal}@${domain}`;
    }
    
    // Fallback: hash completo
    const hash = crypto
      .createHash('sha256')
      .update(email + this.salt)
      .digest('hex')
      .substring(0, 10);
    
    return `user_${hash}@anonymized.domain`;
  }

  /**
   * Pseudonimização de User ID
   */
  pseudonymizeUserId(userId) {
    const method = this.methods.userId;
    
    if (method === 'pseudonymization') {
      // Gera ID pseudônimo consistente
      const hash = crypto
        .createHash('sha256')
        .update(userId.toString() + this.salt)
        .digest('hex')
        .substring(0, 16);
      
      return `usr_${hash}`;
    }
    
    return `anon_${crypto.randomBytes(8).toString('hex')}`;
  }

  /**
   * Redução de precisão de localização
   */
  anonymizeLocation(location) {
    if (!location || !location.latitude || !location.longitude) {
      return { latitude: 0, longitude: 0, precision: 'country' };
    }
    
    // Reduz precisão para nível de cidade (~10km)
    const precision = 100; // 1 grau ≈ 111km, 0.1 ≈ 11km
    const lat = Math.round(location.latitude * precision) / precision;
    const lon = Math.round(location.longitude * precision) / precision;
    
    return {
      latitude: lat,
      longitude: lon,
      precision: 'city',
      anonymized: true
    };
  }

  /**
   * Verifica se dados são sensíveis
   */
  isSensitiveData(fieldName, value) {
    const fieldLower = fieldName.toLowerCase();
    
    // Verifica categorias sensíveis
    for (const category of config.dataProtection.sensitiveDataCategories) {
      if (fieldLower.includes(category)) {
        return true;
      }
    }
    
    // Verifica padrões de dados sensíveis
    const sensitivePatterns = [
      /senha|password|passwd/i,
      /token|auth|credential/i,
      /cpf|cnpj|ssn|social.security/i,
      /cartao|credit.card|card.number/i,
      /conta.bancaria|bank.account/i
    ];
    
    return sensitivePatterns.some(pattern => pattern.test(fieldName));
  }

  /**
   * Processa objeto completo anonimizando campos sensíveis
   */
  processObject(obj, context = 'telemetry') {
    if (!obj || typeof obj !== 'object') {
      return obj;
    }
    
    const processed = Array.isArray(obj) ? [] : {};
    
    for (const [key, value] of Object.entries(obj)) {
      if (value === null || value === undefined) {
        processed[key] = value;
        continue;
      }
      
      // Anonimiza dados sensíveis
      if (this.isSensitiveData(key, value)) {
        processed[key] = '[SENSITIVE_DATA_REMOVED]';
        continue;
      }
      
      // Anonimiza tipos conhecidos
      if (typeof value === 'string') {
        if (key.includes('ip') || key === 'ipAddress' || key === 'clientIp') {
          processed[key] = this.anonymizeIpAddress(value);
        } else if (key.includes('email') || key === 'userEmail') {
          processed[key] = this.anonymizeEmail(value);
        } else if (key.includes('phone') || key === 'telefone') {
          processed[key] = this.anonymizePhone(value);
        } else if (key.includes('id') && !key.includes('anonymous')) {
          processed[key] = this.pseudonymizeUserId(value);
        } else {
          processed[key] = value;
        }
      } else if (typeof value === 'object') {
        // Recursão para objetos aninhados
        processed[key] = this.processObject(value, context);
      } else {
        processed[key] = value;
      }
    }
    
    // Adiciona metadados de anonimização
    if (typeof processed === 'object' && !Array.isArray(processed)) {
      processed._compliance = {
        anonymized: true,
        timestamp: new Date().toISOString(),
        context: context,
        method: 'v12_anonymization_suite'
      };
    }
    
    return processed;
  }

  /**
   * Gera hash determinístico para análise agregada
   */
  generateAggregateHash(data, aggregationLevel) {
    const dataString = JSON.stringify(data);
    return crypto
      .createHash('sha256')
      .update(dataString + this.salt + aggregationLevel)
      .digest('hex')
      .substring(0, 12);
  }
}

module.exports = new DataAnonymizer();
```

    const consentId = this.generateConsentId(sessionId);
    
    const consent = {
      id: consentId,
      sessionId: sessionId,
      timestamp: new Date().toISOString(),
      version: this.consentVersion,
      preferences: userPreferences || config.consent.defaultConsent,
      legalBasis: {
        necessary: 'legitimate_interest',
        performance: userPreferences?.performance ? 'consent' : 'none',
        functional: userPreferences?.functional ? 'consent' : 'none',
        marketing: userPreferences?.marketing ? 'consent' : 'none'
      },
      history: []
    };

   */

    const consent = this.consentStore.get(consentId);
    if (!consent) {
      throw new Error('Consent not found');
    }
    
    const oldPreferences = { ...consent.preferences };
    
    // Atualiza apenas as chaves permitidas
    Object.keys(updates).forEach(key => {
      if (key in consent.preferences) {
        consent.preferences[key] = updates[key];
        
        // Atualiza base legal
        if (key !== 'necessary') {
          consent.legalBasis[key] = updates[key] ? 'consent' : 'none';
        }
      }
    });
    
    consent.timestamp = new Date().toISOString();
    consent.history.push({
      action: 'updated',
      timestamp: consent.timestamp,
      oldPreferences,
      newPreferences: { ...consent.preferences }
    });
    
    this.consentStore.set(consentId, consent);
    this.logConsentChange(consentId, 'updated', updates);
    
    return consent;
  }

    }

```python
        """Adiciona um critério ao conjunto"""
        self.criteria.append(criterion)

        for criteria_type in AcceptanceCriteriaType:
```
