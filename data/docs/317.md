# DIYAPP Evolution - V12 Core - Arquitetura de Microsserviços

## ADR-001: Arquitetura de Microsserviços V12

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista Backend

### CONTEXTO:
O DIYAPP evoluiu de uma aplicação monolítica para uma arquitetura distribuída com mais de 15 serviços. A comunicação atual via REST/HTTP está causando:
1. Acoplamento temporal entre serviços
2. Dificuldade em rastrear transações distribuídas
3. Latência acumulada em cadeias de chamadas síncronas
4. Complexidade no versionamento de APIs

### DECISÃO:
Implementar arquitetura de microsserviços V12 baseada em:
1. **gRPC para comunicação síncrona** entre serviços com contratos fortemente tipados
2. **Apache Kafka para eventos assíncronos** e desacoplamento temporal
3. **Service Mesh com Istio** para gerenciamento de tráfego e resiliência
4. **OpenTelemetry para observabilidade unificada**

### OPÇÕES CONSIDERADAS:

**Opção A: REST + RabbitMQ (Status Quo)**
- Prós: Equipe familiarizada, fácil debug com HTTP
- Contras: Acoplamento síncrono, schemas não validados em tempo de compilação
- Escolhida: ❌

**Opção B: gRPC + Kafka + Istio**
- Prós: Performance 7-10x melhor que REST, schemas protobuf, streaming bidirecional, observabilidade nativa
- Contras: Curva de aprendizado, tooling menos maduro para debug
- Escolhida: ✅

**Opção C: GraphQL + NATS**
- Prós: Flexibilidade no frontend, schemas tipados
- Contras: Complexidade no backend, over-fetching em microserviços
- Escolhida: ❌

### CONSEQUÊNCIAS:
**Positivas:**
- Redução de latência em 60-70%
- Contratos de API validados em tempo de compilação
- Desacoplamento temporal com eventos
- Observabilidade unificada com tracing distribuído

**Negativas:**
- 3-4 semanas para treinamento da equipe
- Complexidade operacional aumentada
- Necessidade de ferramentas específicas para debug gRPC

**Riscos:**
1. **Risco de adoção:** Mitigar com pair programming e workshops
2. **Risco de performance Kafka:** Mitigar com particionamento adequado e monitoring
3. **Risco de complexidade Istio:** Mitigar com configuração como código e GitOps

**Revisão:** 2024-04-15 (3 meses após implementação)

  repeated string roles = 4;

  string user_id = 1;
  EventType event_type = 2;
  google.protobuf.Timestamp timestamp = 3;
  map<string, string> metadata = 4;
}

  USER_UPDATED = 1;
  USER_DELETED = 2;

### 2. Serviço de Autenticação (`services/auth-service/src/grpc/server.ts`)

```typescript
import * as grpc from '@grpc/grpc-js';
import * as protoLoader from '@grpc/proto-loader';
import { AuthServiceHandlers } from '../proto/auth/v1/auth';
import { login, validateToken, createUser } from '../controllers/auth';
import { setupTracing } from '../observability/tracing';
import { recordMetrics } from '../observability/metrics';
import { KafkaProducer } from '../kafka/producer';

const PROTO_PATH = __dirname + '/../../../proto/auth/v1/auth.proto';

const packageDefinition = protoLoader.loadSync(PROTO_PATH, {
  keepCase: true,
  longs: String,
  enums: String,
  defaults: true,
  oneofs: true,
});

const protoDescriptor = grpc.loadPackageDefinition(packageDefinition);
const authProto = protoDescriptor.diyapp.auth.v1 as any;

class AuthServer {
  private server: grpc.Server;
  private kafkaProducer: KafkaProducer;
  private tracer: any;

  constructor() {
    this.server = new grpc.Server();
    this.kafkaProducer = new KafkaProducer();
    this.tracer = setupTracing('auth-service');
  }

  async start(): Promise<void> {
    const handlers: AuthServiceHandlers = {
      Login: this.wrapWithObservability(this.handleLogin.bind(this), 'Login'),
      ValidateToken: this.wrapWithObservability(this.handleValidateToken.bind(this), 'ValidateToken'),
      CreateUser: this.wrapWithObservability(this.handleCreateUser.bind(this), 'CreateUser'),
      StreamUserEvents: this.handleStreamUserEvents.bind(this),
    };

    this.server.addService(authProto.AuthService.service, handlers);
    
    await this.kafkaProducer.connect();
    
    this.server.bindAsync(
      `0.0.0.0:${process.env.GRPC_PORT || 50051}`,
      grpc.ServerCredentials.createInsecure(),
      (error, port) => {
        if (error) {
          console.error('Failed to bind gRPC server:', error);
          process.exit(1);
        }
        console.log(`gRPC server running on port ${port}`);
      }
    );
  }

  private wrapWithObservability(
    handler: Function,
    operationName: string
  ): Function {
    return async (call: any, callback: any) => {
      const span = this.tracer.startSpan(operationName);
      
      try {
        // Adicionar contexto de tracing aos headers
        const metadata = new grpc.Metadata();
        this.tracer.inject(span, 'grpc', metadata);
        
        // Executar handler
        const result = await handler(call, callback);
        
        // Registrar métricas
        recordMetrics(operationName, 'success', Date.now() - span.startTime);
        
        span.setStatus({ code: 1 }); // OK
        span.end();
        
        return result;
      } catch (error) {
        // Registrar métricas de erro
        recordMetrics(operationName, 'error', Date.now() - span.startTime);
        
        span.setStatus({ code: 2, message: error.message }); // ERROR
        span.recordException(error);
        span.end();
        
        throw error;
      }
    };
  }

  private async handleLogin(call: any, callback: any): Promise<void> {
    const { email, password } = call.request;
    
    try {
      const result = await login(email, password);
      
      // Publicar evento no Kafka
      await this.kafkaProducer.send('user-events', {
        user_id: result.user.id,
        event_type: 'LOGIN_SUCCESS',
        timestamp: new Date().toISOString(),
        metadata: {
          ip_address: call.metadata.get('x-forwarded-for')[0] || 'unknown',
          user_agent: call.metadata.get('user-agent')[0] || 'unknown',
        },
      });
      
      callback(null, result);
    } catch (error) {
      callback({
        code: grpc.status.UNAUTHENTICATED,
        message: 'Invalid credentials',
      });
    }
  }

  private async handleValidateToken(call: any, callback: any): Promise<void> {
    const token = call.request.token;
    const result = await validateToken(token);
    callback(null, result);
  }

  private async handleCreateUser(call: any, callback: any): Promise<void> {
    const userData = call.request;
    const result = await createUser(userData);
    
    // Publicar evento no Kafka
    await this.kafkaProducer.send('user-events', {
      user_id: result.user.id,
      event_type: 'USER_CREATED',
      timestamp: new Date().toISOString(),
    });
    
    callback(null, result);
  }

  private handleStreamUserEvents(call: any): void {
    const { user_id } = call.request;
    
    // Configurar stream de eventos em tempo real
    const eventInterval = setInterval(() => {
      if (call.cancelled) {
        clearInterval(eventInterval);
        return;
      }
      
      call.write({
        user_id,
        event_type: 'HEARTBEAT',
        timestamp: new Date().toISOString(),
      });
    }, 30000);
    
    call.on('cancelled', () => {
      clearInterval(eventInterval);
    });
  }

  async shutdown(): Promise<void> {
    await new Promise((resolve) => {
      this.server.tryShutdown(resolve);
    });
    await this.kafkaProducer.disconnect();
  }
}

export { AuthServer };
```

import { NodeTracerProvider } from '@opentelemetry/sdk-trace-node';
import { SimpleSpanProcessor } from '@opentelemetry/sdk-trace-base';
import { JaegerExporter } from '@opentelemetry/exporter-jaeger';
import { Resource } from '@opentelemetry/resources';
import { SemanticResourceAttributes } from '@opentelemetry/semantic-conventions';
import { GrpcInstrumentation } from '@opentelemetry/instrumentation-grpc';
import { KafkaJsInstrumentation } from 'opentelemetry-instrumentation-kafkajs';
import { registerInstrumentations } from '@opentelemetry/instrumentation';

export function setupTracing(serviceName: string): any {
  // Configurar provedor de tracing
  const provider = new NodeTracerProvider({
    resource: new Resource({
      [SemanticResourceAttributes.SERVICE_NAME]: serviceName,
      [SemanticResourceAttributes.DEPLOYMENT_ENVIRONMENT]: process.env.NODE_ENV || 'development',
    }),
  });

  // Configurar exportador Jaeger
  const jaegerExporter = new JaegerExporter({
    endpoint: process.env.JAEGER_ENDPOINT || 'http://jaeger:14268/api/traces',
  });

  // Adicionar processador de spans
  provider.addSpanProcessor(new SimpleSpanProcessor(jaegerExporter));

  // Registrar provedor
  provider.register();

  // Registrar instrumentações automáticas
  registerInstrumentations({
    instrumentations: [
      new GrpcInstrumentation({
        ignoreGrpcMethods: ['health.check'],
      }),
      new KafkaJsInstrumentation(),
    ],
  });

  return provider.getTracer(serviceName);
}

// Middleware para tracing HTTP (para APIs REST ainda existentes)
import { context, trace } from '@opentelemetry/api';

export function tracingMiddleware(req: any, res: any, next: Function): void {
  const tracer = trace.getTracer('http-server');
  const span = tracer.startSpan(`${req.method} ${req.path}`);
  
  // Adicionar atributos ao span
  span.setAttributes({
    'http.method': req.method,
    'http.path': req.path,
    'http.user_agent': req.get('user-agent'),
    'http.client_ip': req.ip,
  });

  // Armazenar span no contexto da requisição
  const ctx = trace.setSpan(context.active(), span);
  req.tracingContext = ctx;

  // Finalizar span quando a resposta for enviada
  res.on('finish', () => {
    span.setAttribute('http.status_code', res.statusCode);
    span.setStatus({
      code: res.statusCode < 400 ? 1 : 2,
      message: res.statusMessage,
    });
    span.end();
  });

  next();
}

  // Exportador para Prometheus

      port: 9464,

    () => {
      console.log(`Prometheus metrics server running on port 9464`);

  // Exportador para OpenTelemetry Collector

    url: process.env.OTLP_ENDPOINT || 'http://otel-collector:4318/v1/metrics',
  });

  // Adicionar leitores de métricas
  provider.addMetricReader(new PeriodicExportingMetricReader({
    exporter: otlpExporter,
    exportIntervalMillis: 60000, // 1 minuto
  }));

  provider.addMetricReader(prometheusExporter);
}

export function recordMetrics(
  operation: string,
  status: 'success' | 'error',
  duration: number
): void {
  const key = `${operation}_${status}`;
  
  if (!metrics.has(key)) {
    metrics.set(key, {
      count: 0,
      totalDuration: 0,
      lastUpdated: Date.now(),
    });
  }

  const metric = metrics.get(key);
  metric.count++;
  metric.totalDuration += duration;
  metric.lastUpdated = Date.now();

  // Aqui você enviaria para o sistema de métricas
  console.log(`[METRIC] ${operation}.${status}: ${duration}ms`);
}

// Métricas customizadas
export class BusinessMetrics {
  private static instance: BusinessMetrics;
  private userRegistrations: Map<string, number> = new Map();
  private failedLogins: Map<string, number> = new Map();

  static getInstance(): BusinessMetrics {
    if (!BusinessMetrics.instance) {
      BusinessMetrics.instance = new BusinessMetrics();
    }
    return BusinessMetrics.instance;
  }

  recordUserRegistration(source: string): void {
    const count = this.userRegistrations.get(source) || 0;
    this.userRegistrations.set(source, count + 1);
    
    // Enviar para sistema de métricas
    console.log(`[BUSINESS_METRIC] user_registration source=${source} count=${count + 1}`);
  }

  recordFailedLogin(userId: string, reason: string): void {
    const key = `${userId}_${reason}`;
    const count = this.failedLogins.get(key) || 0;
    this.failedLogins.set(key, count + 1);
    
    // Alertar se muitas tentativas falhas
    if (count + 1 >= 5) {
      console.warn(`[SECURITY_ALERT] Multiple failed logins for user ${userId}`);
    }
  }
}

class StructuredLogger:
    """Logger estruturado em JSON com correlation ID"""
    
    def __init__(self, service_name: str = "v12-core"):
        self.service_name = service_name
        self.logger = logging.getLogger(service_name)
        self.logger.setLevel(logging.INFO)
        
        # Handler para JSON
        handler = logging.StreamHandler()
        formatter = logging.Formatter(
            '{"timestamp": "%(asctime)s", "level": "%(levelname)s", '
            '"service": "%(name)s", "correlation_id": "%(correlation_id)s", '
            '"user_id": "%(user_id)s", "endpoint": "%(endpoint)s", '
            '"duration_ms": %(duration_ms)d, "message": "%(message)s"}'
        )
        handler.setFormatter(formatter)
        self.logger.addHandler(handler)
    
    def log_request(
        self,
        correlation_id: str,
        user_id: Optional[str],
        endpoint: str,
        duration_ms: int,
        level: str = "info",
        message: str = "",
        **extra
    ):
        """Log estruturado para requests HTTP"""
        log_data = {
            "correlation_id": correlation_id,
            "user_id": user_id or "anonymous",
            "endpoint": endpoint,
            "duration_ms": duration_ms,
            **extra
        }
        
        log_method = getattr(self.logger, level.lower(), self.logger.info)
        log_method(message, extra=log_data)

logger = StructuredLogger()

class CircuitState(Enum):
    CLOSED = "closed"      # Operação normal
    OPEN = "open"          # Circuito aberto - falhas frequentes
    HALF_OPEN = "half_open" # Testando se serviço recuperou

class CircuitBreaker:
    """Implementação de Circuit Breaker com estado e métricas"""
    
    def __init__(
        self,
        name: str,
        failure_threshold: int = 5,
        recovery_timeout: int = 30,
        half_open_max_calls: int = 3
    ):
        self.name = name
        self.failure_threshold = failure_threshold
        self.recovery_timeout = recovery_timeout
        self.half_open_max_calls = half_open_max_calls
        
        self.state = CircuitState.CLOSED
        self.failure_count = 0
        self.last_failure_time = None
        self.half_open_success_count = 0
        
    def record_success(self):
        """Registra sucesso e reseta contadores"""
        if self.state == CircuitState.HALF_OPEN:
            self.half_open_success_count += 1
            if self.half_open_success_count >= self.half_open_max_calls:
                self._close_circuit()
        else:
            self._close_circuit()
    
    def record_failure(self):
        """Registra falha e atualiza estado do circuito"""
        self.failure_count += 1
        self.last_failure_time = time.time()
        
        if self.state == CircuitState.CLOSED:
            if self.failure_count >= self.failure_threshold:
                self._open_circuit()
        elif self.state == CircuitState.HALF_OPEN:
            self._open_circuit()
    
    def _open_circuit(self):
        """Abre o circuito"""
        self.state = CircuitState.OPEN
        logger.logger.warning(
            f"Circuit {self.name} OPENED",
            extra={"circuit_name": self.name, "state": "open"}
        )
    
    def _close_circuit(self):
        """Fecha o circuito"""
        self.state = CircuitState.CLOSED
        self.failure_count = 0
        self.half_open_success_count = 0
        self.last_failure_time = None
    
    def _try_half_open(self):
        """Tenta transicionar para half-open"""
        if (self.last_failure_time and 
            time.time() - self.last_failure_time > self.recovery_timeout):
            self.state = CircuitState.HALF_OPEN
            logger.logger.info(
                f"Circuit {self.name} HALF_OPEN",
                extra={"circuit_name": self.name, "state": "half_open"}
            )
    
    def allow_request(self) -> bool:
        """Verifica se request é permitido"""
        if self.state == CircuitState.OPEN:
            self._try_half_open()
            return self.state != CircuitState.OPEN
        return True
    
    def execute(self, func, *args, **kwargs):
        """Executa função com proteção do circuit breaker"""
        if not self.allow_request():
            raise CircuitBreakerOpenError(f"Circuit {self.name} is OPEN")
        
        try:
            result = func(*args, **kwargs)
            self.record_success()
            return result
        except Exception as e:
            self.record_failure()
            raise

class CircuitBreakerOpenError(Exception):
    """Exceção quando circuit breaker está aberto"""
    pass

class EventType(Enum):
    USER_CREATED = "user_created"
    USER_UPDATED = "user_updated"
    USER_DELETED = "user_deleted"

class DomainEvent(BaseModel):
    """Evento de domínio para Event Sourcing"""
    event_id: str = Field(default_factory=lambda: str(uuid4()))
    aggregate_id: str
    event_type: EventType
    payload: Dict[str, Any]
    metadata: Dict[str, Any] = Field(default_factory=dict)
    timestamp: datetime = Field(default_factory=datetime.utcnow)
    version: int = Field(default=1)

class EventStore:
    """Armazenamento de eventos para Event Sourcing"""
    
    def __init__(self, db_pool: asyncpg.Pool):
        self.db_pool = db_pool
    
    async def create_tables(self):
        """Cria tabelas para event store"""
        async with self.db_pool.acquire() as conn:
            await conn.execute("""
                CREATE TABLE IF NOT EXISTS events (
                    event_id UUID PRIMARY KEY,
                    aggregate_id VARCHAR(255) NOT NULL,
                    event_type VARCHAR(50) NOT NULL,
                    payload JSONB NOT NULL,
                    metadata JSONB NOT NULL,
                    timestamp TIMESTAMPTZ NOT NULL,
                    version INTEGER NOT NULL,
                    created_at TIMESTAMPTZ DEFAULT NOW()
                );
                
                CREATE INDEX IF NOT EXISTS idx_events_aggregate_id 
                ON events(aggregate_id);
                
                CREATE INDEX IF NOT EXISTS idx_events_timestamp 
                ON events(timestamp);
            """)
    
    async def save_event(self, event: DomainEvent) -> str:
        """Salva evento no event store"""
        async with self.db_pool.acquire() as conn:
            await conn.execute("""
                INSERT INTO events 
                (event_id, aggregate_id, event_type, payload, metadata, timestamp, version)
                VALUES ($1, $2, $3, $4, $5, $6, $7)
            """, 
            event.event_id, event.aggregate_id, event.event_type.value,
            json.dumps(event.payload), json.dumps(event.metadata),
            event.timestamp, event.version)
            
            return event.event_id
    
    async def get_events_for_aggregate(
        self, 
        aggregate_id: str,
        from_version: int = 0
    ) -> List[DomainEvent]:
        """Recupera eventos de um aggregate"""
        async with self.db_pool.acquire() as conn:
            rows = await conn.fetch("""
                SELECT * FROM events 
                WHERE aggregate_id = $1 AND version > $2
                ORDER BY version ASC
            """, aggregate_id, from_version)
            
            events = []
            for row in rows:
                event = DomainEvent(
                    event_id=str(row['event_id']),
                    aggregate_id=row['aggregate_id'],
                    event_type=EventType(row['event_type']),
                    payload=row['payload'],
                    metadata=row['metadata'],
                    timestamp=row['timestamp'],
                    version=row['version']
                )
                events.append(event)
            
            return events

class UserReadRepository:
    """Repository para queries (read side)"""
    
    def __init__(self, db_pool: asyncpg.Pool):
        self.db_pool = db_pool
    
    async def create_tables(self):
        """Cria tabelas para read model"""
        async with self.db_pool.acquire() as conn:
            await conn.execute("""
                CREATE TABLE IF NOT EXISTS users_read (
                    user_id VARCHAR(255) PRIMARY KEY,
                    email VARCHAR(255) UNIQUE NOT NULL,
                    name VARCHAR(100) NOT NULL,
                    role VARCHAR(20) NOT NULL,
                    created_at TIMESTAMPTZ NOT NULL,
                    updated_at TIMESTAMPTZ NOT NULL,
                    version INTEGER NOT NULL DEFAULT 1
                );
                
                CREATE INDEX IF NOT EXISTS idx_users_email 
                ON users_read(email);
                
                CREATE INDEX IF NOT EXISTS idx_users_role 
                ON users_read(role);
            """)
    
    @retry(
        retry=retry_if_exception_type(asyncpg.PostgresConnectionError),
        stop=stop_after_attempt(3),
        wait=wait_exponential(multiplier=1, min=4, max=10),
        before_sleep=before_sleep_log(logger.logger, logging.WARNING)
    )
    async def get_user(self, user_id: str) -> Optional[Dict]:
        """Busca usuário por ID (parametrizado - segurança)"""
        async with self.db_pool.acquire() as conn:
            row = await conn.fetchrow(
                "SELECT * FROM users_read WHERE user_id = $1",
                user_id  # Parâmetro seguro
            )
            
            if row:
                return dict(row)
            return None
    
    @retry(
        retry=retry_if_exception_type(asyncpg.PostgresConnectionError),
        stop=stop_after_attempt(3),
        wait=wait_exponential(multiplier=1, min=4, max=10)
    )
    async def list_users(
        self, 
        page: int = 1, 
        limit: int = 20,
        role: Optional[str] = None
    ) -> Tuple[List[Dict], int]:
        """Lista usuários com paginação"""
        offset = (page - 1) * limit
        
        async with self.db_pool.acquire() as conn:
            # Query parametrizada
            if role:
                rows = await conn.fetch(
                    "SELECT * FROM users_read WHERE role = $1 "
                    "ORDER BY created_at DESC LIMIT $2 OFFSET $3",
                    role, limit, offset
                )
                count_row = await conn.fetchrow(
                    "SELECT COUNT(*) FROM users_read WHERE role = $1",
                    role
                )
            else:
                rows = await conn.fetch(
                    "SELECT * FROM users_read "
                    "ORDER BY created_at DESC LIMIT $1 OFFSET $2",
                    limit, offset
                )
                count_row = await conn.fetchrow(
                    "SELECT COUNT(*) FROM users_read"
                )
            
            total = count_row['count'] if count_row else 0
            return [dict(row) for row in rows], total

class UserWriteRepository:
    """Repository para commands (write side)"""
    
    def __init__(self, db_pool: asyncpg.Pool, event_store: EventStore):
        self.db_pool = db_pool
        self.event_store = event_store
    
    async def create_user(self, command: CreateUserCommand, metadata: Dict) -> str:
        """Cria usuário e dispara evento"""
        user_id = str(uuid4())
        now = datetime.utcnow()
        
        async with self.db_pool.acquire() as conn:
            async with conn.transaction():
                # Inserção parametrizada
                await conn.execute("""
                    INSERT INTO users_read 
                    (user_id, email, name, role, created_at, updated_at)
                    VALUES ($1, $2, $3, $4, $5, $6)
                """, 
                user_id, command.email, command.name, 
                command.role, now, now)
                
                # Salva evento
                event = DomainEvent(
                    aggregate_id=user_id,
                    event_type=EventType.USER_CREATED,
                    payload={
                        "email": command.email,
                        "name": command.name,
                        "role": command.role
                    },
                    metadata=metadata
                )
                
                await self.event_store.save_event(event)
                
                return user_id
    
    async def update_user(self, command: UpdateUserCommand, metadata: Dict) -> bool:
        """Atualiza usuário e dispara evento"""
        async with self.db_pool.acquire() as conn:
            async with conn.transaction():
                # Busca usuário atual
                current = await conn.fetchrow(
                    "SELECT * FROM users_read WHERE user_id = $1",
                    command.user_id
                )
                
                if not current:
                    return False
                
                # Prepara updates
                updates = {}
                if command.name is not None:
                    updates['name'] = command.name
                if command.role is not None:
                    updates['role'] = command.role
                
                if not updates:
                    return True
                
                # Query dinâmica parametrizada
                set_clause = []
                params = []
                param_idx = 1
                
                for field, value in updates.items():
                    set_clause.append(f"{field} = ${param_idx}")
                    params.append(value)
                    param_idx += 1
                
                set_clause.append("updated_at = $" + str(param_idx))
                params.append(datetime.utcnow())
                param_idx += 1
                
                params.append(command.user_id)
                
                query = f"""
                    UPDATE users_read 
                    SET {', '.join(set_clause)}
                    WHERE user_id = ${param_idx}
                    RETURNING *
                """
                
                updated = await conn.fetchrow(query, *params)
                
                # Salva evento
                event = DomainEvent(
                    aggregate_id=command.user_id,
                    event_type=EventType.USER_UPDATED,
                    payload=updates,
                    metadata={
                        **metadata,
                        "previous_state": dict(current)
                    }
                )
                
                await self.event_store.save_event(event)
                
                return True

import { useState, useEffect } from 'react';

export function useDebounce<T>(value: T, delay: number): T {
  const [debouncedValue, setDebouncedValue] = useState<T>(value);

  useEffect(() => {
    const timer = setTimeout(() => {
      setDebouncedValue(value);
    }, delay);

    return () => {
      clearTimeout(timer);
    };
  }, [value, delay]);

  return debouncedValue;
}

import { useState, useEffect } from 'react';
import { tokens } from '../styles/tokens';

export function useMediaQuery(query: string): boolean {
  const [matches, setMatches] = useState(false);

  useEffect(() => {
    const media = window.matchMedia(query);
    
    if (media.matches !== matches) {
      setMatches(media.matches);
    }

    const listener = (event: MediaQueryListEvent) => {
      setMatches(event.matches);
    };

    media.addEventListener('change', listener);
    
    return () => {
      media.removeEventListener('change', listener);
    };
  }, [matches, query]);

  return matches;
}

  const query = `(min-width: ${tokens.breakpoints[breakpoint]})`;
  return useMediaQuery(query);
}

### 1.2 Análise STRIDE por Componente

#### **API Gateway (Kong/Nginx)**
- **Spoofing:** Autenticação centralizada via JWT tokens
- **Tampering:** TLS 1.3 obrigatório, assinatura de payloads
- **Repudiation:** Logs de todas as requisições com correlation ID
- **Info Disclosure:** Rate limiting, WAF integrado
- **Denial of Service:** Circuit breaker, rate limiting por IP/usuário
- **Elevation of Privilege:** Validação de escopo em todas as rotas

- **Spoofing:** MFA obrigatório para admin, refresh tokens com rotação
- **Tampering:** JWT assinado com algoritmo forte (RS256)
- **Repudiation:** Logs de login, logout e tentativas falhas
- **Info Disclosure:** Senhas com bcrypt (cost 12), salt único
- **Denial of Service:** Lockout após 5 tentativas falhas
- **Elevation of Privilege:** RBAC granular, princípio do menor privilégio

#### **LLM Service**
- **Spoofing:** Validação de origem das requisições
- **Tampering:** Sanitização de prompts e outputs
- **Repudiation:** Logs de prompts e respostas (sem dados sensíveis)
- **Info Disclosure:** Context window limitado, PII redaction
- **Denial of Service:** Rate limiting por modelo/custo
- **Elevation of Privilege:** Validação de permissão por operação LLM

#### **User Service (PII)**
- **Spoofing:** Verificação de propriedade de dados
- **Tampering:** Dados criptografados em repouso (AES-256-GCM)
- **Repudiation:** Audit trail de todas as operações CRUD
- **Info Disclosure:** Máscara automática em logs e respostas
- **Denial of Service:** Quotas por usuário
- **Elevation of Privilege:** Row-level security no PostgreSQL

### 2.3 Script de Monitoramento de Dependências

```javascript
// scripts/security-monitor.js
#!/usr/bin/env node

const { execSync } = require('child_process');
const fs = require('fs');
const path = require('path');

class SecurityMonitor {
  constructor() {
    this.report = {
      timestamp: new Date().toISOString(),
      critical: [],
      high: [],
      medium: [],
      low: []
    };
  }

  async runDependencyScan() {
    console.log('🔍 Scanning dependencies for vulnerabilities...');
    
    try {
      // Check npm audit
      const auditResult = execSync('npm audit --json', { encoding: 'utf8' });
      const auditData = JSON.parse(auditResult);
      
      this.parseAuditData(auditData);
      
      // Check for outdated packages with security updates
      const outdatedResult = execSync('npm outdated --json', { encoding: 'utf8' });
      const outdatedData = JSON.parse(outdatedResult);
      
      this.checkSecurityUpdates(outdatedData);
      
      this.generateReport();
      
    } catch (error) {
      console.error('❌ Security scan failed:', error.message);
      process.exit(1);
    }
  }

  parseAuditData(auditData) {
    if (auditData.vulnerabilities) {
      Object.values(auditData.vulnerabilities).forEach(vuln => {
        const vulnerability = {
          name: vuln.name,
          severity: vuln.severity,
          via: vuln.via[0]?.title || 'Unknown',
          fixAvailable: vuln.fixAvailable,
          paths: vuln.nodes || []
        };

        switch (vuln.severity) {
          case 'critical':
            this.report.critical.push(vulnerability);
            break;
          case 'high':
            this.report.high.push(vulnerability);
            break;
          case 'medium':
            this.report.medium.push(vulnerability);
            break;
          case 'low':
            this.report.low.push(vulnerability);
            break;
        }
      });
    }
  }

  checkSecurityUpdates(outdatedData) {
    console.log('📦 Checking for security updates...');
    
    Object.entries(outdatedData).forEach(([pkg, info]) => {
      if (info.current !== info.latest) {
        console.log(`  ${pkg}: ${info.current} → ${info.latest}`);
        
        // Check if update is security-related
        if (this.isSecurityUpdate(info.current, info.latest)) {
          this.report.high.push({
            name: pkg,
            severity: 'high',
            via: 'Outdated with security fixes',
            fixAvailable: true,
            paths: ['package.json']
          });
        }
      }
    });
  }

  isSecurityUpdate(current, latest) {
    // Simple heuristic: major version change often includes security fixes
    const currentMajor = parseInt(current.split('.')[0]);
    const latestMajor = parseInt(latest.split('.')[0]);
    return latestMajor > currentMajor;
  }

  generateReport() {
    const reportPath = path.join(__dirname, '../reports/security-scan.json');
    
    // Ensure reports directory exists
    const reportsDir = path.dirname(reportPath);
    if (!fs.existsSync(reportsDir)) {
      fs.mkdirSync(reportsDir, { recursive: true });
    }
    
    fs.writeFileSync(reportPath, JSON.stringify(this.report, null, 2));
    
    // Print summary
    console.log('\n📊 Security Scan Summary:');
    console.log(`  Critical: ${this.report.critical.length}`);
    console.log(`  High: ${this.report.high.length}`);
    console.log(`  Medium: ${this.report.medium.length}`);
    console.log(`  Low: ${this.report.low.length}`);
    
    // Exit with error if critical vulnerabilities found
    if (this.report.critical.length > 0) {
      console.error('\n❌ Critical vulnerabilities found! Blocking deployment.');
      process.exit(1);
    }
    
    if (this.report.high.length > 0) {
      console.warn('\n⚠️  High severity vulnerabilities found. Review required.');
    }
  }
}

// Run the monitor
const monitor = new SecurityMonitor();
monitor.runDependencyScan();
```

```python
"""
Sistema de Detecção de Anomalias em Logs com Machine Learning
"""
import numpy as np
import pandas as pd
from datetime import datetime, timedelta
from typing import Dict, List, Any, Optional, Tuple
import joblib
import json
import warnings
warnings.filterwarnings('ignore')

from sklearn.ensemble import IsolationForest, RandomForestClassifier
from sklearn.preprocessing import StandardScaler
from sklearn.cluster import DBSCAN
from sklearn.svm import OneClassSVM
import xgboost as xgb
from prophet import Prophet
import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers

class LogAnomalyDetector:
    """Detector de anomalias em logs usando múltiplos algoritmos ML"""
    
    def __init__(self, config_path: str = "config/monitoring/ml_config.json"):
        self.config = self._load_config(config_path)
        self.models = {}
        self.scalers = {}
        self.feature_columns = []
        self.anomaly_history = []
        self.initialize_models()
        
    def _load_config(self, config_path: str) -> Dict:
        """Carrega configuração do modelo ML"""
        default_config = {
            "models": {
                "isolation_forest": {
                    "contamination": 0.1,
                    "n_estimators": 100,
                    "random_state": 42
                },
                "one_class_svm": {
                    "nu": 0.1,
                    "kernel": "rbf",
                    "gamma": "auto"
                },
                "dbscan": {
                    "eps": 0.5,
                    "min_samples": 5
                },
                "xgboost": {
                    "n_estimators": 100,
                    "max_depth": 6,
                    "learning_rate": 0.1
                },
                "lstm_autoencoder": {
                    "sequence_length": 10,
                    "encoding_dim": 32,
                    "epochs": 50,
                    "batch_size": 32
                }
            },
            "thresholds": {
                "anomaly_score": 0.7,
                "confidence_threshold": 0.8,
                "minimum_samples": 100
            },
            "retraining": {
                "interval_hours": 24,
                "minimum_new_samples": 1000
            }
        }
        
        try:
            with open(config_path, 'r') as f:
                user_config = json.load(f)
                # Merge configurations
                for key in default_config:
                    if key in user_config:
                        if isinstance(default_config[key], dict):
                            default_config[key].update(user_config[key])
                        else:
                            default_config[key] = user_config[key]
        except FileNotFoundError:
            print(f"Config file {config_path} not found, using defaults")
            
        return default_config
    
    def initialize_models(self):
        """Inicializa todos os modelos de ML"""
        # Isolation Forest
        iso_config = self.config["models"]["isolation_forest"]
        self.models["isolation_forest"] = IsolationForest(
            contamination=iso_config["contamination"],
            n_estimators=iso_config["n_estimators"],
            random_state=iso_config["random_state"]
        )
        
        # One-Class SVM
        svm_config = self.config["models"]["one_class_svm"]
        self.models["one_class_svm"] = OneClassSVM(
            nu=svm_config["nu"],
            kernel=svm_config["kernel"],
            gamma=svm_config["gamma"]
        )
        
        # DBSCAN
        dbscan_config = self.config["models"]["dbscan"]
        self.models["dbscan"] = DBSCAN(
            eps=dbscan_config["eps"],
            min_samples=dbscan_config["min_samples"]
        )
        
        # XGBoost para classificação
        xgb_config = self.config["models"]["xgboost"]
        self.models["xgboost"] = xgb.XGBClassifier(
            n_estimators=xgb_config["n_estimators"],
            max_depth=xgb_config["max_depth"],
            learning_rate=xgb_config["learning_rate"],
            use_label_encoder=False,
            eval_metric='logloss'
        )
        
        # Scaler para normalização
        self.scalers["standard"] = StandardScaler()
        
        # Inicializa LSTM Autoencoder
        self._initialize_lstm_autoencoder()
        
    def _initialize_lstm_autoencoder(self):
        """Inicializa autoencoder LSTM para detecção de anomalias temporais"""
        lstm_config = self.config["models"]["lstm_autoencoder"]
        
        # Encoder
        encoder_input = keras.Input(shape=(lstm_config["sequence_length"], 1))
        x = layers.LSTM(64, activation='relu', return_sequences=True)(encoder_input)
        x = layers.LSTM(32, activation='relu', return_sequences=False)(x)
        encoded = layers.Dense(lstm_config["encoding_dim"], activation='relu')(x)
        
        # Decoder
        x = layers.RepeatVector(lstm_config["sequence_length"])(encoded)
        x = layers.LSTM(32, activation='relu', return_sequences=True)(x)
        x = layers.LSTM(64, activation='relu', return_sequences=True)(x)
        decoded = layers.TimeDistributed(layers.Dense(1))(x)
        
        # Autoencoder completo
        self.models["lstm_autoencoder"] = keras.Model(encoder_input, decoded)
        self.models["lstm_autoencoder"].compile(
            optimizer='adam',
            loss='mse'
        )
        
        # Encoder separado para extração de features
        self.models["lstm_encoder"] = keras.Model(encoder_input, encoded)
        
    def extract_features(self, logs: List[Dict]) -> pd.DataFrame:
        """Extrai features dos logs para análise de ML"""
        features = []
        
        for log in logs:
            feature_vector = {
                # Features temporais
                "hour": log.get("timestamp", datetime.now()).hour,
                "day_of_week": log.get("timestamp", datetime.now()).weekday(),
                "minute_of_hour": log.get("timestamp", datetime.now()).minute,
                
                # Features de conteúdo
                "log_length": len(str(log.get("message", ""))),
                "has_error": 1 if "error" in str(log.get("level", "")).lower() else 0,
                "has_warning": 1 if "warn" in str(log.get("level", "")).lower() else 0,
                "has_exception": 1 if "exception" in str(log.get("message", "")).lower() else 0,
                
                # Features de frequência
                "same_source_count": 0,  # Será calculado posteriormente
                "same_message_pattern": 0,  # Será calculado posteriormente
                
                # Features numéricas extraídas
                "numeric_values_count": self._count_numeric_values(log.get("message", "")),
                "url_count": self._count_urls(log.get("message", "")),
                "ip_count": self._count_ips(log.get("message", "")),
                
                # Metadata
                "response_time": float(log.get("response_time", 0)) if log.get("response_time") else 0,
                "memory_usage": float(log.get("memory_usage", 0)) if log.get("memory_usage") else 0,
                "cpu_usage": float(log.get("cpu_usage", 0)) if log.get("cpu_usage") else 0,
            }
            features.append(feature_vector)
        
        df = pd.DataFrame(features)
        
        # Calcula features de frequência
        if len(df) > 0:
            df["same_source_count"] = self._calculate_frequency_features(logs, "source")
            df["same_message_pattern"] = self._calculate_message_pattern_frequency(logs)
        
        self.feature_columns = df.columns.tolist()
        return df
    
    def _count_numeric_values(self, text: str) -> int:
        """Conta valores numéricos no texto"""
        import re
        return len(re.findall(r'\b\d+\b', str(text)))
    
    def _count_urls(self, text: str) -> int:
        """Conta URLs no texto"""
        import re
        url_pattern = r'https?://(?:[-\w.]|(?:%[\da-fA-F]{2}))+'
        return len(re.findall(url_pattern, str(text)))
    
    def _count_ips(self, text: str) -> int:
        """Conta endereços IP no texto"""
        import re
        ip_pattern = r'\b(?:\d{1,3}\.){3}\d{1,3}\b'
        return len(re.findall(ip_pattern, str(text)))
    
    def _calculate_frequency_features(self, logs: List[Dict], field: str) -> List[int]:
        """Calcula frequência de valores em um campo específico"""
        from collections import Counter
        values = [log.get(field, "unknown") for log in logs]
        counter = Counter(values)
        return [counter[val] for val in values]
    
    def _calculate_message_pattern_frequency(self, logs: List[Dict]) -> List[int]:
        """Calcula frequência de padrões de mensagem"""
        from collections import Counter
        import re
        
        patterns = []
        for log in logs:
            message = str(log.get("message", ""))
            # Simplifica mensagem removendo valores variáveis
            pattern = re.sub(r'\b\d+\b', '#NUM', message)
            pattern = re.sub(r'0x[0-9a-fA-F]+', '#HEX', pattern)
            pattern = re.sub(r'https?://\S+', '#URL', pattern)
            patterns.append(pattern)
        
        counter = Counter(patterns)
        return [counter[pattern] for pattern in patterns]
    
    def train_models(self, training_logs: List[Dict], labels: Optional[List[int]] = None):
        """Treina os modelos de ML com logs históricos"""
        print("Iniciando treinamento dos modelos de detecção de anomalias...")
        
        # Extrai features
        X = self.extract_features(training_logs)
        
        # Normaliza os dados
        X_scaled = self.scalers["standard"].fit_transform(X)
        
        # Treina modelos não supervisionados
        print("Treinando Isolation Forest...")
        self.models["isolation_forest"].fit(X_scaled)
        
        print("Treinando One-Class SVM...")
        self.models["one_class_svm"].fit(X_scaled)
        
        # Treina DBSCAN (não supervisionado, não precisa de fit com labels)
        print("DBSCAN configurado para detecção de clusters anômalos")
        
        # Se temos labels, treina modelo supervisionado
        if labels and len(set(labels)) > 1:
            print("Treinando XGBoost (supervisionado)...")
            self.models["xgboost"].fit(X_scaled, labels)
        
        # Treina LSTM Autoencoder se temos dados sequenciais suficientes
        if len(training_logs) >= 100:
            print("Treinando LSTM Autoencoder...")
            sequences = self._create_sequences(X_scaled)
            if len(sequences) > 0:
                lstm_config = self.config["models"]["lstm_autoencoder"]
                self.models["lstm_autoencoder"].fit(
                    sequences, sequences,
                    epochs=lstm_config["epochs"],
                    batch_size=lstm_config["batch_size"],
                    verbose=0,
                    validation_split=0.1
                )
        
        print("Treinamento concluído!")
        return True
    
    def _create_sequences(self, data: np.ndarray, sequence_length: int = 10) -> np.ndarray:
        """Cria sequências para LSTM"""
        sequences = []
        for i in range(len(data) - sequence_length + 1):
            sequences.append(data[i:i + sequence_length])
        return np.array(sequences)
    
    def detect_anomalies(self, logs: List[Dict]) -> List[Dict]:
        """Detecta anomalias em novos logs usando ensemble de modelos"""
        if len(logs) < self.config["thresholds"]["minimum_samples"]:
            return []
        
        # Extrai features
        X = self.extract_features(logs)
        
        if len(X) == 0:
            return []
        
        # Normaliza
        X_scaled = self.scalers["standard"].transform(X)
        
        anomalies = []
        
        for i, (log, features) in enumerate(zip(logs, X_scaled)):
            anomaly_scores = {}
            features_2d = features.reshape(1, -1)
            
            # Isolation Forest
            iso_pred = self.models["isolation_forest"].predict(features_2d)
            anomaly_scores["isolation_forest"] = 1 if iso_pred[0] == -1 else 0
            
            # One-Class SVM
            svm_pred = self.models["one_class_svm"].predict(features_2d)
            anomaly_scores["one_class_svm"] = 1 if svm_pred[0] == -1 else 0
            
            # DBSCAN
            dbscan_pred = self.models["dbscan"].fit_predict(features_2d.reshape(1, -1))
            anomaly_scores["dbscan"] = 1 if dbscan_pred[0] == -1 else 0
            
            # XGBoost (se treinado)
            if hasattr(self.models["xgboost"], 'predict_proba'):
                try:
                    xgb_pred = self.models["xgboost"].predict_proba(features_2d)
                    anomaly_scores["xgboost"] = xgb_pred[0][1]  # Probabilidade de ser anomalia
                except:
                    anomaly_scores["xgboost"] = 0
            
            # LSTM Autoencoder reconstruction error
            lstm_score = 0

        const technicalControls = this.defineTechnicalControls(architectureSpec, riskAssessment);
        
        // 5. Processos de auditoria
        const auditProcesses = this.defineAuditProcesses();
        
        // 6. Rastreabilidade
        const traceabilityFramework = this.defineTraceabilityFramework();

        const complianceMap = {
            mappingId,
            timestamp,
            architectureVersion: architectureSpec.version,
            status: 'COMPLETED',
            personalDataAnalysis,
            legalBasesMapping,
            riskAssessment,
            technicalControls,
            auditProcesses,
            traceabilityFramework,
            validationChecks: this.performValidationChecks()
        };

        // Salva o mapeamento
        this.saveComplianceMapping(mappingId, complianceMap);
        
        // Atualiza ROPA
        this.updateROPARegister(complianceMap);
        
        this.logAuditEvent('MAPPING_COMPLETE', 'Compliance mapping completed successfully', {
            mappingId,
            dataCategories: personalDataAnalysis.categories.length,
            highRiskProcesses: riskAssessment.highRisk.length
        });

        return complianceMap;
    }

    /**
     * Analisa fluxos de dados pessoais na arquitetura
     */
    analyzePersonalDataFlows(architectureSpec) {
        const dataFlows = [];
        const dataCategories = new Set();
        const storageLocations = new Set();
        const thirdParties = new Set();

        // Análise de componentes da arquitetura
        architectureSpec.components?.forEach(component => {
            if (this.containsPersonalData(component)) {
                const flow = {
                    component: component.name,
                    dataTypes: this.extractDataTypes(component),
                    purpose: component.purpose || 'Not specified',
                    retentionPeriod: component.retention || 'Not defined',
                    storageLocation: component.storageLocation || 'Primary database',
                    thirdPartyInvolved: component.thirdParty || false
                };

                dataFlows.push(flow);
                
                // Coleta categorias
                flow.dataTypes.forEach(type => dataCategories.add(type));
                storageLocations.add(flow.storageLocation);
                
                if (flow.thirdPartyInvolved) {
                    thirdParties.add(component.thirdPartyName || 'Unknown');
                }
            }
        });

        // Análise de APIs e endpoints
        architectureSpec.apis?.forEach(api => {
            if (api.processesPersonalData) {
                const flow = {
                    component: `API: ${api.name}`,
                    dataTypes: api.dataFields || ['Personal data'],
                    purpose: api.purpose || 'API processing',
                    retentionPeriod: 'Session-based',
                    storageLocation: 'Memory/Logs',
                    thirdPartyInvolved: api.external || false
                };

                dataFlows.push(flow);
                
                api.dataFields?.forEach(field => dataCategories.add(field));
                if (api.external) {
                    thirdParties.add(api.provider || 'External API');
                }
            }
        });

        return {
            totalFlows: dataFlows.length,
            flows: dataFlows,
            categories: Array.from(dataCategories),
            storageLocations: Array.from(storageLocations),
            thirdParties: Array.from(thirdParties),
            analysisDate: new Date().toISOString()
        };
    }

    /**
     * Verifica se componente contém dados pessoais
     */
    containsPersonalData(component) {
        const personalDataIndicators = [
            'user', 'customer', 'client', 'person', 'individual',
            'name', 'email', 'cpf', 'cnpj', 'address', 'phone',
            'birth', 'location', 'ip', 'device', 'biometric'
        ];

        const componentStr = JSON.stringify(component).toLowerCase();
        return personalDataIndicators.some(indicator => 
            componentStr.includes(indicator)
        );
    }

    /**
     * Extrai tipos de dados do componente
     */
    extractDataTypes(component) {
        const dataTypes = [];
        
        if (component.dataModel) {
            Object.keys(component.dataModel).forEach(field => {
                if (this.isPersonalDataField(field)) {
                    dataTypes.push(field);
                }
            });
        }
        
        return dataTypes.length > 0 ? dataTypes : ['Personal data (unspecified)'];
    }

    isPersonalDataField(fieldName) {
        const personalDataFields = [
            'name', 'email', 'cpf', 'rg', 'passport', 'birth_date',
            'address', 'phone', 'ip_address', 'device_id', 'location',
            'salary', 'health', 'biometric', 'photo', 'voice'
        ];
        
        return personalDataFields.some(pdf => 
            fieldName.toLowerCase().includes(pdf)
        );
    }

    /**
     * Mapeia bases legais para cada processamento
     */
    mapLegalBases(architectureSpec, dataAnalysis) {
        const legalBases = [];
        
        dataAnalysis.flows.forEach(flow => {
            const legalBasis = this.determineLegalBasis(flow);
            
            legalBases.push({
                processingActivity: flow.component,
                dataTypes: flow.dataTypes,
                purpose: flow.purpose,
                legalBasis: legalBasis.type,
                justification: legalBasis.justification,
                consentRequired: legalBasis.type === 'consent',
                documentationRequired: this.getDocumentationRequirements(legalBasis.type),
                applicableArticles: this.getApplicableArticles(legalBasis.type)
            });
        });

        return {
            legalBases,
            summary: {
                totalActivities: legalBases.length,
                byBasis: this.groupByLegalBasis(legalBases),
                complianceStatus: this.checkLegalBasisCompliance(legalBases)
            }
        };
    }

    determineLegalBasis(flow) {
        // Lógica para determinar base legal baseada no propósito
        const purpose = flow.purpose.toLowerCase();
        
        if (purpose.includes('contract') || purpose.includes('agreement')) {
            return {
                type: 'contract',
                justification: 'Processing necessary for performance of a contract'
            };
        }
        
        if (purpose.includes('legal') || purpose.includes('obligation')) {
            return {
                type: 'legal_obligation',
                justification: 'Processing necessary for compliance with legal obligation'
            };
        }
        
        if (purpose.includes('consent') || purpose.includes('opt-in')) {
            return {
                type: 'consent',
                justification: 'Data subject has given consent'
            };
        }
        
        if (purpose.includes('legitimate') || purpose.includes('interest')) {
            return {
                type: 'legitimate_interest',
                justification: 'Processing necessary for legitimate interests pursued by controller'
            };
        }
        
        // Default para novos processamentos - requer análise específica
        return {
            type: 'requires_assessment',
            justification: 'Needs specific legal basis assessment'
        };
    }

    /**
     * Realiza avaliação de riscos
     */
    performRiskAssessment(dataAnalysis) {
        const risks = [];
        
        dataAnalysis.flows.forEach(flow => {
            const riskLevel = this.calculateRiskLevel(flow);
            
            risks.push({
                processingActivity: flow.component,
                riskLevel: riskLevel.level,
                score: riskLevel.score,
                factors: riskLevel.factors,
                recommendations: this.getRiskMitigations(riskLevel.level)
            });
        });

        return {
            risks,
            summary: {
                highRisk: risks.filter(r => r.riskLevel === 'HIGH').length,
                mediumRisk: risks.filter(r => r.riskLevel === 'MEDIUM').length,
                lowRisk: risks.filter(r => r.riskLevel === 'LOW').length,
                overallRisk: this.calculateOverallRisk(risks)
            }
        };
    }

    calculateRiskLevel(flow) {
        let score = 0;
        const factors = [];
        
        // Fator: Volume de dados
        if (flow.dataTypes.length > 5) {
            score += 2;
            factors.push('High volume of personal data types');
        }
        
        // Fator: Terceiros envolvidos
        if (flow.thirdPartyInvolved) {
            score += 3;
            factors.push('Third-party data processing');
        }
        
        // Fator: Retenção prolongada
        if (flow.retentionPeriod.includes('year') || flow.retentionPeriod === 'Indefinite') {
            score += 2;
            factors.push('Long retention period');
        }
        
        // Fator: Dados sensíveis
        const sensitiveData = ['health', 'biometric', 'racial', 'political', 'religious'];
        if (flow.dataTypes.some(dt => sensitiveData.some(sd => dt.toLowerCase().includes(sd)))) {
            score += 4;
            factors.push('Sensitive personal data involved');
        }
        
        // Determina nível baseado no score
        let level;
        if (score >= 7) level = 'HIGH';
        else if (score >= 4) level = 'MEDIUM';
        else level = 'LOW';
        
        return { level, score, factors };
    }
