# DIYAPP Evolution - V12 Core - Arquitetura Técnica

## ADR-001: Arquitetura V12 - Microserviços Otimizados com Gateway Unificado

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead V12 Core

### CONTEXTO:
O DIYAPP evoluiu para uma plataforma complexa com múltiplos módulos (IA Agents, WhatsApp Integration, Dashboard, Analytics, Payment). A arquitetura monolítica atual apresenta:
1. Acoplamento excessivo entre módulos
2. Dificuldade de deploy independente
3. Escalabilidade limitada por componentes específicos
4. Tempos de build/test crescentes (>15min)
5. Single point of failure em componentes críticos

**Forças em jogo:**
- Necessidade de 99.9% uptime
- Time-to-market rápido para novos features
- Equipe distribuída trabalhando em paralelo
- Orçamento limitado para infraestrutura
- Requisito de <200ms P95 para todas APIs públicas

### DECISÃO:
Adotar arquitetura de microserviços otimizados com Gateway API unificado e comunicação assíncrona via Redis Streams. Cada módulo crítico será um serviço independente, mas compartilharemos banco de dados PostgreSQL por domínio para reduzir complexidade de transações distribuídas.

### OPÇÕES CONSIDERADAS:

**Opção A: Monolito Otimizado com Modularização Interna**
- **Prós:** 
  - Transações ACID garantidas
  - Debug mais simples
  - Menor overhead de rede
  - Deploy único
- **Contras:**
  - Escalabilidade vertical limitada
  - Acoplamento persistente
  - Build/test lentos
  - Risco de single point of failure

**Opção B: Microserviços Puros (12-Factor)**
- **Prós:**
  - Escalabilidade independente por serviço
  - Deploy contínuo independente
  - Isolamento de falhas
  - Poliglotismo tecnológico
- **Contras:**
  - Complexidade operacional alta
  - Transações distribuídas complexas
  - Latência de rede agregada
  - Custo infra mais alto

**Opção C: Microserviços Otimizados (Escolhida)**
- **Prós:**
  - Balanceamento entre autonomia e simplicidade
  - Gateway unificado simplifica client-side
  - Redis Streams para comunicação assíncrona robusta
  - PostgreSQL compartilhado por domínio reduz complexidade
  - Pode escalar componentes críticos independentemente
- **Contras:**
  - Gerenciamento de múltiplos serviços
  - Necessidade de monitoramento distribuído
  - Debug distribuído requer ferramentas específicas

**Justificativa da escolha:** A Opção C oferece melhor trade-off para nosso contexto. Mantemos autonomia de desenvolvimento por squad (Modo Hive) enquanto controlamos complexidade operacional com padrões claros e ferramentas unificadas.

### CONSEQUÊNCIAS:

**Positivas:**
1. Cada squad pode desenvolver/deployar independentemente
2. Componentes críticos (WhatsApp Gateway, LLM Orchestrator) escalam horizontalmente
3. Isolamento de falhas - problema em um serviço não derruba toda plataforma
4. Time-to-market reduzido para novos features
5. Stack tecnológica pode evoluir por serviço

**Negativas:**
1. Overhead de rede entre serviços (mitigado com Redis local)
2. Monitoramento distribuído necessário
3. Transações cross-service mais complexas
4. Maior consumo de recursos em staging/dev

**Riscos:**
1. **Latência agregada:** Mitigar com cache Redis e conexões persistentes
2. **Consistência eventual:** Implementar SAGA pattern para transações críticas
3. **Debug distribuído:** Implementar correlation IDs e centralized logging
4. **Custo infra:** Usar auto-scaling e otimizar resource allocation

### REVISÃO: 2024-04-15 (3 meses para avaliação de performance)

```javascript
// src/infrastructure/cache/redis-manager.js
const Redis = require('ioredis');
const { EventEmitter } = require('events');

class RedisCacheManager extends EventEmitter {
  constructor(config) {
    super();
    this.config = {
      clusterMode: config.clusterMode || false,
      nodes: config.nodes || [{ host: 'localhost', port: 6379 }],
      ttl: config.ttl || 3600, // 1 hour default
      maxRetries: config.maxRetries || 3,
      enableOfflineQueue: config.enableOfflineQueue || true,
      keyPrefix: config.keyPrefix || 'diyapp:',
    };

    this.redis = this.config.clusterMode
      ? new Redis.Cluster(this.config.nodes, {
          scaleReads: 'slave',
          redisOptions: {
            keyPrefix: this.config.keyPrefix,
            enableOfflineQueue: this.config.enableOfflineQueue,
            retryStrategy: (times) => Math.min(times * 50, 2000),
          },
        })
      : new Redis({
          ...this.config.nodes[0],
          keyPrefix: this.config.keyPrefix,
          enableOfflineQueue: this.config.enableOfflineQueue,
          retryStrategy: (times) => Math.min(times * 50, 2000),
        });

    this.streams = new Map();
    this.init();
  }

  async init() {
    // Health check
    setInterval(async () => {
      try {
        await this.redis.ping();
        this.emit('healthy');
      } catch (error) {
        this.emit('unhealthy', error);
      }
    }, 30000);

    // Monitor cache hit rate
    setInterval(() => this.monitorPerformance(), 60000);
  }

  // Cache operations with circuit breaker
  async get(key, options = {}) {
    const circuit = options.circuit || this.createCircuitBreaker();
    
    return circuit.execute(async () => {
      const start = Date.now();
      try {
        const value = await this.redis.get(key);
        const duration = Date.now() - start;
        
        this.emit('cache_operation', {
          operation: 'get',
          key,
          hit: value !== null,
          duration,
          timestamp: new Date().toISOString(),
        });

        return value ? JSON.parse(value) : null;
      } catch (error) {
        this.emit('cache_error', {
          operation: 'get',
          key,
          error: error.message,
          timestamp: new Date().toISOString(),
        });
        throw error;
      }
    });
  }

  async set(key, value, ttl = this.config.ttl) {
    const start = Date.now();
    try {
      await this.redis.setex(
        key,
        ttl,
        JSON.stringify(value)
      );
      
      const duration = Date.now() - start;
      this.emit('cache_operation', {
        operation: 'set',
        key,
        ttl,
        duration,
        size: JSON.stringify(value).length,
        timestamp: new Date().toISOString(),
      });
    } catch (error) {
      this.emit('cache_error', {
        operation: 'set',
        key,
        error: error.message,
        timestamp: new Date().toISOString(),
      });
      throw error;
    }
  }

  // Redis Streams for async communication
  async publishToStream(stream, message, options = {}) {
    const messageId = options.messageId || `${Date.now()}-${Math.random().toString(36).substr(2, 9)}`;
    const maxLen = options.maxLen || 1000; // Keep last 1000 messages
    
    try {
      const id = await this.redis.xadd(
        stream,
        'MAXLEN', '~', maxLen,
        '*',
        'payload', JSON.stringify(message),
        'timestamp', new Date().toISOString(),
        'source', options.source || 'unknown'
      );

      this.emit('stream_published', {
        stream,
        messageId: id,
        size: JSON.stringify(message).length,
        timestamp: new Date().toISOString(),
      });

      return id;
    } catch (error) {
      this.emit('stream_error', {
        stream,
        error: error.message,
        timestamp: new Date().toISOString(),
      });
      throw error;
    }
  }

  async createConsumerGroup(stream, group, options = {}) {
    try {
      await this.redis.xgroup(
        'CREATE',
        stream,
        group,
        options.startId || '0',
        'MKSTREAM'
      );
    } catch (error) {
      if (!error.message.includes('BUSYGROUP')) {
        throw error;
      }
    }
  }

  async consumeFromStream(stream, group, consumer, options = {}) {
    const count = options.count || 10;
    const block = options.block || 5000; // 5 seconds
    
    try {
      const messages = await this.redis.xreadgroup(
        'GROUP', group, consumer,
        'COUNT', count,
        'BLOCK', block,
        'STREAMS', stream, options.lastId || '>'
      );

      if (messages && messages.length > 0) {
        const processed = [];
        for (const [_, streamMessages] of messages) {
          for (const [id, fields] of streamMessages) {
            const message = {
              id,
              payload: JSON.parse(fields[1]), // payload field
              timestamp: fields[3], // timestamp field
              source: fields[5], // source field
            };
            processed.push(message);
            
            // Auto-acknowledge if configured
            if (options.autoAck) {
              await this.redis.xack(stream, group, id);
            }
          }
        }
        
        this.emit('stream_consumed', {
          stream,
          group,
          consumer,
          count: processed.length,
          timestamp: new Date().toISOString(),
        });
        
        return processed;
      }
      
      return [];
    } catch (error) {
      this.emit('stream_error', {
        stream,
        group,
        consumer,
        error: error.message,
        timestamp: new Date().toISOString(),
      });
      throw error;
    }
  }

  // Performance monitoring
  async monitorPerformance() {
    try {
      const info = await this.redis.info();
      const hitRate = await this.calculateHitRate();
      
      this.emit('performance_metrics', {
        hitRate,
        connectedClients: this.extractInfoValue(info, 'connected_clients'),
        usedMemory: this.extractInfoValue(info, 'used_memory_human'),
        opsPerSecond: this.extractInfoValue(info, 'instantaneous_ops_per_sec'),
        timestamp: new Date().toISOString(),
      });
    } catch (error) {
      this.emit('monitoring_error', error);
    }
  }

  async calculateHitRate() {
    const stats = await this.redis.info('stats');
    const keyspaceHits = this.extractInfoValue(stats, 'keyspace_hits');
    const keyspaceMisses = this.extractInfoValue(stats, 'keyspace_misses');
    
    const total = keyspaceHits + keyspaceMisses;
    return total > 0 ? (keyspaceHits / total) * 100 : 0;
  }

  extractInfoValue(info, key) {
    const match = info.match(new RegExp(`${key}:([^\\r\\n]+)`));
    return match ? parseFloat(match[1]) : 0;
  }

  createCircuitBreaker() {
    let failures = 0;
    let lastFailureTime = 0;
    const resetTimeout = 30000; // 30 seconds
    
    return {
      execute: async (fn) => {
        const now = Date.now();
        
        // Circuit is open
        if (failures >= 3 && (now - lastFailureTime) < resetTimeout) {
          this.emit('circuit_open');
          throw new Error('Circuit breaker open - cache unavailable');
        }
        
        try {
          const result = await fn();
          
          // Reset on success
          if (failures > 0 && (now - lastFailureTime) >= resetTimeout) {
            failures = 0;
            this.emit('circuit_closed');
          }
          
          return result;
        } catch (error) {
          failures++;
          lastFailureTime = now;
          this.emit('circuit_failure', { failures, error: error.message });
          throw error;
        }
      },
    };
  }

  // Cache patterns
  async cacheAside(key, fetchFn, ttl = this.config.ttl) {
    // Try cache first
    const cached = await this.get(key);
    if (cached !== null) {
      return cached;
    }
    
    // Cache miss - fetch from source
    const data = await fetchFn();
    
    // Update cache async
    this.set(key, data, ttl).catch(error => {
      this.emit('cache_aside_error', { key, error: error.message });
    });
    
    return data;
  }

  async writeThrough(key, value, writeFn, ttl = this.config.ttl) {
    // Write to source first
    const result = await writeFn(value);
    
    // Update cache
    await this.set(key, result, ttl);
    
    return result;
  }

  // Invalidation strategies
  async invalidatePattern(pattern) {
    const keys = await this.redis.keys(`${this.config.keyPrefix}${pattern}`);
    if (keys.length > 0) {
      await this.redis.del(...keys);
      this.emit('cache_invalidated', { pattern, keys: keys.length });
    }
  }

  async invalidateWithTags(tags) {
    for (const tag of tags) {
      await this.invalidatePattern(`tag:${tag}:*`);
    }
  }

  // Graceful shutdown
  async shutdown() {
    try {
      await this.redis.quit();
      this.emit('shutdown');
    } catch (error) {
      this.emit('shutdown_error', error);
    }
  }
}

module.exports = RedisCacheManager;
```

import asyncio
from typing import Optional, Tuple
import redis.asyncio as redis
from datetime import datetime, timedelta
import logging

logger = logging.getLogger(__name__)

class RateLimiter:
    """Rate limiting distribuído usando Redis"""
    
    def __init__(self, storage_url: str, enabled: bool = True):
        self.enabled = enabled
        self.redis_client = None
        
        if enabled:
            try:
                self.redis_client = redis.from_url(
                    storage_url,
                    encoding="utf-8",
                    decode_responses=True
                )
                logger.info("Rate limiter initialized with Redis")
            except Exception as e:
                logger.error(f"Failed to initialize Redis: {str(e)}")
                self.enabled = False
    
    async def is_allowed(
        self, 
        identifier: str, 
        endpoint: str,
        limit: Optional[int] = None,
        window: Optional[int] = None
    ) -> bool:
        """
        Verifica se uma requisição é permitida baseada no rate limit
        
        Args:
            identifier: Identificador único (ex: IP:endpoint)
            endpoint: Endpoint sendo acessado
            limit: Número máximo de requisições (default: 100)
            window: Janela de tempo em segundos (default: 60)
        
        Returns:
            bool: True se permitido, False se excedeu o limite
        """
        if not self.enabled or not self.redis_client:
            return True
        
        try:
            # Configurações padrão por endpoint
            if "auth" in endpoint:
                limit = limit or 10  # Limite mais baixo para auth
                window = window or 60
            elif "api" in endpoint:
                limit = limit or 100
                window = window or 60
            else:
                limit = limit or 50
                window = window or 60
            
            key = f"rate_limit:{identifier}:{window}"
            
            # Usar pipeline para atomicidade
            async with self.redis_client.pipeline() as pipe:
                pipe.incr(key)
                pipe.expire(key, window)
                results = await pipe.execute()
            
            current_count = results[0]
            
            if current_count > limit:
                logger.warning(
                    f"Rate limit exceeded for {identifier}",
                    extra={
                        "identifier": identifier,
                        "current_count": current_count,
                        "limit": limit,
                        "window": window
                    }
                )
                return False
            
            return True
            
        except Exception as e:
            logger.error(f"Rate limiter error: {str(e)}")
            # Em caso de falha no Redis, permitir requisições
            return True
    
    async def get_usage(self, identifier: str) -> dict:
        """Retorna uso atual do rate limit"""
        if not self.enabled or not self.redis_client:
            return {"enabled": False}
        
        try:
            # Buscar todas as chaves para o identificador
            pattern = f"rate_limit:{identifier}:*"
            keys = await self.redis_client.keys(pattern)
            
            usage = {}
            for key in keys:
                window = key.split(":")[-1]
                count = await self.redis_client.get(key)
                ttl = await self.redis_client.ttl(key)
                
                usage[window] = {
                    "count": int(count) if count else 0,
                    "ttl": ttl
                }
            
            return {
                "enabled": True,
                "identifier": identifier,
                "usage": usage
            }
            
        except Exception as e:
            logger.error(f"Failed to get rate limit usage: {str(e)}")
            return {"enabled": False, "error": str(e)}

```typescript
import React from 'react';
import clsx from 'clsx';
import { tokens } from '../../tokens';

interface ButtonProps extends React.ButtonHTMLAttributes<HTMLButtonElement> {
  variant?: 'primary' | 'secondary' | 'outline' | 'ghost' | 'danger';
  size?: 'sm' | 'md' | 'lg';
  isLoading?: boolean;
  leftIcon?: React.ReactNode;
  rightIcon?: React.ReactNode;
  fullWidth?: boolean;
}

export const Button: React.FC<ButtonProps> = ({
  children,
  variant = 'primary',
  size = 'md',
  isLoading = false,
  leftIcon,
  rightIcon,
  fullWidth = false,
  className,
  disabled,
  ...props
}) => {
  const baseStyles = 'inline-flex items-center justify-center font-medium rounded-lg transition-colors focus:outline-none focus:ring-2 focus:ring-offset-2 disabled:opacity-50 disabled:cursor-not-allowed';
  
  const variants = {
    primary: `bg-${tokens.colors.primary[600]} text-white hover:bg-${tokens.colors.primary[700]} focus:ring-${tokens.colors.primary[500]}`,
    secondary: `bg-${tokens.colors.neutral[200]} text-${tokens.colors.text.primary} hover:bg-${tokens.colors.neutral[300]} focus:ring-${tokens.colors.neutral[400]}`,
    outline: `border border-${tokens.colors.border.default} bg-transparent text-${tokens.colors.text.primary} hover:bg-${tokens.colors.background.secondary} focus:ring-${tokens.colors.primary[500]}`,
    ghost: `bg-transparent text-${tokens.colors.text.primary} hover:bg-${tokens.colors.background.secondary} focus:ring-${tokens.colors.primary[500]}`,
    danger: `bg-${tokens.colors.error[600]} text-white hover:bg-${tokens.colors.error[700]} focus:ring-${tokens.colors.error[500]}`,
  };
  
  const sizes = {
    sm: `px-${tokens.spacing[2]} py-${tokens.spacing[1]} text-${tokens.typography.fontSize.sm}`,
    md: `px-${tokens.spacing[4]} py-${tokens.spacing[2]} text-${tokens.typography.fontSize.base}`,
    lg: `px-${tokens.spacing[6]} py-${tokens.spacing[3]} text-${tokens.typography.fontSize.lg}`,
  };
  
  const widthClass = fullWidth ? 'w-full' : '';
  
  return (
    <button
      className={clsx(
        baseStyles,
        variants[variant],
        sizes[size],
        widthClass,
        className
      )}
      disabled={disabled || isLoading}
      aria-busy={isLoading}
      {...props}
    >
      {isLoading && (
        <svg
          className="animate-spin -ml-1 mr-2 h-4 w-4"
          xmlns="http://www.w3.org/2000/svg"
          fill="none"
          viewBox="0 0 24 24"
        >
          <circle
            className="opacity-25"
            cx="12"
            cy="12"
            r="10"
            stroke="currentColor"
            strokeWidth="4"
          />
          <path
            className="opacity-75"
            fill="currentColor"
            d="M4 12a8 8 0 018-8V0C5.373 0 0 5.373 0 12h4zm2 5.291A7.962 7.962 0 014 12H0c0 3.042 1.135 5.824 3 7.938l3-2.647z"
          />
        </svg>
      )}
      {!isLoading && leftIcon && <span className="mr-2">{leftIcon}</span>}
      {children}
      {!isLoading && rightIcon && <span className="ml-2">{rightIcon}</span>}
    </button>
  );
};
```

```sql
PARTITION BY toYYYYMM(metric_date)

    mau UInt32,
    new_users UInt32,

    total_sessions UInt32,

    retention_7d Float32,
    retention_30d Float32,

    metric_name String,
    metric_value Float64,
    expected_value Float64,
    deviation_pct Float64,
    severity Enum8('low' = 1, 'medium' = 2, 'high' = 3, 'critical' = 4),
    detected_at DateTime,
    time_window String,
    description String,
    status Enum8('new' = 1, 'investigating' = 2, 'resolved' = 3, 'false_positive' = 4),
    assigned_to Nullable(String),
    resolved_at Nullable(DateTime),
    resolution_notes Nullable(String)
)

-- View materializada para métricas em tempo real
CREATE MATERIALIZED VIEW IF NOT EXISTS diyapp_analytics.realtime_metrics_mv
TO diyapp_analytics.product_metrics_daily AS
SELECT
    event_date as metric_date,
    uniq(user_id) as dau,
    uniqIf(user_id, event_date >= today() - 30) as mau,
    uniqIf(user_id, created_at >= today() - 1 AND created_at < today()) as new_users,
    dau - new_users as returning_users,
    uniq(session_id) as total_sessions,
    avgIf(
        dateDiff('second', min(created_at), max(created_at)),
        event_type = 'session_end'
    ) as avg_session_duration,
    countIf(event_type = 'page_view') as total_page_views,
    mapFromArrays(
        groupArray(feature_name),
        groupArray(feature_count)
    ) as feature_adoption,
    mapFromArrays(
        ['visitors', 'engaged', 'converted'],
        [
            uniq(user_id),
            uniqIf(user_id, arrayCount(x -> x IN ['feature_click', 'llm_call'], groupArray(event_type)) > 0),
            uniqIf(user_id, event_type = 'conversion')
        ]
    ) as conversion_funnel,
    countIf(arrayCount(x -> x = 'page_view', groupArray(event_type)) = 1) / total_sessions as bounce_rate
FROM diyapp_analytics.raw_events
WHERE event_date = today()
GROUP BY event_date;
```

```python
# src/database/compliance_db.py
import os
import json
from datetime import datetime, timedelta
from typing import Dict, List, Optional, Any
from sqlalchemy import create_engine, Column, String, DateTime, JSON, Boolean, Integer, Text, ForeignKey
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy.orm import sessionmaker, relationship
from sqlalchemy.sql import func
import hashlib
import hmac

Base = declarative_base()

class DataSubject(Base):
    """Titular de dados conforme LGPD/GDPR"""
    __tablename__ = 'data_subjects'
    
    id = Column(String(36), primary_key=True)
    external_id = Column(String(255), unique=True, nullable=False)
    email = Column(String(255), index=True)
    created_at = Column(DateTime, default=func.now())
    updated_at = Column(DateTime, default=func.now(), onupdate=func.now())
    
    # Metadados de conformidade
    data_provenance = Column(JSON)  # Origem dos dados
    legal_basis = Column(String(50), nullable=False)  # consentimento, contrato, legítimo interesse
    data_categories = Column(JSON)  # Categorias de dados pessoais
    
    # Relacionamentos
    consents = relationship("UserConsent", back_populates="subject", cascade="all, delete-orphan")
    rights_requests = relationship("RightsRequest", back_populates="subject")
    audit_logs = relationship("AuditLog", back_populates="subject")

class UserConsent(Base):
    """Registro granular de consentimentos"""
    __tablename__ = 'user_consents'
    
    id = Column(String(36), primary_key=True)
    subject_id = Column(String(36), ForeignKey('data_subjects.id'), nullable=False)
    purpose = Column(String(100), nullable=False)  # Finalidade específica
    version = Column(String(20), nullable=False)  # Versão da política
    granted = Column(Boolean, default=False)
    granted_at = Column(DateTime)
    revoked_at = Column(DateTime)
    
    # Detalhes do consentimento
    method = Column(String(50))  # web, mobile, api, etc.
    ip_address = Column(String(45))
    user_agent = Column(Text)
    
    # Metadados
    created_at = Column(DateTime, default=func.now())
    updated_at = Column(DateTime, default=func.now(), onupdate=func.now())
    
    # Relacionamentos
    subject = relationship("DataSubject", back_populates="consents")

class DataRetentionRule(Base):
    """Regras de retenção de dados por finalidade"""
    __tablename__ = 'data_retention_rules'
    
    id = Column(String(36), primary_key=True)
    purpose = Column(String(100), unique=True, nullable=False)
    retention_period_days = Column(Integer, nullable=False)
    legal_basis = Column(String(50), nullable=False)
    description = Column(Text)
    
    # Configuração de exclusão
    auto_delete = Column(Boolean, default=True)
    archive_before_delete = Column(Boolean, default=False)
    
    created_at = Column(DateTime, default=func.now())
    updated_at = Column(DateTime, default=func.now(), onupdate=func.now())

class AuditLog(Base):
    """Logs de auditoria imutáveis (WORM - Write Once Read Many)"""
    __tablename__ = 'audit_logs'
    
    id = Column(String(36), primary_key=True)
    event_id = Column(String(64), unique=True, nullable=False)  # Hash imutável
    subject_id = Column(String(36), ForeignKey('data_subjects.id'))
    event_type = Column(String(50), nullable=False)  # access, modification, deletion, consent_change
    resource_type = Column(String(50), nullable=False)
    resource_id = Column(String(255))
    
    # Dados do evento
    before_state = Column(JSON)
    after_state = Column(JSON)
    changes = Column(JSON)
    
    # Contexto
    user_id = Column(String(255))
    ip_address = Column(String(45))
    user_agent = Column(Text)
    endpoint = Column(String(500))
    
    # Metadados imutáveis
    timestamp = Column(DateTime, default=func.now())
    log_hash = Column(String(64), nullable=False)  # Hash para verificação de integridade
    
    # Relacionamentos
    subject = relationship("DataSubject", back_populates="audit_logs")

class RightsRequest(Base):
    """Solicitações de direitos do titular"""
    __tablename__ = 'rights_requests'
    
    id = Column(String(36), primary_key=True)
    subject_id = Column(String(36), ForeignKey('data_subjects.id'), nullable=False)
    request_type = Column(String(20), nullable=False)  # access, rectification, erasure, portability
    status = Column(String(20), default='pending')  # pending, processing, completed, rejected
    requested_at = Column(DateTime, default=func.now())
    
    # SLA Tracking
    deadline = Column(DateTime)
    completed_at = Column(DateTime)
    
    # Dados da solicitação
    request_data = Column(JSON)
    response_data = Column(JSON)
    
    # Metadados
    created_at = Column(DateTime, default=func.now())
    updated_at = Column(DateTime, default=func.now(), onupdate=func.now())
    
    # Relacionamentos
    subject = relationship("DataSubject", back_populates="rights_requests")

class ComplianceDB:
    """Gerenciador do banco de dados de conformidade"""
    
    def __init__(self, database_url: str = None):
        self.database_url = database_url or os.getenv('COMPLIANCE_DB_URL', 'postgresql://user:pass@localhost:5432/compliance_db')
        self.engine = create_engine(self.database_url)
        self.SessionLocal = sessionmaker(autocommit=False, autoflush=False, bind=self.engine)
        
    def init_db(self):
        """Inicializa o banco de dados"""
        Base.metadata.create_all(bind=self.engine)
        
    def get_session(self):
        """Retorna uma sessão do banco"""
        return self.SessionLocal()
```
