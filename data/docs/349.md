# DIYAPP Evolution - V12 Core - Arquitetura de Microsserviços

## 1. ADR-001: Arquitetura de Microsserviços V12

**ADR-001: Adoção de Arquitetura de Microsserviços para V12**  
**Data:** 2024-01-15  
**Status:** Aceita  
**Autores:** Tech Lead + Especialista Infra + Especialista Backend  

### CONTEXTO:
A V11 atual do DIYAPP utiliza uma arquitetura monolítica que está enfrentando:
1. Dificuldades de escalabilidade horizontal
2. Acoplamento excessivo entre funcionalidades
3. Tempos de deploy longos (>15 minutos)
4. Dificuldade em adotar novas tecnologias por partes
5. Single point of failure em componentes críticos

### DECISÃO:
Adotar arquitetura de microsserviços com os seguintes princípios:
- 6 serviços independentes com responsabilidades bem definidas
- Comunicação via API REST (síncrona) e eventos (assíncrona)
- Cada serviço com seu próprio banco de dados
- Deploy independente por serviço
- API Gateway como ponto único de entrada

### OPÇÕES CONSIDERADAS:
- **Opção A: Monólito modularizado** — Prós: Menor complexidade operacional, debugging mais fácil. Contras: Não resolve problemas de escalabilidade, acoplamento ainda existe.
- **Opção B: Microsserviços completos** — Prós: Escalabilidade independente, deploy rápido, resiliência. Contras: Complexidade operacional maior, necessidade de observabilidade robusta.
- **Opção C: Arquitetura híbrida** — Prós: Balanceamento entre complexidade e benefícios. Contras: Pode gerar inconsistências arquiteturais.

**Opção escolhida: B** — Justificativa: A necessidade de escalabilidade independente e deploy rápido justifica a complexidade adicional. A squad já possui experiência com containers e CI/CD.

### CONSEQUÊNCIAS:
**Positivas:**
- Escalabilidade granular por funcionalidade
- Deploy independente (2-3 minutos por serviço)
- Resiliência melhorada (falha em um serviço não derruba o sistema)
- Adoção de tecnologias específicas por serviço

**Negativas:**
- Complexidade de deploy e monitoramento aumentada
- Necessidade de gerenciamento de comunicação entre serviços
- Overhead de rede para chamadas entre serviços
- Consistência eventual em alguns cenários

**Riscos:**
1. **Latência de rede:** Mitigar com circuit breakers e timeouts adequados
2. **Debugging distribuído:** Implementar distributed tracing com correlation IDs
3. **Gestão de dados distribuídos:** Usar SAGA pattern para transações distribuídas

**REVISÃO:** 2024-04-15 (3 meses após implementação)

## 3. Padrões de API REST

### 3.1 Versionamento
- URL versioning: `/api/v1/resource`
- Header `Accept: application/vnd.diyapp.v1+json`

### 3.2 Formato de Resposta
```json
{
  "success": true,
  "data": { /* dados da resposta */ },
  "meta": {
    "timestamp": "2024-01-15T10:30:00Z",
    "version": "v1",
    "requestId": "req_123456"
  },
  "error": null
}
```

### 3.3 Status Codes
- 200: Sucesso
- 201: Criado
- 400: Bad Request (validação)
- 401: Não autenticado
- 403: Proibido (sem permissão)
- 404: Não encontrado
- 429: Too Many Requests
- 500: Erro interno

### 3.4 Paginação
```
GET /api/v1/posts?page=1&limit=20&sort=-createdAt
```
Resposta inclui:
```json
{
  "data": [],
  "meta": {
    "pagination": {
      "page": 1,
      "limit": 20,
      "total": 150,
      "pages": 8
    }
  }
}
```

## 4. Padrões de Commits
- **feat:** Nova funcionalidade
- **fix:** Correção de bug
- **docs:** Documentação
- **style:** Formatação (sem alteração de código)
- **refactor:** Refatoração
- **test:** Testes
- **chore:** Tarefas de build/CI

Exemplo: `feat(auth): implementa login com Google OAuth`

## 5. Estratégia de Branching
- `main`: Produção (protected)
- `develop`: Desenvolvimento (protected)
- `feature/*`: Novas funcionalidades
- `hotfix/*`: Correções urgentes
- `release/*`: Preparação de release

## 6. Testes
- **Cobertura mínima:** 80%
- **Testes unitários:** Jest
- **Testes de integração:** Supertest + Jest
- **Testes E2E:** Cypress (frontend)

## 7. Logging
- Níveis: error, warn, info, debug
- Formato JSON para ingestão em ELK
- Correlation ID em todas as requisições

## 8. Segurança
- Input validation com Joi/Zod
- Rate limiting por IP/serviço
- Headers de segurança (CSP, HSTS)
- Secrets em variáveis de ambiente
- Scanning de dependências (Snyk/Dependabot)
```

**services/template-service/src/index.ts:**
```typescript
import express from 'express';
import cors from 'cors';
import helmet from 'helmet';
import rateLimit from 'express-rate-limit';
import { config } from './config';
import { logger } from './utils/logger';
import { errorHandler } from './middleware/error-handler';
import { requestLogger } from './middleware/request-logger';
import { healthRouter } from './routes/health';
import { apiRouter } from './routes/api';

const app = express();

// Security middleware
app.use(helmet());
app.use(cors({
  origin: config.cors.origin,
  credentials: true
}));

// Rate limiting
const limiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 minutes
  max: 100 // limit each IP to 100 requests per windowMs
});
app.use('/api', limiter);

// Body parsing
app.use(express.json({ limit: '10mb' }));
app.use(express.urlencoded({ extended: true }));

// Request logging
app.use(requestLogger);

// Routes
app.use('/health', healthRouter);
app.use('/api/v1', apiRouter);

// 404 handler
app.use('*', (req, res) => {
  res.status(404).json({
    success: false,
    error: {
      code: 'NOT_FOUND',
      message: `Route ${req.originalUrl} not found`
    }
  });
});

// Error handler (must be last)
app.use(errorHandler);

// Start server
const server = app.listen(config.port, () => {
  logger.info(`🚀 ${config.serviceName} service running on port ${config.port}`);
  logger.info(`📝 Environment: ${config.nodeEnv}`);
  logger.info(`🔗 Health check: http://localhost:${config.port}/health`);
});

// Graceful shutdown
process.on('SIGTERM', () => {
  logger.info('SIGTERM received, shutting down gracefully');
  server.close(() => {
    logger.info('Process terminated');
    process.exit(0);
  });
});

process.on('SIGINT', () => {
  logger.info('SIGINT received, shutting down gracefully');
  server.close(() => {
    logger.info('Process terminated');
    process.exit(0);
  });
});

export { app };
```

import winston from 'winston';
import { config } from '../config';

const { combine, timestamp, json, colorize, printf } = winston.format;

const logFormat = printf(({ level, message, timestamp, ...metadata }) => {
  let msg = `${timestamp} [${level}] : ${message}`;
  
  if (Object.keys(metadata).length > 0) {
    msg += ` ${JSON.stringify(metadata)}`;
  }
  
  return msg;
});

export const logger = winston.createLogger({
  level: config.logLevel,
  format: combine(
    timestamp({ format: 'YYYY-MM-DD HH:mm:ss' }),
    config.nodeEnv === 'production' ? json() : combine(colorize(), logFormat)
  ),
  transports: [
    new winston.transports.Console(),
    new winston.transports.File({ 
      filename: 'logs/error.log', 
      level: 'error' 
    }),
    new winston.transports.File({ 
      filename: 'logs/combined.log' 
    }),
  ],
  exceptionHandlers: [
    new winston.transports.File({ 
      filename: 'logs/exceptions.log' 
    }),
  ],
  rejectionHandlers: [
    new winston.transports.File({ 
      filename: 'logs/rejections.log' 
    }),
  ],
});

### `src/services/auth/auth.service.ts`

```typescript
import crypto from 'crypto';
import jwt from 'jsonwebtoken';
import bcrypt from 'bcrypt';
import { Redis } from 'ioredis';
import { CircuitBreaker, RetryPolicy } from '../../lib/resilience';
import { IdempotencyService } from '../../lib/idempotency';
import { logger } from '../../lib/logger';

export interface AuthConfig {
  jwtSecret: string;
  jwtExpiresIn: string;
  refreshTokenExpiresIn: number;
  bcryptRounds: number;
  redisUrl: string;
}

export interface LoginRequest {
  email: string;
  password: string;
  idempotencyKey?: string;
}

export interface LoginResponse {
  accessToken: string;
  refreshToken: string;
  expiresIn: number;
  userId: string;
  sessionId: string;
}

export interface TokenPayload {
  userId: string;
  sessionId: string;
  role: string;
  iat: number;
  exp: number;
}

export class AuthService {
  private redis: Redis;
  private circuitBreaker: CircuitBreaker;
  private idempotencyService: IdempotencyService;
  private config: AuthConfig;

  constructor(config: AuthConfig) {
    this.config = config;
    this.redis = new Redis(config.redisUrl, {
      retryStrategy: (times) => Math.min(times * 50, 2000),
      maxRetriesPerRequest: 3,
    });

    this.circuitBreaker = new CircuitBreaker({
      failureThreshold: 5,
      resetTimeout: 60000,
      halfOpenMaxAttempts: 3,
    });

    this.idempotencyService = new IdempotencyService(this.redis);
  }

  async login(request: LoginRequest): Promise<LoginResponse> {
    const correlationId = crypto.randomUUID();
    const startTime = Date.now();

    try {
      // Validação de input
      if (!request.email || !request.password) {
        throw new Error('Email and password are required');
      }

      // Verificação de idempotência
      if (request.idempotencyKey) {
        const cachedResponse = await this.idempotencyService.get(
          request.idempotencyKey
        );
        if (cachedResponse) {
          logger.info('Returning idempotent response', {
            correlationId,
            idempotencyKey: request.idempotencyKey,
          });
          return cachedResponse as LoginResponse;
        }
      }

      // Circuit breaker para acesso ao banco de dados
      const user = await this.circuitBreaker.execute(async () => {
        // Simulação de busca de usuário - em produção, integrar com repositório
        return await this.findUserByEmail(request.email);
      });

      if (!user) {
        throw new Error('Invalid credentials');
      }

      // Verificação de senha com bcrypt
      const passwordValid = await bcrypt.compare(
        request.password,
        user.passwordHash
      );

      if (!passwordValid) {
        await this.recordFailedAttempt(request.email);
        throw new Error('Invalid credentials');
      }

      // Reset de tentativas falhas
      await this.resetFailedAttempts(request.email);

      // Geração de tokens
      const sessionId = crypto.randomUUID();
      const accessToken = this.generateAccessToken(user.id, sessionId, user.role);
      const refreshToken = this.generateRefreshToken(sessionId);

      // Armazenamento de refresh token com TTL
      await this.storeRefreshToken(
        sessionId,
        user.id,
        refreshToken,
        this.config.refreshTokenExpiresIn
      );

      const response: LoginResponse = {
        accessToken,
        refreshToken,
        expiresIn: 3600, // 1 hora
        userId: user.id,
        sessionId,
      };

      // Cache de resposta idempotente
      if (request.idempotencyKey) {
        await this.idempotencyService.set(
          request.idempotencyKey,
          response,
          3600 // TTL de 1 hora
        );
      }

      const duration = Date.now() - startTime;
      logger.info('Login successful', {
        correlationId,
        userId: user.id,
        sessionId,
        duration,
      });

      return response;
    } catch (error) {
      const duration = Date.now() - startTime;
      logger.error('Login failed', {
        correlationId,
        error: error.message,
        email: this.maskEmail(request.email),
        duration,
      });

      if (error.message.includes('circuit open')) {
        throw new Error('Authentication service temporarily unavailable');
      }

      throw new Error('Authentication failed');
    }
  }

  async validateToken(token: string): Promise<TokenPayload> {
    try {
      const decoded = jwt.verify(token, this.config.jwtSecret) as TokenPayload;
      
      // Verificar se a sessão ainda está ativa
      const sessionActive = await this.redis.get(`session:${decoded.sessionId}`);
      if (!sessionActive) {
        throw new Error('Session expired');
      }

      return decoded;
    } catch (error) {
      logger.error('Token validation failed', {
        error: error.message,
        tokenHash: crypto.createHash('sha256').update(token).digest('hex'),
      });
      throw new Error('Invalid token');
    }
  }

  async refreshToken(refreshToken: string): Promise<LoginResponse> {
    const correlationId = crypto.randomUUID();

    try {
      // Verificar refresh token
      const sessionId = await this.verifyRefreshToken(refreshToken);
      if (!sessionId) {
        throw new Error('Invalid refresh token');
      }

      // Buscar informações da sessão
      const sessionData = await this.redis.hgetall(`session:${sessionId}`);
      if (!sessionData.userId) {
        throw new Error('Session not found');
      }

      // Gerar novo access token
      const newAccessToken = this.generateAccessToken(
        sessionData.userId,
        sessionId,
        sessionData.role
      );

      // Gerar novo refresh token (rotation)
      const newRefreshToken = this.generateRefreshToken(sessionId);
      await this.storeRefreshToken(
        sessionId,
        sessionData.userId,
        newRefreshToken,
        this.config.refreshTokenExpiresIn
      );

      logger.info('Token refreshed', {
        correlationId,
        userId: sessionData.userId,
        sessionId,
      });

      return {
        accessToken: newAccessToken,
        refreshToken: newRefreshToken,
        expiresIn: 3600,
        userId: sessionData.userId,
        sessionId,
      };
    } catch (error) {
      logger.error('Token refresh failed', {
        correlationId,
        error: error.message,
      });
      throw new Error('Token refresh failed');
    }
  }

  async logout(sessionId: string, userId: string): Promise<void> {
    const correlationId = crypto.randomUUID();

    try {
      // Remover sessão
      await this.redis.del(`session:${sessionId}`);
      await this.redis.del(`refresh:${sessionId}`);

      logger.info('Logout successful', {
        correlationId,
        userId,
        sessionId,
      });
    } catch (error) {
      logger.error('Logout failed', {
        correlationId,
        userId,
        sessionId,
        error: error.message,
      });
      throw new Error('Logout failed');
    }
  }

  private generateAccessToken(
    userId: string,
    sessionId: string,
    role: string
  ): string {
    return jwt.sign(
      { userId, sessionId, role },
      this.config.jwtSecret,
      { expiresIn: this.config.jwtExpiresIn }
    );
  }

  private generateRefreshToken(sessionId: string): string {
    return crypto.randomBytes(64).toString('hex');
  }

  private async storeRefreshToken(
    sessionId: string,
    userId: string,
    refreshToken: string,
    ttl: number
  ): Promise<void> {
    await this.redis.hset(`refresh:${sessionId}`, {
      token: refreshToken,
      userId,
      createdAt: Date.now().toString(),
    });
    await this.redis.expire(`refresh:${sessionId}`, ttl);
  }

  private async verifyRefreshToken(refreshToken: string): Promise<string | null> {
    // Buscar sessão pelo refresh token
    const keys = await this.redis.keys('refresh:*');
    
    for (const key of keys) {
      const sessionData = await this.redis.hgetall(key);
      if (sessionData.token === refreshToken) {
        return key.split(':')[1];
      }
    }
    
    return null;
  }

  private async findUserByEmail(email: string): Promise<any> {
    // Implementação real integraria com repositório
    // Simulação para demonstração
    return {
      id: 'user-123',
      email,
      passwordHash: await bcrypt.hash('password', this.config.bcryptRounds),
      role: 'user',
    };
  }

  private async recordFailedAttempt(email: string): Promise<void> {
    const key = `failed_attempts:${email}`;
    const attempts = await this.redis.incr(key);
    await this.redis.expire(key, 900); // 15 minutos

    if (attempts >= 5) {
      await this.redis.setex(`locked:${email}`, 1800, 'true'); // 30 minutos
    }
  }

  private async resetFailedAttempts(email: string): Promise<void> {
    await this.redis.del(`failed_attempts:${email}`);
    await this.redis.del(`locked:${email}`);
  }

  private maskEmail(email: string): string {
    const [local, domain] = email.split('@');
    if (local.length <= 2) return '*'.repeat(local.length) + '@' + domain;
    return local[0] + '*'.repeat(local.length - 2) + local.slice(-1) + '@' + domain;
  }
}

    body: {
      type: 'object',
      required: ['email', 'password'],
      properties: {
        email: { type: 'string', format: 'email' },
        password: { type: 'string', minLength: 8 },
        idempotencyKey: { type: 'string' },
      },
    },
  })
  @rateLimiter({ windowMs: 15 * 60 * 1000, max: 5 }) // 5 tentativas por 15 minutos
  async login(req: Request, res: Response): Promise<void> {
    const correlationId = req.headers['x-correlation-id'] as string || crypto.randomUUID();
    const startTime = Date.now();

    try {
      const request: LoginRequest = {
        email: req.body.email,
        password: req.body.password,
        idempotencyKey: req.headers['x-idempotency-key'] as string,
      };

      const result = await this.authService.login(request);

      const duration = Date.now() - startTime;
      logger.info('Login API completed', {
        correlationId,
        userId: result.userId,
        duration,
        status: 200,
      });

      res.status(200).json({
        success: true,
        data: result,
        correlationId,
      });
    } catch (error) {
      const duration = Date.now() - startTime;
      logger.error('Login API failed', {
        correlationId,
        error: error.message,
        duration,
        status: 401,
      });

      res.status(401).json({
        success: false,
        error: 'Authentication failed',
        correlationId,
      });
    }
  }

  async refresh(req: Request, res: Response): Promise<void> {
    const correlationId = req.headers['x-correlation-id'] as string || crypto.randomUUID();
    const startTime = Date.now();

    try {
      const refreshToken = req.body.refreshToken;
      if (!refreshToken) {
        throw new Error('Refresh token is required');
      }

      const result = await this.authService.refreshToken(refreshToken);

      const duration = Date.now() - startTime;
      logger.info('Token refresh completed', {
        correlationId,
        userId: result.userId,
        duration,
        status: 200,
      });

      res.status(200).json({
        success: true,
        data: result,
        correlationId,
      });
    } catch (error) {
      const duration = Date.now() - startTime;
      logger.error('Token refresh failed', {
        correlationId,
        error: error.message,
        duration,
        status: 401,
      });

      res.status(401).json({
        success: false,
        error: 'Token refresh failed',
        correlationId,
      });
    }
  }

  async logout(req: Request, res: Response): Promise<void> {
    const correlationId = req.headers['x-correlation-id'] as string || crypto.randomUUID();
    const startTime = Date.now();

    try {
      // Extrair token do header
      const authHeader = req.headers.authorization;
      if (!authHeader || !authHeader.startsWith('Bearer ')) {
        throw new Error('Authorization header missing');
      }

      const token = authHeader.split(' ')[1];
      const payload = await this.authService.validateToken(token);

      await this.authService.logout(payload.sessionId, payload.userId);

      const duration = Date.now() - startTime;
      logger.info('Logout completed', {
        correlationId,
        userId: payload.userId,
        duration,
        status: 200,
      });

      res.status(200).json({
        success: true,
        message: 'Logged out successfully',
        correlationId,
      });
    } catch (error) {
      const duration = Date.now() - startTime;
      logger.error('Logout failed', {
        correlationId,
        error: error.message,
        duration,
        status: 400,
      });

      res.status(400).json({
        success: false,
        error: 'Logout failed',
        correlationId,
      });
    }
  }

  async validate(req: Request, res: Response): Promise<void> {

        correlationId,
        userId: payload.userId,

        status: 200,
      });

        success: true,
        data: {
          userId: payload.userId,

          expiresAt: payload.exp,
        },
        correlationId,
      });
    } catch (error) {
      const duration = Date.now() - startTime;
      logger.error('Token validation failed', {
        correlationId,
        error: error.message,
        duration,
        status: 401,
      });

      res.status(401).json({
        success: false,
        error: 'Invalid token',
        correlationId,
      });
    }
  }

def validate_source_data(**context):
    """
    Valida dados de origem antes do processamento
    """
    hook = SnowflakeHook(snowflake_conn_id='snowflake_default')
    
    # Verificar completude dos dados
    queries = [
        """
        SELECT 
            COUNT(*) as total_events,
            MIN(event_timestamp) as min_timestamp,
            MAX(event_timestamp) as max_timestamp,
            COUNT(DISTINCT DATE(event_timestamp)) as days_with_data
        FROM raw_data.app_events 
        WHERE event_timestamp >= DATEADD(hour, -2, CURRENT_TIMESTAMP())
        """,
        """
        SELECT 
            COUNT(*) as total_llm_calls,
            SUM(CASE WHEN status_code != 200 THEN 1 ELSE 0 END) as failed_calls,
            AVG(latency_ms) as avg_latency
        FROM raw_data.llm_calls 
        WHERE call_timestamp >= DATEADD(hour, -2, CURRENT_TIMESTAMP())
        """
    ]
    
    validation_results = {}
    for i, query in enumerate(queries):
        result = hook.get_first(query)
        validation_results[f'validation_{i}'] = result
    
    # Verificar se há dados suficientes
    if validation_results['validation_0'][0] < 100:  # Menos de 100 eventos
        raise ValueError(f"Dados insuficientes: {validation_results['validation_0'][0]} eventos")
    
    # Verificar latência aceitável
    if validation_results['validation_1'][2] > 10000:  # Latência > 10s
        logging.warning(f"Latência LLM alta: {validation_results['validation_1'][2]}ms")
    
    context['ti'].xcom_push(key='validation_results', value=validation_results)
    return validation_results

def process_staging_layer(**context):
    """
    Processa camada STAGING - Limpeza e padronização
    """
    hook = SnowflakeHook(snowflake_conn_id='snowflake_default')

      AND user_id IS NOT NULL;
    """

    staging_llm_query = """

def build_product_marts(**context):
    """
    Constrói marts de produto para o PM
    """
    hook = SnowflakeHook(snowflake_conn_id='snowflake_default')
    
    # Mart: Métricas diárias de usuários ativos
    product_dau_query = """
    CREATE OR REPLACE TABLE marts.product_daily_active_users AS
    SELECT 
        event_date,
        COUNT(DISTINCT user_id) as daily_active_users,
        COUNT(DISTINCT session_id) as daily_sessions,

        COUNT(DISTINCT feature_name) as features_used,
        AVG(CASE WHEN event_type = 'session_duration' 
              THEN event_properties:duration::FLOAT END) as avg_session_duration
    FROM staging.events
    WHERE event_date >= DATEADD(day, -30, CURRENT_DATE())
    GROUP BY event_date
    ORDER BY event_date DESC;
    """
    
    # Mart: Adoção de features
    product_features_query = """
    CREATE OR REPLACE TABLE marts.product_feature_adoption AS
    SELECT 
        feature_name,
        event_date,
        COUNT(DISTINCT user_id) as daily_users,
        COUNT(*) as daily_events,
        SUM(CASE WHEN event_type = 'feature_completed' THEN 1 ELSE 0 END) as completions,
        AVG(CASE WHEN event_type = 'feature_time' 
              THEN event_properties:time_spent::FLOAT END) as avg_time_spent
    FROM staging.events
    WHERE feature_name IS NOT NULL
      AND event_date >= DATEADD(day, -30, CURRENT_DATE())
    GROUP BY feature_name, event_date
    ORDER BY event_date DESC, daily_users DESC;
    """
    
    # Mart: Funnel de conversão
    product_funnel_query = """
    CREATE OR REPLACE TABLE marts.product_funnel_conversion AS
    WITH funnel_steps AS (
        SELECT 
            user_id,
            session_id,
            MIN(CASE WHEN event_type = 'app_launch' THEN event_timestamp END) as step_1,
            MIN(CASE WHEN event_type = 'feature_view' THEN event_timestamp END) as step_2,
            MIN(CASE WHEN event_type = 'feature_interaction' THEN event_timestamp END) as step_3,
            MIN(CASE WHEN event_type = 'feature_completed' THEN event_timestamp END) as step_4
        FROM staging.events
        WHERE event_date = CURRENT_DATE() - 1
        GROUP BY user_id, session_id
    )
    SELECT 
        event_date,
        COUNT(DISTINCT user_id) as total_users,
        COUNT(DISTINCT CASE WHEN step_1 IS NOT NULL THEN user_id END) as step_1_count,
        COUNT(DISTINCT CASE WHEN step_2 IS NOT NULL THEN user_id END) as step_2_count,
        COUNT(DISTINCT CASE WHEN step_3 IS NOT NULL THEN user_id END) as step_3_count,
        COUNT(DISTINCT CASE WHEN step_4 IS NOT NULL THEN user_id END) as step_4_count,
        ROUND(step_2_count * 100.0 / NULLIF(step_1_count, 0), 2) as step_1_to_2_conversion,
        ROUND(step_3_count * 100.0 / NULLIF(step_2_count, 0), 2) as step_2_to_3_conversion,
        ROUND(step_4_count * 100.0 / NULLIF(step_3_count, 0), 2) as step_3_to_4_conversion
    FROM funnel_steps
    CROSS JOIN (SELECT CURRENT_DATE() - 1 as event_date)
    GROUP BY event_date;
    """
    
    hook.run(product_dau_query)
    hook.run(product_features_query)
    hook.run(product_funnel_query)
    
    logging.info("Product marts construídos com sucesso")
    return "Product marts built"

// components/performance-monitor.tsx - Core Web Vitals monitor
'use client'

import { useEffect } from 'react'
import { useReportWebVitals } from 'next/web-vitals'

export function PerformanceMonitor() {
  useReportWebVitals((metric) => {
    console.log(metric)
    
    // Send to analytics
    if (window.gtag) {
      window.gtag('event', metric.name, {
        value: Math.round(metric.name === 'CLS' ? metric.value * 1000 : metric.value),
        event_label: metric.id,
        non_interaction: true,
      })
    }
  })

  useEffect(() => {
    // Monitor LCP
    const observer = new PerformanceObserver((entryList) => {
      const entries = entryList.getEntries()
      const lastEntry = entries[entries.length - 1]
      console.log('LCP:', lastEntry.startTime)
    })

    observer.observe({ type: 'largest-contentful-paint', buffered: true })

    // Monitor CLS
    let clsValue = 0
    let clsEntries: LayoutShift[] = []

    const clsObserver = new PerformanceObserver((entryList) => {
      for (const entry of entryList.getEntries()) {
        if (!entry.hadRecentInput) {
          clsEntries.push(entry as LayoutShift)
          clsValue += entry.value
          console.log('CLS:', clsValue)
        }
      }
    })

    clsObserver.observe({ type: 'layout-shift', buffered: true })

    return () => {
      observer.disconnect()
      clsObserver.disconnect()
    }
  }, [])

  return null
}

// components/lazy-image.tsx - Image com lazy loading e WebP
'use client'

import * as React from 'react'
import { cn } from '@/lib/utils'

interface LazyImageProps extends React.ImgHTMLAttributes<HTMLImageElement> {
  src: string
  alt: string
  width: number
  height: number
  placeholderSrc?: string
  className?: string
  priority?: boolean
}

export function LazyImage({
  src,
  alt,
  width,
  height,
  placeholderSrc,
  className,
  priority = false,
  ...props
}: LazyImageProps) {
  const [isLoaded, setIsLoaded] = React.useState(false)
  const [error, setError] = React.useState(false)

  React.useEffect(() => {
    const img = new Image()
    img.src = src
    img.onload = () => setIsLoaded(true)
    img.onerror = () => setError(true)
  }, [src])

  // Generate WebP src if possible
  const webpSrc = src.replace(/\.(jpg|jpeg|png)$/i, '.webp')

  return (
    <picture>
      <source srcSet={webpSrc} type="image/webp" />
      <img
        src={error ? placeholderSrc || '/placeholder.svg' : src}
        alt={alt}
        width={width}
        height={height}
        className={cn(
          'transition-opacity duration-300',
          isLoaded ? 'opacity-100' : 'opacity-0',
          className
        )}
        loading={priority ? 'eager' : 'lazy'}
        decoding="async"
        {...props}
      />
    </picture>
  )
}

### 2. ANÁLISE STRIDE POR COMPONENTE

#### API Gateway (Kong)
- **Spoofing**: Tokens JWT podem ser roubados/reutilizados
- **Tampering**: Requests podem ser modificados em trânsito
- **Repudiation**: Logs de acesso podem ser alterados
- **Info Disclosure**: Headers sensíveis podem vazar
- **DoS**: Rate limiting insuficiente
- **Elevation**: Configuração incorreta de rotas

#### Auth Service
- **Spoofing**: Weak password policies, no MFA
- **Tampering**: Tokens sem assinatura forte
- **Repudiation**: Logs de autenticação não imutáveis
- **Info Disclosure**: Tokens em logs, PII exposto
- **DoS**: Account lockout attacks
- **Elevation**: Role escalation via JWT manipulation

#### Message Queue (RabbitMQ)
- **Spoofing**: Credenciais padrão não alteradas
- **Tampering**: Mensagens podem ser interceptadas/modificadas
- **Repudiation**: Não há garantia de entrega/processamento
- **Info Disclosure**: Dados sensíveis em filas sem criptografia
- **DoS**: Queue flooding
- **Elevation**: Permissões excessivas entre serviços

#### AI Service (LLM)
- **Spoofing**: Prompt injection para assumir identidade
- **Tampering**: Manipulação de contexto/input do modelo
- **Repudiation**: Não há logs de prompts/respostas
- **Info Disclosure**: Vazamento de dados via LLM output
- **DoS**: Modelo sobrecarregado com requests complexos
- **Elevation**: LLM usado para gerar código/credenciais

### 3. CONTROLES DE MITIGAÇÃO

#### API Gateway
1. WAF com regras OWASP Top 10
2. Rate limiting por IP/usuario
3. Validação de JWT com assinatura RSA256
4. Headers de segurança (CSP, HSTS)
5. Logs centralizados imutáveis

#### Auth Service
1. MFA obrigatório para admin
2. Tokens JWT com expiração curta (15min access, 7d refresh)
3. Redis para blacklist de tokens
4. Password policy: 12 chars, especial, número, maiúscula
5. Audit logs para todas as operações privilegiadas

#### Message Queue
1. TLS 1.3 para comunicação entre serviços
2. Autenticação mutual TLS (mTLS)
3. Criptografia de mensagens em repouso
4. Quotas por serviço
5. Dead letter queues para análise

#### AI Service
1. Sanitização de input do usuário
2. System prompt hardening
3. Output validation antes de renderizar
4. Context window limits
5. Monitoring de prompt injection patterns

### 4. RISCOS ACEITOS (DOCUMENTADOS)
1. **Risco BA-001**: Rate limiting pode bloquear usuários legítimos durante picos

   - Aceito por: Compliance + Security

   - Aceito por: Tech Lead + Security

const helmet = require('helmet');
const xss = require('xss-clean');
const hpp = require('hpp');
const mongoSanitize = require('express-mongo-sanitize');

class WAFMiddleware {
    constructor() {
        this.attackPatterns = {
            sqlInjection: [
                /(\%27)|(\')|(\-\-)|(\%23)|(#)/gi,
                /((\%3D)|(=))[^\n]*((\%27)|(\')|(\-\-)|(\%3B)|(;))/gi,
                /\w*((\%27)|(\'))((\%6F)|o|(\%4F))((\%72)|r|(\%52))/gi,
                /((\%27)|(\'))union/gi,
                /exec(\s|\+)+(s|x)p\w+/gi
            ],
            xss: [
                /<script\b[^<]*(?:(?!<\/script>)<[^<]*)*<\/script>/gi,
                /javascript:/gi,
                /on\w+\s*=/gi,
                /<\s*iframe/gi,
                /<\s*object/gi
            ],
            pathTraversal: [
                /\.\.\//gi,
                /\.\.\\/gi,
                /\/etc\/passwd/gi,
                /\/proc\/self\/environ/gi
            ],
            commandInjection: [
                /;|\||&|`|\$(?:\{|\(|\[)|\n|\r/gi,
                /(ping|nslookup|dig|whois)\s+.*/gi
            ]
        };
    }

    // Rate limiting configuration
    getRateLimiters() {
        return {
            api: rateLimit({
                windowMs: 15 * 60 * 1000, // 15 minutes
                max: 100, // limit each IP to 100 requests per windowMs
                message: 'Too many requests from this IP, please try again later',
                standardHeaders: true,
                legacyHeaders: false
            }),
            auth: rateLimit({
                windowMs: 15 * 60 * 1000,
                max: 5, // 5 login attempts per 15 minutes
                message: 'Too many authentication attempts',
                skipSuccessfulRequests: true
            }),
            ai: rateLimit({
                windowMs: 60 * 1000, // 1 minute
                max: 10, // 10 AI requests per minute
                message: 'AI service rate limit exceeded'
            })
        };
    }

    // Security headers middleware
    securityHeaders() {
        return helmet({
            contentSecurityPolicy: {
                directives: {
                    defaultSrc: ["'self'"],
                    styleSrc: ["'self'", "'unsafe-inline'"],
                    scriptSrc: ["'self'", "'unsafe-inline'", "'unsafe-eval'"],
                    imgSrc: ["'self'", "data:", "https:"],
                    connectSrc: ["'self'", "https://api.diyapp.com"],
                    fontSrc: ["'self'", "https://fonts.gstatic.com"],
                    objectSrc: ["'none'"],
                    mediaSrc: ["'self'"],
                    frameSrc: ["'none'"]
                }
            },
            hsts: {
                maxAge: 31536000,
                includeSubDomains: true,
                preload: true
            },
            referrerPolicy: { policy: 'strict-origin-when-cross-origin' }
        });
    }

    // Input validation and sanitization
    inputSanitization() {
        return [
            mongoSanitize(), // NoSQL injection protection
            xss(), // XSS protection
            hpp(), // HTTP Parameter Pollution protection
            this.sanitizeInputs.bind(this)
        ];
    }

    // Custom input sanitization
    sanitizeInputs(req, res, next) {
        const sanitize = (input) => {
            if (!input || typeof input !== 'string') return input;
            
            // Check for attack patterns
            for (const [attackType, patterns] of Object.entries(this.attackPatterns)) {
                for (const pattern of patterns) {
                    if (pattern.test(input)) {
                        // Log the attack attempt
                        this.logAttackAttempt(req, attackType, input);
                        
                        // For production, you might want to throw an error
                        // For now, we'll sanitize by removing dangerous patterns
                        input = input.replace(pattern, '');
                    }
                }
            }
            
            return input;
        };

        // Sanitize all inputs
        if (req.body) {
            for (const key in req.body) {
                if (typeof req.body[key] === 'string') {
                    req.body[key] = sanitize(req.body[key]);
                }
            }
        }

        if (req.query) {
            for (const key in req.query) {
                if (typeof req.query[key] === 'string') {
                    req.query[key] = sanitize(req.query[key]);
                }
            }
        }

        if (req.params) {
            for (const key in req.params) {
                if (typeof req.params[key] === 'string') {
                    req.params[key] = sanitize(req.params[key]);
                }
            }
        }

        next();
    }

    // Log attack attempts
    logAttackAttempt(req, attackType, payload) {
        const logEntry = {
            timestamp: new Date().toISOString(),
            ip: req.ip,
            userAgent: req.get('User-Agent'),
            attackType: attackType,
            endpoint: req.originalUrl,
            method: req.method,
            payload: payload.substring(0, 500), // Limit payload length
            headers: {
                referer: req.get('Referer'),
                origin: req.get('Origin')
            }
        };

        // Send to security monitoring system
        console.warn(`[WAF] Attack detected: ${attackType} from ${req.ip}`);
        
        // In production, send to SIEM/Security monitoring
        // this.sendToSIEM(logEntry);
    }

    // WAF monitoring endpoint
    monitorMiddleware() {
        return (req, res, next) => {
            const startTime = Date.now();
            
            res.on('finish', () => {
                const duration = Date.now() - startTime;
                const logData = {
                    timestamp: new Date().toISOString(),
                    ip: req.ip,
                    method: req.method,
                    url: req.originalUrl,
                    status: res.statusCode,
                    duration: duration,
                    userAgent: req.get('User-Agent'),
                    contentLength: res.get('Content-Length') || 0
                };
                
                // Log slow requests (> 5 seconds)
                if (duration > 5000) {
                    console.warn(`[WAF] Slow request detected: ${duration}ms for ${req.method} ${req.originalUrl}`);
                }
                
                // Log 4xx and 5xx errors
                if (res.statusCode >= 400) {
                    console.warn(`[WAF] Error ${res.statusCode} for ${req.method} ${req.originalUrl}`);
                }
            });
            
            next();
        };
    }
}

const http = require('http');
const socketIo = require('socket.io');
const cors = require('cors');
const helmet = require('helmet');
const path = require('path');
require('dotenv').config();

// Importar serviços
const { setupWebSocketServer } = require('./src/websocket/websocket-server');
const { setupAIAssistant } = require('./src/ai-assistant/assistant');
const logger = require('./src/utils/logger');

const app = express();
const server = http.createServer(app);
const io = socketIo(server, {
  cors: {
    origin: "*",
    methods: ["GET", "POST"]
  }
});

// Middleware
app.use(helmet());
app.use(cors());
app.use(express.json());
app.use(express.static(path.join(__dirname, 'public')));

// Configurar WebSockets
setupWebSocketServer(io);

// Configurar Assistente de IA
const aiAssistant = setupAIAssistant(io);

// Rotas da API
app.get('/api/health', (req, res) => {
  res.json({
    status: 'healthy',
    version: 'v12.0.0',
    features: ['ai-assistant', 'websockets', 'real-time-notifications'],
    timestamp: new Date().toISOString()
  });
});

app.post('/api/chat', async (req, res) => {
  try {
    const { message, userId, context } = req.body;
    
    if (!message) {
      return res.status(400).json({ error: 'Message is required' });
    }

    const response = await aiAssistant.processMessage(message, userId, context);
    
    // Enviar notificação em tempo real via WebSocket
    io.to(userId).emit('ai_response', {
      message: response.content,
      timestamp: new Date().toISOString(),
      context: response.context
    });

    res.json(response);
  } catch (error) {
    logger.error('Chat API error:', error);
    res.status(500).json({ error: 'Internal server error' });
  }
});

app.get('/api/notifications/:userId', (req, res) => {
  const { userId } = req.params;
  // Em produção, buscar do banco de dados
  res.json({
    userId,
    notifications: [],
    unreadCount: 0
  });
});

// Servir o dashboard principal
app.get('/', (req, res) => {
  res.sendFile(path.join(__dirname, 'index.html'));
});

// Tratamento de erros
app.use((err, req, res, next) => {
  logger.error('Unhandled error:', err);
  res.status(500).json({ error: 'Something went wrong!' });
});

const PORT = process.env.PORT || 3000;

server.listen(PORT, () => {

## 1. ADR-001: Arquitetura V12 Core

**docs/architecture/ADR-001-v12-architecture.md**

```markdown
# ADR-001: Arquitetura V12 Core - Modo Hive

## Status
Aceito

## Data de Decisão
2024-01-15

## Decisores
- Tech Lead: Arquitetura de Sistema
- SRE: Operações e Estabilidade
- Dev Backend: Implementação
- Dev Frontend: Interface

## Contexto
A V11 do DIYAPP enfrentava problemas de escalabilidade devido ao acoplamento entre módulos. O sistema precisava evoluir para suportar:
1. Refatoração paralela de módulos (Modo Hive)
2. 100% de autonomia operacional
3. Zero-downtime deployments
4. Monitoramento granular por módulo

## Opções Consideradas

### Opção 1: Monólito com Módulos Internos
- **Prós**: Simples de desenvolver, deploy único
- **Contras**: Acoplamento alto, difícil refatoração paralela, risco de downtime total

### Opção 2: Microsserviços Tradicionais
- **Prós**: Desacoplamento total, escalabilidade independente
- **Contras**: Complexidade operacional, latência entre serviços, custo infraestrutura

### Opção 3: Arquitetura Híbrida (Modo Hive)
- **Prós**: Balanceia acoplamento vs autonomia, permite refatoração paralela
- **Contras**: Requer orquestração inteligente

## Decisão
Adotamos a **Opção 3: Arquitetura Híbrida (Modo Hive)** com os seguintes componentes:

### Componentes Principais
1. **Core Engine**: Orquestrador central que gerencia módulos
2. **Módulos Autônomos**: Unidades funcionais independentes
3. **Message Bus**: Comunicação assíncrona entre módulos
4. **Health Monitor**: Sistema de monitoramento distribuído
5. **Config Manager**: Gerenciamento centralizado de configuração

### Princípios Arquiteturais
- **Autonomia Controlada**: Módulos são autônomos mas seguem contratos
- **Graceful Degradation**: Falha em um módulo não derruba o sistema
- **Observabilidade Nativa**: Todos os módulos exportam métricas
- **Deployment Independente**: Cada módulo pode ser deployado separadamente

### Diagrama de Arquitetura
```
┌─────────────────────────────────────────────────────────────┐
│                    Load Balancer                            │
└────────────────┬────────────────────────────────────────────┘
                 │
    ┌────────────▼────────────┐
    │      Core Engine        │
    │  - Orchestration        │
    │  - Service Discovery    │
    │  - Circuit Breaker      │
    └────────────┬────────────┘
                 │
    ┌────────────┼────────────┐
    ▼            ▼            ▼
┌───────┐  ┌───────┐  ┌───────┐
│ Mod A │  │ Mod B │  │ Mod C │  ← Módulos Autônomos
│ Hive  │  │ Hive  │  │ Hive  │     (Podem ser refatorados
└───┬───┘  └───┬───┘  └───┬───┘      paralelamente)
    │          │          │
    └──────────┼──────────┘
               │
    ┌──────────▼──────────┐
    │    Message Bus      │ ← Comunicação Assíncrona
    │  (Redis Streams)    │
    └──────────┬──────────┘
               │
    ┌──────────▼──────────┐
    │   Health Monitor    │ ← Coleta métricas de todos os módulos
    │   (Prometheus)      │
    └─────────────────────┘
```

## Consequências

### Positivas
- Refatoração paralela possível via Modo Hive
- Isolamento de falhas por módulo
- Deployment independente (zero-downtime)
- Monitoramento granular

### Negativas
- Complexidade aumentada de orquestração
- Necessidade de contrato de API bem definido
- Overhead de comunicação entre módulos

### Mitigações
- Testes de contrato automatizados
- Circuit breakers para falhas em cascata
- Cache distribuído para reduzir latência

import pandas as pd

class MLflowMonitor:
    """MLflow-based monitoring for LLM models"""
    
    def __init__(self, config: MLflowConfig):
        self.config = config
        mlflow.set_tracking_uri(config.tracking_uri)
        mlflow.set_registry_uri(config.registry_uri)
        
        # Ensure experiment exists
        try:
            self.experiment = mlflow.get_experiment_by_name(config.experiment_name)
            if self.experiment is None:
                self.experiment_id = mlflow.create_experiment(config.experiment_name)
            else:
                self.experiment_id = self.experiment.experiment_id
        except Exception as e:
            logger.error(f"Failed to setup MLflow experiment: {e}")
            raise
    
    def log_inference(self, 
                     model_name: str,
                     input_tokens: int,
                     output_tokens: int,
                     latency_ms: float,
                     cost_usd: float,
                     success: bool,
                     metadata: Dict[str, Any] = None,
                     tags: Dict[str, str] = None):
        """Log a single inference to MLflow"""
        
        with mlflow.start_run(experiment_id=self.experiment_id, run_name=f"inference_{datetime.now().isoformat()}"):
            # Log metrics
            mlflow.log_metric("input_tokens", input_tokens)
            mlflow.log_metric("output_tokens", output_tokens)
            mlflow.log_metric("total_tokens", input_tokens + output_tokens)
            mlflow.log_metric("latency_ms", latency_ms)
            mlflow.log_metric("cost_usd", cost_usd)
            mlflow.log_metric("success", 1 if success else 0)
            
            # Log parameters
            mlflow.log_param("model_name", model_name)
            mlflow.log_param("timestamp", datetime.now().isoformat())
            
            # Log metadata
            if metadata:
                mlflow.log_dict(metadata, "metadata.json")
            
            # Set tags
            if tags:
                mlflow.set_tags(tags)
            
            # Add to model registry if this is a production model
            if model_name == self.config.model_name:
                try:
                    model_uri = f"models:/{model_name}/{self.config.stage}"
                    mlflow.log_artifact(model_uri, "model_info")
                except:
                    pass
    
    def log_batch_metrics(self, 
                         model_name: str,
                         metrics: Dict[str, float],
                         period_minutes: int = 5):
        """Log aggregated metrics for a time period"""
        
        with mlflow.start_run(experiment_id=self.experiment_id, 
                            run_name=f"batch_{datetime.now().strftime('%Y%m%d_%H%M')}"):
            
            # Log all metrics
            for metric_name, value in metrics.items():
                mlflow.log_metric(metric_name, value)
            
            # Log parameters
            mlflow.log_param("model_name", model_name)
            mlflow.log_param("period_minutes", period_minutes)
            mlflow.log_param("batch_timestamp", datetime.now().isoformat())
            
            # Calculate and log derived metrics
            if "total_requests" in metrics and "failed_requests" in metrics:
                error_rate = metrics["failed_requests"] / max(metrics["total_requests"], 1)
                mlflow.log_metric("error_rate", error_rate)
            
            if "total_tokens" in metrics and "total_cost" in metrics:
                cost_per_token = metrics["total_cost"] / max(metrics["total_tokens"], 1)
                mlflow.log_metric("cost_per_token", cost_per_token)
    
    def get_production_metrics(self, 
                              start_date: datetime,
                              end_date: datetime = None) -> pd.DataFrame:
        """Retrieve production metrics for analysis"""
        
        if end_date is None:
            end_date = datetime.now()
        
        # Query MLflow for runs in date range
        runs = mlflow.search_runs(
            experiment_ids=[self.experiment_id],
            filter_string=f"attributes.start_time >= {start_date.timestamp()} "
                         f"AND attributes.start_time <= {end_date.timestamp()} "
                         f"AND tags.mlflow.runName LIKE 'batch_%'"
        )
        
        return runs
    
    def get_model_performance_history(self, 
                                     model_name: str,
                                     days: int = 30) -> Dict[str, List[float]]:
        """Get historical performance metrics for a model"""
        
        end_date = datetime.now()
        start_date = end_date - timedelta(days=days)
        
        runs = self.get_production_metrics(start_date, end_date)
        
        # Filter for specific model and extract metrics
        model_runs = runs[runs["params.model_name"] == model_name]
        
        performance = {
            "latency": model_runs["metrics.latency_ms"].tolist() if "metrics.latency_ms" in model_runs.columns else [],
            "error_rate": model_runs["metrics.error_rate"].tolist() if "metrics.error_rate" in model_runs.columns else [],
            "cost_per_token": model_runs["metrics.cost_per_token"].tolist() if "metrics.cost_per_token" in model_runs.columns else [],
            "throughput": model_runs["metrics.total_requests"].tolist() if "metrics.total_requests" in model_runs.columns else [],
            "timestamps": model_runs["params.batch_timestamp"].tolist()
        }
        
        return performance
    
    def register_model_version(self,
                              model_path: str,
                              model_name: str,
                              metrics: Dict[str, float],
                              description: str = "") -> str:
        """Register a new model version in MLflow registry"""
        
        # Log model
        run_id = mlflow.active_run().info.run_id if mlflow.active_run() else None
        
        if run_id:
            mlflow.log_artifact(model_path, "model")
            model_uri = f"runs:/{run_id}/model"
        else:
            with mlflow.start_run(experiment_id=self.experiment_id):
                mlflow.log_artifact(model_path, "model")
                model_uri = f"runs:/{mlflow.active_run().info.run_id}/model"
        
        # Register model
        mv = mlflow.register_model(model_uri, model_name)
        
        # Add description and metrics
        mlflow.tracking.MlflowClient().update_model_version(
            name=model_name,
            version=mv.version,
            description=description
        )
        
        # Log metrics for this version
        for metric_name, value in metrics.items():
            mlflow.tracking.MlflowClient().log_metric(
                run_id=mv.run_id,
                key=f"v{mv.version}_{metric_name}",
                value=value
            )
        
        return mv.version

            conn.execute("""

                    entities JSON,

                    FOREIGN KEY (prompt_hash) REFERENCES prompts(prompt_hash)
                )
            """)
            
            conn.execute("""

                    id INTEGER PRIMARY KEY AUTOINCREMENT,

            conn.execute("CREATE INDEX IF NOT EXISTS idx_prompts_feature ON prompts(feature_name)")
            conn.execute("CREATE INDEX IF NOT EXISTS idx_prompts_user ON prompts(user_id)")
            
            conn.commit()
    
    @contextmanager
    def _get_connection(self) -> Connection:
        """Get database connection with context manager"""
        conn = sqlite3.connect(self.db_config.database_url)
        conn.row_factory = sqlite3.Row
        try:
            yield conn
        finally:
            conn.close()
    
    def _hash_prompt(self, prompt_text: str) -> str:
        """Create hash for prompt deduplication"""
        return hashlib.sha256(prompt_text.encode()).hexdigest()
    
    def log_prompt(self,
                  prompt_text: str,
                  model_name: str,
                  response_text: Optional[str] = None,
                  prompt_tokens: Optional[int] = None,
                  response_tokens: Optional[int] = None,
                  latency_ms: Optional[float] = None,
                  cost_usd: Optional[float] = None,
                  success: bool = True,
                  user_id: Optional[str] = None,
                  session_id: Optional[str] = None,
                  feature_name: Optional[str] = None,
                  metadata: Optional[Dict[str, Any]] = None) -> str:
        """Log a prompt and response"""
        
        prompt_hash = self._hash_prompt(prompt_text)
        
        # Truncate if too long
        if len(prompt_text) > self.db_config.max_prompt_length:
            prompt_text = prompt_text[:self.db_config.max_prompt_length] + "... [TRUNCATED]"
        
        if response_text and len(response_text) > self.db_config.max_prompt_length:
            response_text = response_text[:self.db_config.max_prompt_length] + "... [TRUNCATED]"
        
        with self._get_connection() as conn:
            cursor = conn.cursor()
            
            # Check if prompt already exists
            cursor.execute(
                "SELECT id FROM prompts WHERE prompt_hash = ?",
                (prompt_hash,)
            )
            existing = cursor.fetchone()
            
            if existing:
                # Update existing record with new response
                cursor.execute("""
                    UPDATE prompts SET
                        response_text = ?,
                        response_tokens = ?,
                        latency_ms = ?,
                        cost_usd = ?,
                        success = ?,
                        indexed_at = CURRENT_TIMESTAMP
                    WHERE prompt_hash = ?
                """, (response_text, response_tokens, latency_ms, cost_usd, success, prompt_hash))
            else:
                # Insert new record
                cursor.execute("""
                    INSERT INTO prompts (
                        prompt_hash, prompt_text, prompt_tokens, model_name,
                        response_text, response_tokens, latency_ms, cost_usd,
                        success, user_id, session_id, feature_name, metadata
                    ) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
                """, (
                    prompt_hash, prompt_text, prompt_tokens, model_name,
                    response_text, response_tokens, latency_ms, cost_usd,
                    success, user_id, session_id, feature_name,
                    json.dumps(metadata) if metadata else None
                ))
            
            conn.commit()
        
        logger.debug(f"Logged prompt: {prompt_hash[:8]}...")
        return prompt_hash
    
    def get_prompts_for_fine_tuning(self,
