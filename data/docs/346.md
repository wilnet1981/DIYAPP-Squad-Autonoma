# DIYAPP Evolution - V12 Core - Arquitetura de Microsserviços

## 1. Documento de Arquitetura - V12 Core

### 1.1 Visão Geral
A arquitetura V12 evolui de uma aplicação monolítica para microsserviços, mantendo 100% de estabilidade autônoma. O sistema será composto por serviços independentes com responsabilidades bem definidas, comunicando-se via APIs REST e mensageria.

### 1.2 Princípios Arquiteturais
1. **Autonomia**: Cada serviço é independente em deploy, escala e falha
2. **Resiliência**: Circuit breakers, retry policies e fallbacks
3. **Observabilidade**: Logs estruturados, métricas e traces distribuídos
4. **Segurança por Design**: Zero-trust, autenticação centralizada
5. **DevOps**: Infrastructure as Code, CI/CD automatizado

## 3. ADRs (Architecture Decision Records)

### ADR-001: Estratégia de Comunicação entre Serviços
**Data**: 2024-01-15
**Status**: Aceita
**Autores**: Tech Lead + Especialista Infra

**CONTEXTO**:
Necessidade de comunicação assíncrona e síncrona entre microsserviços com garantias de entrega e baixa latência.

**DECISÃO**:
Implementar comunicação híbrida:
- APIs REST para operações síncronas (com timeout configurável)
- RabbitMQ para eventos assíncronos com DLQ (Dead Letter Queue)
- gRPC para comunicação interna entre serviços com alta performance

**OPÇÕES CONSIDERADAS**:
- **Opção A**: REST apenas - Prós: Simples, HTTP padrão. Contras: Acoplamento temporal, sem garantias de entrega
- **Opção B**: Kafka apenas - Prós: Alta throughput, persistência. Contras: Complexidade operacional, overkill para nosso volume
- **Opção C**: Híbrido (REST + RabbitMQ) - Prós: Balance entre simplicidade e robustez, DLQ nativa. Contras: Duas tecnologias para manter

**Opção escolhida**: C - Justificativa: Atende todos os casos de uso com maturidade operacional

**CONSEQUÊNCIAS**:
- Positivas: Resiliência a falhas, desacoplamento temporal, garantias de entrega
- Negativas: Complexidade aumentada, necessidade de monitoramento do message broker
- Riscos: Message loss se mal configurado. Mitigação: Configuração de persistência e replicação

### ADR-002: Estratégia de Banco de Dados
**Data**: 2024-01-15
**Status**: Aceita
**Autores**: Tech Lead + Especialista Infra

**CONTEXTO**:
Cada microsserviço precisa de isolamento de dados enquanto mantemos consistência eventual.

**DECISÃO**:
- Database per Service pattern
- PostgreSQL para dados transacionais
- Redis para cache e sessões
- Event Sourcing para consistência eventual entre serviços

**OPÇÕES CONSIDERADAS**:
- **Opção A**: Banco único compartilhado - Prós: Simples, ACID garantido. Contras: Acoplamento forte, single point of failure
- **Opção B**: Database per Service - Prós: Isolamento, escalabilidade independente. Contras: Consistência eventual, complexidade de queries distribuídas
- **Opção C**: Polyglot Persistence - Prós: Otimizado para cada caso de uso. Contras: Complexidade operacional extrema

**Opção escolhida**: B - Justificativa: Alinhado com princípios de microsserviços, permite evolução independente

**CONSEQUÊNCIAS**:
- Positivas: Escalabilidade independente, isolamento de falhas
- Negativas: SAGAs necessárias para transações distribuídas, eventual consistency
- Riscos: Data duplication. Mitigação: Design cuidadoso de bounded contexts

```typescript
// src/app.ts - Template padrão
import express from 'express';
import helmet from 'helmet';
import cors from 'cors';
import rateLimit from 'express-rate-limit';
import { createLogger, format, transports } from 'winston';
import { errorHandler } from './middleware/errorHandler';
import { requestLogger } from './middleware/requestLogger';
import healthRouter from './routes/health';
import apiRouter from './routes/api';

// Configuração do logger estruturado
const logger = createLogger({
  level: process.env.LOG_LEVEL || 'info',
  format: format.combine(
    format.timestamp(),
    format.errors({ stack: true }),
    format.json()
  ),
  defaultMeta: { service: process.env.SERVICE_NAME },
  transports: [
    new transports.Console(),
    new transports.File({ filename: 'logs/error.log', level: 'error' }),
    new transports.File({ filename: 'logs/combined.log' }),
  ],
});

const app = express();

// Security middleware
app.use(helmet());
app.use(cors({
  origin: process.env.ALLOWED_ORIGINS?.split(',') || [],
  credentials: true,
}));

// Rate limiting
const limiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 minutos
  max: 100, // limite por IP
  message: 'Too many requests from this IP',
});
app.use('/api', limiter);

// Logging middleware
app.use(requestLogger(logger));

// Body parsing
app.use(express.json({ limit: '10mb' }));
app.use(express.urlencoded({ extended: true }));

// Routes
app.use('/health', healthRouter);
app.use('/api/v1', apiRouter);

// Error handling (deve ser o último middleware)
app.use(errorHandler(logger));

// Graceful shutdown
process.on('SIGTERM', () => {
  logger.info('SIGTERM received, shutting down gracefully');
  process.exit(0);
});

export { app, logger };
```

### 4.4 Template de Controller
```typescript
// src/controllers/userController.ts
import { Request, Response, NextFunction } from 'express';
import { UserService } from '../services/userService';
import { ApiResponse } from '../models/ApiResponse';
import { validateRequest } from '../middleware/validation';
import { createUserSchema, updateUserSchema } from '../schemas/userSchemas';

export class UserController {
  constructor(private userService: UserService) {}

  @validateRequest(createUserSchema)
  async createUser(req: Request, res: Response, next: NextFunction): Promise<void> {
    try {
      const user = await this.userService.create(req.body);
      const response: ApiResponse = {
        success: true,
        data: user,
        timestamp: new Date().toISOString(),
      };
      res.status(201).json(response);
    } catch (error) {
      next(error);
    }
  }

  async getUser(req: Request, res: Response, next: NextFunction): Promise<void> {
    try {
      const userId = req.params.id;
      const user = await this.userService.findById(userId);
      
      if (!user) {
        res.status(404).json({
          success: false,
          error: 'User not found',
          timestamp: new Date().toISOString(),
        });
        return;
      }

      const response: ApiResponse = {
        success: true,
        data: user,
        timestamp: new Date().toISOString(),
      };
      res.json(response);
    } catch (error) {
      next(error);
    }
  }

  @validateRequest(updateUserSchema)
  async updateUser(req: Request, res: Response, next: NextFunction): Promise<void> {
    try {
      const userId = req.params.id;
      const updates = req.body;
      
      const user = await this.userService.update(userId, updates);
      
      const response: ApiResponse = {
        success: true,
        data: user,
        timestamp: new Date().toISOString(),
      };
      res.json(response);
    } catch (error) {
      next(error);
    }
  }

  async deleteUser(req: Request, res: Response, next: NextFunction): Promise<void> {
    try {
      const userId = req.params.id;
      await this.userService.delete(userId);
      
      const response: ApiResponse = {
        success: true,
        message: 'User deleted successfully',
        timestamp: new Date().toISOString(),
      };
      res.status(204).json(response);
    } catch (error) {
      next(error);
    }
  }
}
```

```tsx
  return {
    getItem(_key: string) {
      return Promise.resolve(null);
    },
    setItem(_key: string, value: any) {
      return Promise.resolve(value);
    },
    removeItem(_key: string) {
      return Promise.resolve();
    },
  };

  ? createWebStorage('local')
  : createNoopStorage();

  key: 'root',

  migrate: (state: any) => {
    // Migration logic for future version changes
```

## 4. Hook de performance (`src/hooks/usePerformance.ts`)

```tsx
import { useEffect, useCallback, useRef } from 'react';

interface PerformanceMetrics {
  lcp: number | null;
  fid: number | null;
  cls: number | null;
  ttfb: number | null;
}

export const usePerformance = () => {
  const metricsRef = useRef<PerformanceMetrics>({
    lcp: null,
    fid: null,
    cls: null,
    ttfb: null,
  });

  const reportMetric = useCallback((name: string, value: number) => {
    // Report to analytics service
    console.log(`[Performance] ${name}: ${value.toFixed(2)}ms`);
    
    // Send to backend for monitoring
    if (typeof window !== 'undefined' && 'sendBeacon' in navigator) {
      const data = {
        metric: name,
        value,
        timestamp: Date.now(),
        path: window.location.pathname,
      };
      navigator.sendBeacon('/api/metrics', JSON.stringify(data));
    }
  }, []);

  useEffect(() => {
    if (typeof window === 'undefined' || !('PerformanceObserver' in window)) {
      return;
    }

    // LCP Observer
    const lcpObserver = new PerformanceObserver((entryList) => {
      const entries = entryList.getEntries();
      const lastEntry = entries[entries.length - 1];
      metricsRef.current.lcp = lastEntry.startTime;
      reportMetric('LCP', lastEntry.startTime);
    });
    lcpObserver.observe({ type: 'largest-contentful-paint', buffered: true });

    // FID Observer
    const fidObserver = new PerformanceObserver((entryList) => {
      const entries = entryList.getEntries();
      entries.forEach((entry) => {
        if (entry instanceof PerformanceEventTiming) {
          metricsRef.current.fid = entry.processingStart - entry.startTime;
          reportMetric('FID', entry.processingStart - entry.startTime);
        }
      });
    });
    fidObserver.observe({ type: 'first-input', buffered: true });

    // CLS Observer
    const clsObserver = new PerformanceObserver((entryList) => {
      let clsValue = 0;
      entryList.getEntries().forEach((entry) => {
        if (!entry.hadRecentInput) {
          clsValue += entry.value;
        }
      });
      metricsRef.current.cls = clsValue;
      reportMetric('CLS', clsValue);
    });
    clsObserver.observe({ type: 'layout-shift', buffered: true });

    // TTFB
    const navigationEntry = performance.getEntriesByType('navigation')[0] as PerformanceNavigationTiming;
    if (navigationEntry) {
      metricsRef.current.ttfb = navigationEntry.responseStart;
      reportMetric('TTFB', navigationEntry.responseStart);
    }

    // Long Tasks monitoring
    const longTaskObserver = new PerformanceObserver((entryList) => {
      entryList.getEntries().forEach((entry) => {
        if (entry.duration > 50) {
          console.warn(`[Performance] Long task detected: ${entry.duration.toFixed(2)}ms`);
        }
      });
    });
    longTaskObserver.observe({ entryTypes: ['longtask'] });

    return () => {
      lcpObserver.disconnect();
      fidObserver.disconnect();
      clsObserver.disconnect();
      longTaskObserver.disconnect();
    };
  }, [reportMetric]);

  const getMetrics = useCallback(() => metricsRef.current, []);

  return { getMetrics };
};
```

class DatabaseConnection:
    """Connection pool PostgreSQL com resiliência"""
    
    def __init__(self):
        self._engine: Optional[AsyncEngine] = None
        self._session_factory: Optional[async_sessionmaker] = None
        self._connection_pool = None
        
    @retry(
        stop=stop_after_attempt(3),
        wait=wait_exponential(multiplier=1, min=4, max=10),
        retry=retry_if_exception_type((OperationalError, ConnectionError)),
        reraise=True,
    )
    async def initialize(self):
        """Inicializa pool de conexões com retry automático"""
        try:
            # Configuração otimizada para alta carga
            self._engine = create_async_engine(
                str(settings.SQLALCHEMY_DATABASE_URI),
                echo=False,
                poolclass=QueuePool,
                pool_size=settings.DB_POOL_SIZE,
                max_overflow=settings.DB_MAX_OVERFLOW,
                pool_timeout=settings.DB_POOL_TIMEOUT,
                pool_recycle=settings.DB_POOL_RECYCLE,
                pool_pre_ping=True,  # Verifica conexão antes de usar
                connect_args={
                    "command_timeout": 60,
                    "server_settings": {
                        "jit": "off",  # Desativa JIT para queries simples
                        "statement_timeout": "30000",  # 30s timeout por query
                    }
                }
            )
            
            self._session_factory = async_sessionmaker(
                bind=self._engine,
                class_=AsyncSession,
                expire_on_commit=False,
                autocommit=False,
                autoflush=False,
            )
            
            # Testa conexão
            async with self._engine.connect() as conn:
                await conn.execute("SELECT 1")
            
            logger.info("Database connection pool initialized successfully")
            
        except Exception as e:
            logger.error(f"Failed to initialize database connection: {e}")
            raise
    
    async def close(self):
        """Fecha pool de conexões graciosamente"""
        if self._engine:
            await self._engine.dispose()
            logger.info("Database connection pool closed")
    
    @contextlib.asynccontextmanager
    async def get_session(self) -> AsyncGenerator[AsyncSession, None]:
        """Context manager para sessões de banco com tratamento de erro"""
        if not self._session_factory:
            raise RuntimeError("Database not initialized")
        
        session: AsyncSession = self._session_factory()
        try:
            yield session
            await session.commit()
        except SQLAlchemyError as e:
            await session.rollback()
            logger.error(f"Database session error: {e}", exc_info=True)
            raise
        finally:
            await session.close()
    
    async def health_check(self) -> bool:
        """Health check do banco de dados"""
        try:
            async with self._engine.connect() as conn:
                result = await conn.execute("SELECT 1")
                return result.scalar() == 1
        except Exception as e:
            logger.error(f"Database health check failed: {e}")
            return False
    
    def get_engine(self) -> AsyncEngine:
        """Retorna engine para migrações"""
        if not self._engine:
            raise RuntimeError("Database not initialized")
        return self._engine

```javascript
const rules = require('./rules');
const threatIntelligence = require('./threat-intelligence');
const { EventEmitter } = require('events');

class WAFMiddleware extends EventEmitter {
  constructor(config) {
    super();
    this.config = config;
    this.blockedIPs = new Map();
    this.threatScore = new Map();
    this.rulesEngine = rules;
    this.threatIntel = threatIntelligence;
    
    // Initialize threat intelligence
    if (config.threatIntelligence.enabled) {
      this.threatIntel.initialize();
      setInterval(() => this.threatIntel.update(), 
        this.parseInterval(config.threatIntelligence.updateInterval));
    }
  }

  parseInterval(interval) {
    const units = {
      's': 1000,
      'm': 60 * 1000,
      'h': 60 * 60 * 1000,
      'd': 24 * 60 * 60 * 1000
    };
    const value = parseInt(interval);
    const unit = interval.replace(value, '');
    return value * (units[unit] || 1000);
  }

  async middleware(req, res, next) {
    const clientIP = req.headers['x-forwarded-for'] || req.socket.remoteAddress;
    const userAgent = req.headers['user-agent'] || '';
    
    // Check if IP is blocked
    if (this.isBlocked(clientIP)) {
      this.logSecurityEvent('BLOCKED_IP', {
        ip: clientIP,
        path: req.path,
        reason: 'IP in blocklist'
      });
      return res.status(403).json({
        error: 'Access denied',
        code: 'WAF_BLOCKED',
        incidentId: this.generateIncidentId()
      });
    }

    // Check threat intelligence
    if (await this.threatIntel.isMalicious(clientIP)) {
      this.blockIP(clientIP, 'Threat intelligence match');
      return res.status(403).json({
        error: 'Access denied',
        code: 'THREAT_INTEL_BLOCK'
      });
    }

    // Analyze request
    const threatAnalysis = await this.analyzeRequest(req);
    
    if (threatAnalysis.block) {
      this.handleThreat(clientIP, threatAnalysis);
      return res.status(403).json({
        error: 'Potential security threat detected',
        code: 'WAF_THREAT_DETECTED',
        threats: threatAnalysis.detectedThreats
      });
    }

    // Add security headers
    this.addSecurityHeaders(res);

    // Attach security context to request
    req.security = {
      threatScore: threatAnalysis.score,
      threatsDetected: threatAnalysis.detectedThreats,
      clientIP,
      timestamp: new Date().toISOString()
    };

    next();
  }

  async analyzeRequest(req) {
    const threats = [];
    let score = 0;

    // SQL Injection detection
    if (this.config.rules.sqlInjection) {
      const sqlThreats = this.rulesEngine.detectSQLInjection(req);
      if (sqlThreats.length > 0) {
        threats.push(...sqlThreats);
        score += sqlThreats.length * 10;
      }
    }

    // XSS detection
    if (this.config.rules.xss) {
      const xssThreats = this.rulesEngine.detectXSS(req);
      if (xssThreats.length > 0) {
        threats.push(...xssThreats);
        score += xssThreats.length * 8;
      }
    }

    // Path traversal
    if (this.config.rules.pathTraversal) {
      const pathThreats = this.rulesEngine.detectPathTraversal(req);
      if (pathThreats.length > 0) {
        threats.push(...pathThreats);
        score += pathThreats.length * 7;
      }
    }

    // Command injection
    if (this.config.rules.commandInjection) {
      const cmdThreats = this.rulesEngine.detectCommandInjection(req);
      if (cmdThreats.length > 0) {
        threats.push(...cmdThreats);
        score += cmdThreats.length * 15;
      }
    }

    // Suspicious headers
    if (this.config.rules.suspiciousHeaders) {
      const headerThreats = this.rulesEngine.detectSuspiciousHeaders(req);
      if (headerThreats.length > 0) {
        threats.push(...headerThreats);
        score += headerThreats.length * 5;
      }
    }

    return {
      block: score >= this.config.blockThreshold,
      score,
      detectedThreats: threats
    };
  }

  handleThreat(ip, analysis) {
    const currentScore = this.threatScore.get(ip) || 0;
    const newScore = currentScore + analysis.score;
    this.threatScore.set(ip, newScore);

    this.logSecurityEvent('THREAT_DETECTED', {
      ip,
      score: newScore,
      threats: analysis.detectedThreats,
      path: analysis.path
    });

    if (newScore >= this.config.blockThreshold * 2) {
      this.blockIP(ip, `Accumulated threat score: ${newScore}`);
    }

    // Emit event for monitoring
    this.emit('threatDetected', {
      ip,
      score: newScore,
      threats: analysis.detectedThreats,
      timestamp: new Date().toISOString()
    });
  }

  blockIP(ip, reason) {
    this.blockedIPs.set(ip, {
      timestamp: Date.now(),
      reason,
      expires: Date.now() + (24 * 60 * 60 * 1000) // 24 hours
    });

    this.logSecurityEvent('IP_BLOCKED', { ip, reason });
  }

  isBlocked(ip) {
    const block = this.blockedIPs.get(ip);
    if (!block) return false;
    
    if (block.expires < Date.now()) {
      this.blockedIPs.delete(ip);
      return false;
    }
    
    return true;
  }

  addSecurityHeaders(res) {
    Object.entries(this.config.headers).forEach(([header, value]) => {
      res.setHeader(header, value);
    });
  }

  logSecurityEvent(type, data) {
    const logEntry = {
      timestamp: new Date().toISOString(),
      type,
      ...data
    };

    // Write to security log
    const fs = require('fs');
    const path = require('path');
    fs.appendFileSync(
      path.join(__dirname, '../../../logs/security.log'),
      JSON.stringify(logEntry) + '\n'
    );

    // Console in development
    if (process.env.NODE_ENV !== 'production') {
      console.log(`[SECURITY] ${type}:`, data);
    }
  }

  generateIncidentId() {
    return `INC-${Date.now()}-${Math.random().toString(36).substr(2, 9)}`;
  }

  getStats() {
    return {
      blockedIPs: this.blockedIPs.size,
      threatScoreSize: this.threatScore.size,
      rules: Object.keys(this.config.rules).filter(k => this.config.rules[k]),
      lastThreatIntelUpdate: this.threatIntel.lastUpdate
    };
  }
}

module.exports = WAFMiddleware;
```

```javascript
class SecurityRules {
  constructor() {
    this.sqlPatterns = [
      /(\%27)|(\')|(\-\-)|(\%23)|(#)/gi,
      /((\%3D)|(=))[^\n]*((\%27)|(\')|(\-\-)|(\%3B)|(;))/gi,
      /\w*((\%27)|(\'))((\%6F)|o|(\%4F))((\%72)|r|(\%52))/gi,
      /((\%27)|(\'))union/gi,
      /exec(\s|\+)+(s|x)p\w+/gi,
      /UNION.*SELECT/gi,
      /INSERT.*INTO/gi,
      /DROP.*TABLE/gi,
      /DELETE.*FROM/gi,
      /UPDATE.*SET/gi
    ];

    this.xssPatterns = [
      /<script\b[^<]*(?:(?!<\/script>)<[^<]*)*<\/script>/gi,
      /javascript:/gi,
      /on\w+\s*=/gi,
      /<iframe/gi,
      /<object/gi,
      /<embed/gi,
      /data:/gi,
      /vbscript:/gi
    ];

    this.pathTraversalPatterns = [
      /\.\.\//gi,
      /\.\.\\/gi,
      /\/etc\/passwd/gi,
      /\/proc\/self/gi,
      /\\windows\\system32/gi
    ];

    this.commandInjectionPatterns = [
      /;\s*\w+/gi,
      /\|\s*\w+/gi,
      /&\s*\w+/gi,
      /\$\s*\(/gi,
      /`/gi,
      /\${\w+}/gi
    ];

    this.suspiciousHeaders = [
      'x-forwarded-host',
      'x-original-url',
      'x-rewrite-url',
      'proxy-connection'
    ];
  }

  detectSQLInjection(req) {
    const threats = [];
    const checkString = JSON.stringify({
      url: req.url,
      query: req.query,
      body: req.body,
      headers: req.headers
    }).toLowerCase();

    this.sqlPatterns.forEach((pattern, index) => {
      if (pattern.test(checkString)) {
        threats.push({
          type: 'SQL_INJECTION',
          pattern: pattern.toString(),
          location: this.findLocation(checkString, pattern)
        });
      }
    });

    return threats;
  }

  detectXSS(req) {
    const threats = [];
    const checkString = JSON.stringify({
      url: req.url,
      query: req.query,
      body: req.body,
      headers: req.headers
    });

    this.xssPatterns.forEach((pattern, index) => {
      if (pattern.test(checkString)) {
        threats.push({
          type: 'XSS',
          pattern: pattern.toString(),
          location: this.findLocation(checkString, pattern)
        });
      }
    });

    return threats;
  }

  detectPathTraversal(req) {
    const threats = [];
    const url = req.url.toLowerCase();
    const checkString = JSON.stringify(req.query).toLowerCase();

    this.pathTraversalPatterns.forEach(pattern => {
      if (pattern.test(url) || pattern.test(checkString)) {
        threats.push({
          type: 'PATH_TRAVERSAL',
          pattern: pattern.toString(),
          location: pattern.test(url) ? 'url' : 'query'
        });
      }
    });

    return threats;
  }

  detectCommandInjection(req) {
    const threats = [];
    const checkString = JSON.stringify({
      query: req.query,
      body: req.body
    });

    this.commandInjectionPatterns.forEach(pattern => {
      if (pattern.test(checkString)) {
        threats.push({
          type: 'COMMAND_INJECTION',
          pattern: pattern.toString(),
          location: this.findLocation(checkString, pattern)
        });
      }
    });

    return threats;
  }

  detectSuspiciousHeaders(req) {
    const threats = [];
    
    this.suspiciousHeaders.forEach(header => {
      if (req.headers[header]) {
        threats.push({
          type: 'SUSPICIOUS_HEADER',
          header,
          value: req.headers[header]
        });
      }
    });

    return threats;
  }

  findLocation(str, pattern) {
    const match = str.match(pattern);
    if (!match) return 'unknown';
    
    const index = str.indexOf(match[0]);
    if (index < 100) return 'beginning';
    if (index > str.length - 100) return 'end';
    return 'middle';
  }
}

module.exports = new SecurityRules();
```

```javascript
function calculateSimilarity(str1, str2) {
  // Implementação simplificada de similaridade de cosseno
  const words1 = str1.toLowerCase().split(/\s+/);
  const words2 = str2.toLowerCase().split(/\s+/);
  const allWords = [...new Set([...words1, ...words2])];
  
  const vector1 = allWords.map(word => words1.filter(w => w === word).length);
  const vector2 = allWords.map(word => words2.filter(w => w === word).length);
  
  const dotProduct = vector1.reduce((sum, val, i) => sum + val * vector2[i], 0);
  const magnitude1 = Math.sqrt(vector1.reduce((sum, val) => sum + val * val, 0));
  const magnitude2 = Math.sqrt(vector2.reduce((sum, val) => sum + val * val, 0));
  
  return dotProduct / (magnitude1 * magnitude2);
}
```

**src/ml/anomaly_detector.py**
```python
import numpy as np
import pandas as pd
from sklearn.ensemble import IsolationForest
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import precision_score, recall_score, f1_score
import joblib
import json
from datetime import datetime
from typing import Dict, List, Tuple, Optional
import warnings
warnings.filterwarnings('ignore')

class AnomalyDetector:
    """Sistema principal de detecção de anomalias com múltiplos algoritmos"""
    
    def __init__(self, config: Dict):
        self.config = config
        self.model = None
        self.scaler = StandardScaler()
        self.feature_names = []
        self.metrics_history = []
        self.anomaly_threshold = config['ml']['threshold_percentile']
        
    def train_isolation_forest(self, X_train: np.ndarray) -> None:
        """Treina modelo Isolation Forest"""
        print("Treinando Isolation Forest...")
        
        # Normalizar dados
        X_scaled = self.scaler.fit_transform(X_train)
        
        # Treinar modelo
        self.model = IsolationForest(
            n_estimators=self.config['ml']['n_estimators'],
            contamination=self.config['ml']['contamination'],
            random_state=self.config['ml']['random_state'],
            n_jobs=-1
        )
        
        self.model.fit(X_scaled)
        print(f"Modelo Isolation Forest treinado com {len(X_train)} amostras")
        
    def train_autoencoder(self, X_train: np.ndarray) -> None:
        """Treina Autoencoder para detecção de anomalias"""
        try:
            from tensorflow.keras.models import Model
            from tensorflow.keras.layers import Input, Dense, Dropout
            from tensorflow.keras.callbacks import EarlyStopping
            import tensorflow as tf
            
            print("Treinando Autoencoder...")
            
            # Normalizar dados
            X_scaled = self.scaler.fit_transform(X_train)
            input_dim = X_scaled.shape[1]
            
            # Construir autoencoder
            inputs = Input(shape=(input_dim,))
            encoded = Dense(64, activation='relu')(inputs)
            encoded = Dropout(0.2)(encoded)
            encoded = Dense(32, activation='relu')(encoded)
            encoded = Dropout(0.2)(encoded)
            encoded = Dense(16, activation='relu')(encoded)
            
            decoded = Dense(32, activation='relu')(encoded)
            decoded = Dropout(0.2)(decoded)
            decoded = Dense(64, activation='relu')(decoded)
            decoded = Dropout(0.2)(decoded)
            decoded = Dense(input_dim, activation='linear')(decoded)
            
            autoencoder = Model(inputs, decoded)
            autoencoder.compile(optimizer='adam', loss='mse')
            
            # Callbacks
            early_stopping = EarlyStopping(
                monitor='val_loss',
                patience=self.config['training']['early_stopping_patience'],
                restore_best_weights=True
            )
            
            # Treinar
            history = autoencoder.fit(
                X_scaled, X_scaled,
                epochs=self.config['training']['epochs'],
                batch_size=self.config['training']['batch_size'],
                validation_split=self.config['training']['validation_split'],
                callbacks=[early_stopping],
                verbose=0
            )
            
            self.model = autoencoder
            print(f"Autoencoder treinado com loss final: {history.history['loss'][-1]:.4f}")
            
        except ImportError:
            print("TensorFlow não disponível, usando Isolation Forest")
            self.train_isolation_forest(X_train)
    
    def train(self, X_train: np.ndarray, y_train: Optional[np.ndarray] = None) -> Dict:
        """Treina o modelo selecionado"""
        model_type = self.config['ml']['model_type']
        
        if model_type == 'autoencoder':
            self.train_autoencoder(X_train)
        else:  # isolation_forest padrão
            self.train_isolation_forest(X_train)
        
        # Avaliar se temos labels
        if y_train is not None:
            metrics = self.evaluate(X_train, y_train)
            self.metrics_history.append({
                'timestamp': datetime.now().isoformat(),
                'metrics': metrics
            })
            return metrics
        
        return {}
    
    def predict(self, X: np.ndarray) -> Tuple[np.ndarray, np.ndarray]:
        """Faz predição de anomalias"""
        if self.model is None:
            raise ValueError("Modelo não treinado. Chame train() primeiro.")
        
        # Normalizar
        X_scaled = self.scaler.transform(X)
        
        if self.config['ml']['model_type'] == 'autoencoder':
            # Para autoencoder, usar erro de reconstrução
            reconstructions = self.model.predict(X_scaled, verbose=0)
            mse = np.mean(np.power(X_scaled - reconstructions, 2), axis=1)
            
            # Calcular threshold baseado no percentil
            threshold = np.percentile(mse, self.anomaly_threshold)
            anomaly_scores = mse / threshold  # Normalizar pelo threshold
            predictions = (mse > threshold).astype(int)
            
        else:  # Isolation Forest
            anomaly_scores = -self.model.score_samples(X_scaled)  # Quanto maior, mais anômalo
            predictions = self.model.predict(X_scaled)
            predictions = (predictions == -1).astype(int)  # Converter -1/1 para 0/1
        
        return predictions, anomaly_scores
    
    def evaluate(self, X_test: np.ndarray, y_test: np.ndarray) -> Dict:
        """Avalia performance do modelo"""
        predictions, scores = self.predict(X_test)
        
        # Calcular métricas
        precision = precision_score(y_test, predictions, zero_division=0)
        recall = recall_score(y_test, predictions, zero_division=0)
        f1 = f1_score(y_test, predictions, zero_division=0)
        
        # Calcular acurácia
        accuracy = np.mean(predictions == y_test)
        
        # Detalhes das predições
        true_positives = np.sum((predictions == 1) & (y_test == 1))
        false_positives = np.sum((predictions == 1) & (y_test == 0))
        true_negatives = np.sum((predictions == 0) & (y_test == 0))
        false_negatives = np.sum((predictions == 0) & (y_test == 1))
        
        metrics = {
            'accuracy': float(accuracy),
            'precision': float(precision),
            'recall': float(recall),
            'f1_score': float(f1),
            'true_positives': int(true_positives),
            'false_positives': int(false_positives),
            'true_negatives': int(true_negatives),
            'false_negatives': int(false_negatives),
            'total_samples': len(X_test),
            'anomaly_rate': float(np.mean(predictions)),
            'threshold_percentile': self.anomaly_threshold
        }
        
        print(f"Performance do modelo:")
        print(f"  Acurácia: {accuracy:.4f}")
        print(f"  Precisão: {precision:.4f}")
        print(f"  Recall: {recall:.4f}")
        print(f"  F1-Score: {f1:.4f}")
        print(f"  Taxa de anomalias detectadas: {np.mean(predictions):.4f}")
        
        return metrics
    
    def save_model(self, path: str) -> None:
        """Salva modelo e scaler"""
        model_data = {
            'model': self.model,
            'scaler': self.scaler,
            'config': self.config,
            'feature_names': self.feature_names,
            'metrics_history': self.metrics_history,
            'timestamp': datetime.now().isoformat()
        }
        joblib.dump(model_data, path)
        print(f"Modelo salvo em: {path}")
    
    def load_model(self, path: str) -> None:
        """Carrega modelo salvo"""
        model_data = joblib.load(path)
        self.model = model_data['model']
        self.scaler = model_data['scaler']
        self.config = model_data['config']
        self.feature_names = model_data['feature_names']
        self.metrics_history = model_data['metrics_history']
        print(f"Modelo carregado de: {path}")
    
    def optimize_threshold(self, X_val: np.ndarray, y_val: np.ndarray) -> float:
        """Otimiza threshold para melhor F1-Score"""
        predictions, scores = self.predict(X_val)
        
        # Testar diferentes thresholds
        thresholds = np.percentile(scores, range(90, 100))
        best_f1 = 0
        best_threshold = self.anomaly_threshold
        
        for threshold in thresholds:
            preds = (scores > threshold).astype(int)
            f1 = f1_score(y_val, preds, zero_division=0)
            
            if f1 > best_f1:
                best_f1 = f1
                best_threshold = threshold
        
        self.anomaly_threshold = best_threshold
        print(f"Threshold otimizado: {best_threshold:.4f} (F1-Score: {best_f1:.4f})")
        return best_threshold
```

        metrics = {}
        
        # CPU
        cpu_percent = psutil.cpu_percent(interval=0.1, percpu=True)
        metrics['cpu_usage'] = np.mean(cpu_percent)
        metrics['cpu_usage_std'] = np.std(cpu_percent)
        metrics['cpu_usage_max'] = np.max(cpu_percent)
        
        # Memória
        memory = psutil.virtual_memory()
        metrics['memory_usage'] = memory.percent
        metrics['memory_available_gb'] = memory.available / (1024**3)
        
        # Disco

        metrics['disk_usage'] = disk.percent

        # Rede

        metrics['network_packets'] = net_io.packets_sent + net_io.packets_recv
        
        # Processos
        metrics['process_count'] = len(psutil.pids())
        
        # Timestamp
        metrics['timestamp'] = datetime.now().isoformat()
        metrics['hour_of_day'] = datetime.now().hour
        metrics['day_of_week'] = datetime.now().weekday()

```javascript
// Sistema de busca para documentação
class DocumentationSearch {
    constructor() {
        this.searchIndex = [];
        this.init();
    }
    
    async init() {
        await this.buildSearchIndex();
        this.setupEventListeners();
    }
    
    async buildSearchIndex() {
        // Índice de busca com todos os conteúdos
        this.searchIndex = [
            {
                id: 'getting-started',
                title: 'Começando',
                content: 'Guia de início rápido para usar a API DIYAPP V12',
                category: 'Guia',
                url: '#getting-started'
            },
            {
                id: 'authentication',
                title: 'Autenticação',
                content: 'Sistema de autenticação JWT e chaves de API',
                category: 'API',
                url: '#authentication'
            },
            {
                id: 'endpoints',
                title: 'Endpoints',
                content: 'Documentação completa de todos os endpoints da API',
                category: 'API',
                url: '#endpoints'
            }
        ];
    }
    
    setupEventListeners() {
        const searchInput = document.getElementById('searchInput');
        if (searchInput) {
            searchInput.addEventListener('input', (e) => this.search(e.target.value));
        }
    }
    
    search(query) {
        if (!query || query.length < 2) {
            this.clearResults();
            return;
        }
        
        const results = this.searchIndex.filter(item => {
            const searchText = `${item.title} ${item.content} ${item.category}`.toLowerCase();
            return searchText.includes(query.toLowerCase());
        });
        
        this.displayResults(results);
    }
    
    displayResults(results) {
        // Implementar exibição de resultados
        console.log('Resultados:', results);
    }
    
    clearResults() {
        // Limpar resultados
    }
}

// Inicializar quando o DOM estiver pronto
document.addEventListener('DOMContentLoaded', () => {
    new DocumentationSearch();

    updateEndpointStatus();
});

async function updateEndpointStatus() {
    const endpoints = [
        { name: '/auth/login', method: 'POST', status: 'operational' },
        { name: '/users', method: 'GET', status: 'operational' },
        { name: '/projects', method: 'POST', status: 'operational' },
        { name: '/automations', method: 'GET', status: 'degraded' },
        { name: '/system/health', method: 'GET', status: 'operational' }
    ];
    
    const tbody = document.getElementById('endpointStatus');
    if (tbody) {
        tbody.innerHTML = endpoints.map(endpoint => `
            <tr>
                <td><code>${endpoint.name}</code></td>
                <td><span class="badge bg-${getMethodColor(endpoint.method)}">${endpoint.method}</span></td>
                <td><span class="badge bg-${getStatusColor(endpoint.status)}">${endpoint.status}</span></td>
                <td>${getRandomLatency()}ms</td>
                <td>${new Date().toLocaleTimeString()}</td>
            </tr>
        `).join('');
    }
}

function getMethodColor(method) {
    const colors = {
        'GET': 'success',
        'POST': 'warning',
        'PUT': 'info',
        'DELETE': 'danger',
        'PATCH': 'secondary'
    };
    return colors[method] || 'secondary';
}

function getStatusColor(status) {
    const colors = {
        'operational': 'success',
        'degraded': 'warning',
        'outage': 'danger',
        'maintenance': 'info'
    };
    return colors[status] || 'secondary';
}

function getRandomLatency() {
    return Math.floor(Math.random() * 100) + 50;
}
```
