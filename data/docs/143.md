ADR-001: Decisão de Arquitetura V12 - Microserviços
Data: 2024-01-15
Status: Aceita
Autores: Tech Lead, Especialista Infra, Especialista Backend

CONTEXTO:
O DIYAPP atual (monolito) atingiu limites de escalabilidade. Novas features de IA
estão causando acoplamento excessivo e dificultando deploy independente.
Necessidade de: 1) Deploy independente por domínio 2) Escala seletiva 3) Resiliência
4) Observabilidade granular.

DECISÃO:
Adotar arquitetura de microserviços com os seguintes princípios:
1. Cada domínio de negócio vira um microserviço independente
2. Comunicação via eventos assíncronos (80%) e APIs síncronas (20%)
3. Cada serviço tem seu próprio banco de dados
4. Gateway de API único para clientes externos
5. Service Mesh para comunicação interna

OPÇÕES CONSIDERADAS:
- Opção A: Monolito modularizado (atual) - Prós: Simplicidade, transações ACID fáceis
  Contras: Acoplamento, deploy único, escala limitada
- Opção B: Microsserviços completos - Prós: Independência, escala seletiva, resiliência
  Contras: Complexidade operacional, consistência eventual
- Opção C: Arquitetura hexagonal - Prós: Desacoplamento, testabilidade
  Contras: Não resolve escala operacional

Opção escolhida: B - Justificativa: Alinhamento com roadmap de features independentes,
necessidade de escalar serviços de IA separadamente, tolerância a consistência eventual.

CONSEQUÊNCIAS:
Positivas: Deploy independente, escala seletiva, resiliência por isolamento,
tecnologias específicas por domínio.
Negativas: Complexidade distribuída, debugging mais difícil, consistência eventual,
overhead de comunicação.
Riscos: Latência em chamadas cross-service, problemas de rede, monitoramento complexo.
Mitigação: Implementar circuit breakers, retry policies, tracing distribuído.

REVISÃO: 2024-04-15 (3 meses para avaliação de custo/benefício)
*/

CONTEXTO:
Necessidade de padronizar tecnologias para garantir interoperabilidade,

DECISÃO:
Stack padrão por camada:
1. Runtime: Node.js 18+ (LTS) com TypeScript
2. Framework: NestJS para serviços, Express para utilities leves
3. Banco de Dados: PostgreSQL (transacional), MongoDB (documentos), Redis (cache)
4. Mensageria: Apache Kafka para eventos críticos, RabbitMQ para workers
5. Service Mesh: Istio (produção), Consul Connect (desenvolvimento)
6. Observabilidade: OpenTelemetry + Jaeger (tracing), Prometheus + Grafana (métricas)
7. Container: Docker + Kubernetes (EKS na AWS)
8. CI/CD: GitHub Actions + ArgoCD

OPÇÕES CONSIDERADAS:
- Opção A: Java/Spring Boot - Prós: Mature, enterprise-grade
  Contras: Overhead de memória, curva mais íngreme
- Opção B: Go - Prós: Performance, concorrência nativa
  Contras: Ecossistema menor para IA
- Opção C: Node.js/NestJS - Prós: Produtividade, ecossistema rico, bom para IA
  Contras: Single-threaded, menos maduro para transações complexas

Opção escolhida: C - Justificativa: Alinhamento com stack atual da squad,
produtividade para features de IA, comunidade ativa.

CONSEQUÊNCIAS:
Positivas: Consistência tecnológica, reutilização de código, comunidade ativa.
Negativas: Limitações de performance CPU-intensive, necessidade de workers separados.
Riscos: Callback hell se não bem gerenciado. Mitigação: Padrões async/await,
uso de RxJS onde apropriado.

import { Injectable, Logger, OnModuleInit, OnModuleDestroy } from '@nestjs/common';
import { MetricsService } from './metrics/metrics.service';
import { TracingService } from './tracing/tracing.service';
import { HealthService } from './health/health.service';

@Injectable()
export abstract class BaseService implements OnModuleInit, OnModuleDestroy {
  protected readonly logger = new Logger(this.constructor.name);
  protected serviceName: string;

  constructor(
    protected readonly metricsService: MetricsService,
    protected readonly tracingService: TracingService,
    protected readonly healthService: HealthService
  ) {
    this.serviceName = this.constructor.name.replace('Service', '').toLowerCase();
  }

  async onModuleInit(): Promise<void> {
    this.logger.log(`${this.serviceName} service initialized`);
    await this.healthService.registerService(this.serviceName);
  }

  async onModuleDestroy(): Promise<void> {
    this.logger.log(`${this.serviceName} service shutting down`);
    await this.healthService.deregisterService(this.serviceName);
  }

  protected async withMetricsAndTracing<T>(
    operationName: string,
    tags: Record<string, string>,
    fn: () => Promise<T>
  ): Promise<T> {
    const startTime = Date.now();
    const span = this.tracingService.startSpan(operationName, tags);

    try {
      const result = await fn();
      const duration = Date.now() - startTime;
      
      this.metricsService.recordHistogram(
        `${this.serviceName}_${operationName}_duration`,
        duration,
        tags
      );
      
      this.metricsService.incrementCounter(
        `${this.serviceName}_${operationName}_success`,
        1,
        tags
      );
      
      span.setStatus({ code: 1 }); // OK
      return result;
    } catch (error) {
      const duration = Date.now() - startTime;
      
      this.metricsService.recordHistogram(
        `${this.serviceName}_${operationName}_duration`,
        duration,
        { ...tags, error: error.constructor.name }
      );
      
      this.metricsService.incrementCounter(
        `${this.serviceName}_${operationName}_error`,
        1,
        { ...tags, error_type: error.constructor.name }
      );
      
      span.setStatus({ code: 2, message: error.message }); // ERROR
      span.recordException(error);
      throw error;
    } finally {
      span.end();
    }
  }
}

# src/core/auth/service.py
from datetime import datetime, timedelta
from typing import Optional, Dict, Any
from jose import JWTError, jwt
from passlib.context import CryptContext
import uuid
from sqlalchemy.orm import Session
from sqlalchemy.exc import SQLAlchemyError
import logging

from ..config import settings
from .models import UserCreate, UserUpdate, Token, TokenData
from ..database.models import User, RefreshToken

logger = logging.getLogger(__name__)

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")

class AuthService:
    """Serviço de autenticação com JWT e refresh tokens"""
    
    def __init__(self, db: Session):
        self.db = db
    
    def verify_password(self, plain_password: str, hashed_password: str) -> bool:
        """Verifica senha com bcrypt"""
        try:
            return pwd_context.verify(plain_password, hashed_password)
        except Exception as e:
            logger.error(f"Erro ao verificar senha: {str(e)}")
            return False
    
    def get_password_hash(self, password: str) -> str:
        """Gera hash da senha"""
        return pwd_context.hash(password)
    
    def authenticate_user(self, email: str, password: str) -> Optional[User]:
        """Autentica usuário por email e senha"""
        try:
            user = self.db.query(User).filter(
                User.email == email,
                User.is_active == True
            ).first()
            
            if not user:
                logger.warning(f"Tentativa de login com email não encontrado: {email}")
                return None
            
            if not self.verify_password(password, user.hashed_password):
                logger.warning(f"Senha incorreta para usuário: {email}")
                return None
            
            logger.info(f"Usuário autenticado: {email}")
            return user
            
        except SQLAlchemyError as e:
            logger.error(f"Erro de database na autenticação: {str(e)}")
            raise
    
    def create_access_token(self, data: Dict[str, Any], expires_delta: Optional[timedelta] = None) -> str:
        """Cria JWT access token"""
        to_encode = data.copy()
        
        if expires_delta:
            expire = datetime.utcnow() + expires_delta
        else:
            expire = datetime.utcnow() + timedelta(minutes=settings.ACCESS_TOKEN_EXPIRE_MINUTES)
        
        to_encode.update({"exp": expire, "type": "access"})
        encoded_jwt = jwt.encode(to_encode, settings.SECRET_KEY, algorithm=settings.ALGORITHM)
        return encoded_jwt
    
    def create_refresh_token(self, user_id: int) -> RefreshToken:
        """Cria e armazena refresh token"""
        try:
            # Revoga tokens antigos do usuário
            self.db.query(RefreshToken).filter(
                RefreshToken.user_id == user_id,
                RefreshToken.is_revoked == False
            ).update({"is_revoked": True})
            
            # Cria novo token
            token = str(uuid.uuid4())
            expires_at = datetime.utcnow() + timedelta(days=settings.REFRESH_TOKEN_EXPIRE_DAYS)
            
            refresh_token = RefreshToken(
                user_id=user_id,
                token=token,
                expires_at=expires_at
            )
            
            self.db.add(refresh_token)
            self.db.commit()
            
            logger.info(f"Refresh token criado para usuário {user_id}")
            return refresh_token
            
        except SQLAlchemyError as e:
            self.db.rollback()
            logger.error(f"Erro ao criar refresh token: {str(e)}")
            raise
    
    def verify_refresh_token(self, token: str) -> Optional[User]:
        """Verifica refresh token e retorna usuário"""
        try:
            refresh_token = self.db.query(RefreshToken).filter(
                RefreshToken.token == token,
                RefreshToken.is_revoked == False,
                RefreshToken.expires_at > datetime.utcnow()
            ).first()
            
            if not refresh_token:
                return None
            
            user = self.db.query(User).filter(
                User.id == refresh_token.user_id,
                User.is_active == True
            ).first()
            
            return user
            
        except SQLAlchemyError as e:
            logger.error(f"Erro ao verificar refresh token: {str(e)}")
            raise
    
    def revoke_refresh_token(self, token: str) -> bool:
        """Revoga um refresh token"""
        try:
            result = self.db.query(RefreshToken).filter(
                RefreshToken.token == token
            ).update({"is_revoked": True})
            
            self.db.commit()
            return result > 0
            
        except SQLAlchemyError as e:
            self.db.rollback()
            logger.error(f"Erro ao revogar refresh token: {str(e)}")
            raise
    
    def create_user(self, user_data: Dict[str, Any]) -> User:
        """Cria novo usuário"""
        try:
            # Verifica se email já existe
            existing_user = self.db.query(User).filter(
                User.email == user_data["email"]
            ).first()
            
            if existing_user:
                raise ValueError("Email já registrado")

import React, { createContext, useContext } from 'react';
import { tokens } from './tokens';

interface ThemeContextType {
  tokens: typeof tokens;
  mode: 'light' | 'dark';
  toggleMode: () => void;
}

const ThemeContext = createContext<ThemeContextType | undefined>(undefined);

export const ThemeProvider: React.FC<{ children: React.ReactNode }> = ({ children }) => {
  const [mode, setMode] = React.useState<'light' | 'dark'>('light');

  const toggleMode = () => {
    setMode(prev => prev === 'light' ? 'dark' : 'light');
  };

  return (
    <ThemeContext.Provider value={{ tokens, mode, toggleMode }}>
      <div className={`theme-${mode}`} style={tokens.global}>
        {children}
      </div>
    </ThemeContext.Provider>
  );
};

export const useTheme = () => {
  const context = useContext(ThemeContext);
  if (!context) {
    throw new Error('useTheme must be used within ThemeProvider');
  }
  return context;
};

class IAMValidator:
    """Validador de políticas IAM"""
    
    def __init__(self):
        self.policies = IAM_POLICIES
        
    def has_permission(self, role: Role, permission: Permission) -> bool:
        """Verifica se um papel tem determinada permissão"""
        if role not in self.policies:
            return False
        return permission in self.policies[role].permissions
    
    def validate_access(self, user_role: Role, required_permissions: List[Permission]) -> Dict:
        """Valida se usuário tem todas as permissões necessárias"""
        missing_permissions = []
        
        for perm in required_permissions:
            if not self.has_permission(user_role, perm):
                missing_permissions.append(perm.value)
        
        return {
            "authorized": len(missing_permissions) == 0,
            "missing_permissions": missing_permissions,
            "user_role": user_role.value,
            "mfa_required": self.policies[user_role].mfa_required if user_role in self.policies else False
        }
    
    def get_role_hierarchy(self) -> Dict:
        """Retorna hierarquia de papéis para auditoria"""
        hierarchy = {}
        for role, policy in self.policies.items():
            hierarchy[role.value] = {
                "permissions": [p.value for p in policy.permissions],
                "mfa_required": policy.mfa_required,
                "session_timeout": policy.session_timeout,
                "description": policy.description
            }
        return hierarchy

# Middleware de autorização
def authorization_middleware(required_permissions: List[Permission]):
    """Decorator para validação de autorização em endpoints"""
    def decorator(func):
        def wrapper(*args, **kwargs):
            # Em produção, isso viria do JWT/contexto
            user_role = kwargs.get('user_role', Role.GUEST)
            
            validator = IAMValidator()
            validation_result = validator.validate_access(user_role, required_permissions)
            
            if not validation_result["authorized"]:
                return {
                    "error": "Unauthorized",
                    "message": f"Missing permissions: {validation_result['missing_permissions']}",
                    "required_role": user_role.value,
                    "code": 403
                }, 403
            
            # Verificar MFA se necessário
            if validation_result["mfa_required"]:
                mfa_verified = kwargs.get('mfa_verified', False)
                if not mfa_verified:
                    return {
                        "error": "MFA Required",
                        "message": "Multi-factor authentication required for this role",
                        "code": 401
                    }, 401
            
            return func(*args, **kwargs)
        return wrapper
    return decorator

import re

class SecureCodingStandards:
    """Implementação dos padrões de codificação segura"""
    
    # Configurações de segurança
    JWT_ALGORITHM = "HS256"
    JWT_ACCESS_EXPIRY = timedelta(minutes=15)
    JWT_REFRESH_EXPIRY = timedelta(days=7)
    BCRYPT_COST = 12
    MIN_PASSWORD_LENGTH = 12
    
    @staticmethod
    def validate_password(password: str) -> Dict[str, Any]:
        """Validação robusta de senha"""
        if len(password) < SecureCodingStandards.MIN_PASSWORD_LENGTH:
            return {"valid": False, "error": f"Password must be at least {SecureCodingStandards.MIN_PASSWORD_LENGTH} characters"}
        
        checks = {
            "uppercase": any(c.isupper() for c in password),
            "lowercase": any(c.islower() for c in password),
            "digit": any(c.isdigit() for c in password),
            "special": any(c in "!@#$%^&*()_+-=[]{}|;:,.<>?" for c in password),
            "no_common": password.lower() not in [
                "password", "123456", "qwerty", "admin", "welcome"
            ]
        }
        
        if not all(checks.values()):
            missing = [k for k, v in checks.items() if not v]
            return {"valid": False, "error": f"Password missing: {', '.join(missing)}"}
        
        return {"valid": True, "strength": "strong"}
    
    @staticmethod
    def hash_password(password: str) -> str:
        """Hash seguro de senha usando bcrypt"""
        salt = bcrypt.gensalt(rounds=SecureCodingStandards.BCRYPT_COST)
        return bcrypt.hashpw(password.encode('utf-8'), salt).decode('utf-8')

        """Verificação segura de senha"""
