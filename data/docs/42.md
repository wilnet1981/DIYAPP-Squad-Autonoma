## ADR-001: Arquitetura de Microsserviços V11

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra

**CONTEXTO:**
A V10 do DIYAPP enfrenta problemas de escalabilidade devido à arquitetura monolítica. O crescimento do código base tornou deploys lentos e testes complexos. Necessitamos de uma arquitetura que permita:
1. Escalabilidade independente por funcionalidade
2. Deploys contínuos sem downtime
3. Isolamento de falhas
4. Equipes autônomas por serviço

**DECISÃO:**
Adotar arquitetura de microsserviços com:
- 6 serviços principais + API Gateway
- Comunicação via HTTP/REST para síncrono e RabbitMQ para assíncrono
- Banco de dados por serviço (exceto shared-db para dados comuns)
- Monorepo com Turborepo para gerenciamento de dependências
- Service Discovery via Consul
- Observabilidade centralizada (Prometheus + Grafana + ELK)

**OPÇÕES CONSIDERADAS:**
- **Opção A:** Arquitetura monolítica aprimorada
  - Prós: Simplicidade inicial, deploy único
  - Contras: Escalabilidade limitada, acoplamento alto
- **Opção B:** Microsserviços com monorepo
  - Prós: Reuso de código, CI/CD unificado, dependências gerenciadas
  - Contras: Complexidade inicial maior
- **Opção C:** Microsserviços com repositórios separados
  - Prós: Independência total por serviço
  - Contras: Versionamento complexo, CI/CD fragmentado

**Opção escolhida:** B - Justificativa: Equilíbrio entre autonomia e consistência, ideal para squads autônomas.

**CONSEQUÊNCIAS:**
**Positivas:**
- Escalabilidade horizontal por serviço
- Deploys independentes
- Stack tecnológica adequada por serviço
- Isolamento de falhas

**Negativas:**
- Complexidade operacional aumentada
- Latência em comunicações entre serviços
- Gerenciamento de transações distribuídas

**Riscos:**
1. **Service Discovery falha:** Mitigar com health checks e fallback
2. **Cascata de falhas:** Implementar circuit breaker (Resilience4j)
3. **Consistência de dados:** Usar padrão Saga para transações

**REVISÃO:** 2024-04-15

## Template de Serviço

```typescript
// apps/auth-service/src/index.ts
import express from 'express';
import cors from 'cors';
import helmet from 'helmet';
import rateLimit from 'express-rate-limit';
import { ConsulService } from '@diyapp/shared-config';
import { logger } from '@diyapp/shared-utils';
import { errorHandler } from './middleware/errorHandler';
import authRoutes from './routes/auth.routes';
import healthRoutes from './routes/health.routes';

const app = express();
const PORT = process.env.PORT || 3002;

// Security middleware
app.use(helmet());
app.use(cors());
app.use(express.json());

// Rate limiting
const limiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 minutes
  max: 100 // limit each IP to 100 requests per windowMs
});
app.use('/api/', limiter);

// Routes
app.use('/api/auth', authRoutes);
app.use('/health', healthRoutes);

// Error handling
app.use(errorHandler);

// Service registration
if (process.env.NODE_ENV !== 'test') {
  const consul = new ConsulService('auth-service', PORT);
  consul.register();
}

const server = app.listen(PORT, () => {
  logger.info(`Auth service running on port ${PORT}`);
});

// Graceful shutdown
process.on('SIGTERM', () => {
  logger.info('SIGTERM received, shutting down gracefully');
  server.close(() => {
    logger.info('Process terminated');
  });
});

export default app;
```

```typescript
// 📁 src/store/agent-store.ts
import { create } from 'zustand'
import { persist } from 'zustand/middleware'

export interface Agent {
  id: string
  name: string
  status: 'idle' | 'running' | 'error' | 'completed'
  type: 'frontend' | 'backend' | 'qa' | 'llm'
  cpuUsage: number
  memoryUsage: number
  lastActive: Date
  logs: LogEntry[]
  metrics: AgentMetrics
}

export interface LogEntry {
  id: string
  timestamp: Date
  level: 'info' | 'warning' | 'error' | 'success'
  message: string
  source: string
}

export interface AgentMetrics {
  requestsPerMinute: number
  averageResponseTime: number
  errorRate: number
  uptime: number
}

interface AgentStore {
  agents: Agent[]
  selectedAgentId: string | null
  isStreaming: boolean
  
  // Actions
  addAgent: (agent: Omit<Agent, 'id' | 'logs' | 'metrics' | 'lastActive'>) => void
  updateAgentStatus: (agentId: string, status: Agent['status']) => void
  updateAgentMetrics: (agentId: string, metrics: Partial<AgentMetrics>) => void
  addLogEntry: (agentId: string, log: Omit<LogEntry, 'id' | 'timestamp'>) => void
  selectAgent: (agentId: string | null) => void
  startStreaming: () => void
  stopStreaming: () => void
  clearAgentLogs: (agentId: string) => void
  
  // Computed
  getSelectedAgent: () => Agent | undefined
  getAgentLogs: (agentId: string) => LogEntry[]
  getSystemMetrics: () => SystemMetrics
}

interface SystemMetrics {
  totalAgents: number
  activeAgents: number
  totalErrors: number
  averageCpuUsage: number
  averageMemoryUsage: number
}

export const useAgentStore = create<AgentStore>()(
  persist(
    (set, get) => ({
      agents: [],
      selectedAgentId: null,
      isStreaming: false,

      addAgent: (agentData) => {
        const newAgent: Agent = {
          ...agentData,
          id: `agent-${Date.now()}`,
          logs: [],
          metrics: {
            requestsPerMinute: 0,
            averageResponseTime: 0,
            errorRate: 0,
            uptime: 100,
          },
          lastActive: new Date(),
        }

        set((state) => ({
          agents: [...state.agents, newAgent],
        }))
      },

      updateAgentStatus: (agentId, status) => {
        set((state) => ({
          agents: state.agents.map((agent) =>
            agent.id === agentId
              ? { ...agent, status, lastActive: new Date() }
              : agent
          ),
        }))
      },

      updateAgentMetrics: (agentId, metrics) => {
        set((state) => ({
          agents: state.agents.map((agent) =>
            agent.id === agentId
              ? {
                  ...agent,
                  metrics: { ...agent.metrics, ...metrics },
                  lastActive: new Date(),
                }
              : agent
          ),
        }))
      },

      addLogEntry: (agentId, log) => {
        const newLog: LogEntry = {
          ...log,
          id: `log-${Date.now()}-${Math.random().toString(36).substr(2, 9)}`,
          timestamp: new Date(),
        }

        set((state) => ({
          agents: state.agents.map((agent) =>
            agent.id === agentId
              ? {
                  ...agent,
                  logs: [newLog, ...agent.logs].slice(0, 100), // Keep last 100 logs
                  lastActive: new Date(),
                }
              : agent
          ),
        }))
      },

      selectAgent: (agentId) => {
        set({ selectedAgentId: agentId })
      },

      startStreaming: () => {
        set({ isStreaming: true })
      },

      stopStreaming: () => {
        set({ isStreaming: false })
      },

      clearAgentLogs: (agentId) => {
        set((state) => ({
          agents: state.agents.map((agent) =>
            agent.id === agentId ? { ...agent, logs: [] } : agent
          ),
        }))
      },

      getSelectedAgent: () => {
        const { agents, selectedAgentId } = get()
        return agents.find((agent) => agent.id === selectedAgentId)
      },

      getAgentLogs: (agentId) => {
        const { agents } = get()
        const agent = agents.find((a) => a.id === agentId)
        return agent?.logs || []
      },

      getSystemMetrics: () => {
        const { agents } = get()
        const totalAgents = agents.length
        const activeAgents = agents.filter((a) => a.status === 'running').length
        const totalErrors = agents.reduce(
          (sum, agent) => sum + agent.metrics.errorRate,
          0
        )
        const averageCpuUsage =
          agents.reduce((sum, agent) => sum + agent.cpuUsage, 0) / totalAgents || 0
        const averageMemoryUsage =
          agents.reduce((sum, agent) => sum + agent.memoryUsage, 0) / totalAgents || 0

        return {
          totalAgents,
          activeAgents,
          totalErrors,
          averageCpuUsage,
          averageMemoryUsage,
        }
      },
    }),
    {
      name: 'agent-store',
      partialize: (state) => ({ agents: state.agents }), // Only persist agents
    }
  )
)
```

```typescript
import Fastify, { FastifyInstance, FastifyServerOptions } from 'fastify';
import fastifyHelmet from 'fastify-helmet';
import fastifyCors from 'fastify-cors';
import fastifyRateLimit from '@fastify/rate-limit';
import fastifySwagger from '@fastify/swagger';
import fastifySwaggerUi from '@fastify/swagger-ui';
import { Server } from 'http';

import { logger } from './utils/logger';
import { errorHandler } from './middleware/error-handler';
import { authMiddleware } from './middleware/auth';
import { validationMiddleware } from './middleware/validation';
import { rateLimitMiddleware } from './middleware/rate-limit';

import healthRoutes from './routes/health';
import agentRoutes from './routes/agents';
import taskRoutes from './routes/tasks';
import metricsRoutes from './routes/metrics';

import { config } from '@/shared/config';

export async function buildServer(
  opts: FastifyServerOptions = {}
): Promise<FastifyInstance> {
  const server = Fastify({
    logger,
    disableRequestLogging: config.NODE_ENV === 'production',
    ...opts,
  });

  // Register security plugins
  await server.register(fastifyHelmet, {
    contentSecurityPolicy: {
      directives: {
        defaultSrc: ["'self'"],
        styleSrc: ["'self'", "'unsafe-inline'"],
        scriptSrc: ["'self'"],
      },
    },
  });

  await server.register(fastifyCors, {
    origin: config.CORS_ORIGINS,
    credentials: true,
  });

  // Rate limiting
  await server.register(fastifyRateLimit, {
    global: true,
    max: config.RATE_LIMIT_MAX,
    timeWindow: config.RATE_LIMIT_TIME_WINDOW,
    redis: config.REDIS_URL,
    enableDraftSpec: false,
  });

  // Swagger documentation
  if (config.NODE_ENV !== 'production') {
    await server.register(fastifySwagger, {
      swagger: {
        info: {
          title: 'DIYAPP V11 Core API',
          description: 'API Gateway for DIYAPP Evolution V11',
          version: config.API_VERSION,
        },
        host: `localhost:${config.PORT}`,
        schemes: ['http', 'https'],
        consumes: ['application/json'],
        produces: ['application/json'],
        securityDefinitions: {
          bearerAuth: {
            type: 'apiKey',
            name: 'Authorization',
            in: 'header',
            description: 'Bearer token',
          },
        },
      },
    });

    await server.register(fastifySwaggerUi, {
      routePrefix: '/docs',
      uiConfig: {
        docExpansion: 'list',
        deepLinking: false,
      },
    });
  }

  // Custom middleware
  server.addHook('onRequest', authMiddleware);
  server.addHook('preValidation', validationMiddleware);
  server.addHook('preHandler', rateLimitMiddleware);

  // Error handling
  server.setErrorHandler(errorHandler);

  // Register routes
  server.register(healthRoutes, { prefix: '/health' });
  server.register(agentRoutes, { prefix: '/api/v1/agents' });
  server.register(taskRoutes, { prefix: '/api/v1/tasks' });
  server.register(metricsRoutes, { prefix: '/api/v1/metrics' });

  // Health check endpoint
  server.get('/', async () => {
    return {
      status: 'ok',
      service: 'diyapp-v11-api-gateway',
      version: config.API_VERSION,
      timestamp: new Date().toISOString(),
    };
  });

  return server;
}

export async function startServer(): Promise<FastifyInstance> {
  try {
    const server = await buildServer();
    
    await server.listen({
      port: config.PORT,
      host: config.HOST,
    });

    logger.info(`Server listening on ${config.HOST}:${config.PORT}`);
    
    if (config.NODE_ENV !== 'production') {
      logger.info(`Swagger docs available at http://${config.HOST}:${config.PORT}/docs`);
    }

    // Graceful shutdown
    const signals = ['SIGINT', 'SIGTERM'];
    signals.forEach(signal => {
      process.on(signal, async () => {
        logger.info(`Received ${signal}, shutting down gracefully`);
        await server.close();
        process.exit(0);
      });
    });

    return server;
  } catch (error) {
    logger.error(error, 'Failed to start server');
    process.exit(1);
  }
}
```

## 1. ADR-001: Arquitetura de Microsserviços V11

**ADR-001: Arquitetura Baseada em Microsserviços com Monorepo Turborepo**

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra

**CONTEXTO:**
A V10 do DIYAPP enfrenta problemas de escalabilidade devido à arquitetura monolítica. O crescimento da base de usuários e a complexidade das novas features exigem:
- Deploy independente por serviço
- Escala horizontal seletiva
- Stack tecnológica específica por domínio
- Manutenção simplificada de dependências compartilhadas

**DECISÃO:**
Adotar arquitetura de microsserviços organizada em monorepo usando Turborepo. Cada serviço será independente, com seu próprio banco de dados quando necessário, comunicando-se via eventos (Kafka) e REST/GraphQL.

**OPÇÕES CONSIDERADAS:**
- **Opção A:** Monorepo com Turborepo - Prós: Compartilhamento de código fácil, CI/CD simplificado, consistência de versões. Contras: Acoplamento no repositório, build times maiores.
- **Opção B:** Multi-repos - Prós: Total independência, equipes autônomas. Contras: Versionamento complexo, duplicação de código, CI/CD fragmentado.
- **Opção C:** Monolito modular - Prós: Simplicidade inicial, deploy único. Contras: Escalabilidade limitada, acoplamento técnico.

**Opção escolhida:** A - Justificativa: Turborepo oferece cache inteligente de builds, permite deploy seletivo e mantém consistência enquanto permite independência operacional.

**CONSEQUÊNCIAS:**
**Positivas:** 
- Deploy independente por serviço
- Escala horizontal granular
- Stack tecnológica específica por domínio
- Cache de builds otimizado

**Negativas:**
- Complexidade operacional aumentada
- Necessidade de orquestração (Docker Compose/K8s)
- Overhead de comunicação entre serviços

**Riscos:**
- Latência em chamadas entre serviços (mitigar com cache e design assíncrono)
- Consistência eventual (mitigar com sagas e compensação)
- Debug distribuído (mitigar com tracing distribuído)

```typescript
// lib/utils.ts
import { type ClassValue, clsx } from 'clsx'
import { twMerge } from 'tailwind-merge'

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}

export function formatBytes(bytes: number, decimals = 2): string {
  if (bytes === 0) return '0 Bytes'
  
  const k = 1024
  const dm = decimals < 0 ? 0 : decimals
  const sizes = ['Bytes', 'KB', 'MB', 'GB', 'TB']
  
  const i = Math.floor(Math.log(bytes) / Math.log(k))
  
  return `${parseFloat((bytes / Math.pow(k, i)).toFixed(dm))} ${sizes[i]}`
}

export function formatDuration(ms: number): string {
  if (ms < 1000) return `${ms}ms`
  if (ms < 60000) return `${(ms / 1000).toFixed(1)}s`
  if (ms < 3600000) return `${Math.floor(ms / 60000)}m ${Math.floor((ms % 60000) / 1000)}s`
  return `${Math.floor(ms / 3600000)}h ${Math.floor((ms % 3600000) / 60000)}m`
}

export function truncateText(text: string, maxLength: number): string {
  if (text.length <= maxLength) return text
  return `${text.slice(0, maxLength)}...`
}
```

```javascript
function errorHandler(error, request, reply) {
  const correlationId = request.correlationId;

  const message = statusCode === 500 ? 'Internal Server Error' : error.message;

      stack: error.stack,
      code: error.code,
    },
    request: {
      method: request.method,
      url: request.url,
      params: request.params,
      query: request.query,
      body: request.body,
    },
  }, 'Request error');
  
  // Never expose stack trace to client

    error: message,
    correlationId,
    timestamp: new Date().toISOString(),
  };
  
  // Add validation details for 400 errors
  if (statusCode === 400 && error.validation) {
    response.validation = error.validation;
  }
  
  reply.code(statusCode).send(response);
}
```

```javascript
        retryStrategy: (times) => {
          this.connectionAttempts = times;
          if (times > this.maxConnectionAttempts) {
            logger.error('Max Redis connection attempts reached');
            return null;
          }
          const delay = Math.min(times * 1000, 5000);
          logger.warn(`Redis connection attempt ${times}, retrying in ${delay}ms`);
          return delay;
        },
        maxRetriesPerRequest: 3,
        enableReadyCheck: true,
      });

      this.client.on('error', (error) => {
        logger.error({ error }, 'Redis client error');
        this.isConnected = false;
      });

      this.client.on('connect', () => {
        logger.info('Redis connected');
        this.isConnected = true;
        this.connectionAttempts = 0;
      });

      this.client.on('close', () => {
        logger.warn('Redis connection closed');
        this.isConnected = false;
      });

      // Test connection
      await this.client.ping();
      logger.info('Redis connection test successful');
      
      return this.client;
    } catch (error) {
      logger.error({ error }, 'Failed to connect to Redis');
      throw error;
    }
  }

  async disconnect() {
    if (this.client) {
      await this.client.quit();
      this.isConnected = false;
    }
  }

  getClient() {
    if (!this.isConnected) {
      throw new Error('Redis client not connected');
    }
    return this.client;
  }

        latency,
        connectionAttempts: this.connectionAttempts,
      };
    } catch (error) {
      return {
        healthy: false,
        error: error.message,
        connectionAttempts: this.connectionAttempts,
      };
    }
  }
```

```javascript
      return this.queues.get(name);
    }

    const defaultOptions = {
      connection: this.redisClient,
      defaultJobOptions: {
        attempts: 3,
        backoff: {
          type: 'exponential',
          delay: 1000,
        },
        removeOnComplete: 100,
        removeOnFail: 1000,
      },
    };

    const queue = new Queue(name, { ...defaultOptions, ...options });
    
    // Create QueueEvents for monitoring
    const queueEvents = new QueueEvents(name, { connection: this.redisClient });
    
    // Set up event listeners
    queueEvents.on('completed', ({ jobId, returnvalue }) => {
      logger.info({ jobId, queue: name }, 'Job completed');
    });

    queueEvents.on('failed', ({ jobId, failedReason }) => {
      logger.error({ jobId, queue: name, reason: failedReason }, 'Job failed');
    });

    queueEvents.on('stalled', ({ jobId }) => {
      logger.warn({ jobId, queue: name }, 'Job stalled');
    });

    this.queues.set(name, queue);
    this.queueEvents.set(name, queueEvents);

    logger.info({ queue: name }, 'Queue created');
    return queue;
  }

  async createWorker(queueName, processor, options = {}) {
    if (this.workers.has(queueName)) {
      return this.workers.get(queueName);
    }

    const defaultOptions = {
      connection: this.redisClient,
      concurrency: 5,
      limiter: {
        max: 10,
        duration: 1000,
      },
    };

    const worker = new Worker(queueName, processor, { ...defaultOptions, ...options });

    worker.on('completed', (job) => {
      logger.info({
        jobId: job.id,
        queue: queueName,
        duration: Date.now() - job.timestamp,
      }, 'Worker completed job');
    });

    worker.on('failed', (job, err) => {
      logger.error({
        jobId: job?.id,
        queue: queueName,
        error: err.message,
        attemptsMade: job?.attemptsMade,
      }, 'Worker failed job');
    });

    worker.on('error', (err) => {
      logger.error({ error: err.message, queue: queueName }, 'Worker error');
    });

    this.workers.set(queueName, worker);
    logger.info({ queue: queueName }, 'Worker created');

    return worker;
  }

  async addJob(queueName, jobName, data, options = {}) {
    const queue = this.queues.get(queueName);
    if (!queue) {
      throw new Error(`Queue ${queueName} not found`);
    }

    const job = await queue.add(jobName, data, options);
    
    logger.info({
      jobId: job.id,
      queue: queueName,
      jobName,
    }, 'Job added to queue');

    return job;
  }

  async getQueueMetrics(queueName) {
    const queue = this.queues.get(queueName);
    if (!queue) {
      throw new Error(`Queue ${queueName} not found`);
    }

    const [
      waiting,
      active,
      completed,
      failed,
      delayed,
    ] = await Promise.all([
      queue.getWaitingCount(),
      queue.getActiveCount(),
      queue.getCompletedCount(),
      queue.getFailedCount(),
      queue.getDelayedCount(),
    ]);

    return {

      active,
      completed,
      failed,
      delayed,
      total: waiting + active + completed + failed + delayed,
    };
  }

    for (const [name, worker] of this.workers) {
      await worker.close();
      logger.info({ worker: name }, 'Worker closed');
    }

    for (const [name, queue] of this.queues) {
      await queue.close();
      logger.info({ queue: name }, 'Queue closed');

    // Close all queue events
    for (const [name, queueEvents] of this.queueEvents) {

      };
    } catch (error) {
      return {
        healthy: false,
        error: error.message,
      };
    }
  }
}
```

## 1. Arquitetura de Microsserviços - ADR-001

**ADR-001: Arquitetura Baseada em Microsserviços para V11**

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead

**CONTEXTO:**
DIYAPP V10 evoluiu para uma arquitetura monolítica que dificulta:
- Deploy independente de funcionalidades
- Escalabilidade seletiva
- Manutenção paralela por squads
- Isolamento de falhas

**DECISÃO:**
Adotar arquitetura de microsserviços com:
1. API Gateway como ponto único de entrada
2. 4 serviços core inicialmente (Auth, User, Content, Notification)
3. Comunicação via HTTP/REST para síncrono, RabbitMQ para assíncrono
4. Banco de dados independente por serviço (Database per Service)
5. Monorepo com Turborepo para gerenciamento

**OPÇÕES CONSIDERADAS:**
- **Opção A:** Monolito com módulos - Prós: Simples deploy, debugging fácil. Contras: Acoplamento alto, escala limitada.
- **Opção B:** Microsserviços completos - Prós: Escala independente, deploy independente. Contras: Complexidade operacional.
- **Opção C:** Arquitetura híbrida (esteira + serviços) - Prós: Balanceamento. Contras: Inconsistência.

**Escolhida: Opção B** - Justificativa: Alinha com objetivo de 100% autonomia da squad e Modo Hive.

**CONSEQUÊNCIAS:**
- Positivas: Deploy independente, escalabilidade granular, isolamento de falhas
- Negativas: Complexidade aumentada, necessidade de orquestração, latência de rede
- Riscos: Falhas em cascata, consistência eventual - Mitigação: Circuit breakers, sagas

```typescript
        meta: {
          timestamp: new Date().toISOString()
        }
      });
    } catch (error) {
      logger.error('Registration error:', error);
      next(error);
    }
  };

  login = async (req: Request, res: Response, next: NextFunction) => {
    try {
      const { email, password } = req.body;
      const result = await this.authService.login(email, password);
      
      res.json({
        success: true,
        data: result,
        error: null
      });
    } catch (error) {
      logger.error('Login error:', error);
      next(error);
    }
  };

  refreshToken = async (req: Request, res: Response, next: NextFunction) => {
    try {
      const { refreshToken } = req.body;
      const result = await this.authService.refreshToken(refreshToken);
      
      res.json({
        success: true,
        data: result,
        error: null
      });
    } catch (error) {
      logger.error('Token refresh error:', error);
      next(error);
    }
  };

  logout = async (req: Request, res: Response, next: NextFunction) => {
    try {
      const { refreshToken } = req.body;
      await this.authService.logout(refreshToken);
      
      res.json({
        success: true,
        data: { message: 'Logged out successfully' },
        error: null
      });
    } catch (error) {
      logger.error('Logout error:', error);
      next(error);
    }
  };
}
```

```javascript
// WCAG Contrast Checker Utility
export const contrastRatios = {
  getContrast: (foreground, background) => {
    // Simplified contrast calculation
    const lum1 = getLuminance(foreground);
    const lum2 = getLuminance(background);
    const brightest = Math.max(lum1, lum2);
    const darkest = Math.min(lum1, lum2);
    return (brightest + 0.05) / (darkest + 0.05);
  },
  
  isAACompliant: (foreground, background, size = 'normal') => {
    const ratio = contrastRatios.getContrast(foreground, background);
    if (size === 'large') return ratio >= 3.0; // WCAG AA for large text
    return ratio >= 4.5; // WCAG AA for normal text
  }
};

function getLuminance(hex) {
  const rgb = hexToRgb(hex);
  const [r, g, b] = Object.values(rgb).map(val => {
    val = val / 255;
    return val <= 0.03928 ? val / 12.92 : Math.pow((val + 0.055) / 1.055, 2.4);
  });
  return 0.2126 * r + 0.7152 * g + 0.0722 * b;
}

function hexToRgb(hex) {
  const result = /^#?([a-f\d]{2})([a-f\d]{2})([a-f\d]{2})$/i.exec(hex);
  return result ? {
    r: parseInt(result[1], 16),
    g: parseInt(result[2], 16),
    b: parseInt(result[3], 16)
  } : { r: 0, g: 0, b: 0 };
}
```

```javascript
import React from 'react';
import './Button.css';
import { colors, typography, spacing, shadows } from '../../design-tokens';

const Button = ({
  children,
  variant = 'primary',
  size = 'medium',
  disabled = false,
  loading = false,
  fullWidth = false,
  startIcon,
  endIcon,
  onClick,
  type = 'button',
  accessibilityLabel,
  ...props
}) => {
  const baseClasses = ['button', `button--${variant}`, `button--${size}`];
  
  if (disabled) baseClasses.push('button--disabled');
  if (loading) baseClasses.push('button--loading');
  if (fullWidth) baseClasses.push('button--full-width');
  
  const handleClick = (e) => {
    if (!disabled && !loading && onClick) {
      onClick(e);
    }
  };
  
  return (
    <button
      className={baseClasses.join(' ')}
      disabled={disabled || loading}
      onClick={handleClick}
      type={type}
      aria-label={accessibilityLabel || (typeof children === 'string' ? children : undefined)}
      aria-busy={loading}
      {...props}
    >
      {loading && (
        <span className="button__loader" aria-hidden="true">
          <svg width="16" height="16" viewBox="0 0 16 16" fill="none">
            <circle cx="8" cy="8" r="7" stroke="currentColor" strokeWidth="2" strokeOpacity="0.3" />
            <path d="M15 8a7 7 0 0 1-7 7" stroke="currentColor" strokeWidth="2" strokeLinecap="round" />
          </svg>
        </span>
      )}
      {startIcon && !loading && <span className="button__icon button__icon--start">{startIcon}</span>}
      <span className="button__content">{children}</span>
      {endIcon && !loading && <span className="button__icon button__icon--end">{endIcon}</span>}
    </button>
  );
};

export default Button;
```

```typescript
// src/store/agent-store.ts
import { create } from 'zustand'
import { persist } from 'zustand/middleware'

export interface Agent {
  id: string
  name: string
  role: string
  status: 'idle' | 'running' | 'error' | 'completed'
  cpuUsage: number
  memoryUsage: number
  uptime: number
  lastActivity: Date
  logs: LogEntry[]
  metrics: AgentMetric[]
}

export interface LogEntry {
  id: string
  timestamp: Date
  level: 'info' | 'warning' | 'error' | 'success'
  message: string
  source: string
}

export interface AgentMetric {
  timestamp: Date
  value: number
  metricType: 'cpu' | 'memory' | 'requests' | 'latency'
}

interface AgentStore {
  agents: Agent[]
  selectedAgentId: string | null
  isStreaming: boolean
  addAgent: (agent: Omit<Agent, 'id' | 'logs' | 'metrics'>) => void
  updateAgentStatus: (agentId: string, status: Agent['status']) => void
  updateAgentMetrics: (agentId: string, metrics: Partial<Agent>) => void
  addLogEntry: (agentId: string, log: Omit<LogEntry, 'id'>) => void
  selectAgent: (agentId: string | null) => void
  toggleStreaming: () => void
  clearLogs: (agentId: string) => void
}

export const useAgentStore = create<AgentStore>()(
  persist(
    (set, get) => ({
      agents: [],
      selectedAgentId: null,
      isStreaming: false,
      
      addAgent: (agentData) => {
        const newAgent: Agent = {
          ...agentData,
          id: `agent-${Date.now()}`,
          logs: [],
          metrics: [],
        }
        set((state) => ({
          agents: [...state.agents, newAgent],
        }))
      },
      
      updateAgentStatus: (agentId, status) => {
        set((state) => ({
          agents: state.agents.map((agent) =>
            agent.id === agentId ? { ...agent, status } : agent
          ),
        }))
      },
      
      updateAgentMetrics: (agentId, metrics) => {
        set((state) => ({
          agents: state.agents.map((agent) =>
            agent.id === agentId ? { ...agent, ...metrics } : agent
          ),
        }))
      },
      
      addLogEntry: (agentId, log) => {
        const newLog: LogEntry = {
          ...log,
          id: `log-${Date.now()}-${Math.random().toString(36).substr(2, 9)}`,
        }
        
        set((state) => ({
          agents: state.agents.map((agent) =>
            agent.id === agentId
              ? {
                  ...agent,
                  logs: [newLog, ...agent.logs].slice(0, 1000), // Keep last 1000 logs
                  lastActivity: new Date(),
                }
              : agent
          ),
        }))
      },
      
      selectAgent: (agentId) => {
        set({ selectedAgentId: agentId })
      },
      
      toggleStreaming: () => {
        set((state) => ({ isStreaming: !state.isStreaming }))
      },
      
      clearLogs: (agentId) => {
        set((state) => ({
          agents: state.agents.map((agent) =>
            agent.id === agentId ? { ...agent, logs: [] } : agent
          ),
        }))
      },
    }),
    {
      name: 'agent-storage',
      partialize: (state) => ({ agents: state.agents }),
    }
  )
)
```

```javascript
const EventEmitter = require('events');
const logger = require('../../utils/logger');
const metrics = require('../../utils/metrics');

class CircuitBreaker extends EventEmitter {
  constructor(options = {}) {
    super();
    
    this.options = {
      failureThreshold: options.failureThreshold || 5,
      resetTimeout: options.resetTimeout || 60000,
      halfOpenMaxAttempts: options.halfOpenMaxAttempts || 3,
      ...options
    };

    this.state = 'CLOSED';
    this.failureCount = 0;
    this.successCount = 0;
    this.nextAttempt = null;
    this.halfOpenAttempts = 0;
    
    this.provider = options.provider || 'unknown';
  }

  async execute(fn, ...args) {
    if (this.state === 'OPEN') {
      if (Date.now() > this.nextAttempt) {
        this.state = 'HALF_OPEN';
        this.halfOpenAttempts = 0;
        logger.warn(`Circuit breaker for ${this.provider} moved to HALF_OPEN`);
        metrics.recordCircuitBreakerState(this.provider, 'half-open');
      } else {
        throw new Error(`Circuit breaker OPEN for ${this.provider}`);
      }
    }

    try {
      const result = await fn(...args);
      this.onSuccess();
      return result;
    } catch (error) {
      this.onFailure(error);
      throw error;
    }
  }

  onSuccess() {
    this.failureCount = 0;
    this.halfOpenAttempts = 0;
    
    if (this.state === 'HALF_OPEN') {
      this.successCount++;
      
      if (this.successCount >= this.options.halfOpenMaxAttempts) {
        this.state = 'CLOSED';
        this.successCount = 0;
        logger.info(`Circuit breaker for ${this.provider} moved to CLOSED`);
        metrics.recordCircuitBreakerState(this.provider, 'closed');
        this.emit('closed', this.provider);
      }
    }
  }

  onFailure(error) {
    this.failureCount++;
    this.successCount = 0;
    
    metrics.recordCircuitBreakerFailure(this.provider);
    
    if (this.state === 'HALF_OPEN') {
      this.halfOpenAttempts++;
      
      if (this.halfOpenAttempts >= this.options.halfOpenMaxAttempts) {
        this.state = 'OPEN';
        this.nextAttempt = Date.now() + this.options.resetTimeout;
        logger.error(`Circuit breaker for ${this.provider} moved to OPEN`);
        metrics.recordCircuitBreakerState(this.provider, 'open');
        this.emit('open', this.provider, error);
      }
    } else if (this.failureCount >= this.options.failureThreshold) {
      this.state = 'OPEN';
      this.nextAttempt = Date.now() + this.options.resetTimeout;
      logger.error(`Circuit breaker for ${this.provider} moved to OPEN`);
      metrics.recordCircuitBreakerState(this.provider, 'open');
      this.emit('open', this.provider, error);
    }
  }

  getState() {
    return {
      state: this.state,
      failureCount: this.failureCount,
      successCount: this.successCount,
      nextAttempt: this.nextAttempt,
      halfOpenAttempts: this.halfOpenAttempts
    };
  }

  reset() {
    this.state = 'CLOSED';
    this.failureCount = 0;
    this.successCount = 0;
    this.nextAttempt = null;
    this.halfOpenAttempts = 0;
    logger.info(`Circuit breaker for ${this.provider} reset`);
    metrics.recordCircuitBreakerState(this.provider, 'closed');
  }
}

module.exports = CircuitBreaker;
```

class SemanticCache {
  constructor(options = {}) {
    this.options = {
      ttl: options.ttl || 3600000,
      maxSize: options.maxSize || 1000,
      similarityThreshold: options.similarityThreshold || 0.85,
      ...options
    };

    this.cache = new lru({
      max: this.options.maxSize,
      ttl: this.options.ttl,
      updateAgeOnGet: true
    });

    this.vectorCache = new Map();
    this.agentStats = new Map();
  }

  generateKey(prompt, context = {}) {
    const content = typeof prompt === 'string' ? prompt : JSON.stringify(prompt);
    const contextStr = JSON.stringify(context);
    
    return crypto
      .createHash('sha256')
      .update(content + contextStr)
      .digest('hex');
  }

  async get(prompt, context = {}, agent = 'unknown') {
    const key = this.generateKey(prompt, context);
    const cached = this.cache.get(key);

    if (cached) {
      logger.debug(`Cache hit for agent: ${agent}`);
      metrics.recordCacheHit(agent);
      this.updateStats(agent, true);
      return cached;
    }

    // Busca semântica
    const semanticMatch = await this.findSemanticMatch(prompt, context, agent);
    if (semanticMatch) {
      logger.debug(`Semantic cache hit for agent: ${agent}`);
      metrics.recordCacheHit(agent);
      this.updateStats(agent, true);
      return semanticMatch;
    }

    logger.debug(`Cache miss for agent: ${agent}`);
    metrics.recordCacheMiss(agent);
    this.updateStats(agent, false);
    return null;
  }

  async set(prompt, context = {}, value, agent = 'unknown') {
    const key = this.generateKey(prompt, context);
    this.cache.set(key, value);

    // Armazenar vetor para busca semântica
    if (this.options.similarityThreshold > 0) {
      const vector = await this.generateVector(prompt);
      this.vectorCache.set(key, {
        vector,
        value,
        timestamp: Date.now(),
        agent
      });
    }

    return key;
  }

  async generateVector(text) {
    // Implementação simplificada de vetorização
    // Em produção, usar modelo de embeddings
    const words = text.toLowerCase().split(/\W+/).filter(w => w.length > 2);
    const vector = {};
    
    words.forEach(word => {
      vector[word] = (vector[word] || 0) + 1;
    });

    return this.normalizeVector(vector);
  }

  normalizeVector(vector) {
    const values = Object.values(vector);
    const magnitude = Math.sqrt(values.reduce((sum, val) => sum + val * val, 0));
    
    if (magnitude === 0) return vector;
    
    const normalized = {};
    Object.keys(vector).forEach(key => {
      normalized[key] = vector[key] / magnitude;
    });
    
    return normalized;
  }

  cosineSimilarity(vecA, vecB) {
    const keys = new Set([...Object.keys(vecA), ...Object.keys(vecB)]);
    let dotProduct = 0;
    let magA = 0;
    let magB = 0;

    for (const key of keys) {
      const a = vecA[key] || 0;
      const b = vecB[key] || 0;
      dotProduct += a * b;
      magA += a * a;
      magB += b * b;
    }

    if (magA === 0 || magB === 0) return 0;
    return dotProduct / (Math.sqrt(magA) * Math.sqrt(magB));
  }

    if (this.options.similarityThreshold <= 0) return null;

      typeof prompt === 'string' ? prompt : JSON.stringify(prompt)

        bestSimilarity = similarity;
        bestMatch = entry.value;
