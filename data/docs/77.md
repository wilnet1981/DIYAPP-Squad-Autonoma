# DIYAPP Evolution - V12 Core - Arquitetura Técnica

## ADR-001: Arquitetura V12 - Microserviços vs Monólito Modular

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead, Especialista Infra, Especialista Backend

### CONTEXTO:
O DIYAPP V11 está enfrentando desafios de escalabilidade e manutenção. O monólito atual dificulta:
1. Deploy independente de funcionalidades
2. Escalonamento seletivo de componentes críticos
3. Isolamento de falhas
4. Adoção de diferentes stacks tecnológicas por domínio

Forças em jogo:
- Time reduzido (5 desenvolvedores)
- Necessidade de deploy contínuo
- Recursos limitados de infraestrutura
- Complexidade de gerenciamento distribuído

### DECISÃO:
Adotar arquitetura **Monólito Modular com Event-Driven Communication**, com caminho claro para microserviços futuros.

### OPÇÕES CONSIDERADAS:

**Opção A: Microserviços Puros**
- **Prós:** 
  - Escalabilidade granular
  - Isolamento completo de falhas
  - Deploy independente
  - Poliglotismo tecnológico
- **Contras:**
  - Complexidade operacional alta
  - Overhead de comunicação
  - Necessidade de equipe SRE dedicada
  - Debug distribuído complexo

**Opção B: Monólito Tradicional**
- **Prós:**
  - Simplicidade de deploy
  - Debug facilitado
  - Menor latência entre componentes
  - Menor custo operacional
- **Contras:**
  - Acoplamento alto
  - Escalabilidade limitada
  - Dificuldade de evolução tecnológica
  - Deploy all-or-nothing

**Opção C: Monólito Modular + Event-Driven (ESCOLHIDA)**
- **Prós:**
  - Baixo acoplamento entre módulos
  - Caminho evolutivo para microserviços
  - Simplicidade operacional mantida
  - Isolamento de falhas por módulo
  - Deploy seletivo possível
- **Contras:**
  - Necessidade de disciplina arquitetural
  - Eventual complexidade de eventos
  - Gerenciamento de estado distribuído limitado

**Justificativa:** Para uma squad de 5 devs com foco em autonomia, a Opção C oferece o melhor balanço entre escalabilidade futura e simplicidade operacional atual. Permite evolução gradual com baixo risco.

### CONSEQUÊNCIAS:
**Positivas:**
- Módulos podem evoluir para microserviços sem refatoração massiva
- Comunicação assíncrona reduz acoplamento
- Testabilidade melhorada por módulo
- Onboarding mais simples que microserviços puros

**Negativas:**
- Necessidade de padrões rígidos de comunicação
- Event bus torna-se single point of failure
- Gerenciamento de transações distribuídas limitado

**Riscos:**
1. Degradação para monólito acoplado se padrões não forem seguidos
   - Mitigação: Code reviews rigorosos e ADRs para cada novo módulo
2. Performance do event bus
   - Mitigação: Monitoramento proativo e escalabilidade vertical inicial

**REVISÃO:** 2024-04-15 (3 meses para reavaliar necessidade de migração para microserviços)

### 3. Padrões de Código

#### JavaScript/TypeScript
```typescript
// Nomenclatura
interface IUserService {}          // Interface prefixada
class UserRepository {}            // Classes PascalCase
const MAX_RETRIES = 3;            // Constantes UPPER_SNAKE
let isLoading = false;            // Variáveis camelCase

// Estrutura de arquivo
// 1. Imports (externos → internos)
// 2. Interfaces/Types
// 3. Constantes
// 4. Funções/Classes
// 5. Export

// Exemplo de módulo
export class UserService implements IUserService {
  private readonly userRepository: UserRepository;
  
  constructor(dependencies: Dependencies) {
    this.userRepository = dependencies.userRepository;
  }
  
  async createUser(userData: CreateUserDto): Promise<User> {
    // Validação
    const validated = await this.validateUserData(userData);
    
    // Lógica de negócio
    const user = await this.userRepository.create(validated);
    
    // Evento
    await this.eventBus.publish('user.created', user);
    
    return user;
  }
}
```

#### Comunicação entre Módulos
```typescript
// Event-Driven Pattern
interface IEvent {
  type: string;
  payload: any;
  timestamp: Date;
  correlationId: string;
}

// Producer
class TaskService {
  async completeTask(taskId: string) {
    const task = await this.taskRepository.complete(taskId);
    
    await this.eventBus.publish({
      type: 'task.completed',
      payload: task,
      timestamp: new Date(),
      correlationId: uuidv4()
    });
  }
}

// Consumer
class NotificationService {
  @Subscribe('task.completed')
  async handleTaskCompleted(event: IEvent) {
    const task = event.payload;
    await this.notifyUser(task.userId, `Task "${task.title}" completed`);
  }
}
```

### 6. Testes

```typescript
// Estrutura AAA (Arrange, Act, Assert)
describe('UserService', () => {
  describe('createUser', () => {
    it('should create user with valid data', async () => {
      // Arrange
      const userData = { email: 'test@example.com', name: 'Test' };
      const mockRepo = { create: jest.fn() };
      const service = new UserService({ userRepository: mockRepo });
      
      // Act
      const result = await service.createUser(userData);
      
      // Assert
      expect(mockRepo.create).toHaveBeenCalledWith(userData);
      expect(result).toHaveProperty('id');
    });
    
    it('should throw error for duplicate email', async () => {
      // Arrange
      const userData = { email: 'duplicate@example.com' };
      const mockRepo = { 
        create: jest.fn().mockRejectedValue(new Error('Duplicate')) 
      };
      const service = new UserService({ userRepository: mockRepo });
      
      // Act & Assert
      await expect(service.createUser(userData))
        .rejects.toThrow('Duplicate email');
    });
  });
});
```

export const verifyToken = (req, res, next) => {
  try {
    const authHeader = req.headers.authorization;
    
    if (!authHeader || !authHeader.startsWith('Bearer ')) {
      return res.status(401).json({
        error: 'Unauthorized',
        message: 'No token provided',
        correlation_id: req.correlation_id,
      });
    }
    
    const token = authHeader.substring(7);
    
    const decoded = jwt.verify(token, config.JWT_SECRET);
    
    // Check token type
    if (decoded.type !== 'access') {
      return res.status(401).json({
        error: 'Unauthorized',
        message: 'Invalid token type',
        correlation_id: req.correlation_id,
      });
    }
    
    req.user = {
      id: decoded.sub,
      role: decoded.role,
    };
    
    next();
  } catch (error) {
    logger.error('Token verification failed', {
      correlation_id: req.correlation_id,
      error: error.message,
      ip: req.ip,
    });
    
    if (error.name === 'TokenExpiredError') {
      return res.status(401).json({
        error: 'Unauthorized',
        message: 'Token expired',
        correlation_id: req.correlation_id,
      });
    }
    
    return res.status(401).json({
      error: 'Unauthorized',

    });
  }

-- Update updated_at timestamp
CREATE OR REPLACE FUNCTION update_updated_at_column()
RETURNS TRIGGER AS $$
BEGIN
    NEW.updated_at = CURRENT_TIMESTAMP;
    RETURN NEW;
END;
$$ language 'plpgsql';

-- Create triggers for tables with updated_at
DO $$ 
DECLARE 
    t text;
BEGIN
    FOR t IN 
        SELECT table_name 
        FROM information_schema.columns 
        WHERE column_name = 'updated_at' 
        AND table_schema = 'public'
    LOOP
        EXECUTE format('
            DROP TRIGGER IF EXISTS update_%s_updated_at ON %s;
            CREATE TRIGGER update_%s_updated_at
            BEFORE UPDATE ON %s
            FOR EACH ROW
            EXECUTE FUNCTION update_updated_at_column();
        ', t, t, t, t);
    END LOOP;
END;
$$;

-- Function to update project activity timestamp

RETURNS TRIGGER AS $$
BEGIN
    IF TG_TABLE_NAME = 'activities' AND NEW.project_id IS NOT NULL THEN
        UPDATE projects 
        SET last_activity_at = CURRENT_TIMESTAMP
        WHERE id = NEW.project_id;
    END IF;
    RETURN NEW;
END;
$$ language 'plpgsql';

-- Function to update project stats
CREATE OR REPLACE FUNCTION update_project_stats()
RETURNS TRIGGER AS $$
BEGIN
    IF TG_TABLE_NAME = 'project_stars' THEN
        IF TG_OP = 'INSERT' THEN
            UPDATE projects 
            SET star_count = star_count + 1
            WHERE id = NEW.project_id;
        ELSIF TG_OP = 'DELETE' THEN
            UPDATE projects 
            SET star_count = star_count - 1
            WHERE id = OLD.project_id;
        END IF;
    END IF;
    RETURN NULL;
END;
$$ language 'plpgsql';

CREATE TRIGGER update_project_stats_stars
AFTER INSERT OR DELETE ON project_stars
FOR EACH ROW
EXECUTE FUNCTION update_project_stats();

-- Function for audit logging
CREATE OR REPLACE FUNCTION audit_trigger_func()
RETURNS TRIGGER AS $$
BEGIN
    IF TG_OP = 'INSERT' THEN
        INSERT INTO audit_log (table_name, record_id, operation, new_data, changed_by)
        VALUES (TG_TABLE_NAME, NEW.id, 'INSERT', to_jsonb(NEW), current_setting('app.user_id', TRUE)::UUID);
        RETURN NEW;
    ELSIF TG_OP = 'UPDATE' THEN
        INSERT INTO audit_log (table_name, record_id, operation, old_data, new_data, changed_by)
        VALUES (TG_TABLE_NAME, NEW.id, 'UPDATE', to_jsonb(OLD), to_jsonb(NEW), current_setting('app.user_id', TRUE)::UUID);
        RETURN NEW;
    ELSIF TG_OP = 'DELETE' THEN
        INSERT INTO audit_log (table_name, record_id, operation, old_data, changed_by)
        VALUES (TG_TABLE_NAME, OLD.id, 'DELETE', to_jsonb(OLD), current_setting('app.user_id', TRUE)::UUID);
        RETURN OLD;
    END IF;
    RETURN NULL;
END;
$$ language 'plpgsql';

```javascript
const { execSync } = require('child_process');

class SecurityAuditor {
  constructor() {
    this.report = {
      timestamp: new Date().toISOString(),
      dependencies: {},
      vulnerabilities: {
        critical: [],
        high: [],
        moderate: [],
        low: []
      },
      recommendations: []

  async runNpmAudit() {
    try {
      console.log('🔍 Executando npm audit...');
      const result = execSync('npm audit --json', { encoding: 'utf8' });
      const auditData = JSON.parse(result);
      
      if (auditData.vulnerabilities) {
        Object.entries(auditData.vulnerabilities).forEach(([pkg, vuln]) => {
          const severity = vuln.severity?.toLowerCase();
          const entry = {
            package: pkg,
            version: vuln.via?.[0]?.name || vuln.name,
            vulnerability: vuln.via?.[0]?.title || 'Unknown',
            fix: vuln.fixAvailable || false,
            path: vuln.nodes || [],
            cvss: vuln.via?.[0]?.cvss?.score || 0
          };

          switch(severity) {
            case 'critical':
              this.report.vulnerabilities.critical.push(entry);
              break;
            case 'high':
              this.report.vulnerabilities.high.push(entry);
              break;
            case 'moderate':
              this.report.vulnerabilities.moderate.push(entry);
              break;
            case 'low':
              this.report.vulnerabilities.low.push(entry);
              break;
          }
        });
      }
    } catch (error) {
      console.error('❌ Erro ao executar npm audit:', error.message);
    }
  }

  async checkDependencies() {
    try {
      const packageJson = JSON.parse(fs.readFileSync('package.json', 'utf8'));
      
      this.report.dependencies = {
        total: Object.keys(packageJson.dependencies || {}).length + 
               Object.keys(packageJson.devDependencies || {}).length,
        production: packageJson.dependencies || {},
        development: packageJson.devDependencies || {}
      };

      // Verificar dependências abandonadas
      console.log('📦 Analisando dependências...');
      
      const abandonedPackages = [];
      const highRiskPackages = [
        'lodash.defaultsdeep', 'hoek', 'minimist', 
        'axios', 'moment', 'request', 'underscore'
      ];

      Object.keys(packageJson.dependencies).forEach(pkg => {
        if (highRiskPackages.includes(pkg)) {
          this.report.recommendations.push({
            type: 'HIGH_RISK_PACKAGE',
            package: pkg,
            message: `Pacote ${pkg} tem histórico de vulnerabilidades. Considere alternativas.`
          });
        }
      });
    } catch (error) {
      console.error('❌ Erro ao verificar dependências:', error);
    }

    await this.runNpmAudit();
    await this.checkDependencies();

    // SLAs baseados em CVSS
    const slaRecommendations = [];
    
    this.report.vulnerabilities.critical.forEach(vuln => {
      slaRecommendations.push({

        cvss: vuln.cvss,
        sla: '< 24h',
        action: 'CORREÇÃO IMEDIATA REQUERIDA'
      });
    });

    this.report.vulnerabilities.high.forEach(vuln => {
      slaRecommendations.push({
        package: vuln.package,
        cvss: vuln.cvss,
        sla: '< 72h',
        action: 'Prioridade máxima no próximo sprint'
      });
    });

    this.report.recommendations.push(...slaRecommendations);

    // Salvar relatório
    const reportPath = path.join(__dirname, '../security-audit-report.json');

    this.printSummary();

    console.log(`Moderadas: ${this.report.vulnerabilities.moderate.length}`);
```

```javascript
      },
```

```javascript
      database: {

      },

      }),

  sanitizeHtml(input) {

      /system prompt/i,
      /as a (?:helpful|developer|assistant)/i,
      /you are now/i,
      /forget (?:that|all)/i,

    sanitized = sanitized.replace(/[<>{}[\]]/g, '');

      /document\./i,
      /window\./i,
      /alert\(/i,
      /prompt\(/i,
      /confirm\(/i,
      /localStorage/i,
      /sessionStorage/i,
      /cookie/i,
      /fetch\(/i,
      /XMLHttpRequest/i
    ];

    return maliciousPatterns.some(pattern => pattern.test(input));

    try {
      const schema = this.getSchema(schemaName);
      const result = await schema.parseAsync(data);
      
      // Log de validação para auditoria
      if (options.logValidation) {

          timestamp: new Date().toISOString(),
          schema: schemaName,
          hasSensitiveData: this.containsSensitiveData(data)
        });
      }

      return { success: true, data: result };
    } catch (error) {

        timestamp: new Date().toISOString(),
        schema: schemaName,
        error: error.errors,
        input: this.maskSensitiveData(data)
      });

      return {
        success: false,
        errors: error.errors.map(err => ({
          path: err.path.join('.'),
          message: err.message,
          code: err.code
        }))
      };
    }
  }

  getSchema(schemaPath) {
    const parts = schemaPath.split('.');
    let schema = this.schemas;
    
    for (const part of parts) {

      if (!schema) throw new Error(`Schema não encontrado: ${schemaPath}`);

      if (masked[field]) {
        masked[field] = '***MASKED***';
      }
    });
```

```javascript
DataProcessingActivity.prototype.requiresDPIA = function() {
  const config = require('../config/compliance-config');
  
  // Critérios para DPIA obrigatória
  const criteria = [
    this.sensitiveData,
    this.dataCategories.some(cat => config.dataClassification.sensitive.includes(cat)),
    this.thirdCountries,
    this.processingName.toLowerCase().includes('monitoramento') ||
    this.processingName.toLowerCase().includes('vigilância'),
    this.purpose.toLowerCase().includes('avaliação') ||
    this.purpose.toLowerCase().includes('pontuação')
  ];
  
  return criteria.some(criterion => criterion === true);
};

    justification: this.retentionJustification,
    autoDelete: true, // Implementado no DataRetentionController
    legalBasis: this.legalBasis
  };
};
```

const cron = require('node-cron');
const { Op } = require('sequelize');
const config = require('../../config/compliance-config');
const logger = require('../../utils/logger');

class DataRetentionController {
  constructor(models) {
    this.models = models;
    this.scheduleJobs();
  }
  
  /**
   * Agenda tarefas de limpeza automática
   */
  scheduleJobs() {
    // Executa diariamente às 02:00
    cron.schedule('0 2 * * *', () => {
      this.executeRetentionCleanup();
    });
    
    // Executa semanalmente aos domingos para revisão
    cron.schedule('0 3 * * 0', () => {
      this.reviewRetentionPolicies();
    });
    
    logger.info('Data retention jobs scheduled');
  }
  
  /**
   * Executa limpeza de dados expirados
   */
  async executeRetentionCleanup() {
    const startTime = Date.now();
    let deletedCount = 0;
    
    try {
      logger.info('Starting data retention cleanup');
      
      // 1. Limpar logs de acesso expirados
      const accessLogsDeleted = await this.models.AccessLog.destroy({
        where: {
          createdAt: {
            [Op.lt]: new Date(Date.now() - config.dataRetention.accessLogs.retentionDays * 24 * 60 * 60 * 1000)
          }
        }
      });
      
      // 2. Limpar perfis de usuário inativos
      const userProfilesDeleted = await this.models.User.destroy({
        where: {
          deletedAt: {
            [Op.not]: null
          },
          deletedAt: {
            [Op.lt]: new Date(Date.now() - 30 * 24 * 60 * 60 * 1000) // 30 dias após deleção
          }
        }
      });
      
      // 3. Limpar dados anonimizados
      const anonymizedDataDeleted = await this.models.AnonymizedData.destroy({
        where: {
          anonymizedAt: {
            [Op.lt]: new Date(Date.now() - config.dataRetention.auditLogs.retentionDays * 24 * 60 * 60 * 1000)
          }
        }
      });
      
      deletedCount = accessLogsDeleted + userProfilesDeleted + anonymizedDataDeleted;
      
      // Registrar auditoria
      await this.models.AuditLog.create({
        action: 'DATA_RETENTION_CLEANUP',
        userId: 'system',
        resource: 'compliance',
        status: 'success',
        metadata: {
          deletedCount,
          executionTime: Date.now() - startTime,
          details: {
            accessLogs: accessLogsDeleted,
            userProfiles: userProfilesDeleted,
            anonymizedData: anonymizedDataDeleted
          }
        }
      });
      
      logger.info(`Data retention cleanup completed: ${deletedCount} records deleted`);
      
    } catch (error) {
      logger.error('Data retention cleanup failed:', error);
      
      await this.models.AuditLog.create({
        action: 'DATA_RETENTION_CLEANUP',
        userId: 'system',
        resource: 'compliance',
        status: 'error',
        metadata: {
          error: error.message,
          executionTime: Date.now() - startTime
        }
      });
    }
  }
  
  /**
   * Processa solicitação de exclusão de titular
   */
  async processDataSubjectDeletion(userId, requestId) {
    const startTime = Date.now();
    
    try {
      logger.info(`Processing data deletion for user: ${userId}`);
      
      // 1. Encontrar usuário
      const user = await this.models.User.findByPk(userId);
      if (!user) {
        throw new Error('User not found');
      }
      
      // 2. Verificar se há obrigação legal de retenção
      const legalHold = await this.checkLegalHold(userId);
      if (legalHold) {
        return {
          success: false,
          reason: 'legal_hold',
          details: legalHold
        };
      }
      
      // 3. Anonimizar dados pessoais
      await this.anonymizeUserData(userId);
      
      // 4. Marcar como deletado (soft delete)
      user.deletedAt = new Date();
      user.email = `deleted_${userId}@anonymized.com`;
      user.name = 'Usuário Deletado';
      user.phone = null;
      user.address = null;
      await user.save();
      
      // 5. Registrar auditoria
      await this.models.AuditLog.create({
        action: 'DATA_SUBJECT_DELETION',
        userId: 'system',
        resource: `user:${userId}`,
        status: 'success',
        metadata: {
          requestId,
          executionTime: Date.now() - startTime,
          method: 'anonymization'
        }
      });
      
      // 6. Atualizar solicitação do titular
      await this.models.DataSubjectRequest.update(
        { status: 'completed', completedAt: new Date() },
        { where: { id: requestId } }
      );
      
      logger.info(`Data deletion completed for user: ${userId}`);
      
      return {
        success: true,
        message: 'Data anonymized successfully',
        timestamp: new Date().toISOString()
      };
      
    } catch (error) {
      logger.error(`Data deletion failed for user ${userId}:`, error);
      
      await this.models.AuditLog.create({
        action: 'DATA_SUBJECT_DELETION',
        userId: 'system',
        resource: `user:${userId}`,
        status: 'error',

```javascript
// src/services/feature-flags/flagsmith-service.js
const axios = require('axios');
const NodeCache = require('node-cache');

class FlagsmithService {
  constructor() {
    this.cache = new NodeCache({ stdTTL: 30, checkperiod: 60 });
    this.baseURL = process.env.FLAGSMITH_API_URL || 'http://localhost:8000/api/v1';
    this.environmentKey = process.env.FLAGSMITH_ENV_KEY;
    this.apiClient = axios.create({
      baseURL: this.baseURL,
      headers: {
        'Authorization': `Token ${this.environmentKey}`,
        'Content-Type': 'application/json'
      }
    });
  }

  async getFeatureFlags(identity = null) {
    const cacheKey = identity ? `flags_${identity}` : 'flags_global';
    const cached = this.cache.get(cacheKey);
    
    if (cached) {
      return cached;
    }

    try {
      const endpoint = identity 
        ? `/identities/?identifier=${encodeURIComponent(identity)}`
        : '/flags/';
      
      const response = await this.apiClient.get(endpoint);
      const flags = this._parseFlags(response.data);
      
      this.cache.set(cacheKey, flags);
      return flags;
    } catch (error) {
      console.error('Error fetching feature flags:', error.message);
      return this._getDefaultFlags();
    }
  }

  async toggleFeature(featureName, enabled, segment = null) {
    try {
      const payload = {
        feature: { name: featureName },
        enabled,
        segment: segment ? { id: segment } : null
      };

      await this.apiClient.post('/features/toggle/', payload);
      this.cache.flushAll(); // Invalidate cache
      return true;
    } catch (error) {
      console.error('Error toggling feature:', error.message);
      return false;
    }
  }

  async createRollout(featureName, percentage, segment = null) {
    try {
      const payload = {
        feature_name: featureName,
        rollout_percentage: percentage,
        segment: segment
      };

      await this.apiClient.post('/rollouts/', payload);
      return true;
    } catch (error) {
      console.error('Error creating rollout:', error.message);
      return false;
    }
  }

  _parseFlags(data) {
    if (data.flags) {
      return data.flags.reduce((acc, flag) => {
        acc[flag.feature.name] = flag.enabled;
        return acc;
      }, {});
    }
    return {};
  }

  _getDefaultFlags() {
    return {
      'new_dashboard': false,
      'ai_suggestions': false,
      'dark_mode': true,
      'export_pdf': true,
      'real_time_updates': false
    };
  }
}

module.exports = new FlagsmithService();
```

```javascript
// public/js/feature-flags-client.js
class FeatureFlagsClient {
  constructor(options = {}) {
    this.options = {
      apiUrl: options.apiUrl || '/api/feature-flags',
      cacheTTL: options.cacheTTL || 30000, // 30 seconds
      identity: options.identity || null,
      onFlagsLoaded: options.onFlagsLoaded || null,
      onFlagsChanged: options.onFlagsChanged || null
    };
    
    this.flags = {};
    this.cacheTimestamp = 0;
    this.identity = this.options.identity || this._getUserId();
    this.pollingInterval = null;
  }

  async init() {
    await this.loadFlags();
    
    // Start polling for flag changes
    if (this.options.pollInterval) {
      this.startPolling(this.options.pollInterval);
    }
    
    // Listen for flag change events from server-sent events
    this._setupSSE();
    
    return this.flags;
  }

  async loadFlags() {
    try {
      const response = await fetch(
        `${this.options.apiUrl}?identity=${encodeURIComponent(this.identity)}&t=${Date.now()}`
      );
      
      if (!response.ok) throw new Error('Failed to load flags');
      
      const newFlags = await response.json();
      const flagsChanged = this._compareFlags(this.flags, newFlags);
      
      this.flags = newFlags;
      this.cacheTimestamp = Date.now();
      
      if (this.options.onFlagsLoaded) {
        this.options.onFlagsLoaded(this.flags);
      }
      
      if (flagsChanged && this.options.onFlagsChanged) {
        this.options.onFlagsChanged(this.flags, this._getChangedFlags(this.flags, newFlags));
      }
      
      return this.flags;
    } catch (error) {
      console.warn('Using default feature flags:', error.message);
      return this._getDefaultFlags();
    }
  }

  isEnabled(featureName, defaultValue = false) {
    return this.flags[featureName] !== undefined 
      ? this.flags[featureName] 
      : defaultValue;
  }

  getVariant(featureName, defaultVariant = 'control') {
    const variantFlag = this.flags[`${featureName}_variant`];
    return variantFlag || defaultVariant;
  }

  startPolling(interval = 30000) {
    this.stopPolling();
    this.pollingInterval = setInterval(() => {
      this.loadFlags();
    }, interval);
  }

  stopPolling() {
    if (this.pollingInterval) {
      clearInterval(this.pollingInterval);
      this.pollingInterval = null;
    }
  }

  setIdentity(identity) {
    this.identity = identity;
    this.cacheTimestamp = 0;
    return this.loadFlags();
  }

  _getUserId() {
    // Try to get user ID from various sources
    if (window.currentUser && window.currentUser.id) {
      return `user_${window.currentUser.id}`;
    }
    
    const sessionId = sessionStorage.getItem('session_id') || 
                     localStorage.getItem('anonymous_id') ||
                     `anon_${Math.random().toString(36).substr(2, 9)}`;
    
    if (!localStorage.getItem('anonymous_id')) {
      localStorage.setItem('anonymous_id', sessionId);
    }
    
    return sessionId;
  }

  _setupSSE() {
    if (typeof EventSource !== 'undefined') {
      const eventSource = new EventSource(`${this.options.apiUrl}/stream`);
      
      eventSource.addEventListener('flags-updated', (event) => {
        const data = JSON.parse(event.data);
        if (data.identity === this.identity || !data.identity) {
          this.loadFlags();
        }
      });
      
      eventSource.onerror = () => {
        eventSource.close();
        // Fallback to polling
        this.startPolling(60000);
      };
    }
  }

  _compareFlags(oldFlags, newFlags) {
    const oldKeys = Object.keys(oldFlags);
    const newKeys = Object.keys(newFlags);
    
    if (oldKeys.length !== newKeys.length) return true;
    
    for (const key of oldKeys) {
      if (oldFlags[key] !== newFlags[key]) return true;
    }
    
    return false;
  }

  _getChangedFlags(oldFlags, newFlags) {
    const changed = {};
    
    for (const key in newFlags) {
      if (oldFlags[key] !== newFlags[key]) {
        changed[key] = {
          old: oldFlags[key],
          new: newFlags[key]
        };
      }
    }
    
    return changed;
  }

  _getDefaultFlags() {
    return {
      'new_dashboard': false,
      'ai_suggestions': false,
      'dark_mode': true
    };
  }
}

// Export for global use
window.FeatureFlags = FeatureFlagsClient;
```

```javascript
const flagsmithService = require('../services/feature-flags/flagsmith-service');

async function featureFlagsMiddleware(req, res, next) {
  try {
    // Get user identity from session, JWT, or headers
    const identity = req.user?.id || 
                    req.headers['x-user-id'] || 
                    req.session?.userId ||
                    `ip_${req.ip.replace(/[^a-zA-Z0-9]/g, '_')}`;
    
    // Get feature flags for this identity
    const flags = await flagsmithService.getFeatureFlags(identity);
    
    // Attach to request object
    req.featureFlags = flags;
    req.featureFlagsIdentity = identity;
    
    // Also attach to res.locals for templates
    res.locals.featureFlags = flags;
    
    // Add helper methods
    res.locals.isFeatureEnabled = (featureName) => {
      return flags[featureName] || false;
    };
    
    next();
  } catch (error) {
    console.error('Feature flags middleware error:', error);
    // Continue with default flags
    req.featureFlags = flagsmithService._getDefaultFlags();
    res.locals.featureFlags = req.featureFlags;
    next();
  }
}
```
