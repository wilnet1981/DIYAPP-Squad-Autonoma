# ==================== ADR-001: ARQUITETURA DE MICROSSERVIÇOS V11 ====================
"""
Architecture Decision Record — ADR-001: Arquitetura de Microsserviços V11
Data: 2024-01-15
Status: Aceita
Autores: Tech Lead + Especialista Infra + Especialista LLM

CONTEXTO:
A V10 do DIYAPP está monolítica, dificultando deploy independente, escalabilidade
seletiva e evolução tecnológica por componente. A V11 precisa suportar:
1. Escalonamento independente do serviço de IA (alto consumo CPU/GPU)
2. Deploy contínuo sem downtime total
3. Observabilidade granular por serviço
4. Comunicação assíncrona para operações de longa duração

DECISÃO:
Adotar arquitetura de microsserviços com os seguintes princípios:
- Cada serviço é responsável por um domínio de negócio específico
- Comunicação síncrona via gRPC para operações críticas de baixa latência
- Comunicação assíncrona via Apache Kafka para eventos de domínio
- API Gateway único como ponto de entrada
- Banco de dados por serviço (exceto para cache compartilhado Redis)
- Observabilidade unificada com OpenTelemetry

OPÇÕES CONSIDERADAS:
- Opção A: Monólito modularizado — Prós: Simplicidade operacional, transações ACID fáceis
  Contras: Acoplamento técnico, dificuldade de escalar componentes específicos
- Opção B: Microsserviços completos — Prós: Independência de deploy, escalabilidade granular
  Contras: Complexidade operacional, necessidade de SRE dedicado
- Opção C: Arquitetura híbrida (microsserviços + módulos) — Prós: Flexibilidade
  Contras: Inconsistência arquitetural, curva de aprendizado maior

Opção escolhida: B — Justificativa: A necessidade de escalar independentemente
o serviço de IA (que consome GPU) e o serviço de notificações (que depende de
APIs externas com rate limiting) justifica a complexidade adicional.

CONSEQUÊNCIAS:
Positivas:
- Deploy independente por serviço
- Escalabilidade granular (K8s HPA por serviço)
- Evolução tecnológica independente
- Isolamento de falhas

Negativas:
- Transações distribuídas complexas
- Necessidade de monitoramento sofisticado
- Overhead de rede
- Testes end-to-end mais complexos

Riscos:
- Latência aumentada em chamadas entre serviços
- Complexidade de debug em produção
- Gestão de schema de eventos

REVISÃO: 2024-04-15 (3 meses após implementação)
"""
# ==================== FIM ADR-001 ====================

# ==================== ADR-002: PADRÕES DE COMUNICAÇÃO ====================
"""
Architecture Decision Record — ADR-002: Padrões de Comunicação entre Serviços
Data: 2024-01-15
Status: Aceita
Autores: Tech Lead + Especialista Infra

CONTEXTO:
Microsserviços precisam se comunicar de forma eficiente, confiável e observável.
Necessidades identificadas:
1. Chamadas síncronas para operações que precisam de resposta imediata
2. Eventos assíncronos para desacoplamento temporal
3. Streaming para dados em tempo real
4. Garantia de entrega para operações críticas

DECISÃO:
Implementar três padrões de comunicação:
1. gRPC com Protobuf para chamadas síncronas (baixa latência, tipagem forte)
2. Apache Kafka para eventos de domínio (garantia de entrega, replay)
3. WebSocket (via Socket.io) para notificações em tempo real para UI

Protocolo por caso de uso:
- Autenticação/login: gRPC (síncrono)
- Criação de projeto: gRPC + evento "ProjectCreated" no Kafka
- Processamento de IA: Evento "AIProcessingRequested" → gRPC callback
- Notificações push: WebSocket

OPÇÕES CONSIDERADAS:
- Opção A: REST/JSON para tudo — Prós: Simplicidade, debugging fácil
  Contras: Overhead alto, sem contrato forte, streaming limitado
- Opção B: gRPC para tudo — Prós: Performance, contrato forte
  Contras: Complexidade para clientes web, falta de padrão para eventos
- Opção C: Mix gRPC + RabbitMQ — Prós: Mensageria madura
  Contras: Menos escalável que Kafka para volumes altos

Opção escolhida: Mix gRPC + Kafka + WebSocket — Justificativa: Cobre todos os
padrões necessários com ferramentas especializadas para cada caso.

CONSEQUÊNCIAS:
Positivas:
- Performance otimizada para cada caso de uso
- Contratos fortes com Protobuf
- Capacidade de replay de eventos
- Suporte a clientes web via WebSocket

Negativas:
- Stack tecnológica mais complexa
- Necessidade de schema registry para Protobuf
- Learning curve para times

Riscos:
- Falhas em cascata se Kafka falhar
- Versionamento de schemas Protobuf

REVISÃO: 2024-03-15
"""

import { NodeSDK } from '@opentelemetry/sdk-node';
import { ConsoleSpanExporter } from '@opentelemetry/sdk-trace-node';
import { getNodeAutoInstrumentations } from '@opentelemetry/auto-instrumentations-node';
import { OTLPTraceExporter } from '@opentelemetry/exporter-trace-otlp-grpc';
import { OTLPMetricExporter } from '@opentelemetry/exporter-metrics-otlp-grpc';
import { PeriodicExportingMetricReader } from '@opentelemetry/sdk-metrics';
import { Resource } from '@opentelemetry/resources';
import { SemanticResourceAttributes } from '@opentelemetry/semantic-conventions';
import { WinstonInstrumentation } from '@opentelemetry/instrumentation-winston';
import { KafkaJsInstrumentation } from 'opentelemetry-instrumentation-kafkajs';
import { GrpcInstrumentation } from '@opentelemetry/instrumentation-grpc';

export class TelemetrySDK {
  private sdk: NodeSDK;
  
  constructor(serviceName: string, environment: string = 'production') {
    const resource = new Resource({
      [SemanticResourceAttributes.SERVICE_NAME]: serviceName,
      [SemanticResourceAttributes.DEPLOYMENT_ENVIRONMENT]: environment,
      'version': process.env.APP_VERSION || '1.0.0',
    });
    
    // Exportador para Jaeger/Grafana Tempo
    const traceExporter = process.env.NODE_ENV === 'development' 
      ? new ConsoleSpanExporter()
      : new OTLPTraceExporter({
          url: process.env.OTLP_ENDPOINT || 'http://jaeger:4317',
        });
    
    // Exportador de métricas para Prometheus
    const metricExporter = new OTLPMetricExporter({
      url: process.env.OTLP_METRICS_ENDPOINT || 'http://prometheus:4317',
    });
    
    this.sdk = new NodeSDK({
      resource,
      traceExporter,
      metricReader: new PeriodicExportingMetricReader({
        exporter: metricExporter,
        exportIntervalMillis: 60000, // 1 minuto
      }),
      instrumentations: [
        getNodeAutoInstrumentations({
          '@opentelemetry/instrumentation-http': {
            ignoreIncomingPaths: ['/health', '/metrics'],
          },
        }),
        new GrpcInstrumentation(),
        new WinstonInstrumentation(),
        new KafkaJsInstrumentation(),
      ],
    });
  }
  
  async start(): Promise<void> {
    await this.sdk.start();
    console.log(`OpenTelemetry iniciado para serviço`);
  }
  
  async shutdown(): Promise<void> {
    await this.sdk.shutdown();
  }
}

import { metrics, Counter, Histogram, ObservableGauge } from '@opentelemetry/api';

export class ServiceMetrics {
  private meter = metrics.getMeter('diyapp-service');
  
  // Contadores
  public readonly httpRequests: Counter;
  public readonly grpcRequests: Counter;
  public readonly kafkaMessages: Counter;
  public readonly errors: Counter;
  
  // Histogramas (latência)
  public readonly httpLatency: Histogram;
  public readonly grpcLatency: Histogram;
  public readonly dbQueryLatency: Histogram;
  
  // Gauges
  public readonly activeConnections: ObservableGauge;
  public readonly memoryUsage: ObservableGauge;
  public readonly queueSize: ObservableGauge;
  
  private connections = 0;
  private queueItems = 0;
  
  constructor(serviceName: string) {
    this.httpRequests = this.meter.createCounter('http_requests_total', {
      description: 'Total de requisições HTTP',
      unit: '1',
    });
    
    this.grpcRequests = this.meter.createCounter('grpc_requests_total', {
      description: 'Total de requisições gRPC',
      unit: '1',
    });
    
    this.kafkaMessages = this.meter.createCounter('kafka_messages_total', {
      description: 'Total de mensagens Kafka processadas',
      unit: '1',
    });
    
    this.errors = this.meter.createCounter('errors_total', {
      description: 'Total de erros',
      unit: '1',
    });
    
    this.httpLatency = this.meter.createHistogram('http_request_duration_seconds', {
      description: 'Duração das requisições HTTP',
      unit: 's',
    });
    
    this.grpcLatency = this.meter.createHistogram('grpc_request_duration_seconds', {
      description: 'Duração das requisições gRPC',
      unit: 's',
    });
    
    this.dbQueryLatency = this.meter.createHistogram('db_query_duration_seconds', {
      description: 'Duração das queries de banco',
      unit: 's',
    });
    
    this.activeConnections = this.meter.createObservableGauge('active_connections', {
      description: 'Conexões ativas',
      unit: '1',
    });
    
    this.memoryUsage = this.meter.createObservableGauge('memory_usage_bytes', {
      description: 'Uso de memória',
      unit: 'bytes',
    });
    
    this.queueSize = this.meter.createObservableGauge('queue_size', {
      description: 'Tamanho da fila interna',
      unit: '1',
    });
    
    // Configurar callbacks para gauges
    this.activeConnections.addCallback((observableResult) => {
      observableResult.observe(this.connections, { service: serviceName });
    });
    
    this.memoryUsage.addCallback((observableResult) => {
      const memory = process.memoryUsage();
      observableResult.observe(memory.heapUsed, { type: 'heap_used', service: serviceName });
      observableResult.observe(memory.heapTotal, { type: 'heap_total', service: serviceName });
      observableResult.observe(memory.rss, { type: 'rss', service: serviceName });
    });
    
    this.queueSize.addCallback((observableResult) => {
      observableResult.observe(this.queueItems, { service: serviceName });
    });
  }
  
  incrementConnections(): void {
    this.connections++;
  }
  
  decrementConnections(): void {
    this.connections--;
  }
  
  setQueueSize(size: number): void {
    this.queueItems = size;
  }
}

import { Injectable, LoggerService as NestLoggerService } from '@nestjs/common';
import * as winston from 'winston';
import { ConfigService } from '@nestjs/config';

@Injectable()
export class LoggerService implements NestLoggerService {
  private logger: winston.Logger;

  constructor(private configService: ConfigService) {
    const isProduction = this.configService.get('NODE_ENV') === 'production';

    this.logger = winston.createLogger({
      level: isProduction ? 'info' : 'debug',
      format: winston.format.combine(
        winston.format.timestamp(),
        winston.format.errors({ stack: true }),
        winston.format.json(),
      ),
      defaultMeta: { service: 'diyapp-v11-core' },
      transports: [
        new winston.transports.Console({
          format: winston.format.combine(
            winston.format.colorize(),
            winston.format.printf(({ timestamp, level, message, correlationId, userId, ...meta }) => {
              return `${timestamp} [${correlationId || 'no-correlation-id'}] ${level}: ${message} ${
                userId ? `user:${userId}` : ''
              } ${Object.keys(meta).length ? JSON.stringify(meta) : ''}`;
            }),
          ),
        }),
        new winston.transports.File({
          filename: 'logs/error.log',
          level: 'error',
          maxsize: 10485760, // 10MB
          maxFiles: 10,
        }),
        new winston.transports.File({
          filename: 'logs/combined.log',
          maxsize: 10485760,
          maxFiles: 10,
        }),
      ],
    });
  }

  log(message: string, context?: string, meta?: Record<string, any>) {
    this.logger.info(message, { context, ...meta });
  }

  error(message: string, trace?: string, context?: string, meta?: Record<string, any>) {
    this.logger.error(message, { trace, context, ...meta });
  }

  warn(message: string, context?: string, meta?: Record<string, any>) {
    this.logger.warn(message, { context, ...meta });
  }

  debug(message: string, context?: string, meta?: Record<string, any>) {
    this.logger.debug(message, { context, ...meta });
  }

  verbose(message: string, context?: string, meta?: Record<string, any>) {
    this.logger.verbose(message, { context, ...meta });
  }

  withCorrelation(correlationId: string, userId?: string) {
    return {
      log: (message: string, context?: string, meta?: Record<string, any>) => 
        this.log(message, context, { correlationId, userId, ...meta }),
      error: (message: string, trace?: string, context?: string, meta?: Record<string, any>) => 
        this.error(message, trace, context, { correlationId, userId, ...meta }),
      warn: (message: string, context?: string, meta?: Record<string, any>) => 
        this.warn(message, context, { correlationId, userId, ...meta }),
      debug: (message: string, context?: string, meta?: Record<string, any>) => 
        this.debug(message, context, { correlationId, userId, ...meta }),
      verbose: (message: string, context?: string, meta?: Record<string, any>) => 
        this.verbose(message, context, { correlationId, userId, ...meta }),
    };
  }
}

"""
Producer de eventos para Kafka/Redpanda
"""
import json
import logging
from datetime import datetime
from typing import Dict, Any
from confluent_kafka import Producer
from confluent_kafka.schema_registry import SchemaRegistryClient
from confluent_kafka.schema_registry.avro import AvroSerializer
from confluent_kafka.serialization import SerializationContext, MessageField

from .event_schema import USER_EVENT_SCHEMA, UserEvent

logger = logging.getLogger(__name__)

class EventProducer:
    """Producer de eventos comportamentais"""
    
    def __init__(self, bootstrap_servers: str = 'localhost:9092',
                 schema_registry_url: str = 'http://localhost:8081'):
        self.bootstrap_servers = bootstrap_servers
        self.schema_registry_url = schema_registry_url
        
        # Configuração do producer
        self.producer_config = {
            'bootstrap.servers': bootstrap_servers,
            'client.id': 'diyapp-event-producer',
            'acks': 'all',  # Garantia de entrega
            'retries': 5,
            'compression.type': 'snappy',
            'linger.ms': 5,
            'batch.size': 16384
        }
        
        # Configuração do schema registry
        self.schema_registry_config = {
            'url': schema_registry_url
        }
        
        self._init_producer()
    
    def _init_producer(self):
        """Inicializa producer com serialização Avro"""
        try:
            schema_registry_client = SchemaRegistryClient(self.schema_registry_config)
            
            # Serializador Avro
            self.avro_serializer = AvroSerializer(
                schema_registry_client,
                json.dumps(USER_EVENT_SCHEMA),
                lambda obj, ctx: obj.to_dict()
            )
            
            # Producer
            self.producer = Producer(self.producer_config)
            logger.info("Producer inicializado com sucesso")
            
        except Exception as e:
            logger.error(f"Erro ao inicializar producer: {e}")
            raise
    
    def produce_event(self, topic: str, event: UserEvent):
        """Produz um evento para o tópico especificado"""
        try:
            # Serializa o evento
            serialized_event = self.avro_serializer(
                event,
                SerializationContext(topic, MessageField.VALUE)
            )
            
            # Produz a mensagem
            self.producer.produce(
                topic=topic,
                key=event.user_id.encode('utf-8'),
                value=serialized_event,
                callback=self._delivery_callback
            )
            
            # Poll para trigger de callbacks
            self.producer.poll(0)
            
            logger.debug(f"Evento {event.event_id} enviado para tópico {topic}")
            
        except Exception as e:
            logger.error(f"Erro ao produzir evento {event.event_id}: {e}")
            raise
    
    def produce_batch(self, topic: str, events: list[UserEvent]):
        """Produz um batch de eventos"""
        for event in events:
            self.produce_event(topic, event)
        
        # Flush para garantir entrega
        self.flush()
    
    def flush(self):
        """Garante que todas as mensagens foram entregues"""
        self.producer.flush()
    
    def _delivery_callback(self, err, msg):
        """Callback de entrega de mensagens"""
        if err:
            logger.error(f"Falha na entrega da mensagem: {err}")
        else:
            logger.debug(f"Mensagem entregue para {msg.topic()} [{msg.partition()}]")
    
    def close(self):
        """Fecha o producer"""
        self.producer.flush()
        logger.info("Producer fechado")

class CollaborativeFilteringRecommender(BaseRecommender):
    """Collaborative filtering based on user-item interactions"""
    
    def __init__(self, model_id: int, name: str):
        super().__init__(model_id, name, "custom")
        self.user_item_matrix = None
        self.item_similarity = None
        self.user_ids = None
        self.item_ids = None
        
    def fit(self, interactions: pd.DataFrame):
        """Train the model on user-item interactions"""
        # Create user-item matrix
        self.user_ids = interactions['user_id'].unique()
        self.item_ids = interactions['item_id'].unique()
        
        self.user_item_matrix = pd.DataFrame(
            0, index=self.user_ids, columns=self.item_ids
        )
        
        for _, row in interactions.iterrows():
            self.user_item_matrix.loc[row['user_id'], row['item_id']] = row['rating']
            
        # Calculate item-item similarity
        self.item_similarity = cosine_similarity(self.user_item_matrix.T)
        self.item_similarity_df = pd.DataFrame(
            self.item_similarity,
            index=self.item_ids,
            columns=self.item_ids
        )
        
    def predict(self, user_id: str, context: Dict[str, Any], k: int = 10) -> List[Dict[str, Any]]:
        """Generate recommendations using collaborative filtering"""
        if user_id not in self.user_item_matrix.index:
            # Cold start: return popular items
            return self._get_popular_items(k)
            
        user_ratings = self.user_item_matrix.loc[user_id]
        rated_items = user_ratings[user_ratings > 0].index
        
        if len(rated_items) == 0:
            return self._get_popular_items(k)
            
        # Calculate scores for all items
        scores = {}
        for item in self.item_ids:
            if item in rated_items:
                continue
                
            # Weighted average of similarities to rated items
            similarities = []
            weights = []
            
            for rated_item in rated_items:
                sim = self.item_similarity_df.loc[item, rated_item]
                if sim > 0:
                    similarities.append(sim)
                    weights.append(user_ratings[rated_item])
                    
            if similarities:
                scores[item] = np.average(similarities, weights=weights)
            else:
                scores[item] = 0
                
        # Get top-k items
        top_items = sorted(scores.items(), key=lambda x: x[1], reverse=True)[:k]
        
        return [
            {
                "item_id": item_id,
                "score": float(score),
                "model": self.name,
                "model_id": self.model_id,
                "provider": self.provider
            }
            for item_id, score in top_items
        ]
        
    def _get_popular_items(self, k: int) -> List[Dict[str, Any]]:
        """Get most popular items (fallback for cold start)"""
        item_popularity = self.user_item_matrix.sum(axis=0)
        top_items = item_popularity.sort_values(ascending=False).head(k)
        
        return [
            {
                "item_id": item_id,
                "score": float(popularity),
                "model": self.name,
                "model_id": self.model_id,
                "provider": self.provider,
                "is_fallback": True
            }
            for item_id, popularity in top_items.items()
        ]

class HybridRecommender(BaseRecommender):
    """Hybrid recommender combining multiple strategies"""
    
    def __init__(self, model_id: int, name: str, providers: List[BaseRecommender], weights: List[float]):
        super().__init__(model_id, name, "hybrid")
        self.providers = providers
        self.weights = weights
        assert len(providers) == len(weights)
        assert abs(sum(weights) - 1.0) < 0.001
        
    def predict(self, user_id: str, context: Dict[str, Any], k: int = 10) -> List[Dict[str, Any]]:
        """Generate recommendations using weighted ensemble"""
        all_recommendations = []

            try:

     */
    async checkAuthorization(authContext, resource, action) {
        const input = {
            auth: {
                authenticated: authContext.authenticated,
                user_id: authContext.userId,
                roles: authContext.roles || [],
                attributes: authContext.attributes || {},
                mfa_verified: authContext.mfaVerified || false,
                ip: authContext.ip
            },
            resource: resource,
            action: action,
            timestamp: new Date().toISOString()
        };

        try {
            const response = await this.client.post('/v1/data/diyapp/authz/allow', {
                input
            });

            const result = response.data.result;
            
            // Log de auditoria
            await logger.securityAudit({
                userId: authContext.userId,
                action: 'authorization_check',
                resource: resource,
                actionType: action,
                allowed: result,
                timestamp: new Date(),
                metadata: {
                    roles: authContext.roles,
                    ip: authContext.ip
                }

            logger.error('OPA authorization check failed:', {
                error: error.message,
                resource,
                action,
                userId: authContext.userId
            });
            
            // Fail closed - nega acesso em caso de erro
            return false;
        }
    }

     * Verifica políticas de recurso específico
     */
    async checkResourcePolicy(resource, action, conditions) {
        const input = {
            resource: resource,
            action: action,
            conditions: conditions
        };

            const response = await this.client.post('/v1/data/diyapp/resource/allow_resource', {
                input
            });

        } catch (error) {
            logger.error('OPA resource policy check failed:', error);
            return false;

 * Middleware de autorização fine-grained
 */
function authorize(resourcePattern, action) {
    return async (req, res, next) => {
        try {
            // Extrai contexto de autenticação
            const authContext = {
                authenticated: req.user !== undefined,
                userId: req.user?.id,
                roles: req.user?.roles || [],
                attributes: req.user?.attributes || {},
                mfaVerified: req.user?.mfaVerified || false,
                ip: req.ip || req.connection.remoteAddress
            };

            // Resolve padrão de recurso (ex: user:${userId}:profile)
            const resource = resolveResourcePattern(resourcePattern, req);

            // Verifica autorização com OPA
            const allowed = await opaClient.checkAuthorization(authContext, resource, action);

            if (!allowed) {
                logger.warn('Authorization denied', {
                    userId: authContext.userId,
                    resource,
                    action,
                    ip: authContext.ip
                });

                return res.status(403).json({
                    error: 'Forbidden',
                    message: 'You do not have permission to perform this action',
                    requestId: req.id
                });
            }

            // Adiciona contexto de autorização à request
            req.authorization = {
                resource,
                action,
                grantedAt: new Date()
            };

            next();
        } catch (error) {
            logger.error('Authorization middleware error:', error);
            res.status(500).json({
                error: 'Internal Server Error',
                message: 'Authorization check failed'
            });
        }
    };
}

/**
 * Resolve padrões de recurso dinâmicos
 */
function resolveResourcePattern(pattern, req) {
    if (typeof pattern === 'function') {
        return pattern(req);
    }

    // Substitui placeholders dinâmicos
    return pattern
        .replace('${userId}', req.user?.id || 'anonymous')
        .replace('${resourceId}', req.params.id || '*')
        .replace('${tenantId}', req.user?.tenantId || 'default');
}

/**
 * Middleware para validação de atributos (ABAC)
 */
function attributeBasedAuth(attributeRules) {
    return async (req, res, next) => {
        try {
            const userAttributes = req.user?.attributes || {};
            const resourceAttributes = req.resource?.attributes || {};

            const input = {
                auth: {
                    attributes: userAttributes
                },
                resource: {
                    attributes: resourceAttributes
                }
            };

            // Verifica regras com OPA
            const response = await opaClient.client.post('/v1/data/diyapp/rbac/allow_abac', {
                input
            });

            if (!response.data.result) {
                return res.status(403).json({
                    error: 'Forbidden',
                    message: 'Attribute-based authorization failed'
                });
            }

            next();
        } catch (error) {
            logger.error('ABAC middleware error:', error);
            res.status(500).json({
                error: 'Internal Server Error'
            });
        }
    };
}

  const pactBrokerUrl = process.env.PACT_BROKER_URL || 'http://localhost:9292';
  const pactBrokerToken = process.env.PACT_BROKER_TOKEN;
  const gitSha = process.env.GIT_SHA || process.env.GITHUB_SHA;
  const branch = process.env.GIT_BRANCH || process.env.GITHUB_REF_NAME;

  const opts = {
    pactFilesOrDirs: [path.resolve(__dirname, './pacts')],
    pactBroker: pactBrokerUrl,
    pactBrokerToken: pactBrokerToken,
    consumerVersion: gitSha || '1.0.0',
    tags: [branch || 'main'],
  };

  try {
    await pact.publishPacts(opts);
    console.log('Pacts published successfully');
  } catch (error) {
    console.error('Failed to publish pacts:', error);
    process.exit(1);
  }
}
