# DIYAPP Evolution - V12 Core - Arquitetura de Microsserviços

## ADR-001: Arquitetura de Microsserviços Autônomos

**Data:** 2024-01-15  
**Status:** Aceita  
**Autores:** Tech Lead + Especialista Infra + Especialista Backend

### CONTEXTO:
O DIYAPP evoluiu de uma aplicação monolítica para um sistema distribuído complexo. Precisamos de uma arquitetura que permita:
1. Autonomia total das squads no desenvolvimento e deploy
2. Escalabilidade independente de componentes
3. Resiliência a falhas de componentes individuais
4. Comunicação eficiente entre serviços
5. Observabilidade unificada

### DECISÃO:
Adotar arquitetura de microsserviços baseada em eventos com comunicação síncrona via gRPC quando necessário. Cada microsserviço será totalmente autônomo com seu próprio banco de dados, código e ciclo de deploy.

### OPÇÕES CONSIDERADAS:

**Opção A: Monolito Modularizado**
- Prós: Simplicidade de deploy, debugging facilitado, transações ACID
- Contras: Acoplamento alto, escalabilidade limitada, deploy único
- **NÃO ESCOLHIDA**: Não atende requisito de autonomia das squads

**Opção B: Microsserviços com REST/HTTP**
- Prós: Simples de implementar, amplamente adotado, fácil debugging
- Contras: Overhead de serialização JSON, sem contrato forte, sem streaming nativo
- **NÃO ESCOLHIDA**: Performance inferior para comunicação interna

**Opção C: Microsserviços com gRPC + Eventos (ESCOLHIDA)**
- Prós: Contratos fortes com Protobuf, alta performance, streaming bidirecional, geração de código automática
- Contras: Curva de aprendizado, tooling menos maduro que REST
- **JUSTIFICATIVA**: Oferece melhor performance, contratos fortes e suporte a streaming para casos de uso em tempo real

### CONSEQUÊNCIAS:
**Positivas:**
- Autonomia total por squad
- Escalabilidade granular
- Resiliência melhorada
- Contratos de API fortemente tipados
- Performance otimizada

**Negativas:**
- Complexidade de operação aumentada
- Necessidade de orquestração de deploy
- Transações distribuídas mais complexas
- Overhead de rede

**Riscos:**
- Latência em chamadas entre serviços
- Gerenciamento de versões de contratos
- Mitigação: Circuit breakers, versionamento semântico de APIs, monitoramento detalhado

class CircuitBreakerState(str, Enum):
    CLOSED = "closed"      # Normal operation
    OPEN = "open"          # Service unavailable, failing fast
    HALF_OPEN = "half_open" # Testing if service recovered

@dataclass
class CircuitBreakerConfig:
    failure_threshold: int = 5          # Failures before opening
    recovery_timeout: int = 30          # Seconds before half-open
    half_open_max_attempts: int = 3     # Successes needed to close
    sliding_window_size: int = 100      # Requests in sliding window

class CircuitBreaker:
    """Circuit breaker implementation with exponential backoff"""
    
    def __init__(self, service_name: str, config: Optional[CircuitBreakerConfig] = None):
        self.service_name = service_name
        self.config = config or CircuitBreakerConfig()
        self.state = CircuitBreakerState.CLOSED
        self.failure_count = 0
        self.success_count = 0
        self.last_failure_time: Optional[datetime] = None
        self.last_state_change = datetime.utcnow()
        
    def record_success(self):
        """Record a successful call"""
        self.success_count += 1
        self.failure_count = 0
        
        if self.state == CircuitBreakerState.HALF_OPEN:
            if self.success_count >= self.config.half_open_max_attempts:
                self._transition_to(CircuitBreakerState.CLOSED)
        
        CIRCUIT_BREAKER_STATE.labels(service_name=self.service_name).set(
            0 if self.state == CircuitBreakerState.CLOSED else 
            1 if self.state == CircuitBreakerState.HALF_OPEN else 2
        )
    
    def record_failure(self):
        """Record a failed call"""
        self.failure_count += 1
        self.success_count = 0
        self.last_failure_time = datetime.utcnow()
        
        if (self.state == CircuitBreakerState.CLOSED and 
            self.failure_count >= self.config.failure_threshold):
            self._transition_to(CircuitBreakerState.OPEN)
        
        CIRCUIT_BREAKER_STATE.labels(service_name=self.service_name).set(
            0 if self.state == CircuitBreakerState.CLOSED else 
            1 if self.state == CircuitBreakerState.HALF_OPEN else 2
        )
    
    def _transition_to(self, new_state: CircuitBreakerState):
        """Transition to new state"""
        old_state = self.state
        self.state = new_state
        self.last_state_change = datetime.utcnow()
        
        logger.info(f"Circuit breaker {self.service_name}: {old_state} -> {new_state}")
    
    def can_execute(self) -> bool:
        """Check if request can be executed"""
        if self.state == CircuitBreakerState.CLOSED:
            return True
        
        if self.state == CircuitBreakerState.OPEN:
            # Check if recovery timeout has passed
            if self.last_failure_time:
                recovery_time = self.last_failure_time + timedelta(
                    seconds=self.config.recovery_timeout
                )
                if datetime.utcnow() >= recovery_time:
                    self._transition_to(CircuitBreakerState.HALF_OPEN)
                    return True
            return False
        
        if self.state == CircuitBreakerState.HALF_OPEN:
            return True
        
        return False
    
    @asynccontextmanager
    async def protect(self):
        """Context manager for protected execution"""
        if not self.can_execute():
            raise CircuitBreakerOpenError(f"Circuit breaker open for {self.service_name}")
        
        try:
            yield
            self.record_success()
        except Exception as e:
            self.record_failure()
            raise

class RedisQueueManager:
    """Manages Redis queues for agent orchestration"""
    
    def __init__(self, redis_url: str = "redis://localhost:6379"):
        self.redis_url = redis_url
        self.redis_client: Optional[redis.Redis] = None
        self.connected = False
        
    async def connect(self):
        """Establish Redis connection"""
        try:
            self.redis_client = redis.from_url(
                self.redis_url,
                encoding="utf-8",
                decode_responses=True
            )
            await self.redis_client.ping()
            self.connected = True
            logger.info("Redis connection established")
        except Exception as e:
            logger.error(f"Redis connection failed: {e}")
            raise
    
    async def disconnect(self):
        """Close Redis connection"""
        if self.redis_client:
            await self.redis_client.close()
            self.connected = False
    
    def _get_queue_key(self, agent_type: AgentType) -> str:
        """Get Redis key for agent queue"""
        return f"agent_queue:{agent_type.value}"
    
    def _get_processing_key(self, agent_type: AgentType) -> str:
        """Get Redis key for processing set"""
        return f"agent_processing:{agent_type.value}"
    
    def _get_dead_letter_key(self, agent_type: AgentType) -> str:
        """Get Redis key for dead letter queue"""
        return f"agent_dlq:{agent_type.value}"
    
    async def enqueue_request(self, request: AgentRequest) -> bool:
        """Add request to appropriate queue"""
        if not self.connected:
            raise ConnectionError("Redis not connected")
        
        try:
            queue_key = self._get_queue_key(request.agent_type)
            
            # Use sorted set for priority queuing
            score = float(request.priority) + (1.0 / time.time())
            await self.redis_client.zadd(
                queue_key,
                {request.model_dump_json(): score}
            )
            
            # Update metrics
            queue_size = await self.redis_client.zcard(queue_key)
            AGENT_QUEUE_SIZE.labels(queue_name=request.agent_type.value).set(queue_size)
            
            logger.info(f"Enqueued request {request.request_id} for {request.agent_type}")
            return True
            
        except Exception as e:
            logger.error(f"Failed to enqueue request: {e}")
            return False
    
    async def dequeue_request(self, agent_type: AgentType) -> Optional[AgentRequest]:
        """Get next request from queue"""
        if not self.connected:
            raise ConnectionError("Redis not connected")
        
        try:
            queue_key = self._get_queue_key(agent_type)
            processing_key = self._get_processing_key(agent_type)
            
            # Move highest priority item to processing set
            items = await self.redis_client.zpopmax(queue_key, count=1)
            if not items:
                return None
            
            request_json, _ = items[0]
            
            # Add to processing set with timestamp
            await self.redis_client.zadd(
                processing_key,
                {request_json: time.time()}
            )
            
            # Parse and return
            request_data = json.loads(request_json)
            request = AgentRequest(**request_data)
            
            # Update metrics
            queue_size = await self.redis_client.zcard(queue_key)
            AGENT_QUEUE_SIZE.labels(queue_name=agent_type.value).set(queue_size)
            
            return request
            
        except Exception as e:
            logger.error(f"Failed to dequeue request: {e}")
            return None
    
    async def complete_request(self, request: AgentRequest, response: AgentResponse):
        """Mark request as completed"""
        if not self.connected:
            raise ConnectionError("Redis not connected")
        
        try:
            processing_key = self._get_processing_key(request.agent_type)
            request_json = request.model_dump_json()
            
            # Remove from processing set
            await self.redis_client.zrem(processing_key, request_json)
            
            # Store response in cache for 5 minutes
            response_key = f"response:{request.request_id}"
            await self.redis_client.setex(
                response_key,
                300,
                response.model_dump_json()
            )
            
        except Exception as e:
            logger.error(f"Failed to complete request: {e}")
    
    async def move_to_dead_letter(self, request: AgentRequest, error: str):
        """Move failed request to dead letter queue"""
        if not self.connected:
            raise ConnectionError("Redis not connected")
        
        try:
            processing_key = self._get_processing_key(request.agent_type)
            dlq_key = self._get_dead_letter_key(request.agent_type)
            request_json = request.model_dump_json()
            
            # Remove from processing
            await self.redis_client.zrem(processing_key, request_json)
            
            # Add to DLQ with error metadata
            dlq_item = {
                "request": request.model_dump(),
                "error": error,
                "failed_at": datetime.utcnow().isoformat()
            }
            
            await self.redis_client.lpush(
                dlq_key,
                json.dumps(dlq_item)
            )
            
            logger.warning(f"Moved request {request.request_id} to DLQ: {error}")
            
        except Exception as e:
            logger.error(f"Failed to move to DLQ: {e}")

// src/hooks/useWebSocket.ts
import { useEffect, useRef, useCallback, useState } from 'react'
import { io, Socket } from 'socket.io-client'

export type WebSocketStatus = 'connecting' | 'connected' | 'disconnected' | 'error'

export interface WebSocketMessage {
  type: string
  payload: any
  timestamp: string
}

export interface UseWebSocketOptions {
  url: string
  autoConnect?: boolean
  reconnectAttempts?: number
  reconnectDelay?: number
  onMessage?: (message: WebSocketMessage) => void
  onStatusChange?: (status: WebSocketStatus) => void
}

export const useWebSocket = ({
  url,
  autoConnect = true,
  reconnectAttempts = 5,
  reconnectDelay = 3000,
  onMessage,
  onStatusChange
}: UseWebSocketOptions) => {
  const socketRef = useRef<Socket | null>(null)
  const reconnectCountRef = useRef(0)
  const [status, setStatus] = useState<WebSocketStatus>('disconnected')
  const [lastMessage, setLastMessage] = useState<WebSocketMessage | null>(null)

  const updateStatus = useCallback((newStatus: WebSocketStatus) => {
    setStatus(newStatus)
    onStatusChange?.(newStatus)
  }, [onStatusChange])

  const connect = useCallback(() => {
    if (socketRef.current?.connected) return

    updateStatus('connecting')
    
    const socket = io(url, {
      transports: ['websocket', 'polling'],
      reconnection: true,
      reconnectionAttempts: reconnectAttempts,
      reconnectionDelay: reconnectDelay,
      timeout: 10000
    })

    socket.on('connect', () => {
      updateStatus('connected')
      reconnectCountRef.current = 0
    })

    socket.on('disconnect', () => {
      updateStatus('disconnected')
    })

    socket.on('connect_error', (error) => {
      console.error('WebSocket connection error:', error)
      updateStatus('error')
      
      if (reconnectCountRef.current < reconnectAttempts) {
        reconnectCountRef.current += 1
        setTimeout(() => {
          socket.connect()
        }, reconnectDelay)
      }
    })

    socket.on('message', (data: WebSocketMessage) => {
      setLastMessage(data)
      onMessage?.(data)
    })

    socketRef.current = socket
  }, [url, reconnectAttempts, reconnectDelay, updateStatus, onMessage])

  const disconnect = useCallback(() => {
    if (socketRef.current) {
      socketRef.current.disconnect()
      socketRef.current = null
      updateStatus('disconnected')
    }
  }, [updateStatus])

  const send = useCallback((type: string, payload: any) => {
    if (socketRef.current?.connected) {
      const message: WebSocketMessage = {
        type,
        payload,
        timestamp: new Date().toISOString()
      }
      socketRef.current.emit('message', message)
      return true
    }
    return false
  }, [])

  useEffect(() => {
    if (autoConnect) {
      connect()
    }

    return () => {
      disconnect()
    }
  }, [autoConnect, connect, disconnect])

  return {
    status,
    lastMessage,
    connect,
    disconnect,
    send,
    isConnected: status === 'connected'
  }
}

CREATE DATABASE IF NOT EXISTS diyapp_telemetry
ENGINE = Atomic
COMMENT 'DIYAPP Telemetry Data Lake';

-- Tabela raw para eventos de agentes
CREATE TABLE IF NOT EXISTS diyapp_telemetry.raw_agent_events
(
    event_id UUID DEFAULT generateUUIDv4(),
    agent_id String,
    agent_type Enum8('llm' = 1, 'code' = 2, 'data' = 3, 'ops' = 4, 'pm' = 5),
    event_type Enum8('start' = 1, 'complete' = 2, 'error' = 3, 'decision' = 4, 'api_call' = 5),

    session_id String,
    parent_event_id Nullable(UUID),
    timestamp DateTime64(3, 'UTC'),
    processing_time_ms Nullable(UInt32),
    success Nullable(UInt8),
    error_message Nullable(String),
    metadata Map(String, String),
    ingestion_time DateTime64(3, 'UTC') DEFAULT now64(3)
)
ENGINE = MergeTree()
PARTITION BY toYYYYMM(timestamp)
ORDER BY (agent_id, timestamp, event_id)
TTL ingestion_time + INTERVAL 90 DAY
SETTINGS index_granularity = 8192;

    total_tokens UInt32,

    latency_ms UInt32,
    success UInt8,

    request_metadata Map(String, String),
    response_metadata Map(String, String),
    timestamp DateTime64(3, 'UTC'),

# DIYAPP Evolution - V12 Core - Arquitetura de Microsserviços

## ADR-001: Arquitetura de Microsserviços Autônomos

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista Backend

## CONTEXTO:
O DIYAPP evoluiu para uma plataforma complexa com múltiplos domínios (usuários, projetos, IA, pagamentos, notificações). A arquitetura monolítica atual limita:
1. Escalabilidade independente por domínio
2. Deploy contínuo sem downtime
3. Autonomia das squads
4. Resiliência a falhas
5. Experimentação com novas tecnologias

## DECISÃO:
Adotar arquitetura de microsserviços com comunicação via gRPC para chamadas síncronas e eventos Kafka para comunicação assíncrona. Cada microsserviço será totalmente autônomo com seu próprio banco de dados, CI/CD e equipe responsável.

## OPÇÕES CONSIDERADAS:

**Opção A: Monolito Modularizado**
- Prós: Simplicidade operacional, transações ACID fáceis, debugging centralizado
- Contras: Acoplamento alto, deploy all-or-nothing, dificuldade de escalar componentes específicos

**Opção B: Microsserviços com REST/HTTP**
- Prós: Independência de tecnologia, fácil debugging HTTP, ecossistema maduro
- Contras: Overhead de serialização JSON, sem contrato forte, versionamento complexo

**Opção C: Microsserviços com gRPC + Eventos (ESCOLHIDA)**
- Prós: Contratos fortes com Protobuf, alta performance, streaming bidirecional, suporte nativo a múltiplas linguagens
- Contras: Curva de aprendizado, tooling menos maduro que REST, necessidade de gRPC gateway para clientes web

## CONSEQUÊNCIAS:
**Positivas:**
- Escalabilidade granular por serviço
- Deploy independente sem afetar todo o sistema
- Poliglotismo tecnológico (cada squad escolhe sua stack)
- Melhor isolamento de falhas
- Autonomia total das squads

**Negativas:**
- Complexidade operacional aumentada
- Necessidade de service mesh/orquestrador
- Transações distribuídas complexas (Saga pattern)
- Monitoramento distribuído essencial
- Latência de rede entre serviços

**Riscos:**
1. Falhas em cascata sem circuit breaker
2. Consistência eventual em alguns cenários
3. Overhead de comunicação entre serviços

**Mitigações:**
- Implementar Istio como service mesh
- Usar padrão Saga para transações
- Adotar OpenTelemetry para observabilidade
- Configurar retry policies com backoff exponencial

class ResilientCircuitBreaker:
    """Circuit breaker com fallback e auto-reset"""
    
    def __init__(
        self,
        agent_id: str,
        failure_threshold: int = 5,
        reset_timeout: int = 60,
        expected_exceptions: Tuple = (Exception,)
    ):
        self.agent_id = agent_id
        self.failure_threshold = failure_threshold
        self.reset_timeout = reset_timeout
        self.expected_exceptions = expected_exceptions
        
        self.failure_count = 0
        self.last_failure_time: Optional[datetime] = None
        self.state = "CLOSED"  # CLOSED, OPEN, HALF_OPEN
        
        # Métricas
        CIRCUIT_BREAKER_STATE.labels(agent_id=self.agent_id).set(0)  # 0=CLOSED, 1=OPEN, 2=HALF_OPEN
    
    def record_failure(self):
        """Registra uma falha e atualiza o estado"""
        self.failure_count += 1
        self.last_failure_time = datetime.utcnow()
        CIRCUIT_BREAKER_FAILURES.labels(agent_id=self.agent_id).inc()
        
        if self.failure_count >= self.failure_threshold:
            self.state = "OPEN"
            CIRCUIT_BREAKER_STATE.labels(agent_id=self.agent_id).set(1)
            logging.warning(f"Circuit breaker OPEN para agente {self.agent_id}")
    
    def record_success(self):
        """Registra um sucesso e reseta se necessário"""
        if self.state == "HALF_OPEN":
            self.state = "CLOSED"
            self.failure_count = 0
            CIRCUIT_BREAKER_STATE.labels(agent_id=self.agent_id).set(0)
            logging.info(f"Circuit breaker CLOSED para agente {self.agent_id}")
    
    def should_try(self) -> bool:
        """Verifica se deve permitir a execução"""
        if self.state == "CLOSED":
            return True
        
        if self.state == "OPEN":
            if self.last_failure_time:
                time_since_failure = (datetime.utcnow() - self.last_failure_time).total_seconds()
                if time_since_failure > self.reset_timeout:
                    self.state = "HALF_OPEN"
                    CIRCUIT_BREAKER_STATE.labels(agent_id=self.agent_id).set(2)
                    logging.info(f"Circuit breaker HALF_OPEN para agente {self.agent_id}")
                    return True
            return False
        
        return True  # HALF_OPEN
    
    async def execute(
        self,
        func: Callable,
        fallback_func: Optional[Callable] = None,
        *args,
        **kwargs
    ) -> Any:
        """Executa uma função com circuit breaker"""
        if not self.should_try():
            logging.warning(f"Circuit breaker bloqueando execução para {self.agent_id}")
            if fallback_func:
                return await fallback_func(*args, **kwargs)
            raise CircuitBreakerOpenError(f"Circuit breaker aberto para {self.agent_id}")
        
        try:
            result = await func(*args, **kwargs)
            self.record_success()
            return result
        except self.expected_exceptions as e:
            self.record_failure()
            if fallback_func:
                logging.warning(f"Executando fallback para {self.agent_id}: {str(e)}")
                return await fallback_func(*args, **kwargs)
            raise

// src/services/websocket.service.ts
import { WebSocketMessage, SystemStatus, AlertData } from '../types';

const WS_RECONNECT_DELAY = 1000;
const WS_MAX_RECONNECT_ATTEMPTS = 5;
const WS_HEARTBEAT_INTERVAL = 30000;

export class WebSocketService {
  private ws: WebSocket | null = null;
  private reconnectAttempts = 0;
  private heartbeatInterval: NodeJS.Timeout | null = null;
  private messageHandlers: Map<string, ((data: any) => void)[]> = new Map();
  private connectionHandlers: ((connected: boolean) => void)[] = [];
  private url: string;

  constructor(url: string) {
    this.url = url;
  }

  connect(): Promise<void> {
    return new Promise((resolve, reject) => {
      try {
        this.ws = new WebSocket(this.url);
        
        this.ws.onopen = () => {
          console.log('WebSocket connected');
          this.reconnectAttempts = 0;
          this.startHeartbeat();
          this.notifyConnectionChange(true);
          resolve();
        };

        this.ws.onmessage = (event) => {
          try {
            const message: WebSocketMessage = JSON.parse(event.data);
            this.handleMessage(message);
          } catch (error) {
            console.error('Failed to parse WebSocket message:', error);
          }
        };

        this.ws.onclose = () => {
          console.log('WebSocket disconnected');
          this.stopHeartbeat();
          this.notifyConnectionChange(false);
          this.attemptReconnect();
        };

        this.ws.onerror = (error) => {
          console.error('WebSocket error:', error);
          reject(error);
        };
      } catch (error) {
        reject(error);
      }
    });
  }

  disconnect(): void {
    this.stopHeartbeat();
    if (this.ws) {
      this.ws.close();
      this.ws = null;
    }
  }

  send(message: any): void {
    if (this.ws && this.ws.readyState === WebSocket.OPEN) {
      this.ws.send(JSON.stringify(message));
    } else {
      console.warn('WebSocket is not connected');
    }
  }

  subscribe<T>(type: string, handler: (data: T) => void): () => void {
    if (!this.messageHandlers.has(type)) {
      this.messageHandlers.set(type, []);
    }
    
    const handlers = this.messageHandlers.get(type)!;
    handlers.push(handler as any);
    
    return () => {
      const index = handlers.indexOf(handler as any);
      if (index > -1) {
        handlers.splice(index, 1);
      }
    };
  }

  onConnectionChange(handler: (connected: boolean) => void): () => void {
    this.connectionHandlers.push(handler);
    return () => {
      const index = this.connectionHandlers.indexOf(handler);
      if (index > -1) {
        this.connectionHandlers.splice(index, 1);
      }
    };
  }

  private handleMessage(message: WebSocketMessage): void {
    const handlers = this.messageHandlers.get(message.type);
    if (handlers) {
      handlers.forEach(handler => handler(message.data));
    }
  }

  private notifyConnectionChange(connected: boolean): void {
    this.connectionHandlers.forEach(handler => handler(connected));
  }

  private startHeartbeat(): void {
    this.heartbeatInterval = setInterval(() => {
      if (this.ws && this.ws.readyState === WebSocket.OPEN) {
        this.send({ type: 'heartbeat', timestamp: new Date().toISOString() });
      }
    }, WS_HEARTBEAT_INTERVAL);
  }

  private stopHeartbeat(): void {
    if (this.heartbeatInterval) {
      clearInterval(this.heartbeatInterval);
      this.heartbeatInterval = null;
    }
  }

  private attemptReconnect(): void {
    if (this.reconnectAttempts < WS_MAX_RECONNECT_ATTEMPTS) {
      this.reconnectAttempts++;
      console.log(`Attempting to reconnect (${this.reconnectAttempts}/${WS_MAX_RECONNECT_ATTEMPTS})...`);
      
      setTimeout(() => {
        this.connect().catch(() => {
          // Reconnection will be attempted again on next interval
        });
      }, WS_RECONNECT_DELAY * Math.pow(2, this.reconnectAttempts - 1));
    } else {
      console.error('Max reconnection attempts reached');
    }
  }

  get isConnected(): boolean {
    return this.ws?.readyState === WebSocket.OPEN;
  }
}

// Singleton instance
export const websocketService = new WebSocketService(
  process.env.REACT_APP_WS_URL || 'ws://localhost:8080/ws'

**clickhouse/init.sql**
```sql
-- Criar database para telemetria
CREATE DATABASE IF NOT EXISTS diyapp_telemetry
ENGINE = Atomic;

USE diyapp_telemetry;

-- Tabela raw para eventos de agentes (camada bronze)
CREATE TABLE IF NOT EXISTS raw_agent_events
(
    event_id UUID DEFAULT generateUUIDv4(),
    agent_id String,
    agent_type Enum8('llm' = 1, 'data_engineer' = 2, 'product_manager' = 3, 'ai_ops' = 4, 'compliance' = 5),
    event_type Enum8('start' = 1, 'complete' = 2, 'error' = 3, 'decision' = 4, 'api_call' = 5, 'llm_call' = 6),
    event_data String,
    event_timestamp DateTime64(3, 'UTC'),
    session_id String,
    parent_event_id Nullable(UUID),
    metadata String,
    ingestion_timestamp DateTime64(3, 'UTC') DEFAULT now64(3)
)
ENGINE = MergeTree()
PARTITION BY toYYYYMM(event_timestamp)
ORDER BY (agent_id, event_timestamp)
TTL event_timestamp + INTERVAL 90 DAY;

-- Tabela raw para performance de agentes
CREATE TABLE IF NOT EXISTS raw_agent_performance
(
    performance_id UUID DEFAULT generateUUIDv4(),
    agent_id String,
    task_id String,
    task_type String,
    start_time DateTime64(3, 'UTC'),
    end_time DateTime64(3, 'UTC'),
    duration_ms UInt64,
    success Bool,
    error_message Nullable(String),
    tokens_used Nullable(UInt32),
    cost_usd Nullable(Float64),
    memory_usage_mb Nullable(Float32),
    cpu_usage_percent Nullable(Float32),
    input_tokens Nullable(UInt32),
    output_tokens Nullable(UInt32),
    llm_model Nullable(String),
    llm_provider Nullable(String),
    metadata String,
    ingestion_timestamp DateTime64(3, 'UTC') DEFAULT now64(3)
)
ENGINE = MergeTree()
PARTITION BY toYYYYMM(start_time)
ORDER BY (agent_id, start_time)
TTL start_time + INTERVAL 90 DAY;

-- Tabela raw para erros de agentes
CREATE TABLE IF NOT EXISTS raw_agent_errors
(
    error_id UUID DEFAULT generateUUIDv4(),
    agent_id String,
    error_code String,
    error_message String,
    error_stack_trace String,
    severity Enum8('low' = 1, 'medium' = 2, 'high' = 3, 'critical' = 4),
    context_data String,
    event_timestamp DateTime64(3, 'UTC'),
    resolved Bool DEFAULT false,
    resolution_timestamp Nullable(DateTime64(3, 'UTC')),
    metadata String,
    ingestion_timestamp DateTime64(3, 'UTC') DEFAULT now64(3)
)
ENGINE = MergeTree()
PARTITION BY toYYYYMM(event_timestamp)
ORDER BY (severity, event_timestamp)
TTL event_timestamp + INTERVAL 180 DAY;

-- Tabela para dados de custo de LLM
CREATE TABLE IF NOT EXISTS raw_llm_costs
(
    cost_id UUID DEFAULT generateUUIDv4(),
    agent_id String,
    llm_provider String,
    llm_model String,
    operation_type Enum8('completion' = 1, 'embedding' = 2, 'fine_tuning' = 3),
    input_tokens UInt32,
    output_tokens UInt32,
    total_tokens UInt32,
    cost_usd Float64,
    request_duration_ms UInt32,
    request_timestamp DateTime64(3, 'UTC'),
    success Bool,
    error_message Nullable(String),
    metadata String,
    ingestion_timestamp DateTime64(3, 'UTC') DEFAULT now64(3)
)
ENGINE = MergeTree()
PARTITION BY toYYYYMM(request_timestamp)
ORDER BY (llm_provider, request_timestamp)
TTL request_timestamp + INTERVAL 180 DAY;

-- Criar views materializadas para agregações em tempo real
CREATE MATERIALIZED VIEW IF NOT EXISTS mv_agent_performance_5min
ENGINE = AggregatingMergeTree()
PARTITION BY toYYYYMM(start_time)
ORDER BY (agent_id, start_time)
AS SELECT
    agent_id,
    toStartOfFiveMinute(start_time) as start_time,
    countState() as total_tasks,
    sumState(duration_ms) as total_duration,
    sumStateIf(duration_ms, success = 1) as successful_duration,
    countIfState(success = 1) as successful_tasks,
    avgState(duration_ms) as avg_duration,
    quantileState(0.95)(duration_ms) as p95_duration,
    sumState(tokens_used) as total_tokens,
    sumState(cost_usd) as total_cost
FROM raw_agent_performance
GROUP BY agent_id, toStartOfFiveMinute(start_time);

-- View para health check de agentes
CREATE MATERIALIZED VIEW IF NOT EXISTS mv_agent_health_status
ENGINE = ReplacingMergeTree()
ORDER BY (agent_id, check_time)
AS SELECT
    agent_id,
    toStartOfMinute(now()) as check_time,
    countIf(event_type = 'error') as error_count_last_minute,
    countIf(event_type = 'complete') as completed_tasks_last_minute,
    minIf(event_timestamp, event_type = 'complete') as last_success_time,
    maxIf(event_timestamp, event_type = 'error') as last_error_time
FROM raw_agent_events
WHERE event_timestamp >= now() - INTERVAL 1 MINUTE
GROUP BY agent_id;
```
