## ADR-001: Padrões de Comunicação entre Microsserviços

```markdown
# ADR-001: Padrões de Comunicação entre Microsserviços

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra

## CONTEXTO:
Precisamos definir padrões de comunicação claros para garantir:
1. Baixo acoplamento entre serviços
2. Alta disponibilidade
3. Tolerância a falhas
4. Observabilidade unificada

## DECISÃO:
Adotar comunicação híbrida:
- Síncrona: REST/HTTP para operações críticas que exigem resposta imediata
- Assíncrona: Eventos via RabbitMQ para operações de background e notificações
- Cache distribuído: Redis para sessões e dados frequentemente acessados

## OPÇÕES CONSIDERADAS:
- **Opção A**: REST puro - Simples, mas acoplamento temporal
- **Opção B**: Mensageria pura - Desacoplado, mas complexidade aumentada
- **Opção C**: Híbrido (escolhida) - Balanceia simplicidade e resiliência

## CONSEQUÊNCIAS:
**Positivas:**
- Resiliência a falhas parciais
- Escalabilidade independente por serviço
- Melhor experiência do usuário (operações assíncronas)

**Negativas:**
- Maior complexidade operacional
- Necessidade de monitoramento distribuído
- Garantia de entrega de mensagens

**Riscos:**
- Perda de mensagens (mitigar com DLQ e retry policies)
- Latência em cadeias de eventos longas (monitorar com tracing)

**REVISÃO:** 2024-04-15
```

## Configuração do API Gateway

```typescript
// src/api-gateway/src/index.ts
import express from 'express';
import cors from 'cors';
import helmet from 'helmet';
import rateLimit from 'express-rate-limit';
import { createProxyMiddleware } from 'http-proxy-middleware';
import { logger, metricsMiddleware, errorHandler } from '../../shared/middleware';
import { healthCheck } from '../../shared/utils/health-check';

const app = express();
const PORT = process.env.PORT || 3000;

// Security middleware
app.use(helmet());
app.use(cors({
  origin: process.env.ALLOWED_ORIGINS?.split(',') || [],
  credentials: true,
}));

// Rate limiting
const limiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 minutes
  max: 100, // limit each IP to 100 requests per windowMs
  message: 'Too many requests from this IP, please try again later.',
});
app.use('/api/', limiter);

// Metrics and logging
app.use(metricsMiddleware);
app.use(logger);

// Health check endpoint
app.get('/health', async (req, res) => {
  const health = await healthCheck();
  res.status(health.status === 'healthy' ? 200 : 503).json(health);
});

// Service routing with circuit breaker pattern
const circuitBreakerOptions = {
  circuitBreaker: {
    windowDuration: 10000,
    numBuckets: 10,
    timeoutDuration: 5000,
    errorThreshold: 50,
    volumeThreshold: 10,
  },
};

// Proxy middleware for each service
app.use('/api/v1/auth', createProxyMiddleware({
  target: process.env.AUTH_SERVICE_URL || 'http://auth-service:3001',
  changeOrigin: true,
  pathRewrite: { '^/api/v1/auth': '/' },
  ...circuitBreakerOptions,
}));

app.use('/api/v1/users', createProxyMiddleware({
  target: process.env.USER_SERVICE_URL || 'http://user-service:3002',
  changeOrigin: true,
  pathRewrite: { '^/api/v1/users': '/' },
  ...circuitBreakerOptions,
}));

app.use('/api/v1/tasks', createProxyMiddleware({
  target: process.env.TASK_SERVICE_URL || 'http://task-service:3003',
  changeOrigin: true,
  pathRewrite: { '^/api/v1/tasks': '/' },
  ...circuitBreakerOptions,
}));

// Error handling
app.use(errorHandler);

// Start server
app.listen(PORT, () => {
  console.log(`API Gateway running on port ${PORT}`);
  console.log(`Health check available at http://localhost:${PORT}/health`);
});
```

// PostgreSQL Connection Pool with optimized settings
class PostgresPool {
  constructor() {
    this.pool = new Pool({
      host: process.env.PG_HOST || 'localhost',
      port: process.env.PG_PORT || 5432,
      database: process.env.PG_DATABASE || 'diyapp',
      user: process.env.PG_USER || 'postgres',
      password: process.env.PG_PASSWORD || 'postgres',
      max: 20, // Maximum number of clients in pool
      idleTimeoutMillis: 30000,
      connectionTimeoutMillis: 2000,
      ssl: process.env.NODE_ENV === 'production' ? { rejectUnauthorized: false } : false
    });

    this.pool.on('error', (err) => {
      logger.error('Unexpected PostgreSQL pool error', {
        error: err.message,
        stack: err.stack
      });
    });
  }

  async query(text, params) {
    const start = Date.now();
    try {
      const result = await this.pool.query(text, params);
      const duration = Date.now() - start;
      
      logger.debug('PostgreSQL query executed', {
        query: text,
        params: this.maskSensitiveData(params),
        duration,
        rowCount: result.rowCount
      });
      
      return result;
    } catch (error) {
      const duration = Date.now() - start;
      logger.error('PostgreSQL query failed', {
        query: text,
        params: this.maskSensitiveData(params),
        duration,
        error: error.message
      });
      throw error;
    }
  }

  maskSensitiveData(params) {
    if (!params) return params;
    return params.map(param => {
      if (typeof param === 'string' && (
        param.includes('@') || // email
        param.match(/^\d{3}\.\d{3}\.\d{3}-\d{2}$/) || // CPF
        param.match(/^\d{16}$/) // credit card
      )) {
        return '[MASKED]';
      }
      return param;
    });
  }
}

// Redis Cache with Circuit Breaker
class RedisCache {
  constructor() {
    this.client = createClient({
      url: process.env.REDIS_URL || 'redis://localhost:6379',
      socket: {
        reconnectStrategy: (retries) => {
          if (retries > 10) {
            logger.error('Redis connection failed after max retries');
            return new Error('Max retries exceeded');
          }
          return Math.min(retries * 100, 3000);
        }
      }
    });

    this.circuitState = 'CLOSED';
    this.failureCount = 0;
    this.nextAttempt = Date.now();
    this.resetTimeout = 30000; // 30 seconds circuit reset

    this.client.on('error', (err) => {
      logger.error('Redis client error', { error: err.message });
      this.handleFailure();
    });

    this.client.on('connect', () => {
      logger.info('Redis connected successfully');
      this.resetCircuit();
    });
  }

  async connect() {
    try {
      await this.client.connect();
      return true;
    } catch (error) {
      logger.error('Redis connection failed', { error: error.message });
      this.handleFailure();
      return false;
    }
  }

  handleFailure() {
    this.failureCount++;
    if (this.failureCount >= 5 && this.circuitState === 'CLOSED') {
      this.circuitState = 'OPEN';
      this.nextAttempt = Date.now() + this.resetTimeout;
      logger.warn('Redis circuit breaker OPEN', { nextAttempt: new Date(this.nextAttempt) });
    }
  }

  resetCircuit() {
    this.circuitState = 'CLOSED';
    this.failureCount = 0;
    this.nextAttempt = Date.now();
  }

  async get(key) {
    if (this.circuitState === 'OPEN') {
      if (Date.now() < this.nextAttempt) {
        logger.warn('Redis circuit breaker is OPEN, skipping get');
        return null;
      }
      // Try to reset circuit
      this.circuitState = 'HALF_OPEN';
    }

    try {
      const start = Date.now();
      const value = await this.client.get(key);
      const duration = Date.now() - start;
      
      if (this.circuitState === 'HALF_OPEN') {
        this.resetCircuit();
      }
      
      logger.debug('Redis GET', { key, duration, hit: value !== null });
      return value ? JSON.parse(value) : null;
    } catch (error) {
      this.handleFailure();
      logger.error('Redis GET failed', { key, error: error.message });
      return null;
    }
  }

  async set(key, value, ttl = 300) { // Default 5 minutes TTL
    if (this.circuitState === 'OPEN') return false;

    try {
      const start = Date.now();
      await this.client.setEx(key, ttl, JSON.stringify(value));
      const duration = Date.now() - start;
      
      logger.debug('Redis SET', { key, ttl, duration });
      return true;
    } catch (error) {
      this.handleFailure();
      logger.error('Redis SET failed', { key, error: error.message });
      return false;
    }
  }

  async invalidate(pattern) {
    try {
      const keys = await this.client.keys(pattern);
      if (keys.length > 0) {
        await this.client.del(keys);
        logger.debug('Redis invalidated keys', { pattern, count: keys.length });
      }
    } catch (error) {
      logger.error('Redis invalidation failed', { pattern, error: error.message });
    }
  }
}

// Bull Queue with optimized job processing
const Queue = require('bull');
class TaskQueue {
  constructor() {
    this.queues = new Map();
    this.redisOptions = {
      redis: {
        url: process.env.REDIS_URL || 'redis://localhost:6379',
        maxRetriesPerRequest: 3,
        enableReadyCheck: false
      },
      defaultJobOptions: {
        removeOnComplete: 100, // Keep last 100 completed jobs
        removeOnFail: 500, // Keep last 500 failed jobs
        attempts: 3,
        backoff: {
          type: 'exponential',
          delay: 1000
        }
      }
    };
  }

  createQueue(name) {
    if (this.queues.has(name)) {
      return this.queues.get(name);
    }

    const queue = new Queue(name, this.redisOptions);
    
    queue.on('error', (error) => {
      logger.error(`Queue ${name} error`, { error: error.message });
    });

    queue.on('failed', (job, error) => {
      logger.error(`Job ${job.id} failed in queue ${name}`, {
        jobId: job.id,
        data: job.data,
        error: error.message,
        attemptsMade: job.attemptsMade
      });
    });

    queue.on('completed', (job) => {
      logger.debug(`Job ${job.id} completed in queue ${name}`, {
        jobId: job.id,
        data: job.data,
        duration: job.finishedOn - job.processedOn
      });
    });

    this.queues.set(name, queue);
    return queue;
  }

  async addJob(queueName, jobData, options = {}) {
    const queue = this.createQueue(queueName);
    const job = await queue.add(jobData, options);
    
    logger.info(`Job added to queue ${queueName}`, {
      jobId: job.id,
      data: this.maskSensitiveJobData(jobData)
    });
    
    return job;
  }

  maskSensitiveJobData(data) {
    if (!data) return data;
    
    const masked = { ...data };
    const sensitiveFields = ['email', 'password', 'token', 'cpf', 'creditCard', 'apiKey'];
    
    sensitiveFields.forEach(field => {
      if (masked[field]) {
        masked[field] = '[MASKED]';
      }
    });
    
    return masked;
  }
}

const DataLoader = require('dataloader');
class GraphQLService {
  constructor(postgresPool, redisCache) {
    this.postgresPool = postgresPool;
    this.redisCache = redisCache;
    this.loaders = new Map();
  }

  getUserLoader() {
    if (!this.loaders.has('user')) {
      this.loaders.set('user', new DataLoader(async (userIds) => {
        const cacheKey = 'users:batch:';
        const cachePromises = userIds.map(id => 
          this.redisCache.get(`${cacheKey}${id}`)
        );
        
        const cachedResults = await Promise.all(cachePromises);
        const missingIds = userIds.filter((_, index) => !cachedResults[index]);
        
        let dbResults = [];
        if (missingIds.length > 0) {
          const query = `
            SELECT id, email, name, created_at, updated_at 
            FROM users 
            WHERE id = ANY($1)
          `;
          const result = await this.postgresPool.query(query, [missingIds]);
          dbResults = result.rows;
          
          // Cache the results
          await Promise.all(
            dbResults.map(user => 
              this.redisCache.set(`${cacheKey}${user.id}`, user, 600) // 10 minutes
            )
          );
        }
        
        // Merge cached and db results
        return userIds.map(id => {
          const cached = cachedResults.find(r => r && r.id === id);
          if (cached) return cached;
          
          const db = dbResults.find(r => r.id === id);
          return db || null;
        });
      }));
    }
    return this.loaders.get('user');
  }

  // Optimized resolver for user with caching
  async getUser(id, useCache = true) {
    const correlationId = `user_${id}_${Date.now()}`;
    
    try {
      if (useCache) {
        const cached = await this.redisCache.get(`user:${id}`);
        if (cached) {
          logger.debug('User cache hit', { correlationId, userId: id });
          return cached;
        }
      }

      const query = `
        SELECT id, email, name, created_at, updated_at 
        FROM users 
        WHERE id = $1
        LIMIT 1
      `;
      
      const result = await this.postgresPool.query(query, [id]);
      
      if (result.rows.length === 0) {
        return null;
      }
      
      const user = result.rows[0];
      
      // Cache with 5 minute TTL
      await this.redisCache.set(`user:${id}`, user, 300);
      
      logger.debug('User fetched from database', { correlationId, userId: id });
      return user;
    } catch (error) {
      logger.error('Get user failed', { correlationId, userId: id, error: error.message });
      throw error;
    }
  }

  // Batch update users with queue
  async batchUpdateUsers(userUpdates) {
    const queue = new TaskQueue();
    
    // Split into chunks of 100 for processing
    const chunkSize = 100;
    const chunks = [];
    
    for (let i = 0; i < userUpdates.length; i += chunkSize) {
      chunks.push(userUpdates.slice(i, i + chunkSize));
    }
    
    // Add jobs to queue
    const jobPromises = chunks.map((chunk, index) => 
      queue.addJob('user-updates', {
        chunkIndex: index,
        updates: chunk,
        totalChunks: chunks.length
      }, {
        priority: index === 0 ? 1 : 2, // First chunk higher priority
        timeout: 30000 // 30 second timeout per job
      })
    );
    
    await Promise.all(jobPromises);
    
    // Invalidate user cache
    await this.redisCache.invalidate('user:*');
    
    return {
      success: true,
      message: `${userUpdates.length} users queued for update`,
      jobCount: chunks.length
    };
  }
}

class UserController {
  constructor(graphQLService) {
    this.graphQLService = graphQLService;
  }

  async getUser(req, res) {
    const correlationId = req.headers['x-correlation-id'] || `req_${Date.now()}`;
    const startTime = Date.now();
    
    try {
      const userId = req.params.id;
      
      if (!userId) {
        return res.status(400).json({
          error: 'User ID is required',
          correlationId
        });
      }

      // Rate limiting check (would integrate with rate limit middleware)
      const user = await this.graphQLService.getUser(userId);
      
      if (!user) {
        return res.status(404).json({
          error: 'User not found',
          correlationId
        });
      }

      const duration = Date.now() - startTime;
      
      logger.info('GET /users/:id', {
        correlationId,
        userId,
        duration,
        status: 200
      });
      
      return res.status(200).json({
        data: user,
        meta: {
          correlationId,
          duration
        }
      });
    } catch (error) {
      const duration = Date.now() - startTime;
      
      logger.error('GET /users/:id failed', {
        correlationId,
        error: error.message,
        stack: error.stack,
        duration,
        status: 500
      });
      
      return res.status(500).json({
        error: 'Internal server error',
        correlationId
      });
    }
  }

  async updateUser(req, res) {
    const correlationId = req.headers['x-correlation-id'] || `req_${Date.now()}`;
    const startTime = Date.now();
    
    try {
      const userId = req.params.id;
      const updates = req.body;
      
      if (!userId || !updates || Object.keys(updates).length === 0) {
        return res.status(400).json({
          error: 'User ID and update data are required',
          correlationId
        });
      }

      // Validate email format if present
      if (updates.email && !this.isValidEmail(updates.email)) {
        return res.status(422).json({
          error: 'Invalid email format',
          correlationId
        });
      }

      const query = `
        UPDATE users 
        SET name = COALESCE($1, name),
            email = COALESCE($2, email),
            updated_at = NOW()
        WHERE id = $3
        RETURNING id, email, name, created_at, updated_at
      `;
      
      const result = await this.graphQLService.postgresPool.query(query, [
        updates.name,
        updates.email,
        userId
      ]);
      
      if (result.rows.length === 0) {
        return res.status(404).json({
          error: 'User not found',
          correlationId
        });
      }

      const updatedUser = result.rows[0];
      
      // Invalidate cache
      await this.graphQLService.redisCache.invalidate(`user:${userId}`);
      await this.graphQLService.redisCache.invalidate('users:batch:*');
      
      const duration = Date.now() - startTime;
      
      logger.info('PUT /users/:id', {
        correlationId,
        userId,
        duration,
        status: 200
      });
      
      return res.status(200).json({
        data: updatedUser,
        meta: {
          correlationId,
          duration
        }
      });
    } catch (error) {
      const duration = Date.now() - startTime;
      
      logger.error('PUT /users/:id failed', {
        correlationId,
        error: error.message,
        stack: error.stack,
        duration,
        status: 500
      });
      
      // Check for duplicate email constraint
      if (error.code === '23505') { // PostgreSQL unique violation
        return res.status(409).json({
          error: 'Email already exists',
          correlationId
        });
      }
      
      return res.status(500).json({
        error: 'Internal server error',
        correlationId
      });
    }
  }

  isValidEmail(email) {
    const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    return emailRegex.test(email);
  }
}

from clickhouse_loader import ClickHouseLoader

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)

class KafkaEventConsumer:
    def __init__(self, bootstrap_servers='localhost:9092', group_id='diyapp-analytics'):
        self.consumer_config = {
            'bootstrap.servers': bootstrap_servers,
            'group.id': group_id,
            'auto.offset.reset': 'earliest',
            'enable.auto.commit': False
        }
        self.consumer = Consumer(self.consumer_config)
        self.clickhouse = ClickHouseLoader()
        
    def subscribe_to_topics(self, topics):
        """Subscribe to Kafka topics"""
        self.consumer.subscribe(topics)
        logger.info(f"Subscribed to topics: {topics}")
    
    def process_event(self, event):
        """Process and validate event before loading"""
        try:
            # Basic validation
            required_fields = ['event_id', 'user_id', 'event_type', 'timestamp']
            for field in required_fields:
                if field not in event:
                    raise ValueError(f"Missing required field: {field}")
            
            # Add processing metadata
            event['_processed_at'] = datetime.utcnow().isoformat()
            event['_ingestion_batch'] = datetime.utcnow().strftime('%Y%m%d%H')
            
            return event
        except Exception as e:
            logger.error(f"Event validation failed: {e}, Event: {event}")
            return None
    
    def consume_events(self):
        """Main consumption loop"""
        try:
            while True:
                msg = self.consumer.poll(1.0)
                
                if msg is None:
                    continue
                if msg.error():
                    if msg.error().code() == KafkaError._PARTITION_EOF:
                        continue
                    else:
                        logger.error(f"Kafka error: {msg.error()}")
                        continue
                
                try:
                    # Parse message
                    event = json.loads(msg.value().decode('utf-8'))
                    event['_topic'] = msg.topic()
                    event['_partition'] = msg.partition()
                    event['_offset'] = msg.offset()
                    
                    # Process event
                    processed_event = self.process_event(event)
                    if processed_event:
                        # Load to ClickHouse
                        self.clickhouse.insert_event('raw_user_events', processed_event)
                        
                        # Commit offset
                        self.consumer.commit(msg)
                        
                        logger.info(f"Processed event: {event['event_id']}")
                        
                except json.JSONDecodeError as e:
                    logger.error(f"JSON decode error: {e}")
                except Exception as e:
                    logger.error(f"Processing error: {e}")
                    
        except KeyboardInterrupt:
            logger.info("Shutting down consumer...")
        finally:
            self.consumer.close()
            self.clickhouse.close()

        """Insert event into ClickHouse"""
        try:
            # Convert properties to JSON string
            if 'properties' in event and isinstance(event['properties'], dict):
                event['properties'] = json.dumps(event['properties'])
            
            if 'device_info' in event and isinstance(event['device_info'], dict):
                event['device_info'] = json.dumps(event['device_info'])
            
            # Prepare columns and values
            columns = list(event.keys())
            values = [event[col] for col in columns]
            
            # Build query
            query = f"INSERT INTO {table_name} ({', '.join(columns)}) VALUES"
            
            self.client.execute(query, [values])
            
            logger.debug(f"Inserted into {table_name}: {event.get('event_id', 'N/A')}")
            
        except Exception as e:
            logger.error(f"Failed to insert event: {e}")
            raise
    
    def close(self):
        """Close connection"""
        self.client.disconnect()

```python
"""
Sistema de Otimização de Prompts com A/B Testing Automático
"""
import re
import json
import hashlib
from typing import Dict, List, Any, Optional, Tuple
from dataclasses import dataclass, asdict
from datetime import datetime
import yaml
from enum import Enum


class PromptType(Enum):
    SUMMARIZATION = "summarization"
    CLASSIFICATION = "classification"
    EXTRACTION = "extraction"
    GENERATION = "generation"
    CONVERSATION = "conversation"


@dataclass
class PromptVariant:
    """Variante de prompt para A/B testing"""
    id: str
    name: str
    prompt_template: str
    prompt_type: PromptType
    version: str
    parameters: Dict[str, Any]
    created_at: datetime
    is_active: bool = True
    performance_score: float = 0.0
    token_efficiency: float = 0.0
    cost_per_call: float = 0.0
    call_count: int = 0
    success_rate: float = 0.0


class PromptOptimizer:
    """Otimizador de prompts com tracking de performance"""
    
    def __init__(self, config_path: str = "config/prompts_config.yaml"):
        self.config_path = config_path
        self.prompts: Dict[str, PromptVariant] = {}
        self.active_variants: Dict[PromptType, List[str]] = {}
        self.load_config()
        
    def load_config(self):
        """Carrega configurações de prompts"""
        try:
            with open(self.config_path, 'r') as f:
                config = yaml.safe_load(f)
                
            for prompt_data in config.get('prompts', []):
                variant = PromptVariant(
                    id=prompt_data['id'],
                    name=prompt_data['name'],
                    prompt_template=prompt_data['template'],
                    prompt_type=PromptType(prompt_data['type']),
                    version=prompt_data['version'],
                    parameters=prompt_data.get('parameters', {}),
                    created_at=datetime.fromisoformat(prompt_data['created_at'])
                )
                self.prompts[variant.id] = variant
                
                if variant.is_active:
                    if variant.prompt_type not in self.active_variants:
                        self.active_variants[variant.prompt_type] = []
                    self.active_variants[variant.prompt_type].append(variant.id)
                    
        except FileNotFoundError:
            print(f"Config file {self.config_path} not found, starting with empty config")
            
    def optimize_prompt(self, prompt_type: PromptType, context: Dict[str, Any]) -> str:
        """
        Seleciona a melhor variante de prompt baseado em performance
        e aplica otimizações específicas
        """
        # Seleciona variante ativa com melhor score
        best_variant = self.get_best_variant(prompt_type)
        
        if not best_variant:
            # Fallback para template padrão
            return self.get_default_prompt(prompt_type, context)
            
        # Aplica otimizações dinâmicas
        optimized_prompt = self.apply_optimizations(
            best_variant.prompt_template, 
            context
        )
        
        # Atualiza estatísticas
        self.record_prompt_usage(best_variant.id)
        
        return optimized_prompt
        
    def get_best_variant(self, prompt_type: PromptType) -> Optional[PromptVariant]:
        """Retorna a variante com melhor performance score"""
        active_ids = self.active_variants.get(prompt_type, [])
        if not active_ids:
            return None
            
        variants = [self.prompts[pid] for pid in active_ids]
        return max(variants, key=lambda v: v.performance_score)
        
    def apply_optimizations(self, template: str, context: Dict[str, Any]) -> str:
        """
        Aplica otimizações específicas baseadas no contexto
        """
        prompt = template
        
        # 1. Otimização de tokens para contexto longo
        if 'text' in context and len(context['text']) > 4000:
            prompt = self._optimize_for_long_context(prompt, context)
            
        # 2. Otimização para sumarização
        if 'task' in context and context['task'] == 'summarize':
            prompt = self._optimize_for_summarization(prompt, context)
            
        # 3. Remoção de redundâncias
        prompt = self._remove_redundancies(prompt)
        
        # 4. Injeção de contexto específico
        prompt = self._inject_context(prompt, context)
        
        return prompt
        
    def _optimize_for_long_context(self, prompt: str, context: Dict) -> str:
        """Otimiza prompt para contexto longo"""
        # Adiciona instruções específicas para lidar com texto longo
        optimization = """
        IMPORTANTE: O texto fornecido é extenso. Por favor:
        1. Foque nos pontos principais
        2. Use estrutura hierárquica se aplicável
        3. Mantenha a concisão sem perder informações críticas
        """
        
        return f"{prompt}\n{optimization}"
        
    def _optimize_for_summarization(self, prompt: str, context: Dict) -> str:
        """Otimizações específicas para sumarização"""
        # Adiciona parâmetros de sumarização baseados no contexto
        if 'summary_length' in context:
            length = context['summary_length']
            if length == 'short':
                addition = "Resuma em no máximo 3 frases concisas."
            elif length == 'medium':
                addition = "Resuma em 5-7 frases mantendo os pontos principais."
            else:
                addition = "Forneça um resumo detalhado com estrutura clara."
                
            return f"{prompt}\n{addition}"
            
        return prompt
        
    def _remove_redundancies(self, prompt: str) -> str:
        """Remove redundâncias e repetições do prompt"""
        # Remove linhas duplicadas
        lines = prompt.split('\n')
        unique_lines = []
        seen = set()
        
        for line in lines:
            line_stripped = line.strip()
            if line_stripped and line_stripped not in seen:
                seen.add(line_stripped)
                unique_lines.append(line)
                
        return '\n'.join(unique_lines)
        
    def _inject_context(self, prompt: str, context: Dict) -> str:
        """Injeta contexto específico no prompt"""
        # Substitui placeholders
        for key, value in context.items():
            if isinstance(value, str):
                placeholder = f"{{{key}}}"
                if placeholder in prompt:
                    prompt = prompt.replace(placeholder, value)
                    
        return prompt
        
    def get_default_prompt(self, prompt_type: PromptType, context: Dict) -> str:
        """Template padrão para cada tipo de prompt"""
        defaults = {
            PromptType.SUMMARIZATION: """
            Por favor, resuma o seguinte texto mantendo os pontos principais:
            
            {text}
            
            Forneça um resumo conciso e informativo.
            """,
            
            PromptType.CLASSIFICATION: """
            Classifique o seguinte texto nas categorias fornecidas:
            
            Texto: {text}
            Categorias: {categories}
            
            Retorne apenas o nome da categoria mais apropriada.
            """,
            
            PromptType.EXTRACTION: """
            Extraia as seguintes informações do texto:
            
            Texto: {text}
            Informações para extrair: {entities}
            
            Retorne no formato JSON especificado.
            """,
            
            PromptType.GENERATION: """
            Gere um texto sobre o seguinte tópico:
            
            Tópico: {topic}
            
            Requisitos: {requirements}
            """,
            
            PromptType.CONVERSATION: """
            Você é um assistente útil. Responda à seguinte mensagem:
            
            {message}
            
            Contexto da conversa: {conversation_history}
            """
        }
        
        template = defaults.get(prompt_type, "{text}")
        return self._inject_context(template, context)
        
    def record_prompt_usage(self, variant_id: str):
        """Registra uso da variante para estatísticas"""
        if variant_id in self.prompts:
            self.prompts[variant_id].call_count += 1
            
    def update_performance(self, variant_id: str, score: float, 
                          tokens_used: int, cost: float, success: bool):
        """Atualiza métricas de performance da variante"""
        if variant_id in self.prompts:
            variant = self.prompts[variant_id]
            
            # Atualiza score com média móvel
            variant.performance_score = (
                0.7 * variant.performance_score + 0.3 * score
            )
            
            # Calcula eficiência de tokens
            if tokens_used > 0:
                variant.token_efficiency = variant.token_efficiency * 0.9 + 0.1 * (1 / tokens_used)
                
            # Atualiza custo médio
            variant.cost_per_call = (
                0.9 * variant.cost_per_call + 0.1 * cost
            )
            
            # Atualiza taxa de sucesso
            success_value = 1.0 if success else 0.0
            variant.success_rate = (
                0.95 * variant.success_rate + 0.05 * success_value
            )
            
    def create_variant(self, base_variant_id: str, modifications: Dict[str, Any]) -> str:
        """Cria nova variante baseada em uma existente"""
        if base_variant_id not in self.prompts:
            raise ValueError(f"Variant {base_variant_id} not found")
            
        base = self.prompts[base_variant_id]
        
        # Gera ID único
        variant_hash = hashlib.md5(
            f"{base_variant_id}{datetime.now().isoformat()}".encode()
        ).hexdigest()[:8]
        
        new_id = f"{base.name}_{variant_hash}"
        
        # Cria nova variante
        new_variant = PromptVariant(
            id=new_id,
            name=f"{base.name} (Optimized)",
            prompt_template=modifications.get('template', base.prompt_template),
            prompt_type=base.prompt_type,
            version=f"{base.version}.1",
            parameters={**base.parameters, **modifications.get('parameters', {})},
            created_at=datetime.now(),
            is_active=True
        )
        
        self.prompts[new_id] = new_variant
        
        # Adiciona aos ativos
        if new_variant.prompt_type not in self.active_variants:
            self.active_variants[new_variant.prompt_type] = []
        self.active_variants[new_variant.prompt_type].append(new_id)
        
        return new_id
        
    def get_metrics_report(self) -> Dict[str, Any]:
        """Gera relatório de métricas de prompts"""
        report = {
            'timestamp': datetime.now().isoformat(),
            'total_variants': len(self.prompts),
            'active_variants': sum(len(v) for v in self.active_variants.values()),
            'by_type': {},
            'top_performers': []
        }
        
        # Agrupa por tipo
        for prompt_type in PromptType:
            variants = [v for v in self.prompts.values() 
                       if v.prompt_type == prompt_type and v.is_active]
            
            if variants:
                report['by_type'][prompt_type.value] = {
                    'count': len(variants),
                    'avg_score': sum(v.performance_score for v in variants) / len(variants),
                    'avg_cost': sum(v.cost_per_call for v in variants) / len(variants),
                    'total_calls': sum(v.call_count for v in variants)
                }
                
        # Top performers
        active_variants = [v for v in self.prompts.values() if v.is_active]
        top_performers = sorted(
            active_variants, 
            key=lambda v: v.performance_score, 
            reverse=True
        )[:5]
        
        report['top_performers'] = [
            {
                'id': v.id,
                'name': v.name,
                'type': v.prompt_type.value,
                'score': v.performance_score,
                'cost_per_call': v.cost_per_call,
                'calls': v.call_count,
                'success_rate': v.success_rate
            }
            for v in top_performers
        ]
        
        return report
        
    def save_config(self):
        """Salva configurações atualizadas"""
        config = {'prompts': []}
        
        for variant in self.prompts.values():
            prompt_data = {
                'id': variant.id,
                'name': variant.name,
                'template': variant.prompt_template,
                'type': variant.prompt_type.value,
                'version': variant.version,
                'parameters': variant.parameters,
                'created_at': variant.created_at.isoformat(),
                'is_active': variant.is_active
            }
            config['prompts'].append(prompt_data)
            
        with open(self.config_path, 'w') as f:
            yaml.dump(config, f, default_flow_style=False)
```

## 2. Rate Limiting Middleware

**src/security/middleware/rateLimiter.js**
```javascript
const rateLimit = require('express-rate-limit');
const RedisStore = require('rate-limit-redis');
const Redis = require('ioredis');
const securityConfig = require('../../config/securityConfig');

class RateLimiter {
  constructor() {
    this.redisClient = new Redis({
      host: process.env.REDIS_HOST || 'localhost',
      port: process.env.REDIS_PORT || 6379,
      password: process.env.REDIS_PASSWORD,
      enableOfflineQueue: false
    });

    this.limits = {};
    this.initializeLimits();
  }

  initializeLimits() {
    const { maxRequests, windowMs } = securityConfig.rateLimiting;

    // Limite geral para todas as rotas
    this.limits.general = rateLimit({
      store: new RedisStore({
        client: this.redisClient,
        prefix: 'rl:general:'
      }),
      windowMs,
      max: maxRequests.general,
      message: {
        error: 'Too many requests',
        retryAfter: '15 minutes'
      },
      standardHeaders: true,
      legacyHeaders: false,
      skip: (req) => req.ip === '127.0.0.1' // Skip localhost for testing
    });

    // Limite específico para autenticação
    this.limits.auth = rateLimit({
      store: new RedisStore({
        client: this.redisClient,
        prefix: 'rl:auth:'
      }),
      windowMs,
      max: maxRequests.auth,
      message: {
        error: 'Too many authentication attempts',
        retryAfter: '15 minutes'
      },
      skipSuccessfulRequests: true // Não conta tentativas bem-sucedidas
    });

    // Limite para API endpoints
    this.limits.api = rateLimit({
      store: new RedisStore({
        client: this.redisClient,
        prefix: 'rl:api:'
      }),
      windowMs,
      max: maxRequests.api,
      message: {
        error: 'API rate limit exceeded',
        retryAfter: '15 minutes'
      }
    });

    // Limite para endpoints administrativos
    this.limits.admin = rateLimit({
      store: new RedisStore({
        client: this.redisClient,
        prefix: 'rl:admin:'
      }),
      windowMs,
      max: maxRequests.admin,
      message: {
        error: 'Admin rate limit exceeded',
        retryAfter: '15 minutes'
      }
    });

    // Limite dinâmico baseado em complexidade da requisição
    this.limits.dynamic = rateLimit({
      store: new RedisStore({
        client: this.redisClient,
        prefix: 'rl:dynamic:'
      }),
      windowMs,
      max: (req) => {
        // Ajusta limite baseado na complexidade da requisição
        const complexity = this.calculateRequestComplexity(req);
        return Math.max(10, Math.floor(1000 / complexity));
      },
      keyGenerator: (req) => {
        return `${req.ip}:${req.path}:${req.user?.id || 'anonymous'}`;
      }
    });
  }

  calculateRequestComplexity(req) {
    let complexity = 1;
    
    // Aumenta complexidade para operações pesadas
    if (req.method === 'POST' || req.method === 'PUT') complexity *= 2;
    if (req.body && Object.keys(req.body).length > 10) complexity *= 1.5;
    if (req.query && Object.keys(req.query).length > 5) complexity *= 1.2;
    
    // Endpoints específicos com maior complexidade
    if (req.path.includes('/search')) complexity *= 3;
    if (req.path.includes('/report')) complexity *= 4;
    if (req.path.includes('/export')) complexity *= 5;
    
    return complexity;
  }

  // Middleware para IP blocking baseado em comportamento malicioso
  ipBlockingMiddleware() {
    return async (req, res, next) => {
      const ip = req.ip;
      const key = `blocked:ip:${ip}`;
      
      try {
        const isBlocked = await this.redisClient.get(key);
        if (isBlocked) {
          const ttl = await this.redisClient.ttl(key);
          return res.status(429).json({
            error: 'IP temporarily blocked',
            retryAfter: `${ttl} seconds`,
            reason: 'Suspicious activity detected'
          });
        }
        
        // Monitora comportamento suspeito
        await this.monitorSuspiciousActivity(req);
        next();
      } catch (error) {
        console.error('IP blocking error:', error);
        next();
      }
    };
  }

  async monitorSuspiciousActivity(req) {
    const ip = req.ip;
    const suspiciousKey = `suspicious:${ip}`;
    
    // Incrementa contador de atividades suspeitas
    const count = await this.redisClient.incr(suspiciousKey);
    
    // Define TTL de 1 hora
    if (count === 1) {
      await this.redisClient.expire(suspiciousKey, 3600);
    }
    
    // Bloqueia IP se exceder limite
    if (count > 10) {
      await this.redisClient.setex(`blocked:ip:${ip}`, 3600, 'true');
      await this.redisClient.del(suspiciousKey);
      
      // Log de segurança
      console.warn(`IP ${ip} blocked for 1 hour due to suspicious activity`);
    }
  }

  // Limpa limites para um IP específico (para admin)
  async clearLimits(ip) {
    const pattern = `rl:*:${ip}*`;
    const keys = await this.redisClient.keys(pattern);
    
    if (keys.length > 0) {
      await this.redisClient.del(...keys);
    }
    
    // Remove bloqueios
    await this.redisClient.del(`blocked:ip:${ip}`);
    await this.redisClient.del(`suspicious:${ip}`);
    
    return keys.length;
  }

  getMiddleware(type = 'general') {
    return this.limits[type] || this.limits.general;
  }
}
```

## Componentes Principais

### 1. API Gateway (Kong)
- **Função**: Roteamento, autenticação, rate limiting
- **Configuração**: Plugin JWT, CORS, logging
- **Monitoramento**: Métricas em tempo real

### 2. Service Mesh (Istio)
- **Função**: Comunicação entre serviços, observabilidade
- **Recursos**:
  - Circuit breaking
  - Retry policies
  - Distributed tracing
  - Load balancing inteligente

### 3. Microserviços

#### Users Service
- Gestão de usuários e autenticação
- RBAC (Role-Based Access Control)
- Sessions e tokens JWT

#### Flows Service
- Execução de workflows
- Engine de automação
- Estado dos fluxos

#### Integrations Service
- Conexão com APIs externas
- WhatsApp Business, Email, CRM
- Webhooks e callbacks

#### AI Service
- Processamento de linguagem natural
- Classificação de intenções
- Geração de respostas inteligentes

#### Analytics Service
- Coleta e análise de métricas
- Dashboards em tempo real
- Relatórios personalizados

#### Notifications Service
- Envio de notificações
- Multi-canal (email, push, SMS)
- Agendamento e priorização

### 4. Message Queue (RabbitMQ)
- **Função**: Comunicação assíncrona entre serviços
- **Exchanges**:
  - `flows.execution` - Execução de fluxos
  - `integrations
