# DIYAPP Evolution - V12 Core - Arquitetura Técnica

## ADR-001: Arquitetura V12 - Padrões de Código e Estrutura Modular

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead V12

### CONTEXTO:
A versão V12 do DIYAPP precisa evoluir para suportar:
1. 100% de autonomia operacional
2. Refatoração paralela via Modo Hive
3. Health checks distribuídos
4. Manutenção de estabilidade durante evolução contínua
5. Integração com múltiplos provedores de IA (OpenAI, Anthropic, Local)

### DECISÃO:
Implementar arquitetura hexagonal com módulos independentes, comunicação via eventos e health checks centralizados. Cada módulo será um microsserviço lógico dentro do monorepo.

### OPÇÕES CONSIDERADAS:
- **Opção A:** Monolito modularizado - Prós: Simplicidade inicial, deploy único. Contras: Acoplamento alto, difícil refatoração paralela.
- **Opção B:** Microsserviços completos - Prós: Isolamento total. Contras: Complexidade operacional, latência de rede.
- **Opção C:** Arquitetura hexagonal com módulos independentes - Prós: Isolamento lógico, fácil testabilidade, suporte a refatoração paralela. Contras: Curva de aprendizado.

**Opção escolhida:** C - Justificativa: Balanceia isolamento com simplicidade operacional, ideal para Modo Hive.

### CONSEQUÊNCIAS:
**Positivas:** 
- Módulos podem ser refatorados independentemente
- Health checks por módulo
- Testabilidade individual
- Troca de provedores de IA sem impacto no core

**Negativas:**
- Overhead de comunicação entre módulos
- Necessidade de contrato de interfaces bem definido

**Riscos:**
- Complexidade excessiva se mal implementado
- Mitigação: Documentação clara e exemplos de implementação

## Health Checks Distribuídos

```typescript
// src/infrastructure/monitoring/health-check.ts
export interface HealthCheck {
  name: string;
  type: 'internal' | 'external' | 'dependency';
  critical: boolean;
  check(): Promise<HealthStatus>;
}

export interface HealthStatus {
  status: 'healthy' | 'degraded' | 'unhealthy';
  details?: Record<string, unknown>;
  timestamp: string;
  responseTime?: number;
}

export class HealthCheckRegistry {
  private checks: Map<string, HealthCheck> = new Map();
  
  register(check: HealthCheck): void {
    this.checks.set(check.name, check);
  }
  
  async runAll(): Promise<Map<string, HealthStatus>> {
    const results = new Map();
    
    for (const [name, check] of this.checks) {
      const startTime = Date.now();
      try {
        const status = await check.check();
        status.responseTime = Date.now() - startTime;
        results.set(name, status);
      } catch (error) {
        results.set(name, {
          status: 'unhealthy',
          details: { error: error.message },
          timestamp: new Date().toISOString(),
          responseTime: Date.now() - startTime
        });
      }
    }
    
    return results;
  }
  
  async getOverallStatus(): Promise<{
    status: 'healthy' | 'degraded' | 'unhealthy';
    checks: Map<string, HealthStatus>;
    timestamp: string;
  }> {
    const checks = await this.runAll();
    let hasUnhealthy = false;
    let hasDegraded = false;
    
    for (const status of checks.values()) {
      if (status.status === 'unhealthy') hasUnhealthy = true;
      if (status.status === 'degraded') hasDegraded = true;
    }
    
    return {
      status: hasUnhealthy ? 'unhealthy' : hasDegraded ? 'degraded' : 'healthy',
      checks,
      timestamp: new Date().toISOString()
    };
  }
}

// Exemplo de health check para módulo AI
export class AIProviderHealthCheck implements HealthCheck {
  name = 'ai-provider-openai';
  type = 'external' as const;
  critical = true;
  
  constructor(private provider: AIProvider) {}
  
  async check(): Promise<HealthStatus> {
    try {
      const startTime = Date.now();
      const response = await this.provider.healthCheck();
      const responseTime = Date.now() - startTime;
      
      return {
        status: response.healthy ? 'healthy' : 'degraded',
        details: {
          model: response.model,
          provider: response.provider,
          responseTime
        },
        timestamp: new Date().toISOString()
      };
    } catch (error) {
      return {
        status: 'unhealthy',
        details: { error: error.message },
        timestamp: new Date().toISOString()
      };
    }
  }
}
```

```javascript
export const accessibility = {
  /**
   * Verifica contraste WCAG AA
   * @param {string} foreground - Cor do texto
   * @param {string} background - Cor do fundo
   * @returns {boolean} Passa no WCAG AA
   */
  checkContrast: (foreground, background) => {
    // Implementação simplificada - em produção usar biblioteca como chroma-js
    const luminance = (color) => {
      const rgb = color.replace('#', '').match(/.{2}/g);
      const [r, g, b] = rgb.map(c => parseInt(c, 16) / 255);
      const [rs, gs, bs] = [r, g, b].map(c => 
        c <= 0.03928 ? c / 12.92 : Math.pow((c + 0.055) / 1.055, 2.4)
      );
      return 0.2126 * rs + 0.7152 * gs + 0.0722 * bs;
    };
    
    const l1 = luminance(foreground);
    const l2 = luminance(background);
    const contrast = (Math.max(l1, l2) + 0.05) / (Math.min(l1, l2) + 0.05);
    return contrast >= 4.5; // WCAG AA mínimo para texto normal
  },
  
  /**
   * Gera gradiente de foco acessível
   */
  focusRing: {
    outline: '2px solid transparent',
    outlineOffset: '2px',
    boxShadow: '0 0 0 2px #FFFFFF, 0 0 0 4px #1890FF'
  }
};
```

### index.js (tokens)
```javascript
/**
 * Design Tokens - Exportação principal
 */

import { colors } from './colors';
import { typography } from './typography';
import { spacing, layout } from './spacing';
import { elevation } from './elevation';

export const designTokens = {
  colors,
  typography,
  spacing,
  layout,
  elevation,
  
  // Funções de utilidade
  getToken: (category, path) => {
    const parts = path.split('.');
    let value = designTokens[category];
    
    for (const part of parts) {
      if (value && value[part] !== undefined) {
        value = value[part];
      } else {
        console.warn(`Token não encontrado: ${category}.${path}`);
        return null;
      }
    }
    
    return value;
  },
  
  // CSS custom properties generator
  generateCSSVariables: () => {
    const variables = {};
    
    // Colors
    Object.entries(colors).forEach(([category, values]) => {
      if (typeof values === 'object') {
        Object.entries(values).forEach(([key, value]) => {
          variables[`--color-${category}-${key}`] = value;
        });
      }
    });
    
    // Typography
    Object.entries(typography.fontSize).forEach(([key, value]) => {
      variables[`--font-size-${key}`] = value;
    });
    
    // Spacing
    Object.entries(spacing).forEach(([key, value]) => {
      variables[`--spacing-${key}`] = value;
    });
    
    return variables;
  }
};

export default designTokens;
```

class StructuredJSONFormatter(logging.Formatter):
    """Formatter para logs estruturados em JSON"""
    
    def format(self, record: logging.LogRecord) -> str:
        log_data = {
            "timestamp": datetime.utcnow().isoformat() + "Z",
            "level": record.levelname,
            "service": SERVICE_NAME,
            "environment": ENV.value,
            "correlation_id": getattr(record, 'correlation_id', 'N/A'),
            "user_id": getattr(record, 'user_id', 'N/A'),
            "endpoint": getattr(record, 'endpoint', 'N/A'),
            "method": getattr(record, 'method', 'N/A'),
            "duration_ms": getattr(record, 'duration_ms', 0),
            "status_code": getattr(record, 'status_code', 0),
            "message": record.getMessage(),
            "module": record.module,
            "function": record.funcName,
            "line": record.lineno
        }
        
        # Adiciona exception info se existir
        if record.exc_info:
            log_data["exception"] = self.formatException(record.exc_info)
        
        # Mascara dados sensíveis
        self._mask_sensitive_data(log_data)
        
        return json.dumps(log_data)
    
    def _mask_sensitive_data(self, log_data: Dict) -> None:
        """Mascara dados sensíveis nos logs"""
        sensitive_fields = ['email', 'cpf', 'password', 'token', 'credit_card', 'api_key']
        
        if isinstance(log_data.get('message'), str):
            for field in sensitive_fields:
                if field in log_data['message'].lower():
                    log_data['message'] = log_data['message'].replace(
                        field, f"[MASKED_{field.upper()}]"
                    )

def setup_logging():
    """Configura logging estruturado"""
    
    # Remove handlers padrão
    for handler in logging.root.handlers[:]:
        logging.root.removeHandler(handler)
    
    # Configura logger principal
    logger = logging.getLogger()
    logger.setLevel(logging.INFO if ENV == Environment.PRODUCTION else logging.DEBUG)
    
    # Handler para console (desenvolvimento)
    console_handler = logging.StreamHandler()
    console_handler.setFormatter(StructuredJSONFormatter())
    logger.addHandler(console_handler)
    
    # Handler para arquivo (produção)
    if ENV == Environment.PRODUCTION:
        file_handler = RotatingFileHandler(
            filename=f"/var/log/{SERVICE_NAME}/app.log",
            maxBytes=10485760,  # 10MB
            backupCount=10
        )
        file_handler.setFormatter(StructuredJSONFormatter())
        logger.addHandler(file_handler)
    
    # Logger específico do serviço
    return logging.getLogger(SERVICE_NAME)

class DatabasePool:
    """Pool de conexões PostgreSQL com circuit breaker"""
    
    _instance = None
    _pool = None
    
    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
        return cls._instance
    
    @circuit(failure_threshold=5, recovery_timeout=60)
    def initialize(self):
        """Inicializa pool de conexões"""
        try:
            self._pool = SimpleConnectionPool(
                minconn=1,
                maxconn=20,
                host=os.getenv("DB_HOST", "localhost"),
                port=os.getenv("DB_PORT", "5432"),
                database=os.getenv("DB_NAME", "diyapp"),
                user=os.getenv("DB_USER", "postgres"),
                password=os.getenv("DB_PASSWORD", ""),
                cursor_factory=RealDictCursor
            )
            logger.info("Database pool initialized successfully")
        except Exception as e:
            logger.error(f"Failed to initialize database pool: {str(e)}")
            raise
    
    @contextmanager
    def get_connection(self):
        """Obtém conexão do pool com timeout"""
        if not self._pool:
            self.initialize()
        
        conn = None
        try:
            conn = self._pool.getconn(timeout=5)
            yield conn
        except psycopg2.OperationalError as e:
            logger.error(f"Database connection error: {str(e)}")
            raise HTTPException(
                status_code=status.HTTP_503_SERVICE_UNAVAILABLE,
                detail="Database temporarily unavailable"
            )
        finally:
            if conn:
                self._pool.putconn(conn)
    
    def close_all(self):
        """Fecha todas as conexões do pool"""
        if self._pool:
            self._pool.closeall()
            logger.info("Database pool closed")

class RedisCache:
    """Cliente Redis com fallback e circuit breaker"""
    
    def __init__(self):
        self._client = None
        self._circuit_open = False
        self._last_failure = None
    
    @circuit(failure_threshold=3, recovery_timeout=30)
    def connect(self):
        """Conecta ao Redis"""
        try:
            self._client = redis.Redis(
                host=os.getenv("REDIS_HOST", "localhost"),
                port=int(os.getenv("REDIS_PORT", "6379")),
                password=os.getenv("REDIS_PASSWORD", None),
                decode_responses=True,
                socket_timeout=5,
                socket_connect_timeout=5,
                retry_on_timeout=True,
                max_connections=50
            )
            # Testa conexão
            self._client.ping()
            logger.info("Redis connected successfully")
        except RedisError as e:
            logger.error(f"Redis connection error: {str(e)}")
            self._circuit_open = True
            self._last_failure = time.time()
            raise
    
    def get(self, key: str) -> Optional[Any]:
        """Obtém valor do cache com fallback silencioso"""
        if self._circuit_open:
            # Circuit breaker aberto - retorna None sem tentar
            if time.time() - self._last_failure > 30:  # Recovery timeout
                self._circuit_open = False
            return None
        
        if not self._client:
            try:
                self.connect()
            except:
                return None
        
        try:
            value = self._client.get(key)
            if value:
                return json.loads(value)
            return None
        except (RedisError, json.JSONDecodeError) as e:
            logger.warning(f"Cache get failed for key {key}: {str(e)}")
            self._circuit_open = True
            self._last_failure = time.time()
            return None
    
    def set(self, key: str, value: Any, ttl: int = CACHE_TTL) -> bool:
        """Define valor no cache com fallback silencioso"""
        if self._circuit_open:
            return False
        
        if not self._client:
            try:
                self.connect()
            except:
                return False
        
        try:
            serialized = json.dumps(jsonable_encoder(value))
            return self._client.setex(key, ttl, serialized)
        except (RedisError, TypeError) as e:
            logger.warning(f"Cache set failed for key {key}: {str(e)}")
            self._circuit_open = True
            self._last_failure = time.time()
            return False
    
    def delete(self, key: str) -> bool:
        """Remove valor do cache"""
        if not self._client or self._circuit_open:
            return False
        
        try:
            return bool(self._client.delete(key))
        except RedisError:
            return False

async def correlation_middleware(request: Request, call_next):
    """Middleware para adicionar correlation ID a cada request"""
    correlation_id = request.headers.get('X-Correlation-ID', str(uuid.uuid4()))
    
    # Adiciona ao state do request
    request.state.correlation_id = correlation_id
    
    # Mede tempo de execução
    start_time = time.time()
    
    try:
        response = await call_next(request)
        duration_ms = int((time.time() - start_time) * 1000)
        
        # Adiciona correlation ID ao header da resposta
        response.headers['X-Correlation-ID'] = correlation_id
        
        # Log estruturado
        logger.info(
            f"Request completed: {request.method} {request.url.path}",
            extra={
                'correlation_id': correlation_id,
                'endpoint': request.url.path,
                'method': request.method,
                'duration_ms': duration_ms,
                'status_code': response.status_code,
                'user_id': getattr(request.state, 'user_id', 'N/A')
            }
        )
        
        return response
        
    except Exception as e:
        duration_ms = int((time.time() - start_time) * 1000)
        logger.error(
            f"Request failed: {str(e)}",
            extra={
                'correlation_id': correlation_id,
                'endpoint': request.url.path,
                'method': request.method,
                'duration_ms': duration_ms,
                'status_code': 500,
                'exception': str(e)
            },
            exc_info=True
        )
        raise

REQUEST_LATENCY = Histogram(

    ['method', 'endpoint', 'status']

# Contador de erros
ERROR_COUNT = Counter(
    'http_errors_total',
    'Total HTTP errors',
    ['method', 'endpoint', 'error_type']
)

async def metrics_middleware(request: Request, call_next):
    """Middleware para coletar métricas"""
    start_time = time.time()
    
    try:
        response = await call_next(request)
        duration = time.time() - start_time
        
        # Registra métricas
        REQUEST_LATENCY.labels(
            method=request.method,
            endpoint=request.url.path,
            status=response.status_code
        ).observe(duration)
        
        REQUEST_COUNT.labels(
            method=request.method,
            endpoint=request.url.path,
            status=response.status_code
        ).inc()
        
        return response
        
    except HTTPException as e:
        duration = time.time() - start_time
        ERROR_COUNT.labels(
            method=request.method,
            endpoint=request.url.path,
            error_type=f"http_{e.status_code}"
        ).inc()
        raise
        
    except Exception as e:
        duration = time.time() - start_time
        ERROR_COUNT.labels(
            method=request.method,
            endpoint=request.url.path,
            error_type="internal_error"
        ).inc()
        raise

    dependencies = {}
    
    # Verifica banco de dados
    try:
        with db.cursor() as cursor:
            cursor.execute("SELECT 1")
            dependencies["database"] = "healthy"
    except Exception as e:
        logger.error(f"Database health check failed: {str(e)}")
        dependencies["database"] = "unhealthy"
    
    # Verifica cache
    try:
        test_key = f"health_check_{int(time.time())}"
        cache.set(test_key, {"test": True}, ttl=10)
        cached = cache.get(test_key)
        dependencies["cache"] = "healthy" if cached else "unhealthy"
    except Exception as e:
        logger.error(f"Cache health check failed: {str(e)}")
        dependencies["cache"] = "unhealthy"
    
    # Coleta métricas
    metrics = {
        "uptime": time.time() - app.startup_time,
        "memory_usage": os.getpid(),
        "active_requests": len(request.app.state.requests) if hasattr(request.app.state, 'requests') else 0
    }
    
    # Determina status geral
    overall_status = "healthy" if all(v == "healthy" for v in dependencies.values()) else "degraded"
    
    return HealthCheckResponse(
        status=overall_status,
        service=SERVICE_NAME,
        version=VERSION,
        timestamp=datetime.utcnow().isoformat() + "Z",
        dependencies=dependencies,
        metrics=metrics
    )

    current_month DATE;

    partition_name TEXT;
    start_date DATE;

            'CREATE TABLE IF NOT EXISTS %I PARTITION OF events_v12
            FOR VALUES FROM (%L) TO (%L)',

            current_month,
            next_month

        );
        
        EXECUTE format(
            'CREATE INDEX IF NOT EXISTS idx_%s_properties_gin

        );

END;

-- 2.3 PROCEDURE DE BACKUP AUTOMATIZADO
-- ------------------------------------

CREATE OR REPLACE PROCEDURE execute_scheduled_backup()
LANGUAGE plpgsql
AS $$
DECLARE
    config_record RECORD;
    backup_record_id BIGINT;
    backup_start_time TIMESTAMPTZ := NOW();
    backup_file_path TEXT;
BEGIN
    -- Busca configurações ativas que precisam ser executadas
    FOR config_record IN 
        SELECT * FROM backup_config_v12 
        WHERE enabled = true 
        AND (next_run_at IS NULL OR next_run_at <= NOW())
    LOOP
        -- Insere registro no histórico
        INSERT INTO backup_history_v12 (config_id, backup_type, started_at)
        VALUES (config_record.config_id, config_record.backup_type, backup_start_time)
        RETURNING backup_id INTO backup_record_id;
        
        BEGIN
            -- Executa backup baseado no tipo
            CASE config_record.backup_type
                WHEN 'full' THEN
                    -- Backup completo usando pg_dump (simulação)
                    backup_file_path := '/backups/full/diyapp_' || 
                        TO_CHAR(backup_start_time, 'YYYYMMDD_HH24MISS') || '.sql';
                    
                    -- Atualiza histórico como sucesso
                    UPDATE backup_history_v12 
                    SET status = 'completed',
                        completed_at = NOW(),
                        file_path = backup_file_path,
                        size_bytes = 0, -- Seria preenchido pelo pg_dump real
                        checksum = MD5(backup_file_path || backup_start_time::text)
                    WHERE backup_id = backup_record_id;
                    
                WHEN 'incremental' THEN
                    -- Backup incremental (apenas dados recentes)
                    backup_file_path := '/backups/incremental/diyapp_inc_' || 
                        TO_CHAR(backup_start_time, 'YYYYMMDD_HH24MISS') || '.sql';
                    
                    -- Atualiza histórico
                    UPDATE backup_history_v12 
                    SET status = 'completed',
                        completed_at = NOW(),
                        file_path = backup_file_path,
                        size_bytes = 0
                    WHERE backup_id = backup_record_id;
                    
                WHEN 'log' THEN
                    -- Backup de logs de transação
                    backup_file_path := '/backups/wal/diyapp_wal_' || 
                        TO_CHAR(backup_start_time, 'YYYYMMDD_HH24MISS') || '.tar';
                    
                    UPDATE backup_history_v12 
                    SET status = 'completed',
                        completed_at = NOW(),
                        file_path = backup_file_path
                    WHERE backup_id = backup_record_id;
            END CASE;
            
            -- Atualiza próximo horário de execução
            UPDATE backup_config_v12 
            SET last_run_at = backup_start_time,
                next_run_at = backup_start_time + 
                    (CASE 
                        WHEN schedule_cron LIKE '%@daily%' THEN INTERVAL '1 day'
                        WHEN schedule_cron LIKE '%@hourly%' THEN INTERVAL '1 hour'
                        WHEN schedule_cron LIKE '%@weekly%' THEN INTERVAL '7 days'
                        ELSE INTERVAL '1 day'
                    END)
            WHERE config_id = config_record.config_id;
            
        EXCEPTION WHEN OTHERS THEN
            -- Registra falha
            UPDATE backup_history_v12 
            SET status = 'failed',
                completed_at = NOW(),
                error_message = SQLERRM
            WHERE backup_id = backup_record_id;
            
            RAISE NOTICE 'Backup failed for config %: %', 
                config_record.config_id, SQLERRM;
        END;
    END LOOP;
    
    -- Limpeza automática de backups antigos
    PERFORM cleanup_old_backups();
END;
$$;

-- 2.4 PROCEDURE DE LIMPEZA DE BACKUPS ANTIGOS
-- ------------------------------------------

CREATE OR REPLACE PROCEDURE cleanup_old_backups()
LANGUAGE plpgsql
AS $$
DECLARE
    retention_config RECORD;
    delete_count INT := 0;
BEGIN
    -- Para cada configuração de retenção
    FOR retention_config IN 
        SELECT DISTINCT retention_days 
        FROM backup_config_v12 
        WHERE enabled = true
    LOOP
        -- Remove registros de histórico antigos
        DELETE FROM backup_history_v12 
        WHERE completed_at < NOW() - (retention_config.retention_days || ' days')::interval
        AND status = 'completed'
        AND backup_id IN (
            SELECT backup_id 
            FROM backup_history_v12 bh
            JOIN backup_config_v12 bc ON bh.config_id = bc.config_id
            WHERE bc.retention_days = retention_config.retention_days
        );
        
        GET DIAGNOSTICS delete_count = ROW_COUNT;
        
        RAISE NOTICE 'Cleaned up % old backup records for % days retention', 
            delete_count, retention_config.retention_days;
    END LOOP;
END;
$$;

CREATE OR REPLACE FUNCTION update_updated_at_column()
RETURNS TRIGGER AS $$
BEGIN
    NEW.updated_at = NOW();
    RETURN NEW;
END;
$$ LANGUAGE plpgsql;

-- Aplica trigger às tabelas principais
DO $$ 
DECLARE
    tbl_name TEXT;
BEGIN
    FOR tbl_name IN 
        SELECT table_name 
        FROM information_schema.tables 
        WHERE table_schema = 'public' 
        AND table_name LIKE '%_v12'
        AND table_name NOT IN ('backup_history_v12', 'backup_config_v12')
    LOOP
        EXECUTE format('
            DROP TRIGGER IF EXISTS update_%s_timestamp ON %I;
            CREATE TRIGGER update_%s_timestamp
            BEFORE UPDATE ON %I
            FOR EACH ROW
            EXECUTE FUNCTION update_updated_at_column();
        ', tbl_name, tbl_name, tbl_name, tbl_name);
    END LOOP;
END $$;

-- 3.1 VIEW PARA MONITORAMENTO DE PERFORMANCE
-- -----------------------------------------

CREATE OR REPLACE VIEW database_performance_v12 AS
SELECT 
    schemaname,
    tablename,
    pg_size_pretty(pg_total_relation_size(schemaname || '.' || tablename)) as total_size,
    pg_size_pretty(pg_relation_size(schemaname || '.' || tablename)) as table_size,
    pg_size_pretty(pg_total_relation_size(schemaname || '.' || tablename) - 
                   pg_relation_size(schemaname || '.' || tablename)) as index_size,
    n_live_tup as live_rows,
    n_dead_tup as dead_rows,
    CASE 
        WHEN n_live_tup > 0 THEN ROUND((n_dead_tup::numeric / n_live_tup) * 100, 2)
        ELSE 0
    END as dead_row_percentage,
    last_vacuum,
    last_autovacuum,
    last_analyze,
    last_autoanalyze
FROM pg_stat_user_tables
WHERE schemaname = 'public'
ORDER BY pg_total_relation_size(schemaname || '.' || tablename) DESC;

-- 3.2 PROCEDURE DE MANUTENÇÃO AUTOMÁTICA
-- --------------------------------------

CREATE OR REPLACE PROCEDURE perform_maintenance()
LANGUAGE plpgsql
AS $$
DECLARE
    table_record RECORD;
BEGIN
    -- Para cada tabela com mais de 10% de dead rows
    FOR table_record IN 
        SELECT tablename, dead_row_percentage
        FROM database_performance_v12
        WHERE dead_row_percentage > 10
        AND tablename LIKE '%_v12'
    LOOP
        RAISE NOTICE 'Vacuuming table % (dead rows: %%)', 
            table_record.tablename, table_record.dead_row_percentage;
        
        EXECUTE format('VACUUM ANALYZE %I', table_record.tablename);
    END LOOP;
    
    -- Atualiza materialized views
    RAISE NOTICE 'Refreshing materialized views...';
    REFRESH MATERIALIZED VIEW CONCURRENTLY realtime_metrics_v12;
    
    -- Cria novas partições se necessário
    PERFORM create_event_partitions();
    
    RAISE NOTICE 'Maintenance completed successfully';
END;
$$;

```bash
#!/bin/bash
# Backup PostgreSQL para S3 (multi-região)
DATE=$(date +%Y-%m-%d_%H-%M-%S)
PGPASSWORD=$DB_PASSWORD pg_dump -h $DB_HOST -U $DB_USER -d $DB_NAME | gzip > /tmp/diyapp-backup-$DATE.sql.gz

# Upload para AWS S3 (us-east-1 e us-west-2)
aws s3 cp /tmp/diyapp-backup-$DATE.sql.gz s3://diyapp-backups-us-east-1/$DATE/
aws s3 cp /tmp/diyapp-backup-$DATE.sql.gz s3://diyapp-backups-us-west-2/$DATE/

# Limpeza
rm /tmp/diyapp-backup-$DATE.sql.gz
```

### Implementação de Autenticação JWT/OAuth

#### Arquivo: `src/controllers/authController.js`

```javascript
const jwt = require('jsonwebtoken');
const { OAuth2Client } = require('google-auth-library'); // Para OAuth exemplo
const client = new OAuth2Client(process.env.GOOGLE_CLIENT_ID);

exports.login = async (req, res) => {
    const { email, password } = req.body;
    // Validação do usuário e senha no banco de dados
    const token = jwt.sign({ email }, process.env.JWT_SECRET, { expiresIn: '1h' });
    res.json({ token });
};

exports.googleLogin = async (req, res) => {
    const { tokenId } = req.body;
    const ticket = await client.verifyIdToken({
        idToken: tokenId,
        audience: process.env.GOOGLE_CLIENT_ID,
    });
    const { email } = ticket.getPayload();
    const token = jwt.sign({ email }, process.env.JWT_SECRET, { expiresIn: '1h' });
    res.json({ token });
};
```

```javascript
    }
```

```typescript
import { evaluate } from 'promptfoo';
import fs from 'fs';

const goldenDataset = JSON.parse(fs.readFileSync('tests/llm/golden-dataset.json', 'utf8'));

async function runGoldenDataset() {
  const results = await evaluate({
    prompts: goldenDataset.prompts,
    providers: ['http://localhost:3000/api/llm'],
    tests: goldenDataset.tests,
  });

  const drift = results.stats.passed / results.stats.total;
  if (drift < 0.9) {
    throw new Error(`Drift detectado: ${(1 - drift) * 100}% de degradação`);
  }

  fs.writeFileSync('reports/llm-drift.json', JSON.stringify(results, null, 2));
}

runGoldenDataset();
```

```bash
BACKUP_DIR="/backups/postgres"

pg_dump -h postgres-service -U admin -d diyapp > $BACKUP_FILE

# Upload para S3
aws s3 cp $BACKUP_FILE.gz s3://diyapp-backups/postgres/
```

```bash

#!/bin/bash

# Download do backup
aws s3 cp $S3_PATH /tmp/$BACKUP_FILE

# Descompressão

# Restauração
psql -h postgres-service -U admin -d diyapp < /tmp/${BACKUP_FILE%.gz}
```

### auth.service.js
```javascript
const jwt = require('jsonwebtoken');
const bcrypt = require('bcryptjs');
const { OAuth2Client } = require('google-auth-library');
const crypto = require('crypto');
const User = require('../models/User');
const logger = require('../utils/logger');

class AuthService {
  constructor() {
    this.jwtSecret = process.env.JWT_SECRET || crypto.randomBytes(64).toString('hex');
    this.jwtExpiresIn = process.env.JWT_EXPIRES_IN || '15m';
    this.refreshExpiresIn = process.env.REFRESH_EXPIRES_IN || '7d';
    this.googleClient = new OAuth2Client(process.env.GOOGLE_CLIENT_ID);
  }

  async generateTokens(userId, roles) {
    const accessToken = jwt.sign(
      { sub: userId, roles },
      this.jwtSecret,
      { expiresIn: this.jwtExpiresIn }
    );

    const refreshToken = jwt.sign(
      { sub: userId },
      this.jwtSecret,
      { expiresIn: this.refreshExpiresIn }
    );

    return { accessToken, refreshToken };
  }

  async verifyToken(token) {
    try {
      return jwt.verify(token, this.jwtSecret);
    } catch (err) {
      logger.error(`Token verification failed: ${err.message}`);
      return null;
    }
  }

  async verifyGoogleToken(token) {
    try {
      const ticket = await this.googleClient.verifyIdToken({
        idToken: token,
        audience: process.env.GOOGLE_CLIENT_ID
      });
      return ticket.getPayload();
    } catch (err) {
      logger.error(`Google token verification failed: ${err.message}`);
      return null;
    }
  }

  async hashPassword(password) {
    return bcrypt.hash(password, 12);
  }

  async comparePasswords(password, hashedPassword) {
    return bcrypt.compare(password, hashedPassword);
  }

  async invalidateToken(token) {
    // In production, implement a token blacklist or short-lived tokens
    logger.info(`Token invalidated: ${token.substring(0, 10)}...`);
    return true;
  }
}

module.exports = new AuthService();
```

```javascript
          }
```

### auth.controller.js
```javascript
const authService = require('./auth.service');
const User = require('../models/User');
const logger = require('../utils/logger');
const { UnauthorizedError, BadRequestError } = require('../utils/errors');

class AuthController {
  async login(req, res, next) {
    try {
      const { email, password } = req.body;

      const user = await User.findOne({ email });
      if (!user) {
        throw new UnauthorizedError('Invalid credentials');
      }

      const isMatch = await authService.comparePasswords(password, user.password);
      if (!isMatch) {
        throw new UnauthorizedError('Invalid credentials');
      }

      const { accessToken, refreshToken } = await authService.generateTokens(
        user._id,
        user.roles
      );

      res.cookie('refreshToken', refreshToken, {
        httpOnly: true,
        secure: process.env.NODE_ENV === 'production',
        sameSite: 'strict',
        maxAge: 7 * 24 * 60 * 60 * 1000 // 7 days
      });

      res.json({
        accessToken,
        user: {
          id: user._id,
          email: user.email,
          roles: user.roles
        }
      });
    } catch (err) {
      next(err);
    }
  }

  async register(req, res, next) {
    try {
      const { email, password, name } = req.body;

      const existingUser = await User.findOne({ email });
      if (existingUser) {
        throw new BadRequestError('Email already in use');
      }

      const hashedPassword = await authService.hashPassword(password);
      const user = new User({
        email,
        password: hashedPassword,
        name,
        roles: ['user']
      });

      await user.save();

      const { accessToken, refreshToken } = await authService.generateTokens(
        user._id,
        user.roles
      );

      res.cookie('refreshToken', refreshToken, {
        httpOnly: true,
        secure: process.env.NODE_ENV === 'production',
        sameSite: 'strict',
        maxAge: 7 * 24 * 60 * 60 * 1000 // 7 days
      });

      res.status(201).json({
        accessToken,
        user: {
          id: user._id,
          email: user.email,
          roles: user.roles
        }
      });
    } catch (err) {
      next(err);
    }
  }

  async refreshToken(req, res, next) {
    try {
      const refreshToken = req.cookies.refreshToken;
      if (!refreshToken) {
        throw new UnauthorizedError('Refresh token required');
      }

      const decoded = await authService.verifyToken(refreshToken);
      if (!decoded) {
        throw new UnauthorizedError('Invalid refresh token');
      }

      const user = await User.findById(decoded.sub);
      if (!user) {
        throw new UnauthorizedError('User not found');
      }

      const { accessToken, newRefreshToken } = await authService.generateTokens(
        user._id,
        user.roles
      );

      res.cookie('refreshToken', newRefreshToken, {
        httpOnly: true,
        secure: process.env.NODE_ENV === 'production',
        sameSite: 'strict',
        maxAge: 7 * 24 * 60 * 60 * 1000 // 7 days
      });

      res.json({ accessToken });
    } catch (err) {
      next(err);
    }
  }

  async logout(req, res, next) {
    try {
      const refreshToken = req.cookies.refreshToken;
      if (refreshToken) {
        await authService.invalidateToken(refreshToken);
      }

      res.clearCookie('refreshToken');
      res.json({ message: 'Logged out successfully' });
    } catch (err) {
      next(err);
    }
  }

  async googleAuth(req, res, next) {
    try {
      const { token } = req.body;
      const payload = await authService.verifyGoogleToken(token);

      if (!payload) {
        throw new UnauthorizedError('Invalid Google token');
      }

      let user = await User.findOne({ email: payload.email });

      if (!user) {
        user = new User({
          email: payload.email,
          name: payload.name,
          googleId: payload.sub,
          roles: ['user']
        });
        await user.save();
      }

      const { accessToken, refreshToken } = await authService.generateTokens(
        user._id,
        user.roles
      );

      res.cookie('refreshToken', refreshToken, {
        httpOnly: true,
        secure: process.env.NODE_ENV === 'production',
        sameSite: 'strict',
        maxAge: 7 * 24 * 60 * 60 * 1000 // 7 days
      });

      res.json({
        accessToken,
        user: {
          id: user._id,
          email: user.email,
          roles: user.roles
        }
      });
    } catch (err) {
      next(err);
    }
  }

  async getCurrentUser(req, res, next) {
    try {
      const user = await User.findById(req.user.id).select('-password');
      if (!user) {
        throw new UnauthorizedError('User not found');
      }

      res.json(user);
    } catch (err) {
      next(err);
    }
  }
}
```

### utils/sanitizer.js
```javascript
const { JSDOM } = require('jsdom');
const createDOMPurify = require('dompurify');
const { body, validationResult } = require('express-validator');
const logger = require('./logger');

const window = new JSDOM('').window;
const DOMPurify = createDOMPurify(window);

class Sanitizer {
  sanitizeHTML(dirty) {
    return DOMPurify.sanitize(dirty, {
      ALLOWED_TAGS: ['b', 'i', 'em', 'strong', 'a', 'code', 'pre', 'p', 'br', 'ul', 'ol', 'li'],
      ALLOWED_ATTR: ['href', 'title', 'target'],
      TARGET_BLANK: true
    });
  }

  sanitizeInput(input) {
    if (typeof input !== 'string') return input;

    // Basic XSS protection
    return input
      .replace(/</g, '&lt;')
      .replace(/>/g, '&gt;')
      .replace(/"/g, '&quot;')
      .replace(/'/g, '&#39;');
  }

  sanitizeObject(obj) {
    if (typeof obj !== 'object' || obj === null) {
      return this.sanitizeInput(obj);
    }

    if (Array.isArray(obj)) {
      return obj.map(item => this.sanitizeObject(item));
    }

    const sanitized = {};
    for (const key in obj) {
      if (obj.hasOwnProperty(key)) {
        sanitized[key] = this.sanitizeObject(obj[key]);
      }
    }
    return sanitized;
  }

  validateAndSanitize(validations) {
    return async (req, res, next) => {
      await Promise.all(validations.map(validation => validation.run(req)));

      const errors = validationResult(req);
      if (!errors.isEmpty()) {
        return res.status(400).json({ errors: errors.array() });
      }

      // Sanitize all inputs after validation
      req.body = this.sanitizeObject(req.body);
      req.query = this.sanitizeObject(req.query);
      req.params = this.sanitizeObject(req.params);

      next();
    };
  }

  // SQL Injection protection for raw queries
  escapeSQL(input) {
    if (typeof input !== 'string') return input;
    return input.replace(/[\0\x08\x09\x1a\n\r"'\\\%]/g, char => {
      switch (char) {
        case '\0': return '\\0';
        case '\x08': return '\\b';
        case '\x09': return '\\t';
        case '\x1a': return '\\z';
        case '\n': return '\\n';
        case '\r': return '\\r';
        case '"':
        case "'":
        case '\\':
        case '%':
          return '\\' + char;
        default: return char;
      }
    });
  }
}

module.exports = new Sanitizer();
```

```javascript
const helmet = require('helmet');
const rateLimit = require('express-rate-limit');
const mongoSanitize = require('express-mongo-sanitize');

const hpp = require('hpp');
const cors = require('cors');

class SecurityMiddleware {
  constructor() {
    this.cspDirectives = {
      directives: {
        defaultSrc: ["'self'"],
        scriptSrc: ["'self'", "'unsafe-inline'", "https://cdn.jsdelivr.net"],
        styleSrc: ["'self'", "'unsafe-inline'", "https://cdn.jsdelivr.net"],
        imgSrc: ["'self'", "data:", "https://*.googleusercontent.com"],
        fontSrc: ["'self'", "https://fonts.gstatic.com"],
        connectSrc: ["'self'", "https://www.googleapis.com"],
        frameSrc: ["'none'"],
        objectSrc: ["'none'"],
        upgradeInsecureRequests: []
      }
    };

    this.rateLimiter = rateLimit({
      windowMs: 15 * 60 * 1000, // 15 minutes
      max: 100, // limit each IP to 100 requests per windowMs
      message: 'Too many requests from this IP, please try again later'
    });

    this.corsOptions = {
      origin: process.env.ALLOWED_ORIGINS ? process.env.ALLOWED_ORIGINS.split(',') : '*',
      methods: ['GET', 'POST', 'PUT', 'PATCH', 'DELETE'],
      allowedHeaders: ['Content-Type', 'Authorization'],
      credentials: true
    };
  }

  applySecurityHeaders(app) {
    // Set security-related HTTP headers
    app.use(helmet());
    app.use(helmet.contentSecurityPolicy(this.cspDirectives));
    app.use(helmet.hsts({
      maxAge: 31536000, // 1 year
      includeSubDomains: true,
      preload: true
    }));
    app.use(helmet.referrerPolicy({ policy: 'same-origin' }));
    app.use(helmet.noSniff());
    app.use(helmet.frameguard({ action: 'deny' }));
    app.use(helmet.permittedCrossDomainPolicies());

    // Prevent MIME type sniffing
    app.use((req, res, next) => {
      res.setHeader('X-Content-Type-Options', 'nosniff');
      next();
    });

    // Enable CORS with options
    app.use(cors(this.corsOptions));

    // Rate limiting
    app.use(this.rateLimiter);

    // Data sanitization against NoSQL query injection
    app.use(mongoSanitize());

    // Data sanitization against XSS
    app.use(xss());

    // Prevent parameter pollution
    app.use(hpp());

    // Security logging
    app.use((req, res, next) => {
      logger.info(`Security: ${req.method} ${req.path} from ${req.ip}`);
      next();
    });
  }

  checkContentType(req, res, next) {

        return next(new ForbiddenError('Content-Type must be application/json'));
      }
    }
    next();
  }
```

```javascript
class App {
  constructor() {
    this.app = express();
    this.configureMiddleware();
    this.configureRoutes();
    this.configureErrorHandling();
    this.connectToDatabase();
  }

  configureMiddleware() {
    // Apply security middleware first
    securityMiddleware.applySecurityHeaders(this.app);

    // Parse JSON bodies
    this.app.use(express.json({ limit: '10kb' }));

    // Parse URL-encoded bodies
    this.app.use(express.urlencoded({ extended: true, limit: '10kb' }));

    // Cookie parser
    this.app.use(cookieParser());

    // Content type checking
    this.app.use(securityMiddleware.checkContentType);

    // Sanitize all inputs
    this.app.use((req, res, next) => {
      req.body = sanitizer.sanitizeObject(req.body);
      req.query = sanitizer.sanitizeObject(req.query);
      req.params = sanitizer.sanitizeObject(req.params);
      next();
    });
  }

  configureRoutes() {
    // Health check endpoint
    this.app.get('/health', (req, res) => {
      res.json({ status: 'healthy', timestamp: new Date().toISOString() });
    });

    // API routes
    this.app.use('/api/auth', authRoutes);

    // Serve static files
    this.app.use(express.static('public'));

    // Catch-all for undefined routes
    this.app.all('*', (req, res, next) => {
      res.status(404).json({
        status: 'error',
        message: `Can't find ${req.originalUrl} on this server!`
      });
    });
  }

  configureErrorHandling() {
    // Error handling middleware
    this.app.use(errorMiddleware);
  }

  async connectToDatabase() {
    try {
      await mongoose.connect(config.uri, config.options);
      logger.info('Connected to MongoDB database');
    } catch (err) {
      logger.error(`Database connection error: ${err.message}`);
      process.exit(1);
    }
  }
}
```

## 8. Análise de Ameaças (Threat Modeling) para V12 Core

### Documento de Threat Modeling (STRIDE)

**Feature:** Autenticação e Autorização no DIYAPP V12 Core
**Data:** [Data Atual]
**Responsável:** Especialista em Segurança

#### 1. Mapeamento de Ativos
- Credenciais de usuário (email/senha)
- Tokens JWT (access e refresh)
- Dados pessoais dos usuários
- Endpoints de autenticação (/api/auth/*)
- Cookies de sessão
- Configurações de segurança

#### 2. Ameaças Identificadas (STRIDE)

**Spoofing (S)**
- **Ameaça:** Ataque de força bruta em credenciais
  - **Risco:** Alto
  - **Controles:** Rate limiting, bloqueio temporário após tentativas falhas, MFA
  - **Status:** Implementado

- **Ameaça:** Phishing de credenciais
  - **Risco:** Médio
  - **Controles:** Educação do usuário, OAuth com provedores confiáveis
  - **Status:** Implementado (Google OAuth)

**Tampering (T)**
- **Ameaça:** Modificação de tokens JWT
  - **Risco:** Alto
  - **Controles:** Assinatura forte (HMAC-SHA256), expiração curta (15min)
  - **Status:** Implementado

- **Ameaça:** Manipulação de cookies de sessão
  - **Risco:** Médio
  - **Controles:** Cookies HTTP-only, Secure, SameSite=strict
  - **Status:** Implementado

**Repudiation (R)**
- **Ameaça:** Usuário nega ter realizado ação crítica
  - **Risco:** Médio
  - **Controles:** Logs imutáveis de operações críticas (login, logout, alteração de senha)
  - **Status:** Implementado (logs no middleware de segurança)

**Information Disclosure (I)**
- **Ameaça:** Vazamento de tokens em logs ou erros
  - **Risco:** Alto
  - **Controles:** Mascaramento de tokens em logs, tratamento de erros genéricos
  - **Status:** Implementado

- **Ameaça:** Exposição de dados sensíveis em respostas de API
  - **Risco:** Alto
  - **Controles:** Seleção cuidadosa de campos em respostas, mascaramento de senhas
  - **Status:** Implementado (select: false para password no User model)

**Denial of Service (D)**
- **Ameaça:** Ataque de força bruta em endpoints de autenticação
  - **Risco:** Alto
  - **Controles:** Rate limiting (100 requisições/15min por IP)
  - **Status:** Implementado

- **Ameaça:** Envio de payloads muito grandes
  - **Risco:** Médio
  - **Controles:** Limite de tamanho de payload (10kb)
  - **Status:** Implementado

**Elevation of Privilege (E)**
- **Ameaça:** Escalada de privilégios através de manipulação de roles
  - **Risco:** Alto
  - **Controles:** Verificação de roles em cada endpoint, princípio do menor privilégio
  - **Status:** Implementado (middleware de autorização)

- **Ameaça:** Acesso a recursos de outros usuários (IDOR)
  - **Risco:** Alto
  - **Controles:** Verificação de ownership em cada requisição
  - **Status:** Implementado (middleware de autorização verifica user.id)

#### 3. Riscos Residuais Aceitos
- **Risco:** Ataques de phishing direcionados a usuários finais
  - **Justificativa:** Fora do controle técnico direto da aplicação
  - **Mitigação:** Implementação de OAuth com provedores confiáveis e educação do usuário

- **Risco:** Vulnerabilidades em dependências de terceiros
  - **Justificativa:** Dependências são necessárias para o funcionamento
  - **Mitigação:** Monitoramento contínuo com Snyk/Dependabot, atualizações regulares

#### 4. Requisitos de Segurança Derivados
1. Implementar rate limiting em todos os endpoints de autenticação
2. Usar tokens JWT com expiração curta (15min) e refresh tokens (7d)
3. Implementar cookies seguros (HTTP-only, Secure, SameSite=strict)
4. Validar e sanitizar todos os inputs
5. Implementar CSP, HSTS e outros headers de segurança
6. Usar bibliotecas de criptografia aprovadas (bcrypt, jsonwebtoken)
7. Implementar logs imutáveis para operações críticas
8. Verificar autorização em cada endpoint protegido

#### 2. Script para Limpeza de Logs

Implemente um script que limpa os logs automaticamente após o período de retenção:

```javascript
const fs = require('fs');
const path = require('path');
const config = require('./config/log');

const logsDir = path.join(__dirname, 'logs');

function cleanUpLogs() {
  fs.readdir(logsDir, (err, files) => {
    if (err) throw err;

    files.forEach(file => {
      const filePath = path.join(logsDir, file);
      fs.stat(filePath, (err, stats) => {
        if (err) throw err;

        const now = new Date().getTime();
        const endTime = new Date(stats.mtime).getTime() + config.retentionPeriod * 24 * 60 * 60 * 1000;

        if (now > endTime) {
          fs.unlink(filePath, err => {
            if (err) throw err;
          });
        }
      });
    });
  });
}

// Schedule log cleanup according to the config file
setInterval(cleanUpLogs, parseTime(config.cleanupInterval));

function parseTime(interval) {
  const [value, unit] = interval.match(/\d+|\D+/g);
  const timeMultiplier = { 'd': 24 * 60 * 60 * 1000 };
  return value * (timeMultiplier[unit] || 1);
}
```

## 1. Threat Modeling (STRIDE)

Primeiro, vamos documentar as ameaças identificadas no arquivo `threat-model/DIYAPP-V12-STRIDE.md`:

```markdown
# DIYAPP V12 Core - Threat Modeling (STRIDE)

## Ativos Protegidos
- Dados de usuários (PII)
- Credenciais de autenticação
- Tokens JWT
- Configurações do sistema
- Código fonte e dependências
- Endpoints da API

## Ameaças Identificadas por Categoria STRIDE

### Spoofing (S)
1. **Ameaça**: Usuário malicioso se passando por outro usuário
   - **Controles**:
     - Autenticação forte (JWT + OAuth)
     - MFA para operações sensíveis
     - Tokens com expiração curta
     - Rate limiting em endpoints de autenticação

2. **Ameaça**: Serviço malicioso se passando pelo DIYAPP
   - **Controles**:
     - Certificados TLS válidos
     - HSTS (HTTP Strict Transport Security)
     - CSP (Content Security Policy)

### Tampering (T)
1. **Ameaça**: Modificação de dados em trânsito
   - **Controles**:
     - TLS 1.2+ em todas as comunicações
     - HMAC para dados sensíveis
     - Validação de integridade de payloads

2. **Ameaça**: Modificação de dados em repouso
   - **Controles**:
     - Criptografia AES-256 para dados sensíveis
     - Controles de acesso granular
     - Logs imutáveis de alterações

### Repudiation (R)
1. **Ameaça**: Usuário nega ter realizado ação crítica
   - **Controles**:
     - Logs de auditoria imutáveis (com timestamp e assinatura)
     - Assinatura digital de operações críticas
     - Retenção de logs por período regulatório

### Information Disclosure (I)
1. **Ameaça**: Vazamento de dados sensíveis
   - **Controles**:
     - Mascaramento de dados em logs e respostas
     - Criptografia de dados sensíveis
     - Controle de acesso baseado em papéis (RBAC)
     - Sanitização de outputs (especialmente para LLMs)

2. **Ameaça**: Exposição de informações de debug
   - **Controles**:
     - Headers de segurança (X-Content-Type-Options, X-Frame-Options)
     - Remoção de informações sensíveis de mensagens de erro

### Denial of Service (D)
1. **Ameaça**: Sobrecarga de endpoints de autenticação
   - **Controles**:
     - Rate limiting agressivo em endpoints de login
     - Bloqueio temporário após N tentativas falhas
     - Circuit breakers para serviços externos

2. **Ameaça**: Abuso de recursos computacionais
   - **Controles**:
     - Limites de tamanho de payload
     - Timeouts para requisições
     - Quotas de uso por usuário

### Elevation of Privilege (E)
1. **Ameaça**: Acesso a recursos não autorizados
   - **Controles**:
     - Autorização granular (por recurso, não apenas por rota)
     - Princípio do menor privilégio
     - Verificação de IDOR (Insecure Direct Object Reference)
     - Logs de operações privilegiadas

2. **Ameaça**: Escalada de privilégios via vulnerabilidades
   - **Controles**:
     - Atualização constante de dependências
     - SAST em todos os PRs
     - Sandboxing de operações perigosas

1. **Risco**: Uso de JWT com expiração curta (15min) pode impactar UX em operações longas
```

### Serviço de Autenticação (`src/auth/auth.service.js`)

```javascript
const jwt = require('jsonwebtoken');
const bcrypt = require('bcryptjs');
const { jwt: jwtConfig } = require('../config/security.config');
const cryptoUtils = require('../utils/cryptoUtils');
const logger = require('../utils/logger');

class AuthService {
  constructor(userModel) {
    this.userModel = userModel;
  }

  async validateUser(email, password) {
    try {
      const user = await this.userModel.findOne({ email });
      if (!user) {
        logger.warn(`Authentication attempt for non-existent user: ${email}`);
        return null;
      }

      const isValid = await bcrypt.compare(password, user.password);
      if (!isValid) {
        logger.warn(`Invalid password attempt for user: ${email}`);
        return null;
      }

      // Remove sensitive data before returning
      const { password: _, ...userWithoutPassword } = user.toObject();
      return userWithoutPassword;
    } catch (error) {
      logger.error(`Authentication error for ${email}: ${error.message}`);
      throw new Error('Authentication failed');
    }
  }

  generateTokens(user) {
    const accessToken = jwt.sign(
      { sub: user._id, roles: user.roles },
      jwtConfig.accessToken.secret,
      { expiresIn: jwtConfig.accessToken.expiresIn }
    );

    const refreshToken = jwt.sign(
      { sub: user._id },
      jwtConfig.refreshToken.secret,
      { expiresIn: jwtConfig.refreshToken.expiresIn }
    );

    return { accessToken, refreshToken };
  }

  async refreshAccessToken(refreshToken) {
    try {
      const decoded = jwt.verify(refreshToken, jwtConfig.refreshToken.secret);
      const user = await this.userModel.findById(decoded.sub);

      if (!user) {
        throw new Error('User not found');
      }

      return this.generateTokens(user);
    } catch (error) {
      logger.error(`Token refresh failed: ${error.message}`);
      throw new Error('Invalid refresh token');
    }
  }

  async invalidateToken(token) {
    // In a real implementation, you would add the token to a blacklist
    // For this example, we'll just log it
    logger.info(`Token invalidated: ${token}`);
    return true;
  }

  async hashPassword(password) {
    return bcrypt.hash(password, 12);
  }
}

module.exports = AuthService;
```

### Estratégia JWT (`src/auth/jwt.strategy.js`)

```javascript
const { jwt: jwtConfig } = require('../config/security.config');
const logger = require('../utils/logger');

const jwtOptions = {
  jwtFromRequest: ExtractJwt.fromAuthHeaderAsBearerToken(),
  secretOrKey: jwtConfig.accessToken.secret,
  algorithms: ['HS256'],
  issuer: 'diyapp',
  audience: 'diyapp-users'
};

const jwtStrategy = new Strategy(jwtOptions, async (payload, done) => {
  try {
    // In a real implementation, you would fetch the user from the database
    // For this example, we'll just return the payload
    if (!payload.sub) {
      return done(null, false);
    }

    return done(null, {
      id: payload.sub,

      }
```

```javascript
        refreshToken: tokens.refreshToken,
        user: {
          id: user._id,
          email: user.email,
          name: user.name,
          roles: user.roles
        }
      });

        return res.status(400).json({ error: 'Refresh token is required' });

      res.json(tokens);
    } catch (error) {
      logger.error(`Token refresh error: ${error.message}`);
      next(error);
    }
  }

        await this.authService.invalidateToken(token);
      }

      logger.info(`User ${req.user?.id} logged out successfully`);

    } catch (error) {
      logger.error(`Logout error: ${error.message}`);
```

### Criptografia (`src/utils/cryptoUtils.js`)

```javascript
const crypto = require('crypto');
const logger = require('./logger');

const cryptoUtils = {
  encrypt: (text, algorithm = 'aes-256-cbc') => {
    try {
      const key = crypto.scryptSync(process.env.ENCRYPTION_KEY || 'default-encryption-key', 'salt', 32);
      const iv = crypto.randomBytes(16);
      const cipher = crypto.createCipheriv(algorithm, key, iv);
      let encrypted = cipher.update(text, 'utf8', 'hex');
      encrypted += cipher.final('hex');
      return `${iv.toString('hex')}:${encrypted}`;
    } catch (error) {
      logger.error(`Encryption failed: ${error.message}`);
      throw new Error('Encryption failed');
    }
  },

  decrypt: (encryptedText, algorithm = 'aes-256-cbc') => {
    try {
      const key = crypto.scryptSync(process.env.ENCRYPTION_KEY || 'default-encryption-key', 'salt', 32);
      const [ivHex, encrypted] = encryptedText.split(':');
      const iv = Buffer.from(ivHex, 'hex');
      const decipher = crypto.createDecipheriv(algorithm, key, iv);
      let decrypted = decipher.update(encrypted, 'hex', 'utf8');
      decrypted += decipher.final('utf8');
      return decrypted;
    } catch (error) {
      logger.error(`Decryption failed: ${error.message}`);
      throw new Error('Decryption failed');
    }
  },

  hash: (text, algorithm = 'sha256') => {
    return crypto.createHash(algorithm).update(text).digest('hex');
  },

  generateRandomToken: (length = 32) => {
    return crypto.randomBytes(length).toString('hex');
  },

  verifyHmac: (data, signature, secret, algorithm = 'sha256') => {
    const hmac = crypto.createHmac(algorithm, secret);
    hmac.update(data);
    const expectedSignature = hmac.digest('hex');
    return crypto.timingSafeEqual(Buffer.from(signature), Buffer.from(expectedSignature));
  }
};

module.exports = cryptoUtils;
```

### Revisão da Política de Retenção de Logs

Criaremos um script que garante a retenção de logs conforme as diretrizes de conformidade, com exclusão automática após o prazo estabelecido.

```python
import os
import time
from datetime import datetime, timedelta

LOG_DIRECTORY = 'logs/'
RETENTION_DAYS = 30

def cleanup_old_logs():
    now = time.time()
    cutoff = now - (RETENTION_DAYS * 86400)
    for filename in os.listdir(LOG_DIRECTORY):
        file_path = os.path.join(LOG_DIRECTORY, filename)
        if os.path.isfile(file_path):
            file_creation_time = os.path.getctime(file_path)
            if file_creation_time < cutoff:
                os.remove(file_path)
                print(f"Deleted old log file: {filename}")

if __name__ == "__main__":
    cleanup_old_logs()
```

### Gestão de Consentimento do Usuário

Implementaremos um fluxo de consentimento que registra quando os usuários concedem ou revogam seu consentimento, com atualizações em um banco de dados SQLite.

```python
import sqlite3

# Database setup
conn = sqlite3.connect('user_consent.db')
c = conn.cursor()
c.execute('''CREATE TABLE IF NOT EXISTS user_consent (
                user_id INTEGER PRIMARY KEY,
                consent_given BOOLEAN NOT NULL,
                consent_date TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP)''')

def update_consent(user_id, consent_given):
    with conn:
        c.execute('''INSERT INTO user_consent (user_id, consent_given, consent_date)
                     VALUES (?, ?, ?) 
                     ON CONFLICT(user_id) 
                     DO UPDATE SET consent_given=excluded.consent_given, consent_date=excluded.consent_date''',
                  (user_id, consent_given, datetime.now()))
        print(f"User {user_id} consent {'granted' if consent_given else 'revoked'}.")

# Example usage
update_consent(1, True)  # User 1 gives consent
update_consent(2, False) # User 2 revokes consent

conn.close()
```

## **2. Roteiro de Testes de Performance (Load/Stress)**
**Critérios de Aceite:**
- **Tempo de resposta < 2s** para 95% das requisições em carga normal (100 usuários simultâneos).
- **Throughput ≥ 1000 requisições/segundo** em endpoints críticos (ex: `/api/projects`).
- **Tolerância a falhas**: Sistema deve se recuperar em < 30s após pico de carga (2000 usuários).
- **Monitoramento de métricas**:
  - Latência (p95, p99).
  - Taxa de erros (HTTP 5xx).
  - Uso de CPU/Memória.

### **Cenários de Performance**
| ID  | Cenário                          | Ferramenta       | Métricas Alvo                                                                 |
|-----|----------------------------------|------------------|-------------------------------------------------------------------------------|
| PERF-01 | Carga Normal (100 usuários)      | k6 / Grafana     | Latência < 2s, Erros 0%, Throughput ≥ 1000 req/s                              |
| PERF-02 | Pico de Carga (2000 usuários)    | k6               | Latência < 5s, Erros < 5%, Recuperação em < 30s                              |
| PERF-03 | Stress de Banco de Dados         | k6 + PostgreSQL  | Queries < 100ms, Sem deadlocks, Uso de CPU < 80%                             |
| PERF-04 | Endpoint Crítico (`/api/projects`) | k6             | Throughput ≥ 1500 req/s, Latência p99 < 1s                                   |

### **Código de Exemplo (k6)**
```javascript
// tests/performance/load-test.js
import http from 'k6/http';
import { check, sleep } from 'k6';

export const options = {
  stages: [
    { duration: '30s', target: 100 },  // Ramp-up
    { duration: '1m', target: 100 },   // Carga normal
    { duration: '30s', target: 2000 }, // Pico
    { duration: '30s', target: 100 },  // Ramp-down
  ],
  thresholds: {
    http_req_duration: ['p(95)<2000'], // 95% das requisições < 2s
    http_req_failed: ['rate<0.01'],    // Erros < 1%
  },
};

export default function () {
  const payload = JSON.stringify({
    name: 'DIYAPP V12',
    description: 'Performance Test',
  });
  const params = {
    headers: { 'Content-Type': 'application/json' },
  };

  // Teste de criação de projeto
  const res = http.post('http://localhost:3000/api/projects', payload, params);
  check(res, {
    'Status é 201': (r) => r.status === 201,
    'Resposta em < 1s': (r) => r.timings.duration < 1000,
  });

  sleep(1);
}
```
