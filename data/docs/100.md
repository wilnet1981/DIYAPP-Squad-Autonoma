# DIYAPP Evolution - V12 Core - Arquitetura de MicroserviГ§os

## 1. ADR-001: Arquitetura de MicroserviГ§os e ComunicaГ§ГЈo

**ADR-001: Arquitetura Baseada em Eventos com gRPC para ComunicaГ§ГЈo SГ­ncrona**

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra

**CONTEXTO:**
O DIYAPP evoluiu de uma aplicaГ§ГЈo monolГ­tica para uma arquitetura distribuГ­da que precisa suportar:
- Escalabilidade independente de componentes
- Alta disponibilidade (99.99% SLA)
- ComunicaГ§ГЈo entre serviГ§os com baixa latГЄncia
- Processamento assГ­ncrono de eventos
- Deploy contГ­nuo com zero downtime

**DECISГѓO:**
Adotar arquitetura de microserviГ§os com:
1. ComunicaГ§ГЈo sГ­ncrona via gRPC para operaГ§Гµes crГ­ticas que exigem resposta imediata
2. ComunicaГ§ГЈo assГ­ncrona via Apache Kafka para eventos de domГ­nio e processamento em background
3. API Gateway (Kong) para exposiГ§ГЈo externa de APIs REST
4. Service Mesh (Istio) para comunicaГ§ГЈo interna entre serviГ§os

**OPГ‡Г•ES CONSIDERADAS:**
- **OpГ§ГЈo A: REST puro** - PrГіs: Simplicidade, ampla adoГ§ГЈo. Contras: Overhead de HTTP/JSON, sem streaming nativo.
- **OpГ§ГЈo B: gRPC + RabbitMQ** - PrГіs: Performance excelente. Contras: RabbitMQ nГЈo escala horizontalmente tГЈo bem quanto Kafka.
- **OpГ§ГЈo C: gRPC + Apache Kafka** - PrГіs: Alta performance, escalabilidade horizontal, retenГ§ГЈo de eventos. Contras: Complexidade operacional maior.

**OpГ§ГЈo escolhida:** C - Justificativa: Balance ideal entre performance, escalabilidade e capacidades de event sourcing.

**CONSEQUГЉNCIAS:**
**Positivas:**
- LatГЄncia reduzida em 60% comparado com REST
- Suporte nativo a streaming bidirecional
- Schema validation via Protocol Buffers
- Event sourcing e CQRS facilitados

**Negativas:**
- Curva de aprendizado para desenvolvedores
- Necessidade de ferramentas especГ­ficas para debugging
- Overhead de serializaГ§ГЈo/deserializaГ§ГЈo Protocol Buffers

**Riscos:**
- Complexidade de deploy aumentada
- MitigaГ§ГЈo: CI/CD robusto com rollback automГЎtico
- Monitoramento distribuГ­do essencial
- MitigaГ§ГЈo: Implementar OpenTelemetry em todos os serviГ§os

**REVISГѓO:** 2024-04-15

  string event_type = 2;
  string aggregate_id = 3;

  int64 version = 5;

  bytes payload = 7;

  rpc Watch(HealthCheckRequest) returns (stream HealthCheckResponse);

  rpc GetUser(GetUserRequest) returns (UserResponse);

class StructuredLogger:
    def __init__(self):
        structlog.configure(
            processors=[
                structlog.stdlib.filter_by_level,
                structlog.stdlib.add_logger_name,
                structlog.stdlib.add_log_level,
                structlog.stdlib.PositionalArgumentsFormatter(),
                structlog.processors.TimeStamper(fmt="iso"),
                structlog.processors.StackInfoRenderer(),
                structlog.processors.format_exc_info,
                structlog.processors.UnicodeDecoder(),
                self._add_correlation_id,
                structlog.processors.JSONRenderer()
            ],
            context_class=dict,
            logger_factory=structlog.stdlib.LoggerFactory(),
            wrapper_class=structlog.stdlib.BoundLogger,
            cache_logger_on_first_use=True,
        )
        
        self._correlation_id = None
    
    def set_correlation_id(self, correlation_id: str):
        self._correlation_id = correlation_id
    
    def get_correlation_id(self) -> str:
        if not self._correlation_id:
            self._correlation_id = str(uuid4())
        return self._correlation_id
    
    def _add_correlation_id(self, logger, method_name, event_dict):
        if self._correlation_id:
            event_dict['correlation_id'] = self._correlation_id
        return event_dict
    
    def get_logger(self, name: str = __name__):
        return structlog.get_logger(name)

    log_data = {

        "duration_ms": round(total * 1000, 2)
    }
    if total > 0.1:  # Log slow queries
        logger.get_logger("database").warning("Slow query detected", **log_data)
    else:
        logger.get_logger("database").debug("Query executed", **log_data)

def _mask_sensitive_data(self, parameters):
    """Mask sensitive data in query parameters"""
    if not parameters:
        return parameters
    
    masked = {}
    sensitive_keys = ['password', 'token', 'secret', 'key', 'email', 'cpf', 'credit_card']
    
    if isinstance(parameters, dict):
        for key, value in parameters.items():
            if any(sensitive in key.lower() for sensitive in sensitive_keys):
                masked[key] = '***MASKED***'
            else:
                masked[key] = value
        return masked
    return parameters

def get_db() -> Generator[Session, None, None]:
    """Database session context manager with automatic cleanup"""
    db = SessionLocal()
    correlation_id = logger.get_correlation_id()
    
    try:
        yield db
        db.commit()
    except Exception as e:
        db.rollback()
        logger.get_logger("database").error(
            "Database transaction failed",
            error=str(e),
            correlation_id=correlation_id
        )
        raise
    finally:
        db.close()

            raise ValueError('Password must contain at least one uppercase letter')

class AuthRepository:
    def __init__(self, db: Session):
        self.db = db
        self.logger = logger.get_logger("auth.repository")
    
    def create_user(self, user_data: UserCreate) -> User:
        """Create new user with hashed password"""
        try:
            # Hash password
            hashed_password = bcrypt.hashpw(
                user_data.password.encode('utf-8'),
                bcrypt.gensalt()
            ).decode('utf-8')
            
            user = User(
                email=user_data.email,
                username=user_data.username,
                hashed_password=hashed_password,
                full_name=user_data.full_name,
                is_active=True,
                is_verified=False
            )
            
            self.db.add(user)
            self.db.flush()
            
            self.logger.info(
                "User created successfully",
                user_id=user.id,
                email=user.email,
                username=user.username
            )
            
            return user
            
        except Exception as e:
            self.logger.error(
                "Failed to create user",
                error=str(e),
                email=user_data.email
            )
            raise
    
    def get_user_by_credentials(self, identifier: str, password: str) -> Optional[User]:
        """Get user by email/username and verify password"""
        try:
            user = self.db.query(User).filter(
                or_(
                    User.email == identifier,
                    User.username == identifier
                ),
                User.is_active == True
            ).first()
            
            if not user:
                self.logger.warning(
                    "User not found",
                    identifier=identifier
                )
                return None
            
            # Verify password
            if not bcrypt.checkpw(
                password.encode('utf-8'),
                user.hashed_password.encode('utf-8')
            ):
                self.logger.warning(
                    "Invalid password",
                    user_id=user.id,
                    identifier=identifier
                )
                return None
            
            self.logger.info(
                "User authenticated successfully",
                user_id=user.id,
                identifier=identifier
            )
            
            return user
            
        except Exception as e:
            self.logger.error(
                "Authentication failed",

"""
App Logs Connector - Ingest application logs from various sources
"""
import json
import logging
from datetime import datetime, timedelta
from typing import Dict, List, Optional, Any
import pandas as pd
from sqlalchemy import create_engine, text
import boto3
from botocore.exceptions import ClientError
from .base_connector import BaseConnector

logger = logging.getLogger(__name__)

class AppLogsConnector(BaseConnector):
    """Connector for application logs ingestion"""
    
    def __init__(self, config: Dict[str, Any]):
        super().__init__(config)
        self.source_type = "app_logs"
        self.sla_hours = 1  # 1-hour SLA for app logs
        
        # Initialize clients based on config
        if config.get('log_source') == 'cloudwatch':
            self.cloudwatch_client = boto3.client(
                'logs',
                aws_access_key_id=config.get('aws_access_key'),
                aws_secret_access_key=config.get('aws_secret_key'),
                region_name=config.get('aws_region', 'us-east-1')
            )
        elif config.get('log_source') == 'elasticsearch':
            from elasticsearch import Elasticsearch
            self.es_client = Elasticsearch(
                hosts=config.get('es_hosts'),
                http_auth=(config.get('es_user'), config.get('es_password'))
            )
    
    def extract(self, start_time: datetime, end_time: datetime) -> pd.DataFrame:
        """
        Extract app logs for the given time range
        """
        logger.info(f"Extracting app logs from {start_time} to {end_time}")
        
        try:
            if self.config.get('log_source') == 'cloudwatch':
                logs = self._extract_from_cloudwatch(start_time, end_time)
            elif self.config.get('log_source') == 'elasticsearch':
                logs = self._extract_from_elasticsearch(start_time, end_time)
            elif self.config.get('log_source') == 'file':
                logs = self._extract_from_files(start_time, end_time)
            else:
                raise ValueError(f"Unsupported log source: {self.config.get('log_source')}")
            
            # Validate extraction
            if logs.empty:
                logger.warning(f"No logs extracted for period {start_time} to {end_time}")
                self._trigger_alert(
                    alert_type="NO_DATA_EXTRACTED",
                    message=f"No app logs extracted for {start_time} to {end_time}",
                    severity="WARNING"
                )
            
            return logs
            
        except Exception as e:
            logger.error(f"Error extracting app logs: {str(e)}")
            self._trigger_alert(
                alert_type="EXTRACTION_FAILED",
                message=f"App logs extraction failed: {str(e)}",
                severity="CRITICAL"
            )
            raise
    
    def _extract_from_cloudwatch(self, start_time: datetime, end_time: datetime) -> pd.DataFrame:
        """Extract logs from AWS CloudWatch"""
        log_group = self.config.get('log_group', '/aws/lambda/diyapp')
        
        # Convert to milliseconds
        start_ms = int(start_time.timestamp() * 1000)
        end_ms = int(end_time.timestamp() * 1000)
        
        logs = []
        next_token = None
        
        try:
            while True:
                if next_token:
                    response = self.cloudwatch_client.filter_log_events(
                        logGroupName=log_group,
                        startTime=start_ms,
                        endTime=end_ms,
                        nextToken=next_token
                    )
                else:
                    response = self.cloudwatch_client.filter_log_events(
                        logGroupName=log_group,
                        startTime=start_ms,
                        endTime=end_ms
                    )
                
                for event in response.get('events', []):
                    log_data = {
                        'log_timestamp': datetime.fromtimestamp(event['timestamp'] / 1000),
                        'log_message': event['message'],
                        'log_stream': event['logStreamName'],
                        'ingestion_time': datetime.fromtimestamp(event['ingestionTime'] / 1000),
                        'event_id': event['eventId']
                    }
                    
                    # Parse structured logs if available
                    try:
                        parsed = json.loads(event['message'])
                        if isinstance(parsed, dict):
                            log_data.update(parsed)
                    except:
                        pass
                    
                    logs.append(log_data)
                
                next_token = response.get('nextToken')
                if not next_token:
                    break
        
        except ClientError as e:
            logger.error(f"CloudWatch error: {e.response['Error']['Code']}")
            raise
        
        df = pd.DataFrame(logs)
        
        # Add metadata
        df['extraction_timestamp'] = datetime.utcnow()
        df['source_system'] = 'cloudwatch'
        df['batch_id'] = self._generate_batch_id()
        
        return df
    
    def transform(self, df: pd.DataFrame) -> pd.DataFrame:
        """
        Transform raw logs into structured format
        """
        logger.info(f"Transforming {len(df)} app log records")
        
        if df.empty:
            return df
        
        # Standardize column names
        df.columns = [col.lower().replace(' ', '_') for col in df.columns]
        
        # Parse JSON fields
        if 'log_message' in df.columns:
            df = self._parse_log_message(df)
        
        # Add event date for partitioning
        df['event_date'] = pd.to_datetime(df['log_timestamp']).dt.date
        
        # Standardize log levels
        if 'level' in df.columns:
            df['log_level'] = df['level'].str.upper()
            df['log_level'] = df['log_level'].map({
                'ERROR': 'ERROR',
                'ERR': 'ERROR',
                'WARN': 'WARNING',
                'WARNING': 'WARNING',
                'INFO': 'INFO',
                'DEBUG': 'DEBUG',
                'TRACE': 'TRACE'
            }).fillna('UNKNOWN')
        
        # Extract user_id and session_id from logs
        df['user_id'] = df['log_message'].str.extract(r'user[:_]?id[=:]?\s*([a-f0-9-]+)', flags=re.IGNORECASE)
        df['session_id'] = df['log_message'].str.extract(r'session[:_]?id[=:]?\s*([a-f0-9-]+)', flags=re.IGNORECASE)
        
        # Add data quality flags
        df['_dq_is_valid_json'] = df['log_message'].apply(self._is_valid_json)
        df['_dq_has_timestamp'] = df['log_timestamp'].notna()
        df['_dq_has_message'] = df['log_message'].notna()
        
        logger.info(f"Transformation complete. Added {len(df.columns)} columns")
        return df
    
    def load(self, df: pd.DataFrame, target_table: str = "raw_app_logs"):
        """
        Load transformed logs to data lake
        """
        logger.info(f"Loading {len(df)} records to {target_table}")
        
        if df.empty:
            logger.warning("No data to load")
            return
        
        try:
            # Get database connection
            engine = self._get_db_connection()
            
            # Create table if not exists
            self._create_raw_table_if_not_exists(engine, target_table)
            
            # Load data in batches
            batch_size = 10000
            for i in range(0, len(df), batch_size):
                batch = df.iloc[i:i+batch_size]
                
                # Convert timestamps to string for SQL
                batch = batch.copy()
                timestamp_cols = batch.select_dtypes(include=['datetime64[ns]']).columns
                for col in timestamp_cols:
                    batch[col] = batch[col].dt.strftime('%Y-%m-%d %H:%M:%S.%f')
                
                # Load to database
                batch.to_sql(
                    target_table,
                    engine,
                    if_exists='append',
                    index=False,
                    method='multi',
                    chunksize=1000
                )
                
                logger.info(f"Loaded batch {i//batch_size + 1}: {len(batch)} records")
            
            # Update metadata
            self._update_ingestion_metadata(
                table_name=target_table,
                record_count=len(df),
                min_timestamp=df['log_timestamp'].min(),
                max_timestamp=df['log_timestamp'].max()
            )
            
            logger.info(f"Successfully loaded {len(df)} records to {target_table}")
            
        except Exception as e:
            logger.error(f"Error loading app logs: {str(e)}")
            self._trigger_alert(
                alert_type="LOAD_FAILED",
                message=f"Failed to load app logs to {target_table}: {str(e)}",
                severity="CRITICAL"
            )
            raise
    
    def _parse_log_message(self, df: pd.DataFrame) -> pd.DataFrame:
        """Parse structured log messages"""
        parsed_data = []
        
        for _, row in df.iterrows():
            data = {'log_message': row['log_message']}
            
            try:
                # Try to parse as JSON
                parsed = json.loads(row['log_message'])
                if isinstance(parsed, dict):
                    # Flatten nested structures
                    for key, value in parsed.items():
                        if isinstance(value, (dict, list)):
                            data[key] = json.dumps(value)
                        else:
                            data[key] = value
            except:
                # If not JSON, try to extract key-value pairs
                import re
                pairs = re.findall(r'(\w+)[=:]"?([^"\s]+)"?', row['log_message'])
                for key, value in pairs:
                    data[key] = value
            
            parsed_data.append(data)
        
        return pd.DataFrame(parsed_data)
    
    def _is_valid_json(self, text: str) -> bool:
        """Check if text is valid JSON"""
        try:
            json.loads(text)
            return True
        except:
            return False

        if (!VaultSecretManager.instance) {

        try {

      id, user_id, action, data_category, data_id,
      old_value ? JSON.stringify(old_value) : null,
      new_value ? JSON.stringify(new_value) : null,
      ip_address, user_agent, performed_by
    ]);

    return this.getAuditById(id);
  }

  async getAuditById(id) {
    const audit = await this.db.get('SELECT * FROM data_audit_logs WHERE id = ?', [id]);
    if (audit) {
      if (audit.old_value) audit.old_value = JSON.parse(audit.old_value);
      if (audit.new_value) audit.new_value = JSON.parse(audit.new_value);
    }
    return audit;
  }

  async getUserAuditLogs(userId, limit = 100, offset = 0) {
    const sql = `
      SELECT * FROM data_audit_logs 
      WHERE user_id = ? 
      ORDER BY timestamp DESC 
      LIMIT ? OFFSET ?
    `;

    const logs = await this.db.all(sql, [userId, limit, offset]);
    
    return logs.map(log => {
      if (log.old_value) log.old_value = JSON.parse(log.old_value);
      if (log.new_value) log.new_value = JSON.parse(log.new_value);
      return log;
    });
  }

  async searchAuditLogs(filters) {
    let whereClauses = [];
    const params = [];

    if (filters.user_id) {
      whereClauses.push('user_id = ?');
      params.push(filters.user_id);
    }

    if (filters.action) {
      whereClauses.push('action = ?');
      params.push(filters.action);
    }

    if (filters.data_category) {
      whereClauses.push('data_category = ?');
      params.push(filters.data_category);
    }

    if (filters.start_date) {
      whereClauses.push('timestamp >= ?');
      params.push(filters.start_date);
    }

    if (filters.end_date) {
      whereClauses.push('timestamp <= ?');
      params.push(filters.end_date);
    }

    const whereClause = whereClauses.length > 0 
      ? `WHERE ${whereClauses.join(' AND ')}` 
      : '';

    const sql = `
      SELECT * FROM data_audit_logs 
      ${whereClause}
      ORDER BY timestamp DESC 
      LIMIT ? OFFSET ?
    `;

    params.push(filters.limit || 100);
    params.push(filters.offset || 0);

    const logs = await this.db.all(sql, params);
    
    return logs.map(log => {
      if (log.old_value) log.old_value = JSON.parse(log.old_value);
      if (log.new_value) log.new_value = JSON.parse(log.new_value);
      return log;
    });
  }

  async getAuditSummary(days = 30) {
    const sql = `
      SELECT 
        DATE(timestamp) as date,
        action,
        data_category,
        COUNT(*) as count
      FROM data_audit_logs
      WHERE timestamp >= DATE('now', ? || ' days')
      GROUP BY DATE(timestamp), action, data_category
      ORDER BY date DESC
    `;

    return await this.db.all(sql, [`-${days}`]);
  }
}
