# DIYAPP Evolution - V12 Core - Stack Técnica e Padrões Arquiteturais

## Arquitetura de Sistema - ADR-001: Stack Técnica V12

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead V12

### CONTEXTO:
A V12 precisa atingir 100% de autonomia operacional com estabilidade de produção. O sistema atual tem débito técnico acumulado em monitoramento, deploy automatizado e resiliência. Precisamos de uma stack que permita:
1. Auto-healing sem intervenção humana
2. CI/CD completamente automatizado
3. Observabilidade em tempo real
4. Rollback automático em falhas
5. Escalabilidade horizontal automática

### DECISÃO:
Adotar uma arquitetura baseada em microserviços com Kubernetes, Istio para service mesh, e ArgoCD para GitOps. Stack principal:
- **Backend:** Node.js 18+ com TypeScript 5.0+
- **Frontend:** React 18 com Vite
- **Banco de Dados:** PostgreSQL 15 + Redis 7 para cache
- **Message Queue:** RabbitMQ 3.12
- **Infra como Código:** Terraform + Pulumi
- **CI/CD:** GitHub Actions + ArgoCD
- **Monitoramento:** Prometheus + Grafana + Loki
- **Service Mesh:** Istio para observabilidade e resiliência
- **Container Registry:** GitHub Container Registry

### OPÇÕES CONSIDERADAS:
- **Opção A:** Monolito com PM2 (atual) - Prós: Simples, baixa complexidade. Contras: Escala vertical apenas, single point of failure.
- **Opção B:** Microserviços com Docker Swarm - Prós: Mais simples que K8s. Contras: Menos recursos de auto-healing.
- **Opção C:** Microserviços com Kubernetes (escolhida) - Prós: Auto-healing nativo, escalabilidade horizontal, ecossistema rico. Contras: Curva de aprendizado, complexidade operacional.

### CONSEQUÊNCIAS:
**Positivas:**
- Auto-scaling baseado em métricas
- Auto-healing de pods com falha
- Rollback automático via GitOps
- Observabilidade completa

**Negativas:**
- Maior complexidade de deploy
- Overhead de recursos para K8s
- Necessidade de expertise em containers

**Riscos:**
- Configuração incorreta do Istio pode quebrar comunicação
- Mitigação: Testes extensivos em staging, canary deployments

### 2.3 Tratamento de Erros
```typescript
// ERRADO
try {
  const user = await getUser(id);
} catch (error) {
  console.log(error);
}

// CORRETO
import { AppError, ErrorCodes } from '../libs/shared-types';

class UserService {
  async getUser(id: string): Promise<User> {
    try {
      const user = await this.userRepository.findById(id);
      if (!user) {
        throw new AppError('User not found', ErrorCodes.NOT_FOUND);
      }
      return user;
    } catch (error) {
      if (error instanceof AppError) {
        throw error;
      }
      this.logger.error('Failed to get user', { id, error });
      throw new AppError(
        'Internal server error',
        ErrorCodes.INTERNAL_ERROR,
        { originalError: error }
      );
    }
  }
}
```

### 4.2 Repository Pattern
```typescript
interface IUserRepository {
  findById(id: string): Promise<User | null>;
  findByEmail(email: string): Promise<User | null>;
  create(user: CreateUserDto): Promise<User>;
  update(id: string, data: Partial<User>): Promise<User>;
  delete(id: string): Promise<boolean>;
}

class UserRepository implements IUserRepository {
  constructor(private readonly db: DatabaseClient) {}
  
  async findById(id: string): Promise<User | null> {
    return this.db.query(
      'SELECT * FROM users WHERE id = $1',
      [id]
    ).then(rows => rows[0] || null);
  }
}
```

describe('UserService', () => {
  let userService: UserService;
  let mockRepository: jest.Mocked<IUserRepository>;
  
  beforeEach(() => {
    mockRepository = {
      findById: jest.fn(),
      findByEmail: jest.fn(),
      create: jest.fn(),
      update: jest.fn(),
      delete: jest.fn(),
    };
    
    userService = new UserService(mockRepository);
  });
  
  describe('getUser', () => {
    it('should return user when found', async () => {
      const mockUser = { id: '123', email: 'test@example.com' };
      mockRepository.findById.mockResolvedValue(mockUser);
      
      const result = await userService.getUser('123');
      
      expect(result).toEqual(mockUser);
      expect(mockRepository.findById).toHaveBeenCalledWith('123');
    });
    
    it('should throw AppError when user not found', async () => {
      mockRepository.findById.mockResolvedValue(null);
      
      await expect(userService.getUser('123'))
        .rejects
        .toThrow(AppError);
    });
  });
});
```

const cors = require('cors');
const helmet = require('helmet');
const rateLimit = require('express-rate-limit');
const compression = require('compression');
const { createTerminus } = require('@godaddy/terminus');

// Configurações
const config = require('./shared/config');
const logger = require('./infrastructure/logging/logger');
const metrics = require('./infrastructure/monitoring/metrics');
const { initializeDatabase, closeDatabase } = require('./data/database');
const { initializeCache } = require('./infrastructure/cache/redis');
const { initializeCircuitBreakers } = require('./infrastructure/circuit-breaker/manager');

// Importar APIs
const restRoutes = require('./api/rest/routes');
const graphqlSchema = require('./api/graphql/schemas');
const graphqlResolvers = require('./api/graphql/resolvers');

class App {
  constructor() {
    this.app = express();
    this.server = null;
    this.apolloServer = null;
    
    this.initializeMiddlewares();
    this.initializeRoutes();
    this.initializeGraphQL();
    this.initializeErrorHandling();
  }

  async initialize() {
    try {
      // Inicializar infraestrutura
      await initializeDatabase();
      await initializeCache();
      initializeCircuitBreakers();
      
      // Inicializar métricas
      metrics.initialize();
      
      logger.info('Infraestrutura inicializada com sucesso');
    } catch (error) {
      logger.error('Falha na inicialização da infraestrutura:', error);
      process.exit(1);
    }
  }

  initializeMiddlewares() {
    // Segurança
    this.app.use(helmet({
      contentSecurityPolicy: config.isProduction,
      crossOriginEmbedderPolicy: config.isProduction,
    }));
    
    // CORS
    this.app.use(cors({
      origin: config.cors.origins,
      credentials: true,
      methods: ['GET', 'POST', 'PUT', 'DELETE', 'PATCH', 'OPTIONS'],
    }));
    
    // Rate limiting
    const apiLimiter = rateLimit({
      windowMs: 15 * 60 * 1000, // 15 minutos
      max: config.rateLimit.maxRequests,
      message: {
        error: 'Muitas requisições deste IP',
        code: 'RATE_LIMIT_EXCEEDED'
      },
      standardHeaders: true,
      legacyHeaders: false,
    });
    
    // Aplicar rate limit apenas em produção
    if (config.isProduction) {
      this.app.use('/api/', apiLimiter);
    }
    
    // Body parsing
    this.app.use(express.json({ limit: '10mb' }));
    this.app.use(express.urlencoded({ extended: true, limit: '10mb' }));
    
    // Compression
    this.app.use(compression());
    
    // Logging middleware
    this.app.use(require('./infrastructure/logging/request-logger'));
    
    // Correlation ID middleware
    this.app.use(require('./api/rest/middleware/correlation-id'));
  }

  initializeRoutes() {
    // Health check
    this.app.get('/health', (req, res) => {
      res.json({
        status: 'healthy',
        timestamp: new Date().toISOString(),
        uptime: process.uptime(),
        version: config.version,
      });
    });
    
    // Metrics endpoint (apenas para monitoramento interno)
    this.app.get('/metrics', async (req, res) => {
      if (!config.metrics.exposeEndpoint) {
        return res.status(404).end();
      }
      
      res.set('Content-Type', 'text/plain');
      res.end(await metrics.getMetrics());
    });
    
    // API REST
    this.app.use('/api/v1', restRoutes);
    
    // 404 handler
    this.app.use('*', (req, res) => {
      res.status(404).json({
        error: 'Endpoint não encontrado',
        path: req.originalUrl,
        code: 'ENDPOINT_NOT_FOUND'
      });
    });
  }

  async initializeGraphQL() {
    this.apolloServer = new ApolloServer({
      typeDefs: graphqlSchema,
      resolvers: graphqlResolvers,
      context: ({ req }) => ({
        user: req.user,
        correlationId: req.correlationId,
        logger: logger.child({ correlationId: req.correlationId }),
      }),
      plugins: [
        require('./infrastructure/logging/apollo-logger'),
        require('./infrastructure/monitoring/apollo-metrics'),
      ],
      formatError: (error) => {
        logger.error('GraphQL Error:', error);
        
        // Não expor stack trace em produção
        if (config.isProduction) {
          return {
            message: error.message,
            code: error.extensions?.code || 'INTERNAL_ERROR',
          };
        }
        
        return error;
      },
      introspection: config.graphql.introspection,
      playground: config.graphql.playground,
    });
    
    await this.apolloServer.start();
    this.apolloServer.applyMiddleware({
      app: this.app,
      path: '/api/graphql',
      cors: false,
    });
  }

  initializeErrorHandling() {
    // Error handler global
    this.app.use((error, req, res, next) => {
      const correlationId = req.correlationId || 'unknown';
      
      logger.error({
        correlationId,
        error: error.message,
        stack: error.stack,
        endpoint: req.originalUrl,
        method: req.method,
      });
      
      // Erros de validação
      if (error.name === 'ValidationError') {
        return res.status(422).json({
          error: 'Erro de validação',
          details: error.details,
          code: 'VALIDATION_ERROR',
          correlationId,
        });
      }
      
      // Erros de autenticação/autorização
      if (error.name === 'AuthenticationError') {
        return res.status(401).json({
          error: 'Não autenticado',
          code: 'UNAUTHENTICATED',
          correlationId,
        });
      }
      
      if (error.name === 'AuthorizationError') {
        return res.status(403).json({
          error: 'Não autorizado',
          code: 'UNAUTHORIZED',
          correlationId,
        });
      }
      
      // Erros de negócio
      if (error.name === 'BusinessError') {
        return res.status(409).json({
          error: error.message,
          code: error.code || 'BUSINESS_ERROR',
          correlationId,
        });
      }
      
      // Erro interno
      res.status(500).json({
        error: 'Erro interno do servidor',
        code: 'INTERNAL_ERROR',
        correlationId,
      });
    });
  }

  async start() {
    await this.initialize();
    
    const server = this.app.listen(config.port, () => {
      logger.info(`Servidor iniciado na porta ${config.port}`);
      logger.info(`REST API: http://localhost:${config.port}/api/v1`);
      logger.info(`GraphQL: http://localhost:${config.port}/api/graphql`);
      logger.info(`Health: http://localhost:${config.port}/health`);
    });
    
    // Health checks para graceful shutdown
    createTerminus(server, {
      signals: ['SIGTERM', 'SIGINT'],
      timeout: 10000,
      healthChecks: {
        '/health/ready': async () => {
          const checks = {
            database: await this.checkDatabase(),
            cache: await this.checkCache(),
          };
          
          const isReady = Object.values(checks).every(check => check.healthy);
          
          return {
            status: isReady ? 'ok' : 'error',
            checks,
          };
        },
        '/health/live': () => ({ status: 'ok' }),
      },
      onSignal: async () => {
        logger.info('Iniciando graceful shutdown...');
        
        if (this.apolloServer) {
          await this.apolloServer.stop();
        }
        
        await closeDatabase();
        
        logger.info('Graceful shutdown completo');
      },
      onShutdown: () => {
        logger.info('Servidor desligado');
      },
      logger: (msg, err) => {
        logger.error(msg, err);
      },
    });
    
    this.server = server;
    return server;
  }

  async checkDatabase() {
    try {
      const db = require('./data/database').getConnection();
      await db.raw('SELECT 1');
      return { healthy: true };
    } catch (error) {
      logger.error('Health check - Database falhou:', error);
      return { healthy: false, error: error.message };
    }
  }

  async checkCache() {
    try {
      const cache = require('./infrastructure/cache/redis').getClient();
      await cache.ping();
      return { healthy: true };
    } catch (error) {
      logger.error('Health check - Cache falhou:', error);
      return { healthy: false, error: error.message };
    }
  }
}

// Inicializar aplicação
if (require.main === module) {
  const app = new App();
  app.start().catch(error => {
    logger.error('Falha ao iniciar aplicação:', error);
    process.exit(1);
  });
}

module.exports = App;
```

## 3. Middleware de Correlation ID (src/api/rest/middleware/correlation-id.js)

```javascript
const { v4: uuidv4 } = require('uuid');

module.exports = (req, res, next) => {
  // Usar correlation ID do header ou gerar novo
  const correlationId = req.headers['x-correlation-id'] || uuidv4();
  
  // Adicionar ao request
  req.correlationId = correlationId;
  
  // Adicionar ao response header
  res.setHeader('X-Correlation-ID', correlationId);
  
  next();
};
```
