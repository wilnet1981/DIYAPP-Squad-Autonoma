# DIYAPP Evolution - V10 Core - Arquitetura e Pipeline

## 1. Documento de Arquitetura (ADR-001)

### ADR-001: Arquitetura de Microsserviços e Observabilidade

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista Segurança

**CONTEXTO:**
DIYAPP evoluiu para uma plataforma complexa com múltiplos serviços. A arquitetura monolítica atual:
- Dificulta deploy independente
- Limita escalabilidade horizontal
- Cria acoplamento técnico entre times
- Dificulta observabilidade e troubleshooting

**DECISÃO:**
Adotar arquitetura de microsserviços baseada em containers Docker, com:
1. Kubernetes para orquestração
2. Service Mesh (Istio) para comunicação entre serviços
3. OpenTelemetry para telemetria unificada
4. Grafana + Loki + Tempo para observabilidade
5. CI/CD com GitOps (ArgoCD) e rollback automático

**OPÇÕES CONSIDERADAS:**

**Opção A: Monólito modularizado**
- Prós: Simplicidade operacional, debugging mais fácil
- Contras: Escalabilidade limitada, deploy acoplado, dificuldade em adotar novas tecnologias

**Opção B: Microsserviços com comunicação direta**
- Prós: Desacoplamento, deploy independente
- Contras: Complexidade operacional, observabilidade fragmentada

**Opção C: Microsserviços com Service Mesh (ESCOLHIDA)**
- Prós: Observabilidade unificada, resiliência, segurança, deploy granular
- Contras: Curva de aprendizado, overhead de recursos

**CONSEQUÊNCIAS:**
- Positivas: Escalabilidade ilimitada, deploy independente, observabilidade completa
- Negativas: Complexidade aumentada, necessidade de expertise em Kubernetes
- Riscos: Latência do service mesh, custo de infraestrutura

**REVISÃO:** 2024-04-15

### 3.2 Padrões de Microsserviços

```typescript
// src/shared/libs/otel.ts
import { NodeSDK } from '@opentelemetry/sdk-node';
import { getNodeAutoInstrumentations } from '@opentelemetry/auto-instrumentations-node';
import { OTLPTraceExporter } from '@opentelemetry/exporter-trace-otlp-grpc';
import { OTLPMetricExporter } from '@opentelemetry/exporter-metrics-otlp-grpc';
import { Resource } from '@opentelemetry/resources';
import { SemanticResourceAttributes } from '@opentelemetry/semantic-conventions';

export const initTelemetry = (serviceName: string) => {
  const sdk = new NodeSDK({
    resource: new Resource({
      [SemanticResourceAttributes.SERVICE_NAME]: serviceName,
      [SemanticResourceAttributes.DEPLOYMENT_ENVIRONMENT]: process.env.NODE_ENV || 'development',
    }),
    traceExporter: new OTLPTraceExporter({
      url: process.env.OTEL_EXPORTER_OTLP_ENDPOINT || 'http://otel-collector:4317',
    }),
    metricExporter: new OTLPMetricExporter({
      url: process.env.OTEL_EXPORTER_OTLP_ENDPOINT || 'http://otel-collector:4317',
    }),
    instrumentations: [getNodeAutoInstrumentations()],
  });

  sdk.start();
  return sdk;
};

// src/shared/libs/logger.ts
import winston from 'winston';
import { WinstonTransport as AxiomTransport } from '@axiomhq/winston';

export const createLogger = (serviceName: string) => {
  return winston.createLogger({
    level: process.env.LOG_LEVEL || 'info',
    defaultMeta: { service: serviceName },
    format: winston.format.combine(
      winston.format.timestamp(),
      winston.format.json(),
      winston.format.errors({ stack: true })
    ),
    transports: [
      new winston.transports.Console(),
      new AxiomTransport({
        dataset: process.env.AXIOM_DATASET || 'diyapp-logs',
        token: process.env.AXIOM_TOKEN,
      }),
    ],
  });
};
```

```dockerfile
# src/auth-service/Dockerfile
FROM node:18-alpine AS builder

WORKDIR /app

# Instala dependências
COPY package*.json ./
RUN npm ci --only=production

# Copia código
COPY . .

# Compila TypeScript
RUN npm run build

# Runtime image
FROM node:18-alpine

WORKDIR /app

# Instala apenas runtime necessário
RUN apk add --no-cache tini curl

# Copia dependências e build
COPY --from=builder /app/node_modules ./node_modules
COPY --from=builder /app/dist ./dist
COPY --from=builder /app/package.json ./

# Configura usuário não-root
RUN addgroup -g 1001 -S nodejs && \
    adduser -S nodejs -u 1001 && \
    chown -R nodejs:nodejs /app

USER nodejs

# Health check
HEALTHCHECK --interval=30s --timeout=3s --start-period=5s --retries=3 \
  CMD curl -f http://localhost:3000/health || exit 1

# Entrypoint
ENTRYPOINT ["/sbin/tini", "--"]
CMD ["node", "dist/index.js"]
```

```yaml
name: CI Pipeline

on:
  pull_request:

        run: npm run type-check
        
      - name: Unit tests
        run: npm test -- --coverage
        
      - name: Upload coverage
        uses: codecov/codecov-action@v3
        
      - name: Build containers
        run: |
          docker build -t diyapp-auth:${GITHUB_SHA} ./src/auth-service
          docker build -t diyapp-api:${GITHUB_SHA} ./src/api-gateway
          
      - name: Trivy vulnerability scanner
        uses: aquasecurity/trivy-action@master
        with:
          image-ref: 'diyapp-auth:${GITHUB_SHA}'
          format: 'sarif'
          output: 'trivy-results.sarif'
          
      - name: Upload Trivy results
        uses: github/codeql-action/upload-sarif@v2
        with:
          sarif_file: 'trivy-results.sarif'
          
  integration-test:
    runs-on: ubuntu-latest
    needs: quality-gate
    services:
      postgres:
        image: postgres:15
        env:
          POSTGRES_PASSWORD: postgres
        options: >-
          --health-cmd pg_isready
          --health-interval 10s
          --health-timeout 5s
          --health-retries 5
      redis:
        image: redis:7-alpine
        options: >-
          --health-cmd "redis-cli ping"
          --health-interval 10s
          --health-timeout 5s
          --health-retries 5
          
    steps:
      - uses: actions/checkout@v3
      
      - name: Run integration tests
        run: npm run test:integration
        
      - name: Load test

          k6 run --vus 10 --duration 30s ./tests/load-test.js
```

### 4.2 CD com ArgoCD e Rollback Automático

```yaml
# .github/workflows/cd.yml
name: CD Pipeline

on:
  push:
    branches: [main]
    
env:
  REGISTRY: ghcr.io
  IMAGE_NAME: ${{ github.repository }}

jobs:
  deploy-staging:
    runs-on: ubuntu-latest
    environment: staging
    steps:
      - uses: actions/checkout@v3
      
      - name: Log in to Container Registry
        uses: docker/login-action@v2
        with:
          registry: ${{ env.REGISTRY }}
          username: ${{ github.actor }}
          password: ${{ secrets.GITHUB_TOKEN }}
          
      - name: Build and push
        uses: docker/build-push-action@v4
        with:
          context: .
          push: true
          tags: |
            ${{ env.REGISTRY }}/${{ env.IMAGE_NAME }}:latest
            ${{ env.REGISTRY }}/${{ env.IMAGE_NAME }}:${{ github.sha }}
          cache-from: type=gha
          cache-to: type=gha,mode=max
          
      - name: Update Helm chart
        run: |
          sed -i "s/tag: .*/tag: ${{ github.sha }}/" charts/diyapp/values.yaml
          git config user.name "GitHub Actions"
          git config user.email "actions@github.com"
          git add charts/diyapp/values.yaml
          git commit -m "Update image tag to ${{ github.sha }}"
          git push
          
      - name: Sync ArgoCD
        run: |
          curl -X POST ${{ secrets.ARGOCD_SERVER }}/api/v1/applications/diyapp-staging/sync \
            -H "Authorization: Bearer ${{ secrets.ARGOCD_TOKEN }}" \
            -H "Content-Type: application/json" \
            -d '{"revision":"main"}'
            
      - name: Run smoke tests
        run: |
          ./scripts/smoke-test.sh ${{ secrets.STAGING_URL }}
          
      - name: Rollback on failure
        if: failure()
        run: |
          # Rollback automático se smoke tests falharem
          curl -X PUT ${{ secrets.ARGOCD_SERVER }}/api/v1/applications/diyapp-staging \
            -H "Authorization: Bearer ${{ secrets.ARGOCD_TOKEN }}" \
            -H "Content-Type: application/json" \
            -d '{"spec":{"source":{"targetRevision":"HEAD~1"}}}'
            
  canary-production:
    needs: deploy-staging
    runs-on: ubuntu-latest
    environment: production
    steps:
      - name: Deploy canary (10%)
        run: |
          kubectl set image deployment/diyapp-api \
            diyapp-api=${{ env.REGISTRY }}/${{ env.IMAGE_NAME }}:${{ github.sha }} \
            -n production
          kubectl scale deployment/diyapp-api --replicas=1 -n production
          
      - name: Monitor canary
        run: |
          ./scripts/monitor-canary.sh
          
      - name: Full rollout
        if: success()
        run: |
          kubectl scale deployment/diyapp-api --replicas=10 -n production
```

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: api-gateway
  namespace: diyapp
  labels:
    app: api-gateway
    version: v1
spec:
  replicas: 3
  selector:
    matchLabels:
      app: api-gateway
  template:
    metadata:
      labels:
        app: api-gateway
        version: v1
      annotations:
        sidecar.istio.io/inject: "true"
        prometheus.io/scrape: "true"
        prometheus.io/port: "3000"
    spec:
      containers:
      - name: api-gateway
        image: ghcr.io/org/diyapp-api:latest
        ports:
        - containerPort: 3000
        env:
        - name: NODE_ENV
          value: "production"
        - name: OTEL_EXPORTER_OTLP_ENDPOINT
          value: "http://otel-collector.observability:4317"
        resources:
          requests:
            memory: "256Mi"
            cpu: "100m"
          limits:
            memory: "512Mi"
            cpu: "500m"
        livenessProbe:
          httpGet:
            path: /health
            port: 3000
          initialDelaySeconds: 30
          periodSeconds: 10
        readinessProbe:
          httpGet:
            path: /ready
            port: 3000
          initialDelaySeconds: 5
          periodSeconds: 5
---
apiVersion: v1
kind: Service
metadata:
  name: api-gateway
  namespace: diyapp
spec:
  selector:
    app: api-gateway
  ports:
  - port: 80
    targetPort: 3000
    name: http
---
apiVersion: networking.istio.io/v1beta1
kind: VirtualService
metadata:
  name: api-gateway
  namespace: diyapp
spec:
  hosts:
  - api.diyapp.com
  gateways:
  - diyapp-gateway
  http:
  - route:
    - destination:
        host: api-gateway
        port:
          number: 80
      weight: 100
    retries:
      attempts: 3
      perTryTimeout: 2s
    timeout: 10s
```

class DatabaseConnectionError(Exception):
    """Exceção para erros de conexão com banco"""
    pass

class DatabaseClient:
    """Cliente de banco com pool de conexões e circuit breaker"""
    
    def __init__(self):
        self.settings = get_settings()
        self.engine = None
        self.async_session_factory = None
        self.circuit_breaker = CircuitBreaker(
            failure_threshold=5,
            recovery_timeout=60,
            name="postgresql"
        )
        self._connection_pool = None
    
    async def connect(self):
        """Estabelece conexão com pool"""
        try:
            if not self.circuit_breaker.allow_request():
                raise DatabaseConnectionError("Circuit breaker aberto para database")
            
            # Pool nativo asyncpg para performance
            self._connection_pool = await asyncpg.create_pool(
                dsn=self.settings.DATABASE_URL,
                min_size=5,
                max_size=self.settings.DB_POOL_SIZE,
                max_queries=50000,
                max_inactive_connection_lifetime=300,
                command_timeout=self.settings.DATABASE_TIMEOUT
            )
            
            # SQLAlchemy para ORM
            self.engine = create_async_engine(
                self.settings.DATABASE_URL.replace("postgresql://", "postgresql+asyncpg://"),
                pool_size=self.settings.DB_POOL_SIZE,
                max_overflow=self.settings.DB_MAX_OVERFLOW,
                pool_timeout=self.settings.DB_POOL_TIMEOUT,
                pool_pre_ping=True,  # Verifica conexão antes de usar
                echo=False
            )
            
            self.async_session_factory = async_sessionmaker(
                self.engine,
                class_=AsyncSession,
                expire_on_commit=False,
                autocommit=False
            )
            
            self.circuit_breaker.record_success()
            logger.info("Database connection pool established")
            
        except Exception as e:
            self.circuit_breaker.record_failure()
            logger.error(f"Database connection failed: {str(e)}")
            raise DatabaseConnectionError(f"Failed to connect to database: {str(e)}")
    
    async def disconnect(self):
        """Fecha conexões"""
        if self._connection_pool:
            await self._connection_pool.close()
        if self.engine:
            await self.engine.dispose()
        logger.info("Database connections closed")
    
    @asynccontextmanager
    async def get_session(self) -> AsyncGenerator[AsyncSession, None]:
        """Context manager para sessões de banco"""
        if not self.async_session_factory:
            raise DatabaseConnectionError("Database not connected")
        
        session = self.async_session_factory()
        try:
            yield session
            await session.commit()
        except Exception as e:
            await session.rollback()
            logger.error(f"Database session error: {str(e)}", exc_info=True)
            raise
        finally:
            await session.close()
    
    async def raw_query(self, query: str, *args) -> list:
        """Executa query raw com pool asyncpg para performance"""
        if not self._connection_pool:
            raise DatabaseConnectionError("Connection pool not available")
        
        async with self._connection_pool.acquire() as connection:
            try:
                # PARAMETRIZAÇÃO: Nunca concatena strings
                result = await connection.fetch(query, *args)
                return [dict(row) for row in result]
            except Exception as e:
                logger.error(f"Raw query failed: {query[:100]}...", exc_info=True)
                raise
    
    async def health_check(self) -> bool:
        """Health check do banco"""
        try:
            async with self.get_session() as session:
                await session.execute("SELECT 1")
            return True
        except Exception as e:
            logger.error(f"Database health check failed: {str(e)}")
            return False

class CacheMiss(Exception):

    pass

class RedisClient:
    """Cliente Redis com circuit breaker e fallback"""
    
    def __init__(self):
        self.settings = get_settings()
        self.client: Optional[redis.Redis] = None
        self.circuit_breaker = CircuitBreaker(
            failure_threshold=settings.REDIS_CIRCUIT_BREAKER_FAILURES,
            recovery_timeout=settings.REDIS_CIRCUIT_BREAKER_TIMEOUT,
            name="redis"
        )
        self._is_connected = False
    
    async def connect(self):
        """Conecta ao Redis"""
        try:
            if not self.circuit_breaker.allow_request():
                raise ConnectionError("Circuit breaker aberto para Redis")
            
            self.client = redis.from_url(
                self.settings.REDIS_URL,
                socket_timeout=self.settings.REDIS_TIMEOUT,
                socket_connect_timeout=self.settings.REDIS_TIMEOUT,
                retry_on_timeout=True,
                max_connections=50,
                decode_responses=True
            )
            
            # Testa conexão
            await self.client.ping()
            self._is_connected = True
            self.circuit_breaker.record_success()
            logger.info("Redis connection established")
            
        except (ConnectionError, TimeoutError, RedisError) as e:
            self.circuit_breaker.record_failure()
            self._is_connected = False
            logger.error(f"Redis connection failed: {str(e)}")
            # Não levanta exceção - cache é degradável
    
    async def disconnect(self):
        """Desconecta do Redis"""
        if self.client:
            await self.client.close()
        self._is_connected = False
        logger.info("Redis connection closed")
    
    def _safe_operation(func):
        """Decorator para operações seguras com fallback"""
        @wraps(func)
        async def wrapper(self, *args, **kwargs):
            if not self._is_connected or not self.circuit_breaker.allow_request():
                raise CacheMiss("Redis unavailable")
            
            try:
                result = await func(self, *args, **kwargs)
                self.circuit_breaker.record_success()
                return result
            except (RedisError, TimeoutError, ConnectionError) as e:
                self.circuit_breaker.record_failure()
                logger.warning(f"Redis operation failed: {func.__name__}, {str(e)}")
                raise CacheMiss(f"Redis operation failed: {str(e)}")
            except asyncio.TimeoutError:
                self.circuit_breaker.record_failure()
                logger.warning(f"Redis timeout: {func.__name__}")
                raise CacheMiss("Redis timeout")
        
        return wrapper
    
    @_safe_operation
    async def get(self, key: str) -> Optional[Any]:
        """Obtém valor do cache"""
        value = await self.client.get(key)
        if value:
            try:
                return json.loads(value)
            except json.JSONDecodeError:
                return value
        return None
    
    @_safe_operation
    async def set(self, key: str, value: Any, ttl: int = 3600) -> bool:
        """Define valor no cache com TTL"""
        if isinstance(value, (dict, list)):
            value = json.dumps(value)
        return await self.client.setex(key, ttl, value)
    
    @_safe_operation
    async def delete(self, key: str) -> bool:
        """Remove valor do cache"""
        return await self.client.delete(key) > 0
    
    @_safe_operation
    async def increment(self, key: str, amount: int = 1) -> int:
        """Incrementa valor (para rate limiting)"""
        return await self.client.incrby(key, amount)
    
    @_safe_operation
    async def expire(self, key: str, ttl: int) -> bool:
        """Define TTL para chave existente"""
        return await self.client.expire(key, ttl)
    
    async def health_check(self) -> bool:
        """Health check do Redis"""
        try:
            if self._is_connected and self.client:
                await self.client.ping()
                return True
            return False
        except Exception:
            return False

# DIYAPP Evolution - V10 Core - Arquitetura e Pipeline

## 1. Documento de Arquitetura (ADR-001)

### ADR-001: Arquitetura de Microsserviços com Observabilidade e CI/CD Automatizado

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead V10 Core

**CONTEXTO:**
DIYAPP evoluiu para uma plataforma complexa com múltiplos serviços. A arquitetura monolítica atual limita:
- Deploy independente por squad
- Escalabilidade horizontal
- Isolamento de falhas
- Experimentação com novos modelos de IA

Necessitamos de uma arquitetura baseada em microsserviços com:
- Observabilidade completa (métricas, logs, traces)
- Pipeline CI/CD com rollback automático
- 100% de autonomia para squads
- Estabilidade garantida mesmo durante refatorações paralelas

**DECISÃO:**
Adotar arquitetura de microsserviços containerizados com:
1. Kubernetes como orquestrador
2. Service Mesh (Istio) para comunicação entre serviços
3. OpenTelemetry para telemetria unificada
4. Grafana + Loki + Tempo para observabilidade
5. ArgoCD para GitOps e deploy contínuo
6. Pipeline CI/CD com rollback automático baseado em métricas

**OPÇÕES CONSIDERADAS:**

**Opção A: Kubernetes + Istio + OpenTelemetry**
- Prós: Ecossistema maduro, suporte a service mesh nativo, observabilidade integrada
- Contras: Complexidade inicial maior, curva de aprendizado

**Opção B: Docker Swarm + Instrumentação customizada**
- Prós: Mais simples de configurar, menos overhead
- Contras: Menos recursos de observabilidade, menos maduro para produção

**Opção C: Serverless (AWS Lambda/Azure Functions)**
- Prós: Escalabilidade automática, menos gerenciamento de infra
- Contras: Vendor lock-in, custo variável, menos controle

**Opção escolhida: A** - Justificativa: Necessitamos de controle total, observabilidade profunda e maturidade para produção.

**CONSEQUÊNCIAS:**
**Positivas:**
- Deploy independente por squad
- Escalabilidade granular por serviço
- Observabilidade unificada
- Rollback automático baseado em métricas
- Isolamento de falhas

**Negativas:**
- Complexidade operacional aumentada
- Necessidade de expertise em Kubernetes
- Overhead de comunicação entre serviços

**Riscos:**
- Configuração incorreta do service mesh pode causar latência
- Mitigação: Testes de performance rigorosos e canary deployments
- Curva de aprendizado para desenvolvedores
- Mitigação: Documentação extensa e sessões de treinamento

## 5. Configuração OpenTelemetry

```typescript
// libs/telemetry/src/index.ts
import { NodeSDK } from '@opentelemetry/sdk-node';
import { getNodeAutoInstrumentations } from '@opentelemetry/auto-instrumentations-node';
import { OTLPTraceExporter } from '@opentelemetry/exporter-trace-otlp-grpc';
import { OTLPMetricExporter } from '@opentelemetry/exporter-metrics-otlp-grpc';
import { Resource } from '@opentelemetry/resources';
import { SemanticResourceAttributes } from '@opentelemetry/semantic-conventions';
import { PeriodicExportingMetricReader } from '@opentelemetry/sdk-metrics';
import { WinstonInstrumentation } from '@opentelemetry/instrumentation-winston';
import { ExpressInstrumentation } from '@opentelemetry/instrumentation-express';
import { HttpInstrumentation } from '@opentelemetry/instrumentation-http';

export function initTelemetry(serviceName: string) {
  const resource = new Resource({
    [SemanticResourceAttributes.SERVICE_NAME]: serviceName,
    [SemanticResourceAttributes.DEPLOYMENT_ENVIRONMENT]: process.env.NODE_ENV || 'development',
  });

  const traceExporter = new OTLPTraceExporter({
    url: process.env.OTEL_EXPORTER_OTLP_ENDPOINT || 'http://localhost:4317',
  });

  const metricReader = new PeriodicExportingMetricReader({
    exporter: new OTLPMetricExporter({
      url: process.env.OTEL_EXPORTER_OTLP_METRICS_ENDPOINT || 'http://localhost:4318',
    }),
    exportIntervalMillis: 60000,
  });

  const sdk = new NodeSDK({
    resource,
    traceExporter,
    metricReader,
    instrumentations: [
      getNodeAutoInstrumentations({
        '@opentelemetry/instrumentation-fs': { enabled: false },
      }),
      new WinstonInstrumentation(),
      new ExpressInstrumentation(),
      new HttpInstrumentation(),
    ],
  });

  sdk.start();

  // Graceful shutdown
  process.on('SIGTERM', () => {
    sdk.shutdown()
      .then(() => console.log('Telemetry terminated'))
      .catch((error) => console.error('Error terminating telemetry', error))
      .finally(() => process.exit(0));
  });

  return {
    tracer: require('@opentelemetry/api').trace.getTracer(serviceName),
    meter: require('@opentelemetry/api').metrics.getMeter(serviceName),
  };
}
```

import contextlib
from typing import AsyncGenerator, Optional
from sqlalchemy.ext.asyncio import (
    AsyncSession,
    async_sessionmaker,
    create_async_engine,
    AsyncEngine
)
from sqlalchemy.orm import declarative_base
from sqlalchemy.pool import QueuePool
import structlog
from ..config import settings

logger = structlog.get_logger(__name__)

Base = declarative_base()

class Database:
    def __init__(self) -> None:
        self._engine: Optional[AsyncEngine] = None
        self._sessionmaker: Optional[async_sessionmaker[AsyncSession]] = None
    
    async def connect(self) -> None:
        """Initialize database connection with connection pooling"""
        try:
            self._engine = create_async_engine(
                settings.database_url,
                echo=False,
                poolclass=QueuePool,
                pool_size=settings.DATABASE_POOL_SIZE,
                max_overflow=settings.DATABASE_MAX_OVERFLOW,
                pool_pre_ping=True,
                pool_recycle=3600,
            )
            self._sessionmaker = async_sessionmaker(
                self._engine,
                class_=AsyncSession,
                expire_on_commit=False,
                autocommit=False,
                autoflush=False,
            )
            await logger.ainfo("Database connection established")
        except Exception as e:
            await logger.aerror("Database connection failed", error=str(e))
            raise
    
    async def disconnect(self) -> None:
        """Close database connection"""
        if self._engine:
            await self._engine.dispose()
            await logger.ainfo("Database connection closed")
    
    @contextlib.asynccontextmanager
    async def session(self) -> AsyncGenerator[AsyncSession, None]:
        """Get database session with automatic rollback on error"""
        if not self._sessionmaker:
            raise RuntimeError("Database not initialized")
        
        session = self._sessionmaker()
        try:
            yield session
            await session.commit()
        except Exception:
            await session.rollback()
            raise
        finally:
            await session.close()
    
    async def health_check(self) -> bool:
        """Check database health"""
        try:
            async with self.session() as session:
                await session.execute("SELECT 1")
            return True
        except Exception as e:
            await logger.aerror("Database health check failed", error=str(e))
            return False


database = Database()

import redis.asyncio as redis

import structlog

logger = structlog.get_logger(__name__)

class RedisClient:
    def __init__(self):
        self._client: Optional[redis.Redis] = None
        self.circuit_breaker = CircuitBreaker(
            failure_threshold=5,
            recovery_timeout=30,
            name="redis"
        )
    
    async def connect(self) -> None:
        """Initialize Redis connection"""
        try:
            self._client = redis.Redis(
                host=settings.REDIS_HOST,
                port=settings.REDIS_PORT,
                password=settings.REDIS_PASSWORD,
                db=settings.REDIS_DB,
                socket_connect_timeout=settings.REDIS_TIMEOUT,
                socket_timeout=settings.REDIS_TIMEOUT,
                retry_on_timeout=True,
                decode_responses=True,
            )
            # Test connection
            await self._client.ping()
            await logger.ainfo("Redis connection established")
        except Exception as e:
            await logger.aerror("Redis connection failed", error=str(e))
            raise
    
    async def disconnect(self) -> None:
        """Close Redis connection"""
        if self._client:
            await self._client.close()
            await logger.ainfo("Redis connection closed")
    
    @circuit_breaker
    async def get(self, key: str) -> Optional[str]:
        """Get value from cache with circuit breaker"""
        try:
            return await self._client.get(key)
        except (ConnectionError, TimeoutError) as e:
            await logger.awarning("Redis get failed", key=key, error=str(e))
            raise
        except RedisError as e:
            await logger.aerror("Redis get error", key=key, error=str(e))
            return None
    
    @circuit_breaker
    async def set(self, key: str, value: str, ttl: Optional[int] = None) -> bool:
        """Set value in cache with circuit breaker"""
        try:
            if ttl:
                return await self._client.setex(key, ttl, value)
            return await self._client.set(key, value)
        except (ConnectionError, TimeoutError) as e:
            await logger.awarning("Redis set failed", key=key, error=str(e))
            raise
        except RedisError as e:
            await logger.aerror("Redis set error", key=key, error=str(e))
            return False
    
    @circuit_breaker
    async def delete(self, key: str) -> bool:
        """Delete key from cache with circuit breaker"""
        try:
            return await self._client.delete(key) > 0
        except (ConnectionError, TimeoutError) as e:
            await logger.awarning("Redis delete failed", key=key, error=str(e))
            raise
        except RedisError as e:
            await logger.aerror("Redis delete error", key=key, error=str(e))
            return False
    
    @circuit_breaker
    async def health_check(self) -> bool:

def check_data_sources():
    """Verifica disponibilidade das fontes de dados"""
    sources = [
        {'name': 'api_users', 'url': 'https://api.diyapp.com/v1/health'},
        {'name': 'api_orders', 'url': 'https://api.diyapp.com/v1/orders/health'},
        {'name': 'llm_gateway', 'url': 'https://llm.diyapp.com/health'},
        {'name': 'app_events', 'url': 'https://events.diyapp.com/health'}
    ]
    
    unavailable = []
    for source in sources:
        try:
            response = requests.get(source['url'], timeout=10)
            if response.status_code != 200:
                unavailable.append(source['name'])
        except:
            unavailable.append(source['name'])
    
    if unavailable:
        raise Exception(f"Fontes indisponíveis: {unavailable}")
    
    return "Todas as fontes disponíveis"

```javascript
const OAuth2Strategy = require('passport-oauth2').Strategy;

class CustomOAuth2Strategy extends OAuth2Strategy {
  constructor(options, verify) {
    super(options, verify);
    this.name = 'oauth2-custom';
    this._userProfileURL = options.userProfileURL;
  }

  userProfile(accessToken, done) {
    axios.get(this._userProfileURL, {
      headers: {
        'Authorization': `Bearer ${accessToken}`,
        'User-Agent': 'DIYAPP-V10-Security'
      }
    })
    .then(response => {
      // Validate and sanitize user profile
      const profile = this.sanitizeProfile(response.data);
      done(null, profile);
    })
    .catch(error => {
      logger.error('OAuth2 profile fetch error:', error);
      done(new Error('Failed to fetch user profile'));
    });
  }

  sanitizeProfile(profile) {
    // Remove any potentially dangerous fields
    const safeProfile = {
      id: String(profile.id || profile.sub || ''),
      email: this.sanitizeEmail(profile.email),
      name: this.sanitizeString(profile.name || ''),
      given_name: this.sanitizeString(profile.given_name || ''),
      family_name: this.sanitizeString(profile.family_name || ''),
      picture: this.validateURL(profile.picture),
      provider: 'oauth2'
    };

    // Add only verified email
    if (profile.email_verified) {
      safeProfile.email_verified = true;
    }

    return safeProfile;
  }

  sanitizeEmail(email) {
    if (!email || typeof email !== 'string') return '';
    // Basic email validation and sanitization
    const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    return emailRegex.test(email) ? email.toLowerCase().trim() : '';
  }

  sanitizeString(str) {
    if (!str || typeof str !== 'string') return '';
    // Remove HTML tags and limit length
    return str.replace(/<[^>]*>/g, '').substring(0, 100).trim();
  }

  validateURL(url) {
    if (!url || typeof url !== 'string') return '';
    try {
      const parsed = new URL(url);
      // Allow only HTTPS in production
      if (process.env.NODE_ENV === 'production' && parsed.protocol !== 'https:') {
        return '';
      }
      return url;
    } catch {
      return '';
    }
  }
}
```

```javascript
const OpenIDConnectStrategy = require('passport-openidconnect').Strategy;
const jwt = require('jsonwebtoken');
const logger = require('../utils/logger');

const oidcStrategy = new OpenIDConnectStrategy({
  issuer: process.env.OIDC_ISSUER || 'https://oidc-provider.com',
  authorizationURL: process.env.OIDC_AUTHORIZATION_URL || 'https://oidc-provider.com/authorize',
  tokenURL: process.env.OIDC_TOKEN_URL || 'https://oidc-provider.com/token',
  userInfoURL: process.env.OIDC_USERINFO_URL || 'https://oidc-provider.com/userinfo',
  clientID: process.env.OIDC_CLIENT_ID,
  clientSecret: process.env.OIDC_CLIENT_SECRET,
  callbackURL: process.env.OIDC_CALLBACK_URL || 'http://localhost:3000/api/auth/oidc/callback',
  scope: ['openid', 'email', 'profile'],
  state: true,
  nonce: true // Important for OIDC security
}, (issuer, sub, profile, jwtClaims, accessToken, refreshToken, params, done) => {
  try {
    // Validate ID Token
    const idToken = params.id_token;
    
    // Verify the ID token signature (if provided by issuer)
    if (idToken && process.env.OIDC_JWKS_URI) {
      // In production, verify against JWKS
      // This is a simplified version
      const decoded = jwt.decode(idToken, { complete: true });
      
      // Validate claims
      if (decoded.payload.iss !== issuer) {
        throw new Error('Invalid issuer in ID token');
      }
      
      if (decoded.payload.aud !== process.env.OIDC_CLIENT_ID) {
        throw new Error('Invalid audience in ID token');
      }
      
      // Check token expiration
      if (decoded.payload.exp < Date.now() / 1000) {
        throw new Error('ID token has expired');
      }
    }

    // Sanitize and normalize profile
    const sanitizedProfile = {
      id: sub,
      issuer: issuer,
      email: profile.email ? profile.email.toLowerCase().trim() : '',
      email_verified: profile.email_verified || false,
      name: (profile.name || '').replace(/<[^>]*>/g, '').substring(0, 100),
      given_name: (profile.given_name || '').replace(/<[^>]*>/g, '').substring(0, 50),
      family_name: (profile.family_name || '').replace(/<[^>]*>/g, '').substring(0, 50),
      picture: validatePictureURL(profile.picture),
      claims: {
        // Store only non-sensitive claims
        auth_time: jwtClaims.auth_time,
        acr: jwtClaims.acr,
        amr: jwtClaims.amr
      }
    };

    logger.info('OIDC authentication successful', {
      userId: sub,
      issuer: issuer,
      email: sanitizedProfile.email
    });

    const user = {
      id: sub,
      email: sanitizedProfile.email,
      name: sanitizedProfile.name,
      accessToken: accessToken,
      refreshToken: refreshToken,
      idToken: idToken,
      provider: 'oidc',
      claims: sanitizedProfile.claims
    };

    return done(null, user);
  } catch (error) {
    logger.error('OIDC verification error:', {
      error: error.message,
      issuer: issuer,
      sub: sub
    });
    return done(error, false);
  }
});

function validatePictureURL(url) {
  if (!url || typeof url !== 'string') return '';
  try {
    const parsed = new URL(url);
    // Security restrictions for profile pictures
    const allowedDomains = process.env.ALLOWED_IMAGE_DOMAINS 
      ? process.env.ALLOWED_IMAGE_DOMAINS.split(',')
      : ['secure.gravatar.com', 'lh3.googleusercontent.com'];
    
    if (!allowedDomains.includes(parsed.hostname)) {
      return '';
    }
    
    if (process.env.NODE_ENV === 'production' && parsed.protocol !== 'https:') {
      return '';
    }
    
    return url;
  } catch {
    return '';
  }
}
```

```javascript
describe('API Integration: Chat Endpoints', () => {
```

A API utiliza autenticação baseada em JWT (JSON Web Token). Para acessar endpoints protegidos:


   ```bash
   curl -X GET https://api.diyapp.com/v10/projects \
   ```

3. **Renove o token quando expirar:**
   ```bash
   curl -X POST https://api.diyapp.com/v10/auth/refresh \
     -H "Authorization: Bearer SEU_REFRESH_TOKEN"
   ```

```javascript
  try {
    const { audioBase64, sessionId, context } = req.body;
    const startTime = Date.now();

    // 1. Transcrever áudio usando Whisper
    const transcription = await openai.audio.transcriptions.create({
      file: Buffer.from(audioBase64, 'base64'),
      model: "whisper-1",
      response_format: "text"
    });

    // 2. Processar com GPT-4o para entender intenção
    const completion = await openai.chat.completions.create({
      model: "gpt-4o",

          Responda em JSON: { "intent": "comando", "parameters": {}, "response": "resposta em texto", "action": "acao_no_diyapp" }`
        },
        {
          role: "user",
          content: transcription
        }
      ],
      response_format: { type: "json_object" }
    });

    const result = JSON.parse(completion.choices[0].message.content);
    const responseTime = Date.now() - startTime;

    // 3. Registrar métrica
    db.run(
      `INSERT INTO voice_metrics (session_id, command, intent, response_time_ms, success) 
       VALUES (?, ?, ?, ?, ?)`,
      [sessionId, transcription, result.intent, responseTime, true]
    );

    // 4. Executar ação no DIYAPP (simulação)
    const actionResult = await executeDIYAPPAction(result.action, result.parameters);

      action: actionResult,
      responseTime,
      timestamp: new Date().toISOString()
    });

      intent: result.intent,
      responseTime
    });
```
