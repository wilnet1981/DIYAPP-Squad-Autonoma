# DIYAPP Evolution - V12 Core - Arquitetura de Microsserviços

## Arquitetura de Microsserviços V12 - Documento Técnico

### 1. ADR-001: Decisão de Arquitetura de Microsserviços

**ADR-001: Arquitetura Baseada em Microsserviços para V12**
**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista Backend

**CONTEXTO:**
A versão atual do DIYAPP (V11) utiliza arquitetura monolítica que está atingindo limites de escalabilidade. O crescimento da base de usuários (+300% nos últimos 6 meses) e a complexidade das novas features de IA estão causando:
1. Acoplamento excessivo entre módulos
2. Dificuldade em deploy independente
3. Escalabilidade vertical limitada
4. Single point of failure
5. Dificuldade em adotar diferentes stacks tecnológicas por feature

**DECISÃO:**
Adotar arquitetura de microsserviços com os seguintes princípios:
1. Cada domínio de negócio será um microsserviço independente
2. Comunicação síncrona via gRPC com protobuf
3. Comunicação assíncrona via Apache Kafka para eventos de domínio
4. API Gateway para exposição externa
5. Service Mesh (Istio) para observabilidade e resiliência
6. Banco de dados por serviço (Database per Service)

**OPÇÕES CONSIDERADAS:**
- **Opção A:** Arquitetura monolítica modularizada
  - Prós: Simplicidade inicial, transações ACID fáceis
  - Contras: Escalabilidade limitada, deploy acoplado
- **Opção B:** Microsserviços com REST/HTTP
  - Prós: Simples de implementar, ampla adoção
  - Contras: Overhead de serialização JSON, sem contrato forte
- **Opção C:** Microsserviços com gRPC + Eventos (ESCOLHIDA)
  - Prós: Performance superior, contrato forte com protobuf, streaming bidirecional
  - Contras: Curva de aprendizado, tooling menos maduro que REST

**CONSEQUÊNCIAS:**
**Positivas:**
- Escalabilidade horizontal ilimitada
- Deploy independente por serviço
- Resiliência e tolerância a falhas
- Adoção de diferentes tecnologias por serviço
- Melhor isolamento de falhas

**Negativas:**
- Complexidade de distribuição aumentada
- Transações distribuídas necessárias
- Monitoramento mais complexo
- Latência de rede entre serviços
- Overhead operacional

**Riscos:**
1. **Latência de rede:** Mitigar com cache Redis e design assíncrono
2. **Consistência eventual:** Implementar SAGA Pattern e compensações
3. **Monitoramento distribuído:** Usar Jaeger + Prometheus + Grafana
4. **Deploy complexo:** Automação completa com GitOps

**REVISÃO:** 2024-04-15

```proto
syntax = "proto3";

package diyapp.v12.template;

import "google/protobuf/timestamp.proto";

service TemplateService {
  rpc CreateTemplate(CreateTemplateRequest) returns (TemplateResponse);
  rpc GetTemplate(GetTemplateRequest) returns (TemplateResponse);
  rpc UpdateTemplate(UpdateTemplateRequest) returns (TemplateResponse);
  rpc DeleteTemplate(DeleteTemplateRequest) returns (DeleteTemplateResponse);
  rpc StreamTemplates(StreamTemplatesRequest) returns (stream TemplateResponse);
}

message CreateTemplateRequest {
  string name = 1;
  string description = 2;
  map<string, string> metadata = 3;
}

message TemplateResponse {
  string id = 1;
  string name = 2;
  string status = 3;
  google.protobuf.Timestamp created_at = 4;
  google.protobuf.Timestamp updated_at = 5;
}

message GetTemplateRequest {
  string id = 1;
}
```

### 8. Template de Código do Serviço

```typescript
// services/template-service/src/index.ts
import 'reflect-metadata';
import { Server } from './server';
import { Logger } from './utils/logger';
import { Config } from './config/config';
import { Metrics } from './utils/metrics';
import { KafkaConsumer } from './events/kafka-consumer';

class Application {
  private server: Server;
  private logger: Logger;
  private kafkaConsumer: KafkaConsumer;

  constructor() {
    this.logger = Logger.getInstance();
    this.server = new Server();
    this.kafkaConsumer = new KafkaConsumer();
  }

  async start(): Promise<void> {
    try {
      // Inicializar métricas
      await Metrics.init();
      
      // Conectar ao Kafka
      await this.kafkaConsumer.connect();
      
      // Iniciar servidor
      await this.server.start();
      
      this.logger.info('Template Service started successfully', {
        port: Config.PORT,
        environment: Config.NODE_ENV,
        version: Config.VERSION
      });
      
      // Graceful shutdown
      this.setupGracefulShutdown();
      
    } catch (error) {
      this.logger.error('Failed to start Template Service', { error });
      process.exit(1);
    }
  }

  private setupGracefulShutdown(): void {
    const signals: NodeJS.Signals[] = ['SIGTERM', 'SIGINT', 'SIGUSR2'];
    
    signals.forEach(signal => {
      process.on(signal, async () => {
        this.logger.info(`Received ${signal}, starting graceful shutdown`);
        
        try {
          // Parar consumidor Kafka
          await this.kafkaConsumer.disconnect();
          
          // Parar servidor
          await this.server.stop();
          
          // Parar métricas
          await Metrics.stop();
          
          this.logger.info('Graceful shutdown completed');
          process.exit(0);
        } catch (error) {
          this.logger.error('Error during graceful shutdown', { error });
          process.exit(1);
        }
      });
    });
  }
}

// Inicializar aplicação
const app = new Application();
app.start().catch(error => {
  console.error('Fatal error during application startup:', error);
  process.exit(1);
});
```

```typescript
import { CircuitBreakerState } from '../types';

export class CircuitBreaker {
  private state: CircuitBreakerState;
  private failureThreshold: number;
  private resetTimeout: number;
  private halfOpenTimeout: number;

  constructor(
    failureThreshold = 5,
    resetTimeout = 60000, // 1 minute
    halfOpenTimeout = 30000 // 30 seconds
  ) {
    this.state = { isOpen: false, failureCount: 0 };
    this.failureThreshold = failureThreshold;
    this.resetTimeout = resetTimeout;
    this.halfOpenTimeout = halfOpenTimeout;
  }

  async execute<T>(
    operation: () => Promise<T>,
    fallback?: () => Promise<T>
  ): Promise<T> {
    if (this.state.isOpen) {
      const now = new Date();
      if (this.state.nextRetryTime && now > this.state.nextRetryTime) {
        // Try half-open state
        this.state.isOpen = false;
        this.state.failureCount = 0;
      } else {
        // Circuit is open, return fallback or throw
        if (fallback) {
          console.warn('Circuit breaker open, using fallback');
          return fallback();
        }
        throw new Error('Service unavailable - circuit breaker open');
      }
    }

    try {
      const result = await operation();
      // Success - reset failure count
      this.state.failureCount = 0;
      return result;
    } catch (error) {
      this.state.failureCount++;
      
      if (this.state.failureCount >= this.failureThreshold) {
        this.state.isOpen = true;
        this.state.lastFailureTime = new Date();
        this.state.nextRetryTime = new Date(
          Date.now() + this.resetTimeout
        );
        console.error(`Circuit breaker opened for service after ${this.failureThreshold} failures`);
      }

      if (fallback) {
        console.warn('Operation failed, using fallback', { error });
        return fallback();
      }
      
      throw error;
    }
  }

  getState(): CircuitBreakerState {
    return { ...this.state };
  }

  reset(): void {
    this.state = { isOpen: false, failureCount: 0 };
  }
}

export async function retryWithBackoff<T>(
  operation: () => Promise<T>,
  maxRetries = 3,
  baseDelay = 1000
): Promise<T> {
  let lastError: Error;
  
  for (let attempt = 0; attempt <= maxRetries; attempt++) {
    try {
      return await operation();
    } catch (error) {
      lastError = error as Error;
      
      // Don't retry on 4xx errors
      if (error instanceof Error && error.message.includes('4')) {
        throw error;
      }
      
      if (attempt === maxRetries) {
        break;
      }
      
      // Exponential backoff with jitter
      const delay = baseDelay * Math.pow(2, attempt);
      const jitter = delay * 0.1 * Math.random();
      await new Promise(resolve => setTimeout(resolve, delay + jitter));
    }
  }
  
  throw lastError!;
}
```

```typescript
import { Request, Response, NextFunction } from 'express';
import { z, ZodError } from 'zod';

export function validateRequest(schema: z.ZodSchema) {
  return (req: Request, res: Response, next: NextFunction) => {
    try {
      const validated = schema.parse({
        body: req.body,
        query: req.query,
        params: req.params,
      });
      
      req.body = validated.body;
      req.query = validated.query;
      req.params = validated.params;
      
      next();
    } catch (error) {
      if (error instanceof ZodError) {
        res.status(400).json({
          success: false,
          error: {
            code: 'VALIDATION_ERROR',
            message: 'Invalid request data',
            details: error.errors.map(err => ({
              path: err.path.join('.'),
              message: err.message,
            })),
          },
        });
      } else {
        next(error);
      }
    }
  };
}
```

```typescript
// src/hooks/useWebSocket.ts
import { useEffect, useRef, useCallback, useState } from 'react'
import { io, Socket } from 'socket.io-client'

interface WebSocketMessage {
  type: string
  payload: any
  timestamp: string
}

interface UseWebSocketOptions {
  url: string
  onMessage?: (message: WebSocketMessage) => void
  onConnect?: () => void
  onDisconnect?: () => void
  onError?: (error: Error) => void
  autoReconnect?: boolean
  reconnectAttempts?: number
  reconnectDelay?: number
}

export const useWebSocket = (options: UseWebSocketOptions) => {
  const {
    url,
    onMessage,
    onConnect,
    onDisconnect,
    onError,
    autoReconnect = true,
    reconnectAttempts = 5,
    reconnectDelay = 3000
  } = options

  const socketRef = useRef<Socket | null>(null)
  const reconnectCountRef = useRef(0)
  const [isConnected, setIsConnected] = useState(false)
  const [lastMessage, setLastMessage] = useState<WebSocketMessage | null>(null)

  const connect = useCallback(() => {
    if (socketRef.current?.connected) return

    const socket = io(url, {
      transports: ['websocket', 'polling'],
      reconnection: autoReconnect,
      reconnectionAttempts: reconnectAttempts,
      reconnectionDelay
    })

    socket.on('connect', () => {
      setIsConnected(true)
      reconnectCountRef.current = 0
      onConnect?.()
    })

    socket.on('disconnect', (reason) => {
      setIsConnected(false)
      onDisconnect?.()
    })

    socket.on('error', (error: Error) => {
      onError?.(error)
    })

    socket.on('message', (data: WebSocketMessage) => {
      setLastMessage(data)
      onMessage?.(data)
    })

    socketRef.current = socket
  }, [url, onMessage, onConnect, onDisconnect, onError, autoReconnect, reconnectAttempts, reconnectDelay])

  const disconnect = useCallback(() => {
    if (socketRef.current) {
      socketRef.current.disconnect()
      socketRef.current = null
      setIsConnected(false)
    }
  }, [])

  const send = useCallback((type: string, payload: any) => {
    if (socketRef.current?.connected) {
      socketRef.current.emit('message', {
        type,
        payload,
        timestamp: new Date().toISOString()
      })
      return true
    }
    return false
  }, [])

  useEffect(() => {
    connect()

    return () => {
      disconnect()
    }
  }, [connect, disconnect])

  return {
    isConnected,
    lastMessage,
    send,
    disconnect,
    connect
  }
}
```

class AuthManager:
    """Gerenciador de autenticação JWT/OAuth2.1"""
    
    def __init__(self, db_path: str = "security.db"):
        self.db_path = db_path
        self._init_db()
        self.fernet = Fernet(SecurityConfig.ENCRYPTION_KEY.encode())
        self.logger = self._setup_logger()
        
    def _setup_logger(self) -> logging.Logger:

        logger = logging.getLogger("security_auth")
        logger.setLevel(logging.INFO)
        
        handler = logging.FileHandler("logs/security_auth.log")

        return logger
    
    def _init_db(self):
        """Inicializa banco de dados de segurança"""
        os.makedirs("data", exist_ok=True)
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        # Tabela de usuários
        cursor.execute('''
            CREATE TABLE IF NOT EXISTS users (
                id TEXT PRIMARY KEY,
                email TEXT UNIQUE NOT NULL,
                hashed_password TEXT NOT NULL,
                role TEXT NOT NULL,
                mfa_enabled BOOLEAN DEFAULT 0,
                mfa_secret TEXT,
                failed_attempts INTEGER DEFAULT 0,
                locked_until TIMESTAMP,
                created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
            )
        ''')
        
        # Tabela de tokens (OAuth2.1 compliant)
        cursor.execute('''
            CREATE TABLE IF NOT EXISTS tokens (
                id TEXT PRIMARY KEY,
                user_id TEXT NOT NULL,
                access_token TEXT NOT NULL,
                refresh_token TEXT NOT NULL,
                token_type TEXT DEFAULT 'Bearer',
                scope TEXT DEFAULT '',
                expires_at TIMESTAMP NOT NULL,
                revoked BOOLEAN DEFAULT 0,
                created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
                FOREIGN KEY (user_id) REFERENCES users (id)
            )
        ''')
        
        # Tabela de auditoria de login
        cursor.execute('''
            CREATE TABLE IF NOT EXISTS login_audit (
                id TEXT PRIMARY KEY,
                user_id TEXT,
                email TEXT,
                ip_address TEXT,
                user_agent TEXT,
                success BOOLEAN,
                failure_reason TEXT,
                timestamp TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
                FOREIGN KEY (user_id) REFERENCES users (id)
            )
        ''')
        
        conn.commit()
        conn.close()
    
    def hash_password(self, password: str) -> str:
        """Hash seguro de senha com salt"""
        salt = secrets.token_hex(16)
        return hashlib.pbkdf2_hmac(
            'sha256',
            password.encode(),
            salt.encode(),
            100000
        ).hex() + ":" + salt
    
    def verify_password(self, stored_password: str, provided_password: str) -> bool:
        """Verifica senha hash"""
        try:
            hashed, salt = stored_password.split(":")
            new_hash = hashlib.pbkdf2_hmac(
                'sha256',
                provided_password.encode(),
                salt.encode(),
                100000
            ).hex()
            return hashed == new_hash
        except:
            return False
    
    def create_user(self, email: str, password: str, role: str = "user") -> Tuple[bool, str]:
        """Cria novo usuário com validações de segurança"""
        # Validação de email
        if not re.match(r"[^@]+@[^@]+\.[^@]+", email):
            return False, "Invalid email format"
        
        # Validação de força de senha
        if len(password) < 12:
            return False, "Password must be at least 12 characters"
        
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        try:
            user_id = secrets.token_urlsafe(16)
            hashed_password = self.hash_password(password)
            
            cursor.execute('''
                INSERT INTO users (id, email, hashed_password, role)
                VALUES (?, ?, ?, ?)
            ''', (user_id, email, hashed_password, role))
            
            conn.commit()
            self.logger.info(f"User created: {email} (ID: {user_id})")
            return True, user_id
            
        except sqlite3.IntegrityError:
            return False, "Email already exists"
        finally:
            conn.close()
    
    def authenticate(self, email: str, password: str, ip: str, user_agent: str) -> Dict:
        """Autenticação com rate limiting e auditoria"""
        audit_id = secrets.token_urlsafe(16)
        
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        try:
            # Verifica se usuário está bloqueado
            cursor.execute(
                "SELECT id, hashed_password, role, failed_attempts, locked_until FROM users WHERE email = ?",
                (email,)
            )
            result = cursor.fetchone()
            
            if not result:
                self._log_login_attempt(audit_id, None, email, ip, user_agent, False, "User not found")
                return {"success": False, "error": "Invalid credentials"}
            
            user_id, stored_password, role, failed_attempts, locked_until = result
            
            # Verifica bloqueio
            if locked_until and datetime.strptime(locked_until, "%Y-%m-%d %H:%M:%S") > datetime.utcnow():
                self._log_login_attempt(audit_id, user_id, email, ip, user_agent, False, "Account locked")
                return {"success": False, "error": "Account locked"}
            
            # Verifica senha
            if not self.verify_password(stored_password, password):
                # Incrementa tentativas falhas
                new_attempts = failed_attempts + 1
                if new_attempts >= 5:
                    lock_time = datetime.utcnow() + timedelta(minutes=15)
                    cursor.execute(
                        "UPDATE users SET failed_attempts = ?, locked_until = ? WHERE id = ?",
                        (new_attempts, lock_time.isoformat(), user_id)
                    )
                    reason = "Account locked after 5 failed attempts"
                else:
                    cursor.execute(
                        "UPDATE users SET failed_attempts = ? WHERE id = ?",
                        (new_attempts, user_id)
                    )
                    reason = "Invalid password"
                
                conn.commit()
                self._log_login_attempt(audit_id, user_id, email, ip, user_agent, False, reason)
                return {"success": False, "error": "Invalid credentials"}
            
            # Login bem-sucedido - reseta tentativas
            cursor.execute(
                "UPDATE users SET failed_attempts = 0, locked_until = NULL WHERE id = ?",
                (user_id,)
            )
            conn.commit()
            
            # Gera tokens JWT
            tokens = self._generate_tokens(user_id, role)
            
            self._log_login_attempt(audit_id, user_id, email, ip, user_agent, True, "Success")
            self.logger.info(f"Successful login: {email} from {ip}")
            
            return {
                "success": True,
                "user_id": user_id,
                "email": email,
                "role": role,
                **tokens
            }
            
        finally:
            conn.close()
    
    def _generate_tokens(self, user_id: str, role: str) -> Dict:
        """Gera access e refresh tokens JWT (OAuth2.1)"""
        access_token_expires = datetime.utcnow() + timedelta(
            minutes=SecurityConfig.JWT_ACCESS_TOKEN_EXPIRE_MINUTES
        )
        refresh_token_expires = datetime.utcnow() + timedelta(
            days=SecurityConfig.JWT_REFRESH_TOKEN_EXPIRE_DAYS
        )
        
        # Access token
        access_token_payload = {
            "sub": user_id,
            "role": role,
            "type": "access",
            "exp": access_token_expires,
            "iat": datetime.utcnow(),
            "jti": secrets.token_urlsafe(16)
        }
        access_token = jwt.encode(
            access_token_payload,
            SecurityConfig.JWT_SECRET_KEY,
            algorithm=SecurityConfig.JWT_ALGORITHM
        )
        
        # Refresh token
        refresh_token_payload = {
            "sub": user_id,
            "type": "refresh",
            "exp": refresh_token_expires,
            "iat": datetime.utcnow(),
            "jti": secrets.token_urlsafe(16)
        }
        refresh_token = jwt.encode(
            refresh_token_payload,
            SecurityConfig.JWT_SECRET_KEY,
            algorithm=SecurityConfig.JWT_ALGORITHM
        )
        
        # Armazena tokens no banco
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        token_id = secrets.token_urlsafe(16)
        cursor.execute('''
            INSERT INTO tokens (id, user_id, access_token, refresh_token, expires_at)
            VALUES (?, ?, ?, ?, ?)
        ''', (
            token_id,
            user_id,
            self.fernet.encrypt(access_token.encode()).decode(),
            self.fernet.encrypt(refresh_token.encode()).decode(),
            refresh_token_expires.isoformat()
        ))
        
        conn.commit()
        conn.close()
        
        return {
            "access_token": access_token,
            "refresh_token": refresh_token,
            "token_type": "Bearer",
            "expires_in": SecurityConfig.JWT_ACCESS_TOKEN_EXPIRE_MINUTES * 60
        }
    
    def _log_login_attempt(self, audit_id: str, user_id: Optional[str], email: str, 
                          ip: str, user_agent: str, success: bool, reason: str):
        """Registra tentativa de login para auditoria"""
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        cursor.execute('''
            INSERT INTO login_audit (id, user_id, email, ip_address, user_agent, success, failure_reason)
            VALUES (?, ?, ?, ?, ?, ?, ?)
        ''', (audit_id, user_id, email, ip, user_agent, success, reason))
        
        conn.commit()
        conn.close()
    
    def validate_token(self, token: str) -> Dict:
        """Valida token JWT e retorna payload"""
        try:
            # Decodifica token
            payload = jwt.decode(
                token,
                SecurityConfig.JWT_SECRET_KEY,
                algorithms=[SecurityConfig.JWT_ALGORITHM]
            )
            
            # Verifica se token foi revogado
            conn = sqlite3.connect(self.db_path)
            cursor = conn.cursor()
            
            cursor.execute('''
                SELECT revoked FROM tokens 
                WHERE user_id = ? AND revoked = 0
                ORDER BY created_at DESC LIMIT 1
            ''', (payload["sub"],))
            
            result = cursor.fetchone()
            conn.close()
            
            if not result or result[0] == 1:
                return {"valid": False, "error": "Token revoked"}
            
            return {"valid": True, "payload": payload}
            
        except jwt.ExpiredSignatureError:
            return {"valid": False, "error": "Token expired"}
        except jwt.InvalidTokenError:
            return {"valid": False, "error": "Invalid token"}
    
    def refresh_token(self, refresh_token: str) -> Optional[Dict]:
        """Renova access token usando refresh token"""
        validation = self.validate_token(refresh_token)
        
        if not validation["valid"] or validation["payload"]["type"] != "refresh":
            return None
        
        user_id = validation["payload"]["sub"]
        
        # Obtém role do usuário
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        cursor.execute("SELECT role FROM users WHERE id = ?", (user_id,))
        result = cursor.fetchone()
        conn.close()
        
        if not result:
            return None
        
        role = result[0]
        
        # Gera novo access token
        access_token_expires = datetime.utcnow() + timedelta(
            minutes=SecurityConfig.JWT_ACCESS_TOKEN_EXPIRE_MINUTES
        )
        
        access_token_payload = {
            "sub": user_id,
            "role": role,
            "type": "access",
            "exp": access_token_expires,
            "iat": datetime.utcnow(),
            "jti": secrets.token_urlsafe(16)
        }
        
        new_access_token = jwt.encode(
            access_token_payload,
            SecurityConfig.JWT_SECRET_KEY,
            algorithm=SecurityConfig.JWT_ALGORITHM
        )
        
        return {
            "access_token": new_access_token,
            "token_type": "Bearer",
            "expires_in": SecurityConfig.JWT_ACCESS_TOKEN_EXPIRE_MINUTES * 60
        }

```python
import numpy as np
import pandas as pd
from sklearn.ensemble import IsolationForest
from sklearn.preprocessing import StandardScaler
from datetime import datetime, timedelta
import joblib
import json
import warnings
warnings.filterwarnings('ignore')

class CostAnomalyDetector:
    """Detector de anomalias de custo para LLMs em produção"""
    
    def __init__(self, config_path='config/anomaly_config.yaml'):
        self.models = {}
        self.scalers = {}
        self.thresholds = {
            'cost_increase_percentage': 20,
            'feature_cost_percentage': 40,
            'token_spike_multiplier': 3
        }
        self.load_config(config_path)
        
    def load_config(self, config_path):
        """Carrega configurações do detector"""
        import yaml
        try:
            with open(config_path, 'r') as f:
                config = yaml.safe_load(f)
                if 'cost_anomaly' in config:
                    self.thresholds.update(config['cost_anomaly'])
        except:
            # Configurações padrão
            pass
    
    def train_model(self, feature_name, historical_data):
        """
        Treina modelo de detecção para uma feature específica
        
        Args:
            feature_name: Nome da feature (ex: 'summarization', 'classification')
            historical_data: DataFrame com dados históricos
        """
        # Preparar features
        features = self._extract_cost_features(historical_data)
        
        # Normalizar dados
        scaler = StandardScaler()
        X_scaled = scaler.fit_transform(features)
        
        # Treinar Isolation Forest
        model = IsolationForest(
            n_estimators=100,
            contamination=0.1,
            random_state=42,
            n_jobs=-1
        )
        model.fit(X_scaled)
        
        # Salvar modelo e scaler
        self.models[feature_name] = model
        self.scalers[feature_name] = scaler
        
        # Calcular threshold adaptativo
        scores = model.decision_function(X_scaled)
        self.thresholds[f'{feature_name}_anomaly_score'] = np.percentile(scores, 10)
        
        return model
    
    def _extract_cost_features(self, data):
        """Extrai features relevantes para detecção de anomalias de custo"""
        features = []
        
        # Features temporais
        data['hour'] = data['timestamp'].dt.hour
        data['day_of_week'] = data['timestamp'].dt.dayofweek
        data['is_weekend'] = data['day_of_week'].isin([5, 6]).astype(int)
        
        # Features de custo
        features.append(data['cost_per_call'].values)
        features.append(data['total_tokens'].values)
        features.append(data['input_tokens'].values)
        features.append(data['output_tokens'].values)
        
        # Features derivadas
        features.append((data['cost_per_call'] / data['cost_per_call'].rolling(24).mean()).fillna(1).values)
        features.append((data['total_tokens'] / data['total_tokens'].rolling(24).mean()).fillna(1).values)
        
        # Features sazonais (hora do dia)
        hour_sin = np.sin(2 * np.pi * data['hour'] / 24)
        hour_cos = np.cos(2 * np.pi * data['hour'] / 24)
        features.append(hour_sin.values)
        features.append(hour_cos.values)
        
        return np.column_stack(features)
    
    def detect_anomalies(self, feature_name, current_data):
        """
        Detecta anomalias em dados atuais
        
        Returns:
            dict: Resultados da detecção com anomalias identificadas
        """
        if feature_name not in self.models:
            return {'status': 'error', 'message': f'Modelo não treinado para {feature_name}'}
        
        # Extrair features
        features = self._extract_cost_features(current_data)
        
        # Normalizar
        scaler = self.scalers[feature_name]
        X_scaled = scaler.transform(features)
        
        # Prever anomalias
        model = self.models[feature_name]
        predictions = model.predict(X_scaled)
        scores = model.decision_function(X_scaled)
        
        # -1 indica anomalia, 1 indica normal
        anomalies = predictions == -1
        
        # Aplicar regras de negócio adicionais
        business_anomalies = self._apply_business_rules(current_data)
        
        # Combinar detecções
        combined_anomalies = anomalies | business_anomalies
        
        # Preparar resultados
        results = {
            'feature': feature_name,
            'timestamp': datetime.now().isoformat(),
            'total_calls': len(current_data),
            'anomaly_count': int(combined_anomalies.sum()),
            'anomaly_percentage': float(combined_anomalies.mean() * 100),
            'anomaly_indices': np.where(combined_anomalies)[0].tolist(),
            'anomaly_scores': scores.tolist(),
            'details': []
        }
        
        # Adicionar detalhes das anomalias
        for idx in np.where(combined_anomalies)[0]:
            anomaly_detail = {
                'index': int(idx),
                'timestamp': current_data.iloc[idx]['timestamp'].isoformat(),
                'cost': float(current_data.iloc[idx]['cost_per_call']),
                'tokens': int(current_data.iloc[idx]['total_tokens']),
                'model': current_data.iloc[idx]['model'],
                'anomaly_score': float(scores[idx]),
                'reason': self._explain_anomaly(current_data.iloc[idx], scores[idx])
            }
            results['details'].append(anomaly_detail)
        
        return results
    
    def _apply_business_rules(self, data):
        """Aplica regras de negócio específicas para detecção de anomalias"""
        anomalies = pd.Series(False, index=data.index)
        
        # Regra 1: Aumento de custo > 20%
        if len(data) >= 2:
            current_cost = data['cost_per_call'].iloc[-1]
            avg_cost = data['cost_per_call'].iloc[:-1].mean()
            if current_cost > avg_cost * (1 + self.thresholds['cost_increase_percentage'] / 100):
                anomalies.iloc[-1] = True
        
        # Regra 2: Tokens 3x acima da média
        avg_tokens = data['total_tokens'].mean()
        token_anomalies = data['total_tokens'] > avg_tokens * self.thresholds['token_spike_multiplier']
        anomalies = anomalies | token_anomalies
        
        # Regra 3: Custo por feature > 40% do total
        # (Esta regra é aplicada em nível agregado, não por chamada)
        
        return anomalies.values
    
    def _explain_anomaly(self, data_point, anomaly_score):
        """Explica a razão da anomalia detectada"""
        reasons = []
        
        if anomaly_score < -0.5:
            reasons.append("Score de anomalia muito baixo")
        
        if data_point['total_tokens'] > data_point['total_tokens'].mean() * 3:
            reasons.append(f"Tokens ({data_point['total_tokens']}) 3x acima da média")
        
        if data_point['cost_per_call'] > data_point['cost_per_call'].mean() * 1.2:
            reasons.append(f"Custo ({data_point['cost_per_call']:.4f}) 20% acima da média")
        
        if len(reasons) == 0:
            reasons.append("Padrão incomum detectado pelo modelo")
        
        return "; ".join(reasons)
    
    def save_model(self, feature_name, path):
        """Salva modelo treinado"""
        if feature_name in self.models:
            model_data = {
                'model': self.models[feature_name],
                'scaler': self.scalers[feature_name],
                'threshold': self.thresholds.get(f'{feature_name}_anomaly_score', -0.5)
            }
            joblib.dump(model_data, f'{path}/{feature_name}_cost_model.pkl')
    
    def load_model(self, feature_name, path):
        """Carrega modelo salvo"""
        try:
            model_data = joblib.load(f'{path}/{feature_name}_cost_model.pkl')
            self.models[feature_name] = model_data['model']
```

from datetime import datetime, timedelta

class EnsembleAnomalyDetector:
    """Detector de anomalias que combina múltiplos métodos"""

        self.detectors = []
        self.weights = []
        self.confidence_threshold = 0.7
        
    def add_detector(self, detector, weight=1.0):
        """Adiciona um detector ao ensemble"""
        self.detectors.append(detector)
        self.weights.append(weight)
        
    def detect(self, data: pd.DataFrame, feature_name: str) -> Dict[str, Any]:
        """
        Executa detecção usando todos os detectores e combina resultados
        
        Args:
            data: DataFrame com dados para análise
            feature_name: Nome da feature sendo monitorada
            
        Returns:
            Dict com resultados combinados
        """
        if not self.detectors:
            return {'error': 'Nenhum detector configurado'}
        
        # Executar todos os detectores
        results = []
        for detector in self.detectors:
            try:
                result = detector.detect(data, feature_name)
                results.append(result)
            except Exception as e:
                print(f"Erro no detector {detector.__class__.__name__}: {e}")
                continue
        
        # Combinar resultados
        combined = self._combine_results(results, data)
        
        # Calcular confiança
        confidence = self._calculate_confidence(combined, results)
        combined['confidence'] = confidence
        
        # Aplicar threshold de confiança
        if confidence < self.confidence_threshold:
            combined['status'] = 'low_confidence'
        
        return combined
    
    def _combine_results(self, results: List[Dict], data: pd.DataFrame) -> Dict[str, Any]:
        """Combina resultados de múltiplos detectores"""
        # Inicializar estrutura de resultados
        combined = {
            'feature': results[0]['feature'] if results else 'unknown',
            'timestamp': datetime.now().isoformat(),
            'total_samples': len(data),
            'anomaly_indices': set(),
            'detector_agreements': {},
            'anomaly_details': []
        }
        
        # Coletar anomalias de todos os detectores
        for i, result in enumerate(results):
            if 'anomaly_indices' in result:
                detector_name = self.detectors[i].__class__.__name__
                anomalies = set(result['anomaly_indices'])
                combined['anomaly_indices'].update(anomalies)
                
                # Registrar acordo entre detectores
                for idx in anomalies:
                    if idx not in combined['detector_agreements']:
                        combined['detector_agreements'][idx] = []
                    combined['detector_agreements'][idx].append(detector_name)
        
        # Converter para lista
        combined['anomaly_indices'] = list(combined['anomaly_indices'])
        combined['anomaly_count'] = len(combined['anomaly_indices'])
        
        # Coletar detalhes das anomalias
        for idx in combined['anomaly_indices']:
            if idx < len(data):
                detail = {
                    'index': idx,
                    'timestamp': data.iloc[idx]['timestamp'].isoformat() if 'timestamp' in data.columns else 'unknown',
                    'detectors': combined['detector_agreements'][idx],
                    'agreement_count': len(combined['detector_agreements'][idx]),
                    'data_point': data.iloc[idx].to_dict()
                }
                combined['anomaly_details'].append(detail)
        
        return combined
    
    def _calculate_confidence(self, combined: Dict, results: List[Dict]) -> float:
        """Calcula confiança baseada na concordância entre detectores"""
        if not results or combined['total_samples'] == 0:
            return 0.0
        
        total_detectors = len(results)
        if total_detectors == 0:
            return 0.0
        
        # Calcular concordância por anomalia
        agreement_scores = []
        for idx in combined['anomaly_indices']:
            if idx in combined['detector_agreements']:
                agreement = len(combined['detector_agreements'][idx]) / total_detectors
                agreement_scores.append(agreement)
        
        if not agreement_scores:
            return 0.0
        
        # Confiança média ponderada pelo número de detectores em acordo
        avg_agreement = np.mean(agreement_scores)
        anomaly_coverage = combined['anomaly_count'] / combined['total_samples']
        
        # Fórmula de confiança
        confidence = avg_agreement * (1 - anomaly_coverage)  # Penaliza muitas anomalias
        
        return min(max(confidence, 0.0), 1.0)

# DIYAPP V12 - Guia de Deploy Manual

## Pré-requisitos

### Sistema
- Ubuntu 20.04 LTS ou superior
- 4GB RAM mínimo (8GB recomendado)
- 50GB espaço em disco
- Acesso root/sudo

### Dependências
```bash
# Atualizar sistema
sudo apt update && sudo apt upgrade -y

# Instalar dependências básicas
sudo apt install -y curl wget git build-essential

# Instalar Node.js 18.x
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install -y nodejs

# Instalar PostgreSQL 14
sudo apt install -y postgresql postgresql-contrib

# Instalar Redis
sudo apt install -y redis-server

# Instalar Nginx
sudo apt install -y nginx

# Instalar PM2 para gerenciamento de processos
sudo npm install -g pm2

# Verificar instalações
node --version  # Deve mostrar v18.x.x
npm --version
psql --version
redis-cli --version
nginx -v
```

## Configuração do Banco de Dados

```bash
# Acessar PostgreSQL
sudo -u postgres psql

-- Criar banco de dados
CREATE DATABASE diyapp_v12;

-- Criar usuário
CREATE USER diyapp_user WITH ENCRYPTED PASSWORD 'seu_password_seguro_aqui';

-- Conceder privilégios
GRANT ALL PRIVILEGES ON DATABASE diyapp_v12 TO diyapp_user;

-- Sair
\q

sudo nano /etc/redis/redis.conf

# Alterar (se necessário):

```
