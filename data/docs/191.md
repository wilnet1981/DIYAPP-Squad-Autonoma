# DIYAPP Evolution - V12 Core - Arquitetura Técnica

## ADR-001: Arquitetura Base V12

**Data:** 2024-03-15
**Status:** Aceita
**Autores:** Tech Lead

### CONTEXTO:
Precisamos estabelecer uma arquitetura robusta para o DIYAPP V12 que suporte:
1. 100% de autonomia operacional
2. Refatoração paralela via Modo Hive
3. Estabilidade de produção com zero downtime
4. Evolução contínua sem acoplamento excessivo

### DECISÃO:
Adotar arquitetura hexagonal (ports & adapters) com:
- **Backend:** Node.js + Express + TypeScript
- **Frontend:** React + Vite + TypeScript (SSR opcional)
- **Banco de Dados:** PostgreSQL (principal) + Redis (cache/sessões)
- **Mensageria:** RabbitMQ para comunicação entre módulos
- **Monitoramento:** OpenTelemetry + Prometheus + Grafana
- **CI/CD:** GitHub Actions com testes automatizados

### OPÇÕES CONSIDERADAS:
- **Opção A (Monolito modular):** Express + módulos internos
  - Prós: Simplicidade inicial, deploy único
  - Contras: Acoplamento, dificuldade em refatoração paralela
- **Opção B (Microserviços completos):** Cada feature como serviço independente
  - Prós: Isolamento máximo, escalabilidade
  - Contras: Complexidade operacional, overhead de comunicação
- **Opção C (Arquitetura hexagonal):** Core isolado, adaptadores plugáveis
  - Prós: Flexibilidade para Modo Hive, testabilidade, evolução gradual
  - Contras: Curva de aprendizado, boilerplate inicial

**Opção escolhida:** C - Justificativa: Balanceia isolamento para refatoração paralela com simplicidade operacional.

### CONSEQUÊNCIAS:
**Positivas:**
- Módulos podem ser refatorados independentemente
- Testes isolados por camada
- Facilidade para substituir tecnologias específicas
- Suporte a múltiplos frontends (web, mobile, CLI)

**Negativas:**
- Mais arquivos e camadas de abstração
- Necessidade de disciplina nos contratos entre camadas

**Riscos:**
- Desenvolvimento pode ficar mais lento inicialmente
- Mitigação: Templates e generators para boilerplate

## Padrões de Código TypeScript

### Nomenclatura
- Interfaces: `IUserRepository` (prefixo I)
- Classes: `UserService` (PascalCase)
- Funções: `createUser` (camelCase)
- Constantes: `MAX_RETRIES` (UPPER_SNAKE_CASE)
- Arquivos: `user.service.ts` (kebab-case)

### Imports
```typescript
// 1. Bibliotecas externas
import express from 'express';

import { UserService } from '@core/application';

// 3. Tipos

```typescript
// ERRADO
throw new Error('User not found');

```

## Testes
- Cobertura mínima: 80%
- Unitários: Jest + Vitest
- E2E: Playwright
- Testes devem ser independentes e não depender de estado externo

## Commits
- Conventional Commits: `feat:`, `fix:`, `chore:`, `docs:`, `test:`, `refactor:`
- Exemplo: `feat(auth): add JWT authentication`

## Code Review
- PRs > 400 linhas devem ser divididos
- Todo PR precisa de:
  1. Descrição clara do problema/solução
  2. Testes adicionados/modificados
  3. Atualização de documentação se necessário
  4. Aprovação do Tech Lead

## Segurança
- Nenhum segredo em código
- Validação de input em todas as camadas
- Rate limiting por padrão
- Headers de segurança (CSP, HSTS)

```typescript
import { v4 as uuidv4 } from 'uuid';
import { AppError, ErrorCodes } from '@shared/errors';

export interface IUserProps {
  id?: string;
  email: string;
  name: string;
  role: UserRole;
  createdAt?: Date;
  updatedAt?: Date;
}

export enum UserRole {
  ADMIN = 'admin',
  USER = 'user',
  GUEST = 'guest'
}

export class User {
  public readonly id: string;
  public readonly email: string;
  public readonly name: string;
  public readonly role: UserRole;
  public readonly createdAt: Date;
  public readonly updatedAt: Date;

  private constructor(props: IUserProps) {
    this.validateEmail(props.email);
    this.validateName(props.name);

    this.id = props.id || uuidv4();
    this.email = props.email.toLowerCase().trim();
    this.name = props.name.trim();
    this.role = props.role || UserRole.USER;
    this.createdAt = props.createdAt || new Date();
    this.updatedAt = props.updatedAt || new Date();
  }

  public static create(props: IUserProps): User {
    return new User(props);
  }

  public update(props: Partial<IUserProps>): User {
    return new User({
      ...this.toJSON(),
      ...props,
      id: this.id, // ID não pode ser alterado
      updatedAt: new Date()
    });
  }

  public toJSON(): IUserProps {
    return {
      id: this.id,
      email: this.email,
      name: this.name,
      role: this.role,
      createdAt: this.createdAt,
      updatedAt: this.updatedAt
    };
  }

  private validateEmail(email: string): void {
    const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    if (!emailRegex.test(email)) {
      throw new AppError('Invalid email format', ErrorCodes.VALIDATION_ERROR);
    }
  }

  private validateName(name: string): void {
    if (name.length < 2) {
      throw new AppError('Name must be at least 2 characters', ErrorCodes.VALIDATION_ERROR);
    }
    if (name.length > 100) {
      throw new AppError('Name must be less than 100 characters', ErrorCodes.VALIDATION_ERROR);
    }
  }
}

## 3. Circuit Breaker Pattern

```javascript
// src/utils/circuitBreaker.js
const { EventEmitter } = require('events');
const { logger } = require('../app');
const { ns } = require('../app');

class CircuitBreaker extends EventEmitter {
  constructor(options = {}) {
    super();
    
    this.failureThreshold = options.failureThreshold || 5;
    this.resetTimeout = options.resetTimeout || 30000; // 30 segundos
    this.timeout = options.timeout || 10000; // 10 segundos
    
    this.state = 'CLOSED';
    this.failureCount = 0;
    this.nextAttempt = Date.now();
    this.name = options.name || 'unnamed-service';
    
    this.metrics = {
      totalCalls: 0,
      successfulCalls: 0,
      failedCalls: 0,
      circuitOpened: 0,
      lastFailure: null
    };
  }

  async call(serviceFunction, ...args) {
    const correlationId = ns.get('correlationId');
    this.metrics.totalCalls++;
    
    // Circuito aberto - retorna fallback imediatamente
    if (this.state === 'OPEN') {
      if (Date.now() > this.nextAttempt) {
        this.state = 'HALF_OPEN';
        logger.info({
          correlation_id: correlationId,
          message: `Circuit ${this.name} transitioning to HALF_OPEN`,
          circuit: this.name
        });
      } else {
        this.metrics.circuitOpened++;
        this.emit('circuitOpened', this.name);
        
        logger.warn({
          correlation_id: correlationId,
          message: `Circuit ${this.name} is OPEN - using fallback`,
          circuit: this.name,
          next_attempt: new Date(this.nextAttempt).toISOString()
        });
        
        throw new Error(`Service ${this.name} unavailable (circuit open)`);
      }
    }
    
    // Timeout wrapper
    const timeoutPromise = new Promise((_, reject) => {
      setTimeout(() => {
        reject(new Error(`Service ${this.name} timeout after ${this.timeout}ms`));
      }, this.timeout);
    });
    
    try {
      // Race entre serviceFunction e timeout
      const result = await Promise.race([
        serviceFunction(...args),
        timeoutPromise
      ]);
      
      // Sucesso - reset contador de falhas
      this.onSuccess(correlationId);
      this.metrics.successfulCalls++;
      
      return result;
    } catch (error) {
      // Falha - incrementa contador
      this.onFailure(error, correlationId);
      this.metrics.failedCalls++;
      this.metrics.lastFailure = new Date().toISOString();
      
      throw error;
    }
  }

  onSuccess(correlationId) {
    if (this.state === 'HALF_OPEN') {
      this.state = 'CLOSED';
      this.failureCount = 0;
      
      logger.info({
        correlation_id: correlationId,
        message: `Circuit ${this.name} reset to CLOSED`,
        circuit: this.name
      });
    }
  }

  onFailure(error, correlationId) {
    this.failureCount++;
    
    logger.error({
      correlation_id: correlationId,
      message: `Circuit ${this.name} failure (${this.failureCount}/${this.failureThreshold})`,
      circuit: this.name,
      error: error.message
    });
    
    if (this.failureCount >= this.failureThreshold) {
      this.state = 'OPEN';
      this.nextAttempt = Date.now() + this.resetTimeout;
      
      logger.error({
        correlation_id: correlationId,
        message: `Circuit ${this.name} opened`,
        circuit: this.name,
        reset_in: `${this.resetTimeout}ms`,
        next_attempt: new Date(this.nextAttempt).toISOString()
      });
      
      this.emit('circuitOpened', this.name);
    }
  }

  getMetrics() {
    return {
      ...this.metrics,
      state: this.state,
      failureCount: this.failureCount,
      failureThreshold: this.failureThreshold,
      nextAttempt: this.nextAttempt
    };
  }
}

// Factory para circuit breakers
class CircuitBreakerFactory {
  constructor() {
    this.breakers = new Map();
  }

  getBreaker(name, options = {}) {
    if (!this.breakers.has(name)) {
      this.breakers.set(name, new CircuitBreaker({ ...options, name }));
    }
    return this.breakers.get(name);
  }

  getMetrics() {
    const metrics = {};
    for (const [name, breaker] of this.breakers) {
      metrics[name] = breaker.getMetrics();
    }
    return metrics;
  }
}

module.exports = {
  CircuitBreaker,
  circuitBreakerFactory: new CircuitBreakerFactory()
};

// src/services/BaseService.js
const { logger } = require('../app');
const { ns } = require('../app');
const { circuitBreakerFactory } = require('../utils/circuitBreaker');

class BaseService {
  constructor() {
    this.logger = logger;
    this.circuitBreakerFactory = circuitBreakerFactory;
  }

  // Retry com backoff exponencial + jitter
  async withRetry(operation, options = {}) {
    const maxRetries = options.maxRetries || 3;
    const baseDelay = options.baseDelay || 100;
    const correlationId = ns.get('correlationId');
    
    let lastError;
    
    for (let attempt = 0; attempt <= maxRetries; attempt++) {
      try {
        return await operation();
      } catch (error) {
        lastError = error;
        
        // Só retry em erros transitórios (timeout, 503, 429, network)
        const isTransient = this.isTransientError(error);
        
        if (!isTransient || attempt === maxRetries) {
          break;
        }
        
        // Backoff exponencial + jitter
        const delay = baseDelay * Math.pow(2, attempt);
        const jitter = delay * 0.1 * Math.random();
        const waitTime = delay + jitter;
        
        this.logger.warn({
          correlation_id: correlationId,
          message: `Retry attempt ${attempt + 1}/${maxRetries + 1}`,
          error: error.message,
          wait_time_ms: waitTime,
          service: this.constructor.name
        });
        
        await new Promise(resolve => setTimeout(resolve, waitTime));
      }
    }
    
    throw lastError;
  }

  isTransientError(error) {
    const transientStatusCodes = [408, 429, 500, 502, 503, 504];
    const transientMessages = ['timeout', 'network', 'connection', 'econnreset', 'econnrefused'];
    
    if (error.statusCode && transientStatusCodes.includes(error.statusCode)) {
      return true;
    }
    
    if (error.code && transientMessages.some(msg => error.code.toLowerCase().includes(msg))) {
      return true;
    }
    
    if (error.message && transientMessages.some(msg => error.message.toLowerCase().includes(msg))) {
      return true;
    }
    
    return false;
  }

  // Métricas de performance
  async withMetrics(operation, metricName) {
    const start = Date.now();
    const correlationId = ns.get('correlationId');
    
    try {
      const result = await operation();
      const duration = Date.now() - start;
      
      this.logger.info({
        correlation_id: correlationId,
        metric: metricName,
        duration_ms: duration,
        status: 'success'
      });
      
      return result;
    } catch (error) {
      const duration = Date.now() - start;
      
      this.logger.error({
        correlation_id: correlationId,
        metric: metricName,
        duration_ms: duration,
        status: 'error',
        error: error.message
      });
      
      throw error;
    }
  }
}

module.exports = BaseService;

interface DesignSystemContextType {
  tokens: typeof designTokens;
  mode: 'light' | 'dark';
  toggleMode: () => void;

const DesignSystemContext = createContext<DesignSystemContextType | undefined>(undefined);

export const useDesignSystem = () => {
  const context = useContext(DesignSystemContext);
  if (!context) {
    throw new Error('useDesignSystem must be used within DesignSystemProvider');
  }
  return context;
};

interface DesignSystemProviderProps {
  children: ReactNode;
  initialMode?: 'light' | 'dark';
}

export const DesignSystemProvider: React.FC<DesignSystemProviderProps> = ({ 
  children, 
  initialMode = 'light' 
}) => {
  const [mode, setMode] = React.useState<'light' | 'dark'>(initialMode);

  const toggleMode = () => {
    setMode(prev => prev === 'light' ? 'dark' : 'light');
  };

  const value = {
    tokens: designTokens,
    mode,
    toggleMode,
  };

  return (
    <DesignSystemContext.Provider value={value}>
      {children}
    </DesignSystemContext.Provider>
  );
};

CREATE OR REPLACE FUNCTION calculate_session_duration()
RETURNS TRIGGER AS $$
BEGIN
    IF NEW.ended_at IS NOT NULL AND NEW.started_at IS NOT NULL THEN
        NEW.duration_seconds = EXTRACT(EPOCH FROM (NEW.ended_at - NEW.started_at));
    END IF;
    RETURN NEW;
END;
$$ language 'plpgsql';

CREATE TRIGGER calculate_session_duration_trigger BEFORE INSERT OR UPDATE ON user_sessions
    FOR EACH ROW EXECUTE FUNCTION calculate_session_duration();

const jwt = require('jsonwebtoken');
const crypto = require('crypto');
const securityConfig = require('../../config/security-config');

class JWTStrategy {
    constructor() {
        this.config = securityConfig.getConfig();
        this.blacklistedTokens = new Set();
    }

    generateTokens(user) {
        const accessToken = jwt.sign(
            {
                userId: user.id,
                email: user.email,
                role: user.role,
                permissions: user.permissions
            },
            this.config.JWT_SECRET,
            { 
                expiresIn: this.config.JWT_ACCESS_EXPIRY,
                issuer: 'diyapp-v12',
                audience: 'diyapp-client'
            }
        );

        const refreshToken = crypto.randomBytes(64).toString('hex');
        
        // Store refresh token hash in database (simulated)
        const refreshTokenHash = crypto
            .createHash('sha256')
            .update(refreshToken)
            .digest('hex');

        return {
            accessToken,
            refreshToken,
            refreshTokenHash,
            expiresIn: 15 * 60 // 15 minutes in seconds
        };
    }

    verifyToken(token) {
        try {
            // Check blacklist
            if (this.blacklistedTokens.has(token)) {
                throw new Error('Token revoked');
            }

            const decoded = jwt.verify(token, this.config.JWT_SECRET, {
                issuer: 'diyapp-v12',
                audience: 'diyapp-client'
            });

            return {
                valid: true,
                decoded,
                expired: false
            };
        } catch (error) {
            return {
                valid: false,
                error: error.message,
                expired: error.name === 'TokenExpiredError'
            };
        }
    }

    revokeToken(token) {
        this.blacklistedTokens.add(token);
        
        // Schedule cleanup (24 hours)
        setTimeout(() => {
            this.blacklistedTokens.delete(token);
        }, 24 * 60 * 60 * 1000);
    }

    rotateRefreshToken(oldRefreshTokenHash, user) {
        // Verify old refresh token exists and is valid
        // In production, check against database
        
        // Generate new tokens
        return this.generateTokens(user);
    }

    decodeWithoutVerification(token) {
        try {
            return jwt.decode(token);
        } catch (error) {
            return null;
        }
    }
}

class OAuth2Provider {
    constructor() {
        this.config = securityConfig.getConfig();
        this.stateStore = new Map();
    }

    generateState() {
        const state = crypto.randomBytes(16).toString('hex');
        const timestamp = Date.now();
        this.stateStore.set(state, timestamp);
        
        // Cleanup old states (5 minutes)
        setTimeout(() => {
            this.stateStore.delete(state);
        }, 5 * 60 * 1000);
        
        return state;
    }

    validateState(state) {
        const timestamp = this.stateStore.get(state);
        if (!timestamp) return false;
        
        // State valid for 5 minutes
        if (Date.now() - timestamp > 5 * 60 * 1000) {
            this.stateStore.delete(state);
            return false;
        }
        
        this.stateStore.delete(state);
        return true;
    }

    async handleGoogleCallback(code, state) {
        if (!this.validateState(state)) {
            throw new Error('Invalid state parameter');
        }

        try {
            // Exchange code for tokens
            const tokenResponse = await axios.post(
                'https://oauth2.googleapis.com/token',
                {
                    code,
                    client_id: this.config.OAUTH_PROVIDERS.google.clientId,
                    client_secret: this.config.OAUTH_PROVIDERS.google.clientSecret,
                    redirect_uri: this.config.OAUTH_PROVIDERS.google.redirectUri,
                    grant_type: 'authorization_code'
                }
            );

            // Get user info
            const userInfo = await axios.get(
                'https://www.googleapis.com/oauth2/v3/userinfo',
                {
                    headers: {
                        Authorization: `Bearer ${tokenResponse.data.access_token}`
                    }
                }
            );

            return {
                success: true,
                user: {
                    email: userInfo.data.email,
                    name: userInfo.data.name,
                    picture: userInfo.data.picture,
                    provider: 'google',
                    providerId: userInfo.data.sub
                },
                tokens: tokenResponse.data
            };
        } catch (error) {
            throw new Error(`OAuth2 error: ${error.message}`);
        }
    }

    async handleGitHubCallback(code, state) {
        if (!this.validateState(state)) {
            throw new Error('Invalid state parameter');
        }

        try {
            // Exchange code for tokens
            const tokenResponse = await axios.post(
                'https://github.com/login/oauth/access_token',
                {
                    code,
                    client_id: this.config.OAUTH_PROVIDERS.github.clientId,
                    client_secret: this.config.OAUTH_PROVIDERS.github.clientSecret,
                    redirect_uri: this.config.OAUTH_PROVIDERS.github.redirectUri
                },
                {
                    headers: {
                        Accept: 'application/json'
                    }
                }
            );

            // Get user info
            const userInfo = await axios.get(
                'https://api.github.com/user',
                {
                    headers: {
                        Authorization: `Bearer ${tokenResponse.data.access_token}`,
                        'User-Agent': 'DIYAPP-V12'
                    }
                }
            );

            // Get user emails
            const emails = await axios.get(
                'https://api.github.com/user/emails',
                {
                    headers: {
                        Authorization: `Bearer ${tokenResponse.data.access_token}`,
                        'User-Agent': 'DIYAPP-V12'
                    }
                }
            );

            const primaryEmail = emails.data.find(email => email.primary);

            return {
                success: true,
                user: {
                    email: primaryEmail.email,
                    name: userInfo.data.name || userInfo.data.login,
                    picture: userInfo.data.avatar_url,
                    provider: 'github',
                    providerId: userInfo.data.id.toString()
                },
                tokens: tokenResponse.data
            };
        } catch (error) {
            throw new Error(`OAuth2 error: ${error.message}`);
        }
    }

    getAuthorizationUrl(provider) {
        const state = this.generateState();
        
        switch (provider) {
            case 'google':
                return `https://accounts.google.com/o/oauth2/v2/auth?` +

                    `state=${state}&` +
                    `access_type=offline&` +
                    `prompt=consent`;
                    
            case 'github':
                return `https://github.com/login/oauth/authorize?` +
                    `client_id=${this.config.OAUTH_PROVIDERS.github.clientId}&` +
                    `redirect_uri=${encodeURIComponent(this.config.OAUTH_PROVIDERS.github.redirectUri)}&` +
                    `scope=user%20user:email&` +
                    `state=${state}`;
                    
            default:
                throw new Error(`Unsupported provider: ${provider}`);
        }
    }
}

const crypto = require('crypto');
const securityConfig = require('../../config/security-config');

class DataEncryption {
    constructor() {
        this.config = securityConfig.getConfig();
        this.algorithm = this.config.ENCRYPTION_ALGORITHM;
        this.key = Buffer.from(this.config.ENCRYPTION_KEY, 'hex');
        
        if (this.key.length !== 32) {
            throw new Error('Encryption key must be 32 bytes (256 bits)');
        }
    }

    encrypt(text) {
        try {
            const iv = crypto.randomBytes(16);
            const cipher = crypto.createCipheriv(this.algorithm, this.key, iv);
            
            let encrypted = cipher.update(text, 'utf8', 'hex');
            encrypted += cipher.final('hex');
            
            const authTag = cipher.getAuthTag();
            
            return {
                encrypted,
                iv: iv.toString('hex'),
                authTag: authTag.toString('hex'),
                algorithm: this.algorithm
            };
        } catch (error) {
            throw new Error(`Encryption failed: ${error.message}`);
        }
    }

    decrypt(encryptedData) {
        try {
            const decipher = crypto.createDecipheriv(
                this.algorithm,
                this.key,
                Buffer.from(encryptedData.iv, 'hex')
            );
            
            decipher.setAuthTag(Buffer.from(encryptedData.authTag, 'hex'));
            
            let decrypted = decipher.update(encryptedData.encrypted, 'hex', 'utf8');
            decrypted += decipher.final('utf8');
            
            return decrypted;
        } catch (error) {
            throw new Error(`Decryption failed: ${error.message}`);
        }
    }

    hashSensitiveData(data, salt = null) {
        const saltToUse = salt || crypto.randomBytes(16).toString('hex');
        const hash = crypto
            .pbkdf2Sync(data, saltToUse, 10000, 64, 'sha512')
            .toString('hex');
            
        return {
            hash,
            salt: saltToUse
        };
    }

    verifyHash(data, hash, salt) {
        const newHash = this.hashSensitiveData(data, salt).hash;
        return crypto.timingSafeEqual(
            Buffer.from(newHash, 'hex'),
            Buffer.from(hash, 'hex')
        );
    }

    maskPII(data, visibleChars = 4) {
        if (!data || data.length <= visibleChars) {
            return '*'.repeat(data?.length || 0);
        }
        
        const lastVisible = data.slice(-visibleChars);
        return '*'.repeat(data.length - visibleChars) + lastVisible;
    }

    generateSecurePassword(length = 16) {
        const charset = 'abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789!@#$%^&*()_+-=[]{}|;:,.<>?';
        const values = crypto.randomBytes(length);
        let password = '';
        
        for (let i = 0; i < length; i++) {
            password += charset[values[i] % charset.length];
        }
        
        return password;
    }
}

class LLMFeaturePage {
  constructor(page) {
    this.page = page;
    this.promptInput = page.locator('[data-testid="prompt-input"]');
    this.modelSelect = page.locator('[data-testid="model-select"]');
    this.generateButton = page.locator('[data-testid="generate-button"]');
    this.resultContainer = page.locator('[data-testid="result-container"]');
    this.resultText = page.locator('[data-testid="result-text"]');
    this.copyButton = page.locator('[data-testid="copy-button"]');
    this.errorMessage = page.locator('[data-testid="llm-error"]');
    this.consistencyMetric = page.locator('[data-testid="consistency-metric"]');
  }

  async fillPrompt(prompt) {
    await this.promptInput.fill(prompt);
  }

  async selectModel(modelName) {
    await this.modelSelect.selectOption(modelName);
  }

  async clickGenerate() {
    await this.generateButton.click();
    await this.page.waitForTimeout(2000); // Aguarda geração
  }

  async getResultText() {
    return await this.resultText.textContent();
  }

  async getWordCount() {
    const text = await this.getResultText();
    return text.split(/\s+/).filter(word => word.length > 0).length;
  }

  async calculateSemanticSimilarity(results) {
    // Implementação simplificada - na prática usar biblioteca como natural
    if (results.length < 2) return 1;
    
    // Calcula similaridade baseada em palavras comuns
    const wordSets = results.map(r => new Set(r.toLowerCase().split(/\W+/)));
    let totalSimilarity = 0;
    let comparisons = 0;
    
    for (let i = 0; i < wordSets.length; i++) {
      for (let j = i + 1; j < wordSets.length; j++) {
        const intersection = new Set([...wordSets[i]].filter(x => wordSets[j].has(x)));
        const union = new Set([...wordSets[i], ...wordSets[j]]);
        const similarity = intersection.size / union.size;
        totalSimilarity += similarity;
        comparisons++;
      }
    }
    
    return comparisons > 0 ? totalSimilarity / comparisons : 0;
  }

  async getConsistencyMetric() {
    return await this.consistencyMetric.textContent();
  }
}

module.exports = { LLMFeaturePage };

## 3. Serviço de Privacidade

**src/compliance/services/PrivacyService.js**
```javascript
const crypto = require('crypto');
const { DataSubject, ConsentRecord, ProcessingActivity } = require('../models');
const complianceConfig = require('../../config/complianceConfig');
const logger = require('../../utils/logger');

class PrivacyService {
  constructor() {
    this.logger = logger.child({ module: 'PrivacyService' });
  }

  /**
   * Cria um registro de titular de dados
   */
  async createDataSubject(userData) {
    try {
      const identificationHash = this.generateIdentificationHash(userData.email);
      
      const dataSubject = await DataSubject.create({
        userId: userData.id,
        email: userData.email,
        identificationHash,
        dataCategories: this.extractDataCategories(userData),
        consentVersion: complianceConfig.consent.defaultVersion || '1.0',
        metadata: {
          createdAt: new Date(),
          source: userData.source || 'registration'
        }
      });

      this.logger.info('Data subject created', { 
        dataSubjectId: dataSubject.id,
        userId: userData.id 
      });

      return dataSubject;
    } catch (error) {
      this.logger.error('Error creating data subject', { error: error.message });
      throw error;
    }
  }

  /**
   * Processa solicitação de direitos do titular
   */
  async processDataSubjectRequest(request) {
    const { dataSubjectId, requestType, details } = request;
    
    try {
      const dataSubject = await DataSubject.findByPk(dataSubjectId);
      if (!dataSubject) {
        throw new Error('Data subject not found');
      }

      let result;
      switch (requestType) {
        case 'access':
          result = await this.processAccessRequest(dataSubject);
          break;
        case 'correction':
          result = await this.processCorrectionRequest(dataSubject, details);
          break;
        case 'deletion':
          result = await this.processDeletionRequest(dataSubject);
          break;
        case 'portability':
          result = await this.processPortabilityRequest(dataSubject);
          break;
        case 'objection':
          result = await this.processObjectionRequest(dataSubject, details);
          break;
        default:
          throw new Error('Invalid request type');
      }

      // Registra o exercício do direito
      await this.recordRightExercise(dataSubject, requestType, result);

      this.logger.info('Data subject request processed', {
        dataSubjectId,
        requestType,
        result: result.status
      });

      return result;
    } catch (error) {
      this.logger.error('Error processing data subject request', {
        dataSubjectId,
        requestType,
        error: error.message
      });
      throw error;
    }
  }

  /**
   * Processa solicitação de acesso
   */
  async processAccessRequest(dataSubject) {
    // Coleta todos os dados do titular
    const userData = await this.collectUserData(dataSubject.userId);
    const consentHistory = await this.getConsentHistory(dataSubject.id);
    const processingActivities = await this.getProcessingActivities(dataSubject.id);

    // Gera token para portabilidade
    const portabilityToken = crypto.randomBytes(32).toString('hex');
    await dataSubject.update({
      lastAccessRequest: new Date(),
      dataPortabilityToken: portabilityToken,
      rightsExercised: [...dataSubject.rightsExercised, {
        type: 'access',
        date: new Date(),
        token: portabilityToken
      }]
    });

    return {
      status: 'completed',
      data: {
        personalData: userData,
        consentHistory,
        processingActivities,
        portabilityToken,
        generatedAt: new Date().toISOString()
      },
      format: 'json', // Também suporta CSV, XML
      expiresAt: new Date(Date.now() + 30 * 24 * 60 * 60 * 1000) // 30 dias
    };
  }

  /**
   * Processa solicitação de exclusão
   */
  async processDeletionRequest(dataSubject) {
    // Verifica se há impedimentos legais
    const legalImpediments = await this.checkLegalImpediments(dataSubject);
    
    if (legalImpediments.length > 0) {
      return {
        status: 'partial',
        message: 'Deletion cannot be completed due to legal obligations',
        impediments: legalImpediments,
        scheduledFor: new Date(Date.now() + 30 * 24 * 60 * 60 * 1000) // Reavalia em 30 dias
      };
    }

    // Agenda exclusão
    const deletionDate = new Date(Date.now() + 24 * 60 * 60 * 1000); // 24 horas
    await dataSubject.update({
      status: 'deletion_pending',
      deletionScheduledAt: deletionDate
    });

    // Inicia processo de exclusão em background
    this.scheduleDataDeletion(dataSubject, deletionDate);

    return {
      status: 'scheduled',
      scheduledDeletion: deletionDate,
      confirmationId: crypto.randomBytes(16).toString('hex')
    };
  }

  /**
   * Agenda exclusão de dados
   */
  async scheduleDataDeletion(dataSubject, deletionDate) {
    // Implementação de exclusão em background
    setTimeout(async () => {
      try {
        await this.executeDataDeletion(dataSubject);
        this.logger.info('Data deletion completed', { dataSubjectId: dataSubject.id });
      } catch (error) {
        this.logger.error('Error executing data deletion', {
          dataSubjectId: dataSubject.id,
          error: error.message
        });
      }
    }, deletionDate - Date.now());
  }

  /**
   * Gera hash de identificação anônima
   */
  generateIdentificationHash(email) {
    const salt = process.env.HASH_SALT || 'diyapp-compliance-salt';
    return crypto
      .createHash('sha256')
      .update(email + salt)
      .digest('hex');
  }

  /**
   * Extrai categorias de dados do usuário
   */
  extractDataCategories(userData) {
    const categories = new Set(['identification', 'contact']);
    
    if (userData.birthDate) categories.add('demographic');
    if (userData.location) categories.add('location');
    if (userData.paymentInfo) categories.add('financial');
    if (userData.healthData) categories.add('health');
    if (userData.biometricData) categories.add('biometric');
    
    return Array.from(categories);
  }

  /**
   * Verifica impedimentos legais para exclusão
   */
  async checkLegalImpediments(dataSubject) {
    const impediments = [];
    
    // Exemplo: verifica se há transações financeiras pendentes
    const pendingTransactions = await this.checkPendingTransactions(dataSubject.userId);
    if (pendingTransactions) {
      impediments.push({
        type: 'financial_obligation',
        description: 'Pending financial transactions',
        retentionPeriod: 'Until transaction completion + 5 years'
      });
    }

    // Verifica obrigações fiscais
    const taxObligations = await this.checkTaxObligations(dataSubject.userId);
    if (taxObligations) {
      impediments.push({
        type: 'legal_obligation',
        description: 'Tax retention requirements',
        retentionPeriod: '5 years'
      });
