## 2. Proposta Arquitetural V11 - Microserviços

### 2.1 ADR-001: Arquitetura de Microserviços

```markdown
# ADR-001: Migração para Arquitetura de Microserviços

Data: 2024-01-15
Status: Aceita
Autores: Tech Lead + Arquitetura Squad

## CONTEXTO:
A V10 apresenta pontos únicos de falha críticos e gargalos de performance
que impedem escalabilidade horizontal. O sistema monolítico atual tem:
1. Acoplamento forte entre módulos
2. Impossibilidade de deploy independente
3. Resource sharing problemático
4. Dificuldade de testes isolados

## DECISÃO:
Migrar para arquitetura baseada em microserviços com os seguintes princípios:
1. Cada serviço gerencia seu próprio banco de dados
2. Comunicação via eventos assíncronos (Apache Kafka)
3. API Gateway para roteamento
4. Service Discovery com Consul
5. Containerização com Docker

## OPÇÕES CONSIDERADAS:

### Opção A: Monolito Modularizado
- Prós: Menor complexidade, deploy único
- Contras: Não resolve pontos únicos de falha, escalabilidade limitada

### Opção B: Microsserviços Completo
- Prós: Alta disponibilidade, escalabilidade independente, deploy contínuo
- Contras: Complexidade operacional, overhead de comunicação

### Opção C: Arquitetura Híbrida (Strangler Pattern)
- Prós: Migração gradual, menor risco
- Contras: Manutenção de duas arquiteturas simultaneamente

**Opção escolhida: B** - Justificativa: Necessidade de 100% autonomia e estabilidade

## CONSEQUÊNCIAS:

### Positivas:
- Eliminação de pontos únicos de falha
- Escalabilidade horizontal independente
- Isolamento de falhas (circuit breaker)
- Deploy contínuo sem downtime

### Negativas:
- Aumento de complexidade operacional
- Overhead de comunicação entre serviços
- Necessidade de observabilidade distribuída

### Riscos:
1. Latência em chamadas entre serviços
   - Mitigação: Cache Redis + comunicação assíncrona
2. Consistência eventual
   - Mitigação: Sagas pattern para transações distribuídas
3. Debugging distribuído
   - Mitigação: Implementar distributed tracing (Jaeger)

## REVISÃO: 2024-04-15
```

# Logger estruturado
class StructuredLogger:
    def __init__(self, name: str):
        self.logger = logging.getLogger(name)
        self.logger.setLevel(logging.INFO)
        
        handler = logging.StreamHandler()
        formatter = logging.Formatter(
            '{"timestamp": "%(asctime)s", "level": "%(levelname)s", '
            '"correlation_id": "%(correlation_id)s", "service": "%(name)s", '
            '"message": "%(message)s", "data": %(data)s}'
        )
        handler.setFormatter(formatter)
        self.logger.addHandler(handler)
    
    def log(self, level: str, message: str, correlation_id: str, **kwargs):
        log_data = {k: self._mask_sensitive_data(k, v) for k, v in kwargs.items()}
        
        extra = {
            'correlation_id': correlation_id,
            'data': json.dumps(log_data)
        }
        
        getattr(self.logger, level.lower())(message, extra=extra)
    
    @staticmethod
    def _mask_sensitive_data(key: str, value: Any) -> Any:
        """Mascara dados sensíveis nos logs"""
        sensitive_keys = {'password', 'token', 'secret', 'key', 'credit_card', 'cpf', 'email'}
        
        if isinstance(value, str) and any(sensitive in key.lower() for sensitive in sensitive_keys):
            return f"***MASKED_{key.upper()}***"
        return value

class CircuitState(Enum):
    CLOSED = "CLOSED"
    OPEN = "OPEN"
    HALF_OPEN = "HALF_OPEN"

@dataclass
class CircuitBreakerMetrics:
    failures: int = 0
    successes: int = 0
    last_failure_time: Optional[float] = None
    state: CircuitState = CircuitState.CLOSED

class CircuitBreaker:
    """Implementação de Circuit Breaker com estado persistente no Redis"""
    
    def __init__(
        self,
        name: str,
        failure_threshold: int = 5,
        reset_timeout: int = 60,
        redis_client: Optional[Redis] = None
    ):
        self.name = name
        self.failure_threshold = failure_threshold
        self.reset_timeout = reset_timeout
        self.redis = redis_client
        self.metrics = CircuitBreakerMetrics()
    
    async def _get_metrics(self) -> CircuitBreakerMetrics:
        """Recupera métricas do Redis ou usa cache local"""
        if self.redis:
            try:
                data = await self.redis.get(f"circuit:{self.name}")
                if data:
                    return CircuitBreakerMetrics(**json.loads(data))
            except Exception:
                pass
        return self.metrics
    
    async def _save_metrics(self, metrics: CircuitBreakerMetrics):
        """Salva métricas no Redis"""
        self.metrics = metrics
        if self.redis:
            try:
                await self.redis.setex(
                    f"circuit:{self.name}",
                    self.reset_timeout * 2,
                    json.dumps(asdict(metrics))
                )
            except Exception:
                pass
    
    async def can_execute(self) -> bool:
        """Verifica se o circuito permite execução"""
        metrics = await self._get_metrics()
        
        if metrics.state == CircuitState.OPEN:
            if metrics.last_failure_time:
                time_since_failure = time.time() - metrics.last_failure_time
                if time_since_failure > self.reset_timeout:
                    metrics.state = CircuitState.HALF_OPEN
                    await self._save_metrics(metrics)
                    return True
            return False
        
        return True
    
    async def on_success(self):
        """Registra sucesso e fecha o circuito se estava meio-aberto"""
        metrics = await self._get_metrics()
        
        if metrics.state == CircuitState.HALF_OPEN:
            metrics.state = CircuitState.CLOSED
            metrics.failures = 0
        
        metrics.successes += 1
        await self._save_metrics(metrics)
    
    async def on_failure(self):
        """Registra falha e potencialmente abre o circuito"""
        metrics = await self._get_metrics()
        
        metrics.failures += 1
        metrics.last_failure_time = time.time()
        
        if metrics.failures >= self.failure_threshold:
            metrics.state = CircuitState.OPEN
        
        await self._save_metrics(metrics)

class RetryPolicy:
    """Política de retry com backoff exponencial e jitter"""
    
    def __init__(
        self,
        max_retries: int = 3,
        base_delay: float = 0.1,
        max_delay: float = 10.0,
        retry_on_status: List[int] = None
    ):
        self.max_retries = max_retries
        self.base_delay = base_delay
        self.max_delay = max_delay
        self.retry_on_status = retry_on_status or [500, 502, 503, 504, 522, 524, 408, 429]
    
    async def execute_with_retry(self, func, *args, **kwargs):
        """Executa função com retry automático"""
        last_exception = None
        
        for attempt in range(self.max_retries + 1):
            try:
                return await func(*args, **kwargs)
            except Exception as e:
                last_exception = e
                
                # Verifica se deve retentar
                if not self._should_retry(e) or attempt == self.max_retries:
                    raise
                
                # Calcula delay com backoff exponencial e jitter
                delay = min(
                    self.base_delay * (2 ** attempt) + (random.random() * 0.1),
                    self.max_delay
                )
                
                await asyncio.sleep(delay)
        
        raise last_exception
    
    def _should_retry(self, exception: Exception) -> bool:
        """Determina se uma exceção deve ser retentada"""
        if isinstance(exception, httpx.HTTPStatusError):
            return exception.response.status_code in self.retry_on_status
        return True

class MonitoringSetup:
    """Configuração centralizada de monitoramento"""
    
    def __init__(self, config: AppConfig):
        self.config = config
        self.tracer = None
        self.metrics_registry = prometheus_client.CollectorRegistry()
        
        self._setup_tracing()
        self._setup_metrics()
    
    def _setup_tracing(self):
        """Configura OpenTelemetry tracing"""
        if self.config.enable_tracing and self.config.otlp_endpoint:
            trace.set_tracer_provider(TracerProvider())
            
            span_processor = BatchSpanProcessor(
                OTLPSpanExporter(endpoint=self.config.otlp_endpoint)
            )
            trace.get_tracer_provider().add_span_processor(span_processor)
            
            self.tracer = trace.get_tracer(__name__)
    
    def _setup_metrics(self):
        """Configura métricas Prometheus"""
        if self.config.enable_metrics:
            # Métricas customizadas
            self.request_duration = prometheus_client.Histogram(
                'http_request_duration_seconds',
                'HTTP request duration in seconds',
                ['method', 'endpoint', 'status'],
                registry=self.metrics_registry
            )
            
            self.external_api_calls = prometheus_client.Counter(
                'external_api_calls_total',
                'Total external API calls',
                ['service', 'status'],
                registry=self.metrics_registry
            )
            
            self.circuit_breaker_state = prometheus_client.Gauge(
                'circuit_breaker_state',
                'Circuit breaker state (0=closed, 1=half_open, 2=open)',
                ['service'],
                registry=self.metrics_registry
            )
    
    @contextmanager
    def trace_span(self, name: str, attributes: Dict = None):
        """Context manager para tracing"""
        if self.tracer:
            with self.tracer.start_as_current_span(name, attributes=attributes) as span:
                yield span
        else:
            yield None
    
    def record_request_metrics(self, method: str, endpoint: str, status: int, duration: float):
        """Registra métricas de requisição HTTP"""
        if self.config.enable_metrics:
            self.request_duration.labels(method, endpoint, str(status)).observe(duration)

class DatabaseManager:
    """Gerenciador de conexões de banco de dados com pool otimizado"""
    
    def __init__(self, config: AppConfig):
        self.config = config
        self.pool = None
        self.logger = StructuredLogger("database")
    
    async def initialize(self):
        """Inicializa pool de conexões"""
        try:
            self.pool = await asyncpg.create_pool(
                dsn=self.config.database_url,
                min_size=self.config.database_pool_min,
                max_size=self.config.database_pool_max,
                command_timeout=60,
                max_queries=50000,
                max_inactive_connection_lifetime=300,
            )
            
            # Criar tabelas se não existirem
            await self._create_tables()
            
            self.logger.log(
                "INFO",
                "Database pool initialized",
                "system",
                pool_size=f"{self.config.database_pool_min}-{self.config.database_pool_max}"
            )
            
        except Exception as e:
            self.logger.log(
                "ERROR",
                "Failed to initialize database pool",
                "system",
                error=str(e),
                database_url=self._mask_database_url()
            )
            raise
    
    def _mask_database_url(self) -> str:
        """Mascara URL do banco para logs"""
        if "@" in self.config.database_url:
            parts = self.config.database_url.split("@")
            return f"***@{parts[1]}" if len(parts) > 1 else "***"
        return "***"
    
    async def _create_tables(self):
        """Cria tabelas necessárias"""
        async with self.pool.acquire() as conn:
            # Tabela de usuários
            await conn.execute("""
                CREATE TABLE IF NOT EXISTS users (
                    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
                    email VARCHAR(255) UNIQUE NOT NULL,
                    name VARCHAR(255) NOT NULL,
                    hashed_password VARCHAR(255) NOT NULL,
                    is_active BOOLEAN DEFAULT true,
                    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
                    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
                );
                
                CREATE INDEX IF NOT EXISTS idx_users_email ON users(email);
                CREATE INDEX IF NOT EXISTS idx_users_active ON users(is_active) WHERE is_active = true;
            """)
            
            # Tabela de produtos
            await conn.execute("""
                CREATE TABLE IF NOT EXISTS products (
                    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
                    name VARCHAR(255) NOT NULL,
                    description TEXT,
                    price DECIMAL(10,2) NOT NULL,
                    stock INTEGER DEFAULT 0,
                    category VARCHAR(100),
                    metadata JSONB DEFAULT '{}',
                    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
                    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
                );
                
                CREATE INDEX IF NOT EXISTS idx_products_category ON products(category);
                CREATE INDEX IF NOT EXISTS idx_products_price ON products(price);
                CREATE INDEX IF NOT EXISTS idx_products_metadata ON products USING gin(metadata);
            """)
            
            # Tabela de pedidos
            await conn.execute("""
                CREATE TABLE IF NOT EXISTS orders (
                    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
                    user_id UUID REFERENCES users(id),
                    status VARCHAR(50) DEFAULT 'pending',
                    total_amount DECIMAL(10,2) NOT NULL,
                    payment_status VARCHAR(50) DEFAULT 'pending',
                    shipping_address JSONB,
                    items JSONB NOT NULL,
                    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
                    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
                );
                
                CREATE INDEX IF NOT EXISTS idx_orders_user_id ON orders(user_id);
                CREATE INDEX IF NOT EXISTS idx_orders_status ON orders(status);
                CREATE INDEX IF NOT EXISTS idx_orders_created_at ON orders(created_at DESC);
            """)
    
    async def execute_query(self, query: str, *args, correlation_id: str = None):
        """Executa query com parâmetros preparados (zero SQL injection)"""
        start_time = time.time()
        
        try:
            async with self.pool.acquire() as conn:
                result = await conn.fetch(query, *args)
                
                duration = time.time() - start_time
                self.logger.log(
                    "DEBUG",
                    "Query executed successfully",
                    correlation_id or "system",
                    query=query[:100],  # Log apenas parte da query
                    params_count=len(args),
                    duration_ms=round(duration * 1000, 2)
                )
                
                return result
                
        except Exception as e:
            duration = time.time() - start_time
            self.logger.log(
                "ERROR",
                "Query execution failed",
                correlation_id or "system",
                query=query[:100],
                error=str(e),
                duration_ms=round(duration * 1000, 2)
            )
            raise
    
    async def close(self):
        """Fecha pool de conexões"""
        if self.pool:
            await self.pool.close()

  deleteProject: (id: string) => void;
  setSelectedProjectId: (id: string | null) => void;
  toggleSidebar: () => void;
  setTheme: (theme: 'light' | 'dark') => void;

      // Initial state
      user: null,
      projects: [],
      selectedProjectId: null,
      sidebarCollapsed: false,
      theme: 'light',

      // Actions
      setUser: (user) => set({ user }),
      
      addProject: (projectData) => {
        const newProject: Project = {
          ...projectData,
          id: crypto.randomUUID(),
          createdAt: new Date(),
          updatedAt: new Date(),
        };
        set((state) => ({
          projects: [...state.projects, newProject],
        }));
      },

      updateProject: (id, updates) => {
        set((state) => ({
          projects: state.projects.map((project) =>
            project.id === id
              ? { ...project, ...updates, updatedAt: new Date() }
              : project
          ),
        }));
      },

      deleteProject: (id) => {
        set((state) => ({
          projects: state.projects.filter((project) => project.id !== id),
          selectedProjectId: state.selectedProjectId === id ? null : state.selectedProjectId,
        }));
      },

      setSelectedProjectId: (id) => set({ selectedProjectId: id }),

      toggleSidebar: () =>
        set((state) => ({ sidebarCollapsed: !state.sidebarCollapsed })),

      setTheme: (theme) => set({ theme }),

      // Derived state
      selectedProject: () => {
        const { projects, selectedProjectId } = get();
        return projects.find((p) => p.id === selectedProjectId);
      },

      activeProjects: () => {
        const { projects } = get();
        return projects.filter((p) => p.status === 'active');
      },
    }),
    {
      name: 'diyapp-storage',
      partialize: (state) => ({
        user: state.user,
        projects: state.projects,

WITH raw_events AS (
    SELECT
        event_id,
        user_id,
        event_type,
        -- Garantir timestamp em UTC
        CASE 
            WHEN event_timestamp LIKE '%Z' THEN CAST(event_timestamp AS TIMESTAMP)
            ELSE CAST(event_timestamp || 'Z' AS TIMESTAMP AT TIME ZONE 'UTC')
        END AS event_timestamp,
        feature_name,
        properties::JSONB AS properties,
        ingested_at

    WHERE event_timestamp IS NOT NULL
      AND user_id IS NOT NULL
      AND event_id IS NOT NULL
)

    event_id,
    user_id,
    LOWER(TRIM(event_type)) AS event_type,
    event_timestamp,
    CASE 
        WHEN feature_name IS NOT NULL THEN LOWER(TRIM(feature_name))
        ELSE NULL
    END AS feature_name,
    properties,
    ingested_at
FROM raw_events

        DATE(event_timestamp) AS event_date,
        feature_name,
        user_id,
        COUNT(*) AS usage_count
    FROM {{ ref('stg_events') }}
    WHERE feature_name IS NOT NULL
      AND event_type = 'feature_click'
    GROUP BY 1, 2, 3
),

    SELECT
        DATE(event_timestamp) AS event_date,
        COUNT(DISTINCT user_id) AS total_active_users
    FROM {{ ref('stg_events') }}
    WHERE event_type IN ('page_view', 'feature_click', 'conversion')

),

        user_id,
        plan_type
    FROM {{ ref('stg_users') }}
)

SELECT
    fe.event_date,
    fe.feature_name,
    up.plan_type,
    COUNT(DISTINCT fe.user_id) AS unique_users,
    SUM(fe.usage_count) AS total_usage,
    du.total_active_users,
    -- Taxa de adoção: usuários que usaram a feature / total de usuários ativos
    ROUND(
        COUNT(DISTINCT fe.user_id) * 100.0 / NULLIF(du.total_active_users, 0),
        2
    ) AS adoption_rate_percent,
    CURRENT_TIMESTAMP AS calculated_at
FROM feature_events fe
LEFT JOIN user_plan up ON fe.user_id = up.user_id
LEFT JOIN daily_users du ON fe.event_date = du.event_date
GROUP BY 1, 2, 3, du.total_active_users

    this.app.use(cors(config.cors));
    this.app.use(express.json({ limit: '10mb' }));
    this.app.use(express.urlencoded({ extended: true }));
    this.app.use(morgan('combined', { stream: logger.stream }));
    this.app.use(express.static('public'));
  }
  
  setupRoutes() {
    this.app.use('/api/v1', dashboardRoutes);
    
    // Health check
    this.app.get('/health', (req, res) => {
      res.json({
        status: 'healthy',
        timestamp: new Date().toISOString(),
        agents: Object.keys(this.agents).length,
        uptime: process.uptime()
      });
    });
    
    // Metrics endpoint
    this.app.get('/metrics', (req, res) => {
      res.json({
        system: this.metrics.system || {},
        llm: this.metrics.llm || {},
        alerts: this.alerts.slice(-100) // Last 100 alerts
      });
    });
  }
  
  setupWebSocket() {
    this.io.on('connection', (socket) => {
      logger.info(`Client connected: ${socket.id}`);
      
      // Send current metrics on connection
      socket.emit('metrics:update', this.metrics);
      socket.emit('alerts:update', this.alerts.slice(-20));
      
      socket.on('disconnect', () => {
        logger.info(`Client disconnected: ${socket.id}`);
      });
    });
    
    // Broadcast metrics updates every 5 seconds
    setInterval(() => {
      if (Object.keys(this.metrics).length > 0) {
        this.io.emit('metrics:update', this.metrics);
      }
    }, 5000);
  }
  
  setupErrorHandling() {
    this.app.use((err, req, res, next) => {
      logger.error('Unhandled error:', err);
      res.status(500).json({
        error: 'Internal server error',
        message: config.env === 'development' ? err.message : 'An error occurred'
      });
    });
  }
  
  async initialize() {
    try {
      logger.info('Initializing Observability System V11...');
      
      // Initialize database
      await initializeDatabase();
      logger.info('Database initialized');
      
      // Initialize agents
      this.agents = await initializeAgents(this);
      logger.info(`Agents initialized: ${Object.keys(this.agents).join(', ')}`);
      
      // Start agents
      Object.values(this.agents).forEach(agent => agent.start());
      logger.info('All agents started');
      
      // Start server
      const PORT = config.port || 3000;
      this.server.listen(PORT, () => {
        logger.info(`Observability System running on port ${PORT}`);
        logger.info(`Dashboard available at http://localhost:${PORT}`);
      });
      
    } catch (error) {
      logger.error('Failed to initialize system:', error);
      process.exit(1);
    }
  }
  
  updateMetrics(metrics) {
    this.metrics = { ...this.metrics, ...metrics };
    
    // Broadcast via WebSocket
    this.io.emit('metrics:update', this.metrics);
    
    // Check for anomalies
    Object.values(this.agents).forEach(agent => {
      if (agent.checkAnomalies) {
        agent.checkAnomalies(this.metrics);
      }
    });
  }
  
  addAlert(alert) {
    const alertWithId = {
      id: Date.now().toString(36) + Math.random().toString(36).substr(2),
      timestamp: new Date().toISOString(),
      ...alert
    };
    
    this.alerts.push(alertWithId);
    
    // Keep only last 1000 alerts
    if (this.alerts.length > 1000) {
      this.alerts = this.alerts.slice(-1000);
    }

    this.io.emit('alert:new', alertWithId);
    this.io.emit('alerts:update', this.alerts.slice(-20));
    
    logger[alert.severity === 'critical' ? 'error' : 'warn'](

  loadDetectionRules() {

      // Cost anomaly rules
      {
        id: 'cost-spike',
        type: 'cost',
        condition: (current, history) => {
          const avgCost = this.calculateAverage(history, 'totalCost', 7);
          return current.totalCost > avgCost * (1 + config.anomaly.detection.costIncreaseThreshold);
        },
        severity: 'high',
        message: (data) => `Cost spike detected: ${data.increase.toFixed(1)}% increase`
      },
      
      // Latency anomaly rules
      {
        id: 'latency-spike',
        type: 'latency',
        condition: (current, history) => {
          const avgLatency = this.calculateAverage(history, 'latency.p95', 24);
          return current.latency?.p95 > avgLatency * config.anomaly.detection.latencySpikeThreshold;
        },
        severity: 'medium',
        message: (data) => `Latency spike detected: P95 at ${data.latency}ms`
      },
      
      // Token usage anomaly
      {
        id: 'token-spike',
        type: 'usage',
        condition: (current, history) => {
          const avgTokens = this.calculateAverage(history, 'tokensPerCall', 24);
          return current.tokensPerCall > avgTokens * config.anomaly.detection.tokenSpikeThreshold;
        },
        severity: 'medium',
        message: (data) => `Token usage spike: ${data.tokens} tokens/call (${data.increase.toFixed(1)}x average)`
      },
      
      // Error rate anomaly

        id: 'error-spike',

  calculateAverage(history, metricPath, hours) {
    if (history.length === 0) return 0;

    return values.reduce((sum, val) => sum + val, 0) / values.length;
  }

# DIYAPP Evolution - V11 Core - Conformidade LGPD/GDPR para Telemetria

## Análise de Conformidade - Sistema de Telemetria V11

### 1. DPIA (Data Protection Impact Assessment) - Telemetria V11

**DPIA - Sistema de Telemetria V11**
**Data:** 2024-05-20 | **Responsável:** Compliance + Product Manager
**Feature vinculada:** [TEL-001] Implementação de telemetria para monitoramento de uso e performance

#### 1. DESCRIÇÃO DO PROCESSAMENTO:
**Dados coletados:**
- Dados de uso da aplicação (eventos de clique, navegação, tempo de sessão)
- Métricas de performance (tempo de carregamento, erros, latência)
- Informações técnicas do dispositivo (navegador, sistema operacional, resolução)
- ID de sessão anonimizado (não vinculado a usuário identificado)
- País/região (baseado em IP, sem precisão de geolocalização)

**Finalidade:** Melhorar performance da aplicação, identificar problemas técnicos, otimizar experiência do usuário

**Base legal (LGPD):** [X] Legítimo interesse (Art. 7º, IX) - para melhorar qualidade do serviço
**Base legal (GDPR):** [X] Legítimo interesse (Art. 6(1)(f)) - para melhorias no serviço

**Responsável pelo tratamento:** DIYAPP Evolution
**Operadores envolvidos:** AWS CloudWatch (métricas), PostgreSQL (logs), Sentry (erros)

#### 2. AVALIAÇÃO DE NECESSIDADE E PROPORCIONALIDADE:
**Os dados são necessários para a finalidade?** Sim - dados de performance são essenciais para identificar e corrigir problemas técnicos

**Há alternativa menos invasiva?** Sim - implementamos anonimização em tempo real e não coletamos dados pessoais identificáveis

**Prazo de retenção:** 30 dias - suficiente para análise de tendências e troubleshooting

**Processo de exclusão:** Automático - dados são excluídos após 30 dias via política de retenção do banco de dados

#### 3. RISCOS IDENTIFICADOS PARA OS TITULARES:
**Risco:** Possível identificação indireta através de padrões de uso únicos
**Probabilidade:** Baixa | **Impacto:** Baixo
**Medida de mitigação:** Anonimização em tempo real, não coleta de IP completo, hashing de identificadores

**Risco:** Vazamento de dados em caso de ataque
**Probabilidade:** Média | **Impacto:** Médio
**Medida de mitigação:** Criptografia em repouso e trânsito, acesso restrito, monitoramento de segurança

#### 4. USO DE LLM NO PROCESSAMENTO (se aplicável):
Não aplicável - telemetria não utiliza LLMs

#### 5. CONCLUSÃO:
**Risco residual:** Baixo
**Aprovado para implementação:** Sim
**Condições:** 
1. Implementar banner de cookies para usuários da UE
2. Garantir mecanismo de opt-out
3. Documentar medidas de segurança
**Revisão agendada:** 2024-08-20
