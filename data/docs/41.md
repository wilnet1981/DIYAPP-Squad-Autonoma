## 1. Análise da Arquitetura V10

### ADR-001: Análise da Arquitetura V10 e Diretrizes para V11
**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead

**CONTEXTO:**
A arquitetura V10 do DIYAPP evoluiu organicamente, resultando em:
1. Acoplamento excessivo entre módulos
2. Logging inconsistente (console.log misturado com Winston)
3. Estrutura de pastas não padronizada
4. Testes com cobertura insuficiente (< 60%)
5. Dependências críticas desatualizadas

**DECISÃO:**
Implementar arquitetura V11 com:
1. Estrutura modular baseada em domínios
2. Sistema de logging unificado com níveis e rotatividade
3. Divisão estratégica de código em módulos independentes
4. Padrões de código documentados em Engineering Standards

**OPÇÕES CONSIDERADAS:**
- **Opção A:** Refatoração incremental mantendo estrutura atual
  - Prós: Menor risco imediato
  - Contras: Débito técnico permanece, dificulta evolução futura
- **Opção B:** Reestruturação completa com nova arquitetura
  - Prós: Base limpa para evolução, padrões consistentes
  - Contras: Esforço maior inicial, risco de regressão
- **Opção escolhida:** B com Modo Hive para refatoração paralela

**CONSEQUÊNCIAS:**
**Positivas:**
- Base de código sustentável para próximos 2 anos
- Padronização que permite onboarding rápido
- Sistema de logging que facilita debugging em produção

**Negativas:**
- Curva de aprendizado para time atual
- Necessidade de migração gradual de funcionalidades

**Riscos:**
- Regressão de funcionalidades durante migração
- Mitigação: Testes E2E antes de cada deploy, feature flags

## 3. Sistema de Logging Unificado (História 2)

### ADR-002: Sistema de Logging Centralizado
**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra

**DECISÃO:**
Implementar sistema de logging baseado em Winston com:
1. Níveis padrão (error, warn, info, debug)
2. Rotatividade diária com retenção de 30 dias
3. Formato estruturado (JSON em produção)
4. Integração com Sentry para errors
5. Contexto automático (requestId, userId, module)

**Implementação:**

```typescript
// src/core/logging/logger.ts
import winston from 'winston';
import DailyRotateFile from 'winston-daily-rotate-file';
import { config } from '../config';

const { combine, timestamp, json, printf, colorize } = winston.format;

// Formato para desenvolvimento (legível)
const devFormat = printf(({ level, message, timestamp, module, requestId }) => {
  return `${timestamp} [${requestId || 'system'}] ${module || 'app'}.${level}: ${message}`;
});

// Formato para produção (JSON estruturado)
const prodFormat = combine(timestamp(), json());

// Transportes
const transports: winston.transport[] = [
  new DailyRotateFile({
    filename: 'logs/application-%DATE%.log',
    datePattern: 'YYYY-MM-DD',
    maxFiles: '30d',
    level: 'info',
    format: prodFormat
  }),
  new DailyRotateFile({
    filename: 'logs/error-%DATE%.log',
    datePattern: 'YYYY-MM-DD',
    maxFiles: '30d',
    level: 'error',
    format: prodFormat
  })
];

// Console apenas em desenvolvimento
if (config.NODE_ENV !== 'production') {
  transports.push(
    new winston.transports.Console({
      level: 'debug',
      format: combine(timestamp(), colorize(), devFormat)
    })
  );
}

// Logger base
const baseLogger = winston.createLogger({
  level: config.LOG_LEVEL || 'info',
  defaultMeta: {
    service: 'diyapp-v11',
    environment: config.NODE_ENV
  },
  transports,
  exceptionHandlers: [
    new DailyRotateFile({
      filename: 'logs/exceptions-%DATE%.log',
      datePattern: 'YYYY-MM-DD',
      maxFiles: '30d'
    })
  ],
  rejectionHandlers: [
    new DailyRotateFile({
      filename: 'logs/rejections-%DATE%.log',
      datePattern: 'YYYY-MM-DD',
      maxFiles: '30d'
    })
  ]
});

// Factory para loggers com contexto
export class LoggerFactory {
  static create(module: string) {
    return {
      error: (message: string, meta?: any) => 
        baseLogger.error(message, { ...meta, module }),
      warn: (message: string, meta?: any) => 
        baseLogger.warn(message, { ...meta, module }),
      info: (message: string, meta?: any) => 
        baseLogger.info(message, { ...meta, module }),
      debug: (message: string, meta?: any) => 
        baseLogger.debug(message, { ...meta, module }),
      
      // Métodos com contexto de request
      withRequest(requestId: string, userId?: string) {
        return {
          error: (message: string, meta?: any) => 
            baseLogger.error(message, { ...meta, module, requestId, userId }),
          warn: (message: string, meta?: any) => 
            baseLogger.warn(message, { ...meta, module, requestId, userId }),
          info: (message: string, meta?: any) => 
            baseLogger.info(message, { ...meta, module, requestId, userId }),
          debug: (message: string, meta?: any) => 
            baseLogger.debug(message, { ...meta, module, requestId, userId })
        };
      }
    };
  }
}

// Middleware para Express
export const requestLogger = (req: any, res: any, next: any) => {
  const requestId = req.headers['x-request-id'] || Date.now().toString(36);
  req.requestId = requestId;
  
  const logger = LoggerFactory.create('http').withRequest(requestId, req.user?.id);
  
  // Log da requisição
  logger.info(`${req.method} ${req.url}`, {
    ip: req.ip,
    userAgent: req.headers['user-agent']
  });
  
  // Log da resposta
  const originalSend = res.send;
  res.send = function(body: any) {
    logger.info(`Response ${res.statusCode}`, {
      statusCode: res.statusCode,
      contentLength: body?.length || 0
    });
    return originalSend.call(this, body);
  };
  
  next();
};
```

## 4. Estratégia de Divisão de Código (História 3)

### ADR-003: Divisão Modular com Lazy Loading
**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead

**DECISÃO:**
Dividir código em módulos independentes com:
1. Carregamento dinâmico baseado em feature flags
2. Dependências explícitas entre módulos
3. Interface de comunicação via eventos
4. Isolamento de falhas (um módulo não derruba o sistema)

**Implementação:**

```typescript
// src/core/modules/module-manager.ts
import { EventEmitter } from 'events';
import { LoggerFactory } from '../logging';

interface ModuleConfig {
  name: string;
  enabled: boolean;
  dependencies: string[];
  load: () => Promise<any>;
  unload?: () => Promise<void>;
}

export class ModuleManager extends EventEmitter {
  private modules: Map<string, any> = new Map();
  private configs: Map<string, ModuleConfig> = new Map();
  private logger = LoggerFactory.create('modules');

  async registerModule(config: ModuleConfig): Promise<void> {
    this.configs.set(config.name, config);
    this.logger.info(`Module registered: ${config.name}`);
  }

  async loadModule(name: string): Promise<any> {
    const config = this.configs.get(name);
    if (!config) {
      throw new Error(`Module ${name} not registered`);
    }

    // Verificar dependências
    for (const dep of config.dependencies) {
      if (!this.modules.has(dep)) {
        await this.loadModule(dep);
      }
    }

    // Carregar módulo
    if (config.enabled) {
      this.logger.info(`Loading module: ${name}`);
      try {
        const module = await config.load();
        this.modules.set(name, module);
        this.emit('module:loaded', { name, module });
        this.logger.info(`Module loaded successfully: ${name}`);
        return module;
      } catch (error) {
        this.logger.error(`Failed to load module ${name}:`, { error });
        throw error;
      }
    } else {
      this.logger.warn(`Module ${name} is disabled`);
      return null;
    }
  }

  async unloadModule(name: string): Promise<void> {
    const config = this.configs.get(name);
    if (!config) return;

    if (config.unload) {
      this.logger.info(`Unloading module: ${name}`);
      try {
        await config.unload();
        this.modules.delete(name);
        this.emit('module:unloaded', { name });
        this.logger.info(`Module unloaded: ${name}`);
      } catch (error) {
        this.logger.error(`Failed to unload module ${name}:`, { error });
      }
    }
  }

  getModule<T = any>(name: string): T | null {
    return this.modules.get(name) || null;
  }

  async reloadModule(name: string): Promise<any> {
    await this.unloadModule(name);
    return this.loadModule(name);
  }

  listModules(): Array<{ name: string; loaded: boolean; enabled: boolean }> {
    return Array.from(this.configs.values()).map(config => ({
      name: config.name,
      loaded: this.modules.has(config.name),
      enabled: config.enabled
    }));
  }
}

// Singleton global
export const moduleManager = new ModuleManager();
```

import { moduleManager } from './module-manager';
import { LoggerFactory } from '../logging';

export interface IExampleModule {
  doSomething(): Promise<string>;
  getStatus(): { healthy: boolean };
}

class ExampleModule implements IExampleModule {
  private logger = LoggerFactory.create('example-module');
  private healthy = true;

  async initialize(): Promise<void> {
    this.logger.info('Initializing example module');
    // Inicialização assíncrona
    await new Promise(resolve => setTimeout(resolve, 100));
    this.logger.info('Example module initialized');
  }

  async doSomething(): Promise<string> {
    this.logger.debug('Doing something...');
    return 'done';
  }

  getStatus(): { healthy: boolean } {
    return { healthy: this.healthy };
  }

  async cleanup(): Promise<void> {
    this.logger.info('Cleaning up example module');
    this.healthy = false;
  }
}

// Registrar módulo
moduleManager.registerModule({
  name: 'example',
  enabled: true,
  dependencies: ['logging'],
  load: async () => {
    const module = new ExampleModule();
    await module.initialize();
    return module;
  },
  unload: async () => {
    const module = moduleManager.getModule<ExampleModule>('example');
    if (module) {
      await module.cleanup();
    }
  }
});

// src/contexts/ThemeContext.tsx
import React, { createContext, useContext, useState, useEffect, ReactNode } from 'react';

type ThemeMode = 'light' | 'dark' | 'system';

interface ThemeContextType {
  mode: ThemeMode;
  theme: 'light' | 'dark';
  toggleMode: () => void;
  setMode: (mode: ThemeMode) => void;
}

const ThemeContext = createContext<ThemeContextType | undefined>(undefined);

export const useTheme = () => {
  const context = useContext(ThemeContext);
  if (!context) {
    throw new Error('useTheme must be used within ThemeProvider');
  }
  return context;
};

interface ThemeProviderProps {
  children: ReactNode;
}

export const ThemeProvider: React.FC<ThemeProviderProps> = ({ children }) => {
  const [mode, setMode] = useState<ThemeMode>(() => {
    const saved = localStorage.getItem('theme-mode');
    return (saved as ThemeMode) || 'system';
  });

  const [theme, setTheme] = useState<'light' | 'dark'>(() => {
    if (mode === 'system') {
      return window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light';
    }
    return mode;
  });

  useEffect(() => {
    const root = document.documentElement;
    root.setAttribute('data-theme', theme);
    localStorage.setItem('theme-mode', mode);
  }, [theme, mode]);

  useEffect(() => {
    if (mode === 'system') {
      const mediaQuery = window.matchMedia('(prefers-color-scheme: dark)');
      const handleChange = (e: MediaQueryListEvent) => {
        setTheme(e.matches ? 'dark' : 'light');
      };

      mediaQuery.addEventListener('change', handleChange);
      return () => mediaQuery.removeEventListener('change', handleChange);
    }
  }, [mode]);

  const toggleMode = () => {
    setMode(current => {
      if (current === 'light') return 'dark';
      if (current === 'dark') return 'system';
      return 'light';
    });
  };

  const value = {
    mode,
    theme,
    toggleMode,
    setMode,
  };

  return <ThemeContext.Provider value={value}>{children}</ThemeContext.Provider>;
};

  const fetchWithRetry = useCallback(async <T,>(
    fetchFn: () => Promise<T>,
    maxRetries = 3
  ): Promise<ApiResponse<T>> => {
    let retries = 0;
    let lastError: Error | null = null;

    while (retries <= maxRetries) {
      try {
        const data = await fetchFn();
        return { data, loading: false, error: null, refetch: async () => {} };
      } catch (error) {
        lastError = error as Error;
        retries++;
        
        if (retries <= maxRetries) {
          await delay(1000 * Math.pow(2, retries - 1)); // Exponential backoff
        }
      }
    }

    return {
      data: null,
      loading: false,
      error: lastError?.message || 'Request failed after retries',
      refetch: async () => {},
    };
  }, []);

  const fetchDashboardStats = useCallback(async () => {
    return fetchWithRetry(async () => {
      await delay(800); // Simulate network delay
      // Simulate 10% chance of error for testing
      if (Math.random() < 0.1) {
        throw new Error('Failed to fetch dashboard stats');
      }
      return mockDashboardStats;

@dataclass
class LatencyTracker:
    """Rastreia latências para cálculo de percentis."""
    def __init__(self, window_size: int = 1000):
        self.window_size = window_size
        self.latencies = deque(maxlen=window_size)
        self.lock = threading.Lock()
    
    def add_latency(self, latency_ms: float):
        """Adiciona uma nova medição de latência."""
        with self.lock:
            self.latencies.append(latency_ms)
    
    def get_p95(self) -> float:
        """Calcula o P95 das latências."""
        with self.lock:
            if not self.latencies:
                return 0.0
            
            sorted_latencies = sorted(self.latencies)
            index = int(len(sorted_latencies) * 0.95)
            return sorted_latencies[min(index, len(sorted_latencies) - 1)]

class MetricsCollector:
    """Coletor de métricas do sistema."""
    
    def __init__(self, log_repo: LogRepository, collect_interval: int = 60):
        """
        Args:
            log_repo: Repositório para salvar métricas
            collect_interval: Intervalo de coleta em segundos
        """
        self.log_repo = log_repo
        self.collect_interval = collect_interval
        self.latency_tracker = LatencyTracker()
        self.request_counter = 0
        self.error_counter = 0
        self.active_connections = 0
        self.collector_thread: Optional[threading.Thread] = None
        self.running = False
        self.logger = logging.getLogger(__name__)
    
    def start(self):
        """Inicia o coletor de métricas em background."""
        if self.running:
            return
        
        self.running = True
        self.collector_thread = threading.Thread(
            target=self._collect_loop,
            daemon=True,
            name="MetricsCollector"
        )
        self.collector_thread.start()
        self.logger.info(f"Métricas coletor iniciado (intervalo: {self.collect_interval}s)")
    
    def stop(self):
        """Para o coletor de métricas."""
        self.running = False
        if self.collector_thread:
            self.collector_thread.join(timeout=5)
    
    def _collect_loop(self):
        """Loop principal de coleta de métricas."""
        while self.running:
            try:
                self.collect_and_save_metrics()
                time.sleep(self.collect_interval)
            except Exception as e:
                self.logger.error(f"Erro ao coletar métricas: {e}")
                time.sleep(min(self.collect_interval, 10))
    
    def collect_and_save_metrics(self):
        """Coleta e salva métricas do sistema."""
        try:
            # Coleta métricas do sistema
            cpu_percent = psutil.cpu_percent(interval=1)
            memory = psutil.virtual_memory()
            disk = psutil.disk_usage('/')
            
            # Calcula métricas da aplicação
            total_requests = self.request_counter
            error_rate = (self.error_counter / total_requests * 100) if total_requests > 0 else 0.0
            p95_latency = self.latency_tracker.get_p95()
            
            # Cria objeto de métricas
            metrics = SystemMetrics(
                timestamp=datetime.utcnow(),
                cpu_percent=cpu_percent,
                memory_percent=memory.percent,
                memory_used_mb=memory.used / (1024 * 1024),
                memory_total_mb=memory.total / (1024 * 1024),
                disk_usage_percent=disk.percent,
                active_connections=self.active_connections,
                request_count=total_requests,
                error_rate=error_rate,
                p95_latency_ms=p95_latency
            )
            
            # Salva no banco
            self.log_repo.save_metrics(metrics)
            
            # Reseta contadores para o próximo intervalo
            self.request_counter = 0
            self.error_counter = 0
            
            self.logger.debug(f"Métricas coletadas: CPU={cpu_percent}%, Mem={memory.percent}%")
            
        except Exception as e:
            self.logger.error(f"Falha ao coletar métricas: {e}")
    
    def record_request(self, duration_ms: float, is_error: bool = False):
        """Registra uma requisição para métricas."""
        self.request_counter += 1
        if is_error:
            self.error_counter += 1
        
        if duration_ms > 0:
            self.latency_tracker.add_latency(duration_ms)
    
    def increment_connections(self):

        self.active_connections += 1

        """Decrementa contador de conexões ativas."""
        if self.active_connections > 0:
            self.active_connections -= 1

// src/shared/logging/logger.ts
import winston from 'winston';
import { config } from '../../config';

export type LogLevel = 'error' | 'warn' | 'info' | 'debug' | 'trace';
export type LogContext = {
  module: string;
  operation: string;
  correlationId?: string;
  userId?: string;
  [key: string]: any;
};

class Logger {
  private logger: winston.Logger;
  private static instance: Logger;

  private constructor() {
    const isProduction = config.NODE_ENV === 'production';
    
    const formats = [
      winston.format.timestamp({ format: 'YYYY-MM-DD HH:mm:ss' }),
      winston.format.errors({ stack: true }),
    ];

    if (isProduction) {
      formats.push(winston.format.json());
    } else {
      formats.push(
        winston.format.colorize(),
        winston.format.printf(({ timestamp, level, message, module, operation, correlationId, ...meta }) => {
          return `${timestamp} [${correlationId || 'no-id'}] ${level} ${module}.${operation}: ${message} ${
            Object.keys(meta).length ? JSON.stringify(meta) : ''
          }`;
        })
      );
    }

    this.logger = winston.createLogger({
      level: config.LOG_LEVEL || 'info',
      format: winston.format.combine(...formats),
      transports: [
        new winston.transports.Console(),
        ...(config.LOG_FILE ? [new winston.transports.File({ filename: config.LOG_FILE })] : []),
      ],
      silent: config.NODE_ENV === 'test',
    });
  }

  static getInstance(): Logger {
    if (!Logger.instance) {
      Logger.instance = new Logger();
    }
    return Logger.instance;
  }

  log(level: LogLevel, message: string, context: LogContext): void {
    this.logger.log(level, message, context);
  }

  error(message: string, error?: Error, context: LogContext): void {
    this.logger.error(message, {
      ...context,
      error: error ? {
        message: error.message,
        stack: error.stack,
        name: error.name,
      } : undefined,
    });
  }

  warn(message: string, context: LogContext): void {
    this.logger.warn(message, context);
  }

  info(message: string, context: LogContext): void {
    this.logger.info(message, context);
  }

  debug(message: string, context: LogContext): void {
    this.logger.debug(message, context);
  }

  trace(message: string, context: LogContext): void {
    this.logger.verbose(message, context);
  }

  // Método para criar child logger com contexto adicional
  child(additionalContext: Partial<LogContext>): {
    error: (message: string, error?: Error) => void;
    warn: (message: string) => void;
    info: (message: string) => void;
    debug: (message: string) => void;
    trace: (message: string) => void;
  } {
    const baseContext = additionalContext;
    
    return {
      error: (message: string, error?: Error) => 
        this.error(message, error, { ...baseContext } as LogContext),
      warn: (message: string) => 
        this.warn(message, { ...baseContext } as LogContext),
      info: (message: string) => 
        this.info(message, { ...baseContext } as LogContext),
      debug: (message: string) => 
        this.debug(message, { ...baseContext } as LogContext),
      trace: (message: string) => 
        this.trace(message, { ...baseContext } as LogContext),
    };
  }
}

// Factory para injeção de dependência
export const createLogger = (module: string) => {
  const logger = Logger.getInstance();
  
  return {
    error: (operation: string, message: string, error?: Error, additionalContext?: Record<string, any>) => {
      logger.error(message, error, { module, operation, ...additionalContext });
    },
    warn: (operation: string, message: string, additionalContext?: Record<string, any>) => {
      logger.warn(message, { module, operation, ...additionalContext });
    },
    info: (operation: string, message: string, additionalContext?: Record<string, any>) => {
      logger.info(message, { module, operation, ...additionalContext });
    },
    debug: (operation: string, message: string, additionalContext?: Record<string, any>) => {
      logger.debug(message, { module, operation, ...additionalContext });
    },
    trace: (operation: string, message: string, additionalContext?: Record<string, any>) => {
      logger.trace(message, { module, operation, ...additionalContext });
    },
    child: (childModule: string) => createLogger(`${module}.${childModule}`),
  };
};

export type AppLogger = ReturnType<typeof createLogger>;

import { Request, Response, NextFunction } from 'express';
import { v4 as uuidv4 } from 'uuid';
import { createLogger } from './logger';

const logger = createLogger('http');

export const loggingMiddleware = (req: Request, res: Response, next: NextFunction): void => {
  const correlationId = req.headers['x-correlation-id'] as string || uuidv4();
  const startTime = Date.now();

  // Adiciona correlationId ao request
  req.correlationId = correlationId;

  // Log da requisição recebida
  logger.info('request_received', `[${req.method}] ${req.path}`, {
    correlationId,
    method: req.method,
    path: req.path,
    ip: req.ip,
    userAgent: req.get('user-agent'),
  });

  // Intercepta response para log
  const originalSend = res.send;
  res.send = function(body: any): Response {
    const duration = Date.now() - startTime;
    
    logger.info('response_sent', `[${req.method}] ${req.path} - ${res.statusCode} (${duration}ms)`, {
      correlationId,
      method: req.method,
      path: req.path,
      statusCode: res.statusCode,
      duration,
      responseSize: typeof body === 'string' ? body.length : JSON.stringify(body).length,
    });

    return originalSend.call(this, body);
  };

  next();
};

import { Request, Response, NextFunction } from 'express';
import { createLogger } from './logger';

const logger = createLogger('error-handler');

export class AppError extends Error {
  constructor(
    public message: string,
    public statusCode: number = 500,
    public code?: string,
    public isOperational: boolean = true
  ) {
    super(message);
    this.name = this.constructor.name;
    Error.captureStackTrace(this, this.constructor);
  }
}

export const errorHandler = (
  error: Error | AppError,
  req?: Request,
  res?: Response,
  next?: NextFunction
): void | Response => {
  const correlationId = req?.correlationId || 'no-context';
  
  // Log do erro
  logger.error('unhandled_error', error.message, error, {
    correlationId,
    path: req?.path,
    method: req?.method,
    isOperational: error instanceof AppError ? error.isOperational : false,
    stack: error.stack,
  });

  // Se for uma resposta HTTP
  if (res && !res.headersSent) {
    const statusCode = error instanceof AppError ? error.statusCode : 500;
    const response = {
      error: {
        message: error.message,
        code: error instanceof AppError ? error.code : 'INTERNAL_ERROR',
        correlationId,
        timestamp: new Date().toISOString(),
      },
    };

    // Em produção, não expor stack trace
    if (process.env.NODE_ENV !== 'production' && error.stack) {
      response.error['stack'] = error.stack;
    }

    return res.status(statusCode).json(response);
  }

  // Se não for contexto HTTP, apenas log
  if (next) {
    next(error);
  }
};

// src/App.tsx
import React, { useState, useEffect } from 'react';
import { ThemeProvider } from './contexts/ThemeContext';
import Dashboard from './components/Dashboard/Dashboard';
import { fetchDashboardData, fetchUserProfile } from './services/api';
import { DashboardData, UserProfile } from './types';
import './styles/global.css';

const App: React.FC = () => {
  const [dashboardData, setDashboardData] = useState<DashboardData | null>(null);
  const [userProfile, setUserProfile] = useState<UserProfile | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  useEffect(() => {
    const loadData = async () => {
      try {
        setLoading(true);
        const [dashboardResponse, profileResponse] = await Promise.all([
          fetchDashboardData(),
          fetchUserProfile()
        ]);
        
        setDashboardData(dashboardResponse);
        setUserProfile(profileResponse);
        setError(null);
      } catch (err) {
        setError(err instanceof Error ? err.message : 'Failed to load dashboard data');
        console.error('Error loading dashboard data:', err);
      } finally {
        setLoading(false);
      }
    };

    loadData();
  }, []);

  return (
    <ThemeProvider>
      <div className="app" data-testid="app">
        {loading ? (
          <div className="loading-state" role="status" aria-label="Loading dashboard">
            <div className="loading-spinner"></div>
            <p className="loading-text">Loading dashboard...</p>
          </div>
        ) : error ? (
          <div className="error-state" role="alert" aria-label="Error loading dashboard">
            <div className="error-icon">⚠️</div>
            <h2 className="error-title">Unable to load dashboard</h2>
            <p className="error-message">{error}</p>
            <button 
              className="retry-button"
              onClick={() => window.location.reload()}
              aria-label="Retry loading dashboard"
            >
              Retry
            </button>
          </div>
        ) : dashboardData && userProfile ? (
          <Dashboard 
            data={dashboardData} 
            userProfile={userProfile}
          />
        ) : null}
      </div>
    </ThemeProvider>
  );
};

export default App;

export const useTheme = (): ThemeContextType => {
  const context = useContext(ThemeContext);
  if (!context) {
    throw new Error('useTheme must be used within a ThemeProvider');
  }
  return context;
};

interface ThemeProviderProps {
  children: ReactNode;
}

export const ThemeProvider: React.FC<ThemeProviderProps> = ({ children }) => {
  const [themeMode, setThemeMode] = useState<ThemeMode>(() => {
    const savedTheme = localStorage.getItem('theme-mode');
    return (savedTheme as ThemeMode) || 'light';
  });

  const theme = themeMode === 'dark' ? darkTheme : defaultTheme;

  useEffect(() => {
    localStorage.setItem('theme-mode', themeMode);
    document.documentElement.setAttribute('data-theme', themeMode);
  }, [themeMode]);

  const toggleTheme = () => {
    setThemeMode(prev => prev === 'light' ? 'dark' : 'light');
  };

  const value: ThemeContextType = {
    theme,
    themeMode,
    toggleTheme
  };

  return (
    <ThemeContext.Provider value={value}>
      {children}
    </ThemeContext.Provider>
  );
};

"""
Serviço de gerenciamento de logs

import json
import logging
import sqlite3
from datetime import datetime, timedelta
from typing import List, Optional, Dict, Any
from contextlib import contextmanager
from pathlib import Path

from .log_model import LogEntry, LogQueryParams, SystemMetrics, LogLevel, LogSource


class LogService:
    """Serviço para gerenciamento de logs e métricas"""
    
    def __init__(self, db_path: str = "logs.db"):
        self.db_path = db_path
        self._init_database()
        self._setup_logging()
    
    def _init_database(self):
        """Inicializa o banco de dados SQLite para logs"""
        with self._get_connection() as conn:
            cursor = conn.cursor()
            
            # Tabela de logs
            cursor.execute('''
                CREATE TABLE IF NOT EXISTS logs (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    timestamp DATETIME NOT NULL,
                    level TEXT NOT NULL,
                    source TEXT NOT NULL,
                    message TEXT NOT NULL,
                    correlation_id TEXT,
                    user_id TEXT,
                    endpoint TEXT,
                    duration_ms INTEGER,
                    status_code INTEGER,
                    metadata TEXT,
                    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
                )
            ''')
            
            # Tabela de métricas do sistema
            cursor.execute('''
                CREATE TABLE IF NOT EXISTS system_metrics (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    timestamp DATETIME NOT NULL,
                    cpu_percent REAL NOT NULL,
                    memory_percent REAL NOT NULL,
                    memory_used_mb REAL NOT NULL,
                    memory_total_mb REAL NOT NULL,
                    disk_usage_percent REAL NOT NULL,
                    disk_used_gb REAL NOT NULL,
                    disk_total_gb REAL NOT NULL,
                    active_connections INTEGER NOT NULL,
                    request_count INTEGER NOT NULL,
                    error_rate REAL NOT NULL,
                    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
                )
            ''')
            
            # Índices para performance
            cursor.execute('CREATE INDEX IF NOT EXISTS idx_logs_timestamp ON logs(timestamp)')
            cursor.execute('CREATE INDEX IF NOT EXISTS idx_logs_level ON logs(level)')
            cursor.execute('CREATE INDEX IF NOT EXISTS idx_logs_source ON logs(source)')
            cursor.execute('CREATE INDEX IF NOT EXISTS idx_logs_correlation ON logs(correlation_id)')
            cursor.execute('CREATE INDEX IF NOT EXISTS idx_metrics_timestamp ON system_metrics(timestamp)')
            
            conn.commit()
    
    @contextmanager
    def _get_connection(self):
        """Context manager para conexões com banco de dados"""
        conn = sqlite3.connect(self.db_path)
        conn.row_factory = sqlite3.Row
        try:
            yield conn
        finally:
            conn.close()
    
    def _setup_logging(self):
        """Configura logging estruturado"""
        logging.basicConfig(
            level=logging.INFO,
            format='{"timestamp": "%(asctime)s", "level": "%(levelname)s", "message": "%(message)s"}',
            datefmt='%Y-%m-%dT%H:%M:%S'
        )
        self.logger = logging.getLogger(__name__)
    
    def create_log(self, log_entry: LogEntry) -> int:
        """
        Cria uma nova entrada de log no banco de dados
        
        Args:
            log_entry: Entrada de log estruturada
            
        Returns:
            ID do log criado
        """
        try:
            with self._get_connection() as conn:
                cursor = conn.cursor()
                
                cursor.execute('''
                    INSERT INTO logs (
                        timestamp, level, source, message, correlation_id,
                        user_id, endpoint, duration_ms, status_code, metadata
                    ) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
                ''', (

                    log_entry.message,
                    log_entry.correlation_id,
                    log_entry.user_id,
                    log_entry.endpoint,
                    log_entry.duration_ms,
                    log_entry.status_code,
                    json.dumps(log_entry.metadata) if log_entry.metadata else None
                ))
                
                log_id = cursor.lastrowid
                conn.commit()

                        "log_id": log_id,
                        "correlation_id": log_entry.correlation_id,
                        "endpoint": log_entry.endpoint
                    }

    def get_logs(self, query_params: LogQueryParams) -> List[Dict[str, Any]]:
        """
        Consulta logs com base nos parâmetros fornecidos
        
        Args:
            query_params: Parâmetros de consulta
            
        Returns:
            Lista de logs no formato de dicionário
        """
        try:
            query = "SELECT * FROM logs WHERE 1=1"
            params = []
            
            # Filtros dinâmicos
            if query_params.level:
                query += " AND level = ?"
                params.append(query_params.level.value)
            
            if query_params.source:
                query += " AND source = ?"
                params.append(query_params.source.value)
            
            if query_params.start_date:
                query += " AND timestamp >= ?"
                params.append(query_params.start_date.isoformat())
            
            if query_params.end_date:
                query += " AND timestamp <= ?"
                params.append(query_params.end_date.isoformat())
            
            if query_params.correlation_id:
                query += " AND correlation_id = ?"
                params.append(query_params.correlation_id)
            
            if query_params.user_id:
                query += " AND user_id = ?"
                params.append(query_params.user_id)
            
            if query_params.endpoint:
                query += " AND endpoint LIKE ?"
                params.append(f"%{query_params.endpoint}%")
            
            # Ordenação e paginação
            query += " ORDER BY timestamp DESC LIMIT ? OFFSET ?"
            params.extend([query_params.limit, query_params.offset])
            
            with self._get_connection() as conn:
                cursor = conn.cursor()
                cursor.execute(query, params)
                rows = cursor.fetchall()
                
                logs = []
                for row in rows:
                    log = dict(row)
                    # Parse metadata JSON
                    if log['metadata']:
                        log['metadata'] = json.loads(log['metadata'])
                    logs.append(log)
                
                return logs

    def get_system_metrics(self, hours: int = 24) -> List[Dict[str, Any]]:
        """
        Obtém métricas do sistema das últimas N horas
        
        Args:
            hours: Número de horas para buscar (padrão: 24)
            
        Returns:
            Lista de métricas do sistema
        """
        try:
            cutoff_time = datetime.utcnow() - timedelta(hours=hours)
            
            with self._get_connection() as conn:
                cursor = conn.cursor()
                cursor.execute('''
                    SELECT * FROM system_metrics 
                    WHERE timestamp >= ? 
                    ORDER BY timestamp DESC
                ''', (cutoff_time.isoformat(),))
                
                rows = cursor.fetchall()
                return [dict(row) for row in rows]
                
        except Exception as e:
            self.logger.error(f"Failed to get system metrics: {str(e)}")
            raise
    
    def get_error_stats(self, hours: int = 24) -> Dict[str, Any]:
        """
        Obtém estatísticas de erros das últimas N horas
        
        Args:
            hours: Número de horas para analisar
            
        Returns:
            Estatísticas de erros
        """
        try:
            cutoff_time = datetime.utcnow() - timedelta(hours=hours)
            
            with self._get_connection() as conn:
                cursor = conn.cursor()
                
                # Contagem de erros por nível
                cursor.execute('''
                    SELECT level, COUNT(*) as count 
                    FROM logs 
                    WHERE timestamp >= ? AND level IN ('error', 'critical')
                    GROUP BY level
                ''', (cutoff_time.isoformat(),))
                
                error_counts = {row['level']: row['count'] for row in cursor.fetchall()}
                
                # Erros por endpoint
                cursor.execute('''
                    SELECT endpoint, COUNT(*) as count 
                    FROM logs 
                    WHERE timestamp >= ? AND level IN ('error', 'critical') AND endpoint IS NOT NULL
                    GROUP BY endpoint 
                    ORDER BY count DESC 
                    LIMIT 10
                ''', (cutoff_time.isoformat(),))

                # Taxa de erro total
                cursor.execute('''
                    SELECT 
                        COUNT(CASE WHEN level IN ('error', 'critical') THEN 1 END) as error_count,
                        COUNT(*) as total_count
                    FROM logs 
                    WHERE timestamp >= ?
                ''', (cutoff_time.isoformat(),))

                error_rate = (stats_row['error_count'] / stats_row['total_count'] * 100) if stats_row['total_count'] > 0 else 0
                
                return {

                    "top_error_endpoints": top_error_endpoints,
                    "error_rate_percent": round(error_rate, 2),

                }

            raise

  systemMemoryUsage.set({ type: 'total' }, totalMem);
  systemMemoryUsage.set({ type: 'used' }, usedMem);
  systemMemoryUsage.set({ type: 'free' }, freeMem);
  
  // CPU
  const cpus = os.cpus();
  let totalIdle = 0, totalTick = 0;
  
  cpus.forEach(cpu => {
    for (let type in cpu.times) {
      totalTick += cpu.times[type];
    }
    totalIdle += cpu.times.idle;
  });
  
  const idle = totalIdle / cpus.length;
  const total = totalTick / cpus.length;
  const usage = 100 - (100 * idle / total);
  
  systemCpuUsage.set(usage);
  
  // Uptime
  uptime.set((Date.now() - startTime) / 1000);
}
