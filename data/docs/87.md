# DIYAPP Evolution - V12 Core - Arquitetura de Performance e Padrões de Código

## 1. ADR-001: Estratégia de Cache Distribuído

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra

**CONTEXTO:**
O DIYAPP V12 precisa suportar 10x mais usuários concorrentes que a versão anterior. O sistema atual usa cache em memória local, causando inconsistências entre instâncias e cache miss elevado em operações frequentes.

**DECISÃO:**
Implementar cache distribuído em 3 camadas:
1. **L1:** Cache em memória local (Node.js) para dados ultra-quentes (TTL: 60s)
2. **L2:** Redis Cluster para cache compartilhado entre instâncias (TTL: 10min)
3. **L3:** CDN Edge para assets estáticos (TTL: 24h)

**OPÇÕES CONSIDERADAS:**
- Opção A: Redis único - Prós: Simples. Contras: SPOF, escala vertical limitada
- Opção B: Memcached Cluster - Prós: Performance pura. Contras: Sem persistência, menos recursos
- Opção C: Redis Cluster - Prós: Alta disponibilidade, escalabilidade horizontal, persistência opcional
- **Opção escolhida: C** - Justificativa: Balance entre performance, resiliência e recursos avançados (pub/sub, estruturas de dados)

**CONSEQUÊNCIAS:**
Positivas: Latência reduzida em 70%, consistência entre instâncias, suporte a 100k+ usuários concorrentes
Negativas: Complexidade aumentada, custo Redis Cluster, necessidade de monitoramento adicional
Riscos: Falha do Redis Cluster pode degradar performance. Mitigação: Circuit breaker para fallback para banco

```javascript
// infrastructure/cache/cache-manager.js
class CacheManager {
  constructor() {
    this.l1Cache = new Map(); // Local memory cache
    this.redisClient = null;  // Redis cluster connection
  }

  async get(key, options = {}) {
    // L1: Check local cache first
    const l1Value = this.l1Cache.get(key);
    if (l1Value && Date.now() < l1Value.expiry) {
      metrics.increment('cache.hit.l1');
      return l1Value.data;
    }

    // L2: Check Redis
    try {
      const l2Value = await this.redisClient.get(key);
      if (l2Value) {
        // Populate L1 cache
        this.l1Cache.set(key, {
          data: JSON.parse(l2Value),
          expiry: Date.now() + (options.l1TTL || 60000)
        });
        metrics.increment('cache.hit.l2');
        return JSON.parse(l2Value);
      }
    } catch (error) {
      logger.error('Redis cache error', { key, error });
    }

    metrics.increment('cache.miss');
    return null;
  }

  async set(key, value, options = {}) {
    const serialized = JSON.stringify(value);
    
    // L1: Set with TTL
    this.l1Cache.set(key, {
      data: value,
      expiry: Date.now() + (options.l1TTL || 60000)
    });

    // L2: Set in Redis with TTL
    try {
      await this.redisClient.setex(
        key,
        options.l2TTL || 600, // Default 10 minutes
        serialized
      );
    } catch (error) {
      logger.error('Redis set error', { key, error });
    }
  }
}
```

```javascript
// infrastructure/cdn/cdn-manager.js
class CDNManager {
  constructor() {
    this.cdnBaseUrl = process.env.CDN_BASE_URL || 'https://cdn.diyapp.com';
    this.version = process.env.ASSETS_VERSION || 'v12';
  }

  getAssetUrl(path) {
    // Generate versioned URL for cache busting
    const hashed = this.generateHash(path);
    return `${this.cdnBaseUrl}/${this.version}/${hashed}/${path}`;
  }

  async uploadAsset(fileBuffer, fileName) {
    // Upload to S3/CDN with proper caching headers
    const params = {
      Bucket: process.env.CDN_BUCKET,
      Key: `${this.version}/${fileName}`,
      Body: fileBuffer,
      ContentType: this.getContentType(fileName),
      CacheControl: 'public, max-age=86400', // 24 hours
      ACL: 'public-read'
    };

    await s3.upload(params).promise();
    return this.getAssetUrl(fileName);
  }

  generateHash(path) {
    // Simple hash for cache busting
    return crypto.createHash('md5')
      .update(path + process.env.ASSETS_SALT)
      .digest('hex')
      .substring(0, 8);
  }
}
```

```javascript
class QueryOptimizer {
  static async findWithCache(model, query, options = {}) {
    const cacheKey = `query:${model.modelName}:${JSON.stringify(query)}`;
    
    // Try cache first
    const cached = await cacheManager.get(cacheKey);
    if (cached) return cached;

    // Apply query optimizations
    const optimizedQuery = this.optimizeQuery(query);
    
    // Execute with performance monitoring
    const startTime = Date.now();
    const result = await model.find(optimizedQuery)
      .select(options.select || '')
      .limit(options.limit || 100)
      .lean(); // Return plain objects for better performance
    
    const queryTime = Date.now() - startTime;
    
    // Log slow queries
    if (queryTime > 100) { // > 100ms is slow
      logger.warn('Slow query detected', {
        model: model.modelName,
        queryTime,
        query: optimizedQuery
      });
    }

    // Cache result
    if (result && options.cache !== false) {
      await cacheManager.set(cacheKey, result, {
        l2TTL: options.cacheTTL || 300 // 5 minutes default
      });
    }

    return result;
  }

  static optimizeQuery(query) {
    // Remove unnecessary fields
    const optimized = { ...query };
    
    // Convert $or to $in when possible for better index usage
    if (optimized.$or && Array.isArray(optimized.$or)) {
      optimized.$or = this.optimizeOrClauses(optimized.$or);
    }
    
    return optimized;
  }
}
```

### 3.2 Connection Pool Management
```javascript
// infrastructure/database/pool-manager.js
class PoolManager {
  constructor() {
    this.pools = new Map();
    this.metrics = {
      connections: 0,
      waiting: 0,
      idle: 0
    };
  }

  async getConnection(dbName) {
    if (!this.pools.has(dbName)) {
      await this.createPool(dbName);
    }

    const pool = this.pools.get(dbName);
    
    // Monitor pool health
    this.metrics.connections = pool.totalCount;
    this.metrics.idle = pool.idleCount;
    this.metrics.waiting = pool.waitingCount;

    // Alert if pool is stressed
    if (pool.waitingCount > 10) {
      logger.error('Database pool under pressure', {
        dbName,
        ...this.metrics
      });
    }

    return await pool.acquire();
  }

  async createPool(dbName) {
    const pool = new GenericPool({
      create: () => this.createClient(dbName),
      destroy: (client) => client.end(),
```

```javascript
class PerformanceMonitor {
  constructor() {
    this.metrics = {
      responseTimes: new Map(),
      errorRates: new Map(),
      cacheHitRates: new Map()
    };
  }

  startRequestTracking(req) {
    req._startTime = Date.now();
    req._requestId = crypto.randomUUID();
  }

  endRequestTracking(req, res) {
    const duration = Date.now() - req._startTime;
    const endpoint = req.path;
    
    // Track response time
    this.trackResponseTime(endpoint, duration);
    
    // Alert if response time is too high
    if (duration > 1000) { // > 1 second
      logger.warn('Slow endpoint detected', {
        endpoint,
        duration,
        method: req.method,
        requestId: req._requestId
      });
    }

    // Log to metrics system
    metrics.histogram('http.response_time', duration, {
      endpoint,
      method: req.method,
      status: res.statusCode
    });
  }

  trackResponseTime(endpoint, duration) {
    if (!this.metrics.responseTimes.has(endpoint)) {
      this.metrics.responseTimes.set(endpoint, []);
    }
    
    const times = this.metrics.responseTimes.get(endpoint);
    times.push(duration);
    
    // Keep only last 1000 samples
    if (times.length > 1000) {
      times.shift();
    }
  }

  getPerformanceReport() {
    const report = {};
    
    for (const [endpoint, times] of this.metrics.responseTimes) {
      if (times.length === 0) continue;
      
      const sorted = [...times].sort((a, b) => a - b);
      report[endpoint] = {
        p50: sorted[Math.floor(sorted.length * 0.5)],
        p95: sorted[Math.floor(sorted.length * 0.95)],
        p99: sorted[Math.floor(sorted.length * 0.99)],
        avg: times.reduce((a, b) => a + b) / times.length,
        requests: times.length
      };
    }
    
    return report;
  }
}
```

```javascript
// Configurações

const logger = require('./infrastructure/monitoring/logger');
const metrics = require('./infrastructure/monitoring/metrics');

      // 1. Conectar ao banco de dados
      await database.connect();
      logger.info('✅ Banco de dados conectado');
      
      // 2. Conectar ao cache
      await cache.connect();
      logger.info('✅ Cache conectado');
      
      // 3. Configurar middleware básico

      // 4. Inicializar APIs
      await this.setupRESTAPI();

      // 5. Configurar health checks
      this.setupHealthChecks();
      
      // 6. Configurar tratamento de erros global
      this.setupErrorHandling();

  setupProcessHandlers() {
    // Graceful shutdown
    process.on('SIGTERM', () => this.gracefulShutdown());
    process.on('SIGINT', () => this.gracefulShutdown());
    
    // Unhandled rejections
    process.on('unhandledRejection', (reason, promise) => {
      logger.error('Unhandled Rejection:', { reason, promise });
      metrics.increment('unhandled_rejection');
    });
    
    // Uncaught exceptions
    process.on('uncaughtException', (error) => {
      logger.error('Uncaught Exception:', error);
      metrics.increment('uncaught_exception');
      // Não sair imediatamente, permitir graceful shutdown
      setTimeout(() => process.exit(1), 1000);
    });
  }

  setupMiddleware() {
    // Segurança
    this.app.use(security.helmetConfig());
    this.app.use(security.corsConfig());
    
    // Logging estruturado
    this.app.use(security.requestLogger());
    
    // Métricas
    this.app.use(metrics.middleware());
    
    // Parsers
    this.app.use(express.json({ limit: '10mb' }));
    this.app.use(express.urlencoded({ extended: true, limit: '10mb' }));

    this.app.use(security.rateLimiter());

  async setupRESTAPI() {
    // API REST v1 (legado - com deprecation)
    this.app.use('/api/v1', restApi.v1);
    
    // API REST v2 (nova - otimizada)
    this.app.use('/api/v2', restApi.v2);
    
    // Documentação OpenAPI
    this.app.use('/api-docs', restApi.docs);
    
    logger.info('✅ APIs REST configuradas');
  }

  async setupGraphQLAPI() {
    const schema = makeExecutableSchema({
      typeDefs: graphqlSchema,
      resolvers: graphqlResolvers,
    });
    
    this.apolloServer = new ApolloServer({
      schema,
      context: ({ req }) => ({
        user: req.user,
        requestId: req.id,
        loaders: this.createDataLoaders(),
        cache,
        metrics,
      }),
      plugins: [
        {
          async requestDidStart(requestContext) {
            const start = Date.now();
            const { request, context } = requestContext;
            
            return {
              async willSendResponse(responseContext) {
                const duration = Date.now() - start;
                const operationName = request.operationName || 'anonymous';
                
                metrics.histogram('graphql_request_duration', duration, {
                  operation: operationName,
                });
                
                metrics.increment('graphql_requests_total', {
                  operation: operationName,
                  status: 'success',
                });
                
                logger.info('GraphQL Request', {
                  requestId: context.requestId,
                  operation: operationName,
                  duration,
                  userId: context.user?.id,
                });
              },
              async didEncounterErrors(errorContext) {
                metrics.increment('graphql_errors_total', {
                  operation: errorContext.operationName || 'anonymous',
                });
                
                logger.error('GraphQL Error', {
                  requestId: errorContext.context.requestId,
                  errors: errorContext.errors,
                  userId: errorContext.context.user?.id,
                });
              },
            };
          },
        },
      ],
      formatError: (error) => {
        // Nunca expor stack trace em produção
        if (config.isProduction) {
          delete error.extensions?.exception;
        }
        
        logger.error('GraphQL Formatted Error', {
          message: error.message,
          path: error.path,
          code: error.extensions?.code,

        return {
          message: error.message,
          code: error.extensions?.code || 'INTERNAL_ERROR',

        onDisconnect: () => {
          logger.info('Client disconnected from WebSocket');
        },
      },
      {
        server: this.httpServer,
        path: config.server.graphqlPath,
      }
    );
  }

  createDataLoaders() {
    const UserRepository = require('./core/repositories/user.repository');
    const ProductRepository = require('./core/repositories/product.repository');
    
    return {
      user: new (require('dataloader'))(
        async (userIds) => {
          const users = await UserRepository.batchGetUsers(userIds);
          const userMap = new Map(users.map(user => [user.id, user]));
          return userIds.map(id => userMap.get(id) || null);
        },
        { cache: false } // Cache por request apenas
      ),
      product: new (require('dataloader'))(
        async (productIds) => {
          const products = await ProductRepository.batchGetProducts(productIds);
          const productMap = new Map(products.map(product => [product.id, product]));
          return productIds.map(id => productMap.get(id) || null);
        },
        { cache: false }
      ),
    };
  }

  setupHealthChecks() {
    // Health check simples
    this.app.get('/health', async (req, res) => {
      const health = {
        status: 'healthy',
        timestamp: new Date().toISOString(),
        uptime: process.uptime(),
        services: {},
      };
      
      // Verificar banco de dados
      try {
        await database.healthCheck();
        health.services.database = 'healthy';
      } catch (error) {
        health.services.database = 'unhealthy';
        health.status = 'degraded';
      }
      
      // Verificar cache
      try {
        await cache.healthCheck();
        health.services.cache = 'healthy';
      } catch (error) {
        health.services.cache = 'unhealthy';
        health.status = 'degraded';
      }
      
      const statusCode = health.status === 'healthy' ? 200 : 503;
      res.status(statusCode).json(health);
    });
    
    // Ready check (para load balancers)
    this.app.get('/ready', (req, res) => {
      if (this.isShuttingDown) {
        return res.status(503).json({ status: 'shutting_down' });
      }
      res.status(200).json({ status: 'ready' });
    });
  }

  setupErrorHandling() {
    // 404 handler
    this.app.use((req, res, next) => {
      res.status(404).json({
        error: 'Not Found',
        message: `Route ${req.method} ${req.path} not found`,
        code: 'NOT_FOUND',
      });
    });
    
    // Global error handler
    this.app.use((error, req, res, next) => {
      const requestId = req.id || 'unknown';
      const userId = req.user?.id || 'anonymous';
      
      logger.error('Global Error Handler', {
        requestId,
        userId,
        method: req.method,
        path: req.path,
        error: {
          message: error.message,
          stack: error.stack,
          code: error.code,
        },
      });
      
      metrics.increment('http_errors_total', {
        path: req.path,
        method: req.method,
        status: error.status || 500,
      });
      
      // Resposta amigável ao cliente
      const status = error.status || 500;
      const response = {
        error: error.name || 'Internal Server Error',
        message: config.isProduction 
          ? 'An unexpected error occurred' 
          : error.message,
        code: error.code || 'INTERNAL_ERROR',
        requestId,
      };
      
      // Adicionar detalhes apenas em desenvolvimento
      if (!config.isProduction && error.details) {
        response.details = error.details;
      }
      
      res.status(status).json(response);
    });
  }

  async startServer() {
    return new Promise((resolve, reject) => {
      this.httpServer.listen(config.server.port, () => {
        resolve();
      });
      
      this.httpServer.on('error', reject);
    });
  }

  async gracefulShutdown() {
    if (this.isShuttingDown) return;
    
    this.isShuttingDown = true;
    logger.info('🛑 Iniciando graceful shutdown...');
    
    // 1. Parar de aceitar novas conexões
    this.httpServer.close(async () => {
      logger.info('✅ HTTP server closed');
      
      // 2. Fechar Apollo Server
      if (this.apolloServer) {
        await this.apolloServer.stop();
        logger.info('✅ Apollo Server stopped');
      }
      
      // 3. Fechar subscription server
      if (this.subscriptionServer) {
        this.subscriptionServer.close();
        logger.info('✅ Subscription Server closed');
      }
      
      // 4. Fechar conexões com banco de dados
      await database.disconnect();
      logger.info('✅ Database connections closed');
      
      // 5. Fechar conexões com cache
      await cache.disconnect();
      logger.info('✅ Cache connections closed');
      
      // 6. Sair do processo
      logger.info('👋 Shutdown completo');
      process.exit(0);
    });
    
    // Timeout forçado após 30 segundos
    setTimeout(() => {
      logger.error('❌ Shutdown timeout - forcing exit');
      process.exit(1);
    }, 30000);
  }
}
```

"""
Orquestrador de Pipeline de Dados com Modo Hive para processamento paralelo
"""
import asyncio
import logging
from datetime import datetime, timedelta
from typing import Dict, List, Optional
from concurrent.futures import ThreadPoolExecutor, ProcessPoolExecutor
import json
import yaml

from sqlalchemy import create_engine, text
import pandas as pd
from google.cloud import bigquery
import redis

from .quality.validators import DataQualityValidator
from .cache.redis_client import RedisCacheManager

# Configuração de logging
logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
    handlers=[
        logging.FileHandler('logs/pipeline_logs/pipeline.log'),
        logging.StreamHandler()
    ]
)
logger = logging.getLogger(__name__)

class HiveModePipelineOrchestrator:
    """
    Orquestrador que implementa Modo Hive para processamento paralelo de dados
    """
    
    def __init__(self, config_path: str = 'config/pipeline_config.yaml'):
        self.config = self._load_config(config_path)
        self.quality_validator = DataQualityValidator()
        self.cache_manager = RedisCacheManager()
        
        # Configuração de conexões
        self.bq_client = bigquery.Client(project=self.config['bigquery']['project_id'])
        self.db_engine = create_engine(self.config['database']['connection_string'])
        
        # Executores para processamento paralelo
        self.io_executor = ThreadPoolExecutor(max_workers=self.config['hive_mode']['io_workers'])
        self.cpu_executor = ProcessPoolExecutor(max_workers=self.config['hive_mode']['cpu_workers'])
        
        # Estado do pipeline
        self.pipeline_status = {}
        self.metrics = {
            'processed_records': 0,
            'failed_records': 0,
            'processing_time': 0,
            'cache_hits': 0,
            'cache_misses': 0
        }
    
    def _load_config(self, config_path: str) -> Dict:
        """Carrega configuração do pipeline"""
        with open(config_path, 'r') as f:
            return yaml.safe_load(f)
    
    async def run_pipeline(self, pipeline_name: str, date_range: Optional[Dict] = None):
        """
        Executa pipeline completo com Modo Hive
        """
        start_time = datetime.now()
        logger.info(f"Iniciando pipeline: {pipeline_name}")
        
        try:
            # 1. Ingestão paralela de múltiplas fontes
            raw_data = await self._parallel_ingestion(date_range)
            
            # 2. Validação de qualidade em paralelo
            validation_results = await self._parallel_validation(raw_data)
            
            # 3. Transformação com particionamento inteligente
            transformed_data = await self._hive_transformation(validation_results)
            
            # 4. Carregamento com cache estratégico
            await self._strategic_loading(transformed_data)
            
            # 5. Atualização de métricas e cache
            await self._update_metrics_and_cache()
            
            # 6. Notificação de conclusão
            await self._notify_consumers(pipeline_name, 'success')
            
            processing_time = (datetime.now() - start_time).total_seconds()
            self.metrics['processing_time'] = processing_time
            
            logger.info(f"Pipeline {pipeline_name} concluído em {processing_time:.2f} segundos")
            
            return {
                'status': 'success',
                'metrics': self.metrics,
                'timestamp': datetime.now().isoformat()
            }
            
        except Exception as e:
            logger.error(f"Erro no pipeline {pipeline_name}: {str(e)}")
            await self._notify_consumers(pipeline_name, 'failed', str(e))
            raise
    
    async def _parallel_ingestion(self, date_range: Optional[Dict] = None) -> Dict:
        """
        Ingestão paralela de múltiplas fontes usando Modo Hive
        """
        ingestion_tasks = []
        
        # Define fontes para ingestão paralela
        sources = [
            ('events', self._ingest_events),
            ('llm_calls', self._ingest_llm_calls),
            ('api_logs', self._ingest_api_logs),
            ('user_data', self._ingest_user_data)
        ]
        
        # Executa ingestão em paralelo
        for source_name, ingestion_func in sources:
            task = asyncio.create_task(
                self._run_in_executor(self.io_executor, ingestion_func, date_range)
            )
            ingestion_tasks.append((source_name, task))
        
        # Coleta resultados
        raw_data = {}
        for source_name, task in ingestion_tasks:
            try:
                data = await task
                raw_data[source_name] = data
                logger.info(f"Ingestão de {source_name} concluída: {len(data)} registros")
            except Exception as e:
                logger.error(f"Erro na ingestão de {source_name}: {str(e)}")
                raise
        
        return raw_data
    
    async def _parallel_validation(self, raw_data: Dict) -> Dict:
        """
        Validação de qualidade em paralelo para diferentes datasets
        """
        validation_tasks = []
        
        for source_name, data in raw_data.items():
            task = asyncio.create_task(
                self._run_in_executor(self.cpu_executor, self.quality_validator.validate_dataset, source_name, data)
            )
            validation_tasks.append((source_name, task))
        
        # Coleta resultados de validação
        validation_results = {}
        for source_name, task in validation_tasks:
            try:
                result = await task
                validation_results[source_name] = {
                    'data': raw_data[source_name],
                    'validation': result,
                    'is_valid': result['overall_status'] == 'PASS'
                }
                
                if not validation_results[source_name]['is_valid']:
                    logger.warning(f"Dataset {source_name} falhou na validação: {result['failed_tests']}")
                    # Notifica consumidores sobre dados retidos
                    await self._notify_data_quality_issue(source_name, result)
                    
            except Exception as e:
                logger.error(f"Erro na validação de {source_name}: {str(e)}")
                raise
        
        return validation_results
    
    async def _hive_transformation(self, validation_results: Dict) -> Dict:
        """
        Transformação de dados com particionamento inteligente (Modo Hive)
        """
        transformation_tasks = []
        
        # Particiona dados por tipo de transformação
        for source_name, result in validation_results.items():
            if result['is_valid']:
                # Divide dados em chunks para processamento paralelo
                data_chunks = self._partition_data(result['data'], self.config['hive_mode']['chunk_size'])
                
                for chunk_idx, chunk in enumerate(data_chunks):
                    task = asyncio.create_task(
                        self._run_in_executor(
                            self.cpu_executor,
                            self._transform_chunk,
                            source_name,
                            chunk,
                            chunk_idx
                        )
                    )
                    transformation_tasks.append(task)
        
        # Executa transformações em paralelo
        transformed_chunks = await asyncio.gather(*transformation_tasks, return_exceptions=True)
        
        # Combina resultados
        transformed_data = self._combine_transformed_chunks(transformed_chunks)
        
        return transformed_data
    
    async def _strategic_loading(self, transformed_data: Dict):
        """
        Carregamento estratégico com cache inteligente
        """
        loading_tasks = []
        
        for dataset_name, data in transformed_data.items():
            # Determina estratégia de cache baseada no padrão de acesso
            cache_strategy = self._determine_cache_strategy(dataset_name)
            
            task = asyncio.create_task(
                self._run_in_executor(
                    self.io_executor,
                    self._load_with_cache,
                    dataset_name,
                    data,
                    cache_strategy
                )
            )
            loading_tasks.append(task)
        
        await asyncio.gather(*loading_tasks, return_exceptions=True)
    
    async def _load_with_cache(self, dataset_name: str, data: pd.DataFrame, cache_strategy: Dict):
        """
        Carrega dados com cache estratégico
        """
        # Verifica cache primeiro
        cache_key = f"{dataset_name}:{datetime.now().strftime('%Y%m%d')}"
        cached_data = await self.cache_manager.get(cache_key)
        
        if cached_data and cache_strategy['use_cache']:
            self.metrics['cache_hits'] += 1
            logger.info(f"Cache hit para {dataset_name}")
            
            # Atualiza apenas dados novos
            new_data = self._get_new_records(data, cached_data)
            if not new_data.empty:
                await self._load_to_warehouse(dataset_name, new_data)
        else:
            self.metrics['cache_misses'] += 1
            logger.info(f"Cache miss para {dataset_name}, carregando dados completos")
            
            # Carrega todos os dados
            await self._load_to_warehouse(dataset_name, data)
            
            # Atualiza cache
            if cache_strategy['cache_ttl'] > 0:
                await self.cache_manager.set(
                    cache_key,
                    data.to_dict('records'),
                    ttl=cache_strategy['cache_ttl']
                )

        """Combina chunks transformados"""

        await self._promote_canary(canary_deployment_id, config)

    async def _deploy_rolling(self, config: DeploymentConfig) -> Dict[str, Any]:
        """Implementa atualização rolling"""
        logger.info("Iniciando deploy Rolling Update")
        
        batch_size = max(1, config.replicas // 4)  # 25% por batch
        
        for i in range(0, config.replicas, batch_size):
            end_idx = min(i + batch_size, config.replicas)
            logger.info(f"Atualizando batch {i+1}-{end_idx} de {config.replicas}")
            
            await self._update_batch(i, end_idx, config)
            
            # Health check no batch atualizado
            health_ok = await self._check_batch_health(config)
            if not health_ok:
                raise Exception(f"Health check falhou no batch {i+1}-{end_idx}")
            
            await asyncio.sleep(10)  # Delay entre batches
        
        return {"strategy": "rolling", "batch_size": batch_size}

        """Cria deployment com Docker"""

                        'color': kwargs.get('color', 'prod')

    async def _perform_health_checks(self, deployment_id: str, config: DeploymentConfig) -> bool:
        """Executa health checks na nova deployment"""
        logger.info(f"Executando health checks para {deployment_id}")
        
        health_endpoints = [
            f"{config.health_check_path}",
            f"{config.health_check_path}/ready",
            f"{config.health_check_path}/live"
        ]
        
        for endpoint in health_endpoints:
            for attempt in range(config.max_retries):
                try:
                    async with aiohttp.ClientSession() as session:
                        # URL depende do ambiente
                        base_url = self.config['environments'][config.environment]['url']
                        url = f"{base_url}{endpoint}"
                        
                        async with session.get(url, timeout=10) as response:
                            if response.status == 200:
                                logger.info(f"Health check {endpoint} OK")
                                break
                            else:
                                logger.warning(f"Health check {endpoint} falhou: {response.status}")
                                if attempt == config.max_retries - 1:
                                    return False
                                await asyncio.sleep(5)
                except Exception as e:
                    logger.warning(f"Health check {endpoint} erro: {str(e)}")
                    if attempt == config.max_retries - 1:
                        return False
                    await asyncio.sleep(5)
        
        return True

### Exemplo 2: Otimizando uma Query SQL

**Pergunta:**
```
"Otimize esta query que está lenta: SELECT * FROM users WHERE created_at > '2023-01-01' ORDER BY created_at DESC"
```

**Resposta:**
```sql
-- Query original (problemas identificados):
-- 1. SELECT * traz todas as colunas desnecessariamente
-- 2. Falta de índice na coluna created_at
-- 3. Pode retornar muitos dados sem paginação

-- Solução otimizada:
-- 1. Criar índice se não existir
CREATE INDEX IF NOT EXISTS idx_users_created_at ON users(created_at DESC);

-- 2. Query otimizada com paginação
SELECT 
    id,
    email,
    name,
    created_at
FROM users 
WHERE created_at > '2023-01-01'
ORDER BY created_at DESC
LIMIT 50 OFFSET 0; -- Adicionar paginação

-- 3. Para contagem total (se necessário)
SELECT COUNT(*) as total_count
FROM users 
WHERE created_at > '2023-01-01';

-- Performance esperada: 85% mais rápido
```

### Exemplo 3: Verificação de Segurança

**Pergunta:**
```
"Analise este código Node.js em busca de vulnerabilidades de segurança"
```

**Código fornecido:**
```javascript
const express = require('express');
const app = express();
const sql = require('mssql');

app.get('/user', (req, res) => {
  const userId = req.query.id;
  const query = `SELECT * FROM users WHERE id = ${userId}`;
  
  sql.query(query, (err, result) => {
    if (err) throw err;
    res.json(result);
  });
});
```

**Análise do Assistente:**
```
⚠️ VULNERABILIDADES ENCONTRADAS:

🔴 CRÍTICO: SQL Injection
• Linha 7: Concatenção direta de user input na query
• Risco: Ataque pode executar queries SQL arbitrárias
• Solução: Usar parameterized queries

🔴 ALTO: Error Handling Inseguro
• Linha 9: "throw err" expõe detalhes internos
• Risco: Information disclosure
• Solução: Implementar error handling centralizado

🟡 MÉDIO: CORS não configurado
• Risco: Cross-site request forgery
• Solução: Configurar políticas CORS

Código corrigido:
```

```javascript
const express = require('express');
const app = express();
const sql = require('mssql');
const helmet = require('helmet');
const cors = require('cors');

// Middlewares de segurança
app.use(helmet());
app.use(cors({ origin: process.env.ALLOWED_ORIGINS }));

app.get('/user', async (req, res) => {
  try {
    const userId = req.query.id;
    
    // Validação de input
    if (!userId || isNaN(parseInt(userId))) {
      return res.status(400).json({ error: 'ID de usuário inválido' });
    }
    
    // Parameterized query
    const query = 'SELECT id, email, name FROM users WHERE id = @userId';
    const result = await sql.query(query, { userId: parseInt(userId) });
    
    res.json(result.recordset);
  } catch (error) {
    // Log detalhado internamente
    console.error('Database error:', error);
    
    // Resposta genérica ao cliente
    res.status(500).json({ 
      error: 'Erro interno do servidor',
      referenceId: req.id // ID para tracking
    });
  }
});
```
