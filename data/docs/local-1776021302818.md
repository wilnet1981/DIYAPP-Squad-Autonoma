## Estratégia de Failover

### 1. Circuit Breaker Pattern
```javascript
class CircuitBreaker {
  constructor(service, options = {}) {
    this.service = service;
    this.state = 'CLOSED';
    this.failureCount = 0;
    this.successCount = 0;
    this.threshold = options.threshold || 5;
    this.timeout = options.timeout || 10000;
    this.resetTimeout = options.resetTimeout || 60000;
  }

  async call(request) {
    if (this.state === 'OPEN') {
      if (Date.now() - this.lastFailureTime > this.resetTimeout) {
        this.state = 'HALF_OPEN';
      } else {
        throw new Error('Circuit breaker is OPEN');
      }
    }

    try {
      const response = await this.service(request);
      this.onSuccess();
      return response;
    } catch (error) {
      this.onFailure();
      throw error;
    }
  }

  onSuccess() {
    this.successCount++;
    if (this.state === 'HALF_OPEN' && this.successCount >= this.threshold) {
      this.state = 'CLOSED';
      this.failureCount = 0;
    }
  }

  onFailure() {
    this.failureCount++;
    this.lastFailureTime = Date.now();
    
    if (this.failureCount >= this.threshold) {
      this.state = 'OPEN';
      setTimeout(() => {
        this.state = 'HALF_OPEN';
        this.successCount = 0;
      }, this.resetTimeout);
    }
  }
}
```

// health-check.js
class HealthMonitor {
  async checkServiceHealth(serviceName) {
    const checks = {
      database: await this.checkDatabase(),
      cache: await this.checkRedis(),
      external_apis: await this.checkExternalDependencies(),
      disk_space: this.checkDiskSpace(),
      memory: this.checkMemoryUsage()
    };

    const status = Object.values(checks).every(check => check.healthy) 
      ? 'HEALTHY' 
      : 'DEGRADED';

    return {
      status,
      timestamp: new Date().toISOString(),
      checks,
      service: serviceName,
      version: process.env.APP_VERSION
    };
  }
}

## ADRs (Architecture Decision Records)

**Status**: Aceito  
**Contexto**: Definir limites claros entre serviços para evitar acoplamento  
**Decisão**: Cada serviço possui seu próprio banco de dados e expõe API bem definida  
**Consequências**: + Independência de deploy, - Complexidade de transações distribuídas

### ADR-002: Communication Protocol
**Status**: Aceito  
**Contexto**: Escolher protocolo principal para comunicação entre serviços  
**Decisão**: gRPC para comunicação síncrona, Kafka para eventos assíncronos  
**Consequências**: + Performance, - Curva de aprendizado

### ADR-003: Failover Strategy
**Status**: Aceito  
**Contexto**: Garantir 100% de disponibilidade em produção  
**Decisão**: Circuit breaker + Health checks + Multi-region deployment  
**Consequências**: + Resiliência, - Custo de infraestrutura

### 1. ADR-001-service-boundaries.md
```markdown
# ADR-001: Service Boundaries

## Status
Aceito

## Contexto
Na arquitetura monolítica anterior, os componentes estavam fortemente acoplados, dificultando:
- Deploy independente
- Escalabilidade seletiva
- Manutenção de equipes especializadas
- Adoção de novas tecnologias

## Decisão
Cada domínio de negócio será implementado como um microserviço independente com:
1. Banco de dados próprio (Database per Service)
2. API bem definida (gRPC + REST para admin)
3. Contrato de eventos assíncronos
4. Autonomia completa de deploy

## Opções Consideradas

### Opção A: Database per Service
**Prós**: 
- Isolamento completo
- Escalabilidade independente
- Tecnologia adequada por domínio

**Contras**:
- Transações distribuídas complexas
- Duplicação eventual de dados
- Consistência eventual

### Opção B: Shared Database
**Prós**:
- Transações ACID
- Consistência forte
- Menor complexidade inicial

**Contras**:
- Acoplamento forte
- Difícil escalar
- Single point of failure

### Opção C: CQRS + Event Sourcing
**Prós**:
- Alta escalabilidade
- Audit trail completo
- Desacoplamento temporal

**Contras**:
- Complexidade alta
- Curva de aprendizado íngreme
- Over-engineering para nosso caso

## Opção Escolhida: A (Database per Service)
Justificativa: O isolamento e autonomia superam a complexidade de transações distribuídas para nosso caso de uso.

## Consequências
### Positivas
- Equipes podem trabalhar independentemente
- Escalabilidade granular
- Resiliência melhorada (falha isolada)
- Evolução tecnológica independente

### Negativas
- SAGA patterns necessários para transações
- Duplicação eventual de dados
- Monitoramento distribuído necessário

### Riscos
1. **Inconsistência de dados**: Mitigado com eventos e compensação
2. **Complexidade operacional**: Mitigado com observabilidade robusta
3. **Latência**: Mitigado com cache e design assíncrono

## Revisão
Reavaliar em 6 meses ou após 1M transações/mês
```

import json

# Context variable for correlation ID
correlation_id_ctx: ContextVar[str] = ContextVar('correlation_id', default='')


class JSONFormatter(logging.Formatter):
    """Formatter para logs estruturados em JSON"""
    
    def format(self, record: logging.LogRecord) -> str:
        log_object: Dict[str, Any] = {
            "timestamp": datetime.utcnow().isoformat(),
            "level": record.levelname,
            "correlation_id": correlation_id_ctx.get(),
            "logger": record.name,
            "message": record.getMessage(),
            "module": record.module,
            "function": record.funcName,
            "line": record.lineno,
        }
        
        # Adiciona campos extras do log record
        if hasattr(record, 'user_id'):
            log_object['user_id'] = record.user_id
        
        if hasattr(record, 'endpoint'):
            log_object['endpoint'] = record.endpoint
        
        if hasattr(record, 'duration_ms'):
            log_object['duration_ms'] = record.duration_ms
        
        if hasattr(record, 'status'):
            log_object['status'] = record.status
        
        # Adiciona exception info se existir
        if record.exc_info:
            log_object['exception'] = self.formatException(record.exc_info)
        
        # Mascara dados sensíveis
        self._mask_sensitive_data(log_object)
        
        return json.dumps(log_object)
    
    def _mask_sensitive_data(self, log_object: Dict[str, Any]) -> None:
        """Mascara dados sensíveis nos logs"""
        sensitive_fields = {'password', 'token', 'secret', 'authorization', 'credit_card'}
        
        def mask_value(value: Any) -> Any:
            if isinstance(value, str) and any(s in value.lower() for s in sensitive_fields):
                return "***MASKED***"
            return value
        
        def mask_dict(d: Dict[str, Any]) -> Dict[str, Any]:
            masked = {}
            for key, value in d.items():
                if isinstance(value, dict):
                    masked[key] = mask_dict(value)
                elif isinstance(value, list):
                    masked[key] = [mask_dict(v) if isinstance(v, dict) else mask_value(v) for v in value]
                else:
                    masked[key] = mask_value(value)
            return masked
        
        for key in list(log_object.keys()):
            if isinstance(log_object[key], dict):
                log_object[key] = mask_dict(log_object[key])
            else:
                log_object[key] = mask_value(log_object[key])


def setup_logging() -> None:
    """Configura o logging estruturado"""
    logger = logging.getLogger()
    logger.setLevel(logging.INFO)
    
    # Remove handlers existentes
    logger.handlers.clear()
    
    # Console handler com JSON formatter
    console_handler = logging.StreamHandler()
    console_handler.setFormatter(JSONFormatter())
    logger.addHandler(console_handler)
    
    logging.info("Logging configurado", extra={
        "service": "diyapp-core",
        "version": "v12"
    })

    """Define o correlation ID"""

def track_api_request(method: str, endpoint: str):
    """Decorator para trackear requisições API"""
    def decorator(func: Callable):
        async def wrapper(*args, **kwargs):
            start_time = time.time()
            try:
                response = await func(*args, **kwargs)
                status = getattr(response, 'status_code', 200)
                API_REQUEST_COUNT.labels(method=method, endpoint=endpoint, status=status).inc()
                return response
            except Exception as e:
                status = getattr(e, 'status_code', 500)
                API_REQUEST_COUNT.labels(method=method, endpoint=endpoint, status=status).inc()
                raise
            finally:
                duration = time.time() - start_time
                API_REQUEST_DURATION.labels(method=method, endpoint=endpoint).observe(duration)
        return wrapper
    return decorator

    """
    Validate source files before ingestion
    Returns validation summary
    """
    execution_date = context['execution_date']
    batch_id = f"batch_{execution_date.strftime('%Y%m%d_%H%M%S')}"
    
    # Check S3/GCS for new files
    s3_hook = S3Hook(aws_conn_id='aws_diyapp')
    gcs_hook = GCSHook(gcp_conn_id='gcp_diyapp')
    
    files_to_process = {
        'telemetry_events': [],
        'performance_metrics': [],
        'business_events': [],
        'llm_usage': []
    }
    
    # Check telemetry events (from S3)
    telemetry_prefix = f"telemetry/events/{execution_date.strftime('%Y/%m/%d/%H')}/"
    telemetry_files = s3_hook.list_keys(
        bucket_name='diyapp-telemetry',
        prefix=telemetry_prefix
    )
    
    for file_key in telemetry_files or []:
        if file_key.endswith('.json.gz'):
            file_info = s3_hook.get_key(file_key, bucket_name='diyapp-telemetry')
            if file_info.last_modified.replace(tzinfo=None) > execution_date - timedelta(minutes=10):
                files_to_process['telemetry_events'].append({
                    'source': 's3',
                    'bucket': 'diyapp-telemetry',
                    'key': file_key,
                    'size': file_info.content_length
                })
    
    # Check performance metrics (from GCS)
    metrics_prefix = f"metrics/{execution_date.strftime('%Y/%m/%d/%H')}/"
    metrics_files = gcs_hook.list(
        bucket_name='diyapp-metrics',
        prefix=metrics_prefix
    )
    
    for file_name in metrics_files or []:
        if file_name.endswith('.parquet'):
            file_info = gcs_hook.get_metadata(
                bucket_name='diyapp-metrics',
                object_name=file_name
            )
            if file_info['updated'].replace(tzinfo=None) > execution_date - timedelta(minutes=10):
                files_to_process['performance_metrics'].append({
                    'source': 'gcs',
                    'bucket': 'diyapp-metrics',
                    'key': file_name,
                    'size': file_info['size']
                })
    
    context['ti'].xcom_push(key='files_to_process', value=files_to_process)
    context['ti'].xcom_push(key='batch_id', value=batch_id)
    
    # Validate we have data to process
    total_files = sum(len(files) for files in files_to_process.values())
    if total_files == 0:
        logging.info("No new files to process")
        # This is not a failure - just no data in this window
        return {'status': 'no_data', 'batch_id': batch_id}
    
    logging.info(f"Found {total_files} files to process in batch {batch_id}")
    return {'status': 'ready', 'batch_id': batch_id, 'files': files_to_process}

  /**
   * Implementação OAuth2 completa
   */
  class OAuth2Handler {
    constructor(provider) {
      this.provider = provider;
      this.stateStore = new Map();
    }

    /**
     * Gera URL de autorização OAuth2
     */
    generateAuthUrl(options = {}) {
      const state = crypto.randomBytes(16).toString('hex');
      const nonce = crypto.randomBytes(16).toString('hex');
      
      this.stateStore.set(state, {
        nonce,
        redirectUri: options.redirectUri,
        timestamp: Date.now()
      });

      // Limpa estados antigos (mais de 10 minutos)
      this.cleanupOldStates();

      const baseUrls = {
        google: `https://accounts.google.com/o/oauth2/v2/auth?client_id=${process.env.GOOGLE_CLIENT_ID}&redirect_uri=${encodeURIComponent(options.redirectUri)}&response_type=code&scope=${encodeURIComponent(options.scope || 'email profile')}&state=${state}&nonce=${nonce}`,
        github: `https://github.com/login/oauth/authorize?client_id=${process.env.GITHUB_CLIENT_ID}&redirect_uri=${encodeURIComponent(options.redirectUri)}&scope=${encodeURIComponent(options.scope || 'user:email')}&state=${state}`,
        microsoft: `https://login.microsoftonline.com/${process.env.MICROSOFT_TENANT_ID}/oauth2/v2.0/authorize?client_id=${process.env.MICROSOFT_CLIENT_ID}&response_type=code&redirect_uri=${encodeURIComponent(options.redirectUri)}&scope=${encodeURIComponent(options.scope || 'User.Read')}&state=${state}&nonce=${nonce}`
      };

      return baseUrls[this.provider];
    }

    /**
     * Valida callback OAuth2 e troca code por token
     */
    async validateCallback(code, state) {
      // Valida state
      const storedState = this.stateStore.get(state);
      if (!storedState) {
        throw new SecurityError('Estado OAuth2 inválido ou expirado', 'OAUTH_STATE_INVALID');
      }

      // Remove state usado
      this.stateStore.delete(state);

      // Valida timestamp (10 minutos máximo)
      if (Date.now() - storedState.timestamp > 10 * 60 * 1000) {
        throw new SecurityError('Sessão OAuth2 expirada', 'OAUTH_SESSION_EXPIRED');
      }

      // Troca code por token de acesso
      const token = await this.exchangeCodeForToken(code, storedState.redirectUri);
      
      // Valida token
      const userInfo = await this.validateToken(token);
      
      return {
        token,
        userInfo,
        provider: this.provider
      };
    }

    cleanupOldStates() {
      const now = Date.now();
      for (const [state, data] of this.stateStore.entries()) {
        if (now - data.timestamp > 10 * 60 * 1000) {
          this.stateStore.delete(state);
        }
      }
    }
  }

  /**
   * Implementação JWT com refresh tokens
   */
  class JWTManager {
    constructor() {
      this.tokenBlacklist = new Set();
      this.refreshTokens = new Map();
    }

    /**
     * Gera access token (15 minutos)
     */
    generateAccessToken(payload) {
      return jwt.sign(
        {
          ...payload,
          type: 'access',
          iat: Math.floor(Date.now() / 1000)
        },
        process.env.JWT_ACCESS_SECRET,
        { 
          expiresIn: SECURITY_CONFIG.JWT_ACCESS_TOKEN_EXPIRY,
          algorithm: 'HS256'
        }
      );
    }

    /**
     * Gera refresh token (7 dias)
     */
    generateRefreshToken(userId) {
      const refreshToken = crypto.randomBytes(40).toString('hex');
      const expiresAt = new Date(Date.now() + 7 * 24 * 60 * 60 * 1000);
      
      this.refreshTokens.set(refreshToken, {
        userId,
        expiresAt,
        createdAt: new Date()
      });

      return refreshToken;
    }

    /**
     * Valida JWT token
     */
    validateToken(token, type = 'access') {
      try {
        const secret = type === 'access' 
          ? process.env.JWT_ACCESS_SECRET 
          : process.env.JWT_REFRESH_SECRET;
        
        // Verifica se token está na blacklist
        if (this.tokenBlacklist.has(token)) {
          throw new SecurityError('Token revogado', 'TOKEN_REVOKED');
        }

        const decoded = jwt.verify(token, secret, { algorithms: ['HS256'] });
        
        // Valida tipo do token
        if (decoded.type !== type) {
          throw new SecurityError('Tipo de token inválido', 'TOKEN_TYPE_INVALID');
        }

        return decoded;
      } catch (error) {
        if (error.name === 'TokenExpiredError') {
          throw new SecurityError('Token expirado', 'TOKEN_EXPIRED');
        }
        throw new SecurityError('Token inválido', 'TOKEN_INVALID');
      }
    }

    /**
     * Revoga token (logout)
     */
    revokeToken(token) {
      this.tokenBlacklist.add(token);
      
      // Limpa blacklist antiga periodicamente
      setTimeout(() => {
        this.tokenBlacklist.delete(token);
      }, 24 * 60 * 60 * 1000); // 24 horas
    }

    /**
     * Refresh token flow
     */
    refreshAccessToken(refreshToken) {
      // Valida refresh token
      const stored = this.refreshTokens.get(refreshToken);
      if (!stored || stored.expiresAt < new Date()) {
        throw new SecurityError('Refresh token inválido ou expirado', 'REFRESH_TOKEN_INVALID');
      }

      // Remove refresh token usado (one-time use)
      this.refreshTokens.delete(refreshToken);

      // Gera novo par de tokens
      const newAccessToken = this.generateAccessToken({ userId: stored.userId });
      const newRefreshToken = this.generateRefreshToken(stored.userId);

      return {
        accessToken: newAccessToken,
        refreshToken: newRefreshToken
      };
    }
  }

  /**
   * Middleware de autenticação JWT
   */
  function authenticateJWT(req, res, next) {
    const authHeader = req.headers.authorization;
    
    if (!authHeader || !authHeader.startsWith('Bearer ')) {
      return res.status(401).json({ 
        error: 'Token de autenticação necessário',
        code: 'AUTH_TOKEN_REQUIRED'
      });
    }

    const token = authHeader.split(' ')[1];
    
    try {
      const jwtManager = new JWTManager();
      const decoded = jwtManager.validateToken(token, 'access');
      
      req.user = decoded;
      req.token = token;
      next();
    } catch (error) {
      return res.status(401).json({
        error: error.message,
        code: error.code || 'AUTH_FAILED'
      });
    }
  }

  /**
   * Middleware de autorização baseada em roles
   */
  function authorize(roles = []) {
    return (req, res, next) => {
      if (!req.user) {
        return res.status(401).json({ 
          error: 'Usuário não autenticado',
          code: 'USER_NOT_AUTHENTICATED'
        });
      }

      if (roles.length > 0 && !roles.includes(req.user.role)) {
        return res.status(403).json({ 
          error: 'Permissão insuficiente',
          code: 'INSUFFICIENT_PERMISSIONS',
          required: roles,
          current: req.user.role
        });
      }

      next();
    };
  }

    constructor(config) {
      this.baseUrl = config.url;
      this.token = config.token;
      this.mountPath = config.mountPath || 'secret';
      this.cache = new Map();
    }

    async getSecret(path) {
      // Verifica cache primeiro
      const cached = this.cache.get(path);
      if (cached && Date.now() - cached.timestamp < 5 * 60 * 1000) {
        return cached.value;
      }

      try {
        const response = await fetch(`${this.baseUrl}/v1/${this.mountPath}/data/${path}`, {
          headers: {
            'X-Vault-Token': this.token,
            'Content-Type': 'application/json'
          }
        });

        if (!response.ok) {
          throw new Error(`Vault error: ${response.status}`);
        }

        const data = await response.json();
        const secret = data.data.data;

        // Armazena em cache
        this.cache.set(path, {
          value: secret,
          timestamp: Date.now()
        });

        return secret;
      } catch (error) {
        console.error('Erro ao acessar Vault:', error);
        throw new SecurityError('Falha ao recuperar secret', 'SECRET_RETRIEVAL_FAILED');
      }
    }

    async rotateSecret(path, newSecret) {
      try {
        const response = await fetch(`${this.baseUrl}/v1/${this.mountPath}/data/${path}`, {
          method: 'POST',
          headers: {
            'X-Vault-Token': this.token,
            'Content-Type': 'application/json'
          },
          body: JSON.stringify({
            data: newSecret
          })
        });

        if (!response.ok) {
          throw new Error(`Vault rotation error: ${response.status}`);
        }

        // Limpa cache
        this.cache.delete(path);

        return true;
      } catch (error) {
        console.error('Erro ao rotacionar secret:', error);
        throw new SecurityError('Falha ao rotacionar secret', 'SECRET_ROTATION_FAILED');
      }
    }
  }

  class AtRestEncryption {
    constructor() {
      this.algorithm = SECURITY_CONFIG.ENCRYPTION_ALGORITHM;
    }

    /**
     * Criptografa dados sensíveis
     */
    encrypt(data, key) {
      const iv = crypto.randomBytes(SECURITY_CONFIG.IV_LENGTH);
      const cipher = crypto.createCipheriv(
        this.algorithm,
        Buffer.from(key, 'hex'),
        iv
      );

      let encrypted = cipher.update(data, 'utf8', 'hex');
      encrypted += cipher.final('hex');

      const authTag = cipher.getAuthTag();

      return {
        encrypted,
        iv: iv.toString('hex'),
        authTag: authTag.toString('hex'),
        algorithm: this.algorithm
      };
    }

    /**
     * Descriptografa dados
     */
    decrypt(encryptedData, key) {
      const decipher = crypto.createDecipheriv(
        encryptedData.algorithm,
        Buffer.from(key, 'hex'),
        Buffer.from(encryptedData.iv, 'hex')
      );

      decipher.setAuthTag(Buffer.from(encryptedData.authTag, 'hex'));

      let decrypted = decipher.update(encryptedData.encrypted, 'hex', 'utf8');
      decrypted += decipher.final('utf8');

      return decrypted;
    }

    /**
     * Criptografa PII (Personal Identifiable Information)
     */
    encryptPII(data) {
      const piiFields = ['email', 'cpf', 'phone', 'address', 'birthDate'];
      const encrypted = { ...data };

        if (encrypted[field]) {

            encrypted[field],
            process.env.PII_ENCRYPTION_KEY
          );
          delete encrypted[field];
        }
      });

      return encrypted;
    }
