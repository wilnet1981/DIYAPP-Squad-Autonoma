# DIYAPP Evolution - V13 Core - Arquitetura de Microsserviços Baseada em Eventos

## ADR-001: Arquitetura de Microsserviços Baseada em Eventos para V13

**Data:** 2024-03-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista Backend

**CONTEXTO:**
DIYAPP V13 precisa evoluir de uma aplicação monolítica para uma arquitetura de microsserviços para:
1. Permitir escalabilidade independente de componentes
2. Facilitar deploy contínuo sem downtime
3. Isolar falhas entre domínios de negócio
4. Suportar times autônomos trabalhando em paralelo (Modo Hive)
5. Garantir estabilidade 100% autônoma

**DECISÃO:**
Adotar arquitetura de microsserviços baseada em eventos com os seguintes componentes:
1. API Gateway como ponto único de entrada
2. Serviços independentes por domínio de negócio
3. Message Broker (RabbitMQ) para comunicação assíncrona
4. Banco de dados por serviço (Database-per-service)
5. Sistema de observabilidade distribuída
6. Service Discovery dinâmico

**OPÇÕES CONSIDERADAS:**
- **Opção A: Monolito modularizado** — Prós: Simples, baixa latência, transações ACID fáceis. Contras: Escalabilidade limitada, acoplamento alto, deploy único.
- **Opção B: Microsserviços síncronos (REST/GRPC)** — Prós: Desacoplamento, deploy independente. Contras: Latência em cadeias, falhas em cascata, complexidade de versionamento.
- **Opção C: Microsserviços baseados em eventos** — Prós: Desacoplamento total, resiliência a falhas, escalabilidade horizontal. Contras: Complexidade eventual consistency, debugging distribuído.
- **Opção escolhida: C** — Justificativa: Melhor suporte para autonomia de squads, resiliência necessária para estabilidade 100%, alinhamento com Modo Hive.

**CONSEQUÊNCIAS:**
**Positivas:**
- Cada squad pode desenvolver e deployar independentemente
- Falhas isoladas não derrubam sistema inteiro
- Escalabilidade granular por componente crítico
- Evolução tecnológica independente por serviço

**Negativas:**
- Complexidade aumentada em transações distribuídas
- Necessidade de sistema de observabilidade robusto
- Overhead de rede entre serviços
- Eventual consistency requer design cuidadoso

**Riscos:**
1. **Latência aumentada:** Mitigar com cache estratégico e otimização de payloads
2. **Debugging distribuído difícil:** Implementar correlation IDs e logging centralizado
3. **Perda de mensagens:** Configurar persistent queues e dead letter exchanges
4. **Versionamento de eventos:** Implementar schema registry e backward compatibility

**REVISÃO:** 2024-06-15 (3 meses para reavaliar complexidade vs benefícios)

import express from 'express';
import cors from 'cors';
import helmet from 'helmet';
import rateLimit from 'express-rate-limit';
import { createProxyMiddleware } from 'http-proxy-middleware';
import { logger, correlationMiddleware } from '@diyapp/shared';
import { healthCheck } from './health';
import { metricsMiddleware } from './metrics';

const app = express();
const PORT = process.env.PORT || 3000;

// Security middleware
app.use(helmet());
app.use(cors({
  origin: process.env.ALLOWED_ORIGINS?.split(',') || ['http://localhost:8080'],
  credentials: true
}));

// Rate limiting
const limiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 minutes
  max: 100, // limit each IP to 100 requests per windowMs
  message: 'Too many requests from this IP'
});
app.use('/api/', limiter);

// Observability
app.use(correlationMiddleware);
app.use(metricsMiddleware);

// Health check
app.get('/health', healthCheck);

// Service routing
const services = {
  auth: process.env.AUTH_SERVICE_URL || 'http://localhost:3001',
  users: process.env.USER_SERVICE_URL || 'http://localhost:3002',
  projects: process.env.PROJECT_SERVICE_URL || 'http://localhost:3003',
  tasks: process.env.TASK_SERVICE_URL || 'http://localhost:3004',
  notifications: process.env.NOTIFICATION_SERVICE_URL || 'http://localhost:3005'
};

// Proxy configuration
Object.entries(services).forEach(([service, target]) => {
  app.use(`/api/${service}`, createProxyMiddleware({
    target,
    changeOrigin: true,
    pathRewrite: {
      [`^/api/${service}`]: ''
    },
    onError: (err, req, res) => {
      logger.error(`Proxy error for ${service}:`, err);
      res.status(502).json({
        error: 'Service temporarily unavailable',
        service,

import amqp, { Channel, Connection } from 'amqplib';
import { logger } from '../utils/logger';
import { BaseEvent } from './types';
import { EXCHANGES } from './broker-config';

export class EventProducer {
  private connection: Connection | null = null;
  private channel: Channel | null = null;
  private connected = false;

  constructor(private brokerUrl: string) {}

  async connect(): Promise<void> {
    if (this.connected) return;

    try {
      this.connection = await amqp.connect(this.brokerUrl);
      this.channel = await this.connection.createChannel();
      
      // Assert exchanges
      await this.channel.assertExchange(EXCHANGES.DOMAIN_EVENTS, 'topic', { durable: true });
      await this.channel.assertExchange(EXCHANGES.COMMANDS, 'direct', { durable: true });
      await this.channel.assertExchange(EXCHANGES.DEAD_LETTER, 'fanout', { durable: true });
      
      this.connected = true;
      logger.info('Event producer connected to broker');
      
      // Handle connection errors
      this.connection.on('error', (err) => {
        logger.error('Broker connection error:', err);
        this.connected = false;
      });
      
      this.connection.on('close', () => {
        logger.warn('Broker connection closed');
        this.connected = false;
      });
    } catch (error) {
      logger.error('Failed to connect to broker:', error);
      throw error;
    }
  }

  async publishEvent(event: BaseEvent): Promise<boolean> {
    if (!this.connected || !this.channel) {
      throw new Error('Producer not connected to broker');
    }

    try {
      const routingKey = `${event.aggregateType}.${event.eventType.toLowerCase()}`;
      const message = Buffer.from(JSON.stringify(event));
      
      const published = this.channel.publish(
        EXCHANGES.DOMAIN_EVENTS,
        routingKey,
        message,
        {
          persistent: true,
          headers: {
            'x-event-type': event.eventType,
            'x-event-version': event.version.toString(),
            'x-correlation-id': event.correlationId,
            'x-timestamp': event.timestamp.toISOString()
          }
        }
      );
      
      if (published) {
        logger.debug('Event published', {
          eventType: event.eventType,
          aggregateId: event.aggregateId,
          routingKey
        });
      } else {
        logger.warn('Event not published (channel backpressure)', {
          eventType: event.eventType
        });
      }
      
      return published;
    } catch (error) {
      logger.error('Failed to publish event:', error);
      throw error;
    }
  }

  async publishCommand(command: any, queue: string): Promise<boolean> {
    if (!this.connected || !this.channel) {
      throw new Error('Producer not connected to broker');
    }

    try {
      await this.channel.assertQueue(queue, { durable: true });
      
      const message = Buffer.from(JSON.stringify(command));
      const published = this.channel.sendToQueue(
        queue,
        message,
        { persistent: true }
      );
      
      return published;
    } catch (error) {
      logger.error('Failed to publish command:', error);
      throw error;
    }
  }

  async disconnect(): Promise<void> {
    if (this.channel) {
      await this.channel.close();
    }
    if (this.connection) {
      await this.connection.close();
    }
    this.connected = false;
    logger.info('Event producer disconnected');
  }
}

// Singleton instance
let producerInstance: EventProducer | null = null;

export function getEventProducer(): EventProducer {
  if (!producerInstance) {
    const brokerUrl = process.env.RABBITMQ_URL || 'amqp://localhost:5672';
    producerInstance = new EventProducer(brokerUrl);
  }
  return producerInstance;
}

def create_access_token(data: Dict[str, Any], expires_delta: Optional[timedelta] = None) -> str:
    """Cria token JWT de acesso"""
    to_encode = data.copy()
    if expires_delta:
        expire = datetime.utcnow() + expires_delta
    else:
        expire = datetime.utcnow() + timedelta(minutes=settings.ACCESS_TOKEN_EXPIRE_MINUTES)
    
    to_encode.update({"exp": expire, "type": "access"})
    encoded_jwt = jwt.encode(to_encode, settings.SECRET_KEY, algorithm=settings.ALGORITHM)
    
    logger.info("access_token_created", user_id=data.get("sub"), expires_at=expire.isoformat())
    return encoded_jwt


def create_refresh_token(db: Session, user_id: int) -> str:
    """Cria e armazena token de refresh"""
    from uuid import uuid4
    
    expire = datetime.utcnow() + timedelta(days=settings.REFRESH_TOKEN_EXPIRE_DAYS)
    token = str(uuid4())
    
    db_token = RefreshToken(
        user_id=user_id,
        token=token,
        expires_at=expire
    )
    db.add(db_token)
    db.commit()
    
    logger.info("refresh_token_created", user_id=user_id, expires_at=expire.isoformat())
    return token


def verify_token(token: str) -> Optional[Dict[str, Any]]:
    """Verifica e decodifica token JWT"""
    try:
        payload = jwt.decode(token, settings.SECRET_KEY, algorithms=[settings.ALGORITHM])
        return payload
    except JWTError as e:
        logger.warning("token_verification_failed", error=str(e))
        return None


def revoke_refresh_token(db: Session, token: str) -> bool:
    """Revoga um token de refresh"""
    db_token = db.query(RefreshToken).filter(RefreshToken.token == token).first()
    if db_token:
        db.delete(db_token)
        db.commit()
        logger.info("refresh_token_revoked", token=token[:8] + "...")
        return True
    return False

from circuitbreaker import circuit

class CircuitBreaker:
    """Circuit breaker com fallback e métricas"""
    
    def __init__(
        self,
        failure_threshold: int = settings.CIRCUIT_FAILURE_THRESHOLD,
        recovery_timeout: int = settings.CIRCUIT_RECOVERY_TIMEOUT,
        expected_exception: tuple = (httpx.HTTPError, TimeoutError, ConnectionError)
    ):
        self.failure_threshold = failure_threshold
        self.recovery_timeout = recovery_timeout
        self.expected_exception = expected_exception
        self._failure_count = 0
        self._last_failure_time = 0
        self._state = "closed"  # closed, open, half-open
        
    def __call__(self, func: Callable) -> Callable:
        """Decorator para aplicar circuit breaker"""
        def wrapper(*args, **kwargs):
            if self._state == "open":
                # Circuito aberto - retorna fallback imediatamente
                if time.time() - self._last_failure_time > self.recovery_timeout:
                    self._state = "half-open"
                    logger.info("circuit_half_open", function=func.__name__)
                else:
                    logger.warning("circuit_open_blocked", function=func.__name__)
                    raise CircuitOpenException(f"Circuit open for {func.__name__}")
            
            try:
                result = func(*args, **kwargs)
                
                if self._state == "half-open":
                    # Sucesso em half-open - fecha o circuito
                    self._state = "closed"
                    self._failure_count = 0
                    logger.info("circuit_closed", function=func.__name__)
                
                return result
                
            except self.expected_exception as e:
                self._failure_count += 1
                self._last_failure_time = time.time()
                
                logger.warning(
                    "circuit_failure",
                    function=func.__name__,
                    error=str(e),
                    failure_count=self._failure_count
                )
                
                if self._failure_count >= self.failure_threshold:
                    self._state = "open"
                    logger.error("circuit_opened", function=func.__name__)
                
                raise
                
        return wrapper
    
    def with_fallback(self, fallback_func: Callable) -> Callable:
        """Adiciona fallback ao circuit breaker"""
        def decorator(func: Callable) -> Callable:
            @self
            def wrapper(*args, **kwargs):
                try:
                    return func(*args, **kwargs)
                except CircuitOpenException:
                    logger.info("using_fallback", function=func.__name__)
                    return fallback_func(*args, **kwargs)
                except self.expected_exception:
                    logger.info("using_fallback_after_error", function=func.__name__)
                    return fallback_func(*args, **kwargs)
            return wrapper
        return decorator


class CircuitOpenException(Exception):
    """Exceção quando o circuito está aberto"""
    pass

export const useMediaQuery = (query: string): boolean => {
  const [matches, setMatches] = useState(false);

  useEffect(() => {
    const media = window.matchMedia(query);
    if (media.matches !== matches) {
      setMatches(media.matches);
    }
    const listener = () => setMatches(media.matches);
    window.addEventListener('resize', listener);
    return () => window.removeEventListener('resize', listener);
  }, [matches, query]);

  return matches;
};
