# DIYAPP Evolution - V12 Core - Arquitetura de Orquestração de Agentes

## ADR-001: Arquitetura de Orquestração de Agentes V12

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista LLM

### CONTEXTO:
O sistema DIYAPP evoluiu para uma arquitetura multi-agente onde diferentes agentes especializados (Tech Lead, Aprovador, Especialista LLM, etc.) precisam colaborar de forma autônoma. A versão atual usa comunicação direta ponto-a-ponto que não escala e não oferece resiliência adequada. Precisamos de uma arquitetura de orquestração que suporte:
1. Comunicação assíncrona entre agentes
2. Auto-descoberta e auto-cura
3. Balanceamento de carga entre instâncias do mesmo tipo de agente
4. Monitoramento em tempo real
5. Escalabilidade horizontal

### DECISÃO:
Implementar uma arquitetura baseada em Pub/Sub com Redis como backbone de mensagens, complementada por um Orchestrator Service que gerencia o ciclo de vida dos agentes. Cada agente será containerizado e registrado em um Service Registry.

### OPÇÕES CONSIDERADAS:

**Opção A: RabbitMQ + Docker Swarm**
- Prós: Mensagens confiáveis, filas duráveis, suporte a padrões AMQP
- Contras: Mais complexo de configurar, overhead maior, menos adequado para padrões Pub/Sub

**Opção B: Apache Kafka + Kubernetes**
- Prós: Alta throughput, retenção de mensagens, excelente para streams
- Contras: Overkill para nosso volume, complexidade operacional alta, custo

**Opção C: Redis Pub/Sub + Docker Compose com auto-orquestração**
- Prós: Simples, rápido, baixa latência, suporte nativo a Pub/Sub
- Contras: Mensagens não duráveis (perdidas se não consumidas), sem garantia de entrega

**Opção escolhida: C com Redis Streams** - Justificativa: Redis Streams resolve o problema de durabilidade das mensagens enquanto mantém a simplicidade. Para nosso volume atual (centenas de mensagens/segundo), é suficiente. Docker Compose com scripts de auto-cura é mais adequado que orquestradores complexos para nossa squad autônoma.

### CONSEQUÊNCIAS:
**Positivas:**
- Baixa latência de comunicação (< 50ms)
- Setup simples e manutenção fácil
- Escalabilidade horizontal por tipo de agente
- Auto-cura implementada a nível de aplicação

**Negativas:**
- Dependência de um único Redis (ponto único de falha)
- Necessidade de implementar lógica de retry na aplicação
- Mensagens podem ser perdidas em caso de crash do Redis

**Riscos:**
- Redis se tornar gargalo → Mitigação: Monitorar métricas e planejar cluster Redis
- Perda de mensagens críticas → Mitigação: Implementar dead letter queues e retry com backoff exponencial

**REVISÃO:** 2024-04-15 (reavaliar após 3 meses de produção)

### 2. src/orchestrator/index.js
```javascript
const express = require('express');
const Redis = require('ioredis');
const { v4: uuidv4 } = require('uuid');
const logger = require('../shared/logger');
const AgentRegistry = require('./agent-registry');

class Orchestrator {
  constructor() {
    this.app = express();
    this.port = process.env.PORT || 3000;
    this.redis = new Redis(process.env.REDIS_URL);
    this.registry = new AgentRegistry(this.redis);
    this.healthTimeout = parseInt(process.env.AGENT_HEALTH_TIMEOUT) || 30000;
    this.maxRestartAttempts = parseInt(process.env.MAX_RESTART_ATTEMPTS) || 3;
    
    this.setupMiddleware();
    this.setupRoutes();
    this.setupEventHandlers();
    this.startHealthMonitoring();
  }

  setupMiddleware() {
    this.app.use(express.json());
    this.app.use((req, res, next) => {
      logger.info(`${req.method} ${req.path}`, { ip: req.ip });
      next();
    });
  }

  setupRoutes() {
    // Registro de agentes
    this.app.post('/api/agents/register', async (req, res) => {
      try {
        const { agentId, agentType, capabilities, metadata } = req.body;
        const agent = await this.registry.registerAgent({
          id: agentId,
          type: agentType,
          capabilities: capabilities || [],
          metadata: metadata || {},
          lastHeartbeat: Date.now(),
          status: 'healthy',
          restartCount: 0
        });
        
        logger.info(`Agent registered: ${agentId} (${agentType})`);
        res.json({ success: true, agent });
      } catch (error) {
        logger.error('Agent registration failed', { error: error.message });
        res.status(500).json({ success: false, error: error.message });
      }
    });

    // Heartbeat de agentes
    this.app.post('/api/agents/:agentId/heartbeat', async (req, res) => {
      try {
        const { agentId } = req.params;
        await this.registry.updateHeartbeat(agentId);
        res.json({ success: true, timestamp: Date.now() });
      } catch (error) {
        logger.error('Heartbeat update failed', { agentId, error: error.message });
        res.status(404).json({ success: false, error: 'Agent not found' });
      }
    });

    // Lista de agentes
    this.app.get('/api/agents', async (req, res) => {
      try {
        const { type, status } = req.query;
        const agents = await this.registry.getAgents({ type, status });
        res.json({ success: true, agents });
      } catch (error) {
        logger.error('Failed to get agents', { error: error.message });
        res.status(500).json({ success: false, error: error.message });
      }
    });

    // Estatísticas do sistema
    this.app.get('/api/stats', async (req, res) => {
      try {
        const stats = await this.registry.getSystemStats();
        res.json({ success: true, stats });
      } catch (error) {
        logger.error('Failed to get stats', { error: error.message });
        res.status(500).json({ success: false, error: error.message });
      }
    });

    // Endpoint de saúde
    this.app.get('/health', (req, res) => {
      res.json({ 
        status: 'healthy', 
        timestamp: new Date().toISOString(),
        uptime: process.uptime(),
        agents: this.registry.agentCount
      });
    });

    // Dashboard
    this.app.get('/', (req, res) => {
      res.send(`
        <html>
          <head><title>DIYAPP V12 Orchestrator</title></head>
          <body>
            <h1>DIYAPP V12 Agent Orchestrator</h1>
            <p>Status: <strong>Operational</strong></p>
            <p>Agents registered: ${this.registry.agentCount}</p>
            <p><a href="/api/agents">View Agents</a></p>
            <p><a href="/api/stats">System Stats</a></p>
          </body>
        </html>
      `);
    });
  }

  setupEventHandlers() {
    // Escuta eventos de agentes
    this.redis.subscribe('agent-events', (err, count) => {
      if (err) {
        logger.error('Failed to subscribe to agent-events', { error: err.message });
        return;
      }
      logger.info(`Subscribed to ${count} channel(s)`);
    });

    this.redis.on('message', async (channel, message) => {
      if (channel === 'agent-events') {
        try {
          const event = JSON.parse(message);
          await this.handleAgentEvent(event);
        } catch (error) {
          logger.error('Failed to process agent event', { error: error.message, message });
        }
      }
    });

    // Publica eventos do orchestrator
    this.publishEvent = (eventType, data) => {
      const event = {
        id: uuidv4(),
        type: eventType,
        timestamp: Date.now(),
        source: 'orchestrator',
        data
      };
      
      this.redis.publish('orchestrator-events', JSON.stringify(event));
      logger.debug(`Event published: ${eventType}`, { eventId: event.id });
    };
  }

  async handleAgentEvent(event) {
    switch (event.type) {
      case 'agent.started':
        logger.info(`Agent started: ${event.data.agentId}`);
        this.publishEvent('orchestrator.welcome', { 
          agentId: event.data.agentId,
          message: 'Welcome to DIYAPP V12'
        });
        break;
        
      case 'agent.error':
        logger.error(`Agent error: ${event.data.agentId}`, { error: event.data.error });
        
        // Verifica se precisa reiniciar
        const agent = await this.registry.getAgent(event.data.agentId);
        if (agent && agent.restartCount < this.maxRestartAttempts) {
          this.publishEvent('orchestrator.restart', { 
            agentId: event.data.agentId,
            attempt: agent.restartCount + 1
          });
        }
        break;
        
      case 'task.completed':
        logger.info(`Task completed by ${event.data.agentId}`, { 
          taskId: event.data.taskId,
          duration: event.data.duration 
        });
        break;
    }
  }

  startHealthMonitoring() {
    setInterval(async () => {
      try {
        const unhealthyAgents = await this.registry.checkAgentHealth(this.healthTimeout);
        
        for (const agent of unhealthyAgents) {
          logger.warn(`Unhealthy agent detected: ${agent.id}`, { 
            lastHeartbeat: agent.lastHeartbeat,
            status: agent.status 
          });
          
          // Tenta reiniciar se dentro dos limites
          if (agent.restartCount < this.maxRestartAttempts) {
            this.publishEvent('orchestrator.restart', { 
              agentId: agent.id,
              attempt: agent.restartCount + 1,
              reason: 'health_check_failed'
            });
            
            await this.registry.incrementRestartCount(agent.id);
          } else {
            logger.error(`Agent ${agent.id} exceeded max restart attempts`, {
              restartCount: agent.restartCount,
              maxAttempts: this.maxRestartAttempts
            });
            
            await this.registry.updateAgentStatus(agent.id, 'failed');
          }
        }
      } catch (error) {
        logger.error('Health monitoring failed', { error: error.message });
      }
    }, 10000); // Verifica a cada 10 segundos
  }

  start() {
    this.server = this.app.listen(this.port, () => {
      logger.info(`Orchestrator running on port ${this.port}`);
      this.publishEvent('orchestrator.started', { port: this.port });
    });
  }

  async shutdown() {
    logger.info('Shutting down orchestrator...');
    
    if (this.server) {
      this.server.close();
    }
    
    await this.redis.quit();
    logger.info('Orchestrator shutdown complete');
  }
}

// Inicialização
if (require.main === module) {
  const orchestrator = new Orchestrator();
  orchestrator.start();

  // Tratamento de sinais para graceful shutdown
  process.on('SIGTERM', () => orchestrator.shutdown());
  process.on('SIGINT', () => orchestrator.shutdown());
}

module.exports = Orchestrator;
```

import logging
import json
from typing import Dict, Any, Optional
from datetime import datetime
import uuid

class StructuredLogger:
    """Logger estruturado em JSON com correlation ID"""
    
    def __init__(self, name: str, level: int = logging.INFO):
        self.logger = logging.getLogger(name)
        self.logger.setLevel(level)
        
        if not self.logger.handlers:
            handler = logging.StreamHandler()
            handler.setFormatter(JSONFormatter())
            self.logger.addHandler(handler)
        
        self.correlation_id = None
        
    def set_correlation_id(self, correlation_id: str):
        """Define correlation ID para o contexto atual"""
        self.correlation_id = correlation_id
    
    def info(self, message: str, **kwargs):
        self._log(logging.INFO, message, **kwargs)
    
    def error(self, message: str, **kwargs):
        self._log(logging.ERROR, message, **kwargs)
    
    def warning(self, message: str, **kwargs):
        self._log(logging.WARNING, message, **kwargs)
    
    def debug(self, message: str, **kwargs):
        self._log(logging.DEBUG, message, **kwargs)
    
    def _log(self, level: int, message: str, **kwargs):
        extra = {
            'correlation_id': self.correlation_id or str(uuid.uuid4()),
            'timestamp': datetime.utcnow().isoformat(),
            **kwargs
        }
        self.logger.log(level, message, extra=extra)

class JSONFormatter(logging.Formatter):
    """Formatter para logs em JSON"""
    
    def format(self, record: logging.LogRecord) -> str:
        log_data = {
            'level': record.levelname,
            'timestamp': getattr(record, 'timestamp', datetime.utcnow().isoformat()),
            'message': record.getMessage(),
            'correlation_id': getattr(record, 'correlation_id', ''),
            'module': record.module,
            'function': record.funcName,
            'line': record.lineno,
        }
        
        # Adiciona campos extras (exceto os reservados)
        reserved = {'args', 'msg', 'levelname', 'levelno', 'pathname', 'filename', 
                   'module', 'exc_info', 'exc_text', 'stack_info', 'lineno', 'funcName',
                   'created', 'msecs', 'relativeCreated', 'thread', 'threadName',
                   'processName', 'process', 'message', 'asctime', 'timestamp',
                   'correlation_id'}
        
        for key, value in record.__dict__.items():
            if key not in reserved and not key.startswith('_'):
                # Mascara dados sensíveis
                if any(sensitive in key.lower() for sensitive in ['token', 'password', 'secret', 'key', 'email']):
                    log_data[key] = '***MASKED***'
                else:
                    log_data[key] = value
        
        # Adiciona exception se existir
        if record.exc_info:
            log_data['exception'] = self.formatException(record.exc_info)
        
        return json.dumps(log_data, ensure_ascii=False)

import time
import random
from typing import Callable, TypeVar, Optional, Any
from functools import wraps
from datetime import datetime, timedelta
from enum import Enum

T = TypeVar('T')

class CircuitState(str, Enum):
    CLOSED = "closed"
    OPEN = "open"
    HALF_OPEN = "half_open"

class CircuitBreaker:
    """Implementação de circuit breaker"""
    
    def __init__(self, failure_threshold: int = 5, 
                 recovery_timeout: int = 60,
                 name: str = "default"):
        self.failure_threshold = failure_threshold
        self.recovery_timeout = recovery_timeout
        self.name = name
        self.state = CircuitState.CLOSED
        self.failure_count = 0
        self.last_failure_time = None
        self.logger = StructuredLogger(f"circuit_breaker.{name}")
    
    def execute(self, func: Callable[..., T], *args, **kwargs) -> T:
        """Executa função com circuit breaker"""
        if self.state == CircuitState.OPEN:
            # Verifica se pode tentar recovery
            if self._can_attempt_recovery():
                self.state = CircuitState.HALF_OPEN
                self.logger.info(f"Circuit {self.name} moving to HALF_OPEN")
            else:
                raise CircuitBreakerOpenError(f"Circuit {self.name} is OPEN")
        
        try:
            result = func(*args, **kwargs)
            self._on_success()
            return result
        except Exception as e:
            self._on_failure(e)
            raise
    
    def _can_attempt_recovery(self) -> bool:
        if not self.last_failure_time:
            return True
        
        time_since_failure = (datetime.utcnow() - self.last_failure_time).total_seconds()
        return time_since_failure >= self.recovery_timeout
    
    def _on_success(self):
        if self.state == CircuitState.HALF_OPEN:
            self.state = CircuitState.CLOSED
            self.logger.info(f"Circuit {self.name} recovered, moving to CLOSED")
        
        self.failure_count = 0
        self.last_failure_time = None
    
    def _on_failure(self, error: Exception):
        self.failure_count += 1
        self.last_failure_time = datetime.utcnow()
        
        self.logger.error(f"Circuit {self.name} failure #{self.failure_count}", 
                         error=str(error))
        
        if self.failure_count >= self.failure_threshold:
            self.state = CircuitState.OPEN
            self.logger.warning(f"Circuit {self.name} OPEN after {self.failure_count} failures")

class CircuitBreakerOpenError(Exception):
    """Exceção quando circuit breaker está aberto"""
    pass

def retry_with_backoff(
    max_attempts: int = 3,
    base_delay: float = 1.0,
    max_delay: float = 10.0,
    jitter: bool = True,
    retry_on_exceptions: tuple = (Exception,)
):
    """Decorator para retry com backoff exponencial"""
    def decorator(func: Callable[..., T]) -> Callable[..., T]:
        @wraps(func)
        def wrapper(*args, **kwargs) -> T:
            last_exception = None
            
            for attempt in range(1, max_attempts + 1):
                try:
                    return func(*args, **kwargs)
                except retry_on_exceptions as e:
                    last_exception = e
                    
                    if attempt == max_attempts:
                        break
                    
                    # Calcula delay com backoff exponencial
                    delay = min(base_delay * (2 ** (attempt - 1)), max_delay)
                    
                    # Adiciona jitter
                    if jitter:
                        delay = delay * (0.5 + random.random())
                    
                    time.sleep(delay)
            
            raise last_exception
        
        return wrapper
    return decorator

import asyncio
from typing import Dict, List, Optional, Set
from datetime import datetime
import heapq
from collections import defaultdict
from contextlib import asynccontextmanager

class PriorityTaskQueue:
    """Fila de tarefas com prioridade e suporte a múltiplos agentes"""
    
    def __init__(self, max_size: int = 1000):
        self.max_size = max_size
        self.queues: Dict[str, List] = defaultdict(list)  # agent_type -> heap
        self.task_map: Dict[str, Task] = {}  # task_id -> Task
        self.locks: Dict[str, asyncio.Lock] = defaultdict(asyncio.Lock)
        self.condition = asyncio.Condition()
        self.logger = StructuredLogger("task_queue")
    
    async def enqueue(self, task: Task) -> bool:
        """Adiciona tarefa à fila"""
        async with self.condition:
            if len(self.task_map) >= self.max_size:
                self.logger.warning(f"Queue full, rejecting task {task.task_id}")
                return False
            
            # Adiciona à fila de prioridade
            heapq.heappush(self.queues[task.agent_type], 
                          (-task.priority.value, task.created_at.timestamp(), task.task_id))
            self.task_map[task.task_id] = task
            
            self.logger.info(f"Task {task.task_id} enqueued for {task.agent_type}", 
                           task_id=task.task_id, agent_type=task.agent_type, 
                           priority=task.priority.value)
            
            # Notifica workers que há nova tarefa
            self.condition.notify_all()
            return True
    
    async def dequeue(self, agent_type: str) -> Optional[Task]:
        """Remove e retorna a próxima tarefa para o tipo de agente"""
        async with self.condition:
            while not self.queues.get(agent_type):
                # Aguarda até haver tarefas
                await self.condition.wait()
            
            # Pega a tarefa com maior prioridade (menor valor negativo = maior prioridade)
            _, _, task_id = heapq.heappop(self.queues[agent_type])
            task = self.task_map.pop(task_id, None)
            
            if task:
                self.logger.info(f"Task {task_id} dequeued for {agent_type}",
                               task_id=task_id, agent_type=agent_type)
            
            return task
    
    async def get_task(self, task_id: str) -> Optional[Task]:
        """Obtém tarefa pelo ID sem removê-la"""
        return self.task_map.get(task_id)
    
    async def remove_task(self, task_id: str) -> bool:
        """Remove tarefa da fila"""
        task = self.task_map.get(task_id)
        if not task:
            return False
        
        # Precisa remover da heap - O(n) infelizmente
        queue = self.queues[task.agent_type]
        for i, (priority, timestamp, t_id) in enumerate(queue):
            if t_id == task_id:
                queue.pop(i)
                heapq.heapify(queue)  # Re-heapify após remoção
                break
        
        del self.task_map[task_id]
        self.logger.info(f"Task {task_id} removed from queue", task_id=task_id)
        return True
    
    async def get_queue_stats(self) -> Dict[str, int]:
        """Retorna estatísticas das filas"""
        stats = {}
        for agent_type, queue in self.queues.items():
            stats[agent_type] = len(queue)
        return stats
    
    async def cleanup_expired(self, max_age_seconds: int = 3600):
        """Remove tarefas expiradas da fila"""
        now = datetime.utcnow()
        expired_tasks = []
        
        for task_id, task in list(self.task_map.items()):
            age = (now - task.created_at).total_seconds()
            if age > max_age_seconds:
                expired_tasks.append(task_id)
        
        for task_id in expired_tasks:
            await self.remove_task(task_id)
        
        if expired_tasks:
            self.logger.warning(f"Cleaned up {len(expired_tasks)} expired tasks")

class PerformanceMetrics {
  constructor() {
    this.metrics = {
      agentCalls: {},
      llmCalls: {},
      errors: {},
      latencies: {}
    };
    
    // Inicializar contadores para cada agente
    const agents = ['ai-ops', 'llm-specialist', 'llm-gateway', 'qa-engineer', 'security', 'sre', 'pm'];
    agents.forEach(agent => {
      this.metrics.agentCalls[agent] = {
        total: 0,
        success: 0,
        failed: 0,
        avgLatency: 0,
        lastCall: null
      };
    });
  }

  recordAgentCall(agent, success, latency, metadata = {}) {
    if (!this.metrics.agentCalls[agent]) {
      this.metrics.agentCalls[agent] = {
        total: 0,
        success: 0,
        failed: 0,
        avgLatency: 0,
        lastCall: new Date().toISOString()
      };
    }

    const stats = this.metrics.agentCalls[agent];
    stats.total++;
    stats.lastCall = new Date().toISOString();
    
    if (success) {
      stats.success++;
    } else {
      stats.failed++;
    }

    // Calcular média móvel da latência
    stats.avgLatency = (stats.avgLatency * (stats.total - 1) + latency) / stats.total;

    // Log da métrica
    aiOpsLogger.info('Agent call recorded', {
      agent,
      success,
      latency,
      totalCalls: stats.total,
      successRate: (stats.success / stats.total * 100).toFixed(2) + '%',
      ...metadata
    });

    return stats;
  }

  recordLLMCall(provider, model, tokensIn, tokensOut, latency, cost, success) {
    const key = `${provider}:${model}`;
    
    if (!this.metrics.llmCalls[key]) {
      this.metrics.llmCalls[key] = {
        total: 0,
        success: 0,
        failed: 0,
        totalTokensIn: 0,
        totalTokensOut: 0,
        totalCost: 0,
        avgLatency: 0,
        lastCall: null
      };
    }

    const stats = this.metrics.llmCalls[key];
    stats.total++;
    stats.lastCall = new Date().toISOString();
    
    if (success) {
      stats.success++;
      stats.totalTokensIn += tokensIn;
      stats.totalTokensOut += tokensOut;
      stats.totalCost += cost;
      stats.avgLatency = (stats.avgLatency * (stats.success - 1) + latency) / stats.success;
    } else {
      stats.failed++;
    }

    // Log detalhado para AI Ops
    aiOpsLogger.info('LLM call recorded', {
      provider,
      model,
      tokensIn,
      tokensOut,
      latency,
      cost,
      success,
      totalCalls: stats.total,
      successRate: (stats.success / stats.total * 100).toFixed(2) + '%',
      avgTokensPerCall: stats.totalTokensIn / stats.success || 0,
      totalCostToDate: stats.totalCost
    });

    return stats;
  }

  recordError(agent, errorType, errorMessage, stackTrace = null) {
    const timestamp = new Date().toISOString();
    const errorKey = `${agent}:${errorType}`;
    
    if (!this.metrics.errors[errorKey]) {
      this.metrics.errors[errorKey] = {
        count: 0,
        firstOccurrence: timestamp,
        lastOccurrence: timestamp,
        samples: []
      };
    }

    const errorStats = this.metrics.errors[errorKey];
    errorStats.count++;
    errorStats.lastOccurrence = timestamp;
    
    // Manter apenas as últimas 5 amostras de erro
    if (errorStats.samples.length >= 5) {
      errorStats.samples.shift();
    }
    errorStats.samples.push({
      timestamp,
      message: errorMessage,
      stack: stackTrace
    });

    // Log de erro estruturado
    mainLogger.error(`Error recorded for agent ${agent}`, {
      agent,
      errorType,
      errorMessage,
      stackTrace,
      occurrenceCount: errorStats.count,
      firstSeen: errorStats.firstOccurrence
    });

    return errorStats;
  }

        cost: stats.totalCost
      };

      report.llmProviders[provider].totalCalls += stats.total;
      report.llmProviders[provider].totalCost += stats.totalCost;
    });

    // Calcular taxas de sucesso por provedor
    Object.keys(report.llmProviders).forEach(provider => {
      const providerData = report.llmProviders[provider];
      const totalSuccess = Object.values(providerData.models).reduce((sum, model) => sum + model.success, 0);
      providerData.successRate = (totalSuccess / providerData.totalCalls * 100).toFixed(2) + '%';
    });

    // Top 5 erros
    report.topErrors = Object.entries(this.metrics.errors)
      .sort((a, b) => b[1].count - a[1].count)
      .slice(0, 5)
      .map(([errorKey, stats]) => ({
        error: errorKey,
        count: stats.count,
        firstSeen: stats.firstOccurrence,
        lastSeen: stats.lastOccurrence
      }));

    // Gerar recomendações baseadas em métricas
    if (totalCost > 1000) { // Exemplo: custo alto
      report.recommendations.push({
        type: 'cost_optimization',
        priority: 'high',
        message: `Custo total de LLM: $${totalCost.toFixed(2)}. Considere otimizar prompts ou usar modelos mais econômicos.`,
        suggestedAction: 'Review prompts com Especialista LLM'
      });
    }

    // Log do relatório
    aiOpsLogger.info('Weekly metrics report generated', report);

    return report;
  }
