# ADR-001: Arquitetura V12 - Microserviços Event-Driven com Polyglot Persistence

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista Backend

## CONTEXTO:
O DIYAPP evoluiu para uma plataforma complexa com múltiplos domínios: gerenciamento de projetos, execução de agentes IA, orquestração de workflows, monitoramento em tempo real e sistema de aprendizado contínuo. A arquitetura monolítica atual apresenta:

1. **Acoplamento excessivo:** Mudanças no módulo de agentes impactam o sistema de monitoramento
2. **Dificuldade de escalar:** Todo o sistema escala uniformemente, mesmo que apenas um módulo precise de mais recursos
3. **Deploys arriscados:** Uma falha em módulo secundário derruba todo o sistema
4. **Stack tecnológica limitada:** Impossível usar a melhor ferramenta para cada domínio

**Forças em jogo:**
- Necessidade de 100% de autonomia e estabilidade
- Time distribuído trabalhando em paralelo (Modo Hive)
- Requisitos diferentes por domínio (latência, throughput, consistência)
- Manutenibilidade a longo prazo

## DECISÃO:
Adotar arquitetura de microserviços event-driven com os seguintes princípios:

1. **Boundaries por domínio:** Cada bounded context vira um microserviço independente
2. **Comunicação assíncrona:** Event-driven via Apache Kafka para desacoplamento
3. **Polyglot persistence:** Cada serviço escolhe seu banco de dados ideal
4. **API Gateway unificado:** Único ponto de entrada com rate limiting e auth
5. **Service Mesh:** Istio para observabilidade e resiliência
6. **Deploy independente:** Cada serviço versionado e deployado separadamente

## OPÇÕES CONSIDERADAS:

### Opção A: Monólito Modular (Evolução incremental)
**Prós:**
- Menor complexidade operacional inicial
- Transações ACID fáceis
- Debugging simplificado
- Deploy único

**Contras:**
- Acoplamento técnico permanece
- Escalabilidade limitada
- Stack única obrigatória
- Time único de deploy (bottleneck)

### Opção B: Microserviços REST (Síncrono)
**Prós:**
- Independência de deploy
- Escalabilidade granular
- Polyglot possível

**Contras:**
- Acoplamento temporal (falha em cascata)
- Latência acumulada
- Complexidade de versionamento de APIs
- Circuit breaking necessário

### Opção C: Microserviços Event-Driven (Escolhida)
**Prós:**
- Desacoplamento máximo
- Resiliência a falhas
- Escalabilidade independente
- Polyglot persistence natural
- Event sourcing possível

**Contras:**
- Maior complexidade operacional
- Eventual consistency
- Debugging distribuído difícil
- Necessidade de idempotência

**Justificativa:** A natureza assíncrona do DIYAPP (agentes executando tarefas, aprendizado contínuo, monitoramento) se alinha perfeitamente com event-driven. O Modo Hive beneficia-se do desacoplamento para trabalhar em paralelo sem conflitos.

## CONSEQUÊNCIAS:

### Positivas:
1. **Autonomia total:** Squads podem desenvolver, testar e deployar independentemente
2. **Escalabilidade granular:** Serviços críticos (orquestração) escalam separadamente
3. **Resiliência:** Falhas isoladas não derrubam o sistema
4. **Flexibilidade tecnológica:** Cada domínio usa a stack ideal
5. **Paralelismo:** Modo Hive otimizado com menos conflitos de merge

### Negativas (trade-offs aceitos):
1. **Complexidade distribuída:** Necessidade de observabilidade robusta
2. **Consistência eventual:** Sistemas devem lidar com estados intermediários
3. **Operações complexas:** Múltiplos bancos, múltiplos deploys
4. **Latência adicional:** Processamento assíncrono vs síncrono

### Riscos e Mitigações:
| Risco | Mitigação |
|-------|-----------|
| Falha no message broker | Cluster Kafka com replicação 3, monitoramento 24/7 |
| Eventos perdidos | Implementar dead letter queues e retry policies |
| Debugging difícil | Distributed tracing (Jaeger) + logs centralizados (ELK) |
| Versionamento de eventos | Schema Registry + backward compatibility |
| Orquestração complexa | Saga pattern com compensação |

```typescript
// Event Schema Padrão
interface DIYEvent {
  event_id: string; // UUID v4
  event_type: string; // ex: 'agent.execution.started'
  timestamp: string; // ISO 8601
  source_service: string;
  correlation_id: string; // Para tracing
  payload: any; // Schema específico por evento
  metadata: {
    version: string; // Schema version
    priority: 'low' | 'medium' | 'high';
    ttl_seconds?: number;
  };
}

// Exemplo de Producer
class EventProducer {
  async publish(event: DIYEvent): Promise<void> {
    await kafka.producer.send({
      topic: this.getTopic(event.event_type),
      messages: [{
        key: event.correlation_id,
        value: JSON.stringify(event),
        headers: {
          'event-type': event.event_type,
          'source': event.source_service
        }
      }]
    });
  }
}

// Exemplo de Consumer com idempotência
class EventConsumer {
  @KafkaListener('agent.events')
  async handleAgentEvent(message: KafkaMessage): Promise<void> {
    const event: DIYEvent = JSON.parse(message.value.toString());
    
    // Idempotency check
    const processed = await this.idempotencyStore.exists(event.event_id);
    if (processed) {
      logger.warn(`Event ${event.event_id} already processed`);
      return;
    }
    
    try {
      await this.processEvent(event);
      await this.idempotencyStore.store(event.event_id);
    } catch (error) {
      // Dead letter queue para retry
      await this.dlqProducer.send(event);
    }
  }
}
```

## 6. Circuit Breaker (src/utils/circuitBreaker.js)

```javascript
const CircuitBreaker = require('circuit-breaker-js');
const logger = require('./logger');

class ResilientService {
  constructor(serviceName, options = {}) {
    this.serviceName = serviceName;
    
    this.circuitBreaker = new CircuitBreaker({
      windowDuration: options.windowDuration || 10000,
      numBuckets: options.numBuckets || 10,
      timeoutDuration: options.timeoutDuration || 10000,
      errorThreshold: options.errorThreshold || 50,
      volumeThreshold: options.volumeThreshold || 5,
      
      onCircuitOpen: (metrics) => {
        logger.warn({
          service: serviceName,
          event: 'circuit_opened',
          metrics,
          failureRate: metrics.failureRate,
        }, `Circuit breaker opened for ${serviceName}`);
      },
      
      onCircuitClose: (metrics) => {
        logger.info({
          service: serviceName,
          event: 'circuit_closed',
          metrics,
        }, `Circuit breaker closed for ${serviceName}`);
      },
    });
    
    this.fallback = options.fallback || (() => {
      throw new Error(`Service ${serviceName} unavailable - fallback not implemented`);
    });
  }

  async execute(operation, ...args) {
    const startTime = Date.now();
    const correlationId = args[0]?.correlationId || `cb-${Date.now()}-${Math.random().toString(36).substr(2, 9)}`;
    
    try {
      // Check if circuit is open
      if (!this.circuitBreaker.allowRequest()) {
        logger.warn({
          service: this.serviceName,
          event: 'circuit_open_short_circuit',
          correlationId,
        }, `Circuit open for ${this.serviceName}, using fallback`);
        
        return await this.fallback(...args);
      }
      
      // Execute with timeout
      const timeout = this.circuitBreaker.timeoutDuration;
      const timeoutPromise = new Promise((_, reject) => {
        setTimeout(() => reject(new Error(`Timeout after ${timeout}ms`)), timeout);
      });
      
      const result = await Promise.race([
        operation(...args),
        timeoutPromise,
      ]);
      
      const duration = Date.now() - startTime;
      
      // Record success
      this.circuitBreaker.success(duration);
      
      logger.debug({
        service: this.serviceName,
        event: 'circuit_success',
        correlationId,
        duration,
      }, `${this.serviceName} succeeded in ${duration}ms`);
      
      return result;
      
    } catch (error) {
      const duration = Date.now() - startTime;
      
      // Record failure
      this.circuitBreaker.failure(duration);
      
      logger.error({
        service: this.serviceName,
        event: 'circuit_failure',
        correlationId,
        duration,
        error: {
          message: error.message,
          code: error.code,
          stack: error.stack,
        },
      }, `${this.serviceName} failed in ${duration}ms: ${error.message}`);
      
      // Use fallback
      try {
        return await this.fallback(...args);
      } catch (fallbackError) {
        logger.error({
          service: this.serviceName,
          event: 'fallback_failure',
          correlationId,
          error: fallbackError.message,
        }, `Fallback also failed for ${this.serviceName}`);
        
        throw new Error(`Service ${this.serviceName} unavailable and fallback failed`);
      }
    }
  }

  getMetrics() {
    return {
      service: this.serviceName,
      state: this.circuitBreaker.isOpen() ? 'OPEN' : 'CLOSED',
      metrics: this.circuitBreaker.getStats(),
    };
  }
}
```

```javascript
const { Pool } = require('pg');
const logger = require('../utils/logger');
const config = require('./index');

class Database {
  constructor() {
    this.pool = new Pool(config.database);
    
    // Log connection events
    this.pool.on('connect', (client) => {
      logger.debug('New database client connected');
    });
    
    this.pool.on('error', (err, client) => {
      logger.error({
        error: err.message,
        event: 'database_pool_error',
      }, 'Unexpected error on idle database client');
    });
    
    // Test connection on startup
    this.testConnection();
  }

  async testConnection() {
    try {
      const client = await this.pool.connect();
      const result = await client.query('SELECT NOW() as current_time, version() as version');
      logger.info({
        event: 'database_connected',
        time: result.rows[0].current_time,
        version: result.rows[0].version,
      }, 'Database connection established');
      client.release();
    } catch (error) {
      logger.error({
        error: error.message,
        event: 'database_connection_failed',
      }, 'Failed to connect to database');
      process.exit(1);
    }
  }

  async query(text, params) {
    const start = Date.now();
    const correlationId = `db-${Date.now()}-${Math.random().toString(36).substr(2, 9)}`;
    
    try {
      logger.debug({
        event: 'database_query_start',
        correlationId,
        query: text.substring(0, 200), // Log first 200 chars
        params: this.maskSensitiveParams(params),
      });
      
      const result = await this.pool.query(text, params);
      const duration = Date.now() - start;
      
      logger.debug({
        event: 'database_query_complete',
        correlationId,
        duration,
        rowCount: result.rowCount,
      });
      
      return result;
    } catch (error) {
      const duration = Date.now() - start;
      
      logger.error({
        event: 'database_query_error',
        correlationId,
        duration,
        error: {
          message: error.message,
          code: error.code,
          detail: error.detail,
          hint: error.hint,
        },
        query: text.substring(0, 200),
        params: this.maskSensitiveParams(params),
      });
      
      throw error;
    }
  }

  maskSensitiveParams(params) {
    if (!params) return params;
    
    return params.map(param => {
      if (typeof param === 'string') {
        // Mask passwords, tokens, etc.
        if (param.toLowerCase().includes('password') || 
            param.toLowerCase().includes('token') || 
            param.toLowerCase().includes('secret') ||
            param.length > 50) { // Likely JWT or hash
          return '[MASKED]';
        }
      }
      return param;
    });
  }

  async transaction(callback) {
    const client = await this.pool.connect();
    
    try {
      await client.query('BEGIN');
      const result = await callback(client);
      await client.query('COMMIT');
      return result;
    } catch (error) {
      await client.query('ROLLBACK');
      throw error;
    } finally {
      client.release();
    }
  }

  async close() {
    await this.pool.end();
    logger.info('Database pool closed');
  }
}

module.exports = new Database();
```

```typescript
import { create } from 'zustand'
import { persist } from 'zustand/middleware'

interface AppState {
  // Theme
  theme: 'light' | 'dark'
  toggleTheme: () => void
  setTheme: (theme: 'light' | 'dark') => void
  
  // Loading states
  isLoading: boolean
  setIsLoading: (loading: boolean) => void
  
  // Notifications
  notifications: Array<{
    id: string
    type: 'success' | 'error' | 'warning' | 'info'
    message: string
    timestamp: Date
  }>
  addNotification: (notification: Omit<AppState['notifications'][0], 'id' | 'timestamp'>) => void
  removeNotification: (id: string) => void
  clearNotifications: () => void
  
  // Sidebar
  isSidebarOpen: boolean
  toggleSidebar: () => void
  setSidebarOpen: (open: boolean) => void
  
  // User preferences
  preferences: {
    reducedMotion: boolean
    highContrast: boolean
    fontSize: 'normal' | 'large' | 'xlarge'
  }
  updatePreferences: (preferences: Partial<AppState['preferences']>) => void
}

export const useAppStore = create<AppState>()(
  persist(
    (set, get) => ({
      // Theme
      theme: 'light',
      toggleTheme: () => set((state) => ({ theme: state.theme === 'light' ? 'dark' : 'light' })),
      setTheme: (theme) => set({ theme }),
      
      // Loading states
      isLoading: false,
      setIsLoading: (loading) => set({ isLoading: loading }),
      
      // Notifications
      notifications: [],
      addNotification: (notification) => {
        const id = Math.random().toString(36).substring(2, 9)
        const newNotification = {
          ...notification,
          id,
          timestamp: new Date(),
        }
        set((state) => ({
          notifications: [...state.notifications, newNotification].slice(-5), // Keep last 5
        }))
        
        // Auto-remove after 5 seconds
        setTimeout(() => {
          get().removeNotification(id)
        }, 5000)
      },
      removeNotification: (id) => {
        set((state) => ({
          notifications: state.notifications.filter((n) => n.id !== id),
        }))
      },
      clearNotifications: () => set({ notifications: [] }),
      
      // Sidebar
      isSidebarOpen: true,
      toggleSidebar: () => set((state) => ({ isSidebarOpen: !state.isSidebarOpen })),
      setSidebarOpen: (open) => set({ isSidebarOpen: open }),
      
      // User preferences
      preferences: {
        reducedMotion: false,
        highContrast: false,
        fontSize: 'normal',
      },
      updatePreferences: (preferences) =>
        set((state) => ({
          preferences: { ...state.preferences, ...preferences },
        })),
    }),
    {
      name: 'diyapp-app-store',
      partialize: (state) => ({
        theme: state.theme,
        preferences: state.preferences,
      }),
    }
  )
)
```
