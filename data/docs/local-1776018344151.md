# DIYAPP Evolution - V12 Core - Arquitetura Técnica

## ADR-001: Arquitetura V12 - Decisões Estruturais

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista Backend

### CONTEXTO:
O DIYAPP evoluiu para uma plataforma complexa com múltiplos serviços. A arquitetura atual (V11) apresenta:
- Acoplamento excessivo entre módulos
- Dificuldade em escalar componentes individualmente
- CI/CD monolítico que atrasa deploys
- Débito técnico acumulado em testes e documentação
- Necessidade de suporte a IA nativo na arquitetura

### DECISÃO:
Adotar arquitetura de microsserviços com os seguintes padrões:
1. **Backend**: Node.js 20+ com Express.js e TypeScript
2. **Frontend**: React 18 com Vite e TypeScript
3. **Banco de Dados**: PostgreSQL (dados relacionais) + Redis (cache/sessões)
4. **Mensageria**: RabbitMQ para comunicação assíncrona
5. **Containerização**: Docker + Docker Compose para desenvolvimento
6. **Orquestração**: Kubernetes para produção
7. **Monitoramento**: Prometheus + Grafana + ELK Stack

### OPÇÕES CONSIDERADAS:

**Opção A: Monolito Evolutivo (Status Quo)**
- Prós: Menor complexidade operacional, deploy único
- Contras: Escalabilidade limitada, acoplamento alto, risco de single point of failure

**Opção B: Microsserviços Completos**
- Prós: Escalabilidade independente, resiliência, deploy contínuo
- Contras: Complexidade operacional maior, necessidade de SRE dedicado

**Opção C: Arquitetura Híbrida (Escolhida)**
- Prós: Balanceamento entre agilidade e estabilidade, migração gradual
- Contras: Gerenciamento de duas arquiteturas durante transição

**Opção escolhida: C** - Justificativa: Permite migração controlada, mantendo estabilidade do sistema atual enquanto novos serviços são desenvolvidos com arquitetura moderna.

### CONSEQUÊNCIAS:
**Positivas:**
- Escalabilidade horizontal por serviço
- Deploys independentes
- Resiliência através de circuit breakers
- Observabilidade granular

**Negativas:**
- Complexidade de debugging distribuído
- Overhead de comunicação entre serviços
- Necessidade de equipe SRE/DevOps

**Riscos:**
- Latência em chamadas entre serviços (mitigar com cache e design assíncrono)
- Consistência de dados distribuídos (mitigar com sagas e compensação)
- Monitoramento distribuído (mitigar com OpenTelemetry)

### 3. Convenções de Nomenclatura

- **Classes**: PascalCase (ex: `UserService`)
- **Interfaces**: PascalCase com prefixo I (ex: `IUser`)
- **Variáveis/funções**: camelCase (ex: `getUserById`)
- **Constantes**: UPPER_SNAKE_CASE (ex: `MAX_RETRY_COUNT`)
- **Arquivos**: kebab-case (ex: `user-controller.ts`)
- **Pastas**: kebab-case (ex: `user-management`)

## CI/CD Pipeline V12

### 1. GitHub Actions Workflow

```yaml
# .github/workflows/ci.yml
name: CI Pipeline

on:
  push:
    branches: [main, develop]
  pull_request:
    branches: [main, develop]

jobs:
  lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
        with:
          node-version: '20'
      - run: npm ci
      - run: npm run lint

  test:
    runs-on: ubuntu-latest
    needs: lint
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
      - run: npm ci
      - run: npm run test:coverage
      - uses: codecov/codecov-action@v3

  build:
    runs-on: ubuntu-latest
    needs: test
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
      - run: npm ci
      - run: npm run build
      - uses: actions/upload-artifact@v3
        with:
          name: build-artifacts
          path: dist/

  security:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Run Snyk to check for vulnerabilities
        uses: snyk/actions/node@master
        env:
          SNYK_TOKEN: ${{ secrets.SNYK_TOKEN }}
```

## Guia de Contribuição

### 1. Setup do Ambiente

```bash
# 1. Clone o repositório
git clone https://github.com/diyapp/diyapp-v12.git
cd diyapp-v12

# 2. Instale dependências
npm install

# 3. Configure variáveis de ambiente
cp .env.example .env
# Edite .env com suas configurações

# 4. Inicie serviços de infraestrutura
docker-compose up -d postgres redis rabbitmq

# 5. Execute migrações do banco
npm run db:migrate

# 6. Inicie os serviços
npm run dev
```

### 2. Fluxo de Desenvolvimento

```bash
# 1. Crie uma branch a partir de develop
git checkout develop
git pull origin develop
git checkout -b feature/meu-recurso

# 2. Desenvolva com testes
# Escreva código + testes unitários
npm run test:watch

# 3. Execute lint e formatação
npm run lint
npm run format

# 4. Commit seguindo conventional commits
git add .
git commit -m "feat: add user profile management"

# 5. Push e crie PR
git push origin feature/meu-recurso
# Crie PR no GitHub para develop
```

### 3. Checklist de PR

Antes de submeter um PR, verifique:

- [ ] Código segue padrões ESLint/Prettier
- [ ] Testes unitários com cobertura ≥ 80%
- [ ] Testes de integração para APIs
- [ ] Documentação atualizada
- [ ] Sem credenciais hardcoded
- [ ] Tratamento de erros adequado
- [ ] Logs estruturados
- [ ] Métricas expostas (Prometheus)
- [ ] Versionamento de API (se aplicável)

### 4. Review Process

1. **Tech Lead Review** (SLA: 8h)
   - Análise arquitetural
   - Verificação de segurança
   - Avaliação de performance

2. **Code Review pelos pares**
   - 2 aprovações necessárias
   - Foco em legibilidade e manutenibilidade

3. **CI Pipeline**
   - Todos os testes passam
   - Build bem-sucedido
   - Scan de segurança limpo

4. **Merge para develop**
   - Squash commits
   - Delete branch

```javascript
const redis = require('redis');
const winston = require('winston');

const logger = winston.createLogger({
  level: process.env.LOG_LEVEL || 'info',
  format: winston.format.json(),
  transports: [
    new winston.transports.File({ filename: process.env.LOG_FILE || 'logs/app.log' })
  ]
});

class RedisClient {
  constructor() {
    this.client = null;
    this.isConnected = false;
    this.circuitOpen = false;
    this.failureCount = 0;
    this.circuitResetTimeout = null;
  }

  async connect() {
    if (this.circuitOpen) {
      logger.warn('Circuit breaker open for Redis. Using fallback.');
      return false;
    }

    try {
      this.client = redis.createClient({
        url: `redis://${process.env.REDIS_HOST}:${process.env.REDIS_PORT}`,
        password: process.env.REDIS_PASSWORD || undefined,
        socket: {
          reconnectStrategy: (retries) => {
            if (retries > 3) {
              this.tripCircuitBreaker();
              return new Error('Max reconnection attempts reached');
            }
            return Math.min(retries * 100, 3000);
          }
        }
      });

      this.client.on('error', (err) => {
        logger.error('Redis Client Error:', err.message);
        this.failureCount++;
        
        if (this.failureCount >= 3) {
          this.tripCircuitBreaker();
        }
      });

      this.client.on('connect', () => {
        logger.info('Redis connected successfully');
        this.isConnected = true;
        this.failureCount = 0;
        this.resetCircuitBreaker();
      });

      await this.client.connect();
      return true;
    } catch (error) {
      logger.error('Redis connection failed:', error.message);
      this.failureCount++;
      
      if (this.failureCount >= 3) {
        this.tripCircuitBreaker();
      }
      
      return false;
    }
  }

  tripCircuitBreaker() {
    this.circuitOpen = true;
    logger.warn('Redis circuit breaker tripped');
    
    if (this.circuitResetTimeout) {
      clearTimeout(this.circuitResetTimeout);
    }
    
    this.circuitResetTimeout = setTimeout(() => {
      this.circuitOpen = false;
      this.failureCount = 0;
      logger.info('Redis circuit breaker reset');
    }, 30000); // Reset after 30 seconds
  }

  resetCircuitBreaker() {
    if (this.circuitOpen) {
      this.circuitOpen = false;
      this.failureCount = 0;
      logger.info('Redis circuit breaker reset');
    }
  }

  async get(key) {
    if (!this.isConnected || this.circuitOpen) {
      return null;
    }

    try {
      return await this.client.get(key);
    } catch (error) {
      logger.error('Redis get error:', error.message);
      return null;
    }
  }

  async set(key, value, ttl = 3600) {
    if (!this.isConnected || this.circuitOpen) {
      return false;
    }

    try {
      await this.client.set(key, value, { EX: ttl });
      return true;
    } catch (error) {
      logger.error('Redis set error:', error.message);
      return false;
    }
  }

  async del(key) {
    if (!this.isConnected || this.circuitOpen) {
      return false;
    }

    try {
      await this.client.del(key);
      return true;
    } catch (error) {
      logger.error('Redis del error:', error.message);
      return false;
    }
  }
}

module.exports = new RedisClient();
```

import React from 'react';
import { colors, spacing, typography } from '../../../design-system/tokens';

export type ButtonVariant = 'primary' | 'secondary' | 'outline' | 'ghost' | 'danger';
export type ButtonSize = 'sm' | 'md' | 'lg';

interface ButtonProps extends React.ButtonHTMLAttributes<HTMLButtonElement> {
  variant?: ButtonVariant;
  size?: ButtonSize;
  isLoading?: boolean;
  leftIcon?: React.ReactNode;
  rightIcon?: React.ReactNode;
  fullWidth?: boolean;
}

const Button: React.FC<ButtonProps> = ({
  children,
  variant = 'primary',
  size = 'md',
  isLoading = false,
  leftIcon,
  rightIcon,
  fullWidth = false,
  disabled,
  className = '',
  ...props
}) => {
  const baseStyles = {
    display: 'inline-flex',
    alignItems: 'center',
    justifyContent: 'center',
    fontFamily: typography.fontFamily.sans,
    fontWeight: typography.fontWeight.semibold,
    borderRadius: '0.375rem',
    border: '1px solid transparent',
    transition: 'all 150ms cubic-bezier(0.4, 0, 0.2, 1)',
    cursor: disabled || isLoading ? 'not-allowed' : 'pointer',
    opacity: disabled ? 0.6 : 1,
    width: fullWidth ? '100%' : 'auto',
  };

  const sizeStyles = {
    sm: {
      fontSize: typography.fontSize.sm,
      padding: `${spacing[1]} ${spacing[3]}`,
      gap: spacing[1],
    },
    md: {
      fontSize: typography.fontSize.base,
      padding: `${spacing[2]} ${spacing[4]}`,
      gap: spacing[2],
    },
    lg: {
      fontSize: typography.fontSize.lg,
      padding: `${spacing[3]} ${spacing[6]}`,
      gap: spacing[3],
    },
  };

  const variantStyles = {
    primary: {
      backgroundColor: colors.primary[600],
      color: colors.background.primary,
      borderColor: colors.primary[600],
      '&:hover:not(:disabled)': {
        backgroundColor: colors.primary[700],
        borderColor: colors.primary[700],
      },
      '&:focus': {
        outline: 'none',
        boxShadow: `0 0 0 3px ${colors.primary[200]}`,
      },
    },
    secondary: {
      backgroundColor: colors.neutral[200],
      color: colors.text.primary,
      borderColor: colors.neutral[200],
      '&:hover:not(:disabled)': {
        backgroundColor: colors.neutral[300],
        borderColor: colors.neutral[300],
      },
    },
    outline: {
      backgroundColor: 'transparent',
      color: colors.primary[600],
      borderColor: colors.primary[600],
      '&:hover:not(:disabled)': {
        backgroundColor: colors.primary[50],
      },
    },
    ghost: {
      backgroundColor: 'transparent',
      color: colors.text.primary,
      borderColor: 'transparent',
      '&:hover:not(:disabled)': {
        backgroundColor: colors.neutral[100],
      },
    },
    danger: {
      backgroundColor: colors.error,
      color: colors.background.primary,
      borderColor: colors.error,
      '&:hover:not(:disabled)': {
        backgroundColor: '#dc2626',
        borderColor: '#dc2626',
      },
    },
  };

  const styles = {
    ...baseStyles,
    ...sizeStyles[size],
    ...variantStyles[variant],
  };

  return (
    <button
      style={styles}
      disabled={disabled || isLoading}
      className={`button ${className}`}
      aria-busy={isLoading}
      {...props}
    >
      {isLoading && (
        <span className="button__loader" aria-hidden="true">
          <svg
            width="16"
            height="16"
            viewBox="0 0 24 24"
            fill="none"
            stroke="currentColor"
            strokeWidth="2"
            strokeLinecap="round"
            strokeLinejoin="round"

            <path d="M21 12a9 9 0 1 1-6.219-8.56" />
          </svg>
        </span>
      )}

      <span className="button__content">{children}</span>
      {!isLoading && rightIcon && <span className="button__right-icon">{rightIcon}</span>}
    </button>
  );
};

export default Button;

// src/app/store/slices/authSlice.ts
import { createSlice, PayloadAction } from '@reduxjs/toolkit';

interface User {
  id: string;
  email: string;
  name: string;
  avatar?: string;
  role: 'user' | 'admin';
}

interface AuthState {
  user: User | null;
  token: string | null;
  isLoading: boolean;
  error: string | null;
}

const initialState: AuthState = {
  user: null,
  token: localStorage.getItem('auth_token'),
  isLoading: false,
  error: null,
};

const authSlice = createSlice({
  name: 'auth',
  initialState,
  reducers: {
    setCredentials: (
      state,
      action: PayloadAction<{ user: User; token: string }>
    ) => {
      state.user = action.payload.user;
      state.token = action.payload.token;
      localStorage.setItem('auth_token', action.payload.token);
    },
    logout: (state) => {
      state.user = null;
      state.token = null;
      localStorage.removeItem('auth_token');
    },
    setLoading: (state, action: PayloadAction<boolean>) => {
      state.isLoading = action.payload;
    },
    setError: (state, action: PayloadAction<string | null>) => {
      state.error = action.payload;
    },
  },
});

export const { setCredentials, logout, setLoading, setError } = authSlice.actions;
export default authSlice.reducer;

import React from 'react';
import { Navigate, useLocation } from 'react-router-dom';
import { useSelector } from '../store/hooks';

interface ProtectedRouteProps {
  children: React.ReactNode;
  roles?: string[];
}

const ProtectedRoute: React.FC<ProtectedRouteProps> = ({ children, roles }) => {
  const { user, token } = useSelector((state) => state.auth);
  const location = useLocation();

  if (!token) {
    return <Navigate to="/login" state={{ from: location }} replace />;
  }

  if (roles && user && !roles.includes(user.role)) {
    return <Navigate to="/" replace />;
  }

  return <>{children}</>;
};

```sql
        EXTRACT(QUARTER FROM date) AS quarter,
        EXTRACT(MONTH FROM date) AS month,
        EXTRACT(WEEK FROM date) AS week,
        EXTRACT(DAY FROM date) AS day,
        EXTRACT(DOW FROM date) AS day_of_week,
        EXTRACT(DOY FROM date) AS day_of_year,
        TO_CHAR(date, 'Month') AS month_name,
        TO_CHAR(date, 'Day') AS day_name,
        CASE WHEN EXTRACT(DOW FROM date) IN (0, 6) THEN TRUE ELSE FALSE END AS is_weekend,
        CASE WHEN date = CURRENT_DATE THEN TRUE ELSE FALSE END AS is_current_date,
        CASE WHEN date = CURRENT_DATE - INTERVAL '1 day' THEN TRUE ELSE FALSE END AS is_yesterday,
        CASE WHEN date >= DATE_TRUNC('month', CURRENT_DATE) THEN TRUE ELSE FALSE END AS is_current_month,
        CASE WHEN date >= DATE_TRUNC('year', CURRENT_DATE) THEN TRUE ELSE FALSE END AS is_current_year
```

5. **Logs de Auditoria**: Registros de acesso e operações

### Análise STRIDE:

#### **Spoofing (Falsificação)**
- **Ameaça**: Ataque de força bruta em login, token JWT roubado
- **Controles**: MFA obrigatório, rate limiting, tokens com expiração curta (15min), refresh tokens rotativos
- **Risco**: Alto - mitigado com MFA e rate limiting

#### **Tampering (Manipulação)**
- **Ameaça**: Manipulação de dados em trânsito, SQL injection
- **Controles**: TLS 1.3, validação de input, prepared statements, assinatura digital de logs
- **Risco**: Médio - mitigado com validação rigorosa

#### **Repudiation (Repúdio)**
- **Ameaça**: Usuário nega ação realizada
- **Controles**: Logs imutáveis com assinatura, auditoria completa, timestamps precisos
- **Risco**: Baixo - mitigado com sistema de auditoria

#### **Information Disclosure (Divulgação)**
- **Ameaça**: Vazamento de PII, credenciais em logs
- **Controles**: Criptografia AES-256 em repouso, mascaramento em logs, controle de acesso granular
- **Risco**: Alto - mitigado com criptografia e mascaramento

#### **Denial of Service (Negação)**
- **Ameaça**: Ataque DDoS, resource exhaustion
- **Controles**: WAF, rate limiting por IP/user, circuit breakers, auto-scaling
- **Risco**: Médio - mitigado com WAF e rate limiting

#### **Elevation of Privilege (Elevação)**
- **Ameaça**: Acesso não autorizado a recursos privilegiados
- **Controles**: RBAC granular, princípio do menor privilégio, verificação de permissão por recurso
- **Risco**: Alto - mitigado com RBAC estrito

```javascript
// src/security/waf/WAFMiddleware.js
const { createHash } = require('crypto');
const { performance } = require('perf_hooks');

class WAFMiddleware {
    constructor() {
        this.blockedIPs = new Map();
        this.suspiciousPatterns = [
            /(\b)(SELECT|INSERT|UPDATE|DELETE|DROP|UNION)(\b)/gi,
            /(\b)(script|javascript|onload|onerror)(\b)/gi,
            /(\b)(eval|exec|system|shell_exec)(\b)/gi,
            /(\b)(union.*select|select.*from)(\b)/gi,
            /(\b)(alert|confirm|prompt)(\b)/gi,
            /(\b)(base64_decode|file_get_contents)(\b)/gi,
            /(\b)(<|>|"|'|&)(\b)/gi,
            /(\b)(\.\.\/|\.\.\\)(\b)/gi
        ];
        
        this.requestThreshold = 100; // requests por minuto
        this.blockDuration = 15 * 60 * 1000; // 15 minutos
    }

    async checkRequest(req, res, next) {
        const clientIP = req.headers['x-forwarded-for'] || req.socket.remoteAddress;
        const userAgent = req.headers['user-agent'] || '';
        const requestPath = req.path;
        const requestTime = Date.now();

        // Verifica se IP está bloqueado
        if (this.isIPBlocked(clientIP)) {
            this.logThreat('IP_BLOCKED', {
                ip: clientIP,
                path: requestPath,
                reason: 'IP bloqueado temporariamente'
            });
            return res.status(429).json({
                error: 'Acesso temporariamente bloqueado',
                code: 'WAF_BLOCKED'
            });
        }

        // Verifica padrões maliciosos
        const threatDetected = this.detectThreats(req);
        if (threatDetected) {
            this.blockIP(clientIP);
            this.logThreat('THREAT_DETECTED', {
                ip: clientIP,
                path: requestPath,
                userAgent,
                threat: threatDetected,
                body: this.sanitizeForLog(req.body)
            });
            return res.status(403).json({
                error: 'Requisição bloqueada por segurança',
                code: 'WAF_THREAT'
            });
        }

        // Rate limiting por IP
        if (!this.checkRateLimit(clientIP)) {
            this.blockIP(clientIP);
            return res.status(429).json({
                error: 'Muitas requisições. Tente novamente mais tarde.',
                code: 'WAF_RATE_LIMIT'
            });
        }

        next();
    }

    detectThreats(req) {
        // Verifica query parameters
        for (const [key, value] of Object.entries(req.query)) {
            if (this.isMalicious(value)) {
                return `SQLi/XSS em query param: ${key}`;
            }
        }

        // Verifica body
        if (req.body && typeof req.body === 'object') {
            const bodyStr = JSON.stringify(req.body);
            if (this.isMalicious(bodyStr)) {
                return 'SQLi/XSS no body da requisição';
            }
        }

        // Verifica headers
        for (const [header, value] of Object.entries(req.headers)) {
            if (this.isMalicious(value)) {
                return `Header malicioso: ${header}`;
            }
        }

        // Verifica path
        if (this.isMalicious(req.path)) {
            return 'Path malicioso detectado';
        }

        return null;
    }

    isMalicious(input) {
        if (!input || typeof input !== 'string') return false;
        
        const inputLower = input.toLowerCase();
        
        // Verifica padrões suspeitos
        for (const pattern of this.suspiciousPatterns) {
            if (pattern.test(inputLower)) {
                return true;
            }
        }

        // Verifica comprimento excessivo (possível buffer overflow)
        if (input.length > 10000) {
            return true;
        }

        // Verifica encoding anormal
        if (/(%[0-9a-f]{2}){10,}/i.test(input)) {
            return true;
        }

        return false;
    }

    checkRateLimit(ip) {
        const now = Date.now();
        const ipData = this.blockedIPs.get(ip) || { count: 0, firstRequest: now };
        
        // Reset contador se passou 1 minuto
        if (now - ipData.firstRequest > 60000) {
            ipData.count = 1;
            ipData.firstRequest = now;
        } else {
            ipData.count++;
        }

        this.blockedIPs.set(ip, ipData);
        return ipData.count <= this.requestThreshold;
    }

    isIPBlocked(ip) {
        const blockInfo = this.blockedIPs.get(ip);
        if (!blockInfo || !blockInfo.blockedUntil) return false;
        
        if (Date.now() > blockInfo.blockedUntil) {
            this.blockedIPs.delete(ip);
            return false;
        }
        
        return true;
    }

    blockIP(ip) {
        this.blockedIPs.set(ip, {
            blockedUntil: Date.now() + this.blockDuration,
            blockedAt: Date.now()
        });
    }

    sanitizeForLog(data) {
        if (!data) return null;
        
        const sanitized = {};
        const sensitiveFields = ['password', 'token', 'secret', 'credit_card', 'cvv', 'ssn', 'cpf'];
        
        for (const [key, value] of Object.entries(data)) {
            if (sensitiveFields.some(field => key.toLowerCase().includes(field))) {
                sanitized[key] = '***REDACTED***';
            } else if (typeof value === 'string' && value.length > 100) {
                sanitized[key] = value.substring(0, 100) + '...';
            } else {
                sanitized[key] = value;
            }
        }
        
        return sanitized;
    }

    logThreat(type, data) {
        const logEntry = {
            timestamp: new Date().toISOString(),
            type,
            data,
            severity: this.getThreatSeverity(type)
        };
        
        // Em produção, enviar para SIEM/Syslog
        console.warn('[WAF_THREAT]', JSON.stringify(logEntry));
        
        // Registrar em arquivo
        const fs = require('fs');
        fs.appendFileSync(
            './logs/waf-threats.log',
            JSON.stringify(logEntry) + '\n'
        );
    }

    getThreatSeverity(type) {
        const severities = {
            'IP_BLOCKED': 'LOW',
            'THREAT_DETECTED': 'HIGH',
            'SQL_INJECTION': 'CRITICAL',
            'XSS_ATTACK': 'HIGH'
        };
        return severities[type] || 'MEDIUM';
    }
}

module.exports = WAFMiddleware;
```

### 2.2 Rate Limiting Avançado

```javascript
// src/security/rate-limiting/RateLimiter.js
const Redis = require('ioredis');

class RateLimiter {
    constructor() {
        this.redis = new Redis({
            host: process.env.REDIS_HOST || 'localhost',
            port: process.env.REDIS_PORT || 6379,
            password: process.env.REDIS_PASSWORD
        });

        this.limits = {
            'global': { window: 60000, max: 1000 }, // 1000 req/min global
            'auth': { window: 60000, max: 5 }, // 5 tentativas de login/min
            'api': { window: 60000, max: 100 }, // 100 req/min por usuário
            'public': { window: 60000, max: 30 }, // 30 req/min por IP público
            'admin': { window: 60000, max: 500 } // 500 req/min para admin
        };
    }

    async checkLimit(key, type = 'api') {
        const limit = this.limits[type] || this.limits['api'];
        const now = Date.now();
        const windowStart = now - limit.window;

        // Usar Redis para rate limiting distribuído
        const redisKey = `rate_limit:${type}:${key}`;
        
        // Adiciona timestamp atual
        await this.redis.zadd(redisKey, now, now);
        
        // Remove timestamps fora da janela
        await this.redis.zremrangebyscore(redisKey, 0, windowStart);
        
        // Conta requisições na janela
        const requestCount = await this.redis.zcard(redisKey);
        
        // Define expiração
        await this.redis.expire(redisKey, Math.ceil(limit.window / 1000) + 1);

        return {
            allowed: requestCount <= limit.max,
            remaining: Math.max(0, limit.max - requestCount),
            reset: windowStart + limit.window,
            limit: limit.max
        };
    }

    async middleware(req, res, next) {
        const clientIP = req.headers['x-forwarded-for'] || req.socket.remoteAddress;
        const userId = req.user?.id || 'anonymous';
        const endpoint = req.path;
        
        // Determina tipo de limite baseado no endpoint
        let limitType = 'api';
        if (endpoint.includes('/auth/')) limitType = 'auth';
        if (endpoint.includes('/admin/')) limitType = 'admin';
        if (endpoint.includes('/public/')) limitType = 'public';

        // Chave única para rate limiting
        let key;
        if (limitType === 'auth') {
            key = clientIP; // Para auth, limitar por IP
        } else if (req.user) {
            key = `user:${userId}`; // Usuários autenticados
        } else {
            key = `ip:${clientIP}`; // Usuários anônimos
        }

        // Verifica limite global
        const globalLimit = await this.checkLimit('global', 'global');
        if (!globalLimit.allowed) {
            return res.status(429).json({
                error: 'Limite global de requisições excedido',
                retryAfter: Math.ceil((globalLimit.reset - Date.now()) / 1000)
            });
        }

        // Verifica limite específico
        const limit = await this.checkLimit(key, limitType);
        
        // Adiciona headers de rate limiting
        res.set({
            'X-RateLimit-Limit': limit.limit,
            'X-RateLimit-Remaining': limit.remaining,
            'X-RateLimit-Reset': new Date(limit.reset).toISOString()
        });

        if (!limit.allowed) {
            return res.status(429).json({
                error: 'Muitas requisições',
                retryAfter: Math.ceil((limit.reset - Date.now()) / 1000),
                code: 'RATE_LIMIT_EXCEEDED'
            });
        }

        next();
    }

    async cleanup() {
        await this.redis.quit();
    }
}

module.exports = RateLimiter;
```

```sql
    data_subjects JSON NOT NULL,
    processors JSON,
    security_measures TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    deadline_date DATE NOT NULL,

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
