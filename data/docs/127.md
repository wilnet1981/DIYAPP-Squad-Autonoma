# DIYAPP Evolution - V12 Core - Arquitetura Técnica

## ADR-001: Arquitetura de Microsserviços e Containers

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead, Especialista Infra

### CONTEXTO:
O DIYAPP precisa evoluir de uma aplicação monolítica para uma arquitetura de microsserviços para:
1. Permitir deploy independente de componentes
2. Melhorar resiliência e escalabilidade
3. Facilitar manutenção paralela por squads autônomas
4. Suportar Modo Hive (refatoração paralela)

### DECISÃO:
Adotar arquitetura baseada em containers Docker com Kubernetes para orquestração, seguindo padrões de 12-factor apps. Cada microsserviço será independente, com seu próprio banco de dados quando necessário.

### OPÇÕES CONSIDERADAS:
- **Opção A: Monolito com módulos** - Prós: Simples, baixa complexidade. Contras: Acoplamento alto, deploy único, difícil escalar componentes específicos.
- **Opção B: Microsserviços com Docker Compose** - Prós: Isolamento, deploy independente. Contras: Orquestração manual, sem auto-scaling.
- **Opção C: Microsserviços com Kubernetes** - Prós: Auto-scaling, auto-healing, service discovery. Contras: Complexidade operacional maior.
- **Opção escolhida: C** - Justificativa: Kubernetes oferece maturidade para produção, suporte nativo a observabilidade e integração com CI/CD.

### CONSEQUÊNCIAS:
**Positivas:**
- Deploy independente por squad
- Auto-scaling baseado em métricas
- Resiliência com circuit breakers
- Isolamento de falhas

**Negativas:**
- Complexidade de debugging distribuído
- Overhead de comunicação entre serviços
- Gerenciamento de múltiplos bancos de dados

**Riscos:**
- Latência em chamadas entre serviços (mitigar com caching e design assíncrono)
- Consistency em dados distribuídos (mitigar com SAGA pattern)
- Custos operacionais (mitigar com auto-scaling agressivo)

## 2. Padrões de Código

### 2.1 Linguagens e Frameworks
- **Backend Services**: Node.js 18+ com TypeScript, NestJS framework
- **Frontend**: React 18+ com TypeScript, Next.js 14
- **Mobile**: React Native com TypeScript
- **Database**: PostgreSQL (relacional), Redis (cache), MongoDB (documentos)
- **Message Queue**: RabbitMQ para comunicação assíncrona

### 2.2 Convenções de Código
```typescript
// Nomenclatura
- Interfaces: PascalCase com prefixo I (ex: IUserRepository)
- Classes: PascalCase
- Funções/métodos: camelCase
- Constantes: UPPER_SNAKE_CASE
- Arquivos: kebab-case

// Estrutura de Service
src/
├── controllers/
├── services/
├── repositories/
├── entities/
├── dtos/
├── middleware/
├── interceptors/
└── utils/
```

### 2.3 Testes
- Cobertura mínima: 80%
- Unit tests: Jest
- Integration tests: Supertest + TestContainers
- E2E tests: Cypress (frontend), Newman (API)
- Test pyramid: 70% unit, 20% integration, 10% e2e

## 3. Padrões de API

### 3.1 REST APIs
- Versionamento: `/api/v1/resource`
- Métodos HTTP semânticos
- Paginação: `?page=1&limit=20`
- Filtros: `?status=active&created_after=2024-01-01`
- Ordenação: `?sort=-created_at,name`
- HATEOAS para navegação

### 3.2 GraphQL (opcional por serviço)
- Apollo Server
- Schema-first design
- DataLoader para N+1

### 3.3 Documentação
- OpenAPI 3.0 (Swagger)
- Auto-gerada a partir de decorators
- Publicada em `/api-docs`

## 4. Comunicação entre Serviços

### 4.1 Síncrona
- HTTP/2 com gRPC para serviços internos
- REST para APIs públicas
- Timeout: 5s padrão
- Retry com exponential backoff (max 3 tentativas)
- Circuit breaker pattern

### 4.2 Assíncrona
- RabbitMQ com exchanges topic
- Event sourcing para consistência eventual
- Dead letter queues para mensagens falhas
- Idempotência obrigatória

## 5. Segurança
- JWT tokens com refresh rotation
- OAuth2 para integrações externas
- Rate limiting por IP e usuário
- Input validation com class-validator
- SQL injection prevention (TypeORM parametrized)
- Secrets em HashiCorp Vault

  push:
    branches: [ main, develop ]

    branches: [ main ]

  lint-and-test:
    runs-on: ubuntu-latest

      matrix:

    - uses: actions/checkout@v3

        cache: 'npm'
        cache-dependency-path: services/${{ matrix.service }}/package-lock.json
    
    - name: Install dependencies
      working-directory: services/${{ matrix.service }}
      run: npm ci
    
    - name: Lint
      working-directory: services/${{ matrix.service }}
      run: npm run lint
    
    - name: Type check
      working-directory: services/${{ matrix.service }}
      run: npm run type-check
    
    - name: Unit tests
      working-directory: services/${{ matrix.service }}
      run: npm test -- --coverage --coverageThreshold='{"global":{"lines":80}}'
    
    - name: Upload coverage
      uses: codecov/codecov-action@v3
      with:
        file: services/${{ matrix.service }}/coverage/lcov.info
  
  integration-tests:
    runs-on: ubuntu-latest
    needs: lint-and-test
    services:
      postgres:
        image: postgres:15
        env:
          POSTGRES_PASSWORD: postgres
        options: >-
          --health-cmd pg_isready
          --health-interval 10s
          --health-timeout 5s
          --health-retries 5
      redis:
        image: redis:7-alpine
        options: >-
          --health-cmd "redis-cli ping"
          --health-interval 10s
          --health-timeout 5s
          --health-retries 5
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Run integration tests
      run: |
        docker-compose -f docker-compose.test.yml up --abort-on-container-exit
        docker-compose -f docker-compose.test.yml down

    needs: lint-and-test

    - uses: actions/checkout@v3

        args: --severity-threshold=high

      with:
        languages: javascript, typescript

  build-and-push:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        service: [auth, user, content, notification, gateway]
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Set up Docker Buildx
      uses: docker/setup-buildx-action@v2
    
    - name: Login to DockerHub
      uses: docker/login-action@v2
      with:
        username: ${{ secrets.DOCKER_USERNAME }}
        password: ${{ secrets.DOCKER_PASSWORD }}
    
    - name: Build and push
      uses: docker/build-push-action@v4
      with:
        context: ./services/${{ matrix.service }}
        push: true
        tags: |
          diyapp/${{ matrix.service }}:latest
          diyapp/${{ matrix.service }}:${{ github.sha }}
        cache-from: type=gha
        cache-to: type=gha,mode=max
  
  deploy-staging:
    runs-on: ubuntu-latest
    needs: build-and-push
    environment: staging
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Configure kubectl
      uses: azure/setup-kubectl@v3
      with:
        version: 'latest'
    
    - name: Deploy to staging
      run: |
        kubectl apply -f infrastructure/k8s/staging/
        kubectl rollout status deployment/auth-service -n diyapp-staging
        kubectl rollout status deployment/user-service -n diyapp-staging
    
    - name: Run smoke tests
      run: |
        ./scripts/smoke-test.sh
  
  deploy-production:
    runs-on: ubuntu-latest
    needs: deploy-staging
    environment: production
    if: github.ref == 'refs/heads/main'
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Deploy to production (blue-green)
      run: |
        ./scripts/blue-green-deploy.sh
    
    - name: Run canary analysis
      run: |
        ./scripts/canary-analysis.sh
      timeout-minutes: 30

  name: 'errors_total',

  labelNames: ['type', 'service']

// Register metrics
register.registerMetric(httpRequestDuration);
register.registerMetric(databaseQueryDuration);
register.registerMetric(activeConnections);
register.registerMetric(errorCounter);

// OpenTelemetry Setup
export function setupTelemetry(serviceName: string) {
  const provider = new NodeTracerProvider({
    resource: new Resource({
      [SemanticResourceAttributes.SERVICE_NAME]: serviceName,
      [SemanticResourceAttributes.DEPLOYMENT_ENVIRONMENT]: process.env.NODE_ENV || 'development',
    }),
  });

  const exporter = new OTLPTraceExporter({
    url: process.env.OTLP_ENDPOINT || 'http://tempo:4317',
  });

  provider.addSpanProcessor(new SimpleSpanProcessor(exporter));
  
  // Register instrumentations
  provider.register();
  
  const instrumentation = [
    new HttpInstrumentation(),
    new ExpressInstrumentation(),
    new NestInstrumentation(),
    new WinstonInstrumentation(),
    new RedisInstrumentation(),
    new PinoInstrumentation(),
  ];

  return opentelemetry.trace.getTracer(serviceName);
}

// Structured Logging
import winston from 'winston';
import { LokiTransport } from 'winston-loki';

export function createLogger(serviceName: string) {
  const transports = [
    new winston.transports.Console({
      format: winston.format.combine(
        winston.format.colorize(),
        winston.format.timestamp(),
        winston.format.printf(({ timestamp, level, message, ...meta }) => {
          return `${timestamp} [${serviceName}] ${level}: ${message} ${
            Object.keys(meta).length ? JSON.stringify(meta) : ''
          }`;
        })
      ),
    }),
  ];

  if (process.env.LOKI_HOST) {
    transports.push(
      new LokiTransport({
        host: process.env.LOKI_HOST,
        labels: { service: serviceName, env: process.env.NODE_ENV },
        json: true,
        format: winston.format.json(),
        onConnectionError: (err) => console.error('Loki connection error:', err),
      })
    );
  }

  return winston.createLogger({
    level: process.env.LOG_LEVEL || 'info',
    defaultMeta: { service: serviceName },
    transports,
  });
}

// Health Check Endpoint
export function createHealthCheckRouter() {
  const router = require('express').Router();
  
  router.get('/health', async (req, res) => {
    const health = {
      status: 'healthy',
      timestamp: new Date().toISOString(),
      service: process.env.SERVICE_NAME,
      uptime: process.uptime(),
      memory: process.memoryUsage(),
      dependencies: {},
    };

    // Check database connection
    try {
      // Add database health check
      health.dependencies.database = 'healthy';
    } catch (error) {
      health.status = 'unhealthy';
      health.dependencies.database = 'unhealthy';
    }

    // Check Redis connection
    try {
      // Add Redis health check
      health.dependencies.redis = 'healthy';
    } catch (error) {
      health.status = 'unhealthy';
      health.dependencies.redis = 'unhealthy';
    }

    res.status(health.status === 'healthy' ? 200 : 503).json(health);
  });

  router.get('/metrics', async (req, res) => {
    res.set('Content-Type', register.contentType);
    res.end(await register.metrics());
  });

  return router;
}

class Database {
  constructor() {
    this.pool = new Pool({

    });

    this.setupEventListeners();
    this.testConnection();
  }

  setupEventListeners() {
    this.pool.on('connect', () => {
      logger.debug('New database connection established');
    });

    this.pool.on('error', (err) => {
      logger.error('Unexpected database pool error:', {
        error: err.message,
        stack: err.stack,
      });
    });

    this.pool.on('remove', () => {
      logger.debug('Database connection removed');
    });
  }

  async testConnection() {
    try {
      const client = await this.pool.connect();
      const result = await client.query('SELECT NOW()');
      logger.info('Database connection established successfully', {
        timestamp: result.rows[0].now,
      });
      client.release();
    } catch (error) {
      logger.error('Failed to connect to database:', {
        error: error.message,
        stack: error.stack,
      });
      // Don't throw - let the app start and retry
    }
  }

  async query(text, params) {
    const start = Date.now();
    try {
      const result = await this.pool.query(text, params);
      const duration = Date.now() - start;
      
      logger.debug('Database query executed', {
        query: text,
        params: params ? params.map(p => typeof p === 'string' ? p.substring(0, 100) : p) : [],
        duration,
        rowCount: result.rowCount,
      });

      return result;
    } catch (error) {
      const duration = Date.now() - start;
      logger.error('Database query failed:', {
        query: text,
        params: params ? params.map(p => typeof p === 'string' ? p.substring(0, 100) : p) : [],
        duration,
        error: error.message,
        code: error.code,
      });
      
      // Re-throw with sanitized error for production
      if (config.env === 'production') {
        const safeError = new Error('Database operation failed');
        safeError.code = error.code;
        throw safeError;
      }
      throw error;
    }
  }

  async transaction(callback) {
    const client = await this.pool.connect();
    try {
      await client.query('BEGIN');
      const result = await callback(client);
      await client.query('COMMIT');
      return result;
    } catch (error) {
      await client.query('ROLLBACK');
      throw error;
    } finally {
      client.release();
    }
  }

  async close() {
    await this.pool.end();
    logger.info('Database pool closed');
  }
}

module.exports = new Database();

## 1. Análise de Threat Modeling (STRIDE)

### Ativos Críticos Identificados:
1. **Dados de Usuário**: Credenciais, tokens de sessão, dados pessoais
2. **Sistema de IA**: Modelos LLM, prompts, configurações
3. **Infraestrutura**: APIs, banco de dados, serviços externos
4. **Configurações**: Secrets, chaves de API, configurações de ambiente

### Análise STRIDE por Componente:

#### A. Sistema de Autenticação (Auth Service)
- **Spoofing**: Credenciais roubadas, token hijacking
- **Tampering**: Modificação de tokens JWT, manipulação de sessões
- **Repudiation**: Falta de logs de autenticação
- **Info Disclosure**: Tokens em logs, senhas em texto claro
- **DoS**: Ataques de força bruta em login
- **Elevation**: Falhas de autorização pós-autenticação

#### B. API Gateway
- **Spoofing**: IP spoofing, falsificação de requisições
- **Tampering**: Manipulação de payloads, SQL/NoSQL injection
- **Repudiation**: Falta de logs de requisições
- **Info Disclosure**: Headers expostos, erros detalhados
- **DoS**: Rate limiting inadequado, DDoS
- **Elevation**: Acesso a APIs não autorizadas

#### C. Sistema LLM
- **Spoofing**: Prompt injection, personificação de sistema
- **Tampering**: Manipulação de prompts, output poisoning
- **Repudiation**: Falta de auditoria de prompts/respostas
- **Info Disclosure**: Vazamento de dados via LLM, system prompt leakage
- **DoS**: Prompt flooding, resource exhaustion
- **Elevation**: Execução de código via LLM output

// src/security/waf/middleware.js
const patterns = require('./patterns');
const rules = require('./rules');
const { createHash } = require('crypto');

class WAFMiddleware {
  constructor(config) {
    this.config = config;
    this.blockedIPs = new Map();
    this.requestCounts = new Map();
    this.threatPatterns = patterns.getPatterns();
  }

  async inspectRequest(req, res, next) {
    const clientIP = req.ip || req.connection.remoteAddress;
    
    // Check if IP is blocked
    if (this.isIPBlocked(clientIP)) {
      this.logThreat('IP_BLOCKED', { ip: clientIP, reason: 'Previously blocked' });
      return res.status(403).json({ 
        error: 'Access denied',
        requestId: req.requestId 
      });
    }

    // Analyze request
    const threats = await this.analyzeRequest(req);
    
    if (threats.length > 0) {
      this.handleThreats(threats, clientIP, req);
      return res.status(400).json({
        error: 'Request blocked by security policy',
        requestId: req.requestId,
        threats: threats.map(t => t.type)
      });
    }

    // Rate limiting check
    if (!this.checkRateLimit(clientIP)) {
      this.logThreat('RATE_LIMIT_EXCEEDED', { ip: clientIP });
      return res.status(429).json({
        error: 'Too many requests',
        requestId: req.requestId,
        retryAfter: '15 minutes'
      });
    }

    next();
  }

  analyzeRequest(req) {
    const threats = [];
    const { body, query, params, headers, url, method } = req;

    // Check for SQL Injection
    if (this.config.waf.rules.sqlInjection) {
      threats.push(...this.checkSQLInjection({ body, query, params, headers }));
    }

    // Check for XSS
    if (this.config.waf.rules.xss) {
      threats.push(...this.checkXSS({ body, query, params, headers }));
    }

    // Check for Path Traversal
    if (this.config.waf.rules.pathTraversal) {
      threats.push(...this.checkPathTraversal(url, params));
    }

    // Check for Command Injection
    if (this.config.waf.rules.commandInjection) {
      threats.push(...this.checkCommandInjection({ body, query, params }));
    }

    // Check for NoSQL Injection
    if (this.config.waf.rules.nosqlInjection) {
      threats.push(...this.checkNoSQLInjection({ body, query, params }));
    }

    // Check for LLM Injection
    if (this.config.waf.rules.llmInjection && req.path.includes('/api/llm')) {
      threats.push(...this.checkLLMInjection(body));
    }

    return threats;
  }

  checkSQLInjection(data) {
    const threats = [];
    const sqlPatterns = this.threatPatterns.sql;

    Object.entries(data).forEach(([key, value]) => {
      if (typeof value === 'string') {
        sqlPatterns.forEach(pattern => {
          if (pattern.regex.test(value.toLowerCase())) {
            threats.push({
              type: 'SQL_INJECTION',
              location: key,
              pattern: pattern.name,
              value: this.sanitizeLogValue(value)
            });
          }
        });
      }
    });

    return threats;
  }

  checkXSS(data) {
    const threats = [];
    const xssPatterns = this.threatPatterns.xss;

    Object.entries(data).forEach(([key, value]) => {
      if (typeof value === 'string') {
        xssPatterns.forEach(pattern => {
          if (pattern.regex.test(value)) {
            threats.push({
              type: 'XSS',
              location: key,
              pattern: pattern.name,
              value: this.sanitizeLogValue(value)
            });
          }
        });
      }
    });

    return threats;
  }

  checkLLMInjection(body) {
    const threats = [];
    
    if (body.prompt) {
      const llmPatterns = this.threatPatterns.llm;
      
      llmPatterns.forEach(pattern => {
        if (pattern.regex.test(body.prompt.toLowerCase())) {
          threats.push({
            type: 'LLM_INJECTION',
            location: 'prompt',
            pattern: pattern.name,
            value: this.sanitizeLogValue(body.prompt.substring(0, 100))
          });
        }
      });
    }

    return threats;
  }

  checkRateLimit(ip) {
    const now = Date.now();
    const windowMs = 15 * 60 * 1000; // 15 minutes
    const maxRequests = 100;

    if (!this.requestCounts.has(ip)) {
      this.requestCounts.set(ip, []);
    }

    const requests = this.requestCounts.get(ip);
    
    // Remove old requests
    const recentRequests = requests.filter(time => now - time < windowMs);
    
    if (recentRequests.length >= maxRequests) {
      return false;
    }

    recentRequests.push(now);
    this.requestCounts.set(ip, recentRequests);
    
    return true;
  }

  handleThreats(threats, ip, req) {
    threats.forEach(threat => {
      this.logThreat(threat.type, {
        ip,
        path: req.path,
        method: req.method,
        userAgent: req.headers['user-agent'],
        threatDetails: threat
      });
    });

    // Increment threat counter for IP
    const threatCount = (this.blockedIPs.get(ip)?.threatCount || 0) + threats.length;
    
    if (threatCount >= this.config.waf.blockThreshold) {
      this.blockIP(ip, `Exceeded threat threshold: ${threatCount} threats`);
    }

    this.blockedIPs.set(ip, {
      threatCount,
      lastThreat: Date.now(),
      threats: [...(this.blockedIPs.get(ip)?.threats || []), ...threats]
    });
  }

  blockIP(ip, reason) {
    const blockDuration = 24 * 60 * 60 * 1000; // 24 hours
    this.blockedIPs.set(ip, {
      ...this.blockedIPs.get(ip),
      blockedUntil: Date.now() + blockDuration,
      blockReason: reason
    });

    this.logThreat('IP_BLOCKED', { ip, reason, duration: '24h' });
  }

  isIPBlocked(ip) {
    const ipData = this.blockedIPs.get(ip);
    if (!ipData || !ipData.blockedUntil) return false;
    
    if (Date.now() > ipData.blockedUntil) {
      this.blockedIPs.delete(ip);
      return false;
    }
    
    return true;
  }

  sanitizeLogValue(value) {
    if (typeof value !== 'string') return value;
    return value.substring(0, 50) + (value.length > 50 ? '...' : '');
  }

  logThreat(type, data) {
    const logEntry = {
      timestamp: new Date().toISOString(),
      type,
      data,
      service: 'waf'
    };

    // Console log for development
    console.warn(`[WAF THREAT] ${type}:`, data);

    // File log for production
    const fs = require('fs');
    const logLine = JSON.stringify(logEntry) + '\n';
    fs.appendFileSync('./logs/security-audit.log', logLine, 'utf8');
  }
}

module.exports = WAFMiddleware;

"""
Agente Autônomo de Rollback Baseado em Métricas
Monitora error rate, latency e executa rollback automático quando necessário
"""
import asyncio
import time
import json
from datetime import datetime, timedelta
from typing import Dict, List, Optional, Tuple
from dataclasses import dataclass
import psutil
import numpy as np
from prometheus_client import start_http_server, Counter, Histogram, Gauge

from src.metrics.collector import MetricsCollector
from src.metrics.analyzer import MetricsAnalyzer
from src.deployment.rollback_executor import RollbackExecutor
from src.agents.anomaly_detector import AnomalyDetector
from src.agents.alert_manager import AlertManager
from src.utils.logger import setup_logger
from src.utils.config_loader import ConfigLoader

logger = setup_logger("rollback_agent")

# Métricas Prometheus
ERROR_RATE_GAUGE = Gauge('app_error_rate', 'Error rate percentage')
LATENCY_GAUGE = Gauge('app_latency_p95', '95th percentile latency in ms')
ROLLBACK_COUNTER = Counter('rollback_executions', 'Number of rollbacks executed')
ANOMALY_COUNTER = Counter('anomalies_detected', 'Number of anomalies detected')

@dataclass
class DeploymentMetrics:
    """Estrutura para métricas de deployment"""
    error_rate: float
    latency_p95: float
    request_count: int
    error_count: int
    timestamp: datetime
    version: str

class RollbackAgent:
    """Agente principal de rollback autônomo"""
    
    def __init__(self, config_path: str = "config/thresholds.yaml"):
        self.config = ConfigLoader.load_thresholds(config_path)
        self.metrics_collector = MetricsCollector()
        self.metrics_analyzer = MetricsAnalyzer()
        self.rollback_executor = RollbackExecutor()
        self.anomaly_detector = AnomalyDetector()
        self.alert_manager = AlertManager()
        
        # Estado do agente
        self.current_version = None
        self.rollback_history = []
        self.metrics_window = []
        self.window_size = self.config.get('window_size', 60)  # 60 segundos
        self.cooldown_period = self.config.get('cooldown_period', 300)  # 5 minutos
        self.last_rollback_time = None
        
        # Thresholds
        self.error_rate_threshold = self.config.get('error_rate_threshold', 5.0)  # 5%
        self.latency_threshold = self.config.get('latency_threshold', 1000)  # 1000ms
        self.anomaly_score_threshold = self.config.get('anomaly_score_threshold', 0.8)
        
        logger.info(f"Rollback Agent inicializado com thresholds: error_rate={self.error_rate_threshold}%, latency={self.latency_threshold}ms")
    
    async def start_monitoring(self):
        """Inicia o monitoramento contínuo"""
        logger.info("Iniciando monitoramento de métricas...")
        
        # Inicia servidor de métricas Prometheus
        start_http_server(9090)
        
        while True:
            try:
                # Coleta métricas
                metrics = await self.collect_current_metrics()
                
                # Adiciona à janela de métricas
                self.metrics_window.append(metrics)
                if len(self.metrics_window) > self.window_size:
                    self.metrics_window.pop(0)
                
                # Atualiza métricas Prometheus
                ERROR_RATE_GAUGE.set(metrics.error_rate)
                LATENCY_GAUGE.set(metrics.latency_p95)
                
                # Analisa métricas e verifica necessidade de rollback
                should_rollback, reason = await self.analyze_metrics_for_rollback(metrics)
                
                if should_rollback:
                    await self.execute_rollback(metrics, reason)
                
                # Detecta anomalias nos logs
                anomalies = await self.detect_log_anomalies()
                if anomalies:
                    await self.handle_anomalies(anomalies)
                
                # Atualiza dashboard
                await self.update_dashboard(metrics)
                
                await asyncio.sleep(self.config.get('polling_interval', 10))
                
            except Exception as e:
                logger.error(f"Erro no monitoramento: {e}")
                await asyncio.sleep(30)
    
    async def collect_current_metrics(self) -> DeploymentMetrics:
        """Coleta métricas atuais do sistema"""
        try:
            # Coleta métricas de aplicação
            app_metrics = await self.metrics_collector.collect_application_metrics()
            
            # Coleta métricas de sistema
            system_metrics = await self.metrics_collector.collect_system_metrics()
            
            # Calcula error rate
            total_requests = app_metrics.get('request_count', 0)
            error_count = app_metrics.get('error_count', 0)
            error_rate = (error_count / total_requests * 100) if total_requests > 0 else 0
            
            # Obtém latência P95
            latency_p95 = app_metrics.get('latency_p95', 0)
            
            return DeploymentMetrics(
                error_rate=error_rate,
                latency_p95=latency_p95,
                request_count=total_requests,
                error_count=error_count,
                timestamp=datetime.now(),
                version=app_metrics.get('version', 'unknown')
            )
            
        except Exception as e:
            logger.error(f"Erro ao coletar métricas: {e}")
            return DeploymentMetrics(
                error_rate=0,
                latency_p95=0,
                request_count=0,
                error_count=0,
                timestamp=datetime.now(),
                version='unknown'
            )
    
    async def analyze_metrics_for_rollback(self, metrics: DeploymentMetrics) -> Tuple[bool, str]:
        """Analisa métricas para determinar se rollback é necessário"""
        
        # Verifica período de cooldown
        if self.last_rollback_time:
            time_since_last = (datetime.now() - self.last_rollback_time).total_seconds()
            if time_since_last < self.cooldown_period:
                return False, f"Em cooldown ({int(self.cooldown_period - time_since_last)}s restantes)"
        
        reasons = []
        
        # Verifica error rate
        if metrics.error_rate > self.error_rate_threshold:
            reasons.append(f"Error rate alto: {metrics.error_rate:.2f}% > {self.error_rate_threshold}%")
        
        # Verifica latência
        if metrics.latency_p95 > self.latency_threshold:
            reasons.append(f"Latência alta: {metrics.latency_p95:.0f}ms > {self.latency_threshold}ms")
        
        # Verifica tendência de degradação
        if len(self.metrics_window) >= 10:
            trend = self.metrics_analyzer.analyze_degradation_trend(self.metrics_window)
            if trend.get('degrading', False):
                reasons.append(f"Tendência de degradação detectada: {trend.get('severity', 'medium')}")
        
        # Verifica anomalias estatísticas
        anomaly_score = self.anomaly_detector.calculate_anomaly_score(metrics)
        if anomaly_score > self.anomaly_score_threshold:
            reasons.append(f"Anomalia detectada: score={anomaly_score:.2f}")
        
        if reasons:
            return True, " | ".join(reasons)
        
        return False, "Métricas dentro dos limites aceitáveis"
    
    async def execute_rollback(self, metrics: DeploymentMetrics, reason: str):
        """Executa rollback automático"""
        try:
            logger.warning(f"Executando rollback automático. Motivo: {reason}")

                level="CRITICAL",
                title="Rollback Automático Iniciado",
                message=f"Rollback sendo executado devido a: {reason}",
                metadata={
                    "error_rate": metrics.error_rate,
                    "latency": metrics.latency_p95,
                    "version": metrics.version,
                    "timestamp": metrics.timestamp.isoformat()
                }
            )
            
            # Executa rollback
            rollback_result = await self.rollback_executor.execute_rollback(
                current_version=metrics.version,
                reason=reason
            )
            
            if rollback_result.get('success', False):
                # Atualiza estado
                self.last_rollback_time = datetime.now()
                self.current_version = rollback_result.get('previous_version')
                self.rollback_history.append({
                    'timestamp': datetime.now(),
                    'from_version': metrics.version,
                    'to_version': self.current_version,
                    'reason': reason,
                    'metrics': {
                        'error_rate': metrics.error_rate,
                        'latency': metrics.latency_p95
                    }
                })
                
                # Incrementa contador
