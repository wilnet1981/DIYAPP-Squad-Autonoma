## ADR-001: Arquitetura de MicrosserviГ§os V12

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Arquitetura Squad

### CONTEXTO:
DIYAPP evoluiu para uma plataforma complexa com mГєltiplos domГ­nios (usuГЎrios, projetos, IA, pagamentos, notificaГ§Гµes). A arquitetura monolГ­tica atual limita:
- Escalabilidade independente por domГ­nio
- Velocidade de deploy de novas features
- Isolamento de falhas
- AdoГ§ГЈo de tecnologias especГ­ficas por domГ­nio

### DECISГѓO:
Adotar arquitetura de microsserviГ§os baseada em domГ­nios de negГіcio com os seguintes princГ­pios:
1. **Um serviГ§o por domГ­nio de negГіcio** (DDD)
2. **ComunicaГ§ГЈo assГ­ncrona via eventos** como padrГЈo
3. **API Gateway** para comunicaГ§ГЈo sГ­ncrona externa
4. **Service Mesh** para observabilidade e resiliГЄncia
5. **Banco de dados por serviГ§o** (Database per Service)

### OPГ‡Г•ES CONSIDERADAS:
- **OpГ§ГЈo A:** MonГіlito modularizado - PrГіs: Simples deploy, transaГ§Гµes ACID. Contras: Escalabilidade limitada, acoplamento tГ©cnico.
- **OpГ§ГЈo B:** MicrosserviГ§os leves - PrГіs: Escalabilidade independente, resiliГЄncia. Contras: Complexidade operacional, consistГЄncia eventual.
- **OpГ§ГЈo C:** Arquitetura em camadas - PrГіs: SeparaГ§ГЈo de responsabilidades. Contras: NГЈo resolve problemas de escala.

**OpГ§ГЈo escolhida:** B - Justificativa: Alinha com crescimento projetado (10x usuГЎrios em 12 meses), permite adoГ§ГЈo de tecnologias especГ­ficas por domГ­nio (ex: vector DB para IA, Redis para sessГµes).

### CONSEQUГЉNCIAS:
**Positivas:**
- Escalabilidade horizontal independente
- Deploys independentes por squad
- ResiliГЄncia a falhas isoladas
- EvoluГ§ГЈo tecnolГіgica por serviГ§o

**Negativas:**
- Complexidade de deploy e monitoramento
- ConsistГЄncia eventual em alguns fluxos
- Overhead de comunicaГ§ГЈo entre serviГ§os
- Necessidade de ferramentas de observabilidade

**Riscos:**
- LatГЄncia em chamadas entre serviГ§os (mitigar com cache e design assГ­ncrono)
- Debug distribuГ­do complexo (mitigar com correlation IDs e tracing)
- Gerenciamento de dados distribuГ­dos (mitigar com SAGA pattern)

## Template de ServiГ§o

```typescript
// templates/service-template/src/index.ts
import express from 'express';
import helmet from 'helmet';
import cors from 'cors';
import { createLogger } from './lib/logger';
import { metricsMiddleware } from './lib/metrics';
import { errorHandler } from './lib/error-handler';
import { healthRouter } from './routes/health';
import { apiRouter } from './routes/api';
import { connectDatabase } from './database';
import { startEventConsumer } from './events/consumer';

const logger = createLogger('service-template');
const app = express();
const PORT = process.env.PORT || 3000;

// Security middleware
app.use(helmet());
app.use(cors());
app.use(express.json());

// Observability
app.use(metricsMiddleware);
app.use((req, res, next) => {
  req.id = crypto.randomUUID();
  logger.info({ reqId: req.id, method: req.method, path: req.path });
  next();
});

// Routes
app.use('/health', healthRouter);
app.use('/api/v1', apiRouter);

// Error handling
app.use(errorHandler);

async function startServer() {
  try {
    // Database connection
    await connectDatabase();
    
    // Event consumer
    await startEventConsumer();
    
    // Start server
    app.listen(PORT, () => {
      logger.info(`Service started on port ${PORT}`);
      logger.info(`Health check: http://localhost:${PORT}/health`);
    });
  } catch (error) {
    logger.error('Failed to start service', { error });
    process.exit(1);
  }
}

// Graceful shutdown
process.on('SIGTERM', () => {
  logger.info('SIGTERM received, shutting down gracefully');
  process.exit(0);
});

process.on('SIGINT', () => {
  logger.info('SIGINT received, shutting down gracefully');
  process.exit(0);
});

startServer();
```

def create_access_token(data: Dict[str, Any], expires_delta: Optional[timedelta] = None) -> str:
    """Create JWT access token"""
    to_encode = data.copy()
    
    if expires_delta:
        expire = datetime.utcnow() + expires_delta
    else:
        expire = datetime.utcnow() + timedelta(minutes=settings.ACCESS_TOKEN_EXPIRE_MINUTES)
    
    to_encode.update({"exp": expire, "type": "access"})
    encoded_jwt = jwt.encode(to_encode, settings.SECRET_KEY, algorithm=settings.ALGORITHM)
    
    logger.info("access_token_created", user_id=data.get("sub"), expires=expire.isoformat())
    return encoded_jwt

- ComunicaГ§ГЈo assГ­ncrona via eventos para atualizaГ§Гµes em massa

- Message Broker (RabbitMQ) para eventos de domГ­nio

- TransaГ§Гµes distribuГ­das necessГЎrias

Riscos:
- LatГЄncia em chamadas entre serviГ§os

import { Pool, PoolConfig, QueryResult, QueryResultRow } from 'pg';
import { Logger } from '../shared/logger';
import { CircuitBreaker } from '../shared/circuit-breaker';

export interface DatabaseConfig extends PoolConfig {
  maxRetries?: number;
  retryDelay?: number;
}

export class Database {
  private pool: Pool;
  private logger: Logger;
  private circuitBreaker: CircuitBreaker;
  private maxRetries: number;
  private retryDelay: number;

  constructor(config: DatabaseConfig) {
    this.pool = new Pool({
      host: config.host || process.env.DB_HOST || 'localhost',
      port: config.port || parseInt(process.env.DB_PORT || '5432'),
      database: config.database || process.env.DB_NAME || 'diyapp_v12',
      user: config.user || process.env.DB_USER || 'postgres',
      password: config.password || process.env.DB_PASSWORD || 'postgres',
      max: config.max || 20,
      idleTimeoutMillis: config.idleTimeoutMillis || 30000,
      connectionTimeoutMillis: config.connectionTimeoutMillis || 5000,
    });

    this.logger = Logger.getInstance();
    this.circuitBreaker = new CircuitBreaker({
      failureThreshold: 5,
      resetTimeout: 60000,
    });
    
    this.maxRetries = config.maxRetries || 3;
    this.retryDelay = config.retryDelay || 100;

    this.setupEventListeners();
  }

  private setupEventListeners(): void {
    this.pool.on('connect', () => {
      this.logger.info('Database connection established');
    });

    this.pool.on('error', (err) => {
      this.logger.error('Unexpected database pool error', { error: err.message });
    });
  }

  async query<T extends QueryResultRow = any>(
    text: string,
    params?: any[],
    correlationId?: string
  ): Promise<QueryResult<T>> {
    const operation = async () => {
      const startTime = Date.now();
      
      try {
        const result = await this.pool.query<T>(text, params);
        const duration = Date.now() - startTime;
        
        this.logger.debug('Database query executed', {
          correlationId,
          query: this.maskSensitiveData(text),
          params: this.maskSensitiveParams(params),
          duration,
          rowCount: result.rowCount,
        });

        return result;
      } catch (error) {
        const duration = Date.now() - startTime;
        
        this.logger.error('Database query failed', {
          correlationId,
          query: this.maskSensitiveData(text),
          params: this.maskSensitiveParams(params),
          duration,
          error: error instanceof Error ? error.message : 'Unknown error',
        });

        throw error;
      }
    };

    return this.circuitBreaker.execute(operation);
  }

  async transaction<T>(
    callback: (client: any) => Promise<T>,
    correlationId?: string
  ): Promise<T> {
    const client = await this.pool.connect();
    
    try {
      await client.query('BEGIN');
      this.logger.debug('Transaction started', { correlationId });

      const result = await callback(client);

      await client.query('COMMIT');
      this.logger.debug('Transaction committed', { correlationId });

      return result;
    } catch (error) {
      await client.query('ROLLBACK');
      this.logger.error('Transaction rolled back', {
        correlationId,
        error: error instanceof Error ? error.message : 'Unknown error',
      });
      throw error;
    } finally {
      client.release();
    }
  }

  async withRetry<T>(
    operation: () => Promise<T>,
    retries = this.maxRetries,
    delay = this.retryDelay
  ): Promise<T> {
    for (let attempt = 1; attempt <= retries; attempt++) {
      try {
        return await operation();
      } catch (error) {
        if (attempt === retries) {
          throw error;
        }

        const isRetryable = this.isRetryableError(error);
        if (!isRetryable) {
          throw error;
        }

        this.logger.warn(`Retrying operation (attempt ${attempt}/${retries})`, {
          error: error instanceof Error ? error.message : 'Unknown error',
          delay,
        });

        await this.sleep(delay * Math.pow(2, attempt - 1));
      }
    }

    throw new Error('Max retries exceeded');
  }

  private isRetryableError(error: any): boolean {
    if (!(error instanceof Error)) return false;

    const retryableMessages = [
      'connection',
      'timeout',
      'deadlock',
      'lock',
      'temporary',
      'try again',
      'retry',
    ];

    const message = error.message.toLowerCase();
    return retryableMessages.some(keyword => message.includes(keyword));
  }

  private sleep(ms: number): Promise<void> {
    return new Promise(resolve => setTimeout(resolve, ms));
  }

  private maskSensitiveData(text: string): string {
    const sensitivePatterns = [
      { pattern: /password\s*=\s*'[^']*'/gi, replacement: "password = '***'" },
      { pattern: /token\s*=\s*'[^']*'/gi, replacement: "token = '***'" },
      { pattern: /secret\s*=\s*'[^']*'/gi, replacement: "secret = '***'" },
      { pattern: /api_key\s*=\s*'[^']*'/gi, replacement: "api_key = '***'" },
    ];

    let masked = text;
    for (const { pattern, replacement } of sensitivePatterns) {
      masked = masked.replace(pattern, replacement);
    }

    return masked;
  }

  private maskSensitiveParams(params?: any[]): any[] {
    if (!params) return [];

    const sensitiveKeys = ['password', 'token', 'secret', 'apiKey', 'api_key'];
    
    return params.map(param => {
      if (typeof param === 'object' && param !== null) {
        const masked = { ...param };
        for (const key of sensitiveKeys) {
          if (key in masked) {
            masked[key] = '***';
          }
        }
        return masked;
      }
      return param;
    });
  }

  async close(): Promise<void> {
    await this.pool.end();
    this.logger.info('Database connection pool closed');
  }

  getPool(): Pool {
    return this.pool;
  }
}

// Singleton instance
let databaseInstance: Database | null = null;

export const getDatabase = (config?: DatabaseConfig): Database => {
  if (!databaseInstance) {
    if (!config && !process.env.DB_HOST) {
      throw new Error('Database configuration required');
    }
    databaseInstance = new Database(config || {});
  }

WITH daily_latency AS (
    SELECT
        DATE(request_timestamp) as metric_date,
        feature_name,
        endpoint,
        
        -- Basic statistics
        COUNT(*) as total_requests,
        SUM(CASE WHEN is_success THEN 1 ELSE 0 END) as successful_requests,
        AVG(latency_ms) as avg_latency,
        STDDEV(latency_ms) as stddev_latency,
        
        -- Percentiles
        APPROX_QUANTILES(latency_ms, 100)[OFFSET(50)] as p50_latency,
        APPROX_QUANTILES(latency_ms, 100)[OFFSET(90)] as p90_latency,
        APPROX_QUANTILES(latency_ms, 100)[OFFSET(95)] as p95_latency,
        APPROX_QUANTILES(latency_ms, 100)[OFFSET(99)] as p99_latency,
        
        -- SLA compliance
        SUM(CASE WHEN latency_ms <= 1000 THEN 1 ELSE 0 END) as requests_within_sla_1s,
        SUM(CASE WHEN latency_ms <= 3000 THEN 1 ELSE 0 END) as requests_within_sla_3s,
        
        -- Error rates
        SUM(CASE WHEN status_code >= 500 THEN 1 ELSE 0 END) as server_errors,
        SUM(CASE WHEN status_code >= 400 AND status_code < 500 THEN 1 ELSE 0 END) as client_errors
        
    FROM {{ ref('stg_app_metrics') }}
    WHERE request_timestamp >= TIMESTAMP_SUB(CURRENT_TIMESTAMP(), INTERVAL 30 DAY)
    GROUP BY 1, 2, 3
),

percentile_analysis AS (
    SELECT
        metric_date,
        feature_name,
        'p50' as percentile,
        p50_latency as latency_ms
    FROM daily_latency
    UNION ALL
    SELECT
        metric_date,
        feature_name,
        'p90' as percentile,
        p90_latency as latency_ms
    FROM daily_latency
    UNION ALL
    SELECT
        metric_date,
        feature_name,
        'p95' as percentile,
        p95_latency as latency_ms
    FROM daily_latency
    UNION ALL
    SELECT
        metric_date,
        feature_name,
        'p99' as percentile,
        p99_latency as latency_ms
    FROM daily_latency
)

SELECT
    dl.*,
    pa.latency_ms as percentile_latency,
    
    -- SLA compliance rates
    ROUND(SAFE_DIVIDE(requests_within_sla_1s, total_requests) * 100, 2) as sla_1s_compliance_rate,
    ROUND(SAFE_DIVIDE(requests_within_sla_3s, total_requests) * 100, 2) as sla_3s_compliance_rate,
    
    -- Error rates
    ROUND(SAFE_DIVIDE(server_errors, total_requests) * 100, 4) as server_error_rate,
    ROUND(SAFE_DIVIDE(client_errors, total_requests) * 100, 4) as client_error_rate,
    
    -- Health score
    CASE 
        WHEN sla_1s_compliance_rate >= 99.9 AND server_error_rate < 0.1 THEN 'excellent'
        WHEN sla_1s_compliance_rate >= 99.5 AND server_error_rate < 0.5 THEN 'good'
        WHEN sla_1s_compliance_rate >= 99.0 AND server_error_rate < 1.0 THEN 'fair'
        ELSE 'needs_attention'
    END as health_status,
    
    CURRENT_TIMESTAMP() as computed_at
    
FROM daily_latency dl
LEFT JOIN percentile_analysis pa 
    ON dl.metric_date = pa.metric_date

```python
# ingestion/collectors/app_metrics_collector.py
import json
import time
import logging
from datetime import datetime, timedelta
from typing import Dict, List, Optional
from dataclasses import dataclass, asdict
import asyncio
from concurrent.futures import ThreadPoolExecutor
import psutil
import requests
from google.cloud import bigquery
from google.cloud import pubsub_v1

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)

@dataclass
class AppMetric:
    """Data class for application metrics"""
    event_id: str
    session_id: str
    user_id: str
    feature_name: str
    endpoint: str
    request_timestamp: datetime
    response_timestamp: datetime
    status_code: int
    request_size_bytes: int
    response_size_bytes: int
    user_agent: str
    ip_address: str
    metadata: Dict
    
class MetricsCollector:
    """Collects and processes application metrics"""
    
    def __init__(self, project_id: str, dataset_id: str):
        self.project_id = project_id
        self.dataset_id = dataset_id
        self.bq_client = bigquery.Client(project=project_id)
        self.publisher = pubsub_v1.PublisherClient()
        self.topic_path = self.publisher.topic_path(project_id, "app-metrics-topic")
        
        # Buffer for batch processing
        self.metrics_buffer: List[Dict] = []
        self.buffer_max_size = 1000
        self.buffer_max_age = 30  # seconds
        
        # Performance tracking
        self.collection_start_time = datetime.now()
        self.metrics_collected = 0
        
    async def collect_metric(self, metric: AppMetric) -> None:
        """Collect a single metric"""
        try:
            metric_dict = asdict(metric)
            metric_dict['request_timestamp'] = metric.request_timestamp.isoformat()
            metric_dict['response_timestamp'] = metric.response_timestamp.isoformat()
            metric_dict['metadata'] = json.dumps(metric.metadata)
            
            # Add to buffer
            self.metrics_buffer.append(metric_dict)
            self.metrics_collected += 1
            
            # Publish to Pub/Sub for real-time processing
            await self._publish_to_pubsub(metric_dict)
            
            # Check if buffer needs flushing
            if (len(self.metrics_buffer) >= self.buffer_max_size or 
                (datetime.now() - self.collection_start_time).seconds >= self.buffer_max_age):
                await self.flush_buffer()
                
        except Exception as e:
            logger.error(f"Error collecting metric: {e}")
            
    async def _publish_to_pubsub(self, metric: Dict) -> None:
        """Publish metric to Pub/Sub for real-time analytics"""
        try:
            data = json.dumps(metric).encode("utf-8")
            future = self.publisher.publish(self.topic_path, data)
            message_id = future.result()
            logger.debug(f"Published message {message_id}")
        except Exception as e:
            logger.error(f"Error publishing to Pub/Sub: {e}")
            
    async def flush_buffer(self) -> None:
        """Flush metrics buffer to BigQuery"""
        if not self.metrics_buffer:
            return
            
        try:
            table_id = f"{self.project_id}.{self.dataset_id}.raw_app_metrics"
            errors = self.bq_client.insert_rows_json(
                table_id, 
                self.metrics_buffer
            )
            
            if errors:
                logger.error(f"BigQuery insert errors: {errors}")
            else:
                logger.info(f"Flushed {len(self.metrics_buffer)} metrics to BigQuery")
                self.metrics_buffer.clear()
                self.collection_start_time = datetime.now()
                
        except Exception as e:
            logger.error(f"Error flushing buffer to BigQuery: {e}")
            
    def get_performance_stats(self) -> Dict:
        """Get collector performance statistics"""
        uptime = (datetime.now() - self.collection_start_time).total_seconds()
        return {
            "uptime_seconds": uptime,
            "metrics_collected": self.metrics_collected,
            "buffer_size": len(self.metrics_buffer),
            "avg_throughput": self.metrics_collected / uptime if uptime > 0 else 0
        }

class ResourceMonitor:
    """Monitors system resource usage"""
    
    def __init__(self, collector: MetricsCollector):
        self.collector = collector
        self.monitoring = False
        
    async def start_monitoring(self, interval: int = 60):
        """Start continuous resource monitoring"""
        self.monitoring = True
        logger.info(f"Starting resource monitoring with {interval}s interval")
        
        while self.monitoring:
            try:
                metrics = await self.collect_resource_metrics()
                await self.collector.collect_metric(metrics)
                await asyncio.sleep(interval)
            except
