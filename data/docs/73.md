# DIYAPP Evolution - V11 Core - Architecture Decision Record

## ADR-001: V11 Core Architecture Foundation

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista Segurança

## CONTEXTO:
A V10 do DIYAPP atingiu um nível de complexidade onde:
1. A arquitetura monolítica dificulta escalabilidade e deploy independente
2. A falta de padrões consistentes aumenta o débito técnico
3. O sistema de logs e monitoramento é insuficiente para operação 100% autônoma
4. O Modo Hive permite refatoração paralela sem interromper produção

## DECISÃO:
Implementar arquitetura V11 baseada em:
1. **Arquitetura em Camadas com Clean Architecture** para separação de responsabilidades
2. **Estrutura de pastas padronizada** com convenções claras
3. **Sistema de logs estruturado** com múltiplos níveis e persistência
4. **Monitoramento integrado** com métricas de saúde do sistema
5. **Plano de migração gradual** usando Modo Hive

## OPÇÕES CONSIDERADAS:

### Opção A: Refatoração incremental (monolito otimizado)
- **Prós:** Menor risco, migração mais simples
- **Contras:** Mantém limitações de escalabilidade, não resolve problemas estruturais
- **Tecnologia:** Node.js com melhorias no código existente

### Opção B: Microserviços completos
- **Prós:** Máxima escalabilidade, deploy independente
- **Contras:** Complexidade operacional alta, custo inicial elevado
- **Tecnologia:** Docker + Kubernetes + Service Mesh

### Opção C: Arquitetura em Camadas (Clean Architecture)
- **Prós:** Separação clara de responsabilidades, testabilidade, preparação para microserviços
- **Contras:** Curva de aprendizado, overhead inicial
- **Tecnologia:** Node.js com estrutura modular

**Opção escolhida:** C - Justificativa: Balanceia separação de responsabilidades com pragmatismo, permitindo migração gradual e preparando para futura evolução para microserviços.

## CONSEQUÊNCIAS:
**Positivas:**
- Código mais testável e manutenível
- Separação clara entre lógica de negócio e infraestrutura
- Preparação para escalabilidade horizontal
- Melhor onboarding de novos desenvolvedores

**Negativas:**
- Overhead inicial de refatoração
- Necessidade de treinamento da squad
- Complexidade temporária durante migração

**Riscos:**
- Regressões durante migração
- Mitigação: Modo Hive + testes abrangentes
- Performance overhead das camadas
- Mitigação: Benchmarking e otimização seletiva

// src/shared/utils/logger.js
const winston = require('winston');
const { ElasticsearchTransport } = require('winston-elasticsearch');

class StructuredLogger {
  constructor() {
    this.logger = winston.createLogger({
      level: process.env.LOG_LEVEL || 'info',
      format: winston.format.combine(
        winston.format.timestamp(),
        winston.format.errors({ stack: true }),
        winston.format.json()
      ),
      defaultMeta: {
        service: 'diyapp-v11',
        environment: process.env.NODE_ENV,
        squad: 'autonomous-squad',
        version: 'v11'
      },
      transports: [
        new winston.transports.Console({
          format: winston.format.combine(
            winston.format.colorize(),
            winston.format.simple()
          )
        }),
        new winston.transports.File({ 
          filename: 'logs/error.log', 
          level: 'error',
          maxsize: 5242880, // 5MB
          maxFiles: 5
        }),
        new winston.transports.File({ 
          filename: 'logs/combined.log',
          maxsize: 5242880,
          maxFiles: 10
        })
      ]
    });

    // Transporte opcional para Elasticsearch
    if (process.env.ELASTICSEARCH_HOST) {
      this.logger.add(new ElasticsearchTransport({
        level: 'info',
        clientOpts: {
          node: process.env.ELASTICSEARCH_HOST,
          auth: {
            username: process.env.ELASTICSEARCH_USERNAME,
            password: process.env.ELASTICSEARCH_PASSWORD
          }
        }
      }));
    }
  }

  log(level, message, meta = {}) {
    const logData = {
      ...meta,
      timestamp: new Date().toISOString(),
      correlationId: meta.correlationId || this.generateCorrelationId()
    };

    this.logger.log(level, message, logData);
  }

  generateCorrelationId() {
    return `${Date.now()}-${Math.random().toString(36).substr(2, 9)}`;
  }

  // Métodos de conveniência
  info(message, meta) { this.log('info', message, meta); }
  error(message, meta) { this.log('error', message, meta); }
  warn(message, meta) { this.log('warn', message, meta); }
  debug(message, meta) { this.log('debug', message, meta); }
}

module.exports = new StructuredLogger();
```

```javascript
// src/shared/utils/monitoring.js
const promClient = require('prom-client');
const responseTime = require('response-time');

class MonitoringSystem {
  constructor() {
    // Inicializa registro do Prometheus
    promClient.collectDefaultMetrics({
      timeout: 5000,
      prefix: 'diyapp_'
    });

    // Métricas customizadas
    this.httpRequestDuration = new promClient.Histogram({
      name: 'diyapp_http_request_duration_seconds',
      help: 'Duration of HTTP requests in seconds',
      labelNames: ['method', 'route', 'status_code'],
      buckets: [0.1, 0.5, 1, 2, 5]
    });

    this.activeConnections = new promClient.Gauge({
      name: 'diyapp_active_connections',
      help: 'Number of active connections'
    });

    this.businessEvents = new promClient.Counter({
      name: 'diyapp_business_events_total',
      help: 'Total business events',
      labelNames: ['event_type']
    });

    this.errorCounter = new promClient.Counter({
      name: 'diyapp_errors_total',
      help: 'Total errors',
      labelNames: ['error_type', 'component']
    });
  }

  getMiddleware() {
    return responseTime((req, res, time) => {
      this.httpRequestDuration
        .labels(req.method, req.route?.path || req.path, res.statusCode)
        .observe(time / 1000);
    });
  }

  incrementBusinessEvent(eventType) {
    this.businessEvents.labels(eventType).inc();
  }

  incrementError(errorType, component) {
    this.errorCounter.labels(errorType, component).inc();
  }

  setActiveConnections(count) {
    this.activeConnections.set(count);
  }

  async getMetrics() {
    return await promClient.register.metrics();
  }

  async getHealthCheck() {
    const checks = {
      status: 'healthy',
      timestamp: new Date().toISOString(),
      checks: {
        database: await this.checkDatabase(),
        memory: this.checkMemory(),
        disk: this.checkDiskSpace(),
        external_services: await this.checkExternalServices()
      }
    };

    // Se qualquer check falhar, status muda para unhealthy
    Object.values(checks.checks).forEach(check => {
      if (check.status !== 'healthy') {
        checks.status = 'unhealthy';
      }
    });

    return checks;
  }

  async checkDatabase() {
    try {
      // Implementação específica do banco
      return { status: 'healthy', response_time: 50 };
    } catch (error) {
      return { status: 'unhealthy', error: error.message };
    }
  }

  checkMemory() {
    const used = process.memoryUsage();
    const threshold = 0.9; // 90% de uso
    
    return {
      status: used.heapUsed / used.heapTotal < threshold ? 'healthy' : 'warning',
      heap_used: Math.round(used.heapUsed / 1024 / 1024),
      heap_total: Math.round(used.heapTotal / 1024 / 1024),
      rss: Math.round(used.rss / 1024 / 1024)
    };
  }

  checkDiskSpace() {
    // Implementação com check-disk-space
    return { status: 'healthy' };
  }

  async checkExternalServices() {
    // Verifica serviços externos (APIs, cache, etc.)
    return { status: 'healthy' };
  }
}

module.exports = new MonitoringSystem();

import React, { forwardRef, InputHTMLAttributes } from 'react';
import { cn } from '../../utils/cn';

export interface InputProps extends InputHTMLAttributes<HTMLInputElement> {
  label?: string;
  error?: string;
  helperText?: string;
  leftIcon?: React.ReactNode;
  rightIcon?: React.ReactNode;
}

const Input = forwardRef<HTMLInputElement, InputProps>(
  ({ className, type, label, error, helperText, leftIcon, rightIcon, id, ...props }, ref) => {
    const inputId = id || `input-${Math.random().toString(36).substr(2, 9)}`;
    
    return (
      <div className="space-y-2">
        {label && (
          <label
            htmlFor={inputId}
            className="block text-sm font-medium text-neutral-700"
          >
            {label}
          </label>
        )}
        <div className="relative">
          {leftIcon && (
            <div className="absolute inset-y-0 left-0 pl-3 flex items-center pointer-events-none">
              <span className="text-neutral-500 sm:text-sm">{leftIcon}</span>
            </div>
          )}
          <input
            type={type}
            id={inputId}
            className={cn(
              'block w-full rounded-md border border-neutral-300 bg-white px-3 py-2 text-sm placeholder:text-neutral-400 focus:outline-none focus:ring-2 focus:ring-primary-500 focus:border-transparent disabled:cursor-not-allowed disabled:bg-neutral-50 disabled:text-neutral-500',
              leftIcon && 'pl-10',
              rightIcon && 'pr-10',
              error && 'border-error focus:ring-error',
              className
            )}
            ref={ref}
            aria-invalid={!!error}
            aria-describedby={error ? `${inputId}-error` : helperText ? `${inputId}-helper` : undefined}
            {...props}
          />
          {rightIcon && (

              <span className="text-neutral-500 sm:text-sm">{rightIcon}</span>

# Decorators utilitários
def cache_response(ttl: int = Config.CACHE_TTL):
    def decorator(func):
        @wraps(func)
        async def wrapper(*args, **kwargs):
            cache_key = None
            redis_client = await RedisCache.get_client()
            
            # Gerar chave de cache baseada nos argumentos
            if 'request' in kwargs:
                request = kwargs['request']
                cache_key = f"cache:{func.__name__}:{hashlib.md5(str(request.dict()).encode()).hexdigest()}"
            
            if cache_key:
                # Tentar obter do cache
                cached = await redis_client.get(cache_key)
                if cached:
                    return json.loads(cached)
            
            # Executar função original
            result = await func(*args, **kwargs)
            
            # Armazenar no cache
            if cache_key and result:
                await redis_client.setex(cache_key, ttl, json.dumps(result))
            
            return result
        return wrapper
    return decorator

def circuit_breaker(failure_threshold=5, recovery_timeout=60):
    def decorator(func):
        @wraps(func)
        async def wrapper(*args, **kwargs):
            return await circuit(
                failure_threshold=failure_threshold,
                recovery_timeout=recovery_timeout,
                expected_exception=(Exception,)
            )(func)(*args, **kwargs)
        return wrapper
    return decorator

    SUM(CASE WHEN event_id IS NULL OR user_id IS NULL OR event_timestamp IS NULL THEN 1 ELSE 0 END),

  CASE 
    WHEN SAFE_DIVIDE(

      COUNT(*)
    ) * 100 < 1 THEN 'PASS'

FROM `diyapp_v11.events`

        OR CAST(JSON_VALUE(event_data, '$.tokens_used') AS INT64) < 0

      COUNT(*)
    ) * 100 < 0.1 THEN 'PASS'

# DIYAPP Evolution - V11 Core - Análise de Segurança e Implementação

## 1. Análise de Ameaças (STRIDE) - Nova Arquitetura V11

### Assets Críticos Identificados:
1. **Sistema de Autenticação JWT** - Tokens de acesso/refresh
2. **APIs RESTful** - Endpoints de dados sensíveis
3. **Banco de Dados** - Dados de usuários, logs, configurações
4. **Sistema de Logs** - Informações de auditoria
5. **Integração com LLM** - Prompts e respostas do modelo
6. **Upload de Arquivos** - Validação e processamento

### Threat Modeling STRIDE:

#### S - Spoofing (Falsificação)
- **Ameaça**: Token JWT roubado/forjado
- **Controle**: Assinatura HMAC-SHA256, expiração curta (15min), refresh tokens rotativos
- **Risco**: ALTO - Mitigação implementada

#### T - Tampering (Manipulação)
- **Ameaça**: Manipulação de dados em trânsito ou em repouso
- **Controle**: TLS 1.3 obrigatório, hashes de integridade para arquivos, assinatura JWT
- **Risco**: MÉDIO - Mitigação implementada

#### R - Repudiation (Repúdio)
- **Ameaça**: Usuário negar ações realizadas
- **Controle**: Logs de auditoria imutáveis com hash chain, timestamps assinados
- **Risco**: BAIXO - Implementação completa

#### I - Information Disclosure (Divulgação)
- **Ameaça**: Vazamento de dados sensíveis via APIs ou logs
- **Controle**: Criptografia AES-256-GCM para PII, mascaramento em logs, sanitização de respostas
- **Risco**: ALTO - Requer atenção contínua

#### D - Denial of Service (Negação)
- **Ameaça**: Sobrecarga de APIs por ataques de força bruta
- **Controle**: Rate limiting por IP/usuario, circuit breakers, quotas
- **Risco**: MÉDIO - Implementação necessária

#### E - Elevation of Privilege (Elevação)
- **Ameaça**: Acesso além das permissões concedidas
- **Controle**: RBAC granular, verificação por recurso, princípio do menor privilégio
- **Risco**: ALTO - Verificação contínua

#### 2. Middleware de Rate Limiting (`src/security/auth/rate-limiter.js`)
```javascript
const rateLimit = require('express-rate-limit');
const Redis = require('ioredis');
const securityConfig = require('../../../config/security');

class RateLimiter {
  constructor() {
    this.redisClient = new Redis({
      host: process.env.REDIS_HOST || 'localhost',
      port: process.env.REDIS_PORT || 6379,
      enableOfflineQueue: false
    });
    
    this.limits = {
      // Limite para autenticação (prevenção de força bruta)
      auth: rateLimit({
        windowMs: securityConfig.rateLimiting.windowMs,
        max: securityConfig.rateLimiting.maxRequests.auth,
        message: {
          error: 'Muitas tentativas de login. Tente novamente em 15 minutos.',
          code: 'RATE_LIMIT_AUTH'
        },
        skipSuccessfulRequests: true,
        keyGenerator: (req) => req.ip,
        handler: (req, res) => {
          this.logSuspiciousActivity(req.ip, 'auth_bruteforce');
          res.status(429).json({
            error: 'Muitas tentativas de login detectadas',
            retryAfter: Math.ceil(securityConfig.rateLimiting.windowMs / 1000 / 60) + ' minutos'
          });
        }
      }),
      
      // Limite para APIs por usuário
      api: rateLimit({
        windowMs: securityConfig.rateLimiting.windowMs,
        max: securityConfig.rateLimiting.maxRequests.api,
        keyGenerator: (req) => req.user?.id || req.ip,
        skipSuccessfulRequests: false,
        handler: (req, res) => {
          res.status(429).json({
            error: 'Limite de requisições excedido',
            code: 'RATE_LIMIT_API'
          });
        }
      }),
      
      // Limite global por IP
      global: rateLimit({
        windowMs: securityConfig.rateLimiting.windowMs,
        max: securityConfig.rateLimiting.maxRequests.public,
        keyGenerator: (req) => req.ip,
        handler: (req, res) => {
          this.logSuspiciousActivity(req.ip, 'global_rate_limit');
          res.status(429).json({
            error: 'Limite global de requisições excedido',
            code: 'RATE_LIMIT_GLOBAL'
          });
        }
      })
    };
  }
  
  async logSuspiciousActivity(ip, activityType) {
    const key = `suspicious:${ip}:${activityType}`;
    const count = await this.redisClient.incr(key);
    await this.redisClient.expire(key, 3600); // Expira em 1 hora
    
    if (count > 10) {
      // Bloqueio temporário do IP
      await this.redisClient.setex(`blocked:${ip}`, 3600, 'true');
      console.warn(`IP ${ip} bloqueado por atividade suspeita: ${activityType}`);
    }
  }
  
  async isIpBlocked(ip) {
    return await this.redisClient.exists(`blocked:${ip}`);
  }
  
  getMiddleware(type) {
    return this.limits[type] || this.limits.global;
  }
  
  // Circuit Breaker pattern
  async circuitBreaker(serviceName, operation) {
    const circuitKey = `circuit:${serviceName}`;
    const failureKey = `failures:${serviceName}`;
    
    const circuitState = await this.redisClient.get(circuitKey);
    
    if (circuitState === 'open') {
      throw new Error(`Circuit breaker aberto para ${serviceName}`);
    }
    
    try {
      const result = await operation();
      
      // Reset failures on success
      await this.redisClient.del(failureKey);
      return result;
    } catch (error) {
      const failures = await this.redisClient.incr(failureKey);
      
      if (failures >= 5) {
        // Open circuit for 30 seconds
        await this.redisClient.setex(circuitKey, 30, 'open');
        console.error(`Circuit breaker aberto para ${serviceName}`);
      }
      
      throw error;
    }
  }
}

module.exports = new RateLimiter();

#### 3. Sanitização de Inputs e Proteção contra Injection (`src/security/validation/input-sanitizer.js`)
```javascript
const validator = require('validator');
const securityConfig = require('../../../config/security');

class InputSanitizer {
  constructor() {
    this.sqlKeywords = [
      'SELECT', 'INSERT', 'UPDATE', 'DELETE', 'DROP', 'UNION', 'OR', 'AND',
      'EXEC', 'EXECUTE', 'TRUNCATE', 'CREATE', 'ALTER', 'TABLE', 'DATABASE'
    ];
    
    this.xssPatterns = [
      /<script\b[^<]*(?:(?!<\/script>)<[^<]*)*<\/script>/gi,
      /javascript:/gi,
      /on\w+\s*=/gi,
      /data:/gi,
      /vbscript:/gi
    ];
  }
  
  // Validação e sanitização de strings
  sanitizeString(input, options = {}) {
    if (!input || typeof input !== 'string') {
      return '';
    }
    
    // Trim e limite de tamanho
    let sanitized = input.trim();
    
    if (options.maxLength && sanitized.length > options.maxLength) {
      sanitized = sanitized.substring(0, options.maxLength);
    }
    
    // Remove null bytes
    sanitized = sanitized.replace(/\0/g, '');
    
    // Sanitização baseada no tipo
    if (options.type === 'email') {
      return validator.normalizeEmail(sanitized) || '';
    } else if (options.type === 'url') {
      return validator.isURL(sanitized) ? sanitized : '';
    } else if (options.type === 'html') {
      return this.sanitizeHTML(sanitized);
    } else {
      // Sanitização padrão
      sanitized = this.escapeSQL(sanitized);
      sanitized = this.preventXSS(sanitized);
      return sanitized;
    }
  }
  
  // Prevenção de SQL Injection
  escapeSQL(input) {
    if (typeof input !== 'string') return input;
    
    // Escapa caracteres especiais SQL
    let escaped = input
      .replace(/'/g, "''")
      .replace(/\\/g, '\\\\')
      .replace(/\0/g, '\\0')
      .replace(/\n/g, '\\n')
      .replace(/\r/g, '\\r')
      .replace(/\x1a/g, '\\Z');
    
    // Detecção de padrões SQL maliciosos
    const upperInput = input.toUpperCase();
    for (const keyword of this.sqlKeywords) {
      if (upperInput.includes(keyword + ' ') || upperInput.includes(keyword + '(')) {
        console.warn(`Possível SQL injection detectado: ${keyword}`);
        throw new Error('Input contém padrões não permitidos');
      }
    }
    
    return escaped;
  }
  
  // Prevenção de XSS
  preventXSS(input) {
    if (typeof input !== 'string') return input;
    
    let sanitized = input;
    
    // Remove padrões XSS
    this.xssPatterns.forEach(pattern => {
      sanitized = sanitized.replace(pattern, '');
    });
    
    // Escapa caracteres HTML
    sanitized = sanitized
      .replace(/&/g, '&amp;')
      .replace(/</g, '&lt;')
      .replace(/>/g, '&gt;')
      .replace(/"/g, '&quot;')
      .replace(/'/g, '&#x27;')
      .replace(/\//g, '&#x2F;');
    
    return sanitized;
  }
  
  // Sanitização de HTML (para conteúdo rico)
  sanitizeHTML(html) {
    const allowedTags = ['b', 'i', 'u', 'strong', 'em', 'p', 'br', 'ul', 'ol', 'li', 'a'];
    const allowedAttributes = {
      a: ['href', 'title', 'target']
    };
    
    return validator.escape(html);
    // Nota: Para sanitização HTML mais avançada, usar biblioteca como DOMPurify
  }
  
  // Validação de arquivos
  validateFile(file, options = {}) {
    const {
      allowedTypes = securityConfig.validation.allowedFileTypes,
      maxSize = securityConfig.validation.maxFileSize
    } = options;
    
    const errors = [];
    
    // Verifica tamanho
    if (file.size > maxSize) {
      errors.push(`Arquivo excede o tamanho máximo de ${maxSize / 1024 / 1024}MB`);
    }
    
    // Verifica tipo pelo conteúdo (não apenas extensão)
    const fileExtension = file.originalname.split('.').pop().toLowerCase();
    const mimeType = file.mimetype;
    
    if (!allowedTypes.includes(fileExtension)) {
      errors.push(`Tipo de arquivo não permitido: ${fileExtension}`);
    }
    
    // Verificação adicional de MIME type
    const validMimeTypes = {
      'pdf': 'application/pdf',
      'jpg': 'image/jpeg',
      'jpeg': 'image/jpeg',
      'png': 'image/png',
      'txt': 'text/plain',
      'doc': 'application/msword',
      'docx': 'application/vnd.openxmlformats-officedocument.wordprocessingml.document'
    };
    
    if (validMimeTypes[fileExtension] && validMimeTypes[fileExtension] !== mimeType) {
      errors.push('MIME type não corresponde à extensão do arquivo');
    }
    
    // Verificação de conteúdo malicioso em arquivos
    if (this.containsMaliciousContent(file.buffer)) {
      errors.push('Arquivo contém conteúdo potencialmente malicioso');
    }
    
    return {
      isValid: errors.length === 0,
      errors,
      sanitizedFilename: this.sanitizeString(file.originalname)
    };
  }
  
  containsMaliciousContent(buffer) {
    const bufferStr = buffer.toString('binary', 0, Math.min(buffer.length, 1024));
    
    // Verifica por padrões maliciosos comuns
    const maliciousPatterns = [
      /<\?php/i,
      /<script/i,
      /eval\(/i,
      /base64_decode/i,
      /gzinflate/i,
      /shell_exec/i,
      /passthru/i,
      /exec\(/i
    ];
    
    return maliciousPatterns.some(pattern => pattern.test(bufferStr));
  }
  
  // Validação de JSON contra ataques de prototipagem
  sanitizeJSON(obj) {
    if (!obj || typeof obj !== 'object') return obj;
    
    const sanitized = {};
    
    for (const [key, value] of Object.entries(obj)) {
      const sanitizedKey = this.sanitizeString(key);
      
      if (typeof value === 'string') {
        sanitized[sanitizedKey] = this.sanitizeString(value);
      } else if (Array.isArray(value)) {
        sanitized[sanitizedKey] = value.map(item => 
          typeof item === 'string' ? this.sanitizeString(item) : item
        );
      } else if (value && typeof value === 'object') {
        sanitized[sanitizedKey] = this.sanitizeJSON(value);
      } else {
        sanitized[sanitizedKey] = value;
      }
    }
    
    return sanitized;
  }
}

module.exports = new InputSanitizer();

# Guia de Migração: API V10 para V11

**Data da publicação:** 2024-01-15  
**Última atualização:** 2024-01-15  
**Owner:** Tech Writer - DIYAPP Squad  
**Status:** Ativo

## Sumário Executivo

A V11 introduz melhorias significativas em performance, segurança e experiência do desenvolvedor. Este guia cobre todas as breaking changes e fornece instruções passo a passo para migração.

## ⚠️ Breaking Changes

### 1. Autenticação: Sessões → JWT Tokens
**Impacto:** ALTO - Requer mudanças em todos os clientes

**V10 (Antigo):**
```http
POST /api/v10/sessions
Cookie: session_id=abc123
```

**V11 (Novo):**
```http
POST /api/v11/auth/login
Authorization: Bearer eyJhbGciOiJIUzI1NiIs...
```

**Passos para migração:**
1. Substitua chamadas de login para usar `/auth/login`
2. Armazene `access_token` e `refresh_token` localmente
3. Adicione `Authorization: Bearer {token}` em todas as requisições autenticadas
4. Implemente lógica de refresh automático (veja exemplo abaixo)

### 2. Paginação: Offset-based → Cursor-based
**Impacto:** MÉDIO - Apenas para listagens paginadas

**V10 (Antigo):**
```http
GET /api/v10/projects?page=2&limit=20
```

**V11 (Novo):**
```http
GET /api/v11/projects?cursor=abc123&limit=20
```

**Passos para migração:**
1. Substitua parâmetros `page` por `cursor`
2. Use `next_cursor` da resposta para próxima página
3. Remova lógica de cálculo de offset

### 3. WebSocket para Notificações em Tempo Real
**Impacto:** BAIXO/MÉDIO - Opcional, mas recomendado

**V10 (Antigo):** Polling a cada 30 segundos
```javascript
// Polling
setInterval(() => {
  fetch('/api/v10/notifications');
}, 30000);
```

**V11 (Novo):** Conexão WebSocket persistente
```javascript
// WebSocket
const ws = new WebSocket('wss://api.diyapp.com/v11/ws?token=...');
```

### Passo 1: Atualizar Endpoints Base

class DIYAPPClientV11 {
  constructor() {
    this.baseURL = 'https://api.diyapp.com/v11';
    this.accessToken = null;
    this.refreshToken = null;
    this.refreshPromise = null;
  }

  async login(email, password) {
    const response = await fetch(`${this.baseURL}/auth/login`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ email, password, device_id: this.getDeviceId() })
    });

    if (!response.ok) throw new Error('Login failed');
    
    const data = await response.json();
    this.setTokens(data.access_token, data.refresh_token);
    return data.user;
  }

  async request(endpoint, options = {}) {
    // Adiciona token automaticamente
    const headers = {
      'Content-Type': 'application/json',
      ...options.headers
    };

    if (this.accessToken) {
      headers['Authorization'] = `Bearer ${this.accessToken}`;
    }

    let response = await fetch(`${this.baseURL}${endpoint}`, {
      ...options,
      headers
    });

    // Se token expirou, tenta refresh e repete
    if (response.status === 401 && this.refreshToken) {
      await this.refreshAccessToken();
      headers['Authorization'] = `Bearer ${this.accessToken}`;
      response = await fetch(`${this.baseURL}${endpoint}`, {
        ...options,
        headers
      });
    }

    return response;
  }

  async refreshAccessToken() {
    if (this.refreshPromise) return this.refreshPromise;
    
    this.refreshPromise = (async () => {
      try {
        const response = await fetch(`${this.baseURL}/auth/refresh`, {

          this.clearTokens();

        self,
        provider: str,

        feature: str,
        input_tokens: int,
        output_tokens: int,
        latency_ms: float,
        cost_usd: float,

            "session_id": session_id,

            log_data["prompt_hash"] = prompt_hash

            log_data["response_hash"] = response_hash

### Assets Críticos Identificados:

### Matriz STRIDE por Componente:

#### 1. Sistema de Autenticação (Auth Service)
```
Spoofing: 
- Ameaça: Token JWT roubado/forjado
- Controle: Assinatura HMAC-SHA256, expiração curta (15min), refresh tokens rotativos
- Risco: ALTO (mitigado)

Tampering:
- Ameaça: Manipulação de claims JWT
- Controle: Verificação de assinatura, validação de issuer/audience
- Risco: MÉDIO

Repudiation:
- Ameaça: Negação de login/logout
- Controle: Logs de auditoria imutáveis com IP, timestamp, user agent
- Risco: BAIXO

Info Disclosure:
- Ameaça: Vazamento de tokens via logs/erros
- Controle: Mascaramento em logs, headers seguros (HttpOnly, Secure, SameSite)
- Risco: MÉDIO

Denial of Service:
- Ameaça: Ataque de força bruta em login
- Controle: Rate limiting (5 tentativas/15min), circuit breaker
- Risco: ALTO

Elevation of Privilege:
- Ameaça: Acesso a recursos além da role
- Controle: RBAC granular, verificação por recurso (não apenas rota)
- Risco: ALTO
```

#### 2. APIs de Usuários
```
Spoofing: IDOR (Insecure Direct Object Reference)
- Controle: UUIDs não sequenciais, verificação de ownership
- Risco: ALTO

Tampering: Manipulação de dados via API
- Controle: Validação de schema Zod, sanitização de inputs
- Risco: MÉDIO

Info Disclosure: Exposição de PII
- Controle: Criptografia AES-256 em repouso, mascaramento em responses
- Risco: ALTO
```

#### 3. WhatsApp Integration
```
Spoofing: Credenciais expostas
- Controle: Secrets no Vault, rotação automática
- Risco: CRÍTICO

Tampering: Mensagens interceptadas/modificadas
- Controle: TLS 1.3, assinatura HMAC
- Risco: MÉDIO

Info Disclosure: Vazamento de conversas
- Controle: Criptografia de ponta-a-ponta, retenção limitada
- Risco: ALTO
```

#### 4. LLM/APIs de IA
```
Prompt Injection: 
- Controle: Separação system/user prompt, sanitização de inputs
- Risco: ALTO

Data Exfiltration:
- Controle: Não incluir dados sensíveis no contexto, sanitização de outputs

  private static readonly ALG = 'HS256';

  }): Promise<string> {

      ...payload,
      type: 'access'
    })
      .setProtectedHeader({ alg: this.ALG })
      .setIssuedAt()
      .setExpirationTime('15m') // Expiração curta
      .setIssuer('diyapp-v11')
      .setAudience('api.diyapp.com')
      .sign(this.secretKey);
    
    return jwt;
  }
  
  static async createRefreshToken(payload: {
    userId: string;
    sessionId: string;
  }): Promise<string> {
    const jwt = await new SignJWT({
      ...payload,
      type: 'refresh'
    })
      .setProtectedHeader({ alg: this.ALG })
      .setIssuedAt()
      .setExpirationTime('7d')
      .setIssuer('diyapp-v11')
      .setAudience('api.diyapp.com')
      .sign(this.secretKey);

```typescript
// src/security/auth/rate-limiter.ts
import Redis from 'ioredis';
import { Request, Response, NextFunction } from 'express';

export class RateLimiter {
  private redis: Redis;
  private readonly WINDOW_SIZE = 15 * 60; // 15 minutos em segundos
  private readonly MAX_REQUESTS = 100; // Máximo por IP
  
  constructor(redisUrl: string) {
    this.redis = new Redis(redisUrl);
  }
  
  async middleware(req: Request, res: Response, next: NextFunction) {
    const ip = this.getClientIP(req);
    const key = `rate_limit:${ip}:${req.path}`;
    
    try {
      const current = await this.redis.incr(key);
      
      if (current === 1) {
        await this.redis.expire(key, this.WINDOW_SIZE);
      }
      
      if (current > this.MAX_REQUESTS) {
        res.status(429).json({
          error: 'Too many requests',
          retryAfter: this.WINDOW_SIZE
        });
        return;
      }
      
      // Headers de rate limiting
      res.setHeader('X-RateLimit-Limit', this.MAX_REQUESTS.toString());
      res.setHeader('X-RateLimit-Remaining', (this.MAX_REQUESTS - current).toString());
      res.setHeader('X-RateLimit-Reset', (Date.now() + this.WINDOW_SIZE * 1000).toString());
      
      next();
    } catch (error) {
      // Fail open - não bloquear tráfego se Redis falhar
      console.error('Rate limiter error:', error);
      next();
    }
  }
  
  // Rate limiting específico para login
  async loginLimiter(req: Request, res: Response, next: NextFunction) {
    const ip = this.getClientIP(req);
    const key = `login_limit:${ip}`;
    const maxAttempts = 5;
    const lockTime = 15 * 60; // 15 minutos
    
    const attempts = await this.redis.incr(key);
    
    if (attempts === 1) {
      await this.redis.expire(key, lockTime);
    }
    
    if (attempts > maxAttempts) {
      res.status(429).json({
        error: 'Too many login attempts. Please try again later.',
        retryAfter: lockTime
      });
      return;
    }
    
    next();
  }
  
  private getClientIP(req: Request): string {
    return req.ip || 
           req.headers['x-forwarded-for']?.toString() || 
           req.socket.remoteAddress || 
           'unknown';
  }
}

    };
  }

  async generateCode(templateType, variables, context = '') {
    try {
      const promptTemplate = new PromptTemplate({
        template: this.templates[templateType],
        inputVariables: Object.keys(variables)
      });

      const chain = new LLMChain({
        llm: this.openai,
        prompt: promptTemplate
      });

      const result = await chain.call(variables);
      
      // Análise de segurança do código gerado
      const securityCheck = this.securityScan(result.text);
      if (!securityCheck.safe) {
        throw new Error(`Security issues detected: ${securityCheck.issues.join(', ')}`);
      }

      return {
        code: result.text,
        timestamp: new Date().toISOString(),
        tokensUsed: result.usage?.total_tokens || 0,
        securityCheck: securityCheck
      };
    } catch (error) {
      logger.error('AI Code Generation Error:', error);
      throw new Error(`Code generation failed: ${error.message}`);
    }
  }

  securityScan(code) {
    const dangerousPatterns = [
      /eval\(/,
      /Function\(/,
      /require\([^)]*\)/,
      /process\.env/,
      /child_process/,
      /fs\./,
      /__dirname/,
      /__filename/
    ];

    const issues = [];
    dangerousPatterns.forEach(pattern => {
      if (pattern.test(code)) {
        issues.push(pattern.toString());
      }
    });

    return {
      safe: issues.length === 0,
      issues,
      score: issues.length === 0 ? 100 : Math.max(0, 100 - (issues.length * 20))
    };
  }

  async explainCode(code, language) {
    const explanation = await this.openai.chat.completions.create({
      model: "gpt-4o",
      messages: [
        {
          role: "system",
          content: `Explain this ${language} code in simple terms. Highlight key concepts, potential issues, and best practices.`
        },
        {
          role: "user",
          content: code
        }
      ],
      temperature: 0.3
    });

    return explanation.choices[0].message.content;
  }
}

// src/preview/realtime-preview.js
class RealtimePreviewManager {
  constructor(io) {
    this.io = io;
    this.activePreviews = new Map();
    this.deviceSessions = new Map();
    
    this.setupSocketHandlers();
  }

  setupSocketHandlers() {
    this.io.on('connection', (socket) => {
      logger.info(`Client connected: ${socket.id}`);
      
      socket.on('join-preview', (data) => {
        const { projectId, deviceType } = data;
        const room = `preview-${projectId}`;
        
        socket.join(room);
        this.activePreviews.set(socket.id, { projectId, deviceType });
        
        // Notificar outros dispositivos
        socket.to(room).emit('device-connected', {
          deviceId: socket.id,
          deviceType,
          timestamp: new Date().toISOString()
        });
        
        logger.info(`Device ${deviceType} joined preview ${projectId}`);
      });

      socket.on('code-update', (data) => {
        const { projectId, code, fileType } = data;
        const room = `preview-${projectId}`;
        
        // Processar código para preview
        const processed = this.processCodeForPreview(code, fileType);
        
        // Broadcast para todos os dispositivos na sala
        socket.to(room).emit('preview-update', {
          code: processed.safeCode,
          fileType,
          deviceId: socket.id,
          timestamp: new Date().toISOString(),
          warnings: processed.warnings
        });
        
        // Log de performance
        this.logPerformance(socket.id, processed.stats);
      });

      socket.on('device-sync', (data) => {
        const { projectId, deviceType, viewport } = data;
        const room = `preview-${projectId}`;
        
        this.io.to(room).emit('viewport-sync', {
          deviceType,
          viewport,
          sourceDevice: socket.id
        });
      });

      socket.on('disconnect', () => {
        const previewData = this.activePreviews.get(socket.id);
        if (previewData) {
          const room = `preview-${previewData.projectId}`;
          socket.to(room).emit('device-disconnected', {
            deviceId: socket.id,
            deviceType: previewData.deviceType
          });
          this.activePreviews.delete(socket.id);
        }
        logger.info(`Client disconnected: ${socket.id}`);
      });
    });
  }

  processCodeForPreview(code, fileType) {
    const startTime = Date.now();
    
    // Sanitização de código para execução segura em preview
    const sanitizedCode = this.sanitizeCode(code, fileType);
    
    // Análise estática básica
    const warnings = this.staticAnalysis(sanitizedCode, fileType);
    
    const processingTime = Date.now() - startTime;
    
    return {
      safeCode: sanitizedCode,
      warnings,
      stats: {
        processingTime,
        codeSize: code.length,
        warningCount: warnings.length
      }
    };
  }

  sanitizeCode(code, fileType) {
    // Remover patterns perigosos para execução em sandbox
    const dangerousPatterns = [
      /<script\b[^>]*>[\s\S]*?<\/script>/gi,
      /eval\(/g,
      /Function\(/g,
      /document\.cookie/g,
      /localStorage\./g,
      /XMLHttpRequest/g,
      /fetch\(/g
    ];

    let safeCode = code;
    dangerousPatterns.forEach(pattern => {
      safeCode = safeCode.replace(pattern, '// SECURITY: Removed dangerous pattern');
    });

    return safeCode;
  }

  staticAnalysis(code, fileType) {
    const warnings = [];
    
    // Análise para HTML
    if (fileType === 'html') {
      if (!code.includes('<!DOCTYPE html>')) {
        warnings.push('Missing DOCTYPE declaration');
      }
      if ((code.match(/<script>/gi) || []).length > 3) {
        warnings.push('Too many script tags - consider bundling');
      }
    }
    
    // Análise para CSS
    if (fileType === 'css') {
      const importantCount = (code.match(/!important/gi) || []).length;
      if (importantCount > 5) {
        warnings.push(`Too many !important declarations (${importantCount})`);
      }
    }
    
    // Análise para JavaScript
    if (fileType === 'javascript') {
      const consoleCount = (code.match(/console\./g) || []).length;
      if (consoleCount > 10) {
        warnings.push(`Excessive console usage (${consoleCount}) - remove for production`);
      }
    }
    
    return warnings;
  }

  logPerformance(deviceId, stats) {
    logger.info('Preview Performance', {
      deviceId,
      ...stats,
      timestamp: new Date().toISOString()
    });
  }
}

const express = require('express');
const router = express.Router();
const AICodeGenerator = require('../ai/code-generator');
const { authenticatePremium } = require('../middleware/auth');

const aiGenerator = new AICodeGenerator();

// Middleware para verificar limite de uso premium
const checkUsageLimit = async (req, res, next) => {
  const userId = req.user.id;
  const usage = await req.db.getUsage(userId);
  
  if (usage.monthlyTokens >= req.user.tier.limit) {
    return res.status(429).json({
      error: 'Monthly token limit exceeded',
      limit: req.user.tier.limit,
      used: usage.monthlyTokens,
      reset: usage.resetDate
    });
  }
  
  next();
};

// Gerar código com IA
router.post('/generate', authenticatePremium, checkUsageLimit, async (req, res) => {
  try {
    const { template, variables, context } = req.body;
    
    if (!aiGenerator.templates[template]) {
      return res.status(400).json({
        error: 'Invalid template',
        available: Object.keys(aiGenerator.templates)
      });
    }

    const result = await aiGenerator.generateCode(template, variables, context);
    
    // Registrar uso
    await req.db.recordUsage(req.user.id, {
      tokens: result.tokensUsed,
      feature: 'code_generation',
      template
    });

    res.json({
      success: true,
      data: result,
      remaining: req.user.tier.limit - (await req.db.getUsage(req.user.id)).monthlyTokens
    });
  } catch (error) {
    logger.error('Generation endpoint error:', error);
    res.status(500).json({
      error: 'Generation failed',
      message: error.message
    });
  }
});

// Explicar código
router.post('/explain', authenticatePremium, async (req, res) => {
  try {
    const { code, language } = req.body;
    
    if (!code || !language) {
      return res.status(400).json({
        error: 'Code and language are required'
      });
    }

    const explanation = await aiGenerator.explainCode(code, language);
    
    res.json({
      success: true,
      explanation,
      language
    });
  } catch (error) {
    logger.error('Explanation endpoint error:', error);
    res.status(500).json({
      error: 'Explanation failed',
      message: error.message
    });
  }
});

// Obter templates disponíveis
router.get('/templates', authenticatePremium, (req, res) => {
  res.json({
    templates: Object.keys(aiGenerator.templates).map(key => ({
      name: key,
      description: aiGenerator.templates[key].substring(0, 100) + '...',
      variables: aiGenerator.templates[key].match(/\{([^}]+)\}/g)?.map(v => v.slice(1, -1)) || []
    }))
  });
});
