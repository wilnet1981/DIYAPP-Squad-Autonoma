# DIYAPP Evolution - V12 Core - Arquitetura Técnica

## Decisões Arquiteturais (ADRs)

### ADR-001: Stack Tecnológica Principal
**Data:** 2024-01-15
**Status:** Aceita

**CONTEXTO:**
Precisamos de uma stack moderna, escalável e com forte ecossistema para suportar funcionalidades de IA, processamento assíncrono e interface rica.

**DECISÃO:**
- **Frontend:** React 18 + TypeScript + Vite
- **Backend:** Node.js 20 + Express + TypeScript
- **Banco de Dados:** PostgreSQL (principal) + Redis (cache/sessões)
- **Mensageria:** RabbitMQ para filas assíncronas
- **Containerização:** Docker + Docker Compose
- **CI/CD:** GitHub Actions
- **Monitoramento:** Prometheus + Grafana + Winston para logs

**OPÇÕES CONSIDERADAS:**
- **Opção A (Vue.js + Python):** Vue.js para front, FastAPI para back
  - Prós: Vue mais simples, Python excelente para IA
  - Contras: Ecossistema menor, dois ambientes diferentes
- **Opção B (React + Node.js):** Stack JavaScript unificada
  - Prós: Mesma linguagem, ecossistema enorme, manutenção simplificada
  - Contras: Node.js menos maduro para alguns workloads de IA
- **Escolhida:** Opção B - Justificativa: Unificação de stack reduz complexidade operacional, React tem maior mercado, Node.js adequado para nossa escala inicial.

**CONSEQUÊNCIAS:**
- Positivas: Time único de JS/TS, reutilização de código, hiring mais fácil
- Negativas: Pode precisar de microserviços Python futuramente para IA pesada
- Riscos: Performance CPU-intensive; Mitigação: Workers separados, considerar Rust/Python para módulos específicos

### ADR-003: Estratégia de Deploy
**Data:** 2024-01-15
**Status:** Aceita

**DECISÃO:**
- **Ambientes:** Dev → Staging → Production
- **Infra como Código:** Terraform para recursos cloud
- **Orquestração:** Kubernetes (EKS/GKE) para produção

// diyapp-v12/src/frontend/src/pages/Dashboard.tsx
import { useEffect, useState } from 'react';
import { Card, CardContent, CardHeader, CardTitle } from '../components/ui/Card';
import { api } from '../services/api';
import { SystemStatus } from '../types';

export default function Dashboard() {
  const [status, setStatus] = useState<SystemStatus | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetchStatus();
    const interval = setInterval(fetchStatus, 30000);
    return () => clearInterval(interval);
  }, []);

  const fetchStatus = async () => {
    try {
      const data = await api.getSystemStatus();
      setStatus(data);
    } catch (error) {
      console.error('Failed to fetch system status:', error);
    } finally {
      setLoading(false);
    }
  };

  if (loading) {
    return <div>Loading...</div>;
  }

  return (
    <div className="dashboard">
      <h1>DIYAPP V12 Dashboard</h1>
      <div className="grid grid-cols-1 md:grid-cols-3 gap-4">
        <Card>
          <CardHeader>
            <CardTitle>System Health</CardTitle>
          </CardHeader>
          <CardContent>
            <div className={`status ${status?.health}`}>
              {status?.health.toUpperCase()}
            </div>
            <p>Uptime: {status?.uptime}</p>
            <p>Version: {status?.version}</p>
          </CardContent>
        </Card>
        
        <Card>
          <CardHeader>
            <CardTitle>Active Tasks</CardTitle>
          </CardHeader>
          <CardContent>
            <p>Running: {status?.tasks.running}</p>
            <p>Queued: {status?.tasks.queued}</p>
            <p>Completed: {status?.tasks.completed}</p>
          </CardContent>
        </Card>
        
        <Card>
          <CardHeader>
            <CardTitle>Performance</CardTitle>
          </CardHeader>
          <CardContent>
            <p>CPU: {status?.performance.cpu}%</p>
            <p>Memory: {status?.performance.memory}%</p>
            <p>Response Time: {status?.performance.responseTime}ms</p>
          </CardContent>
        </Card>
      </div>
    </div>
  );
}

const { Pool } = require('pg');
const config = require('./environment');
const logger = require('../utils/logger');

class Database {
  constructor() {
    this.pool = null;
    this.isConnected = false;
  }

  async connect() {
    try {
      this.pool = new Pool({
        connectionString: config.database.url,
        max: config.database.pool.max,
        min: config.database.pool.min,
        idleTimeoutMillis: 30000,
        connectionTimeoutMillis: 2000,
        ssl: config.env === 'production' ? { rejectUnauthorized: false } : false
      });

      // Test connection
      const client = await this.pool.connect();
      const result = await client.query('SELECT NOW()');
      client.release();
      
      this.isConnected = true;
      logger.info('Database connected successfully', { timestamp: result.rows[0].now });
      
      // Handle pool errors
      this.pool.on('error', (err) => {
        logger.error('Unexpected database pool error', { error: err.message });
        this.isConnected = false;
      });
      
      return this.pool;
    } catch (error) {
      logger.error('Database connection failed', { error: error.message });
      throw error;
    }
  }

  async query(text, params) {
    const start = Date.now();
    const correlationId = global.correlationId || 'unknown';
    
    try {
      const result = await this.pool.query(text, params);
      const duration = Date.now() - start;
      
      logger.debug('Database query executed', {
        correlationId,
        query: text,
        duration,
        rowCount: result.rowCount
      });
      
      return result;
    } catch (error) {
      logger.error('Database query failed', {
        correlationId,
        query: text,
        error: error.message,
        code: error.code
      });
      
      // Mask sensitive data in error messages
      const safeError = new Error('Database operation failed');
      safeError.code = error.code;
      throw safeError;
    }
  }

  async transaction(callback) {
    const client = await this.pool.connect();
    
    try {
      await client.query('BEGIN');
      const result = await callback(client);
      await client.query('COMMIT');
      return result;
    } catch (error) {
      await client.query('ROLLBACK');
      throw error;
    } finally {
      client.release();
    }
  }

  async disconnect() {
    if (this.pool) {
      await this.pool.end();
      this.isConnected = false;
      logger.info('Database disconnected');
    }
  }

  getConnectionStatus() {
    return {
      connected: this.isConnected,
      totalCount: this.pool ? this.pool.totalCount : 0,
      idleCount: this.pool ? this.pool.idleCount : 0,
      waitingCount: this.pool ? this.pool.waitingCount : 0
    };
  }
}

module.exports = new Database();

const Redis = require('ioredis');
const config = require('./environment');
const logger = require('../utils/logger');

class RedisClient {
  constructor() {
    this.client = null;
    this.publisher = null;
    this.subscriber = null;
    this.isConnected = false;
  }

  async connect() {
    try {
      const options = {
        retryStrategy: (times) => {
          const delay = Math.min(times * 50, 2000);
          logger.warn(`Redis reconnecting attempt ${times}`, { delay });
          return delay;
        },
        maxRetriesPerRequest: 3,
        enableReadyCheck: true,
        connectTimeout: 10000,
        lazyConnect: false
      };

      if (config.redis.password) {
        options.password = config.redis.password;
      }

      if (config.redis.tls) {
        options.tls = {};
      }

      this.client = new Redis(config.redis.url, options);
      this.publisher = new Redis(config.redis.url, options);
      this.subscriber = new Redis(config.redis.url, options);

      // Wait for connection
      await Promise.all([
        this.client.ping(),
        this.publisher.ping(),
        this.subscriber.ping()
      ]);

      this.isConnected = true;
      logger.info('Redis connected successfully');

      // Handle connection events
      this.client.on('error', (error) => {
        logger.error('Redis client error', { error: error.message });
        this.isConnected = false;
      });

      this.client.on('connect', () => {
        logger.info('Redis client reconnected');
        this.isConnected = true;
      });

      return this.client;
    } catch (error) {
      logger.error('Redis connection failed', { error: error.message });
      throw error;
    }
  }

  async get(key) {
    const start = Date.now();
    const correlationId = global.correlationId || 'unknown';
    
    try {
      const value = await this.client.get(key);
      const duration = Date.now() - start;
      
      logger.debug('Redis GET operation', {
        correlationId,
        key,
        duration,
        hit: value !== null
      });
      
      return value;
    } catch (error) {
      logger.error('Redis GET failed', {
        correlationId,
        key,
        error: error.message
      });
      throw error;
    }
  }

  async set(key, value, ttl = 3600) {
    const start = Date.now();
    const correlationId = global.correlationId || 'unknown';
    
    try {
      const result = await this.client.set(key, value, 'EX', ttl);
      const duration = Date.now() - start;
      
      logger.debug('Redis SET operation', {
        correlationId,
        key,
        ttl,
        duration
      });
      
      return result;
    } catch (error) {
      logger.error('Redis SET failed', {
        correlationId,
        key,
        error: error.message
      });
      throw error;
    }
  }

  async del(key) {
    try {
      return await this.client.del(key);
    } catch (error) {
      logger.error('Redis DEL failed', { key, error: error.message });
      throw error;
    }
  }

  async publish(channel, message) {
    try {
      await this.publisher.publish(channel, JSON.stringify(message));
      logger.debug('Redis PUBLISH', { channel });
    } catch (error) {
      logger.error('Redis PUBLISH failed', { channel, error: error.message });
      throw error;
    }
  }

  async subscribe(channel, callback) {
    try {
      await this.subscriber.subscribe(channel);
      this.subscriber.on('message', (ch, msg) => {
        if (ch === channel) {
          callback(JSON.parse(msg));
        }
      });
      logger.debug('Redis SUBSCRIBE', { channel });
    } catch (error) {
      logger.error('Redis SUBSCRIBE failed', { channel, error: error.message });
      throw error;
    }
  }

  async disconnect() {
    if (this.client) {
      await this.client.quit();
      await this.publisher.quit();
      await this.subscriber.quit();
      this.isConnected = false;
      logger.info('Redis disconnected');
    }
  }

  getConnectionStatus() {
    return {
      connected: this.isConnected,
      status: this.client ? this.client.status : 'disconnected'
    };
  }
}

module.exports = new RedisClient();

    """Schema para eventos do aplicativo"""
    event_id: str = Field(..., description="ID único do evento")
    user_id: str = Field(..., description="ID do usuário (anonymized)")
    session_id: str = Field(..., description="ID da sessão")
    event_type: str = Field(..., description="Tipo do evento: page_view, button_click, feature_used")
    feature_name: Optional[str] = Field(None, description="Nome da feature usada")
    page_url: Optional[str] = Field(None, description="URL da página")
    timestamp: datetime = Field(..., description="Timestamp do evento UTC")
    metadata: Dict = Field(default_factory=dict, description="Metadados adicionais")
    app_version: str = Field(..., description="Versão do aplicativo")
    platform: str = Field(..., description="web, mobile_ios, mobile_android")

class AppMetricsCollector:
    """Coletor de métricas do DIYAPP"""
    
    def __init__(self, db_url: str, api_endpoint: str = "http://diyapp-api:8000"):
        self.db_url = db_url
        self.api_endpoint = api_endpoint
        self.engine = create_engine(db_url)
        self.session = None
        
    async def initialize(self):
        """Inicializa conexão async"""
        self.session = aiohttp.ClientSession()
        
    async def collect_recent_events(self, hours: int = 1) -> List[AppEvent]:
        """Coleta eventos das últimas N horas da API do DIYAPP"""
        try:
            cutoff_time = datetime.utcnow() - timedelta(hours=hours)
            
            async with self.session.get(
                f"{self.api_endpoint}/api/v1/analytics/events",
                params={
                    "since": cutoff_time.isoformat(),
                    "limit": 10000
                }
            ) as response:
                if response.status == 200:
                    data = await response.json()
                    events = [AppEvent(**event) for event in data.get("events", [])]
                    logger.info(f"Coletados {len(events)} eventos do DIYAPP")
                    return events
                else:
                    logger.error(f"Falha ao coletar eventos: {response.status}")
                    return []
                    
        except Exception as e:
            logger.error(f"Erro na coleta de eventos: {e}")
            return []
    
    async def store_events_raw(self, events: List[AppEvent]):
        """Armazena eventos na camada RAW"""
        if not events:
            return
            
        try:
            # Converte para DataFrame
            df = pd.DataFrame([event.dict() for event in events])
            df['collected_at'] = datetime.utcnow()
            
            # Insere no PostgreSQL (camada raw)
            df.to_sql(
                'raw_app_events',
                self.engine,
                if_exists='append',
                index=False,
                method='multi'
            )
            
            logger.info(f"Armazenados {len(events)} eventos na camada RAW")
            
            # Registra métrica de ingestão
            await self._log_ingestion_metric(len(events))
            
        except Exception as e:
            logger.error(f"Erro ao armazenar eventos: {e}")
            raise
    
    async def _log_ingestion_metric(self, event_count: int):
        """Registra métrica de ingestão para monitoramento"""
        try:
            metric_data = {
                'metric_name': 'events_ingested',
                'metric_value': event_count,
                'source': 'app_metrics_collector',
                'timestamp': datetime.utcnow()
            }
            
            pd.DataFrame([metric_data]).to_sql(
                'ingestion_metrics',
                self.engine,
                if_exists='append',
                index=False
            )
            
        except Exception as e:
            logger.error(f"Erro ao registrar métrica: {e}")
    
    async def run_collection_cycle(self):
        """Executa um ciclo completo de coleta"""
        logger.info("Iniciando ciclo de coleta de métricas...")
        
        # Coleta eventos
        events = await self.collect_recent_events(hours=1)
        
        # Valida qualidade básica
        valid_events = await self._validate_events(events)
        
        # Armazena
        await self.store_events_raw(valid_events)
        
        logger.info(f"Ciclo completo: {len(valid_events)} eventos processados")
        
        return len(valid_events)
    
    async def _validate_events(self, events: List[AppEvent]) -> List[AppEvent]:
        """Validação básica de qualidade dos eventos"""
        valid_events = []
        
        for event in events:
            # Testes de qualidade
            if not event.event_id:
                logger.warning(f"Evento sem ID: {event}")
                continue
                
            if not event.user_id:
                logger.warning(f"Evento sem user_id: {event.event_id}")
                continue
                
            if event.timestamp > datetime.utcnow() + timedelta(minutes=5):
                logger.warning(f"Evento com timestamp futuro: {event.event_id}")
                continue
                
            valid_events.append(event)
        
        logger.info(f"Validação: {len(valid_events)}/{len(events)} eventos válidos")
        return valid_events
    
    async def cleanup(self):
        """Limpeza de recursos"""
        if self.session:
            await self.session.close()

# DIYAPP Evolution - V12 Core - Threat Modeling e Implementação de Segurança

## 1. Threat Modeling STRIDE - V12 Core

### Ativos Críticos Identificados:
1. **Dados de Usuário**: Credenciais, tokens de sessão, dados pessoais
2. **Recursos do Sistema**: APIs, banco de dados, serviços de IA
3. **Integrações Externas**: WhatsApp API, serviços de terceiros
4. **Modelos de IA**: Prompts do sistema, contextos de conversação

### Análise STRIDE:

#### **Spoofing (Falsificação)**
- **Ameaça**: Ataque de força bruta em login, token roubado, phishing
- **Controles**: MFA obrigatório, rate limiting, tokens JWT com expiração curta
- **Risco**: ALTO - Requer mitigação imediata

#### **Tampering (Manipulação)**
- **Ameaça**: SQL injection, manipulação de parâmetros de API, adulteração de dados em trânsito
- **Controles**: Validação de input, prepared statements, assinatura HMAC para APIs críticas
- **Risco**: ALTO - Dados podem ser corrompidos

#### **Repudiation (Repúdio)**
- **Ameaça**: Usuários negam ações realizadas (exclusões, modificações)
- **Controles**: Logs de auditoria imutáveis, timestamp com assinatura digital
- **Risco**: MÉDIO - Requer rastreabilidade completa

#### **Information Disclosure (Divulgação)**
- **Ameaça**: Vazamento de dados sensíveis, erro de configuração, logs expostos
- **Controles**: Criptografia em repouso, mascaramento de logs, headers de segurança
- **Risco**: ALTO - Conformidade com LGPD

#### **Denial of Service (Negação)**
- **Ameaça**: Ataques DDoS, sobrecarga de recursos, rate limiting ausente
- **Controles**: WAF configurado, circuit breakers, quotas por usuário

#### **Elevation of Privilege (Elevação)**
- **Ameaça**: IDOR, falha de autorização, acesso a recursos de outros usuários
- **Controles**: RBAC granular, verificação de permissão por recurso, princípio do menor privilégio
- **Risco**: ALTO - Acesso não autorizado crítico

const rateLimiter = require('./rate-limiter');
const mfa = require('./mfa');

class SecureAuth {
  constructor() {
    this.jwtSecret = securityConfig.keys.JWT_SECRET;
    this.rateLimiter = new rateLimiter();
    this.mfa = new mfa();
  }

  // Hash de senha com salt
  async hashPassword(password) {
    try {
      const salt = await bcrypt.genSalt(securityConfig.encryption.saltRounds);
      return await bcrypt.hash(password, salt);
    } catch (error) {
      auditLogger.error('Password hash failed', { error: error.message });
      throw new Error('Falha ao processar senha');
    }
  }

  // Verificação de senha
  async verifyPassword(password, hash) {
    try {
      return await bcrypt.compare(password, hash);
    } catch (error) {
      auditLogger.error('Password verification failed', { error: error.message });
      return false;
    }
  }

  // Geração de token JWT seguro
  generateToken(user, type = 'access') {
    const payload = {
      sub: user.id,
      email: user.email,
      role: user.role,
      permissions: user.permissions || [],
      iss: securityConfig.auth.jwt.issuer,
      aud: securityConfig.auth.jwt.audience,
      jti: crypto.randomUUID(),
      type: type
    };

    const options = {
      expiresIn: type === 'access' 
        ? securityConfig.auth.jwt.accessTokenExpiry 
        : securityConfig.auth.jwt.refreshTokenExpiry,
      algorithm: 'HS256'
    };

    const token = jwt.sign(payload, this.jwtSecret, options);
    
    // Log de auditoria
    auditLogger.info('Token generated', {
      userId: user.id,
      tokenType: type,
      jti: payload.jti
    });

    return token;
  }

  // Verificação de token JWT
  async verifyToken(token, type = 'access') {
    try {
      const decoded = jwt.verify(token, this.jwtSecret, {
        issuer: securityConfig.auth.jwt.issuer,
        audience: securityConfig.auth.jwt.audience,
        algorithms: ['HS256']
      });

      if (decoded.type !== type) {
        throw new Error('Tipo de token inválido');
      }

      // Verificar se token está na blacklist
      const isBlacklisted = await this.isTokenBlacklisted(decoded.jti);
      if (isBlacklisted) {
        throw new Error('Token revogado');
      }

      return decoded;
    } catch (error) {
      auditLogger.warn('Token verification failed', { error: error.message });
      throw error;
    }
  }

  // Login seguro com rate limiting e MFA
  async secureLogin(email, password, ipAddress) {
    // Verificar rate limiting
    const isBlocked = await this.rateLimiter.checkAttempts(email, ipAddress);
    if (isBlocked) {
      auditLogger.warn('Login blocked by rate limiter', { email, ipAddress });
      throw new Error('Muitas tentativas. Tente novamente em 30 minutos.');
    }

    // Buscar usuário (simulação)
    const user = await this.findUserByEmail(email);
    if (!user) {
      await this.rateLimiter.recordAttempt(email, ipAddress, false);
      auditLogger.warn('Login attempt with non-existent email', { email, ipAddress });
      throw new Error('Credenciais inválidas');
    }

    // Verificar senha
    const isValidPassword = await this.verifyPassword(password, user.passwordHash);
    if (!isValidPassword) {
      await this.rateLimiter.recordAttempt(email, ipAddress, false);
      auditLogger.warn('Failed login attempt', { userId: user.id, email, ipAddress });
      throw new Error('Credenciais inválidas');
    }

    // Resetar rate limiter para login bem-sucedido
    await this.rateLimiter.resetAttempts(email, ipAddress);

    // Verificar necessidade de MFA
    if (securityConfig.auth.mfa.requiredForAdmin && user.role === 'admin') {
      const mfaRequired = await this.mfa.isMFARequired(user.id);
      if (mfaRequired) {
        return {
          requiresMFA: true,
          userId: user.id,
          mfaMethods: await this.mfa.getAvailableMethods(user.id)
        };
      }
    }

    // Gerar tokens
    const accessToken = this.generateToken(user, 'access');
    const refreshToken = this.generateToken(user, 'refresh');

    // Registrar login bem-sucedido
    auditLogger.info('Successful login', {
      userId: user.id,
      email,
      ipAddress,
      userAgent: 'user-agent-placeholder'
    });

    return {
      accessToken,
      refreshToken,
      user: {
        id: user.id,
        email: user.email,
        role: user.role,
        permissions: user.permissions
      }
    };
  }

  // Logout seguro
  async secureLogout(token, userId) {
    try {
      const decoded = await this.verifyToken(token);
      
      // Adicionar token à blacklist
      await this.blacklistToken(decoded.jti, decoded.exp);

      await this.invalidateUserTokens(userId);

    } catch (error) {

    }

### Oportunidade 1: Assistente de Código por Voz com Contexto Visual
**Problema**: Usuários em dispositivos móveis ou com limitações motoras têm dificuldade em escrever código rapidamente durante sessões criativas.

**Hipótese**: Acreditamos que **desenvolvedores frontend em dispositivos móveis** têm o problema de **baixa velocidade de codificação em telas touch** e que um **assistente de código por voz com reconhecimento visual dos componentes atuais** vai resultar em **aumento de 40% na velocidade de criação de componentes simples**.

### Oportenidade 2: Sugestão Automática de Componentes Baseada em Descrição Natural
**Problema**: Usuários iniciantes sabem o que querem criar mas não conhecem a sintaxe ou componentes disponíveis.
