# DIYAPP Evolution - V12 Core - Arquitetura de Microsserviços e Observabilidade

## ADR-001: Arquitetura de Microsserviços Baseada em Eventos

**Data:** 2024-03-15
**Status:** Aceita
**Autores:** Tech Lead V12 Core

### CONTEXTO:
O DIYAPP evoluiu de uma aplicação monolítica para um sistema distribuído complexo. A V12 precisa:
1. Suportar escalabilidade horizontal independente por domínio
2. Garantir resiliência e tolerância a falhas
3. Manter consistência eventual entre serviços
4. Permitir deploy independente por squad
5. Oferecer observabilidade completa do sistema distribuído

### DECISÃO:
Adotar arquitetura de microsserviços baseada em eventos com os seguintes componentes:
1. **Event Bus**: Apache Kafka como backbone de eventos
2. **Service Mesh**: Istio para comunicação service-to-service
3. **API Gateway**: Kong com plugins customizados
4. **Message Schema**: Apache Avro com Schema Registry
5. **Orquestração**: Kubernetes com GitOps (ArgoCD)

### OPÇÕES CONSIDERADAS:
- **Opção A (Síncrona REST)**: Comunicação HTTP direta entre serviços
  - Prós: Simples de implementar, debugging fácil
  - Contras: Acoplamento temporal, cascata de falhas, difícil escalar

- **Opção B (Eventos com RabbitMQ)**: Mensageria com RabbitMQ
  - Prós: Padrão AMQP, confiável, suporte a patterns complexos
  - Contras: Single point of failure, escalabilidade limitada

- **Opção C (Eventos com Kafka)**: Event streaming com Kafka
  - Prós: Alta throughput, retenção de eventos, reprocessamento
  - Contras: Curva de aprendizado, operação complexa

- **Opção escolhida: C** - Justificativa: Kafka oferece durabilidade de eventos, permite event sourcing, e suporta padrões como CQRS e Saga. A retenção de eventos permite reprocessamento e debugging histórico.

### CONSEQUÊNCIAS:
**Positivas:**
- Desacoplamento temporal entre serviços
- Resiliência a falhas transitórias
- Escalabilidade independente por tópico
- Audit trail completo de eventos
- Possibilidade de event sourcing

**Negativas:**
- Complexidade operacional aumentada
- Latência adicional em alguns cenários
- Necessidade de monitoramento sofisticado
- Gerenciamento de schemas de eventos

**Riscos:**
- Eventos duplicados (mitigar com idempotência)
- Ordem de eventos (usar partition keys)
- Schema evolution (usar Avro com backward compatibility)

```javascript
const { getNodeAutoInstrumentations } = require('@opentelemetry/auto-instrumentations-node');
const { Resource } = require('@opentelemetry/resources');
const { SemanticResourceAttributes } = require('@opentelemetry/semantic-conventions');
const { OTLPTraceExporter } = require('@opentelemetry/exporter-trace-otlp-grpc');
const { OTLPMetricExporter } = require('@opentelemetry/exporter-metrics-otlp-grpc');
const { PeriodicExportingMetricReader } = require('@opentelemetry/sdk-metrics');
const { WinstonInstrumentation } = require('@opentelemetry/instrumentation-winston');
const { KafkaJsInstrumentation } = require('@opentelemetry/instrumentation-kafkajs');

class ObservabilitySDK {
  constructor(serviceName) {
    this.serviceName = serviceName;
    this.sdk = null;
  }

  initialize() {
    const resource = new Resource({
      [SemanticResourceAttributes.SERVICE_NAME]: this.serviceName,
      [SemanticResourceAttributes.SERVICE_VERSION]: process.env.APP_VERSION || '1.0.0',
      'deployment.environment': process.env.NODE_ENV || 'development',
    });

    const traceExporter = new OTLPTraceExporter({
      url: process.env.OTLP_ENDPOINT || 'http://collector:4317',
    });

    const metricExporter = new OTLPMetricExporter({
      url: process.env.OTLP_ENDPOINT || 'http://collector:4317',
    });

    const metricReader = new PeriodicExportingMetricReader({
      exporter: metricExporter,
      exportIntervalMillis: 60000,
    });

    this.sdk = new NodeSDK({
      resource,
      traceExporter,
      metricReader,
      instrumentations: [
        getNodeAutoInstrumentations({
          '@opentelemetry/instrumentation-http': {
            ignoreIncomingPaths: ['/health', '/metrics'],
          },
        }),
        new WinstonInstrumentation(),
        new KafkaJsInstrumentation(),
      ],
    });

    return this.sdk.start();
  }

  async shutdown() {
    if (this.sdk) {
      await this.sdk.shutdown();
    }
  }
}

module.exports = ObservabilitySDK;
```

```javascript
// service-template/src/utils/logger.js
const winston = require('winston');
const { combine, timestamp, json, errors } = winston.format;

class StructuredLogger {
  constructor(serviceName, context = {}) {
    this.serviceName = serviceName;
    this.context = context;
    
    this.logger = winston.createLogger({
      level: process.env.LOG_LEVEL || 'info',
      defaultMeta: {
        service: serviceName,
        environment: process.env.NODE_ENV,
        version: process.env.APP_VERSION,
        ...context,
      },
      format: combine(
        errors({ stack: true }),
        timestamp(),
        json()
      ),
      transports: [
        new winston.transports.Console(),
        new winston.transports.File({ 
          filename: `logs/${serviceName}-error.log`, 
          level: 'error' 
        }),
        new winston.transports.File({ 
          filename: `logs/${serviceName}-combined.log` 
        }),
      ],
    });

    // Adiciona correlation ID automaticamente
    this.logger.add(new winston.transports.Console({
      format: winston.format.printf((info) => {
        const correlationId = info.correlationId || 'none';
        return `${info.timestamp} [${info.service}] ${info.level}: ${info.message} (correlationId: ${correlationId})`;
      }),
    }));
  }

  // Métodos helpers para logs estruturados
  info(message, meta = {}) {
    this.logger.info(message, { ...this.context, ...meta });
  }

  error(message, error = null, meta = {}) {
    const logMeta = { ...this.context, ...meta };
    
    if (error) {
      logMeta.error = {
        name: error.name,
        message: error.message,
        stack: error.stack,
        code: error.code,
      };
    }
    
    this.logger.error(message, logMeta);
  }

  warn(message, meta = {}) {
    this.logger.warn(message, { ...this.context, ...meta });
  }

  debug(message, meta = {}) {
    this.logger.debug(message, { ...this.context, ...meta });
  }

  // Método para criar child logger com contexto adicional
  child(additionalContext) {
    return new StructuredLogger(this.serviceName, {
      ...this.context,
      ...additionalContext,
    });
  }
}

module.exports = StructuredLogger;
```

### 5. Producer de Eventos
```javascript
// libs/events/producer.js
const { Kafka } = require('kafkajs');
const avro = require('avsc');
const StructuredLogger = require('../utils/logger');

class EventProducer {
  constructor(config) {
    this.config = config;
    this.logger = new StructuredLogger('EventProducer', {
      component: 'events',
    });
    
    this.kafka = new Kafka({
      clientId: config.clientId,
      brokers: config.brokers,
      ssl: config.ssl,
      sasl: config.sasl,
      retry: {
        initialRetryTime: 100,
        retries: 8,
      },
    });
    
    this.producer = this.kafka.producer();
    this.schemas = new Map();
    this.schemaRegistry = config.schemaRegistry;
  }

  async connect() {
    await this.producer.connect();
    this.logger.info('Kafka producer connected');
  }

  async disconnect() {
    await this.producer.disconnect();
    this.logger.info('Kafka producer disconnected');
  }

  async registerSchema(topic, schemaDefinition) {
    const type = avro.Type.forSchema(schemaDefinition);
    this.schemas.set(topic, type);
    
    // Registrar no Schema Registry
    if (this.schemaRegistry) {
      await this.registerSchemaInRegistry(topic, schemaDefinition);
    }
  }

  async publish(topic, event) {
    const span = global.tracer?.startSpan(`publish.${topic}`);
    
    try {
      // Validar schema
      const schema = this.schemas.get(topic);
      if (!schema) {
        throw new Error(`Schema not registered for topic: ${topic}`);
      }

      // Validar evento contra schema
      const isValid = schema.isValid(event);
      if (!isValid) {
        const errors = schema.check(event);
        throw new Error(`Event validation failed: ${JSON.stringify(errors)}`);
      }

      // Serializar para Avro
      const buffer = schema.toBuffer(event);

      // Publicar no Kafka
      const result = await this.producer.send({
        topic,
        messages: [
          {
            key: event.eventId,
            value: buffer,
            headers: {
              'event-type': event.eventType,
              'correlation-id': event.correlationId,
              'timestamp': event.timestamp.toString(),
              'content-type': 'application/avro',
            },
          },
        ],
      });

      this.logger.info('Event published successfully', {
        topic,
        eventId: event.eventId,
        eventType: event.eventType,
        partition: result[0].partition,
        offset: result[0].offset,
      });

      span?.setAttributes({
        'event.id': event.eventId,
        'event.type': event.eventType,
        'kafka.topic': topic,
        'kafka.partition': result[0].partition,
        'kafka.offset': result[0].offset,
      });

      return result;
    } catch (error) {
      this.logger.error('Failed to publish event', error, {
        topic,
        eventId: event.eventId,
      });
      
      span?.recordException(error);
      span?.setStatus({ code: 2, message: error.message });
      throw error;
    } finally {
      span?.end();
    }
  }

  async publishWithRetry(topic, event, maxRetries = 3) {
    for (let attempt = 1; attempt <= maxRetries; attempt++) {
      try {
        return await this.publish(topic, event);
      } catch (error) {
        if (attempt === maxRetries) {
          throw error;
        }
        
        const delay = Math.pow(2, attempt) * 1000;
        this.logger.warn(`Retrying event publish (attempt ${attempt}/${maxRetries})`, {
          topic,
          eventId: event.eventId,
          delay,
          error: error.message,
        });
        
        await new Promise(resolve => setTimeout(resolve, delay));
      }
    }
  }
}

module.exports = EventProducer;
```

// libs/events/consumer.js
const { Kafka } = require('kafkajs');
const avro = require('avsc');
const StructuredLogger = require('../utils/logger');

class EventConsumer {
  constructor(config) {
    this.config = config;
    this.logger = new StructuredLogger('EventConsumer', {
      component: 'events',
      groupId: config.groupId,
    });
    
    this.kafka = new Kafka({
      clientId: config.clientId,
      brokers: config.brokers,
      ssl: config.ssl,
      sasl: config.sasl,
    });
    
    this.consumer = this.kafka.consumer({
      groupId: config.groupId,
      sessionTimeout: config.sessionTimeout || 30000,
      heartbeatInterval: config.heartbeatInterval || 3000,
      maxBytesPerPartition: config.maxBytesPerPartition || 1048576,
    });
    
    this.schemas = new Map();
    this.handlers = new Map();
  }

  async connect() {
    await this.consumer.connect();
    this.logger.info('Kafka consumer connected');
  }

  async disconnect() {
    await this.consumer.disconnect();
    this.logger.info('Kafka consumer disconnected');
  }

  async subscribe(topic, schemaDefinition, handler) {
    // Registrar schema
    const type = avro.Type.forSchema(schemaDefinition);
    this.schemas.set(topic, type);
    this.handlers.set(topic, handler);

    // Subscrever no tópico
    await this.consumer.subscribe({
      topic,
      fromBeginning: this.config.fromBeginning || false,
    });

    this.logger.info(`Subscribed to topic: ${topic}`);
  }

  async run() {
    await this.consumer.run({
      eachMessage: async ({ topic, partition, message, heartbeat, pause }) => {
        const span = global.tracer?.startSpan(`consume.${topic}`);
        const correlationId = message.headers?.['correlation-id']?.toString();
        
        const childLogger = this.logger.child({ correlationId });
        
        try {
          // Desserializar mensagem Avro
          const schema = this.schemas.get(topic);
          if (!schema) {
            throw new Error(`Schema not found for topic: ${topic}`);
          }

          const event = schema.fromBuffer(message.value);
          
          // Adicionar metadados do Kafka
          event._kafka = {
            topic,
            partition,
            offset: message.offset,
            timestamp: message.timestamp,
            headers: message.headers,
          };

          childLogger.info('Processing event', {
            topic,
            partition,
            offset: message.offset,
            eventId: event.eventId,
            eventType: event.eventType,
          });

          span?.setAttributes({
            'event.id': event.eventId,
            'event.type': event.eventType,
            'kafka.topic': topic,
            'kafka.partition': partition,
            'kafka.offset': message.offset,
            'correlation.id': correlationId,
          });

          // Executar handler
          const handler = this.handlers.get(topic);
          if (handler) {
            await handler(event);
          }

          // Commit do offset
          await heartbeat();

          childLogger.info('Event processed successfully', {
            eventId: event.eventId,
            processingTime: span ? Date.now() - span.startTime : null,

# .github/workflows/v12-ci-cd-pipeline.yaml
name: V12 Core CI/CD Pipeline
on:
  push:
    branches: [ main, release/v12.* ]
    paths:
      - 'src/**'
      - 'Dockerfile'
      - 'package.json'
      - 'requirements.txt'
  pull_request:
    branches: [ main ]
  workflow_dispatch:
    inputs:
      environment:
        description: 'Deployment environment'
        required: true
        default: 'production'
        type: choice
        options:
        - staging
        - production
      force_rollback:
        description: 'Force rollback to previous version'
        required: false
        default: false
        type: boolean

env:
  REGISTRY: ghcr.io
  IMAGE_NAME: ${{ github.repository }}/v12-core-api
  K8S_NAMESPACE: v12-production
  ECS_CLUSTER: v12-failover-cluster
  SLO_AVAILABILITY_TARGET: 99.95
  SLO_LATENCY_TARGET: 200
  ROLLBACK_WINDOW_MINUTES: 15

jobs:
  # Job 1: Quality Gates & Security Scanning
  quality-gates:
    name: Quality Gates & Security
    runs-on: ubuntu-latest
    outputs:
      security_score: ${{ steps.sonarqube.outputs.quality_gate_status }}
      vulnerability_count: ${{ steps.trivy.outputs.vulnerability_count }}
    steps:
    - uses: actions/checkout@v3
      with:
        fetch-depth: 0
    
    - name: Run SonarQube Analysis
      id: sonarqube
      uses: SonarSource/sonarqube-scan-action@master
      env:
        SONAR_TOKEN: ${{ secrets.SONAR_TOKEN }}
        SONAR_HOST_URL: ${{ secrets.SONAR_HOST_URL }}
      with:
        args: >
          -Dsonar.projectKey=diyapp-v12-core
          -Dsonar.sources=src
          -Dsonar.tests=tests
          -Dsonar.javascript.lcov.reportPaths=coverage/lcov.info
          -Dsonar.python.coverage.reportPaths=coverage.xml
          -Dsonar.qualitygate.wait=true
    
    - name: Run Trivy Vulnerability Scanner
      id: trivy
      run: |
        docker run --rm -v /var/run/docker.sock:/var/run/docker.sock \
          aquasec/trivy:latest image --exit-code 0 --severity CRITICAL \
          --format json --output trivy-report.json \
          ${{ env.REGISTRY }}/${{ env.IMAGE_NAME }}:latest
        VULN_COUNT=$(jq '.Results[].Vulnerabilities | length' trivy-report.json | awk '{sum+=$1} END {print sum}')
        echo "vulnerability_count=$VULN_COUNT" >> $GITHUB_OUTPUT
        
    - name: Check SLO Compliance in Code
      run: |
        python scripts/check_slo_compliance.py \
          --availability ${{ env.SLO_AVAILABILITY_TARGET }} \
          --latency ${{ env.SLO_LATENCY_TARGET }}
    
    - name: Fail if Critical Vulnerabilities > 0
      if: steps.trivy.outputs.vulnerability_count > 0
      run: |
        echo "❌ Critical vulnerabilities found. Blocking deployment."
        exit 1

  # Job 2: Build & Test with SLO Validation
  build-and-test:
    name: Build, Test & SLO Validation
    runs-on: ubuntu-latest
    needs: quality-gates
    outputs:
      image_tag: ${{ steps.meta.outputs.tags }}
      test_coverage: ${{ steps.coverage.outputs.percentage }}
    steps:
    - uses: actions/checkout@v3
    
    - name: Set up Docker Buildx
      uses: docker/setup-buildx-action@v2
    
    - name: Log in to Container Registry
      uses: docker/login-action@v2
      with:
        registry: ${{ env.REGISTRY }}
        username: ${{ github.actor }}
        password: ${{ secrets.GITHUB_TOKEN }}
    
    - name: Extract metadata for Docker
      id: meta
      uses: docker/metadata-action@v4
      with:
        images: ${{ env.REGISTRY }}/${{ env.IMAGE_NAME }}
        tags: |
          type=sha,prefix={{branch}}-
          type=ref,event=branch
          type=ref,event=pr
          type=semver,pattern={{version}}
          type=raw,value=latest,enable={{is_default_branch}}
    
    - name: Build and push Docker image
      uses: docker/build-push-action@v4
      with:
        context: .
        push: true
        tags: ${{ steps.meta.outputs.tags }}
        labels: ${{ steps.meta.outputs.labels }}
        cache-from: type=gha
        cache-to: type=gha,mode=max
    
    - name: Run Unit Tests with Coverage
      id: coverage

from sqlalchemy.ext.declarative import declarative_base

class Database:
    def __init__(self, settings: Settings):
        self.settings = settings
        self.engine = None
        self.SessionLocal = None
        
    def connect(self):
        try:
            self.engine = create_engine(
                str(self.settings.DATABASE_URL),
                poolclass=QueuePool,
                pool_size=self.settings.DATABASE_POOL_SIZE,
                max_overflow=self.settings.DATABASE_MAX_OVERFLOW,
                pool_timeout=self.settings.DATABASE_POOL_TIMEOUT,
                pool_pre_ping=True,  # Verifica conexão antes de usar
                echo=False  # Em produção, sempre False
            )
            
            self.SessionLocal = sessionmaker(
                autocommit=False,
                autoflush=False,
                bind=self.engine
            )
            
            logger.info("Database connection established successfully")
            
        except Exception as e:
            logger.error(f"Failed to connect to database: {str(e)}")
            raise
    
    @contextmanager
    def get_session(self) -> Generator[Session, None, None]:
        """Context manager para sessões do banco com tratamento de erros"""
        session = self.SessionLocal()
        correlation_id = get_correlation_id()
        
        try:
            start_time = time.time()
            session.info = {"correlation_id": correlation_id}
            yield session
            session.commit()
            
            duration_ms = (time.time() - start_time) * 1000
            if duration_ms > 1000:  # Log queries lentas
                logger.warning(
                    "Slow database transaction",
                    extra={
                        "correlation_id": correlation_id,
                        "duration_ms": round(duration_ms, 2),
                        "threshold_ms": 1000
                    }
                )
                
        except Exception as e:
            session.rollback()
            logger.error(
                "Database transaction failed",
                extra={
                    "correlation_id": correlation_id,
                    "error": str(e),
                    "error_type": type(e).__name__
                }
            )
            raise
        finally:
            session.close()

import redis
from redis.exceptions import RedisError, ConnectionError, TimeoutError
from typing import Optional, Any
import json
import logging
from functools import wraps
import time

logger = logging.getLogger(__name__)

class CircuitBreaker:
    """Circuit breaker pattern para Redis"""
    def __init__(self, failure_threshold=5, recovery_timeout=30):
        self.failure_threshold = failure_threshold
        self.recovery_timeout = recovery_timeout
        self.failure_count = 0
        self.last_failure_time = None
        self.state = "CLOSED"  # CLOSED, OPEN, HALF_OPEN
    
    def record_failure(self):
        self.failure_count += 1
        self.last_failure_time = time.time()
        
        if self.failure_count >= self.failure_threshold:
            self.state = "OPEN"
            logger.warning(f"Circuit breaker OPENED for Redis")
    
    def record_success(self):
        self.failure_count = 0
        self.state = "CLOSED"
    
    def can_execute(self) -> bool:
        if self.state == "CLOSED":
            return True
        
        if self.state == "OPEN":
            if time.time() - self.last_failure_time > self.recovery_timeout:
                self.state = "HALF_OPEN"
                return True
            return False
        
        return True  # HALF_OPEN

class RedisCache:
    def __init__(self, settings: Settings):
        self.settings = settings
        self.client: Optional[redis.Redis] = None
        self.circuit_breaker = CircuitBreaker()
        self.is_connected = False
    
    def connect(self):
        try:
            self.client = redis.Redis.from_url(
                str(self.settings.REDIS_URL),
                socket_connect_timeout=5,
                socket_timeout=5,
                retry_on_timeout=True,
                max_connections=20
            )
            
            # Test connection
            self.client.ping()
            self.is_connected = True
            logger.info("Redis connection established successfully")
            
        except (ConnectionError, TimeoutError) as e:
            logger.error(f"Failed to connect to Redis: {str(e)}")
            self.is_connected = False
            raise
    
    def _with_circuit_breaker(func):
        @wraps(func)
        def wrapper(self, *args, **kwargs):
            if not self.circuit_breaker.can_execute():
                logger.warning("Circuit breaker is OPEN, skipping Redis operation")
                return None
            
            try:
                result = func(self, *args, **kwargs)
                self.circuit_breaker.record_success()
                return result
            except RedisError as e:
                self.circuit_breaker.record_failure()
                logger.error(f"Redis operation failed: {str(e)}")
                return None
        return wrapper
    
    @_with_circuit_breaker
    def get(self, key: str) -> Optional[Any]:
        if not self.is_connected or not self.client:
            return None
        
        try:
            value = self.client.get(key)
            if value:
                return json.loads(value)
            return None
        except (json.JSONDecodeError, RedisError) as e:
            logger.error(f"Failed to get key {key} from Redis: {str(e)}")
            return None
    
    @_with_circuit_breaker
    def set(self, key: str, value: Any, ttl: Optional[int] = None) -> bool:
        if not self.is_connected or not self.client:
            return False
        
        try:
            ttl = ttl or self.settings.REDIS_DEFAULT_TTL
            serialized = json.dumps(value)
            return self.client.setex(key, ttl, serialized)
        except (TypeError, RedisError) as e:
            logger.error(f"Failed to set key {key} in Redis: {str(e)}")
            return False
    
    @_with_circuit_breaker
    def delete(self, key: str) -> bool:
        if not self.is_connected or not self.client:
            return False
        
        try:
            return bool(self.client.delete(key))
        except RedisError as e:
            logger.error(f"Failed to delete key {key} from Redis: {str(e)}")
            return False
    
    @_with_circuit_breaker
    def increment(self, key: str, amount: int = 1) -> Optional[int]:
        if not self.is_connected or not self.client:
            return None
        
        try:
            return self.client.incrby(key, amount)
        except RedisError as e:
            logger.error(f"Failed to increment key {key} in Redis: {str(e)}")
            return None

import pika
from pika.exceptions import AMQPConnectionError, AMQPChannelError
from typing import Callable, Optional, Dict, Any
import json
import logging
import time
from functools import wraps

logger = logging.getLogger(__name__)

class RabbitMQQueue:
    def __init__(self, settings: Settings):
        self.settings = settings
        self.connection: Optional[pika.BlockingConnection] = None
        self.channel: Optional[pika.adapters.blocking_connection.BlockingChannel] = None
        self.is_connected = False
    
    def connect(self):
        """Estabelece conexão com RabbitMQ com retry automático"""
        max_retries = 3
        retry_delay = 2
        
        for attempt in range(max_retries):
            try:
                parameters = pika.URLParameters(str(self.settings.RABBITMQ_URL))
                parameters.heartbeat = 600
                parameters.blocked_connection_timeout = 300
                
                self.connection = pika.BlockingConnection(parameters)
                self.channel = self.connection.channel()
                
                # Declara queues duráveis
                self.channel.queue_declare(
                    queue=self.settings.TASK_QUEUE_NAME,
                    durable=True,
                    arguments={'x-max-priority': 10}
                )
                
                self.channel.queue_declare(
                    queue=self.settings.EMAIL_QUEUE_NAME,
                    durable=True
                )
                
                self.is_connected = True
                logger.info("RabbitMQ connection established successfully")
                return
                
            except AMQPConnectionError as e:
                if attempt == max_retries - 1:
                    logger.error(f"Failed to connect to RabbitMQ after {max_retries} attempts: {str(e)}")
                    raise
                logger.warning(f"RabbitMQ connection attempt {attempt + 1} failed, retrying in {retry_delay}s")
                time.sleep(retry_delay)
                retry_delay *= 2
    
    def publish(self, queue_name: str, message: Dict[str, Any], priority: int = 0):
        """Publica mensagem na fila com tratamento de erro"""
        if not self.is_connected or not self.channel:
            logger.error("Cannot publish message: not connected to RabbitMQ")
            return False
        
        try:
            self.channel.basic_publish(
                exchange='',
                routing_key=queue_name,
                body=json.dumps(message),
                properties=pika.BasicProperties(
                    delivery_mode=2,  # Persistente
                    priority=priority,
                    content_type='application/json'
                )
            )
            logger.info(f"Message published to {queue_name}", extra={"queue": queue_name})
            return True
            
        except (AMQPConnectionError, AMQPChannelError) as e:
            logger.error(f"Failed to publish message to {queue_name}: {str(e)}")
            self.is_connected = False
            return False
    
    def consume(self, queue_name: str, callback: Callable, auto_ack: bool = False):
        """Consome mensagens da fila"""
        if not self.is_connected or not self.channel:
            logger.error("Cannot consume messages: not connected to RabbitMQ")
            return
        
        try:
            self.channel.basic_qos(prefetch_count=1)
            self.channel.basic_consume(
                queue=queue_name,
                on_message_callback=callback,
                auto_ack=auto_ack
            )
            
            logger.info(f"Started consuming from {queue_name}")
            self.channel.start_consuming()
            
        except (AMQPConnectionError, AMQPChannelError) as e:
            logger.error(f"Failed to consume from {queue_name}: {str(e)}")
            self.is_connected = False
    
    def close(self):
        """Fecha conexão graciosamente"""
        if self.channel and self.channel.is_open:
            self.channel.close()
        if self.connection and self.connection.is_open:
            self.connection.close()
        logger.info("RabbitMQ connection closed")
