# ADR-001: Arquitetura Hexagonal com Clean Architecture

Data: 2024-01-15
Status: Aceita
Autores: Tech Lead + Especialista Infra + Especialista Backend

## CONTEXTO:
A V11 sofre de:
1. Acoplamento rígido entre camadas
2. Testabilidade quase impossível
3. Dificuldade de evolução paralela (Modo Hive)
4. Zero isolamento de domínio

## DECISÃO:
Adotar Arquitetura Hexagonal (Ports & Adapters) com Clean Architecture:

```
src/
├── domain/           # Entidades e regras de negócio puro
├── application/      # Casos de uso (services)
├── infrastructure/   # Implementações concretas (DB, APIs externas)
└── interfaces/       # Controllers, CLI, Webhooks
```

## OPÇÕES CONSIDERADAS:
- Opção A: MVC Tradicional (atual) - Prós: Familiar, rápido início. Contras: Acoplamento, difícil manutenção.
- Opção B: Microserviços - Prós: Escalabilidade. Contras: Overkill, complexidade operacional.
- Opção C: Hexagonal + Clean - Prós: Testabilidade, evolução paralela, baixo acoplamento.

Escolha: C - Justificativa: Alinha com Modo Hive, permite refatoração incremental.

## CONSEQUÊNCIAS:
Positivas:
- Testes unitários isolados
- Squads podem trabalhar em domínios independentes
- Troca de tecnologias sem afetar core business

Negativas:
- Curva de aprendizado inicial
- Mais arquivos e abstrações

Riscos:
- Implementação parcial gera complexidade desnecessária
Mitigação: Code reviews rigorosos + ADRs para cada adaptador

## REVISÃO: 2024-04-15
```

## 3. Stack de Performance - ADR-002

```markdown
# ADR-002: Stack de Performance Multi-camada

Data: 2024-01-15
Status: Aceita
Autores: Tech Lead + Especialista Infra

## CONTEXTO:
Performance degradada em:
- 95th percentile latency: 1200ms (alvo: <200ms)
- Database CPU: 85% constante
- Cache hit ratio: 12% (alvo: >80%)

## DECISÃO:
Implementar caching estratificado:

```
L1: In-memory (Node.js) - 10ms TTL
L2: Redis Cluster - 5min TTL  
L3: CDN Edge (Cloudflare) - 1h TTL
L4: Database com índices otimizados
```

## OPÇÕES:
- Opção A: Redis apenas - Prós: Simples. Contras: Single point of failure.
- Opção B: Multi-layer com fallback - Prós: Resiliente, performance ótima.

Escolha: B - Justificativa: Balance entre complexidade e resiliência.

## IMPLEMENTAÇÃO:
```javascript
// infrastructure/cache/multi_layer_cache.js
class MultiLayerCache {
  constructor() {
    this.l1 = new NodeCache({ stdTTL: 10 });

    // L1 → L2 → L3 → DB → Warm all caches
  }
}
```

## CONSEQUÊNCIAS:
Positivas:
- Latência reduzida para <100ms p95
- DB load reduzido em 70%
- Offline resilience melhorada

Negativas:
- Inconsistência eventual (resolve com version keys)

// Multi-layer cache pattern
async function getWithCache(key, fetchFn, ttl = 300) {
  // L1: Memory (10ms)
  const l1 = memoryCache.get(key);
  if (l1) return l1;
  
  // L2: Redis (100ms)
  const l2 = await redis.get(key);
  if (l2) {
    memoryCache.set(key, l2, 10);
    return l2;
  }
  
  // L3: Source + warm cache
  const data = await fetchFn();
  await Promise.all([
    memoryCache.set(key, data, 10),
    redis.setex(key, ttl, data)
  ]);
  
  return data;
}

const Redis = require('ioredis');
const config = require('./index');
const logger = require('../utils/logger');

class RedisClient {
  constructor() {
    this.client = null;
    this.isConnected = false;
  }

  async connect() {
    try {
      this.client = new Redis({
        host: config.REDIS_HOST,
        port: config.REDIS_PORT,
        password: config.REDIS_PASSWORD,
        retryStrategy: (times) => {
          const delay = Math.min(times * 50, 2000);
          return delay;
        },
        maxRetriesPerRequest: 3,
        enableReadyCheck: true,
      });

      this.client.on('error', (error) => {
        logger.error('Redis connection error:', { error: error.message });
        this.isConnected = false;
      });

      this.client.on('connect', () => {
        logger.info('Redis connected successfully');
        this.isConnected = true;
      });

      this.client.on('ready', () => {
        logger.info('Redis ready for commands');
      });

      await this.client.ping();
      return this.client;
    } catch (error) {
      logger.error('Failed to connect to Redis:', { error: error.message });
      throw error;
    }
  }

  async get(key) {
    if (!this.isConnected) {
      logger.warn('Redis not connected, skipping cache get');
      return null;
    }

    try {
      const data = await this.client.get(key);
      return data ? JSON.parse(data) : null;
    } catch (error) {
      logger.error('Redis get error:', { key, error: error.message });
      return null;
    }
  }

  async set(key, value, ttl = config.REDIS_TTL) {
    if (!this.isConnected) {
      logger.warn('Redis not connected, skipping cache set');
      return false;
    }

    try {
      await this.client.setex(key, ttl, JSON.stringify(value));
      return true;
    } catch (error) {
      logger.error('Redis set error:', { key, error: error.message });
      return false;
    }
  }

  async del(key) {
    if (!this.isConnected) {
      logger.warn('Redis not connected, skipping cache delete');
      return false;
    }

    try {
      await this.client.del(key);
      return true;
    } catch (error) {
      logger.error('Redis delete error:', { key, error: error.message });
      return false;
    }
  }

  async invalidatePattern(pattern) {
    if (!this.isConnected) {
      logger.warn('Redis not connected, skipping pattern invalidation');
      return false;
    }

    try {
      const keys = await this.client.keys(pattern);
      if (keys.length > 0) {
        await this.client.del(...keys);
        logger.info(`Invalidated ${keys.length} cache keys matching ${pattern}`);
      }
      return true;
    } catch (error) {
      logger.error('Redis pattern invalidation error:', { pattern, error: error.message });
      return false;
    }
  }

  async disconnect() {
    if (this.client) {
      await this.client.quit();
      this.isConnected = false;
    }
  }
}

module.exports = new RedisClient();

const { Pool } = require('pg');
const config = require('./index');
const logger = require('../utils/logger');

class Database {
  constructor() {
    this.pool = new Pool({
      host: config.DB_HOST,
      port: config.DB_PORT,
      database: config.DB_NAME,
      user: config.DB_USER,
      password: config.DB_PASSWORD,
      max: 20, // max connections
      idleTimeoutMillis: 30000,
      connectionTimeoutMillis: config.QUERY_TIMEOUT_MS,
      statement_timeout: config.QUERY_TIMEOUT_MS,
    });

    this.pool.on('error', (err) => {
      logger.error('Unexpected database pool error:', { error: err.message });
    });

    this.pool.on('connect', () => {
      logger.debug('New database connection established');
    });
  }

  async query(text, params) {
    const start = Date.now();
    const correlationId = require('../utils/logger').getCorrelationId();
    
    try {
      const result = await this.pool.query(text, params);
      const duration = Date.now() - start;
      
      logger.debug('Database query executed', {
        correlationId,
        query: text,
        params: this.maskSensitiveData(params),
        duration,
        rowCount: result.rowCount,
      });

      // Log slow queries
      if (duration > 1000) {
        logger.warn('Slow query detected', {
          correlationId,
          query: text,
          duration,
          threshold: 1000,
        });
      }

      return result;
    } catch (error) {
      const duration = Date.now() - start;
      logger.error('Database query failed', {
        correlationId,
        query: text,
        params: this.maskSensitiveData(params),
        duration,
        error: error.message,
        code: error.code,
      });
      throw error;
    }
  }

  maskSensitiveData(params) {
    if (!params) return params;
    
    return params.map(param => {
      if (typeof param === 'string') {
        // Mask emails
        if (param.includes('@')) {
          const [local, domain] = param.split('@');
          return `${local.charAt(0)}***@${domain}`;
        }
        // Mask tokens (JWT-like)
        if (param.length > 50 && param.includes('.')) {
          return '***MASKED_TOKEN***';
        }
      }
      return param;
    });
  }

  async transaction(callback) {
    const client = await this.pool.connect();
    
    try {
      await client.query('BEGIN');
      const result = await callback(client);
      await client.query('COMMIT');
      return result;
    } catch (error) {
      await client.query('ROLLBACK');
      throw error;
    } finally {
      client.release();
    }
  }

  async close() {
    await this.pool.end();
  }
}

module.exports = new Database();

  format.json()

const sensitiveFields = ['password', 'token', 'jwt', 'secret', 'authorization', 'credit_card', 'cpf', 'ssn'];

const maskSensitiveData = format((info) => {
  const maskedInfo = { ...info };
  
  Object.keys(maskedInfo).forEach(key => {
    const keyLower = key.toLowerCase();
    
    // Check if this is a sensitive field
    if (sensitiveFields.some(field => keyLower.includes(field))) {
      maskedInfo[key] = '***MASKED***';

    // Recursively mask nested objects
    if (typeof maskedInfo[key] === 'object' && maskedInfo[key] !== null) {
      maskedInfo[key] = maskSensitiveData.transform(maskedInfo[key]);
    }
  });
  
  return maskedInfo;
});

// src/components/Button/Button.tsx
import React from 'react';
import { tokens } from '../../design-system/tokens';

export type ButtonVariant = 'primary' | 'secondary' | 'outline' | 'ghost';
export type ButtonSize = 'sm' | 'md' | 'lg';

export interface ButtonProps extends React.ButtonHTMLAttributes<HTMLButtonElement> {
  variant?: ButtonVariant;
  size?: ButtonSize;
  isLoading?: boolean;
  leftIcon?: React.ReactNode;
  rightIcon?: React.ReactNode;
  children: React.ReactNode;
}

const variantStyles: Record<ButtonVariant, string> = {
  primary: `
    background-color: ${tokens.colors.primary[600]};
    color: white;
    border: 1px solid ${tokens.colors.primary[600]};
    
    &:hover:not(:disabled) {
      background-color: ${tokens.colors.primary[700]};
      border-color: ${tokens.colors.primary[700]};
    }
    
    &:focus-visible {
      box-shadow: 0 0 0 3px ${tokens.colors.primary[200]};
    }
  `,
  secondary: `
    background-color: ${tokens.colors.neutral[100]};
    color: ${tokens.colors.neutral[900]};
    border: 1px solid ${tokens.colors.neutral[300]};
    
    &:hover:not(:disabled) {
      background-color: ${tokens.colors.neutral[200]};
      border-color: ${tokens.colors.neutral[400]};
    }
    
    &:focus-visible {
      box-shadow: 0 0 0 3px ${tokens.colors.neutral[200]};
    }
  `,
  outline: `
    background-color: transparent;
    color: ${tokens.colors.primary[600]};
    border: 1px solid ${tokens.colors.primary[600]};
    
    &:hover:not(:disabled) {
      background-color: ${tokens.colors.primary[50]};
    }
    
    &:focus-visible {
      box-shadow: 0 0 0 3px ${tokens.colors.primary[200]};
    }
  `,
  ghost: `
    background-color: transparent;
    color: ${tokens.colors.neutral[700]};
    border: 1px solid transparent;
    
    &:hover:not(:disabled) {
      background-color: ${tokens.colors.neutral[100]};
    }
    
    &:focus-visible {
      box-shadow: 0 0 0 3px ${tokens.colors.neutral[200]};
    }
  `,
};

const sizeStyles: Record<ButtonSize, string> = {
  sm: `
    padding: ${tokens.spacing.xs} ${tokens.spacing.sm};
    font-size: ${tokens.typography.fontSize.sm};
    height: 2rem;
  `,
  md: `
    padding: ${tokens.spacing.sm} ${tokens.spacing.md};
    font-size: ${tokens.typography.fontSize.base};
    height: 2.5rem;
  `,
  lg: `
    padding: ${tokens.spacing.md} ${tokens.spacing.lg};
    font-size: ${tokens.typography.fontSize.lg};
    height: 3rem;
  `,
};

export const Button = React.forwardRef<HTMLButtonElement, ButtonProps>(
  (
    {
      variant = 'primary',
      size = 'md',
      isLoading = false,
      leftIcon,
      rightIcon,
      children,
      disabled,
      className = '',
      ...props
    },
    ref
  ) => {
    const isDisabled = disabled || isLoading;
    
    return (
      <button
        ref={ref}
        disabled={isDisabled}
        aria-busy={isLoading}
        className={`button ${className}`}
        css={`
          display: inline-flex;
          align-items: center;
          justify-content: center;
          gap: ${tokens.spacing.sm};
          border-radius: ${tokens.borderRadius.md};
          font-weight: ${tokens.typography.fontWeight.medium};
          transition: all 0.2s ease;
          cursor: ${isDisabled ? 'not-allowed' : 'pointer'};
          opacity: ${isDisabled ? 0.6 : 1};
          width: fit-content;
          min-width: ${size === 'sm' ? '5rem' : size === 'md' ? '6rem' : '7rem'};
          
          ${variantStyles[variant]}
          ${sizeStyles[size]}
          
          &:disabled {
            cursor: not-allowed;
          }
        `}
        {...props}
      >
        {isLoading ? (
          <span
            css={`
              @keyframes spin {
                to {
                  transform: rotate(360deg);
                }
              }
              
              width: 1em;
              height: 1em;
              border: 2px solid currentColor;
              border-top-color: transparent;
              border-radius: 50%;
              animation: spin 0.6s linear infinite;
            `}
            aria-hidden="true"
          />
        ) : (
          <>
            {leftIcon && <span aria-hidden="true">{leftIcon}</span>}
            <span>{children}</span>
            {rightIcon && <span aria-hidden="true">{rightIcon}</span>}
          </>
        )}
      </button>
    );
  }
);

Button.displayName = 'Button';

import React from 'react';
import { render, screen, fireEvent } from '@testing-library/react';
import { Button } from './Button';
import { describe, it, expect, vi } from 'vitest';

describe('Button', () => {
  it('renders with default props', () => {
    render(<Button>Click me</Button>);
    
    const button = screen.getByRole('button', { name: /click me/i });
    expect(button).toBeInTheDocument();
    expect(button).toHaveAttribute('type', 'button');
  });

  it('handles click events', () => {
    const handleClick = vi.fn();
    render(<Button onClick={handleClick}>Click me</Button>);
    
    const button = screen.getByRole('button');
    fireEvent.click(button);
    
    expect(handleClick).toHaveBeenCalledTimes(1);
  });

  it('shows loading state', () => {
    render(<Button isLoading>Click me</Button>);
    
    const button = screen.getByRole('button');
    expect(button).toHaveAttribute('aria-busy', 'true');
    expect(button).toBeDisabled();
  });

  it('is disabled when disabled prop is true', () => {
    render(<Button disabled>Click me</Button>);
    
    const button = screen.getByRole('button');
    expect(button).toBeDisabled();
  });

  it('renders with left icon', () => {
    const leftIcon = <span data-testid="left-icon">←</span>;
    render(<Button leftIcon={leftIcon}>Back</Button>);
    
    expect(screen.getByTestId('left-icon')).toBeInTheDocument();
  });

  it('renders with right icon', () => {
    const rightIcon = <span data-testid="right-icon">→</span>;
    render(<Button rightIcon={rightIcon}>Next</Button>);
    
    expect(screen.getByTestId('right-icon')).toBeInTheDocument();
  });

  it('supports different variants', () => {
    const { rerender } = render(<Button variant="primary">Primary</Button>);
    const button = screen.getByRole('button');

    rerender(<Button size="lg">Large</Button>);
    expect(button).toHaveStyle({ height: '3rem' });
  });
