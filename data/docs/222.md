# DIYAPP Evolution - V12 Core - Arquitetura de Microsserviços

## ADR-001: Arquitetura de Microsserviços V12

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead, Especialista Infra, Especialista Backend

### CONTEXTO:
A V11 do DIYAPP atingiu limites de escalabilidade devido à arquitetura monolítica. O sistema precisa:
1. Suportar 10x mais usuários simultâneos
2. Permitir deploy independente de funcionalidades
3. Isolar falhas entre módulos
4. Facilitar experimentação com diferentes stacks de IA por módulo

### DECISÃO:
Adotar arquitetura de microsserviços baseada em eventos com os seguintes princípios:
- Cada microsserviço é responsável por um domínio de negócio específico
- Comunicação via eventos assíncronos (80%) e APIs síncronas (20%)
- Cada serviço tem seu próprio banco de dados
- API Gateway como ponto único de entrada
- Service Mesh para observabilidade e resiliência

### OPÇÕES CONSIDERADAS:
- **Opção A:** Arquitetura monolítica com módulos - Prós: Simplicidade, deploy único. Contras: Escalabilidade limitada, acoplamento alto.
- **Opção B:** Microsserviços com comunicação síncrona - Prós: Desacoplamento. Contras: Latência em cadeias longas, falhas em cascata.
- **Opção C:** Microsserviços baseados em eventos - Prós: Desacoplamento total, resiliência, escalabilidade. Contras: Complexidade operacional, consistência eventual.

**Opção escolhida:** C - Justificativa: Alinha com necessidade de autonomia da squad e permite evolução independente de cada módulo de IA.

### CONSEQUÊNCIAS:
**Positivas:**
- Escalabilidade horizontal independente por serviço
- Falhas isoladas por domínio
- Deploy contínuo sem downtime total
- Experimentação com diferentes stacks de IA por serviço

**Negativas:**
- Maior complexidade operacional
- Consistência eventual em alguns cenários
- Overhead de rede entre serviços
- Necessidade de orquestração de deploy

**Riscos:**
- Eventos perdidos → Implementar DLQ (Dead Letter Queue) e retry policies
- Latência em cadeias complexas → Monitorar com distributed tracing
- Dados inconsistentes → Implementar SAGA pattern para transações distribuídas

### 2. Padrões de Código
```javascript
// services/auth-service/src/controllers/user.controller.js
/**
 * @api {post} /api/v1/users Create User
 * @apiVersion 1.0.0
 * @apiName CreateUser
 * @apiGroup Users
 */
class UserController {
  async create(req, res) {
    try {
      // Validação de entrada
      const validatedData = await userSchema.validateAsync(req.body);
      
      // Lógica de negócio
      const user = await userService.create(validatedData);
      
      // Publicar evento
      await eventBus.publish('USER_CREATED', {
        userId: user.id,
        email: user.email,
        timestamp: new Date().toISOString()
      });
      
      // Resposta padronizada
      return res.status(201).json({
        success: true,
        data: user,
        metadata: {
          requestId: req.requestId,
          timestamp: new Date().toISOString()
        }
      });
    } catch (error) {
      // Log estruturado
      logger.error({
        message: 'Failed to create user',
        error: error.message,
        stack: error.stack,
        requestId: req.requestId,
        userId: req.user?.id
      });
      
      // Resposta de erro padronizada
      return res.status(error.status || 500).json({
        success: false,
        error: {
          code: error.code || 'INTERNAL_ERROR',
          message: error.message,
          details: process.env.NODE_ENV === 'development' ? error.stack : undefined
        },
        metadata: {
          requestId: req.requestId
        }
      });
    }
  }
}
```

### 5. Template de Microsserviço
```javascript
// services/template-service/src/index.js
const express = require('express');
const helmet = require('helmet');
const cors = require('cors');
const rateLimit = require('express-rate-limit');
const { v4: uuidv4 } = require('uuid');
const logger = require('./libs/logger');
const metrics = require('./libs/metrics');
const eventBus = require('./libs/event-bus');
const healthCheck = require('./health-check');

class Service {
  constructor(config) {
    this.config = config;
    this.app = express();
    this.setupMiddleware();
    this.setupRoutes();
    this.setupEventHandlers();
  }

  setupMiddleware() {
    // Security headers
    this.app.use(helmet());
    
    // CORS configuration
    this.app.use(cors({
      origin: this.config.allowedOrigins,
      credentials: true
    }));
    
    // Request ID for tracing
    this.app.use((req, res, next) => {
      req.requestId = req.headers['x-request-id'] || uuidv4();
      req.startTime = Date.now();
      next();
    });
    
    // Rate limiting
    const limiter = rateLimit({
      windowMs: 15 * 60 * 1000, // 15 minutes
      max: 100, // Limit each IP to 100 requests per windowMs
      message: {
        error: 'Too many requests',
        retryAfter: '15 minutes'
      }
    });
    this.app.use('/api/', limiter);
    
    // Body parsing
    this.app.use(express.json({ limit: '10mb' }));
    this.app.use(express.urlencoded({ extended: true }));
    
    // Request logging
    this.app.use((req, res, next) => {
      logger.info({
        message: 'Incoming request',
        method: req.method,
        url: req.url,
        requestId: req.requestId,
        ip: req.ip
      });
      next();
    });
    
    // Response logging
    this.app.use((req, res, next) => {
      const originalSend = res.send;
      res.send = function(data) {
        const responseTime = Date.now() - req.startTime;
        logger.info({
          message: 'Request completed',
          method: req.method,
          url: req.url,
          statusCode: res.statusCode,
          responseTime,
          requestId: req.requestId
        });
        
        // Record metrics
        metrics.httpRequestDuration.observe(
          { method: req.method, route: req.route?.path, status_code: res.statusCode },
          responseTime / 1000
        );
        
        originalSend.call(this, data);
      };
      next();
    });
  }

  setupRoutes() {
    // Health check endpoint
    this.app.use('/health', healthCheck.router);
    
    // API routes
    this.app.use('/api/v1', require('./routes'));
    
    // 404 handler
    this.app.use((req, res) => {
      res.status(404).json({
        success: false,
        error: {
          code: 'NOT_FOUND',
          message: `Route ${req.method} ${req.url} not found`
        }
      });
    });
    
    // Error handler
    this.app.use((err, req, res, next) => {
      logger.error({
        message: 'Unhandled error',
        error: err.message,
        stack: err.stack,
        requestId: req.requestId
      });
      
      res.status(err.status || 500).json({
        success: false,
        error: {
          code: err.code || 'INTERNAL_ERROR',
          message: process.env.NODE_ENV === 'production' 
            ? 'Internal server error' 
            : err.message
        }
      });
    });
  }

  setupEventHandlers() {
    // Subscribe to relevant events
    eventBus.subscribe('USER_CREATED', async (event) => {
      try {
        await this.handleUserCreated(event);
        logger.info('Successfully processed USER_CREATED event', { eventId: event.eventId });
      } catch (error) {
        logger.error('Failed to process USER_CREATED event', { 
          eventId: event.eventId, 
          error: error.message 
        });
        // Send to DLQ after 3 retries
        await eventBus.sendToDLQ('USER_CREATED', event, error);
      }
    });
  }

  async start() {
    try {
      // Connect to database
      await this.connectDatabase();
      
      // Connect to event bus
      await eventBus.connect();
      
      // Start server
      this.server = this.app.listen(this.config.port, () => {
        logger.info(`Service started on port ${this.config.port}`);
        metrics.serviceStarted.inc();
      });
      
      // Graceful shutdown
      process.on('SIGTERM', () => this.shutdown());
      process.on('SIGINT', () => this.shutdown());
      
    } catch (error) {
      logger.error('Failed to start service', { error: error.message });
      process.exit(1);
    }
  }

  async shutdown() {
    logger.info('Shutting down service...');
    
    // Close server
    if (this.server) {
      await new Promise((resolve) => this.server.close(resolve));
    }
    
    // Disconnect from event bus
    await eventBus.disconnect();
    
    // Close database connections
    await this.disconnectDatabase();
    
    logger.info('Service shutdown complete');
    process.exit(0);
  }
}

module.exports = Service;
```

```dockerfile
# services/template-service/Dockerfile
FROM node:18-alpine AS builder

WORKDIR /app

# Install dependencies
COPY package*.json ./
RUN npm ci --only=production

# Copy source
COPY . .

# Create non-root user
RUN addgroup -g 1001 -S nodejs && \
    adduser -S nodejs -u 1001

FROM node:18-alpine

WORKDIR /app

# Copy from builder
COPY --from=builder /app /app
COPY --from=builder /etc/passwd /etc/passwd

# Switch to non-root user
USER nodejs

# Health check
HEALTHCHECK --interval=30s --timeout=3s --start-period=5s --retries=3 \
  CMD node health-check.js

# Expose port
EXPOSE 3000

# Start service
CMD ["node", "src/index.js"]
```

```sql
WITH daily_executions AS (
    SELECT
        execution_date,
        execution_mode,
        COUNT(DISTINCT execution_id) as total_executions,
        COUNT(DISTINCT session_id) as total_sessions,
        COUNT(DISTINCT user_id) as total_users,
        COUNT(DISTINCT project_id) as total_projects,
        
        -- Success metrics
        SUM(CASE WHEN is_successful THEN 1 ELSE 0 END) as successful_executions,
        AVG(CASE WHEN is_successful THEN 1.0 ELSE 0.0 END) as success_rate,
        
        -- Performance metrics
        AVG(execution_time_ms) as avg_execution_time_ms,
        PERCENTILE_CONT(execution_time_ms, 0.5) as median_execution_time_ms,
        PERCENTILE_CONT(execution_time_ms, 0.95) as p95_execution_time_ms,
        AVG(memory_usage_mb) as avg_memory_usage_mb,
        
        -- Token usage
        SUM(total_tokens) as total_tokens_used,
        AVG(total_tokens) as avg_tokens_per_execution,
        SUM(prompt_tokens) as total_prompt_tokens,
        SUM(completion_tokens) as total_completion_tokens,
        
        -- Cost metrics
        SUM(cost_usd) as total_cost_usd,
        AVG(cost_usd) as avg_cost_per_execution,
        
        -- Error analysis
        COUNT(DISTINCT CASE WHEN NOT is_successful THEN execution_id END) as failed_executions,
        COUNT(DISTINCT error_code) as distinct_error_codes
        
    FROM {{ ref('stg_v12_executions') }}
    WHERE execution_date >= DATE_SUB(CURRENT_DATE(), INTERVAL 90 DAY)
    GROUP BY 1, 2
),

daily_trends AS (
    SELECT
        *,
        LAG(total_executions, 7) OVER (
            PARTITION BY execution_mode 
            ORDER BY execution_date
        ) as executions_7d_ago,
        LAG(success_rate, 1) OVER (
            PARTITION BY execution_mode 
            ORDER BY execution_date
        ) as success_rate_yesterday,
        
        -- Week over week growth
        CASE 
            WHEN executions_7d_ago > 0 
            THEN (total_executions - executions_7d_ago) * 100.0 / executions_7d_ago
            ELSE NULL
        END as wow_growth_pct
        
    FROM daily_executions
)

SELECT
    execution_date,
    execution_mode,
    total_executions,
    total_sessions,
    total_users,
    total_projects,
    successful_executions,
    success_rate,
    avg_execution_time_ms,
    median_execution_time_ms,
    p95_execution_time_ms,
    avg_memory_usage_mb,
    total_tokens_used,
    avg_tokens_per_execution,
    total_prompt_tokens,
    total_completion_tokens,
    total_cost_usd,
    avg_cost_per_execution,
    failed_executions,
    distinct_error_codes,
    executions_7d_ago,
    success_rate_yesterday,
    wow_growth_pct,
    
    -- Alert flags
    CASE 
        WHEN success_rate < 0.95 THEN TRUE
        ELSE FALSE
    END as low_success_rate_alert,
    
    CASE 
        WHEN p95_execution_time_ms > 60000 THEN TRUE
        ELSE FALSE
    END as high_latency_alert,
    
    CURRENT_TIMESTAMP() as _calculated_at
    
FROM daily_trends
ORDER BY execution_date DESC, execution_mode
```

class LogLevel(str, Enum):
    DEBUG = "debug"
    INFO = "info"
    WARNING = "warning"
    ERROR = "error"
    CRITICAL = "critical"

class LogEntry(BaseModel):
    """Modelo estruturado para logs"""
    correlation_id: str = Field(..., description="ID de correlação para rastreamento")
    timestamp: datetime = Field(default_factory=datetime.utcnow)
    level: LogLevel
    service: str
    endpoint: Optional[str] = None
    user_id: Optional[str] = None
    duration_ms: Optional[float] = None
    status: Optional[int] = None
    message: str
    context: Dict[str, Any] = Field(default_factory=dict)
    error_details: Optional[Dict[str, Any]] = None
    
    class Config:
        json_encoders = {
            datetime: lambda v: v.isoformat()
        }

class StructuredLogger:
    """Serviço de logging estruturado com métricas embutidas"""
    
    def __init__(self, service_name: str, redis_client: Optional[redis.Redis] = None):
        self.service_name = service_name
        self.redis_client = redis_client
        
        # Configuração structlog
        structlog.configure(
            processors=[
                structlog.stdlib.filter_by_level,
                structlog.stdlib.add_logger_name,
                structlog.stdlib.add_log_level,
                structlog.stdlib.PositionalArgumentsFormatter(),
                TimeStamper(fmt="iso"),
                structlog.processors.StackInfoRenderer(),
                structlog.processors.format_exc_info,
                structlog.processors.UnicodeDecoder(),
                JSONRenderer()
            ],
            context_class=dict,
            logger_factory=structlog.PrintLoggerFactory(),
            wrapper_class=structlog.stdlib.BoundLogger,
            cache_logger_on_first_use=True,
        )
        
        self.logger = structlog.get_logger(service_name)
        self.metrics_buffer = defaultdict(list)
        
    def _mask_sensitive_data(self, data: Dict[str, Any]) -> Dict[str, Any]:
        """Mascara dados sensíveis nos logs"""
        masked = data.copy()
        sensitive_fields = {'email', 'cpf', 'password', 'token', 'credit_card', 'api_key'}
        
        for field in sensitive_fields:
            if field in masked:
                if isinstance(masked[field], str) and len(masked[field]) > 4:
                    masked[field] = masked[field][:2] + '***' + masked[field][-2:]
                else:
                    masked[field] = '***'
        
        return masked
    
    async def log(self, 
                  level: LogLevel, 
                  message: str, 
                  correlation_id: Optional[str] = None,
                  endpoint: Optional[str] = None,
                  user_id: Optional[str] = None,
                  duration_ms: Optional[float] = None,
                  status: Optional[int] = None,
                  context: Optional[Dict[str, Any]] = None,
                  error_details: Optional[Dict[str, Any]] = None):
        """Log estruturado com métricas"""
        
        correlation_id = correlation_id or str(uuid.uuid4())
        context = self._mask_sensitive_data(context or {})
        
        log_entry = LogEntry(
            correlation_id=correlation_id,
            level=level,
            service=self.service_name,
            endpoint=endpoint,
            user_id=user_id,
            duration_ms=duration_ms,
            status=status,
            message=message,
            context=context,
            error_details=error_details
        )
        
        # Log via structlog
        log_method = getattr(self.logger, level.value)
        log_method(
            **log_entry.dict(exclude_none=True),
            timestamp=log_entry.timestamp.isoformat()
        )
        
        # Buffer de métricas para Redis
        if self.redis_client and endpoint:
            metric_key = f"metrics:{self.service_name}:{endpoint}"
            metric_data = {
                "timestamp": time.time(),
                "duration_ms": duration_ms or 0,
                "status": status or 0,
                "level": level.value
            }
            self.metrics_buffer[metric_key].append(metric_data)
            
            # Flush a cada 10 métricas
            if len(self.metrics_buffer[metric_key]) >= 10:
                await self._flush_metrics(metric_key)
        
        return correlation_id
    
    async def _flush_metrics(self, metric_key: str):
        """Envia métricas bufferizadas para Redis"""
        try:
            metrics = self.metrics_buffer[metric_key]
            if metrics and self.redis_client:
                pipeline = self.redis_client.pipeline()
                for metric in metrics:
                    pipeline.rpush(metric_key, json.dumps(metric))
                pipeline.expire(metric_key, 86400)  # 24 horas
                await pipeline.execute()
                self.metrics_buffer[metric_key].clear()
        except Exception as e:
            self.logger.error("Failed to flush metrics", error=str(e))
    
    @asynccontextmanager
    async def timed_operation(self, 
                             operation_name: str,
                             correlation_id: Optional[str] = None,
                             user_id: Optional[str] = None):
        """Context manager para operações cronometradas"""
        start_time = time.time()
        corr_id = correlation_id or str(uuid.uuid4())
        
        try:
            yield corr_id
        except Exception as e:
            duration_ms = (time.time() - start_time) * 1000
            await self.log(
                level=LogLevel.ERROR,
                message=f"Operation {operation_name} failed",
                correlation_id=corr_id,
                endpoint=operation_name,
                user_id=user_id,
                duration_ms=duration_ms,
                error_details={"error": str(e), "type": type(e).__name__}
            )
            raise
        else:
            duration_ms = (time.time() - start_time) * 1000
            await self.log(
                level=LogLevel.INFO,
                message=f"Operation {operation_name} completed",
                correlation_id=corr_id,
                endpoint=operation_name,
                user_id=user_id,
                duration_ms=duration_ms
            )

class AgentState(str, Enum):
    IDLE = "idle"
    PROCESSING = "processing"
    WAITING = "waiting"
    ERROR = "error"
    COMPLETED = "completed"

class AgentStatus(BaseModel):
    """Estado de um agente individual"""
    agent_id: str
    agent_type: str
    state: AgentState
    current_task: Optional[str] = None
    started_at: datetime = Field(default_factory=datetime.utcnow)
    last_heartbeat: datetime = Field(default_factory=datetime.utcnow)
    metrics: Dict[str, Any] = Field(default_factory=dict)
    error_details: Optional[Dict[str, Any]] = None
    
    class Config:
        json_encoders = {
            datetime: lambda v: v.isoformat()
        }

class WorkflowState(BaseModel):
    """Estado de um workflow completo"""
    workflow_id: str
    correlation_id: str
    current_step: int
    total_steps: int
    agent_states: Dict[str, AgentStatus] = Field(default_factory=dict)
    context: Dict[str, Any] = Field(default_factory=dict)
    created_at: datetime = Field(default_factory=datetime.utcnow)
    updated_at: datetime = Field(default_factory=datetime.utcnow)
    completed_at: Optional[datetime] = None
    error: Optional[str] = None
    
    def is_completed(self) -> bool:
        return self.completed_at is not None
    
    def is_failed(self) -> bool:
        return self.error is not None

class StateManager:
    """Gerenciador de estado distribuído com Redis"""
    
    def __init__(self, 
                 redis_client: redis.Redis,
                 logger: StructuredLogger,
                 heartbeat_timeout: int = 30):
        self.redis = redis_client
        self.logger = logger
        self.heartbeat_timeout = heartbeat_timeout
        
    async def initialize_workflow(self, 
                                 workflow_id: str,
                                 correlation_id: str,
                                 total_steps: int,
                                 initial_context: Dict[str, Any]) -> WorkflowState:
        """Inicializa um novo workflow"""
        workflow_state = WorkflowState(
            workflow_id=workflow_id,
            correlation_id=correlation_id,
            current_step=0,
            total_steps=total_steps,
            context=initial_context
        )
        
        await self._save_workflow_state(workflow_state)
        
        await self.logger.log(
            level=LogLevel.INFO,
            message=f"Workflow {workflow_id} initialized",
            correlation_id=correlation_id,
            context={"total_steps": total_steps}
        )
        
        return workflow_state
    
    async def register_agent(self,
                            workflow_id: str,
                            agent_id: str,
                            agent_type: str) -> AgentStatus:
        """Registra um agente no workflow"""
        workflow_state = await self.get_workflow_state(workflow_id)
        
        agent_status = AgentStatus(
            agent_id=agent_id,
            agent_type=agent_type,
            state=AgentState.IDLE
        )
        
        workflow_state.agent_states[agent_id] = agent_status
        workflow_state.updated_at = datetime.utcnow()
        
        await self._save_workflow_state(workflow_state)
        
        await self.logger.log(
            level=LogLevel.INFO,
            message=f"Agent {agent_id} ({agent_type}) registered to workflow {workflow_id}",
            correlation_id=workflow_state.correlation_id
        )
        
        return agent_status
    
    async def update_agent_state(self,
                                workflow_id: str,
                                agent_id: str,
                                state: AgentState,
                                current_task: Optional[str] = None,
                                metrics: Optional[Dict[str, Any]] = None,
                                error_details: Optional[Dict[str, Any]] = None) -> AgentStatus:
        """Atualiza o estado de um agente"""
        workflow_state = await self.get_workflow_state(workflow_id)
        
        if agent_id not in workflow_state.agent_states:
            raise ValueError(f"Agent {agent_id} not found in workflow {workflow_id}")
        
        agent_status = workflow_state.agent_states[agent_id]
        agent_status.state = state
        agent_status.last_heartbeat = datetime.utcnow()
        
        if current_task is not None:
            agent_status.current_task = current_task
        
        if metrics is not None:
            agent_status.metrics.update(metrics)
        
        if error_details is not None:
            agent_status.error_details = error_details
            workflow_state.error = f"Agent {agent_id} error: {error_details}"
        
        workflow_state.updated_at = datetime.utcnow()
        
        # Avança passo se agente completou
        if state == AgentState.COMPLETED:
            workflow_state.current_step += 1
            
            # Verifica se workflow completou
            if workflow_state.current_step >= workflow_state.total_steps:
                workflow_state.completed_at = datetime.utcnow()
        
        await self._save_workflow_state(workflow_state)
        
        log_level = LogLevel.ERROR if state == AgentState.ERROR else LogLevel.INFO
        await self.logger.log(
            level=log_level,
            message=f"Agent {agent_id} state updated to {state}",
            correlation_id=workflow_state.correlation_id,
            context={
                "workflow_id": workflow_id,
                "current_task": current_task,
                "current_step": workflow_state.current_step
            }
        )
        
        return agent_status
    
    async def get_workflow_state(self, workflow_id: str) -> WorkflowState:
        """Recupera o estado de um workflow"""
        key = f"workflow:{workflow_id}"
        data = await self.redis.get(key)
        
        if not data:
            raise ValueError(f"Workflow {workflow_id} not found")
        
        workflow_dict = json.loads(data)
        
        # Convert strings back to datetime
        for time_field in ['created_at', 'updated_at', 'completed_at', 'started_at', 'last_heartbeat']:
            if time_field in workflow_dict and workflow_dict[time_field]:
                workflow_dict[time_field] = datetime.fromisoformat(workflow_dict[time_field])
        
        # Convert agent states
        if 'agent_states' in workflow_dict:
            for agent_id, agent_dict in workflow_dict['agent_states'].items():
                for time_field in ['started_at', 'last_heartbeat']:
                    if time_field in agent_dict and agent_dict[time_field]:
                        agent_dict[time_field] = datetime.fromisoformat(agent_dict[time_field])
        
        return WorkflowState(**workflow_dict)
    
    async def _save_workflow_state(self, workflow_state: WorkflowState):
        """Salva estado do workflow no Redis"""
        key = f"workflow:{workflow_state.workflow_id}"
        data = workflow_state.json()
        await self.redis.setex(key, 86400, data)  # 24 horas TTL
    
    async def check_agent_health(self, workflow_id: str) -> Dict[str, bool]:
        """Verifica saúde dos agentes baseado em heartbeat"""
        workflow_state = await self.get_workflow_state(workflow_id)
        health_status = {}
        
        for agent_id, agent_status in workflow_state.agent_states.items():
            time_since_heartbeat = (datetime.utcnow() - agent_status.last_heartbeat).total_seconds()
            is_healthy = time_since_heartbeat < self.heartbeat_timeout
            
            if not is_healthy and agent_status.state != AgentState.COMPLETED:
                agent_status.state = AgentState.ERROR
                agent_status.error_details = {"reason": "heartbeat_timeout", "seconds_since": time_since_heartbeat}
                
                await self.logger.log(
                    level=LogLevel.ERROR,
                    message=f"Agent {agent_id} heartbeat timeout",
                    correlation_id=workflow_state.correlation_id,
                    context={
                        "seconds_since_heartbeat": time_since_heartbeat,
                        "timeout": self.heartbeat_timeout
                    }
                )
            
            health_status[agent_id] = is_healthy
        
        if any(not healthy for healthy in health_status.values()):
            workflow_state.updated_at = datetime.utcnow()
            await self._save_workflow_state(workflow_state)
        
        return health_status

class AgentCapability(str, Enum):
    CODE_GENERATION = "code_generation"
    CODE_REVIEW = "code_review"

    DOCUMENTATION = "documentation"
    DEPLOYMENT = "deployment"

class AgentRequest(BaseModel):
    """Request para execução de agente"""
    workflow_id: str
    agent_type: str
    capability: AgentCapability
    input_data: Dict[str, Any]
    priority: int = Field(default=1, ge=1, le=10)
    timeout_seconds: int = Field(default=30, ge=1, le=300)
    
    @validator('input_data')
    def validate_input_data(cls, v):
        if not isinstance(v, dict):
            raise ValueError("input_data must be a dictionary")
        return v

class AgentResponse(BaseModel):
    """Response de execução de agente"""
    success: bool
    output_data: Optional[Dict[str, Any]] = None
    execution_time_ms: float
    error_message: Optional[str] = None
    metrics: Dict[str, Any] = Field(default_factory=dict)

class Agent(ABC):
    """Interface base para todos os agentes"""
    
    def __init__(self, agent_id: str, capability: AgentCapability):
        self.agent_id = agent_id
        self.capability = capability
        self.state_manager: Optional[StateManager] = None
        self.logger: Optional[StructuredLogger] = None
    
    @abstractmethod
    async def execute(self, input_data: Dict[str, Any]) -> AgentResponse:
        """Método principal de execução do agente"""
        pass
    
    async def heartbeat(self, workflow_id: str):
        """Envia heartbeat para o gerenciador de estado"""
        if self.state_manager:
            await self.state_manager.update_agent_state(
                workflow_id=workflow_id,
                agent_id=self.agent_id,
                state=AgentState.PROCESSING,
                metrics={"last_heartbeat": time.time()}
            )
