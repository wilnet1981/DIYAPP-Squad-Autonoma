# DIYAPP Evolution - V12 Core - Arquitetura Técnica

## 1. ADR-001: Arquitetura V12 - Padrões e Estrutura

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead V12

### CONTEXTO:
O DIYAPP evoluiu para V11 com funcionalidades robustas, mas acumulou débito técnico em:
1. Estrutura de pastas inconsistente entre módulos
2. Padrões de código variáveis (ES5/ES6/TypeScript misturados)
3. Cache implementado de forma ad-hoc sem estratégia unificada
4. Monitoramento básico sem APM integrado
5. Dificuldade de onboarding devido à falta de padrões documentados

A V12 precisa estabelecer fundamentos técnicos sólidos para os próximos 2 anos de evolução.

### DECISÃO:
Adotar arquitetura modular baseada em Clean Architecture com os seguintes pilares:
1. **Estrutura de pastas** padronizada por domínio/funcionalidade
2. **Padrões de código** TypeScript estrito com ESLint/Prettier
3. **Estratégia de cache** em duas camadas (Redis + Local Memory)
4. **APM** com OpenTelemetry + Prometheus + Grafana
5. **Plano de migração** gradual usando Modo Hive

### OPÇÕES CONSIDERADAS:

**Opção A: Refatoração Big Bang**
- Prós: Consistência imediata, padrões uniformes desde dia 1
- Contras: Risco alto de regressão, downtime prolongado

**Opção B: Migração Gradual (Modo Hive)**
- Prós: Zero downtime, rollback fácil, aprendizado contínuo
- Contras: Complexidade de coexistência V11/V12, overhead de compatibilidade

**Opção C: Nova Base Paralela**
- Prós: Liberdade total na V12, sem amarras da V11
- Contras: Custo de desenvolvimento duplicado, migração de dados complexa

**Opção escolhida: B** - Justificativa: Alinhamento com requisito de estabilidade 100% autônoma. Permite refatoração paralela sem impacto nos usuários.

### CONSEQUÊNCIAS:
**Positivas:**
- Zero downtime durante migração
- Rollback instantâneo se necessário
- Time pode aprender novos padrões gradualmente
- Métricas de performance comparáveis entre versões

**Negativas:**
- Overhead de ~15% no código para compatibilidade
- Necessidade de feature flags para funcionalidades novas
- Duplicação temporária de algumas lógicas

**Riscos:**
1. **Complexidade excessiva:** Mitigar com documentação clara e sessões de pairing
2. **Performance overhead:** Monitorar com APM e otimizar gradualmente
3. **Confusão entre versões:** Usar naming conventions claras (v11_, v12_)

  set<T>(key: string, value: T, ttl?: number): Promise<void>;
  delete(key: string): Promise<void>;
  clear(): Promise<void>;
  has(key: string): Promise<boolean>;

// src/infrastructure/cache/two-layer-cache.ts
export class TwoLayerCache implements CacheStrategy {
  constructor(
    private readonly localCache: CacheStrategy,
    private readonly distributedCache: CacheStrategy,
    private readonly options: {
      localTtl: number; // 60 segundos
      distributedTtl: number; // 3600 segundos
      staleWhileRevalidate: boolean;
    }
  ) {}

  async get<T>(key: string): Promise<T | null> {
    // 1. Tentar local cache
    const localValue = await this.localCache.get<T>(key);
    if (localValue !== null) {
      return localValue;
    }

    // 2. Tentar distributed cache
    const distributedValue = await this.distributedCache.get<T>(key);
    if (distributedValue !== null) {
      // Popula local cache
      await this.localCache.set(key, distributedValue, this.options.localTtl);
      return distributedValue;
    }

    return null;
  }

  async set<T>(key: string, value: T, ttl?: number): Promise<void> {
    const effectiveTtl = ttl || this.options.distributedTtl;
    
    // Write-through: ambas as camadas
    await Promise.all([
      this.localCache.set(key, value, this.options.localTtl),
      this.distributedCache.set(key, value, effectiveTtl)
    ]);
  }

  // Implementações de delete, clear, has...
}

    return data ? JSON.parse(data) : null;

    if (ttl) {
      await this.redisClient.setex(key, ttl, serialized);
    } else {
      await this.redisClient.set(key, serialized);
    }
  }

export class MemoryCache implements CacheStrategy {
  private cache = new Map<string, { value: any; expires: number }>();

  async get<T>(key: string): Promise<T | null> {
    const item = this.cache.get(key);
    if (!item) return null;
    
    if (Date.now() > item.expires) {
      this.cache.delete(key);
      return null;
    }
    
    return item.value as T;
  }

  async set<T>(key: string, value: T, ttl: number = 60000): Promise<void> {
    this.cache.set(key, {
      value,
      expires: Date.now() + ttl
    });
  }
}

export function Cacheable(options: {
  key: string | ((...args: any[]) => string);
  ttl?: number;
  layer?: 'local' | 'distributed' | 'both';
}) {
  return function (
    target: any,
    propertyName: string,
    descriptor: PropertyDescriptor
  ) {
    const originalMethod = descriptor.value;
    
    descriptor.value = async function (...args: any[]) {
      const cacheKey = typeof options.key === 'function' 
        ? options.key(...args) 
        : options.key;
      
      const cacheService = (this as any).cacheService as TwoLayerCache;
      
      // Tentar cache primeiro
      if (cacheService) {
        const cached = await cacheService.get(cacheKey);
        if (cached !== null) {
          return cached;
        }
      }
      
      // Executar método original
      const result = await originalMethod.apply(this, args);
      
      // Armazenar em cache
      if (cacheService && result !== undefined) {
        await cacheService.set(cacheKey, result, options.ttl);
      }
      
      return result;
    };
    
    return descriptor;
  };
}

import asyncio
from typing import Optional, Callable, Any
from datetime import datetime, timedelta
import functools
import logging

logger = logging.getLogger(__name__)

class CircuitBreaker:
    """Implementação de Circuit Breaker"""
    
    def __init__(
        self,
        name: str,
        failure_threshold: int = 5,
        recovery_timeout: int = 60,
        expected_exceptions: tuple = (Exception,)
    ):
        self.name = name
        self.failure_threshold = failure_threshold
        self.recovery_timeout = recovery_timeout
        self.expected_exceptions = expected_exceptions
        
        self.failures = 0
        self.state = "CLOSED"  # CLOSED, OPEN, HALF_OPEN
        self.last_failure_time: Optional[datetime] = None
        self._lock = asyncio.Lock()
    
    async def execute(self, func: Callable, *args, **kwargs) -> Any:
        """Executa a função com circuit breaker"""
        async with self._lock:
            # Verificar se o circuito está aberto
            if self.state == "OPEN":
                # Verificar se já pode tentar recuperar
                if self.last_failure_time and (
                    datetime.now() - self.last_failure_time > timedelta(seconds=self.recovery_timeout)
                ):
                    self.state = "HALF_OPEN"
                    logger.info(f"Circuit {self.name} moving to HALF_OPEN")
                else:
                    raise CircuitOpenError(f"Circuit {self.name} is OPEN")
            
            try:
                # Executar a função
                result = await func(*args, **kwargs)
                
                # Se sucesso em HALF_OPEN, fechar o circuito
                if self.state == "HALF_OPEN":
                    self.state = "CLOSED"
                    self.failures = 0
                    self.last_failure_time = None
                    logger.info(f"Circuit {self.name} reset to CLOSED")
                
                return result
                
            except self.expected_exceptions as e:
                # Incrementar falhas
                self.failures += 1
                self.last_failure_time = datetime.now()
                
                # Se atingir threshold, abrir circuito
                if self.failures >= self.failure_threshold:
                    self.state = "OPEN"
                    logger.error(
                        f"Circuit {self.name} opened after {self.failures} failures",
                        extra={"last_error": str(e)}
                    )
                
                raise
    
    def get_state(self) -> dict:
        """Retorna o estado atual do circuit breaker"""
        return {
            "name": self.name,
            "state": self.state,
            "failures": self.failures,
            "last_failure_time": self.last_failure_time.isoformat() if self.last_failure_time else None
        }

class CircuitOpenError(Exception):
    """Exceção lançada quando o circuit breaker está aberto"""

def with_retry(
    max_attempts: int = 3,
    delay: float = 0.1,
    max_delay: float = 10.0,
    backoff_factor: float = 2.0,
    jitter: bool = True,
    retry_on_exceptions: tuple = (Exception,)
):
    """Decorator para retry com backoff exponencial"""
    
    def decorator(func):
        @functools.wraps(func)
        async def wrapper(*args, **kwargs):
            last_exception = None
            
            for attempt in range(1, max_attempts + 1):
                try:
                    return await func(*args, **kwargs)
                    
                except retry_on_exceptions as e:
                    last_exception = e
                    
                    # Não retry em 4xx errors (erros do cliente)
                    if hasattr(e, 'status_code') and 400 <= e.status_code < 500:
                        raise
                    
                    # Última tentativa, propaga exceção
                    if attempt == max_attempts:
                        logger.warning(
                            f"Final attempt failed for {func.__name__}",
                            extra={"attempt": attempt, "error": str(e)}
                        )
                        raise
                    
                    # Calcular delay com backoff exponencial
                    current_delay = min(delay * (backoff_factor ** (attempt - 1)), max_delay)
                    
                    # Adicionar jitter
                    if jitter:
                        current_delay = current_delay * (0.5 + (0.5 * (attempt / max_attempts)))
                    
                    logger.info(
                        f"Retrying {func.__name__}, attempt {attempt}/{max_attempts}",
                        extra={"delay": current_delay, "error": str(e)}
                    )
                    
                    await asyncio.sleep(current_delay)
            
            # Nunca deve chegar aqui
            raise last_exception
        
        return wrapper
    
    return decorator

import { create } from 'zustand';
import { persist } from 'zustand/middleware';
import { immer } from 'zustand/middleware/immer';

interface User {
  id: string;
  name: string;
  email: string;
  avatar?: string;
}

interface AppState {
  // State
  user: User | null;
  theme: 'light' | 'dark';
  sidebarOpen: boolean;
  notifications: Array<{
    id: string;
    title: string;
    message: string;
    type: 'info' | 'success' | 'warning' | 'error';
    read: boolean;
    timestamp: Date;
  }>;
  
  // Actions
  setUser: (user: User | null) => void;
  toggleTheme: () => void;
  toggleSidebar: () => void;
  addNotification: (notification: Omit<AppState['notifications'][0], 'id' | 'timestamp' | 'read'>) => void;
  markNotificationAsRead: (id: string) => void;
  clearNotifications: () => void;
}

export const useAppStore = create<AppState>()(
  persist(
    immer((set) => ({
      // Initial state
      user: null,
      theme: 'light',
      sidebarOpen: true,
      notifications: [],

      // Actions
      setUser: (user) =>
        set((state) => {
          state.user = user;
        }),

      toggleTheme: () =>
        set((state) => {
          state.theme = state.theme === 'light' ? 'dark' : 'light';
        }),

      toggleSidebar: () =>
        set((state) => {
          state.sidebarOpen = !state.sidebarOpen;
        }),

      addNotification: (notification) =>
        set((state) => {
          state.notifications.unshift({
            ...notification,
            id: crypto.randomUUID(),
            timestamp: new Date(),
            read: false,
          });
        }),

      markNotificationAsRead: (id) =>
        set((state) => {
          const notification = state.notifications.find((n) => n.id === id);
          if (notification) {
            notification.read = true;
          }
        }),

      clearNotifications: () =>
        set((state) => {
          state.notifications = [];
        }),
    })),
    {
      name: 'app-storage',
      partialize: (state) => ({
        user: state.user,
        theme: state.theme,
      }),
    }
  )
);

    // Validação JWT com múltiplas camadas de segurança
    async validateJWT(req, res, next) {
        const token = this.extractToken(req);
        
        if (!token) {
            return this.denyAccess(res, 'TOKEN_MISSING', 'Token de autenticação não fornecido');
        }

        // Verifica se token foi revogado
        if (this.revokedTokens.has(token)) {
            return this.denyAccess(res, 'TOKEN_REVOKED', 'Token foi revogado');
        }

        try {
            // Verificação 1: Assinatura
            const decoded = jwt.verify(token, SECURITY_CONFIG.JWT_SECRET, {
                algorithms: ['HS256'],
                clockTolerance: 30 // 30 segundos de tolerância
            });

            // Verificação 2: Claims obrigatórios
            if (!decoded.sub || !decoded.iss || !decoded.aud) {
                return this.denyAccess(res, 'INVALID_CLAIMS', 'Token com claims inválidas');
            }

            // Verificação 3: Expiração
            if (decoded.exp * 1000 < Date.now()) {
                return this.denyAccess(res, 'TOKEN_EXPIRED', 'Token expirado');
            }

            // Verificação 4: IP binding (opcional para alta segurança)
            if (decoded.ip && decoded.ip !== req.ip) {
                logger.warn(`IP mismatch for user ${decoded.sub}`, {
                    tokenIp: decoded.ip,
                    requestIp: req.ip
                });
                // Não negamos acesso, apenas logamos para auditoria
            }

            // Verificação 5: Session validation
            const sessionValid = await this.validateSession(decoded.sub, decoded.jti);
            if (!sessionValid) {
                return this.denyAccess(res, 'SESSION_INVALID', 'Sessão inválida ou expirada');
            }

            // Adiciona informações ao request
            req.user = {
                id: decoded.sub,
                roles: decoded.roles || ['user'],
                permissions: decoded.perms || [],
                sessionId: decoded.jti,
                mfaVerified: decoded.mfa || false
            };

            // Log de acesso bem-sucedido
            logger.info(`Access granted for user ${decoded.sub}`, {
                endpoint: req.path,
                method: req.method,
                ip: req.ip,
                userAgent: req.get('user-agent')
            });

            next();
        } catch (error) {
            logger.error('JWT validation failed', {
                error: error.message,
                token: this.maskToken(token),
                ip: req.ip
            });

            if (error.name === 'TokenExpiredError') {
                return this.denyAccess(res, 'TOKEN_EXPIRED', 'Token expirado');
            } else if (error.name === 'JsonWebTokenError') {
                return this.denyAccess(res, 'TOKEN_INVALID', 'Token inválido');
            } else {
                return this.denyAccess(res, 'AUTH_ERROR', 'Erro na autenticação');
            }
        }

            if (!hasPermission) {

                    resource,
                    userPermissions: req.user.permissions,
                    endpoint: req.path
                });

            }

            // Log de auditoria para operações privilegiadas

                logger.audit(`Privileged operation executed by ${req.user.id}`, {

    // MFA Service
    async requireMFA(req, res, next) {
        if (!req.user) {

        }

        }

        const authHeader = req.headers.authorization;

        }
        return req.cookies?.access_token || req.query?.token;

const winston = require('winston');
const DailyRotateFile = require('winston-daily-rotate-file');
const EventEmitter = require('events');
const promClient = require('prom-client');

// Configuração de logging
const logger = winston.createLogger({
  level: 'info',
  format: winston.format.combine(
    winston.format.timestamp(),
    winston.format.json()
  ),
  transports: [
    new winston.transports.Console(),
    new DailyRotateFile({
      filename: 'logs/autohealing-%DATE%.log',
      datePattern: 'YYYY-MM-DD',
      zippedArchive: true,
      maxSize: '20m',
      maxFiles: '30d'
    })
  ]
});

// Métricas Prometheus
const register = new promClient.Registry();
promClient.collectDefaultMetrics({ register });

const healthCheckCounter = new promClient.Counter({
  name: 'autohealing_health_checks_total',
  help: 'Total de verificações de saúde realizadas',
  labelNames: ['service', 'status']
});

const healingActionsCounter = new promClient.Counter({
  name: 'autohealing_actions_total',
  help: 'Total de ações de auto-healing executadas',
  labelNames: ['action_type', 'success']
});

const serviceLatency = new promClient.Histogram({
  name: 'autohealing_service_latency_seconds',
  help: 'Latência dos serviços monitorados',
  labelNames: ['service'],
  buckets: [0.1, 0.5, 1, 2, 5, 10]
});

class HealthMonitor extends EventEmitter {
  constructor(config) {
    super();
    this.config = config;
    this.services = new Map();
    this.checkInterval = config.checkInterval || 30000; // 30 segundos
    this.failureThreshold = config.failureThreshold || 3;
    this.recoveryThreshold = config.recoveryThreshold || 2;
    this.checkTimer = null;
    
    // Inicializar métricas
    register.registerMetric(healthCheckCounter);
    register.registerMetric(healingActionsCounter);
    register.registerMetric(serviceLatency);
  }

  async registerService(serviceConfig) {
    const service = {
      ...serviceConfig,
      consecutiveFailures: 0,
      consecutiveSuccesses: 0,
      status: 'healthy',
      lastCheck: null,
      metrics: {
        totalChecks: 0,
        totalFailures: 0,
        averageResponseTime: 0
      }
    };
    
    this.services.set(serviceConfig.name, service);
    logger.info(`Serviço registrado: ${serviceConfig.name}`, { service: serviceConfig.name });
  }

  async startMonitoring() {
    logger.info('Iniciando monitoramento de saúde...');
    
    this.checkTimer = setInterval(async () => {
      await this.performHealthChecks();
    }, this.checkInterval);
    
    // Primeira verificação imediata
    await this.performHealthChecks();
  }

  async performHealthChecks() {
    const checkPromises = [];
    
    for (const [serviceName, service] of this.services) {
      checkPromises.push(this.checkServiceHealth(serviceName, service));
    }
    
    await Promise.allSettled(checkPromises);
  }

  async checkServiceHealth(serviceName, service) {
    const startTime = Date.now();
    let checkStatus = 'healthy';
    
    try {
      const response = await this.executeHealthCheck(service);
      const latency = Date.now() - startTime;
      
      serviceLatency.observe({ service: serviceName }, latency / 1000);
      
      if (response.healthy) {
        service.consecutiveFailures = 0;
        service.consecutiveSuccesses++;
        service.lastCheck = new Date();
        service.metrics.totalChecks++;
        service.metrics.averageResponseTime = 
          (service.metrics.averageResponseTime * (service.metrics.totalChecks - 1) + latency) / 
          service.metrics.totalChecks;
        
        // Verificar se recuperou
        if (service.status === 'unhealthy' && 
            service.consecutiveSuccesses >= this.recoveryThreshold) {
          service.status = 'healthy';
          this.emit('serviceRecovered', { serviceName, service });
          logger.info(`Serviço recuperado: ${serviceName}`, { service: serviceName });
        }
        
        healthCheckCounter.inc({ service: serviceName, status: 'success' });
      } else {
        throw new Error(`Health check failed: ${response.message}`);
      }
    } catch (error) {
      const latency = Date.now() - startTime;
      serviceLatency.observe({ service: serviceName }, latency / 1000);
      
      service.consecutiveFailures++;
      service.consecutiveSuccesses = 0;
      service.metrics.totalChecks++;
      service.metrics.totalFailures++;
      
      healthCheckCounter.inc({ service: serviceName, status: 'failure' });
      
      logger.warn(`Falha no serviço ${serviceName}: ${error.message}`, {
        service: serviceName,
        error: error.message,
        consecutiveFailures: service.consecutiveFailures
      });
      
      // Verificar se atingiu threshold de falha
      if (service.consecutiveFailures >= this.failureThreshold && 
          service.status !== 'unhealthy') {
        service.status = 'unhealthy';
        checkStatus = 'unhealthy';
        
        this.emit('serviceUnhealthy', { 
          serviceName, 
          service,
          error: error.message,
          consecutiveFailures: service.consecutiveFailures
        });
        
        logger.error(`Serviço marcado como não saudável: ${serviceName}`, {
          service: serviceName,
          consecutiveFailures: service.consecutiveFailures
        });
      }
    }
    
    return { serviceName, status: checkStatus };
  }

  async executeHealthCheck(service) {
    // Implementação específica do tipo de verificação
    switch (service.type) {
      case 'http':
        return await this.checkHttpService(service);
      case 'tcp':
        return await this.checkTcpService(service);
      case 'docker':
        return await this.checkDockerContainer(service);
      case 'custom':
        return await service.checkFunction();
      default:
        throw new Error(`Tipo de serviço não suportado: ${service.type}`);
    }
  }

  async checkHttpService(service) {
    const axios = require('axios');
    const timeout = service.timeout || 5000;
    
    try {
      const response = await axios.get(service.url, { 
        timeout,
        validateStatus: (status) => status < 500
      });
      
      return {
        healthy: true,
        statusCode: response.status,
        responseTime: response.headers['x-response-time'] || null
      };
    } catch (error) {
      return {
        healthy: false,
        message: error.message,
        code: error.code
      };
    }
  }

  async checkTcpService(service) {
    const net = require('net');
    const { host, port } = service;
    const timeout = service.timeout || 3000;
    
    return new Promise((resolve) => {
      const socket = new net.Socket();
      let timedOut = false;
      
      const timer = setTimeout(() => {
        timedOut = true;
        socket.destroy();
        resolve({ healthy: false, message: 'Connection timeout' });
      }, timeout);
      
      socket.on('connect', () => {
        clearTimeout(timer);
        socket.destroy();
        resolve({ healthy: true });
      });
      
      socket.on('error', (error) => {
        clearTimeout(timer);
        if (!timedOut) {
          resolve({ healthy: false, message: error.message });
        }
      });
      
      socket.connect(port, host);
    });
  }

  async checkDockerContainer(service) {
    const Docker = require('dockerode');
    const docker = new Docker();
    
    try {
      const container = docker.getContainer(service.containerId);
      const data = await container.inspect();
      
      return {
        healthy: data.State.Status === 'running',
        status: data.State.Status,
        exitCode: data.State.ExitCode,
        startedAt: data.State.StartedAt
      };
    } catch (error) {
      return {
        healthy: false,
        message: error.message
      };
    }
  }

  stopMonitoring() {
    if (this.checkTimer) {
      clearInterval(this.checkTimer);
      this.checkTimer = null;
      logger.info('Monitoramento parado');
    }
  }

  getServiceStatus(serviceName) {
    const service = this.services.get(serviceName);
    return service ? {
      name: serviceName,
      status: service.status,
      consecutiveFailures: service.consecutiveFailures,
      consecutiveSuccesses: service.consecutiveSuccesses,
      lastCheck: service.lastCheck,
      metrics: { ...service.metrics }
    } : null;
  }

  getAllServicesStatus() {
    const status = {};
    for (const [name, service] of this.services) {
      status[name] = this.getServiceStatus(name);
    }
    return status;
  }

  getMetrics() {
    return register.metrics();
  }
}

module.exports = HealthMonitor;

const winston = require('winston');
const EventEmitter = require('events');

class AutoHealer extends EventEmitter {
  constructor(config) {
    super();
    this.config = config;
    this.healingStrategies = new Map();
    this.healingHistory = [];
    this.maxHistorySize = config.maxHistorySize || 100;
    this.cooldownPeriod = config.cooldownPeriod || 300000; // 5 minutos
    this.lastHealingAttempts = new Map();
    
    this.logger = winston.createLogger({
      level: 'info',
      format: winston.format.combine(
        winston.format.timestamp(),
        winston.format.json()
      ),
      transports: [
        new winston.transports.Console(),
        new winston.transports.File({ 
          filename: 'logs/healing-actions.log' 
        })
      ]
    });
  }

  registerHealingStrategy(serviceType, strategy) {
    this.healingStrategies.set(serviceType, strategy);
    this.logger.info(`Estratégia de healing registrada para: ${serviceType}`);
  }

  async triggerHealing(serviceInfo) {
    const { serviceName, service, error } = serviceInfo;
    
    // Verificar cooldown
    const lastAttempt = this.lastHealingAttempts.get(serviceName);
    if (lastAttempt && (Date.now() - lastAttempt) < this.cooldownPeriod) {
      this.logger.warn(`Healing em cooldown para: ${serviceName}`, {
        service: serviceName,
        cooldownRemaining: this.cooldownPeriod - (Date.now() - lastAttempt)
      });
      return { success: false, reason: 'cooldown' };
    }

    this.logger.info(`Iniciando auto-healing para: ${serviceName}`, {
      service: serviceName,
      error,
      consecutiveFailures: service.consecutiveFailures
    });

    // Determinar estratégia baseada no tipo de serviço
    const strategy = this.healingStrategies.get(service.type) || 
                     this.healingStrategies.get('default');
    
    if (!strategy) {
      this.logger.error(`Nenhuma estratégia de healing encontrada para: ${service.type}`);
      return { success: false, reason: 'no_strategy' };
    }

    try {
      this.lastHealingAttempts.set(serviceName, Date.now());
      
      // Executar estratégia de healing
      const result = await strategy.execute(serviceInfo);
      
      // Registrar no histórico
      const healingRecord = {
        timestamp: new Date(),
        serviceName,
        serviceType: service.type,
        action: result.action,
        success: result.success,
        details: result.details,
        error: result.error
      };
      
      this.addToHistory(healingRecord);
      
      // Emitir evento de healing
      this.emit('healingAttempted', healingRecord);
      
      if (result.success) {
        this.logger.info(`Auto-healing bem-sucedido para: ${serviceName}`, {
          service: serviceName,
          action: result.action,
          details: result.details
        });
        
        // Atualizar métricas
        const { healingActionsCounter } = require('./health-monitor');
        healingActionsCounter.inc({ action_type: result.action, success: 'true' });
      } else {
        this.logger.error(`Auto-healing falhou para: ${serviceName}`, {
          service: serviceName,
          action: result.action,
          error: result.error
        });
        
        // Atualizar métricas
        const { healingActionsCounter } = require('./health-monitor');
        healingActionsCounter.inc({ action_type: result.action, success: 'false' });
        
        // Escalonar para ação mais agressiva se disponível
        if (strategy.escalate) {
          this.logger.info(`Escalonando healing para: ${serviceName}`);
          const escalatedResult = await strategy.escalate(serviceInfo);
          return escalatedResult;
        }
      }
      
      return result;
      
    } catch (error) {
      this.logger.error(`Erro durante auto-healing: ${error.message}`, {
        service: serviceName,
        error: error.stack
      });
      
      return { 
        success: false, 
        reason: 'execution_error',
        error: error.message 
      };
    }
  }

  addToHistory(record) {
    this.healingHistory.unshift(record);
    
    // Manter histórico limitado
    if (this.healingHistory.length > this.maxHistorySize) {
      this.healingHistory = this.healingHistory.slice(0, this.maxHistorySize);
    }
  }

  getHealingHistory(limit = 20) {
    return this.healingHistory.slice(0, limit);
  }

  getServiceHealingHistory(serviceName, limit = 10) {
    return this.healingHistory
      .filter(record => record.serviceName === serviceName)
      .slice(0, limit);
  }

  clearCooldown(serviceName) {
    this.lastHealingAttempts.delete(serviceName);
    this.logger.info(`Cooldown limpo para: ${serviceName}`);
  }

  getCooldownStatus(serviceName) {
    const lastAttempt = this.lastHealingAttempts.get(serviceName);
    if (!lastAttempt) return { inCooldown: false };
    
    const remaining = this.cooldownPeriod - (Date.now() - lastAttempt);
    return {
      inCooldown: remaining > 0,
      remainingMs: remaining > 0 ? remaining : 0,
      lastAttempt: new Date(lastAttempt)
    };
  }
}

    { duration: '2m', target: 200 },  // Pico de carga

  },
};

export default function () {
  const baseUrl = __ENV.BASE_URL || 'http://localhost:3000';
  const vuId = __VU;

  // 1. Criação de agente sob estresse
  const startCreate = Date.now();
  const agentRes = http.post(`${baseUrl}/api/agents`, JSON.stringify({
    name: `stress-agent-${vuId}-${Date.now()}`,
    type: 'stress-test',
    instructions: 'Handle concurrent load'
  }), {
    headers: { 'Content-Type': 'application/json' },
  });

  agentCreationTime.add(Date.now() - startCreate);
  
  check(agentRes, {
    'agent created under stress': (r) => r.status === 201,
  });

  if (agentRes.status === 201) {
    const agentId = JSON.parse(agentRes.body).id;

    // 2. Execução de tarefa concorrente
    const taskStart = Date.now();
    const taskRes = http.post(`${baseUrl}/api/agents/${agentId}/tasks`, JSON.stringify({
      type: 'process',
      items: Array.from({ length: 10 }, (_, i) => `item-${i}`)
    }));

    taskProcessingTime.add(Date.now() - taskStart);

    check(taskRes, {
      'task processed under load': (r) => r.status === 200,
      'task completed successfully': (r) => JSON.parse(r.body).status === 'completed',
    });

    // 3. Verificação de estado do agente
    const statusRes = http.get(`${baseUrl}/api/agents/${agentId}/status`);
    check(statusRes, {
      'agent remains stable': (r) => JSON.parse(r.body).status === 'active',
    });
  }

  errorRate.add(agentRes.status >= 400);
  sleep(Math.random() * 2 + 1); // Sleep aleatório entre 1-3s
}

const jwt = require('jsonwebtoken');
const bcrypt = require('bcrypt');
const crypto = require('crypto');
const { RateLimiterRedis } = require('rate-limiter-flexible');

class AuthSystem {
  constructor(redisClient) {
    this.redis = redisClient;
    this.jwtSecret = process.env.JWT_SECRET || crypto.randomBytes(64).toString('hex');
    this.jwtRefreshSecret = process.env.JWT_REFRESH_SECRET || crypto.randomBytes(64).toString('hex');
    
    // Rate limiter para tentativas de login
    this.loginLimiter = new RateLimiterRedis({
      storeClient: redisClient,
      keyPrefix: 'login_fail',
      points: 5, // 5 tentativas
      duration: 900, // 15 minutos
      blockDuration: 1800 // Bloqueia por 30 minutos após exceder
    });
  }

  async generateTokens(user) {
    const accessToken = jwt.sign(
      {
        userId: user.id,
        email: user.email,
        role: user.role,
        permissions: user.permissions
      },
      this.jwtSecret,
      {
        expiresIn: '15m', // Access token curto
        issuer: 'diyapp-v12',
        audience: 'user'
      }
    );

    const refreshToken = jwt.sign(
      {
        userId: user.id,
        tokenVersion: user.tokenVersion || 0
      },
      this.jwtRefreshSecret,
      {
        expiresIn: '7d', // Refresh token mais longo
        issuer: 'diyapp-v12',
        audience: 'refresh'
      }
    );

    // Armazenar refresh token no Redis com TTL
    await this.redis.setex(
      `refresh_token:${user.id}:${crypto.createHash('sha256').update(refreshToken).digest('hex')}`,
      7 * 24 * 60 * 60, // 7 dias
      JSON.stringify({
        userId: user.id,
        createdAt: new Date().toISOString(),
        userAgent: 'web' // Poderia ser extraído do request
      })
    );

    return { accessToken, refreshToken };
  }

  async verifyAccessToken(token) {
    try {
      const decoded = jwt.verify(token, this.jwtSecret, {
        issuer: 'diyapp-v12',
        audience: 'user'
      });

      // Verificar se o token está na blacklist (logout)
      const isBlacklisted = await this.redis.get(`token_blacklist:${token}`);
      if (isBlacklisted) {
        throw new Error('Token invalidado');
      }

      return decoded;
    } catch (error) {
      throw new Error(`Token inválido: ${error.message}`);
    }
  }

  async verifyRefreshToken(token) {
    try {
      const decoded = jwt.verify(token, this.jwtRefreshSecret, {
        issuer: 'diyapp-v12',
        audience: 'refresh'
      });

      // Verificar se o refresh token existe no Redis
      const tokenHash = crypto.createHash('sha256').update(token).digest('hex');
      const storedToken = await this.redis.get(`refresh_token:${decoded.userId}:${tokenHash}`);
      
      if (!storedToken) {
        throw new Error('Refresh token não encontrado');
      }

      return decoded;
    } catch (error) {
      throw new Error(`Refresh token inválido: ${error.message}`);
    }
  }

  async invalidateTokens(userId, token) {
    // Adicionar access token à blacklist (TTL de 15 minutos)
    await this.redis.setex(
      `token_blacklist:${token}`,
      15 * 60, // 15 minutos
      'invalidated'
    );

    // Invalidar todos os refresh tokens do usuário
    const keys = await this.redis.keys(`refresh_token:${userId}:*`);
    if (keys.length > 0) {
      await Promise.all(keys.map(key => this.redis.del(key)));
    }

    // Incrementar token version para invalidar tokens futuros
    await this.redis.incr(`token_version:${userId}`);
  }

  async hashPassword(password) {
    const saltRounds = 12;
    return bcrypt.hash(password, saltRounds);
  }

  async verifyPassword(password, hash) {
    return bcrypt.compare(password, hash);
  }

  async checkLoginAttempts(ip) {
    try {
      await this.loginLimiter.consume(ip);
      return { allowed: true };
    } catch (error) {
      return {
        allowed: false,
        retryAfter: Math.ceil(error.msBeforeNext / 1000) || 1800
      };
    }
  }

  async recordFailedLogin(ip) {
    await this.loginLimiter.penalty(ip);
  }
}

module.exports = AuthSystem;

// security/rbac-system.js
class RBACSystem {
  constructor() {
    this.roles = {
      user: {
        permissions: [
          'read:own_profile',
          'update:own_profile',
          'create:content',
          'read:public_content',
          'use:ai_features'
        ]
      },
      
      premium_user: {
        permissions: [
          ...this.roles.user.permissions,
          'create:advanced_content',
          'use:premium_ai_features',
          'export:data'
        ]
      },
      
      moderator: {
        permissions: [
          ...this.roles.premium_user.permissions,
          'read:all_profiles',
          'update:user_content',
          'delete:inappropriate_content',
          'ban:users'
        ]
      },
      
      admin: {
        permissions: [
          ...this.roles.moderator.permissions,
          'create:users',
          'delete:users',
          'update:system_settings',
          'read:audit_logs',
          'manage:roles'
        ]
      },
      
      system: {
        permissions: ['*'] // Permissão total
      }
    };

    this.resourceValidators = {
      'own_profile': this.validateOwnProfile.bind(this),
      'user_content': this.validateUserContent.bind(this)
    };
  }

  async hasPermission(user, permission, resource = null) {
    // Verificar permissões do usuário
    const userPermissions = user.permissions || this.roles[user.role]?.permissions || [];
    
    // Permissão curinga
    if (userPermissions.includes('*')) {
      return true;
    }

    // Verificar permissão específica
    if (!userPermissions.includes(permission)) {
      return false;
    }

    // Validação específica do recurso
    if (resource && this.resourceValidators[permission.split(':')[1]]) {
      const resourceType = permission.split(':')[1];
      const validator = this.resourceValidators[resourceType];
      if (validator) {
        return await validator(user, resource);
      }
    }

    return true;
  }

  async validateOwnProfile(user, resource) {
    // resource deve conter userId
    return user.id === resource.userId;
  }

  async validateUserContent(user, resource) {
    // Moderadores podem modificar conteúdo de usuários
    if (user.role === 'moderator' || user.role === 'admin') {
      return true;
    }
    
    // Usuários só podem modificar seu próprio conteúdo
    return user.id === resource.authorId;
  }

  async checkAccess(user, action, resourceType, resourceId) {
    const permission = `${action}:${resourceType}`;
    
    // Construir objeto de recurso para validação
    const resource = { 
      userId: resourceId,
      authorId: resourceId,
      [resourceType + 'Id']: resourceId 
    };

    return this.hasPermission(user, permission, resource);
  }

  // Middleware para autorização
  authorize(permission, resourceGetter = null) {
    return async (req, res, next) => {
      try {
        const user = req.user; // Definido pelo middleware de autenticação
        
        if (!user) {
          return res.status(401).json({ error: 'Não autenticado' });
        }

        let resource = null;
        if (resourceGetter) {
          resource = await resourceGetter(req);
        }

        const hasAccess = await this.hasPermission(user, permission, resource);
        
        if (!hasAccess) {
          return res.status(403).json({ 
            error: 'Acesso negado',
            requiredPermission: permission,
            userPermissions: user.permissions
          });
        }

        next();
      } catch (error) {
        console.error('Erro na autorização:', error);
        res.status(500).json({ error: 'Erro interno na autorização' });
      }
    };
  }
}

WITH raw_performance AS (
    SELECT
        event_id,
        session_id,
        user_id,
        feature_name,
        operation_name,
        duration_ms,
        memory_usage_mb,
        cpu_percent,
        timestamp,
        environment,
        app_version,
        metadata,
        source_system,
        ingested_at
    FROM {{ source('raw', 'performance_metrics_raw') }}
    WHERE timestamp >= CURRENT_DATE - INTERVAL '90 days'  -- Data retention window
),

cleaned_performance AS (
    SELECT
        event_id,
        session_id,
        COALESCE(user_id, 'anonymous') AS user_id,
        LOWER(TRIM(feature_name)) AS feature_name,
        LOWER(TRIM(operation_name)) AS operation_name,
        -- Validate and clean duration
        CASE 
            WHEN duration_ms < 0 THEN 0
            WHEN duration_ms > 3600000 THEN 3600000  -- Cap at 1 hour
            ELSE duration_ms
        END AS duration_ms,
        -- Validate memory usage
        CASE 
            WHEN memory_usage_mb < 0 THEN NULL
            WHEN memory_usage_mb > 100000 THEN NULL  -- Cap at 100GB
            ELSE memory_usage_mb
        END AS memory_usage_mb,
        -- Validate CPU percent
        CASE 
            WHEN cpu_percent < 0 THEN 0
            WHEN cpu_percent > 1000 THEN 1000  -- Cap at 1000%
            ELSE cpu_percent
        END AS cpu_percent,
        timestamp,
        COALESCE(environment, 'production') AS environment,
        COALESCE(app_version, 'unknown') AS app_version,
        ingested_at,
        -- Calculate statistical anomaly flag
        CASE 
            WHEN duration_ms > (
                SELECT AVG(duration_ms) + 3 * STDDEV(duration_ms) 
                FROM {{ source('raw', 'performance_metrics_raw') }}
                WHERE feature_name = raw_performance.feature_name
                AND timestamp >= CURRENT_DATE - INTERVAL '7 days'
            ) THEN TRUE
            ELSE FALSE
        END AS is_anomaly,
        -- Normalize duration by feature baseline
        duration_ms / NULLIF(
            (
                SELECT AVG(duration_ms)
                FROM {{ source('raw', 'performance_metrics_raw') }}
                WHERE feature_name = raw_performance.feature_name
                AND timestamp >= CURRENT_DATE - INTERVAL '7 days'
            ), 0
        ) AS normalized_duration
    FROM raw_performance
    WHERE feature_name IS NOT NULL
    AND duration_ms IS NOT NULL
)

WITH daily_metrics AS (
    SELECT
        DATE(timestamp) AS date,
        feature_name,
        environment,
        app_version,
        COUNT(*) AS total_operations,
        COUNT(DISTINCT session_id) AS unique_sessions,
        COUNT(DISTINCT user_id) AS unique_users,
        AVG(duration_ms) AS avg_duration_ms,
        PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY duration_ms) AS median_duration_ms,
        PERCENTILE_CONT(0.95) WITHIN GROUP (ORDER BY duration_ms) AS p95_duration_ms,
        PERCENTILE_CONT(0.99) WITHIN GROUP (ORDER BY duration_ms) AS p99_duration_ms,
        STDDEV(duration_ms) AS stddev_duration_ms,
        AVG(memory_usage_mb) AS avg_memory_mb,
        AVG(cpu_percent) AS avg_cpu_percent,
        SUM(CASE WHEN is_anomaly THEN 1 ELSE 0 END) AS anomaly_count,
        MIN(timestamp) AS first_event_time,
        MAX(timestamp) AS last_event_time
    FROM {{ ref('stg_performance_metrics') }}
    WHERE timestamp >= CURRENT_DATE - INTERVAL '30 days'
    GROUP BY 1, 2, 3, 4
),

trend_analysis AS (
    SELECT
        *,
        avg_duration_ms - LAG(avg_duration_ms, 1) OVER (
            PARTITION BY feature_name, environment 
            ORDER BY date
        ) AS duration_change_day_over_day,
        (avg_duration_ms - LAG(avg_duration_ms, 7) OVER (
            PARTITION BY feature_name, environment 
            ORDER BY date
        )) / NULLIF(LAG(avg_duration_ms, 7) OVER (
            PARTITION BY feature_name, environment 
            ORDER BY date
        ), 0) * 100 AS duration_change_week_over_week_pct,
        CASE 
            WHEN p95_duration_ms > (
                SELECT AVG(p95_duration_ms) * 1.5 
                FROM daily_metrics 
                WHERE feature_name = dm.feature_name 
                AND date >= CURRENT_DATE - INTERVAL '14 days'
            ) THEN 'DEGRADED'
            WHEN p95_duration_ms < (
                SELECT AVG(p95_duration_ms) * 0.7 
                FROM daily_metrics 
                WHERE feature_name = dm.feature_name 
                AND date >= CURRENT_DATE - INTERVAL '14 days'
            ) THEN 'IMPROVED'
            ELSE 'STABLE'
        END AS performance_status
    FROM daily_metrics dm
)

SELECT
    date,
    feature_name,
    environment,
    app_version,
    total_operations,
    unique_sessions,
    unique_users,
    ROUND(avg_duration_ms, 2) AS avg_duration_ms,
    ROUND(median_duration_ms, 2) AS median_duration_ms,
    ROUND(p95_duration_ms, 2) AS p95_duration_ms,
    ROUND(p99_duration_ms, 2) AS p99_duration_ms,
    ROUND(stddev_duration_ms, 2) AS stddev_duration_ms,
    ROUND(avg_memory_mb, 2) AS avg_memory_mb,
    ROUND(avg_cpu_percent, 2) AS avg_cpu_percent,
    anomaly_count,
    ROUND(duration_change_day_over_day, 2) AS duration_change_day_over_day,
    ROUND(duration_change_week_over_week_pct, 2) AS duration_change_week_over_week_pct,
    performance_status,
    first_event_time,
    last_event_time,
    CURRENT_TIMESTAMP AS generated_at
FROM trend_analysis
ORDER BY date DESC, feature_name
