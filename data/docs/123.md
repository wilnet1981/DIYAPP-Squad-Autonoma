## ADR-001: Arquitetura de Microsserviços

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead

### CONTEXTO:
Precisamos evoluir de uma arquitetura monolítica para microsserviços para alcançar:
- Escalabilidade independente por serviço
- Deploy contínuo sem downtime
- Isolamento de falhas
- Autonomia das squads

### DECISÃO:
Adotar arquitetura baseada em microsserviços com os seguintes princípios:
1. Cada serviço é independente e possui seu próprio banco de dados
2. Comunicação via eventos assíncronos (Kafka) para desacoplamento
3. API Gateway como ponto único de entrada
4. Service Mesh (Istio) para comunicação entre serviços
5. Observabilidade centralizada (logs, métricas, traces)

### OPÇÕES CONSIDERADAS:
- **Opção A:** Monolito modularizado - Prós: Simplicidade inicial. Contras: Escalabilidade limitada, deploy acoplado.
- **Opção B:** Microsserviços com REST síncrono - Prós: Desacoplamento. Contras: Latência, falhas em cascata.
- **Opção C:** Microsserviços com eventos - Prós: Desacoplamento total, resiliência. Contras: Complexidade, eventual consistency.

**Opção escolhida:** C - Justificativa: Alinha com objetivo de 100% autonomia e estabilidade.

### CONSEQUÊNCIAS:
**Positivas:** 
- Escalabilidade granular
- Deploy independente
- Isolamento de falhas
- Tecnologias heterogêneas por serviço

**Negativas:**
- Complexidade operacional aumentada
- Necessidade de orquestração
- Eventual consistency
- Overhead de rede

**Riscos:**
- Distributed tracing complexo
- Debug cross-service
- Mitigação: Investir em observabilidade desde o início

```javascript
      const { id } = req.params;
      const user = await this.userService.getUser(id);
      
      if (!user) {
        throw new NotFoundError('User not found');
      }
      
      res.json({

        data: user,

      });
    } catch (error) {
```

## Monitoramento e Observabilidade

```javascript
// libs/core/metrics/metrics.js
const promClient = require('prom-client');
const responseTime = require('response-time');

class MetricsCollector {
  constructor() {
    this.register = new promClient.Registry();
    
    // Default metrics
    promClient.collectDefaultMetrics({ register: this.register });
    
    // Custom metrics
    this.httpRequestDuration = new promClient.Histogram({
      name: 'http_request_duration_seconds',
      help: 'Duration of HTTP requests in seconds',
      labelNames: ['method', 'route', 'status_code'],
      buckets: [0.1, 0.5, 1, 2, 5]
    });
    
    this.serviceErrors = new promClient.Counter({
      name: 'service_errors_total',
      help: 'Total number of service errors',
      labelNames: ['service', 'error_type']
    });
    
    this.register.registerMetric(this.httpRequestDuration);
    this.register.registerMetric(this.serviceErrors);
  }
  
  middleware() {
    return responseTime((req, res, time) => {
      if (req.url !== '/metrics') {
        this.httpRequestDuration
          .labels(req.method, req.route?.path || req.url, res.statusCode)
          .observe(time / 1000);
      }
    });
  }
  
  incrementError(service, errorType) {
    this.serviceErrors.labels(service, errorType).inc();
  }
  
  async getMetrics() {
    return await this.register.metrics();
  }
}

module.exports = new MetricsCollector();
```

from datetime import datetime, timedelta
from typing import Optional, Dict, Any
from jose import JWTError, jwt
from passlib.context import CryptContext
from fastapi import HTTPException, status, Depends
from fastapi.security import HTTPBearer, HTTPAuthorizationCredentials
import structlog

logger = structlog.get_logger(__name__)

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")
security = HTTPBearer(auto_error=False)


def verify_password(plain_password: str, hashed_password: str) -> bool:
    """Verifica senha com bcrypt"""
    return pwd_context.verify(plain_password, hashed_password)


def get_password_hash(password: str) -> str:
    """Gera hash bcrypt da senha"""
    return pwd_context.hash(password)


def create_access_token(data: Dict[str, Any], expires_delta: Optional[timedelta] = None) -> str:
    """Cria JWT access token"""
    to_encode = data.copy()
    if expires_delta:
        expire = datetime.utcnow() + expires_delta
    else:
        expire = datetime.utcnow() + timedelta(minutes=settings.ACCESS_TOKEN_EXPIRE_MINUTES)
    
    to_encode.update({"exp": expire, "type": "access"})
    encoded_jwt = jwt.encode(to_encode, settings.SECRET_KEY, algorithm=settings.ALGORITHM)
    
    # Log seguro (sem dados sensíveis)
    logger.info("access_token_created", user_id=data.get("sub"), expires=expire.isoformat())
    return encoded_jwt


def create_refresh_token(data: Dict[str, Any]) -> str:
    """Cria JWT refresh token"""
    to_encode = data.copy()
    expire = datetime.utcnow() + timedelta(days=settings.REFRESH_TOKEN_EXPIRE_DAYS)
    to_encode.update({"exp": expire, "type": "refresh"})
    return jwt.encode(to_encode, settings.SECRET_KEY, algorithm=settings.ALGORITHM)


async def verify_token(
    credentials: Optional[HTTPAuthorizationCredentials] = Depends(security)
) -> Dict[str, Any]:
    """Valida token JWT - Middleware de autenticação"""
    if credentials is None:
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED,
            detail="Not authenticated",
            headers={"WWW-Authenticate": "Bearer"},
        )
    
    token = credentials.credentials
    
    try:
        payload = jwt.decode(token, settings.SECRET_KEY, algorithms=[settings.ALGORITHM])
        
        # Verifica tipo do token
        if payload.get("type") != "access":
            raise HTTPException(
                status_code=status.HTTP_401_UNAUTHORIZED,
                detail="Invalid token type"
            )
        
        user_id: str = payload.get("sub")
        if user_id is None:
            raise HTTPException(
                status_code=status.HTTP_401_UNAUTHORIZED,
                detail="Invalid token payload"
            )
        
        # Log seguro
        logger.info("token_validated", user_id=user_id)
        return payload
        
    except JWTError as e:
        logger.warning("token_validation_failed", error=str(e))
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED,
            detail="Could not validate credentials"
        )


async def authorize_resource(
    user_id: str,
    resource_owner_id: str,
    resource_type: str,
    action: str
) -> bool:
    """Autorização baseada em recursos"""
    # Implementação básica - em produção, usar sistema de roles/permissions
    if user_id == resource_owner_id:
        return True
    
    # Log de tentativa de acesso não autorizado

        "unauthorized_access_attempt",
        user_id=user_id,
        resource_owner_id=resource_owner_id,
        resource_type=resource_type,
        action=action
    )
    
    return False

import redis.asyncio as redis
from typing import Optional, Any
import pickle
import json
from functools import wraps
import asyncio
from tenacity import retry, stop_after_attempt, wait_exponential
import structlog

logger = structlog.get_logger(__name__)


class CacheManager:
    """Gerenciador de cache com Redis e fallback"""
    
    def __init__(self):
        self.redis_client: Optional[redis.Redis] = None
        self.cache_enabled = settings.CACHE_ENABLED
        self.local_cache = {}  # Fallback local
        
    async def connect(self):
        """Conecta ao Redis com circuit breaker"""
        if not self.cache_enabled:
            return
            
        try:
            self.redis_client = redis.from_url(
                str(settings.REDIS_URL),
                encoding="utf-8",
                decode_responses=False,
                socket_connect_timeout=2,
                socket_timeout=2
            )
            await self.redis_client.ping()
            logger.info("redis_connected")
        except Exception as e:
            logger.error("redis_connection_failed", error=str(e))
            self.cache_enabled = False
            self.redis_client = None
    
    async def disconnect(self):
        """Desconecta do Redis"""
        if self.redis_client:
            await self.redis_client.close()
    
    @retry(
        stop=stop_after_attempt(3),
        wait=wait_exponential(multiplier=1, min=2, max=10)
    )
    async def get(self, key: str) -> Optional[Any]:
        """Obtém valor do cache com retry"""
        if not self.cache_enabled:
            return None
            
        try:
            if self.redis_client:
                data = await self.redis_client.get(key)
                if data:
                    return pickle.loads(data)
            return self.local_cache.get(key)
        except Exception as e:
            logger.warning("cache_get_failed", key=key, error=str(e))
            return None
    
    async def set(self, key: str, value: Any, ttl: Optional[int] = None) -> bool:
        """Define valor no cache"""
        if not self.cache_enabled:
            return False
            
        try:
            if self.redis_client:
                ttl = ttl or settings.CACHE_TTL
                serialized = pickle.dumps(value)
                await self.redis_client.setex(key, ttl, serialized)
            else:
                self.local_cache[key] = value
            return True
        except Exception as e:
            logger.warning("cache_set_failed", key=key, error=str(e))
            return False
    
    async def delete(self, key: str) -> bool:
        """Remove valor do cache"""
        try:
            if self.redis_client:
                await self.redis_client.delete(key)
            if key in self.local_cache:
                del self.local_cache[key]
            return True
        except Exception as e:
            logger.warning("cache_delete_failed", key=key, error=str(e))
            return False
    
    async def delete_pattern(self, pattern: str) -> int:
        """Remove múltiplas chaves por pattern"""
        try:
            if self.redis_client:
                keys = await self.redis_client.keys(pattern)
                if keys:
                    return await self.redis_client.delete(*keys)
            return 0
        except Exception as e:
            logger.warning("cache_delete_pattern_failed", pattern=pattern, error=str(e))
            return 0


def cache_response(ttl: int = 300, key_prefix: str = "api"):
    """Decorator para cache de responses de API"""
    def decorator(func):
        @wraps(func)
        async def wrapper(*args, **kwargs):
            cache_manager = kwargs.get('cache_manager') or CacheManager()
            
            # Gera chave de cache baseada nos argumentos
            cache_key = f"{key_prefix}:{func.__name__}:{str(args)}:{str(kwargs)}"
            
            # Tenta obter do cache
            cached = await cache_manager.get(cache_key)
            if cached is not None:
                logger.info("cache_hit", key=cache_key)
                return cached
            
            # Executa função se não tem cache
            result = await func(*args, **kwargs)
            
            # Armazena no cache
            await cache_manager.set(cache_key, result, ttl)
            
            return result
        return wrapper
    return decorator

from circuitbreaker import circuit
import time
from typing import Callable, Any
import structlog

logger = structlog.get_logger(__name__)


class CircuitBreaker:
    """Circuit Breaker com fallback e métricas"""
    
    def __init__(
        self,
        failure_threshold: int = settings.CIRCUIT_FAILURE_THRESHOLD,
        recovery_timeout: int = settings.CIRCUIT_RECOVERY_TIMEOUT,
        name: str = "default"
    ):
        self.failure_threshold = failure_threshold
        self.recovery_timeout = recovery_timeout
        self.name = name
        self.failures = 0
        self.last_failure_time = 0
        self.state = "CLOSED"  # CLOSED, OPEN, HALF_OPEN
        
    def record_failure(self):
        """Registra falha e atualiza estado"""
        self.failures += 1
        self.last_failure_time = time.time()
        
        if self.failures >= self.failure_threshold:
            self.state = "OPEN"
            logger.warning(
                "circuit_opened",
                circuit_name=self.name,
                failures=self.failures
            )
    
    def record_success(self):
        """Registra sucesso e reseta contador"""
        self.failures = 0
        if self.state == "HALF_OPEN":
            self.state = "CLOSED"
            logger.info("circuit_closed", circuit_name=self.name)
    
    def can_execute(self) -> bool:
        """Verifica se pode executar"""
        if self.state == "CLOSED":
            return True
        
        if self.state == "OPEN":
            # Verifica se já passou tempo suficiente para tentar recovery
            if time.time() - self.last_failure_time > self.recovery_timeout:
                self.state = "HALF_OPEN"
                logger.info("circuit_half_open", circuit_name=self.name)
                return True
            return False
        
        return True  # HALF_OPEN


def circuit_breaker(
    fallback_func: Optional[Callable] = None,
    failure_threshold: int = 5,
    recovery_timeout: int = 60
):
    """Decorator para circuit breaker"""
    def decorator(func):
        cb = CircuitBreaker(failure_threshold, recovery_timeout, func.__name__)
        
        @wraps(func)
        async def wrapper(*args, **kwargs):
            if not cb.can_execute():
                logger.warning("circuit_blocked", circuit_name=cb.name)
                if fallback_func:
                    return await fallback_func(*args, **kwargs)
                raise Exception(f"Circuit breaker '{cb.name}' is OPEN")
            
            try:
                result = await func(*args, **kwargs)
                cb.record_success()
