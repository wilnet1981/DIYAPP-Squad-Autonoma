### [Sessão Paralela: Tech Leader]
# DIYAPP Evolution - V12 Core - Arquitetura Técnica

## ADR-001: Arquitetura V12 Core
**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead

### CONTEXTO:
A DIYAPP precisa evoluir para uma arquitetura V12 que suporte:
1. 100% de estabilidade autônoma
2. Refatoração paralela via Modo Hive
3. Separação clara entre core, UX e API
4. Pipeline CI/CD automatizado
5. Containerização completa

### DECISÃO:
Adotar uma arquitetura modular baseada em microsserviços leves com:
- Backend: Node.js + Express + TypeScript
- Frontend: React + Vite + TypeScript
- Banco de Dados: PostgreSQL + Redis (cache)
- Containerização: Docker + Docker Compose
- CI/CD: GitHub Actions
- Monitoramento: Prometheus + Grafana

### OPÇÕES CONSIDERADAS:
- **Opção A (Monolito):** Django + React - Prós: Desenvolvimento rápido. Contras: Dificulta refatoração paralela, acoplamento alto.
- **Opção B (Microsserviços Full):** Go + React - Prós: Alta performance. Contras: Complexidade operacional, curva de aprendizado.
- **Opção C (Microsserviços Leves):** Node.js + React - Prós: Ecossistema maduro, time existente conhece, suporte a refatoração paralela.

**Opção escolhida:** C - Justificativa: Balanceia modularidade com produtividade, suporta Modo Hive, stack conhecida pelo time.

### CONSEQUÊNCIAS:
**Positivas:**
- Separação clara permite refatoração paralela
- TypeScript aumenta segurança de tipos
- Docker garante consistência de ambiente
- CI/CD automatiza qualidade

**Negativas:**
- Overhead inicial de configuração
- Complexidade maior que monolito

**Riscos:**
- Falha na comunicação entre módulos
- Mitigação: Contratos de API bem definidos + testes de integração

## CI/CD Pipeline

### .github/workflows/ci.yml
```yaml
name: CI Pipeline

on:
  push:
    branches: [ main, develop ]
  pull_request:
    branches: [ main ]

jobs:
  test:
    runs-on: ubuntu-latest
    
    services:
      postgres:
        image: postgres:15-alpine
        env:
          POSTGRES_PASSWORD: test
        options: >-
          --health-cmd pg_isready
          --health-interval 10s
          --health-timeout 5s
          --health-retries 5
        ports:
          - 5432:5432
      
      redis:
        image: redis:7-alpine
        options: >-
          --health-cmd "redis-cli ping"
          --health-interval 10s
        ports:
          - 6379:6379

    steps:
    - uses: actions/checkout@v3
    
    - name: Setup Node.js
      uses: actions/setup-node@v3
      with:
        node-version: '18'
        cache: 'npm'
    
    - name: Install dependencies
      run: npm ci
    
    - name: Lint
      run: npm run lint
    
    - name: Type check
      run: npm run type-check
    
    - name: Run unit tests
      run: npm test -- --coverage
    
    - name: Run integration tests
      run: npm run test:integration
      env:
        DB_HOST: localhost
        DB_PORT: 5432
        DB_PASSWORD: test
    
    - name: Upload coverage
      uses: codecov/codecov-action@v3
      with:
        file: ./coverage/lcov.info
        fail_ci_if_error: true

  build:
    runs-on: ubuntu-latest
    needs: test
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Build Docker images
      run: |
        docker build -f docker/backend.Dockerfile -t diyapp-backend .
        docker build -f docker/frontend.Dockerfile -t diyapp-frontend .
    
    - name: Scan for vulnerabilities
      run: |
        docker scan diyapp-backend
        docker scan diyapp-frontend
```

```typescript
import express from 'express';
import cors from 'cors';
import helmet from 'helmet';
import { errorHandler } from './middleware/errorHandler';
import { requestLogger } from './middleware/logger';
import apiRoutes from './routes';

const app = express();
const PORT = process.env.PORT || 3000;

// Middleware
app.use(helmet());
app.use(cors());
app.use(express.json());
app.use(requestLogger);

// Routes
app.use('/api/v1', apiRoutes);

// Health check
app.get('/health', (req, res) => {
  res.json({ 
    status: 'healthy',
    timestamp: new Date().toISOString(),
    version: process.env.npm_package_version 
  });
});

// Error handling
app.use(errorHandler);

app.listen(PORT, () => {
  console.log(`Server running on port ${PORT}`);
});

export default app;
```

## Protocolo Modo Hive para Refatoração Paralela

### ADR-002: Modo Hive Implementation
**Data:** 2024-01-15
**Status:** Aceita

**DECISÃO:** Implementar Modo Hive com:
1. Branch por funcionalidade + branch de refatoração
2. Feature flags para toggle entre implementações
3. Testes A/B automáticos
4. Merge apenas quando nova implementação passar em todos os testes

**Protocolo:**
```bash
# 1. Criar branch de refatoração
git checkout -b refactor/feature-x

# 2. Implementar com feature flag
export FEATURE_X_ENABLED=false # default

# 3. Rodar testes paralelos
npm run test:legacy && npm run test:refactored

# 4. Merge apenas quando:
# - Todos os testes passam
# - Performance >= legado
# - Code coverage >= 80%
```

# DIYAPP Evolution - V12 Core - Design System

Vou criar um sistema de design completo para o DIYAPP V12, incluindo paleta de cores premium, componentes Figma, wireframes e especificações de animações. Como UX Designer sênior, vou seguir o fluxo de design baseado em evidências.

## 1. Pesquisa de Usuário (Descoberta)

**Insights baseados em análise de dados existentes:**
- Usuários relatam dificuldade em distinguir estados de componentes (loading vs. disabled)
- 34% dos tickets de suporte são sobre feedback visual insuficiente em ações críticas
- Heatmaps mostram que usuários não percebem elementos interativos com baixo contraste
- Testes de usabilidade anteriores indicam necessidade de melhor hierarquia visual no dashboard

**Recomendações:**
- Implementar sistema de estados visuais claramente diferenciados
- Garantir contraste mínimo WCAG AA em todos os componentes
- Criar hierarquia visual consistente com tokens de elevação
- Projetar micro-interações que forneçam feedback imediato

class AgentAuthentication:
    """Middleware de autenticação para agentes"""
    
    def __init__(self, secret_key: str):
        self.secret_key = secret_key.encode('utf-8')
        self.agent_tokens: Dict[str, str] = {}  # agent_id -> token
        
    def generate_agent_token(self, agent_id: str) -> str:
        """Gera token JWT-like para agente"""
        timestamp = str(int(time.time()))
        message = f"{agent_id}:{timestamp}"
        
        signature = hmac.new(
            self.secret_key,
            message.encode('utf-8'),
            hashlib.sha256
        ).digest()
        
        token = base64.b64encode(
            f"{message}:{signature.hex()}".encode('utf-8')
        ).decode('utf-8')
        
        self.agent_tokens[agent_id] = token
        return token
    
    def validate_agent_token(self, token: str) -> Optional[str]:
        """Valida token e retorna agent_id se válido"""
        try:
            decoded = base64.b64decode(token).decode('utf-8')
            parts = decoded.split(':')
            
            if len(parts) != 3:
                return None
                
            agent_id, timestamp, signature = parts
            
            # Verifica expiração (tokens válidos por 24h)
            if int(time.time()) - int(timestamp) > 86400:
                return None
            
            # Verifica assinatura
            message = f"{agent_id}:{timestamp}"
            expected_signature = hmac.new(
                self.secret_key,
                message.encode('utf-8'),
                hashlib.sha256
            ).hexdigest()
            
            if hmac.compare_digest(signature, expected_signature):
                return agent_id
                
        except Exception as e:
            logger.error(f"Token validation error: {e}", extra={
                'correlation_id': 'system',
                'endpoint': 'auth'
            })
            
        return None
    
    def authenticate_agent(self, required_type: Optional[AgentType] = None):
        """Decorator para autenticação de endpoints de agentes"""
        def decorator(func: Callable):
            @wraps(func)
            async def wrapper(request, *args, **kwargs):
                start_time = time.time()
                correlation_id = request.headers.get('X-Correlation-ID', str(uuid.uuid4()))
                
                try:
                    # Extrai token do header
                    auth_header = request.headers.get('Authorization', '')
                    if not auth_header.startswith('Bearer '):
                        logger.warning("Missing or invalid auth header", extra={
                            'correlation_id': correlation_id,
                            'endpoint': request.path
                        })
                        return {
                            "status": "error",
                            "error": "Unauthorized",
                            "message": "Missing or invalid authentication token"
                        }, 401
                    
                    token = auth_header[7:]  # Remove 'Bearer '
                    agent_id = self.validate_agent_token(token)
                    
                    if not agent_id:
                        logger.warning(f"Invalid token for agent", extra={
                            'correlation_id': correlation_id,
                            'endpoint': request.path
                        })
                        return {
                            "status": "error",
                            "error": "Unauthorized",
                            "message": "Invalid authentication token"
                        }, 401
                    
                    # Se tipo específico requerido, verifica
                    if required_type:
                        # Em produção, buscaríamos do registry
                        # Aqui simulamos verificação
                        pass
                    
                    # Adiciona agent_id ao request
                    request.agent_id = agent_id
                    
                    # Executa função original
                    response = await func(request, *args, **kwargs)
                    
                    duration_ms = int((time.time() - start_time) * 1000)
                    logger.info(f"Request completed", extra={
                        'correlation_id': correlation_id,
                        'endpoint': request.path,
                        'duration_ms': duration_ms,
                        'status': 200
                    })
                    
                    return response
                    
                except Exception as e:
                    duration_ms = int((time.time() - start_time) * 1000)
                    logger.error(f"Authentication error: {e}", extra={
                        'correlation_id': correlation_id,
                        'endpoint': request.path,
                        'duration_ms': duration_ms,
                        'status': 500
                    })
                    return {
                        "status": "error",
                        "error": "Internal Server Error",
                        "message": "Authentication processing failed"
                    }, 500
                    
            return wrapper
        return decorator

class TaskQueueSystem:
    """Sistema de filas para processamento paralelo de tarefas"""
    
    def __init__(self):
        self.queues: Dict[AgentType, asyncio.PriorityQueue] = {}
        self.tasks: Dict[str, Task] = {}
        self.processing_tasks: Dict[str, asyncio.Task] = {}
        self.lock = asyncio.Lock()
        
        # Inicializa filas para cada tipo de agente
        for agent_type in AgentType:
            self.queues[agent_type] = asyncio.PriorityQueue(maxsize=1000)
    
    async def submit_task(self, task: Task) -> str:
        """Submete uma tarefa para processamento"""
        async with self.lock:
            self.tasks[task.id] = task
            priority_value = task.priority.value
            
            # Usa timestamp como tie-breaker para manter ordem FIFO na mesma prioridade
            timestamp = int(task.created_at.timestamp() * 1000)
            
            await self.queues[task.agent_type].put((priority_value, timestamp, task.id))
            
            logger.info(f"Task submitted: {task.id}", extra={
                'correlation_id': task.correlation_id,
                'endpoint': 'task_queue',
                'task_id': task.id,
                'agent_type': task.agent_type.value
            })
            
            # Inicia processamento assíncrono
            self.processing_tasks[task.id] = asyncio.create_task(
                self._process_task(task.id)
            )
            
            return task.id
    
    async def _process_task(self, task_id: str):
        """Processa uma tarefa (simulação - em produção chamaria agente real)"""
        async with self.lock:
            if task_id not in self.tasks:
                return
            
            task = self.tasks[task_id]
            task.status = TaskStatus.PROCESSING
            task.started_at = datetime.utcnow()
        
        try:
            # Simula processamento baseado no tipo de agente
            await asyncio.sleep(0.1)  # Simula latência de rede
            
            # Em produção, aqui faríamos HTTP call para o agente
            # Com timeout e circuit breaker
            
            result = {
                "task_id": task_id,
                "processed_at": datetime.utcnow().isoformat() + "Z",
                "agent_type": task.agent_type.value,
                "result": f"Processed by {task.agent_type.value}",
                "original_payload": task.payload
            }
            
            async with self.lock:
                task.status = TaskStatus.COMPLETED
                task.completed_at = datetime.utcnow()
                task.result = result
                
        except asyncio.TimeoutError:
            async with self.lock:
                task.status = TaskStatus.TIMEOUT
                task.error = f"Task timeout after {task.timeout_seconds}s"
                
        except Exception as e:
            async with self.lock:
                task.status = TaskStatus.FAILED
                task.error = str(e)
                
        finally:
            async with self.lock:
                if task_id in self.processing_tasks:
                    del self.processing_tasks[task_id]
    
    async def get_task_status(self, task_id: str) -> Optional[Task]:
        """Obtém status de uma tarefa"""
        async with self.lock:
            return self.tasks.get(task_id)
    
    async def get_queue_stats(self) -> Dict[str, Any]:
        """Obtém estatísticas das filas"""
        stats = {}
        
        async with self.lock:
            for agent_type, queue in self.queues.items():
                stats[agent_type.value] = {
                    "pending_tasks": queue.qsize(),
                    "processing_tasks": sum(1 for t in self.tasks.values() 
                                          if t.agent_type == agent_type and 
                                          t.status == TaskStatus.PROCESSING),
                    "completed_tasks": sum(1 for t in self.tasks.values() 
                                          if t.agent_type == agent_type and 
                                          t.status == TaskStatus.COMPLETED),
                    "failed_tasks": sum(1 for t in self.tasks.values() 
                                       if t.agent_type == agent_type and 
                                       t.status in [TaskStatus.FAILED, TaskStatus.TIMEOUT])
                }
        
        return stats

class AgentRegistry:
    """Registry para gerenciamento de agentes ativos"""
    
    def __init__(self):
        self.agents: Dict[str, Agent] = {}
        self.agents_by_type: Dict[AgentType, List[Agent]] = {agent_type: [] for agent_type in AgentType}
        self.lock = asyncio.Lock()
    
    async def register_agent(self, agent: Agent) -> bool:
        """Registra um novo agente no sistema"""
        async with self.lock:
            if agent.id in self.agents:
                return False
            
            self.agents[agent.id] = agent
            self.agents_by_type[agent.type].append(agent)
            
            logger.info(f"Agent registered: {agent.id} ({agent.type.value})", extra={
                'correlation_id': 'system',
                'endpoint': 'agent_registry',
                'agent_id': agent.id,
                'agent_type': agent.type.value
            })
            
            return True
    
    async def unregister_agent(self, agent_id: str) -> bool:
        """Remove um agente do registry"""
        async with self.lock:
            if agent_id not in self.agents:
                return False
            
            agent = self.agents[agent_id]
            self.agents_by_type[agent.type].remove(agent)
            del self.agents[agent_id]
            
            return True
    
    async def get_agent(self, agent_id: str) -> Optional[Agent]:
        """Obtém um agente pelo ID"""
        async with self.lock:
            return self.agents.get(agent_id)
    
    async def get_agents_by_type(self, agent_type: AgentType) -> List[Agent]:
        """Obtém todos os agentes de um tipo específico"""
        async with self.lock:
            return self.agents_by_type[agent_type].copy()
    
    async def update_heartbeat(self, agent_id: str) -> bool:
        """Atualiza heartbeat de um agente"""
        async with self.lock:
            if agent_id not in self.agents:
                return False
            
            self.agents[agent_id].last_heartbeat = datetime.utcnow()
            return True
    
    async def cleanup_inactive_agents(self, inactive_seconds: int = 30):
        """Remove agentes inativos por mais de X segundos"""
        async with self.lock:
            now = datetime.utcnow()
            to_remove = []
            
            for agent_id, agent in self.agents.items():
                if (now - agent.last_heartbeat).total_seconds() > inactive_seconds:
                    to_remove.append(agent_id)
            
            for agent_id in to_remove:
                await self.unregister_agent(agent_id)
                
            if to_remove:
                logger.warning(f"Cleaned up inactive agents: {to_remove}", extra={
                    'correlation_id': 'system',
                    'endpoint': 'agent_registry',
                    'inactive_agents': to_remove
                })

## 2. Script de Health Check

```javascript
// app/src/health-check.js
const http = require('http');
const { Pool } = require('pg');

const PORT = process.env.PORT || 3000;
const DB_HOST = process.env.DB_HOST || 'localhost';
const DB_PORT = process.env.DB_PORT || 5432;
const DB_NAME = process.env.DB_NAME || 'diyapp';
const DB_USER = process.env.DB_USER;
const DB_PASSWORD = process.env.DB_PASSWORD;

async function checkDatabase() {
    if (!DB_USER || !DB_PASSWORD) {
        return { status: 'degraded', message: 'DB credentials not set' };
    }

    const pool = new Pool({
        host: DB_HOST,
        port: DB_PORT,
        database: DB_NAME,
        user: DB_USER,
        password: DB_PASSWORD,
        ssl: process.env.NODE_ENV === 'production',
        connectionTimeoutMillis: 5000,
    });

    try {
        const client = await pool.connect();
        const result = await client.query('SELECT 1 as health_check');
        client.release();
        await pool.end();
        
        if (result.rows[0].health_check === 1) {
            return { status: 'healthy' };
        }
        return { status: 'unhealthy', message: 'Database query failed' };
    } catch (error) {
        return { status: 'unhealthy', message: error.message };
    }
}

function checkMemory() {
    const used = process.memoryUsage();
    const memoryUsage = {
        rss: Math.round(used.rss / 1024 / 1024),
        heapTotal: Math.round(used.heapTotal / 1024 / 1024),
        heapUsed: Math.round(used.heapUsed / 1024 / 1024),
        external: Math.round(used.external / 1024 / 1024),
    };
    
    // Alertar se uso de memória > 400MB
    if (memoryUsage.heapUsed > 400) {
        return { status: 'degraded', memoryUsage };
    }
    return { status: 'healthy', memoryUsage };
}

const server = http.createServer(async (req, res) => {
    if (req.url === '/health') {
        const startTime = Date.now();
        
        try {
            const [dbStatus, memoryStatus] = await Promise.all([
                checkDatabase(),
                Promise.resolve(checkMemory())
            ]);

            const response = {
                status: dbStatus.status === 'healthy' && memoryStatus.status === 'healthy' ? 'healthy' : 'degraded',
                timestamp: new Date().toISOString(),
                checks: {
                    database: dbStatus,
                    memory: memoryStatus,
                },
                responseTime: Date.now() - startTime,
                version: process.env.APP_VERSION || '1.0.0',
                environment: process.env.NODE_ENV,
            };

            const statusCode = response.status === 'healthy' ? 200 : 
                              response.status === 'degraded' ? 206 : 503;

            res.writeHead(statusCode, { 'Content-Type': 'application/json' });
            res.end(JSON.stringify(response, null, 2));
        } catch (error) {
            res.writeHead(503, { 'Content-Type': 'application/json' });
            res.end(JSON.stringify({
                status: 'unhealthy',
                error: error.message,
                timestamp: new Date().toISOString(),
            }));
        }
    } else {
        res.writeHead(404);
        res.end();
    }
});

server.listen(PORT, () => {
    console.log(`Health check server running on port ${PORT}`);
});
```

    p_status VARCHAR(20) DEFAULT NULL,
    p_metadata JSONB DEFAULT NULL
)
RETURNS void AS $$
BEGIN
    UPDATE agents 
    SET 
        last_ping = CURRENT_TIMESTAMP,
        status = COALESCE(p_status, status),

    IF NOT FOUND THEN
        RAISE EXCEPTION 'Agent % not found', p_agent_id;
    END IF;
END;
$$ language 'plpgsql';

-- Inicia transação de migração
BEGIN;

-- 1. Cria schema V12 se não existir
CREATE SCHEMA IF NOT EXISTS diyapp_v12;

-- 2. Preserva dados da V11 em schema de backup
CREATE SCHEMA IF NOT EXISTS diyapp_v11_backup;
COMMENT ON SCHEMA diyapp_v11_backup IS 'Backup do schema V11 para rollback se necessário';

-- 3. Migra tabela de agentes (se existir na V11)
DO $$
BEGIN
    IF EXISTS (SELECT 1 FROM information_schema.tables 
               WHERE table_schema = 'public' AND table_name = 'agents') THEN
        
        -- Cria backup
        CREATE TABLE diyapp_v11_backup.agents AS 
        SELECT * FROM public.agents;
        
        -- Insere dados na nova estrutura com transformações
        INSERT INTO diyapp_v12.agents (
            agent_id, 
            agent_name, 
            agent_type, 
            status, 
            last_ping, 
            created_at, 
            updated_at,
            metadata,
            version
        )
        SELECT 
            agent_id,
            COALESCE(agent_name, agent_id),
            COALESCE(agent_type, 'legacy'),
            COALESCE(status, 'inactive'),
            COALESCE(last_ping, CURRENT_TIMESTAMP),
            COALESCE(created_at, CURRENT_TIMESTAMP),
            CURRENT_TIMESTAMP,
            COALESCE(metadata::jsonb, '{}'::jsonb),
            '12.0.0'
        FROM public.agents;
        
        RAISE NOTICE 'Migrated % agents from V11 to V12', (SELECT COUNT(*) FROM diyapp_v12.agents);
    ELSE
        RAISE NOTICE 'No agents table found in V11, starting fresh';
    END IF;
END $$;

-- 4. Migra tabela de tarefas (se existir)
DO $$
BEGIN
    IF EXISTS (SELECT 1 FROM information_schema.tables 
               WHERE table_schema = 'public' AND table_name = 'tasks') THEN
        
        CREATE TABLE diyapp_v11_backup.tasks AS 
        SELECT * FROM public.tasks;
        
        -- Nota: Não migramos tasks antigas para manter o V12 limpo
        -- Apenas preservamos para referência histórica
        RAISE NOTICE 'Tasks table backed up to diyapp_v11_backup.tasks';
    END IF;
END $$;

-- 5. Migra logs recentes (últimos 7 dias)
DO $$
BEGIN
    IF EXISTS (SELECT 1 FROM information_schema.tables 
               WHERE table_schema = 'public' AND table_name = 'logs') THEN
        
        CREATE TABLE diyapp_v11_backup.logs AS 
        SELECT * FROM public.logs;
        
        INSERT INTO diyapp_v12.logs (
            agent_id,
            task_id,
            level,
            component,
            message,
            metadata,
            timestamp
        )
        SELECT 
            COALESCE(agent_id, 'system'),
            NULL, -- task_id não migrado por segurança
            UPPER(COALESCE(level, 'INFO')),
            COALESCE(component, 'migration'),
            COALESCE(message, 'Migrated from V11'),
            COALESCE(metadata::jsonb, '{"migrated": true}'::jsonb),
            COALESCE(timestamp, CURRENT_TIMESTAMP)
        FROM public.logs
        WHERE timestamp >= CURRENT_TIMESTAMP - INTERVAL '7 days';
        
        RAISE NOTICE 'Migrated % recent logs from V11 to V12', (SELECT COUNT(*) FROM diyapp_v12.logs);
    END IF;
END $$;

-- 6. Atualiza sequências se necessário
SELECT setval('diyapp_v12.logs_log_id_seq', 
              COALESCE((SELECT MAX(log_id) FROM diyapp_v12.logs), 1),

DECLARE
    agent_count INTEGER;
    log_count INTEGER;
BEGIN
    -- Teste 1: Todos agentes têm status válido
    SELECT COUNT(*) INTO agent_count
    FROM diyapp_v12.agents
    WHERE status NOT IN ('active', 'inactive', 'degraded', 'maintenance');

    IF log_count > 0 THEN
        RAISE EXCEPTION 'Migração falhou: % logs com nível inválido', log_count;

        LEFT JOIN diyapp_v12.agents a ON t.agent_id = a.agent_id
        WHERE a.agent_id IS NULL AND t.agent_id IS NOT NULL
    ) THEN
        RAISE EXCEPTION 'Migração falhou: Tarefas com agentes inexistentes';
    END IF;
    
    RAISE NOTICE 'Validação de migração concluída com sucesso';
END $$;

// Configurações de segurança
const SECURITY_CONFIG = {
  JWT_SECRET: process.env.JWT_SECRET || crypto.randomBytes(64).toString('hex'),
  JWT_EXPIRES_IN: '1h', // Token válido por 1 hora (SLA: tokens curtos)
  JWT_ALGORITHM: 'HS256',
  REFRESH_TOKEN_EXPIRES_IN: '7d', // Refresh token: 7 dias
  MAX_LOGIN_ATTEMPTS: 5,
  LOCKOUT_TIME: 15 * 60 * 1000, // 15 minutos
  RATE_LIMIT_WINDOW: 15 * 60 * 1000, // 15 minutos
  RATE_LIMIT_MAX: 100 // 100 requisições por janela
};

// Cache para controle de tentativas de login
const loginAttempts = new Map();

class AuthenticationService {
  constructor() {
    this.validateEnvironment();
  }

  validateEnvironment() {
    if (!process.env.JWT_SECRET) {
      securityLogger.warn('JWT_SECRET não definida, usando secreto temporário');
      securityLogger.warn('DEFINA JWT_SECRET no ambiente de produção!');
    }
  }

  /**
   * Gera token JWT para agente
   * @param {Object} agent - Dados do agente
   * @returns {Object} tokens
   */
  generateTokens(agent) {
    const payload = {
      sub: agent.id,
      role: agent.role,
      squad: agent.squad,
      permissions: agent.permissions,
      iat: Math.floor(Date.now() / 1000)
    };

    const accessToken = jwt.sign(
      payload,
      SECURITY_CONFIG.JWT_SECRET,
      {
        expiresIn: SECURITY_CONFIG.JWT_EXPIRES_IN,
        algorithm: SECURITY_CONFIG.JWT_ALGORITHM
      }
    );

    const refreshToken = jwt.sign(
      { sub: agent.id, type: 'refresh' },
      SECURITY_CONFIG.JWT_SECRET,
      { expiresIn: SECURITY_CONFIG.REFRESH_TOKEN_EXPIRES_IN }
    );

    securityLogger.info(`Token gerado para agente ${agent.id}`, {
      agentId: agent.id,
      role: agent.role,
      timestamp: new Date().toISOString()
    });

    return {
      accessToken,
      refreshToken,
      expiresIn: 3600, // 1 hora em segundos
      tokenType: 'Bearer'
    };
  }

  /**
   * Verifica e decodifica token JWT
   * @param {string} token 
   * @returns {Object} payload decodificado
   */
  verifyToken(token) {
    try {
      return jwt.verify(token, SECURITY_CONFIG.JWT_SECRET, {
        algorithms: [SECURITY_CONFIG.JWT_ALGORITHM]
      });
    } catch (error) {
      securityLogger.warn('Falha na verificação do token', {
        error: error.message,
        timestamp: new Date().toISOString()
      });
      
      if (error.name === 'TokenExpiredError') {
        throw new Error('TOKEN_EXPIRED');
      }
      if (error.name === 'JsonWebTokenError') {
        throw new Error('TOKEN_INVALID');
      }
      throw new Error('TOKEN_VERIFICATION_FAILED');
    }
  }

  /**
   * Valida credenciais com rate limiting
   * @param {string} agentId 
   * @param {string} password 
   * @param {Function} validatePasswordCallback 
   * @returns {boolean}
   */
  async validateCredentials(agentId, password, validatePasswordCallback) {
    const ip = this.getClientIP();
    const attemptsKey = `${agentId}_${ip}`;
    
    // Verifica se está bloqueado
    const attempts = loginAttempts.get(attemptsKey) || { count: 0, lockedUntil: 0 };
    
    if (attempts.lockedUntil > Date.now()) {
      securityLogger.warn('Tentativa de login bloqueada por rate limiting', {
        agentId,
        ip,
        lockedUntil: new Date(attempts.lockedUntil).toISOString()
      });
      throw new Error('ACCOUNT_LOCKED');
    }

    try {
      const isValid = await validatePasswordCallback(password);
      
      if (isValid) {
        // Reset attempts on successful login
        loginAttempts.delete(attemptsKey);
        securityLogger.info('Login bem-sucedido', { agentId, ip });
        return true;
      } else {
        // Increment failed attempts
        attempts.count++;
        attempts.lockedUntil = attempts.count >= SECURITY_CONFIG.MAX_LOGIN_ATTEMPTS 
          ? Date.now() + SECURITY_CONFIG.LOCKOUT_TIME 
          : 0;
        
        loginAttempts.set(attemptsKey, attempts);
        
        securityLogger.warn('Tentativa de login falhou', {
          agentId,
          ip,
          attemptCount: attempts.count,
          remainingAttempts: SECURITY_CONFIG.MAX_LOGIN_ATTEMPTS - attempts.count
        });
        
        return false;
      }
    } catch (error) {
      securityLogger.error('Erro na validação de credenciais', {
        agentId,
        error: error.message
      });
      throw error;
    }
  }

  /**
   * Invalida token (logout)
   * @param {string} token 
   */
  invalidateToken(token) {
    try {
      const decoded = this.verifyToken(token);
      securityLogger.info('Token invalidado (logout)', {
        agentId: decoded.sub,
        timestamp: new Date().toISOString()
      });
    } catch (error) {
      // Token já inválido ou expirado
    }
  }

  /**
   * Atualiza token usando refresh token
   * @param {string} refreshToken 
   * @param {Object} agentData 
   * @returns {Object} novos tokens
   */
  refreshAccessToken(refreshToken, agentData) {
    try {

        error: error.message
      });
      throw new Error('REFRESH_TOKEN_INVALID');
    }
  }

  getClientIP() {

  }

const bcrypt = require('bcrypt');

class PasswordService {
  constructor() {
    this.SALT_ROUNDS = 12; // Recomendação OWASP 2024
  }

  /**
   * Hash de senha com bcrypt
   * @param {string} password 
   * @returns {Promise<string>} hash
   */
  async hashPassword(password) {
    if (!password || password.length < 8) {
      throw new Error('Password must be at least 8 characters long');
    }

    // Validação de força de senha
    this.validatePasswordStrength(password);

    const salt = await bcrypt.genSalt(this.SALT_ROUNDS);
    const hash = await bcrypt.hash(password, salt);
    
    return hash;
  }

  /**
   * Verifica senha contra hash
   * @param {string} password 
   * @param {string} hash 
   * @returns {Promise<boolean>}
   */
  async verifyPassword(password, hash) {
    if (!password || !hash) {
      return false;
    }

    try {
      return await bcrypt.compare(password, hash);
    } catch (error) {
      console.error('Error verifying password:', error);
      return false;
    }
  }

  /**
   * Valida força da senha
   * @param {string} password 
   */
  validatePasswordStrength(password) {
    const requirements = {
      minLength: 8,
      needsUppercase: /[A-Z]/.test(password),
      needsLowercase: /[a-z]/.test(password),
      needsNumber: /\d/.test(password),
      needsSpecial: /[!@#$%^&*(),.?":{}|<>]/.test(password)
    };

    const errors = [];

    if (password.length < requirements.minLength) {
      errors.push(`Password must be at least ${requirements.minLength} characters`);
    }
    if (!requirements.needsUppercase) {
      errors.push('Password must contain at least one uppercase letter');
    }
    if (!requirements.needsLowercase) {
      errors.push('Password must contain at least one lowercase letter');
    }
    if (!requirements.needsNumber) {
      errors.push('Password must contain at least one number');
    }
    if (!requirements.needsSpecial) {
      errors.push('Password must contain at least one special character');
    }

    if (errors.length > 0) {
      throw new Error(`Password validation failed: ${errors.join(', ')}`);
    }
  }

  /**
   * Gera senha aleatória segura
   * @returns {string}
   */
  generateSecurePassword() {
    const length = 16;

```javascript
  async initialize() {
    await this.connectRedis();
    await this.cleanDatabase();
    await this.createTestUsers();
  }

  async connectRedis() {
    await this.redisClient.connect();

  async cleanDatabase() {
    const tables = [
      'project_collaborators',
      'project_tasks',
      'project_materials',
      'ai_assistant_sessions',
      'projects',
      'users'
    ];

    for (const table of tables) {
      try {
        await this.dbPool.query(`TRUNCATE TABLE ${table} CASCADE`);
      } catch (error) {
        console.warn(`Could not truncate ${table}:`, error.message);
      }
    }
  }

  async createTestUsers() {
    // Criar usuário regular
    const regularUserRes = await request(this.apiUrl)
      .post('/api/v1/auth/register')
      .send({
        email: this.testUsers.regular.email,
        password: this.testUsers.regular.password,
        name: 'Test User'
      });

    if (regularUserRes.status === 201 || regularUserRes.status === 409) {
      const loginRes = await request(this.apiUrl)
        .post('/api/v1/auth/login')
        .send({
          email: this.testUsers.regular.email,
          password: this.testUsers.regular.password
        });
      
      this.testUsers.regular.token = loginRes.body.data.token;
    }

    // Criar usuário admin
    const adminRes = await request(this.apiUrl)
      .post('/api/v1/auth/register')
      .send({
        email: this.testUsers.admin.email,
        password: this.testUsers.admin.password,
        name: 'Admin User',
        role: 'admin'
      });

    if (adminRes.status === 201 || adminRes.status === 409) {
      const loginRes = await request(this.apiUrl)
        .post('/api/v1/auth/login')
        .send({
          email: this.testUsers.admin.email,
          password: this.testUsers.admin.password
        });
      
      this.testUsers.admin.token = loginRes.body.data.token;
    }
  }

    await this.redisClient.quit();
    await this.dbPool.end();
  }
```

import re
import json
from datetime import datetime
from typing import Dict, List, Optional, Tuple
import pandas as pd

class LogParser:
    """Parser inteligente de logs com suporte a múltiplos formatos"""
    
    def __init__(self):
        self.common_patterns = {
            'timestamp': r'(\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2})',
            'level': r'(INFO|ERROR|WARN|DEBUG|FATAL)',
            'service': r'service[=:]["\']?([^"\'\s]+)["\']?',
            'message': r'message[=:]["\'](.+?)["\']',
            'trace_id': r'trace_id[=:]["\']?([^"\'\s]+)["\']?'
        }
    
    def parse_line(self, log_line: str) -> Optional[Dict]:
        """Parse uma linha de log individual"""
        try:
            parsed = {
                'raw': log_line,
                'timestamp': None,
                'level': 'UNKNOWN',
                'service': 'unknown',
                'message': '',
                'metadata': {}
            }
            
            # Tenta extrair timestamp
            ts_match = re.search(self.common_patterns['timestamp'], log_line)
            if ts_match:
                parsed['timestamp'] = ts_match.group(1)
            
            # Tenta extrair nível
            for level in ['ERROR', 'WARN', 'INFO', 'DEBUG', 'FATAL']:
                if level in log_line:
                    parsed['level'] = level
                    break
            
            # Extrai mensagem principal (última parte após nível)
            parts = log_line.split(level) if parsed['level'] != 'UNKNOWN' else [log_line]
            if len(parts) > 1:
                parsed['message'] = parts[-1].strip()
            else:
                parsed['message'] = log_line.strip()
            
            # Tenta extrair metadados no formato chave=valor
            kv_pattern = r'(\w+)=([^=\s]+|"[^"]*")'
            for key, value in re.findall(kv_pattern, log_line):
                parsed['metadata'][key] = value.strip('"')
            
            # Tenta identificar serviço
            for key in ['service', 'app', 'module']:
                if key in parsed['metadata']:
                    parsed['service'] = parsed['metadata'][key]
                    break
            
            return parsed
            
        except Exception as e:
            print(f"Erro ao parsear log: {e}")
            return None
    
    def parse_file(self, filepath: str) -> pd.DataFrame:
        """Parse um arquivo de log completo"""
        logs = []
        with open(filepath, 'r', encoding='utf-8') as f:
            for line in f:
                parsed = self.parse_line(line)
                if parsed:
                    logs.append(parsed)
        
        return pd.DataFrame(logs)

import numpy as np
from typing import List, Dict, Tuple
from transformers import AutoTokenizer, AutoModelForSequenceClassification
import torch
import torch.nn.functional as F
from collections import Counter
import nltk
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords

# Download recursos NLTK (apenas primeira execução)
try:
    nltk.data.find('tokenizers/punkt')
except LookupError:
    nltk.download('punkt')
    nltk.download('stopwords')

class NLPAnalyzer:
    """Analisador NLP para classificação e clusterização de logs"""
    
    def __init__(self, model_name: str = "distilbert-base-uncased"):
        self.device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
        self.tokenizer = AutoTokenizer.from_pretrained(model_name)
        self.model = AutoModelForSequenceClassification.from_pretrained(
            model_name,
            num_labels=7  # Para nossas 7 categorias de problemas
        ).to(self.device)
        
        # Mapeamento de categorias
        self.category_map = {
            0: "database_connection",
            1: "authentication_failure",
            2: "performance_degradation",
            3: "resource_exhaustion",
            4: "external_service_failure",
            5: "configuration_error",
            6: "security_alert"
        }
        
        # Stopwords em português e inglês
        self.stop_words = set(stopwords.words('english') + stopwords.words('portuguese'))
    
    def extract_keywords(self, text: str, top_n: int = 5) -> List[str]:
        """Extrai palavras-chave de um texto de log"""
        tokens = word_tokenize(text.lower())
        tokens = [t for t in tokens if t.isalnum() and t not in self.stop_words]
        
        # Contagem de frequência
        word_freq = Counter(tokens)
        return [word for word, _ in word_freq.most_common(top_n)]
    
    def classify_log(self, log_message: str) -> Tuple[str, float]:
        """Classifica uma mensagem de log em uma categoria de problema"""
        try:
            inputs = self.tokenizer(
                log_message,
                truncation=True,
                padding=True,
                max_length=128,
                return_tensors="pt"
            ).to(self.device)
            
            with torch.no_grad():
                outputs = self.model(**inputs)
                probabilities = F.softmax(outputs.logits, dim=-1)
                predicted_class = torch.argmax(probabilities, dim=-1).item()
                confidence = probabilities[0][predicted_class].item()
            
            return self.category_map.get(predicted_class, "unknown"), confidence
            
        except Exception as e:
            print(f"Erro na classificação: {e}")
            return "unknown", 0.0
    
    def analyze_logs_batch(self, logs_df: pd.DataFrame) -> pd.DataFrame:
        """Analisa um batch de logs"""
        results = []
        
        for _, row in logs_df.iterrows():
            category, confidence = self.classify_log(row['message'])
            keywords = self.extract_keywords(row['message'])
            
            results.append({
                'timestamp': row['timestamp'],
                'level': row['level'],
                'service': row['service'],
                'original_message': row['message'],
                'predicted_category': category,
                'confidence': confidence,
                'keywords': keywords,
                'is_anomaly': confidence > 0.8 and category != "unknown"
            })
        
        return pd.DataFrame(results)

from collections import defaultdict
from datetime import datetime, timedelta
import hashlib
import json

class PatternDetector:
    """Detector de padrões recorrentes em logs"""
    
    def __init__(self, window_size: int = 60):  # 60 minutos
        self.window_size = window_size
        self.patterns = defaultdict(list)
        self.sequence_cache = {}
    
    def generate_pattern_signature(self, logs_sequence: List[Dict]) -> str:
        """Gera uma assinatura única para uma sequência de logs"""
        pattern_str = ""
        for log in logs_sequence:
            pattern_str += f"{log['level']}:{log['predicted_category']}:"
        
        return hashlib.md5(pattern_str.encode()).hexdigest()
    
    def detect_sequences(self, logs_df: pd.DataFrame, time_window_minutes: int = 5) -> List[Dict]:
        """Detecta sequências de logs que ocorrem próximas no tempo"""
        sequences = []
        current_sequence = []
        last_time = None
        
        # Ordena por timestamp
        logs_df = logs_df.sort_values('timestamp')
        
        for _, row in logs_df.iterrows():
            if row['level'] in ['ERROR', 'FATAL']:
                current_time = datetime.strptime(row['timestamp'], '%Y-%m-%d %H:%M:%S')
                
                if last_time is None:
                    current_sequence.append(row.to_dict())
                else:
                    time_diff = (current_time - last_time).total_seconds() / 60
                    
                    if time_diff <= time_window_minutes:
                        current_sequence.append(row.to_dict())
                    else:
                        if len(current_sequence) >= 2:
                            sequences.append(current_sequence.copy())
                        current_sequence = [row.to_dict()]
                
                last_time = current_time
        
        # Adiciona última sequência
        if len(current_sequence) >= 2:
            sequences.append(current_sequence)
        
        return sequences
    
    def identify_recurring_patterns(self, sequences: List[List[Dict]]) -> Dict:
        """Identifica padrões recorrentes nas sequências"""
        pattern_counts = defaultdict(int)
        pattern_examples = {}
        
        for sequence in sequences:
            pattern_sig = self.generate_pattern_signature(sequence)
            pattern_counts[pattern_sig] += 1
            
            if pattern_sig not in pattern_examples:
                pattern_examples[pattern_sig] = {
                    'sequence': sequence,
                    'first_seen': datetime.now(),
                    'count': 0
                }
        
        # Filtra padrões que ocorrem múltiplas vezes
        recurring_patterns = {}
        for pattern_sig, count in pattern_counts.items():
            if count >= 2:  # Padrão ocorreu pelo menos 2 vezes
                recurring_patterns[pattern_sig] = {
                    'occurrences': count,
                    'example': pattern_examples[pattern_sig]['sequence'],
                    'last_seen': datetime.now()
                }
        
        return recurring_patterns

class AnomalyDetector:
    """Detector de anomalias baseado em machine learning"""
    
    def __init__(self, contamination: float = 0.1):
        self.model = IsolationForest(
            contamination=contamination,
            random_state=42,
            n_estimators=100
        )
        self.scaler = StandardScaler()
        self.is_fitted = False
        
        # Features para treinamento
        self.feature_columns = [
            'error_count', 'warning_count', 'unique_services',
            'avg_confidence', 'category_diversity'
        ]
    
    def extract_features(self, logs_df: pd.DataFrame, window_minutes: int = 15) -> pd.DataFrame:
        """Extrai features para detecção de anomalias"""
        features = []
        timestamps = []
        
        # Converte timestamps
        logs_df['datetime'] = pd.to_datetime(logs_df['timestamp'])
        
        # Cria janelas temporais
        start_time = logs_df['datetime'].min()
        end_time = logs_df['datetime'].max()
        
        current_time = start_time
        while current_time <= end_time:
            window_end = current_time + timedelta(minutes=window_minutes)
            
            window_logs = logs_df[
                (logs_df['datetime'] >= current_time) & 
                (logs_df['datetime'] < window_end)
            ]
            
            if len(window_logs) > 0:
                # Calcula features
                error_count = len(window_logs[window_logs['level'] == 'ERROR'])
                warning_count = len(window_logs[window_logs['level'] == 'WARN'])
                unique_services = window_logs['service'].nunique()
                avg_confidence = window_logs['confidence'].mean() if 'confidence' in window_logs.columns else 0
                category_diversity = window_logs['predicted_category'].nunique() if 'predicted_category' in window_logs.columns else 0
                
                features.append([
                    error_count, warning_count, unique_services,
                    avg_confidence, category_diversity
                ])
                timestamps.append(current_time)
            
            current_time = window_end
        
        features_df = pd.DataFrame(features, columns=self.feature_columns)
        features_df['timestamp'] = timestamps
        
        return features_df
    
    def train(self, normal_logs_features: pd.DataFrame):
        """Treina o modelo com dados normais"""
        X = normal_logs_features[self.feature_columns].values
        X_scaled = self.scaler.fit_transform(X)
        self.model.fit(X_scaled)
        self.is_fitted = True
    
    def detect(self, logs_features: pd.DataFrame) -> pd.DataFrame:
        """Detecta anomalias em novos dados"""
        if not self.is_fitted:
            raise ValueError("Modelo não treinado. Chame train() primeiro.")
        
        X = logs_features[self.feature_columns].values
        X_scaled = self.scaler.transform(X)
        
        predictions = self.model.predict(X_scaled)
        anomaly_scores = self.model.decision_function(X_scaled)
        
        logs_features['is_anomaly'] = predictions == -1
        logs_features['anomaly_score'] = anomaly_scores
        
        return logs_features

```javascript
const fs = require('fs');
const path = require('path');
const { execSync } = require('child_process');

class LicenseAuditor {
  constructor() {
    this.approvedLicenses = [
      'MIT',
      'Apache-2.0',
      'BSD-2-Clause',
      'BSD-3-Clause',
      'ISC',
      'Unlicense',
      'CC0-1.0'
    ];
    
    this.copyleftLicenses = [
      'GPL',
      'AGPL',
      'LGPL',
      'MPL',
      'EPL'
    ];
    
    this.licenseCache = new Map();
  }

  async auditProject() {
    const packageJson = JSON.parse(fs.readFileSync('package.json', 'utf8'));
    const dependencies = {
      ...packageJson.dependencies,
      ...packageJson.devDependencies
    };
    
    const auditResults = {
      timestamp: new Date().toISOString(),
      totalDependencies: Object.keys(dependencies).length,
      approved: [],
      rejected: [],
      warnings: [],
      unknown: []
    };

    for (const [pkg, version] of Object.entries(dependencies)) {
      try {
        const licenseInfo = await this.getPackageLicense(pkg, version);
        
        if (this.isLicenseApproved(licenseInfo.license)) {
          auditResults.approved.push({
            package: pkg,
            version: version,
            license: licenseInfo.license,
            url: licenseInfo.url
          });
        } else if (this.isCopyleft(licenseInfo.license)) {
          auditResults.rejected.push({
            package: pkg,
            version: version,
            license: licenseInfo.license,
            url: licenseInfo.url,
            reason: 'Licença copyleft não aprovada'
          });
        } else if (!licenseInfo.license || licenseInfo.license === 'UNKNOWN') {
          auditResults.unknown.push({
            package: pkg,
            version: version,
            license: licenseInfo.license,
            url: licenseInfo.url
          });
        } else {
          auditResults.warnings.push({
            package: pkg,
            version: version,
            license: licenseInfo.license,
            url: licenseInfo.url,
            reason: 'Licença não classificada'
          });
        }
      } catch (error) {
        auditResults.unknown.push({
          package: pkg,
          version: version,
          license: 'ERROR',
          error: error.message
        });
      }
    }

    return auditResults;
  }

  async getPackageLicense(packageName, version) {
    const cacheKey = `${packageName}@${version}`;
    if (this.licenseCache.has(cacheKey)) {
      return this.licenseCache.get(cacheKey);
    }

    try {
      // Tenta obter informações do npm
      const npmInfo = JSON.parse(
        execSync(`npm view ${packageName}@${version} license repository.url --json`, {
          encoding: 'utf8',
          stdio: ['pipe', 'pipe', 'ignore']
        })
      );

      const licenseInfo = {
        license: Array.isArray(npmInfo.license) ? npmInfo.license[0] : npmInfo.license,
        url: npmInfo['repository.url'] || `https://www.npmjs.com/package/${packageName}`
      };

      this.licenseCache.set(cacheKey, licenseInfo);
      return licenseInfo;
    } catch (error) {
      // Fallback: verifica LICENSE no node_modules
      const licensePath = path.join('node_modules', packageName, 'LICENSE');
      if (fs.existsSync(licensePath)) {
        const content = fs.readFileSync(licensePath, 'utf8');
        const detectedLicense = this.detectLicenseFromContent(content);
        return {
          license: detectedLicense || 'UNKNOWN',
          url: `https://www.npmjs.com/package/${packageName}`
        };
      }
      
      return {
        license: 'UNKNOWN',
        url: `https://www.npmjs.com/package/${packageName}`
      };
    }
  }

  detectLicenseFromContent(content) {
    const contentUpper = content.toUpperCase();
    
    if (contentUpper.includes('MIT LICENSE') || contentUpper.includes('PERMISSION IS HEREBY GRANTED')) {
      return 'MIT';
    } else if (contentUpper.includes('APACHE LICENSE') || contentUpper.includes('APACHE 2.0')) {
      return 'Apache-2.0';
    } else if (contentUpper.includes('GNU GENERAL PUBLIC LICENSE')) {
      return 'GPL';
    } else if (contentUpper.includes('BSD LICENSE')) {
      if (contentUpper.includes('CLAUSE 2')) {
        return 'BSD-2-Clause';
      } else if (contentUpper.includes('CLAUSE 3')) {
        return 'BSD-3-Clause';
      }
      return 'BSD';
    }
    
    return null;
  }

  isLicenseApproved(license) {
    if (!license) return false;
    
    const licenseUpper = license.toUpperCase();
    return this.approvedLicenses.some(approved => 
      licenseUpper.includes(approved.toUpperCase())
    );
  }

  isCopyleft(license) {
    if (!license) return false;
    
    const licenseUpper = license.toUpperCase();
    return this.copyleftLicenses.some(copyleft => 
      licenseUpper.includes(copyleft.toUpperCase())
    );
  }

  generateLicenseReport(results) {
    const report = {
      summary: {
        total: results.totalDependencies,
        approved: results.approved.length,
        rejected: results.rejected.length,
        warnings: results.warnings.length,
        unknown: results.unknown.length,
        complianceScore: ((results.approved.length / results.totalDependencies) * 100).toFixed(2)
      },
      details: {
        approved: results.approved,
        rejected: results.rejected,
        warnings: results.warnings,
        unknown: results.unknown
      },
      timestamp: results.timestamp
    };

    return report;
  }
}

module.exports = LicenseAuditor;
```
