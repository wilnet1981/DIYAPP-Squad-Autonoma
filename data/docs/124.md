# DIYAPP Evolution - V12 Core - Arquitetura de Microsserviços

## 1. ADR-001: Arquitetura de Microsserviços V12

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead V12 Core

### CONTEXTO:
A V11 do DIYAPP atingiu limites de escalabilidade devido à arquitetura monolítica. Necessitamos de:
1. Deploy independente por funcionalidade
2. Escala horizontal específica por domínio
3. Isolamento de falhas
4. Equipes autônomas por microsserviço
5. Suporte a múltiplas linguagens por domínio

### DECISÃO:
Adotar arquitetura de microsserviços com os seguintes princípios:
- **12 microsserviços** baseados em domínios de negócio
- **Comunicação síncrona via gRPC** para operações críticas
- **Comunicação assíncrona via Apache Kafka** para eventos de domínio
- **API Gateway** como ponto único de entrada
- **Service Mesh** (Istio) para observabilidade e resiliência
- **Banco de dados por serviço** (Database-per-service)

### OPÇÕES CONSIDERADAS:
- **Opção A**: Arquitetura monolítica com módulos
  - Prós: Simplicidade inicial, transações ACID fáceis
  - Contras: Escalabilidade limitada, deploy único, acoplamento alto
- **Opção B**: Microsserviços com REST/HTTP
  - Prós: Simples, amplamente adotado, fácil debug
  - Contras: Overhead de serialização, sem contrato forte, versionamento complexo
- **Opção C**: Microsserviços com gRPC + Eventos (ESCOLHIDA)
  - Prós: Contrato forte com Protobuf, alta performance, streaming bidirecional
  - Contras: Curva de aprendizado, tooling menos maduro que REST

### CONSEQUÊNCIAS:
**Positivas:**
- Deploy independente por serviço
- Escala horizontal granular
- Falhas isoladas por domínio
- Equipes autônomas com stack própria
- Performance otimizada com gRPC

**Negativas:**
- Complexidade distribuída
- Transações distribuídas (Saga Pattern)
- Monitoramento distribuído necessário
- Latência de rede entre serviços

**Riscos:**
1. **Cascata de falhas**: Mitigar com circuit breakers e timeouts
2. **Consistência eventual**: Documentar claramente para negócio
3. **Operacional complexo**: Investir em observabilidade desde dia 1

## 3. Protobuf Contracts (Exemplo User Service)

**`protos/user/user.proto`**
```proto
syntax = "proto3";

package diyapp.user.v1;

import "google/protobuf/timestamp.proto";
import "protos/shared/common.proto";

service UserService {
  rpc CreateUser(CreateUserRequest) returns (UserResponse);
  rpc GetUser(GetUserRequest) returns (UserResponse);
  rpc UpdateUser(UpdateUserRequest) returns (UserResponse);
  rpc ListUsers(ListUsersRequest) returns (ListUsersResponse);
  rpc StreamUserUpdates(StreamUserUpdatesRequest) returns (stream UserUpdate);
}

message User {
  string id = 1;
  string email = 2;
  string username = 3;
  UserStatus status = 4;
  google.protobuf.Timestamp created_at = 5;
  google.protobuf.Timestamp updated_at = 6;
  map<string, string> metadata = 7;
}

enum UserStatus {
  USER_STATUS_UNSPECIFIED = 0;
  USER_STATUS_ACTIVE = 1;
  USER_STATUS_INACTIVE = 2;
  USER_STATUS_SUSPENDED = 3;
}

message CreateUserRequest {
  string email = 1;
  string username = 2;
  string password_hash = 3;
  map<string, string> metadata = 4;
}

message UserResponse {
  User user = 1;
  diyapp.shared.v1.OperationStatus status = 2;
}

message GetUserRequest {
  string user_id = 1;
}

message UpdateUserRequest {
  string user_id = 1;
  optional string email = 2;
  optional string username = 3;
  optional UserStatus status = 4;
  map<string, string> metadata = 5;
}

message ListUsersRequest {
  int32 page = 1;
  int32 page_size = 2;
  string filter = 3;
}

message ListUsersResponse {
  repeated User users = 1;
  int32 total = 2;
  int32 page = 3;
  int32 page_size = 4;
}

message StreamUserUpdatesRequest {
  string user_id = 1;
}

message UserUpdate {
  User user = 1;
  UpdateType update_type = 2;
  google.protobuf.Timestamp timestamp = 3;
}

enum UpdateType {
  UPDATE_TYPE_UNSPECIFIED = 0;
  UPDATE_TYPE_CREATED = 1;
  UPDATE_TYPE_UPDATED = 2;
  UPDATE_TYPE_DELETED = 3;
}
```

## 5. CI/CD Pipeline (GitHub Actions)

**`.github/workflows/ci-master.yml`**
```yaml
name: CI Master Pipeline

on:
  push:
    branches: [ main, master ]
  pull_request:
    branches: [ main, master ]

jobs:
  lint-and-test:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        service: [api-gateway, user-service, project-service, auth-service]
    
    steps:
    - uses: actions/checkout@v4
    
    - name: Setup Node.js
      uses: actions/setup-node@v4
      with:
        node-version: '18'
        cache: 'npm'
        cache-dependency-path: ${{ matrix.service }}/package-lock.json
    
    - name: Install dependencies
      working-directory: ./${{ matrix.service }}
      run: npm ci
    
    - name: Lint
      working-directory: ./${{ matrix.service }}
      run: npm run lint
    
    - name: Run tests
      working-directory: ./${{ matrix.service }}
      run: npm test -- --coverage
    
    - name: Upload coverage
      uses: codecov/codecov-action@v3
      with:
        file: ./${{ matrix.service }}/coverage/lcov.info
        flags: ${{ matrix.service }}
        name: codecov-umbrella

  security-scan:
    runs-on: ubuntu-latest
    steps:
    - uses: actions/checkout@v4
    
    - name: Run Trivy vulnerability scanner
      uses: aquasecurity/trivy-action@master
      with:
        scan-type: 'fs'
        scan-ref: '.'
        format: 'sarif'
        output: 'trivy-results.sarif'
    
    - name: Upload Trivy scan results to GitHub Security tab
      uses: github/codeql-action/upload-sarif@v2
      with:
        sarif_file: 'trivy-results.sarif'
    
    - name: Snyk Security Scan
      uses: snyk/actions/node@master
      env:
        SNYK_TOKEN: ${{ secrets.SNYK_TOKEN }}
      with:
        args: --severity-threshold=high

  build-and-push:
    needs: [lint-and-test, security-scan]
    runs-on: ubuntu-latest
    if: github.event_name == 'push'
    
    steps:
    - uses: actions/checkout@v4
    
    - name: Set up Docker Buildx
      uses: docker/setup-buildx-action@v3
    
    - name: Login to DockerHub
      uses: docker/login-action@v3
      with:
        username: ${{ secrets.DOCKERHUB_USERNAME }}
        password: ${{ secrets.DOCKERHUB_TOKEN }}
    
    - name: Build and push
      uses: docker/build-push-action@v5
      with:
        context: .
        file: ./api-gateway/Dockerfile
        push: true
        tags: |
          ${{ secrets.DOCKERHUB_USERNAME }}/diyapp-api-gateway:${{ github.sha }}
          ${{ secrets.DOCKERHUB_USERNAME }}/diyapp-api-gateway:latest
        cache-from: type=gha
        cache-to: type=gha,mode=max
```

**`.github/workflows/cd-canary.yml`**
```yaml
name: CD Canary Deployment

on:
  workflow_dispatch:
    inputs:
      environment:
        description: 'Environment'
        required: true
        default: 'staging'
        type: choice
        options:
        - staging
        - production
      version:
        description: 'Docker image tag'
        required: true
        default: 'latest'

jobs:
  deploy-canary:
    runs-on: ubuntu-latest
    environment: ${{ github.event.inputs.environment }}
    
    steps:
    - name: Checkout
      uses: actions/checkout@v4
    
    - name: Configure AWS credentials
      uses: aws-actions/configure-aws-credentials@v4
      with:
        aws-access-key-id: ${{ secrets.AWS_ACCESS_KEY_ID }}
        aws-secret-access-key: ${{ secrets.AWS_SECRET_ACCESS_KEY }}
        aws-region: us-east-1
    
    - name: Deploy to EKS with Canary
      run: |
        # Update Kubernetes manifests with new image
        sed -i "s|image:.*|image: ${{ secrets.DOCKERHUB_USERNAME }}/diyapp-api-gateway:${{ github.event.inputs.version }}|g" ./k8s/api-gateway-deployment.yaml
        
        # Apply canary deployment (10% traffic)
        kubectl apply -f ./k8s/api-gateway-canary.yaml
        
        # Wait for canary to stabilize
        sleep 60
        
        # Run integration tests against canary
        ./scripts/run-canary-tests.sh
        
        # If tests pass, scale to 100%
        kubectl apply -f ./k8s/api-gateway-deployment.yaml
        
        # Clean up canary
        kubectl delete -f ./k8s/api-gateway-canary.yaml
```

class StructuredLogger {
  constructor() {

      format: combine(
        timestamp(),
        errors({ stack: true }),
        json()
      ),
      defaultMeta: { service: 'hive-manager' },
      transports: [
        new transports.Console({
          format: combine(
            timestamp(),
            format.colorize(),
            format.printf(({ timestamp, level, message, correlation_id, ...meta }) => {
              return `${timestamp} [${level}] [correlation:${correlation_id || 'none'}] ${message} ${Object.keys(meta).length ? JSON.stringify(meta) : ''}`;
            })
          )
        }),
        new transports.File(config.logging.file)
      ]
    });
  }

  info(message, meta = {}) {
    this.logger.info(message, meta);
  }

  error(message, meta = {}) {
    this.logger.error(message, meta);
  }

  warn(message, meta = {}) {
    this.logger.warn(message, meta);
  }

  debug(message, meta = {}) {

const Redis = require('ioredis');
const config = require('../config');
const logger = require('./logger');
const CircuitBreaker = require('opossum');

class ResilientRedisClient {
  constructor() {
    this.client = null;
    this.circuitBreaker = null;
    this.isConnected = false;
    this.connectionAttempts = 0;
    this.MAX_RETRIES = 5;
  }

  async connect() {
    try {
      this.client = new Redis({
        host: config.redis.host,
        port: config.redis.port,
        password: config.redis.password,
        retryStrategy: config.redis.retryStrategy,
        maxRetriesPerRequest: config.redis.maxRetriesPerRequest
      });

      // Event listeners para monitoramento
      this.client.on('connect', () => {
        this.isConnected = true;
        this.connectionAttempts = 0;
        logger.info('Redis connected successfully');
      });

      this.client.on('error', (error) => {
        logger.error('Redis connection error', { error: error.message });
        this.isConnected = false;
        this.connectionAttempts++;
        
        if (this.connectionAttempts >= this.MAX_RETRIES) {
          logger.error('Max Redis connection attempts reached', { attempts: this.connectionAttempts });
        }
      });

      this.client.on('close', () => {
        logger.warn('Redis connection closed');
        this.isConnected = false;
      });

      // Configurar circuit breaker para operações Redis
      this.circuitBreaker = new CircuitBreaker(
        async (command, ...args) => {
          if (!this.isConnected) {
            throw new Error('Redis not connected');
          }
          return await this.client[command](...args);
        },
        {
          timeout: 5000, // 5 segundos timeout para operações Redis
          errorThresholdPercentage: 50,
          resetTimeout: 10000
        }
      );

      this.circuitBreaker.fallback(() => {
        logger.warn('Redis circuit breaker open - using fallback');
        return null; // Fallback: retorna null quando Redis está indisponível
      });

      this.circuitBreaker.on('open', () => {
        logger.error('Redis circuit breaker opened');
      });

      this.circuitBreaker.on('close', () => {
        logger.info('Redis circuit breaker closed');
      });

      await this.client.ping();
      return this.client;
    } catch (error) {
      logger.error('Failed to connect to Redis', { error: error.message });
      throw error;
    }
  }

  async execute(command, ...args) {
    try {
      return await this.circuitBreaker.fire(command, ...args);
    } catch (error) {
      logger.error('Redis operation failed', { 
        command, 
        args: this.maskSensitiveData(args),
        error: error.message 
      });
      throw error;
    }
  }

  // Mascarar dados sensíveis nos logs
  maskSensitiveData(args) {
    return args.map(arg => {
      if (typeof arg === 'string' && arg.length > 10) {
        return arg.substring(0, 4) + '***' + arg.substring(arg.length - 4);
      }
      return arg;
    });
  }

  async disconnect() {
    if (this.client) {
      await this.client.quit();
      this.isConnected = false;
    }
  }

  getClient() {
    if (!this.isConnected) {
      throw new Error('Redis client not connected');
    }
    return this.client;
  }

  getStatus() {
    return {
      isConnected: this.isConnected,
      connectionAttempts: this.connectionAttempts,
      circuitBreakerState: this.circuitBreaker ? this.circuitBreaker.status : 'not_initialized'
    };
  }
}

module.exports = new ResilientRedisClient();

const Bull = require('bull');
const config = require('../config');
const logger = require('../services/logger');
const redisClient = require('../services/redis-client');

class ResilientTaskQueue {
  constructor() {
    this.queues = new Map();
    this.initialized = false;
  }

  async initialize() {
    try {
      await redisClient.connect();
      
      // Criar filas configuradas
      for (const [queueName, queueConfig] of Object.entries(config.queues)) {
        const queue = new Bull(queueConfig.name, {
          redis: {
            host: config.redis.host,
            port: config.redis.port,
            password: config.redis.password
          },
          defaultJobOptions: {
            attempts: queueConfig.attempts || 3,
            backoff: queueConfig.backoff || {
              type: 'exponential',
              delay: 1000
            },
            removeOnComplete: queueConfig.removeOnComplete || 100,
            removeOnFail: queueConfig.removeOnFail || 50
          }
        });

        // Event listeners para monitoramento
        queue.on('completed', (job) => {
          logger.info('Job completed', {
            queue: queueConfig.name,
            jobId: job.id,
            duration: job.finishedOn - job.processedOn
          });
        });

        queue.on('failed', (job, error) => {
          logger.error('Job failed', {
            queue: queueConfig.name,
            jobId: job.id,
            error: error.message,
            attemptsMade: job.attemptsMade
          });
        });

        queue.on('stalled', (job) => {
          logger.warn('Job stalled', {
            queue: queueConfig.name,
            jobId: job.id
          });
        });

        queue.on('error', (error) => {
          logger.error('Queue error', {
            queue: queueConfig.name,
            error: error.message
          });
        });

        this.queues.set(queueName, {
          instance: queue,
          config: queueConfig
        });
      }

      this.initialized = true;
      logger.info('Task queues initialized successfully');
    } catch (error) {
      logger.error('Failed to initialize task queues', { error: error.message });
      throw error;
    }
  }

  async addJob(queueName, jobData, options = {}) {
    if (!this.initialized) {
      throw new Error('Task queues not initialized');
    }

    const queue = this.queues.get(queueName);
    if (!queue) {
      throw new Error(`Queue ${queueName} not found`);
    }

    try {
      const job = await queue.instance.add(jobData, options);
      logger.info('Job added to queue', {
        queue: queueName,
        jobId: job.id,
        data: this.sanitizeJobData(jobData)
      });
      return job;
    } catch (error) {
      logger.error('Failed to add job to queue', {
        queue: queueName,
        error: error.message,
        data: this.sanitizeJobData(jobData)
      });
      throw error;
    }
  }

  async processQueue(queueName, processor) {
    const queue = this.queues.get(queueName);
    if (!queue) {
      throw new Error(`Queue ${queueName} not found`);
    }

    queue.instance.process(queue.config.concurrency || 1, async (job) => {
      const startTime = Date.now();
      const correlationId = job.id;
      const jobLogger = logger.withCorrelation(correlationId);

      try {
        jobLogger.info('Starting job processing', {
          queue: queueName,
          jobId: job.id,
          attempt: job.attemptsMade + 1
        });

        const result = await processor(job.data, job);
        const duration = Date.now() - startTime;

        jobLogger.info('Job processed successfully', {
          queue: queueName,
          jobId: job.id,
          duration,
          result: this.sanitizeJobData(result)
        });

        return result;
      } catch (error) {
        const duration = Date.now() - startTime;
        jobLogger.error('Job processing failed', {
          queue: queueName,
          jobId: job.id,
          duration,
          error: error.message,
          stack: error.stack
        });
        throw error;
      }
    });
  }

  sanitizeJobData(data) {
    // Remove dados sensíveis para logging
    const sanitized = { ...data };
    if (sanitized.apiKey) sanitized.apiKey = '***';
    if (sanitized.password) sanitized.password = '***';
    if (sanitized.token) sanitized.token = '***';
    return sanitized;
  }

  async getQueueMetrics(queueName) {
    const queue = this.queues.get(queueName);
    if (!queue) {
      throw new Error(`Queue ${queueName} not found`);
    }

    const [waiting, active, completed, failed, delayed] = await Promise.all([
      queue.instance.getWaitingCount(),
      queue.instance.getActiveCount(),
      queue.instance.getCompletedCount(),
      queue.instance.getFailedCount(),
      queue.instance.getDelayedCount()
    ]);

    return {
      name: queueName,
      waiting,
      active,
      completed,
      failed,
      delayed,
      concurrency: queue.config.concurrency
    };
  }

  async cleanup() {
    for (const [queueName, queue] of this.queues) {
      await queue.instance.close();
    }
    await redisClient.disconnect();
    this.initialized = false;
  }
}

// src/hooks/useWebSocket.ts
import { useEffect, useRef, useCallback } from 'react';

interface UseWebSocketOptions {
  url: string;
  onMessage: (message: any) => void;
  onConnect?: () => void;
  onDisconnect?: () => void;
  onError?: (error: Event) => void;
  reconnectInterval?: number;
  maxReconnectAttempts?: number;
}

export const useWebSocket = (options: UseWebSocketOptions) => {
  const {
    url,
    onMessage,
    onConnect,
    onDisconnect,
    onError,
    reconnectInterval = 5000,
    maxReconnectAttempts = 10
  } = options;
  
  const wsRef = useRef<WebSocket | null>(null);
  const reconnectAttempts = useRef(0);
  const reconnectTimeoutRef = useRef<NodeJS.Timeout>();
  
  const connect = useCallback(() => {
    if (wsRef.current?.readyState === WebSocket.OPEN) {
      return;
    }
    
    try {
      const ws = new WebSocket(url);
      
      ws.onopen = () => {
        console.log('WebSocket conectado');
        reconnectAttempts.current = 0;
        onConnect?.();
      };
      
      ws.onmessage = (event) => {
        try {
          const message = JSON.parse(event.data);
          onMessage(message);
        } catch (error) {
          console.error('Erro ao processar mensagem WebSocket:', error);
        }
      };
      
      ws.onclose = () => {
        console.log('WebSocket desconectado');
        onDisconnect?.();
        
        // Tentar reconectar
        if (reconnectAttempts.current < maxReconnectAttempts) {
          reconnectAttempts.current += 1;
          reconnectTimeoutRef.current = setTimeout(() => {
            console.log(`Tentando reconectar (${reconnectAttempts.current}/${maxReconnectAttempts})`);
            connect();
          }, reconnectInterval);
        }
      };
      
      ws.onerror = (error) => {
        console.error('WebSocket error:', error);
        onError?.(error);
      };
      
      wsRef.current = ws;
    } catch (error) {
      console.error('Erro ao conectar WebSocket:', error);
    }
  }, [url, onMessage, onConnect, onDisconnect, onError, reconnectInterval, maxReconnectAttempts]);
  
  const disconnect = useCallback(() => {
    if (reconnectTimeoutRef.current) {
      clearTimeout(reconnectTimeoutRef.current);
    }
    
    if (wsRef.current) {
      wsRef.current.close();
      wsRef.current = null;
    }
  }, []);
  
  const sendMessage = useCallback((message: any) => {
    if (wsRef.current?.readyState === WebSocket.OPEN) {
      wsRef.current.send(JSON.stringify(message));
      return true;
    }
    console.warn('WebSocket não está conectado');
    return false;
  }, []);
  
  useEffect(() => {
    return () => {
      disconnect();
    };
  }, [disconnect]);
  
  return {
    connect,
    disconnect,
    sendMessage,
    isConnected: wsRef.current?.readyState === WebSocket.OPEN
  };
};

DECLARE
    v_score DECIMAL(3,2) := 1.0;
    v_error_count INTEGER := 0;
    v_total_checks INTEGER := 0;
BEGIN
    -- Verificar completude
    IF EXISTS (
        SELECT 1 FROM data_lake.agent_telemetry_raw 
        WHERE telemetry_id = p_telemetry_id 
        AND (agent_id IS NULL OR event_type IS NULL)
    ) THEN
        v_error_count := v_error_count + 1;
    END IF;
    v_total_checks := v_total_checks + 1;
    
    -- Verificar timestamps válidos
    IF EXISTS (
        SELECT 1 FROM data_lake.agent_telemetry_raw 
        WHERE telemetry_id = p_telemetry_id 
        AND event_timestamp > NOW() + INTERVAL '5 minutes'
    ) THEN
        v_error_count := v_error_count + 1;
    END IF;
    v_total_checks := v_total_checks + 1;
    
    -- Calcular score
    v_score := 1.0 - (v_error_count::DECIMAL / v_total_checks::DECIMAL);
    RETURN GREATEST(0.0, v_score);
END;
$$ LANGUAGE plpgsql;

-- Procedure para limpeza automática de dados antigos (retention policy)
CREATE OR REPLACE PROCEDURE data_lake.apply_retention_policy(
    p_retention_days INTEGER DEFAULT 90
) AS $$
BEGIN
    -- Raw data: 90 dias
    DELETE FROM data_lake.agent_telemetry_raw 
    WHERE event_timestamp < NOW() - INTERVAL '90 days';
    
    -- Staging: 30 dias
    DELETE FROM data_lake.agent_telemetry_staging 
    WHERE event_timestamp < NOW() - INTERVAL '30 days';
    
    -- Log da execução
    INSERT INTO data_lake.agent_telemetry_raw (
        agent_id, agent_version, agent_role, squad_id,
        execution_id, event_timestamp, event_type, payload
    ) VALUES (
        'retention_policy', '1.0', 'data_engineer', 'system',
        gen_random_uuid(), NOW(), 'data_pipeline',
        jsonb_build_object(
            'action', 'retention_cleanup',
            'retention_days', p_retention_days,
            'rows_deleted_raw', ROW_COUNT
        )
    );
END;
$$ LANGUAGE plpgsql;
