## 1. Documento de Arquitetura (ARCHITECTURE.md)

```markdown
# DIYAPP V13 - Arquitetura de Microsserviços

## Visão Geral
Arquitetura baseada em microsserviços para a V13 do DIYAPP, projetada para 100% de autonomia e estabilidade. Utiliza comunicação gRPC para sincronia e Kafka para eventos assíncronos.

## Princípios Arquiteturais
1. **Autonomia Total**: Cada serviço é independente em deploy, escala e falha
2. **Resiliência**: Circuit breakers, retry com backoff exponencial e fallbacks
3. **Observabilidade**: Logs estruturados, métricas e tracing distribuído
4. **Segurança por Design**: Autenticação JWT, autorização baseada em roles, secrets em Vault
5. **Evolução Contínua**: Versionamento de API, compatibilidade retroativa

## Padrões de Comunicação

### 1. Comunicação Síncrona (gRPC)
```protobuf
// protos/user_service.proto
syntax = "proto3";

package diyapp.v1;

service UserService {
  rpc GetUser(GetUserRequest) returns (UserResponse);
  rpc CreateUser(CreateUserRequest) returns (UserResponse);
  rpc UpdateUser(UpdateUserRequest) returns (UserResponse);
}

message GetUserRequest {
  string user_id = 1;
  string auth_token = 2;
}

message UserResponse {
  string id = 1;
  string email = 2;
  string name = 3;
  UserStatus status = 4;
  google.protobuf.Timestamp created_at = 5;
}

enum UserStatus {
  ACTIVE = 0;
  INACTIVE = 1;
  SUSPENDED = 2;
}
```

## Padrões de Resiliência

### Circuit Breaker Pattern
```yaml
# config/resilience.yml
circuit_breaker:
  user_service:
    failure_threshold: 5
    timeout: 3000
    reset_timeout: 30000
  task_service:
    failure_threshold: 3
    timeout: 5000
    reset_timeout: 60000

retry_policy:
  max_attempts: 3
  backoff:
    initial: 100
    multiplier: 2
    max: 3000
```

### Fallback Strategies
1. **Cache Fallback**: Retorna dados em cache quando serviço indisponível
2. **Default Response**: Retorna resposta padrão pré-definida
3. **Queue and Retry**: Coloca requisição em fila para processamento posterior

## ADRs (Architecture Decision Records)

### ADR-001: Escolha de gRPC sobre REST para comunicação interna
**Status**: Aceito  
**Contexto**: Necessidade de comunicação eficiente entre serviços  
**Decisão**: Usar gRPC com Protobuf para comunicação síncrona  
**Consequências**: +Performance, +Type safety, -Complexidade inicial

### ADR-002: Kafka para eventos assíncronos
**Status**: Aceito  
**Contexto**: Necessidade de desacoplamento e processamento em background  
**Decisão**: Usar Apache Kafka como message broker  
**Consequências**: +Escalabilidade, +Resiliência, -Operacional overhead

### ADR-003: Banco de dados por serviço
**Status**: Aceito  
**Contexto**: Garantir autonomia total dos serviços  
**Decisão**: Cada serviço gerencia seu próprio banco de dados  
**Consequências**: +Autonomia, +Escalabilidade independente, -Consistência eventual
```

// WCAG Contrast Checker Utility
export const checkContrast = (foreground, background) => {
  // Implementation of WCAG 2.1 contrast ratio calculation
  const getLuminance = (color) => {
    const rgb = color.match(/\w\w/g).map(c => parseInt(c, 16) / 255);
    const [r, g, b] = rgb.map(c => 
      c <= 0.03928 ? c / 12.92 : Math.pow((c + 0.055) / 1.055, 2.4)
    );
    return 0.2126 * r + 0.7152 * g + 0.0722 * b;
  };
  
  const lum1 = getLuminance(foreground);
  const lum2 = getLuminance(background);
  const brightest = Math.max(lum1, lum2);
  const darkest = Math.min(lum1, lum2);
  return (brightest + 0.05) / (darkest + 0.05);
};

export class ThemeManager {
  constructor() {
    this.currentTheme = 'light';
    this.themes = { light: lightTheme, dark: darkTheme };
    this.subscribers = new Set();
  }
  
  getTheme() {
    return this.themes[this.currentTheme];
  }
  
  toggleTheme() {
    this.currentTheme = this.currentTheme === 'light' ? 'dark' : 'light';
    this.notifySubscribers();
    this.saveToStorage();
    return this.currentTheme;
  }
  
  setTheme(theme) {
    if (this.themes[theme]) {
      this.currentTheme = theme;
      this.notifySubscribers();
      this.saveToStorage();
    }
  }
  
  subscribe(callback) {
    this.subscribers.add(callback);
    return () => this.subscribers.delete(callback);
  }
  
  notifySubscribers() {
    this.subscribers.forEach(callback => callback(this.getTheme()));
  }
  
  saveToStorage() {
    if (typeof window !== 'undefined') {
      localStorage.setItem('diyapp-theme', this.currentTheme);
    }
  }
  
  loadFromStorage() {
    if (typeof window !== 'undefined') {
      const saved = localStorage.getItem('diyapp-theme');
      if (saved && this.themes[saved]) {
        this.currentTheme = saved;
      }
    }
  }
}

export const themeManager = new ThemeManager();

dotenv.config();

const configSchema = z.object({
  NODE_ENV: z.enum(['development', 'production', 'test']).default('development'),
  PORT: z.string().transform(Number).default('3000'),
  GRAPHQL_PORT: z.string().transform(Number).default('3001'),
  
  // Database
  DB_HOST: z.string().default('localhost'),
  DB_PORT: z.string().transform(Number).default('5432'),
  DB_USER: z.string(),
  DB_PASSWORD: z.string(),
  DB_NAME: z.string(),
  
  // Redis
  REDIS_HOST: z.string().default('localhost'),
  REDIS_PORT: z.string().transform(Number).default('6379'),
  REDIS_PASSWORD: z.string().optional(),
  
  // JWT
  JWT_SECRET: z.string().min(32),
  JWT_EXPIRES_IN: z.string().default('24h'),
  
  // Rate limiting
  RATE_LIMIT_WINDOW_MS: z.string().transform(Number).default('900000'), // 15 minutes
  RATE_LIMIT_MAX_REQUESTS: z.string().transform(Number).default('100'),
  
  // Circuit breaker
  CIRCUIT_BREAKER_FAILURE_THRESHOLD: z.string().transform(Number).default('5'),
  CIRCUIT_BREAKER_RESET_TIMEOUT: z.string().transform(Number).default('30000'), // 30 seconds
  
  // External services timeouts (ms)
  EXTERNAL_API_TIMEOUT: z.string().transform(Number).default('5000'),
  LLM_API_TIMEOUT: z.string().transform(Number).default('30000'),
  
  // Logging
  LOG_LEVEL: z.enum(['error', 'warn', 'info', 'debug']).default('info'),
  LOG_RETENTION_DAYS: z.string().transform(Number).default('30'),
});

export type Config = z.infer<typeof configSchema>;

class ConfigManager {
  private static instance: ConfigManager;
  private config: Config;

  private constructor() {
    this.config = this.validateConfig();
  }

  static getInstance(): ConfigManager {
    if (!ConfigManager.instance) {
      ConfigManager.instance = new ConfigManager();
    }
    return ConfigManager.instance;
  }

  private validateConfig(): Config {
    try {
      return configSchema.parse(process.env);
    } catch (error) {
      if (error instanceof z.ZodError) {
        const missingVars = error.errors.map(err => err.path.join('.'));
        throw new Error(
          `Missing or invalid environment variables: ${missingVars.join(', ')}`
        );
      }
      throw error;
    }
  }

  get<K extends keyof Config>(key: K): Config[K] {
    return this.config[key];
  }

  getAll(): Config {
    return { ...this.config };
  }

  isProduction(): boolean {
    return this.config.NODE_ENV === 'production';
  }

  isDevelopment(): boolean {
    return this.config.NODE_ENV === 'development';
  }

  isTest(): boolean {
    return this.config.NODE_ENV === 'test';
  }
}

export const config = ConfigManager.getInstance();

import Redis from 'ioredis';
import { config } from './index';

class RedisManager {
  private static instance: RedisManager;
  private client: Redis;
  private subscriber: Redis;

  private constructor() {
    const redisConfig = {
      host: config.get('REDIS_HOST'),
      port: config.get('REDIS_PORT'),
      password: config.get('REDIS_PASSWORD'),
      retryStrategy: (times: number) => {
        const delay = Math.min(times * 50, 2000);
        return delay;
      },
      maxRetriesPerRequest: 3,
      enableReadyCheck: true,
      connectTimeout: 10000,
    };

    this.client = new Redis(redisConfig);
    this.subscriber = new Redis(redisConfig);

    this.setupEventListeners();
  }

  static getInstance(): RedisManager {
    if (!RedisManager.instance) {
      RedisManager.instance = new RedisManager();
    }
    return RedisManager.instance;
  }

  private setupEventListeners(): void {
    this.client.on('connect', () => {
      console.log('Redis client connected');
    });

    this.client.on('error', (error) => {
      console.error('Redis client error:', error);
    });

    this.client.on('close', () => {
      console.warn('Redis client connection closed');
    });

    this.client.on('reconnecting', () => {
      console.log('Redis client reconnecting...');
    });
  }

  getClient(): Redis {
    return this.client;
  }

  getSubscriber(): Redis {
    return this.subscriber;
  }

  async ping(): Promise<string> {
    try {
      return await this.client.ping();
    } catch (error) {
      console.error('Redis ping failed:', error);
      throw error;
    }
  }

  async setWithExpiry(
    key: string,
    value: string,
    ttlSeconds: number = 3600
  ): Promise<void> {
    try {
      await this.client.setex(key, ttlSeconds, value);
    } catch (error) {
      console.error('Redis setWithExpiry failed:', error);
      throw error;
    }
  }

  async get(key: string): Promise<string | null> {
    try {
      return await this.client.get(key);
    } catch (error) {
      console.error('Redis get failed:', error);
      throw error;
    }
  }

  async del(key: string): Promise<void> {
    try {
      await this.client.del(key);
    } catch (error) {
      console.error('Redis del failed:', error);
      throw error;
    }
  }

  async disconnect(): Promise<void> {
    try {
      await this.client.quit();
      await this.subscriber.quit();
      console.log('Redis connections closed');
    } catch (error) {
      console.error('Error closing Redis connections:', error);

    }
  }
}

    INDEX idx_feature_usage_feature (feature_id)

-- Função para atualizar updated_at
CREATE OR REPLACE FUNCTION update_updated_at_column()
RETURNS TRIGGER AS $$
BEGIN
    NEW.updated_at = CURRENT_TIMESTAMP;
    RETURN NEW;
END;
$$ language 'plpgsql';

-- Triggers para atualização automática
CREATE TRIGGER update_users_updated_at BEFORE UPDATE ON users
    FOR EACH ROW EXECUTE FUNCTION update_updated_at_column();

-- View para métricas diárias
CREATE VIEW daily_metrics AS

    DATE(created_at) as metric_date,
    COUNT(DISTINCT user_id) as daily_active_users,
    COUNT(*) as total_events,
    COUNT(DISTINCT session_id) as total_sessions,

            else:
                features.append(float(value))
        
        return np.array(features).reshape(1, -1)
    
    def _prepare_training_data(self) -> np.ndarray:
        """Prepara dados para treinamento"""
        if not self.metrics_history:
            return np.array([])
        
        df = pd.DataFrame(self.metrics_history)
        feature_columns = self.config.get('feature_columns', [])
        
        # Filtra colunas existentes
        existing_columns = [col for col in feature_columns if col in df.columns]
        
        if not existing_columns:
            return np.array([])
        
        X = df[existing_columns].values
        
        # Remove NaN
        X = X[~np.isnan(X).any(axis=1)]
        
        return X
    
    def _train_models(self, X: np.ndarray) -> None:
        """Treina os modelos de ML"""
        if X.shape[0] < 10:
            return
        
        # Normaliza dados
        X_scaled = self.scaler.fit_transform(X)
        
        # Treina Isolation Forest
        self.isolation_forest.fit(X_scaled)
        
        # Treina One-Class SVM
        self.one_class_svm.fit(X_scaled)
        
        # Para Random Forest, precisamos de labels
        # Usamos Isolation Forest para gerar labels iniciais
        labels = self.isolation_forest.predict(X_scaled)
        labels = np.where(labels == -1, 1, 0)  # Converte para 0/1
        
        if np.sum(labels) > 0 and np.sum(labels) < len(labels):
            self.random_forest.fit(X_scaled, labels)
    
    def _ensemble_detection(self, features: np.ndarray, X: np.ndarray) -> Dict:
        """Detecção por ensemble de modelos"""
        if X.shape[0] < 10:
            return {'is_anomaly': False, 'votes': {}}

        votes['isolation_forest'] = 1 if if_anomaly == -1 else 0
        
        # One-Class SVM
        svm_anomaly = self.one_class_svm.predict(features_scaled)[0]
        votes['one_class_svm'] = 1 if svm_anomaly == -1 else 0
        
        # Random Forest (se treinado)
        if hasattr(self.random_forest, 'estimators_'):
            rf_pred = self.random_forest.predict(features_scaled)[0]
            votes['random_forest'] = int(rf_pred)
        else:
            votes['random_forest'] = 0
        
        # Votação por maioria
        total_votes = sum(votes.values())
        is_anomaly = total_votes >= 2  # Pelo menos 2 de 3
        
        return {
            'is_anomaly': is_anomaly,

        max_votes = len(votes)
