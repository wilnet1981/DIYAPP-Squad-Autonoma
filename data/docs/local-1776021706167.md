# DIYAPP Evolution - V12 Core - Arquitetura Técnica

## 1. ADR-001: Stack Tecnológica V12

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista LLM

**CONTEXTO:**
A versão V12 precisa suportar:
- 100% de autonomia operacional
- Escalabilidade para 10x o volume atual
- Integração nativa com múltiplos modelos LLM
- Observabilidade completa sem intervenção humana
- Deploy contínuo com zero downtime

**DECISÃO:**
Stack principal:
- **Frontend:** Next.js 15 (App Router) + React 18 + TypeScript 5.5
- **Backend:** Node.js 22 (LTS) + Express 5 + TypeScript
- **Banco de Dados:** PostgreSQL 16 + Redis 7 (cache/queue)
- **LLM Orchestration:** LangChain.js + LiteLLM
- **Infra como Código:** Terraform + Docker + Kubernetes
- **CI/CD:** GitHub Actions + ArgoCD
- **Observabilidade:** OpenTelemetry + Prometheus + Grafana Loki

**OPÇÕES CONSIDERADAS:**
- Opção A: Next.js 15 + Node.js 22 + PostgreSQL
  - Prós: Full-stack TypeScript, SSR nativo, comunidade ativa
  - Contras: Curva de aprendizado para App Router
- Opção B: NestJS + React 18 + MongoDB
  - Prós: Arquitetura bem definida, rápido desenvolvimento
  - Contras: Menos flexível para IA, MongoDB menos adequado para transações
- Opção C: Remix + Cloudflare Workers + D1
  - Prós: Performance edge, custo otimizado
  - Contras: Ecossistema menos maduro para IA

**Escolhida:** Opção A - Justificativa: Melhor equilíbrio entre produtividade, performance e ecossistema de IA.

**CONSEQUÊNCIAS:**
- Positivas: TypeScript end-to-end, SSR otimizado, deploy simplificado
- Negativas: Bundle size maior que SPA puro, necessidade de expertise em App Router
- Riscos: Breaking changes em Next.js 15, mitigado por testes E2E rigorosos

**REVISÃO:** 2024-04-15

```
## Convenções de Código

### TypeScript
- Strict mode obrigatório
- Interfaces para objetos públicos, types para internos
- Nunca use `any` - use `unknown` com type guards

### React/Next.js
- Server Components por padrão, Client Components apenas quando necessário
- Use `async` Server Components para data fetching
- Implemente loading.tsx e error.tsx para todas as rotas

### Nomenclatura
- Components: PascalCase (ButtonPrimary)
- Files: kebab-case (user-service.ts)
- Variables: camelCase (userProfile)
- Constants: UPPER_SNAKE_CASE (API_TIMEOUT)

- Conventional Commits obrigatório
- Format: `type(scope): description`
- Types: feat, fix, docs, style, refactor, test, chore
- Example: `feat(auth): add OAuth2 integration`

- Cobertura mínima: 80%
- E2E: Playwright
- Testes de integração para todas as APIs

- Scan de dependências diário (Dependabot)
```

```javascript
const Redis = require('ioredis');
const config = require('../config/env');
const logger = require('../utils/logger');

class DatabaseClient {
  constructor() {
    this.pgPool = null;
    this.redisClient = null;
    this.isConnected = false;
  }

  async connect() {
    try {
      // PostgreSQL connection with connection pool
      this.pgPool = new Pool({
        connectionString: config.DATABASE_URL,
        max: 20, // Maximum number of clients in the pool
        idleTimeoutMillis: 30000,
        connectionTimeoutMillis: 2000,
      });

      // Test PostgreSQL connection
      const pgClient = await this.pgPool.connect();
      await pgClient.query('SELECT NOW()');
      pgClient.release();
      
      logger.info('PostgreSQL connected successfully');

      // Redis connection with circuit breaker pattern
      this.redisClient = new Redis({
        host: config.REDIS_HOST,
        port: config.REDIS_PORT,
        password: config.REDIS_PASSWORD,
        retryStrategy: (times) => {
          const delay = Math.min(times * 50, 2000);
          return delay;
        },
        maxRetriesPerRequest: 3,
        enableReadyCheck: true,
        connectTimeout: 5000,
      });

      // Test Redis connection
      await this.redisClient.ping();
      logger.info('Redis connected successfully');

      this.isConnected = true;
      
      // Handle connection errors
      this.redisClient.on('error', (error) => {
        logger.error('Redis connection error:', error);
        this.isConnected = false;
      });

      this.pgPool.on('error', (error) => {
        logger.error('PostgreSQL pool error:', error);
        this.isConnected = false;
      });

    } catch (error) {
      logger.error('Database connection failed:', error);
      throw error;
    }
  }

  async disconnect() {
    try {
      if (this.pgPool) {
        await this.pgPool.end();
        logger.info('PostgreSQL disconnected');
      }
      
      if (this.redisClient) {
        await this.redisClient.quit();
        logger.info('Redis disconnected');
      }
    } catch (error) {
      logger.error('Error disconnecting databases:', error);
    }
  }

  getPgPool() {
    if (!this.isConnected) {
      throw new Error('Database not connected');
    }
    return this.pgPool;
  }

  getRedisClient() {
    if (!this.isConnected) {
      throw new Error('Redis not connected');
    }
    return this.redisClient;
  }

  async healthCheck() {
    try {
      const pgCheck = this.pgPool ? await this.pgPool.query('SELECT NOW()') : null;
      const redisCheck = this.redisClient ? await this.redisClient.ping() : null;
      
      return {
        postgresql: pgCheck ? 'healthy' : 'unhealthy',
        redis: redisCheck === 'PONG' ? 'healthy' : 'unhealthy',
        timestamp: new Date().toISOString()
      };
    } catch (error) {
      return {
        postgresql: 'unhealthy',
        redis: 'unhealthy',
        error: error.message,
        timestamp: new Date().toISOString()
      };
    }
  }
}
```

```javascript
const path = require('path');
const fs = require('fs').promises;
const { Pool } = require('pg');
const config = require('../../config/env');
const logger = require('../../utils/logger');

class MigrationRunner {
  constructor() {
    this.pool = new Pool({
      connectionString: config.DATABASE_URL,
    });
    this.migrationsTable = 'schema_migrations';
  }

  async init() {
    await this.pool.query(`
      CREATE TABLE IF NOT EXISTS ${this.migrationsTable} (
        version VARCHAR(255) PRIMARY KEY,
        applied_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
      )
    `);
  }

  async getAppliedMigrations() {
    const result = await this.pool.query(
      `SELECT version FROM ${this.migrationsTable} ORDER BY applied_at`
    );
    return result.rows.map(row => row.version);
  }

  async getMigrationFiles() {
    const migrationsDir = path.join(__dirname);
    const files = await fs.readdir(migrationsDir);
    return files
      .filter(file => file.endsWith('.js') && file !== 'run.js')
      .sort();
  }

  async run(direction = 'up') {
    await this.init();
    const applied = await this.getAppliedMigrations();
    const files = await this.getMigrationFiles();

    const client = await this.pool.connect();
    
    try {
      await client.query('BEGIN');

      for (const file of files) {
        const version = file.replace('.js', '');
        
        if (direction === 'up' && !applied.includes(version)) {
          logger.info(`Applying migration: ${version}`);
          const migration = require(path.join(__dirname, file));
          await migration.up(client);
          
          await client.query(
            `INSERT INTO ${this.migrationsTable} (version) VALUES ($1)`,
            [version]
          );
          
          logger.info(`Migration ${version} applied successfully`);
        }
        
        if (direction === 'down' && applied.includes(version)) {
          logger.info(`Reverting migration: ${version}`);
          const migration = require(path.join(__dirname, file));
          await migration.down(client);
          
          await client.query(
            `DELETE FROM ${this.migrationsTable} WHERE version = $1`,
            [version]
          );
          
          logger.info(`Migration ${version} reverted successfully`);
        }
      }

      await client.query('COMMIT');
      logger.info(`All migrations ${direction} completed successfully`);
    } catch (error) {
      await client.query('ROLLBACK');
      logger.error(`Migration failed: ${error.message}`);
      throw error;
    } finally {
      client.release();
      await this.pool.end();
    }
  }
}

const runner = new MigrationRunner();
const direction = process.argv[2] || 'up';

runner.run(direction).catch(error => {
  console.error('Migration runner failed:', error);
  process.exit(1);
});
```

```javascript
      paramCount++;

      SET ${fields.join(', ')}
```

```sql
DECLARE

BEGIN

    anonymized := anonymized - 'email';
    anonymized := anonymized - 'full_name';
    anonymized := anonymized - 'ip_address';

    END IF;

END;
```

# DIYAPP Evolution - V12 Core - Threat Modeling e Implementação de Segurança

## 1. Threat Modeling STRIDE - V12 Core

### Documento de Threat Modeling
**Feature:** DIYAPP V12 Core - Sistema Autônomo de IA  
**Data:** $(new Date().toISOString().split('T')[0])  
**Responsável:** Especialista em Segurança  
**Status:** Análise Concluída  

### Ativos Críticos Identificados:
1. **Dados de Usuário:** Perfis, histórico de interações, preferências
2. **Modelos de IA:** Prompts do sistema, configurações de LLM
3. **Credenciais:** Tokens de API, chaves de criptografia
4. **Infraestrutura:** Servidores, bancos de dados, filas de mensagens
5. **Código Fonte:** Repositório DIYAPP, agentes especializados

### Análise STRIDE:

#### S - Spoofing (Falsificação)
- **Ameaça:** Ataque de força bruta em endpoints de autenticação
- **Ameaça:** Token JWT roubado/reutilizado
- **Ameaça:** LLM se passando por usuário legítimo via prompt injection
- **Controles Implementados:**
  - MFA obrigatório para contas administrativas
  - Rate limiting por IP/usuario em login
  - Tokens JWT com expiração curta (15min)
  - Validação de origem em todas as requisições

#### T - Tampering (Manipulação)
- **Ameaça:** Modificação de prompts do sistema via input do usuário
- **Ameaça:** Manipulação de dados em trânsito (MITM)
- **Ameaça:** Alteração de configurações de segurança
- **Controles Implementados:**
  - TLS 1.3 obrigatório
  - Assinatura digital em prompts do sistema
  - Validação de integridade em dados críticos
  - WAF com regras específicas para LLM

#### R - Repudiation (Repúdio)
- **Ameaça:** Negação de ações administrativas críticas
- **Ameaça:** Falta de rastreabilidade em operações de LLM
- **Controles Implementados:**
  - Logs imutáveis de todas as operações privilegiadas
  - Auditoria completa de prompts e respostas de LLM
  - Assinatura digital em logs críticos
  - Armazenamento em sistema WORM (Write Once Read Many)

#### I - Information Disclosure (Divulgação de Informação)
- **Ameaça:** Vazamento de prompts do sistema via engenharia de prompts
- **Ameaça:** Exposição de dados sensíveis em respostas de LLM
- **Ameaça:** Logs contendo informações sensíveis
- **Controles Implementados:**
  - Sanitização de outputs de LLM
  - Mascaramento de PII em logs
  - Criptografia AES-256 em repouso para dados sensíveis
  - Controle de acesso baseado em atributos (ABAC)

#### D - Denial of Service (Negação de Serviço)
- **Ameaça:** Ataque DDoS em endpoints de LLM
- **Ameaça:** Exaustão de recursos via prompts complexos
- **Ameaça:** Rate limiting insuficiente
- **Controles Implementados:**
  - WAF com proteção DDoS
  - Rate limiting adaptativo por usuário/endpoint
  - Circuit breakers em chamadas de LLM
  - Quotas de uso por tenant

#### E - Elevation of Privilege (Elevação de Privilégio)
- **Ameaça:** Prompt injection para execução de comandos privilegiados
- **Ameaça:** Exploração de vulnerabilidades em dependências
- **Ameaça:** Configuração incorreta de IAM
- **Controles Implementados:**
  - Execução sandboxed de LLM
  - Princípio do menor privilégio em todas as permissões
  - Revisão regular de dependências (SAST/SCA)
  - Segregação de ambientes

### Riscos Aceitos Conscientemente:
1. **Risco Baixo:** Possibilidade de false positives no WAF (aceitável para segurança)
2. **Risco Médio:** Performance impactada por sanitização em tempo real (otimização contínua)
3. **Risco Alto:** Nenhum - todas as ameaças altas têm controles implementados

### 2.1 Configuração do WAF (Web Application Firewall)

**src/security/waf/index.js:**
```javascript
const rules = require('./rules');
const llmProtection = require('./llm-protection');
const crypto = require('crypto');

class WAF {
    constructor(config) {
        this.config = config;
        this.blockedIPs = new Map();
        this.requestHistory = new Map();
        this.attackPatterns = rules.getAttackPatterns();
        this.llmRules = llmProtection.getLLMRules();
    }

    async inspectRequest(req) {
        const ip = req.ip || req.connection.remoteAddress;
        const timestamp = Date.now();
        
        // Verifica se IP está bloqueado
        if (this.isIPBlocked(ip)) {
            this.logAttack('IP_BLOCKED', ip, req);
            return { blocked: true, reason: 'IP bloqueado' };
        }

        // Aplica regras do WAF
        const wafResult = this.applyWAFRules(req);
        if (wafResult.blocked) {
            this.blockIP(ip, 3600000); // Bloqueia por 1 hora
            return wafResult;
        }

        // Regras específicas para LLM
        if (this.isLLMRequest(req)) {
            const llmResult = this.applyLLMRules(req);
            if (llmResult.blocked) {
                this.blockIP(ip, 7200000); // Bloqueia por 2 horas para ataques LLM
                return llmResult;
            }
        }

        // Rate limiting adaptativo
        const rateLimitResult = this.checkRateLimit(ip, req);
        if (rateLimitResult.blocked) {
            return rateLimitResult;
        }

        return { blocked: false };
    }

    applyWAFRules(req) {
        const { path, method, headers, body, query } = req;
        
        // SQL Injection
        if (this.detectSQLInjection(body, query)) {
            return { blocked: true, reason: 'Padrão de SQL Injection detectado' };
        }

        // XSS
        if (this.detectXSS(body, query, headers)) {
            return { blocked: true, reason: 'Padrão de XSS detectado' };
        }

        // Path Traversal
        if (this.detectPathTraversal(path)) {
            return { blocked: true, reason: 'Path traversal detectado' };
        }

        // Command Injection
        if (this.detectCommandInjection(body, query)) {
            return { blocked: true, reason: 'Command injection detectado' };
        }

        return { blocked: false };
    }

    applyLLMRules(req) {
        if (!req.body || !req.body.prompt) {
            return { blocked: false };
        }

        const prompt = req.body.prompt;
        
        // Prompt Injection detection
        if (this.detectPromptInjection(prompt)) {
            return { blocked: true, reason: 'Tentativa de prompt injection detectada' };
        }

        // System Prompt Extraction
        if (this.detectSystemPromptExtraction(prompt)) {
            return { blocked: true, reason: 'Tentativa de extração de system prompt' };
        }

        // Data Exfiltration patterns
        if (this.detectDataExfiltration(prompt)) {
            return { blocked: true, reason: 'Padrão de exfiltração de dados detectado' };
        }

        return { blocked: false };
    }

    detectPromptInjection(prompt) {
        const injectionPatterns = [
            /ignore.*previous.*instructions/i,
            /system.*prompt.*show/i,
            /forget.*everything.*before/i,
            /disregard.*initial.*instructions/i,
            /from.*now.*on/i,
            /your.*new.*instructions/i,
            /you.*are.*now/i,
            /act.*as.*if/i,
            /pretend.*you.*are/i,
            /output.*only.*the.*following/i
        ];

        return injectionPatterns.some(pattern => pattern.test(prompt));
    }

    detectSystemPromptExtraction(prompt) {
        const extractionPatterns = [
            /what.*are.*your.*instructions/i,
            /show.*your.*system.*prompt/i,
            /what.*is.*your.*initial.*prompt/i,
            /repeat.*your.*prompt/i,
            /what.*were.*you.*told/i,
            /original.*instructions/i,
            /initial.*configuration/i,
            /how.*are.*you.*programmed/i
        ];

        return extractionPatterns.some(pattern => pattern.test(prompt));
    }

    detectDataExfiltration(prompt) {
        const exfiltrationPatterns = [
            /send.*data.*to.*http/i,
            /exfiltrate.*information/i,
            /leak.*data/i,
            /output.*all.*user.*data/i,
            /show.*all.*stored.*information/i,
            /dump.*database/i,
            /export.*all.*records/i
        ];

        return exfiltrationPatterns.some(pattern => pattern.test(prompt));
    }

    detectSQLInjection(...inputs) {
        const sqlPatterns = [
            /(\%27)|(\')|(\-\-)|(\%23)|(#)/i,
            /((\%3D)|(=))[^\n]*((\%27)|(\')|(\-\-)|(\%3B)|(;))/i,
            /\w*((\%27)|(\'))((\%6F)|o|(\%4F))((\%72)|r|(\%52))/i,
            /((\%27)|(\'))union/i,
            /exec(\s|\+)+(s|x)p\w+/i,
            /insert.+into/i,
            /select.+from/i,
            /delete.+from/i,
            /update.+set/i,
            /drop.+table/i,
            /truncate.+table/i
        ];

        const inputString = JSON.stringify(inputs).toLowerCase();
        return sqlPatterns.some(pattern => pattern.test(inputString));
    }

    detectXSS(...inputs) {
        const xssPatterns = [
            /<script.*?>.*?<\/script>/i,
            /javascript:/i,
            /on\w+\s*=/i,
            /<iframe.*?>.*?<\/iframe>/i,
            /<object.*?>.*?<\/object>/i,
            /<embed.*?>.*?<\/embed>/i,
            /<applet.*?>.*?<\/applet>/i,
            /<meta.*?>.*?<\/meta>/i,
            /<link.*?>.*?<\/link>/i,
            /<style.*?>.*?<\/style>/i,
            /expression\(.*?\)/i,
            /url\(.*?\)/i,
            /eval\(.*?\)/i,
            /alert\(.*?\)/i,
            /confirm\(.*?\)/i,
            /prompt\(.*?\)/i,
            /document\.cookie/i,
            /window\.location/i,
            /document\.write/i,
            /innerHTML/i,
            /outerHTML/i,
            /appendChild/i,
            /createElement/i
        ];

        const inputString = JSON.stringify(inputs);
        return xssPatterns.some(pattern => pattern.test(inputString));
    }

    checkRateLimit(ip, req) {
        const now = Date.now();
        const windowMs = 60000; // 1 minuto
        const maxRequests = this.getMaxRequestsForEndpoint(req.path);
        
        if (!this.requestHistory.has(ip)) {
            this.requestHistory.set(ip, []);
        }
        
        const requests = this.requestHistory.get(ip);
        const windowStart = now - windowMs;
        
        // Remove requisições antigas
        while (requests.length > 0 && requests[0] < windowStart) {
            requests.shift();
        }
        
        // Verifica limite
        if (requests.length >= maxRequests) {
            return { blocked: true, reason: 'Rate limit excedido' };
        }
```

```python
import numpy as np
import pandas as pd
from sklearn.ensemble import IsolationForest
from sklearn.preprocessing import StandardScaler
from typing import Tuple, List, Dict, Any
import joblib
import logging
from datetime import datetime
from pathlib import Path

logger = logging.getLogger(__name__)

class IsolationForestAnomalyDetector:
    """Detector de anomalias baseado em Isolation Forest"""
    
    def __init__(self, contamination: float = 0.1, n_estimators: int = 100):
        self.contamination = contamination
        self.n_estimators = n_estimators
        self.model = IsolationForest(
            contamination=contamination,
            n_estimators=n_estimators,
            random_state=42,
            n_jobs=-1
        )
        self.scaler = StandardScaler()
        self.is_trained = False
        self.feature_names = None
        
    def extract_features(self, metrics_data: pd.DataFrame) -> pd.DataFrame:
        """Extrai features para detecção de anomalias"""
        features = pd.DataFrame()
        
        # Features básicas de métricas
        if 'cpu_percent' in metrics_data.columns:
            features['cpu_mean'] = metrics_data['cpu_percent'].rolling(5).mean()
            features['cpu_std'] = metrics_data['cpu_percent'].rolling(5).std()
            features['cpu_trend'] = metrics_data['cpu_percent'].diff().rolling(10).mean()
            
        if 'memory_percent' in metrics_data.columns:
            features['memory_mean'] = metrics_data['memory_percent'].rolling(5).mean()
            features['memory_std'] = metrics_data['memory_percent'].rolling(5).std()
            features['memory_trend'] = metrics_data['memory_percent'].diff().rolling(10).mean()
            
        if 'response_time_ms' in metrics_data.columns:
            features['response_time_mean'] = metrics_data['response_time_ms'].rolling(5).mean()
            features['response_time_std'] = metrics_data['response_time_ms'].rolling(5).std()
            features['response_time_trend'] = metrics_data['response_time_ms'].diff().rolling(10).mean()
            
        if 'request_rate' in metrics_data.columns:
            features['request_rate_mean'] = metrics_data['request_rate'].rolling(5).mean()
            features['request_rate_std'] = metrics_data['request_rate'].rolling(5).std()
            features['request_rate_trend'] = metrics_data['request_rate'].diff().rolling(10).mean()
            
        # Features derivadas de logs
        if 'error_count' in metrics_data.columns:
            features['error_rate'] = metrics_data['error_count'] / (metrics_data['request_rate'] + 1)
            features['error_trend'] = metrics_data['error_count'].diff().rolling(10).mean()
            
        # Features temporais
        if 'timestamp' in metrics_data.columns:
            metrics_data['timestamp'] = pd.to_datetime(metrics_data['timestamp'])
            features['hour_of_day'] = metrics_data['timestamp'].dt.hour
            features['day_of_week'] = metrics_data['timestamp'].dt.dayofweek
            features['is_weekend'] = features['day_of_week'].isin([5, 6]).astype(int)
            
        # Remover NaN
        features = features.fillna(method='ffill').fillna(0)
        
        self.feature_names = features.columns.tolist()
        return features
    
    def train(self, training_data: pd.DataFrame) -> Dict[str, Any]:
        """Treina o modelo Isolation Forest"""
        try:
            logger.info("Iniciando treinamento do Isolation Forest")
            
            # Extrair features
            features = self.extract_features(training_data)
            
            # Normalizar dados
            scaled_features = self.scaler.fit_transform(features)
            
            # Treinar modelo
            self.model.fit(scaled_features)
            self.is_trained = True
            
            # Avaliar treinamento
            predictions = self.model.predict(scaled_features)
            anomaly_indices = np.where(predictions == -1)[0]
            anomaly_rate = len(anomaly_indices) / len(predictions)
            
            logger.info(f"Treinamento concluído. Taxa de anomalias detectadas: {anomaly_rate:.2%}")
            
            return {
                'status': 'success',
                'anomaly_rate': anomaly_rate,
                'n_samples': len(features),
                'n_features': features.shape[1],
                'feature_names': self.feature_names
            }
            
        except Exception as e:
            logger.error(f"Erro no treinamento: {str(e)}")
            return {
                'status': 'error',
                'error': str(e)
            }
    
    def predict(self, metrics_data: pd.DataFrame) -> Tuple[np.ndarray, np.ndarray]:
        """Faz previsões de anomalias"""
        if not self.is_trained:
            raise ValueError("Modelo não treinado. Execute train() primeiro.")
        
        # Extrair features
        features = self.extract_features(metrics_data)
        
        # Garantir mesma ordem de features
        features = features[self.feature_names]
        
        # Normalizar
        scaled_features = self.scaler.transform(features)
        
        # Prever
        predictions = self.model.predict(scaled_features)
        anomaly_scores = self.model.decision_function(scaled_features)
        
        # Converter scores para probabilidades (0 a 1)
        probabilities = 1 / (1 + np.exp(-anomaly_scores))
        
        return predictions, probabilities
    
    def save(self, filepath: str):
        """Salva o modelo treinado"""
        if not self.is_trained:
            raise ValueError("Modelo não treinado")
        
        model_data = {
            'model': self.model,
            'scaler': self.scaler,
            'feature_names': self.feature_names,
            'contamination': self.contamination,
            'n_estimators': self.n_estimators,
            'trained_at': datetime.now().isoformat()
        }
        
        joblib.dump(model_data, filepath)
        logger.info(f"Modelo salvo em {filepath}")
    
    def load(self, filepath: str):
        """Carrega um modelo treinado"""
        model_data = joblib.load(filepath)
        
        self.model = model_data['model']
        self.scaler = model_data['scaler']
        self.feature_names = model_data['feature_names']
        self.contamination = model_data['contamination']
        self.n_estimators = model_data['n_estimators']
        self.is_trained = True
        
        logger.info(f"Modelo carregado de {filepath}")
        
    def explain_anomaly(self, features: pd.Series, score: float) -> Dict[str, Any]:
        """Explica uma anomalia detectada"""
        if not self.is_trained:
            return {}
        
        # Identificar features mais anômalas
        feature_contributions = {}
        
        # Para Isolation Forest, podemos usar a profundidade média
        # como proxy para a contribuição de cada feature
        scaled_features = self.scaler.transform([features])[0]
        
        # Calcular contribuição aproximada
        for i, feature_name in enumerate(self.feature_names):
            feature_value = scaled_features[i]
            feature_mean = self.scaler.mean_[i]
            feature_std = self.scaler.scale_[i]
            
            # Distância da média em termos de desvios padrão
            z_score = abs(feature_value - feature_mean) / feature_std
            feature_contributions[feature_name] = {
                'value': features[feature_name],
                'z_score': z_score,
                'is_extreme': z_score > 3
            }
        
        # Ordenar por z-score
        sorted_contributions = sorted(
            feature_contributions.items(),
            key=lambda x: x[1]['z_score'],
            reverse=True
        )
        
        return {
            'anomaly_score': score,
            'is_anomaly': score > 0.7,
            'top_contributors': sorted_contributions[:5],
            'timestamp': datetime.now().isoformat()
        }
```

        self.encoder = nn.LSTM(

            batch_first=True,

        )

        self.decoder = nn.LSTM(

        # Decoder
        decoded, _ = self.decoder(hidden_repeated)
        
        # Output layer
        output = self.output_layer(decoded)
