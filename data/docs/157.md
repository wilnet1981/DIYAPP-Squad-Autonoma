# DIYAPP Evolution - V12 Core - Arquitetura e Repositório Base

## Arquitetura V12 - ADR-001: Decisões Técnicas Fundamentais

**ADR-001: Stack Tecnológica e Padrões Arquiteturais V12**
**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead V12

### CONTEXTO:
O DIYAPP evolui para a versão V12 com foco em:
1. 100% de estabilidade autônoma
2. Refatoração paralela via Modo Hive
3. Manutenibilidade a longo prazo
4. Observabilidade completa
5. CI/CD totalmente automatizado

### DECISÃO:
Adotar arquitetura modular baseada em Node.js com TypeScript, PostgreSQL, Redis e Docker. Implementar Clean Architecture com separação clara de responsabilidades.

### OPÇÕES CONSIDERADAS:
- **Opção A:** Monolito Node.js + Express + PostgreSQL
  - Prós: Simplicidade inicial, deploy único
  - Contras: Acoplamento alto, difícil refatoração paralela
- **Opção B:** Microserviços completos
  - Prós: Isolamento total, escalabilidade independente
  - Contras: Complexidade operacional, overhead de comunicação
- **Opção C:** Arquitetura Modular (escolhida)
  - Prós: Isolamento lógico, refatoração paralela possível, operação simplificada
  - Contras: Requer disciplina de equipe

### CONSEQUÊNCIAS:
**Positivas:**
- Refatoração paralela via Modo Hive viável
- Testabilidade individual por módulo
- Evolução independente de componentes

**Negativas:**
- Curva de aprendizado para padrões Clean Architecture
- Overhead inicial de configuração

**Riscos:**
- Degradação para monólito acoplado sem disciplina
- Mitigação: Code reviews rigorosos e ADRs

### 2. Dockerfile
```dockerfile
# Build stage
FROM node:18-alpine AS builder

WORKDIR /app

# Copy package files
COPY package*.json ./
COPY nest-cli.json ./
COPY tsconfig*.json ./

# Install dependencies
RUN npm ci --only=production

# Copy source code
COPY src ./src
COPY public ./public
COPY tests ./tests

# Build application
RUN npm run build

# Production stage
FROM node:18-alpine

WORKDIR /app

# Install dumb-init for proper signal handling
RUN apk add --no-cache dumb-init

# Create non-root user
RUN addgroup -g 1001 -S nodejs && \
    adduser -S nodejs -u 1001

# Copy built application
COPY --from=builder --chown=nodejs:nodejs /app/dist ./dist
COPY --from=builder --chown=nodejs:nodejs /app/node_modules ./node_modules
COPY --from=builder --chown=nodejs:nodejs /app/package*.json ./
COPY --from=builder --chown=nodejs:nodejs /app/public ./public

# Switch to non-root user
USER nodejs

# Health check
HEALTHCHECK --interval=30s --timeout=3s --start-period=5s --retries=3 \
  CMD node -e "require('http').get('http://localhost:3000/health', (r) => {if(r.statusCode !== 200) throw new Error()})"

# Start application
ENTRYPOINT ["dumb-init", "--"]
CMD ["node", "dist/src/main.js"]
```

```tsx
import React, { InputHTMLAttributes, forwardRef, useState } from 'react';
import { cn } from '../../../utils/helpers';

export interface InputProps extends InputHTMLAttributes<HTMLInputElement> {
  label?: string;
  error?: string;
  helperText?: string;
  leftIcon?: React.ReactNode;
  rightIcon?: React.ReactNode;
  fullWidth?: boolean;
}

const Input = forwardRef<HTMLInputElement, InputProps>(
  ({ className, type, label, error, helperText, leftIcon, rightIcon, fullWidth, id, ...props }, ref) => {
    const [showPassword, setShowPassword] = useState(false);
    const inputId = id || `input-${Math.random().toString(36).substr(2, 9)}`;

    const isPassword = type === 'password';
    const inputType = isPassword && showPassword ? 'text' : type;

    return (
      <div className={cn('space-y-2', fullWidth && 'w-full')}>
        {label && (
          <label
            htmlFor={inputId}
            className="block text-sm font-medium text-neutral-700 dark:text-neutral-300"
          >
            {label}
            {props.required && <span className="text-error ml-1">*</span>}
          </label>
        )}
        
        <div className="relative">
          {leftIcon && (
            <div className="absolute left-3 top-1/2 -translate-y-1/2 text-neutral-400">
              {leftIcon}
            </div>
          )}
          
          <input
            id={inputId}
            type={inputType}
            className={cn(
              'flex h-10 w-full rounded-md border border-neutral-300 bg-white px-3 py-2 text-sm placeholder:text-neutral-400',
              'focus:outline-none focus:ring-2 focus:ring-primary-500 focus:border-transparent',
              'disabled:cursor-not-allowed disabled:opacity-50',
              'dark:border-neutral-700 dark:bg-neutral-900 dark:text-white',
              error && 'border-error focus:ring-error',
              leftIcon && 'pl-10',
              (rightIcon || isPassword) && 'pr-10',
              className
            )}
            ref={ref}
            aria-invalid={!!error}
            aria-describedby={
              error ? `${inputId}-error` : helperText ? `${inputId}-helper` : undefined
            }
            {...props}
          />
          
          {(rightIcon || isPassword) && (
            <div className="absolute right-3 top-1/2 -translate-y-1/2">
              {isPassword ? (
                <button
                  type="button"
                  onClick={() => setShowPassword(!showPassword)}
                  className="text-neutral-400 hover:text-neutral-600 focus:outline-none focus:ring-2 focus:ring-primary-500 rounded"
                  aria-label={showPassword ? 'Hide password' : 'Show password'}
                  tabIndex={-1}
                >
                  {showPassword ? '👁️' : '👁️‍🗨️'}
                </button>
              ) : (
                rightIcon
              )}
            </div>
          )}
        </div>

          <p
            id={`${inputId}-error`}
            className="text-sm text-error"

          >
            {error}
```

```tsx
import { create } from 'zustand';
import { persist } from 'zustand/middleware';

export type Theme = 'light' | 'dark' | 'system';
export type Notification = {
  id: string;
  type: 'info' | 'success' | 'warning' | 'error';
  title: string;
  message: string;
  timestamp: Date;
  read: boolean;
};

interface AppState {
  // Theme
  theme: Theme;
  setTheme: (theme: Theme) => void;
  
  // Notifications
  notifications: Notification[];
  addNotification: (notification: Omit<Notification, 'id' | 'timestamp' | 'read'>) => void;
  markAsRead: (id: string) => void;
  clearNotifications: () => void;
  
  // Loading states
  isLoading: Record<string, boolean>;
  setLoading: (key: string, loading: boolean) => void;
  
  // Modal
  activeModal: string | null;
  modalData: Record<string, any>;
  openModal: (modalId: string, data?: Record<string, any>) => void;
  closeModal: () => void;
  
  // Sidebar
  sidebarOpen: boolean;
  toggleSidebar: () => void;
  
  // API Status
  apiStatus: 'healthy' | 'degraded' | 'down';
  lastApiCheck: Date | null;
  updateApiStatus: (status: 'healthy' | 'degraded' | 'down') => void;
}

export const useAppStore = create<AppState>()(
  persist(
    immer((set) => ({
      // Theme
      theme: 'system',
      setTheme: (theme) => set({ theme }),
      
      // Notifications
      notifications: [],
      addNotification: (notification) =>
        set((state) => {
          state.notifications.unshift({
            ...notification,
            id: Math.random().toString(36).substr(2, 9),
            timestamp: new Date(),
            read: false,
          });
          
          // Keep only last 50 notifications
          if (state.notifications.length > 50) {
            state.notifications = state.notifications.slice(0, 50);
          }
        }),
      
      markAsRead: (id) =>
        set((state) => {
          const notification = state.notifications.find((n) => n.id === id);
          if (notification) {
            notification.read = true;
          }
        }),
      
      clearNotifications: () => set({ notifications: [] }),
      
      // Loading states
      isLoading: {},
      setLoading: (key, loading) =>
        set((state) => {
          state.isLoading[key] = loading;
        }),
      
      // Modal
      activeModal: null,
      modalData: {},
      openModal: (modalId, data = {}) =>
        set({ activeModal: modalId, modalData: data }),
      closeModal: () => set({ activeModal: null, modalData: {} }),
      
      // Sidebar
      sidebarOpen: true,
      toggleSidebar: () => set((state) => ({ sidebarOpen: !state.sidebarOpen })),
      
      // API Status
      apiStatus: 'healthy',
      lastApiCheck: null,
      updateApiStatus: (status) =>
        set({ apiStatus: status, lastApiCheck: new Date() }),
    })),
    {
      name: 'app-storage',
      partialize: (state) => ({
        theme: state.theme,
        notifications: state.notifications,
      }),
    }
  )
);
```

```javascript
const { Pool } = require('pg');
const logger = require('../utils/logger');
const metrics = require('../utils/metrics');

class Database {
    constructor() {
        this.pool = new Pool({
            host: process.env.DB_HOST || 'localhost',
            port: process.env.DB_PORT || 5432,
            database: process.env.DB_NAME || 'diyapp_v12',
            user: process.env.DB_USER || 'postgres',
            password: process.env.DB_PASSWORD || 'postgres',
            max: process.env.DB_MAX_CONNECTIONS || 20,
            idleTimeoutMillis: 30000,
            connectionTimeoutMillis: 5000,
        });

        this.setupEventListeners();
        this.initializeMetrics();
    }

    setupEventListeners() {
        this.pool.on('connect', (client) => {
            logger.info('Database client connected', { clientId: client.processID });
        });

        this.pool.on('error', (err, client) => {
            logger.error('Unexpected database error', { 
                error: err.message,
                clientId: client?.processID 
            });
            metrics.incrementCounter('database_errors_total');
        });

        this.pool.on('remove', (client) => {
            logger.info('Database client removed', { clientId: client.processID });
        });
    }

    initializeMetrics() {
        // Métricas de conexão
        setInterval(() => {
            metrics.setGauge('database_connections_total', this.pool.totalCount);
            metrics.setGauge('database_connections_idle', this.pool.idleCount);
            metrics.setGauge('database_connections_waiting', this.pool.waitingCount);
        }, 15000);
    }

    async query(text, params) {
        const startTime = Date.now();
        const correlationId = require('crypto').randomBytes(16).toString('hex');
        
        try {
            logger.debug('Database query executing', {
                correlationId,
                query: text.substring(0, 200),
                params: this.maskSensitiveData(params)
            });

            const result = await this.pool.query(text, params);
            const duration = Date.now() - startTime;

            logger.debug('Database query completed', {
                correlationId,
                duration,
                rowCount: result.rowCount
            });

            metrics.recordHistogram('database_query_duration_ms', duration);
            metrics.incrementCounter('database_queries_total');

            return result;
        } catch (error) {
            const duration = Date.now() - startTime;
            
            logger.error('Database query failed', {
                correlationId,
                error: error.message,
                duration,
                query: text.substring(0, 200)
            });

            metrics.incrementCounter('database_query_errors_total');
            metrics.recordHistogram('database_error_duration_ms', duration);

            throw error;
        }
    }

    maskSensitiveData(params) {
        if (!params) return params;
        
        return params.map(param => {
            if (typeof param === 'string') {
                // Mascarar emails
                if (param.includes('@')) {
                    const [local, domain] = param.split('@');
                    return `${local.substring(0, 2)}***@${domain}`;
                }
                // Mascarar tokens JWT
                if (param.length > 100 && param.includes('.')) {
                    return `${param.substring(0, 10)}***`;
                }
            }
            return param;
        });
    }

    async transaction(callback) {
        const client = await this.pool.connect();
        
        try {
            await client.query('BEGIN');
            const result = await callback(client);
            await client.query('COMMIT');
            return result;
        } catch (error) {
            await client.query('ROLLBACK');
            throw error;
        } finally {
            client.release();
        }
    }

    async healthCheck() {
        try {
            const result = await this.query('SELECT NOW() as time');
            return {
                status: 'healthy',
                timestamp: result.rows[0].time,
                connections: {
                    total: this.pool.totalCount,
                    idle: this.pool.idleCount,
                    waiting: this.pool.waitingCount
                }
            };
        } catch (error) {
            return {
                status: 'unhealthy',
                error: error.message,
                timestamp: new Date().toISOString()
            };
        }
    }
}

module.exports = new Database();
```

```javascript
                total: parseInt(countResult.rows[0].total),

            }
        };
    }
```

    p_table_name VARCHAR,
    p_retention_days INTEGER
)
RETURNS INTEGER AS $$
DECLARE
    v_rows_deleted INTEGER;
BEGIN
    EXECUTE format(
        'DELETE FROM %I WHERE created_at < NOW() - INTERVAL ''%s days''',
        p_table_name,
        p_retention_days
    );
    
    GET DIAGNOSTICS v_rows_deleted = ROW_COUNT;

        table_name,
        retention_days,
        rows_deleted,
        executed_at
    ) VALUES (
        p_table_name,
        p_retention_days,
        v_rows_deleted,
        NOW()
    );
    
    RETURN v_rows_deleted;
END;

# DIYAPP Evolution - V12 Core - Threat Modeling e Implementação de Segurança

## 1. Threat Modeling STRIDE para V12

### Ativos Críticos:
1. **Dados de usuários**: Informações pessoais, credenciais, histórico de uso
2. **Sistema de autenticação**: Tokens JWT, sessões, OAuth
3. **APIs de integração**: WhatsApp, Telegram, sistemas externos
4. **Banco de dados**: PostgreSQL com dados sensíveis
5. **Sistema de arquivos**: Uploads de usuários, documentos
6. **LLM/IA**: Prompts, contextos, outputs de modelos

### Análise STRIDE:

#### **Spoofing (Falsificação)**
- **Ameaça**: Ataque de força bruta em login, token JWT roubado, phishing
- **Controles**: MFA obrigatório, rate limiting em login, tokens com expiração curta, validação de origem
- **Risco**: ALTO

#### **Tampering (Manipulação)**
- **Ameaça**: Manipulação de dados em trânsito, SQL injection, XSS
- **Controles**: WAF, sanitização de inputs, prepared statements, CSP headers
- **Risco**: ALTO

#### **Repudiation (Repúdio)**
- **Ameaça**: Usuário nega ação realizada no sistema
- **Controles**: Logs imutáveis com auditoria, assinatura digital de logs, rastreamento completo
- **Risco**: MÉDIO

#### **Information Disclosure (Divulgação)**
- **Ameaça**: Vazamento de dados pessoais, exposição de credenciais, erro messages informativas
- **Controles**: Criptografia em repouso, mascaramento de dados, headers de segurança, LGPD compliance
- **Risco**: CRÍTICO

#### **Denial of Service (Negação)**
- **Ameaça**: Ataques DDoS, sobrecarga de API, resource exhaustion
- **Controles**: Rate limiting em camadas, circuit breakers, quotas, CDN/WAF
- **Risco**: MÉDIO

#### **Elevation of Privilege (Elevação)**
- **Ameaça**: Escalação de privilégios, IDOR, falhas de autorização
- **Controles**: RBAC granular, verificação de permissão por recurso, princípio do menor privilégio

**src/security/waf/waf-middleware.js**
```javascript
const { createRulesEngine } = require('./rules-engine');
const { logAttack } = require('../monitoring/security-logger');

class WAFMiddleware {
    constructor() {
        this.rulesEngine = createRulesEngine();
        this.blockedIPs = new Map();
        this.attackPatterns = this.loadAttackPatterns();
    }

    loadAttackPatterns() {
        return {
            sqlInjection: [
                /(\%27)|(\')|(\-\-)|(\%23)|(#)/i,
                /((\%3D)|(=))[^\n]*((\%27)|(\')|(\-\-)|(\%3B)|(;))/i,
                /\w*((\%27)|(\'))((\%6F)|o|(\%4F))((\%72)|r|(\%52))/i,
                /((\%27)|(\'))union/i,
                /exec(\s|\+)+(s|x)p\w+/i
            ],
            xss: [
                /<script\b[^>]*>([\s\S]*?)<\/script>/gi,
                /javascript:/gi,
                /on\w+\s*=/gi,
                /<iframe\b[^>]*>([\s\S]*?)<\/iframe>/gi,
                /<object\b[^>]*>([\s\S]*?)<\/object>/gi
            ],
            pathTraversal: [
                /\.\.\//gi,
                /\.\.\\/gi,
                /etc\/passwd/gi,
                /proc\/self/gi
            ],
            commandInjection: [
                /;.*(ls|cat|rm|mkdir|chmod|wget|curl|nc|netcat|python|perl|bash|sh)/gi,
                /\|.*(ls|cat|rm|mkdir|chmod|wget|curl|nc|netcat)/gi,
                /\$\(.*\)/gi,
                /`.*`/gi
            ]
        };
    }

    async inspectRequest(req) {
        const ip = req.ip || req.connection.remoteAddress;
        
        // Verifica se IP está bloqueado
        if (this.isIPBlocked(ip)) {
            throw new Error('IP_BLOCKED');
        }

        // Analisa headers
        const headerViolations = this.checkHeaders(req.headers);
        if (headerViolations.length > 0) {
            await this.handleViolation(ip, 'HEADER_VIOLATION', headerViolations);
            return false;
        }

        // Analisa query parameters
        const queryViolations = this.checkQueryParams(req.query);
        if (queryViolations.length > 0) {
            await this.handleViolation(ip, 'QUERY_VIOLATION', queryViolations);
            return false;
        }

        // Analisa body
        if (req.body) {
            const bodyViolations = this.checkBody(req.body);
            if (bodyViolations.length > 0) {
                await this.handleViolation(ip, 'BODY_VIOLATION', bodyViolations);
                return false;
            }
        }

        // Analisa URL
        const urlViolations = this.checkURL(req.url);
        if (urlViolations.length > 0) {
            await this.handleViolation(ip, 'URL_VIOLATION', urlViolations);
            return false;
        }

        return true;
    }

    checkHeaders(headers) {
        const violations = [];
        
        // Verifica headers suspeitos
        Object.entries(headers).forEach(([key, value]) => {
            const lowerValue = String(value).toLowerCase();
            
            // SQL Injection em headers
            this.attackPatterns.sqlInjection.forEach(pattern => {
                if (pattern.test(lowerValue)) {
                    violations.push(`SQLi in header ${key}: ${value}`);
                }
            });

            // XSS em headers
            this.attackPatterns.xss.forEach(pattern => {
                if (pattern.test(lowerValue)) {
                    violations.push(`XSS in header ${key}: ${value}`);
                }
            });
        });

        return violations;
    }

    checkQueryParams(query) {
        const violations = [];
        
        const checkValue = (value, path) => {
            if (!value) return;
            
            const strValue = String(value);
            const lowerValue = strValue.toLowerCase();
            
            // SQL Injection
            this.attackPatterns.sqlInjection.forEach(pattern => {
                if (pattern.test(lowerValue)) {
                    violations.push(`SQLi in ${path}: ${strValue}`);
                }
            });

            // XSS
            this.attackPatterns.xss.forEach(pattern => {
                if (pattern.test(lowerValue)) {
                    violations.push(`XSS in ${path}: ${strValue}`);
                }
            });

            // Path Traversal
            this.attackPatterns.pathTraversal.forEach(pattern => {
                if (pattern.test(lowerValue)) {
                    violations.push(`Path traversal in ${path}: ${strValue}`);
                }
            });

            // Command Injection
            this.attackPatterns.commandInjection.forEach(pattern => {
                if (pattern.test(lowerValue)) {
                    violations.push(`Command injection in ${path}: ${strValue}`);
                }
            });
        };

        const traverse = (obj, path = '') => {
            Object.entries(obj).forEach(([key, value]) => {
                const currentPath = path ? `${path}.${key}` : key;
                
                if (value && typeof value === 'object') {
                    traverse(value, currentPath);
                } else {
                    checkValue(value, currentPath);
                }
            });
        };

        traverse(query);
        return violations;
    }

    checkBody(body) {
        return this.checkQueryParams(body); // Reutiliza mesma lógica
    }

    checkURL(url) {
        const violations = [];
        const lowerUrl = url.toLowerCase();
        
        this.attackPatterns.pathTraversal.forEach(pattern => {
            if (pattern.test(lowerUrl)) {
                violations.push(`Path traversal in URL: ${url}`);
            }
        });

        return violations;
    }

    async handleViolation(ip, violationType, details) {
        // Registra o ataque
        await logAttack({
            ip,
            violationType,
            details: Array.isArray(details) ? details.join(' | ') : details,
            timestamp: new Date().toISOString(),
            userAgent: this.currentRequest?.headers?.['user-agent']
        });

        // Incrementa contador de violações para este IP
        const violationCount = (this.blockedIPs.get(ip)?.violationCount || 0) + 1;
        
        if (violationCount >= 5) {
            // Bloqueia IP por 1 hora após 5 violações
            this.blockIP(ip, 60 * 60 * 1000); // 1 hora
        } else if (violationCount >= 3) {
            // Bloqueia IP por 15 minutos após 3 violações
            this.blockIP(ip, 15 * 60 * 1000); // 15 minutos
        }

        this.blockedIPs.set(ip, {
            violationCount,
            lastViolation: new Date(),
            blockedUntil: violationCount >= 3 ? new Date(Date.now() + (violationCount >= 5 ? 3600000 : 900000)) : null
        });
    }

    blockIP(ip, duration) {
        const blockedUntil = new Date(Date.now() + duration);
        this.blockedIPs.set(ip, {
            ...(this.blockedIPs.get(ip) || {}),
            blockedUntil,
            isBlocked: true
        });

        console.log(`[WAF] IP ${ip} bloqueado até ${blockedUntil.toISOString()}`);
    }

    isIPBlocked(ip) {
        const ipData = this.blockedIPs.get(ip);
        if (!ipData || !ipData.blockedUntil) return false;
        
        if (new Date() > ipData.blockedUntil) {
            this.blockedIPs.delete(ip);
            return false;
        }
        
        return true;
    }

    middleware() {
        return async (req, res, next) => {
            try {
                this.currentRequest = req;
                const isSafe = await this.inspectRequest(req);
                
                if (!isSafe) {
                    return res.status(403).json({
                        error: 'Acesso bloqueado pelo WAF',
                        code: 'WAF_BLOCKED',
                        timestamp: new Date().toISOString()
                    });
                }
                
                next();
            } catch (error) {
                if (error.message === 'IP_BLOCKED') {
                    return res.status(403).json({
                        error: 'IP temporariamente bloqueado',
                        code: 'IP_BLOCKED',
                        timestamp: new Date().toISOString()
                    });
                }
                
                console.error('[WAF Error]', error);
                next();
            }
        };
    }
}

module.exports = WAFMiddleware;
```

```javascript
const Redis = require('ioredis');
const crypto = require('crypto');

class RateLimiter {
    constructor() {
        this.redis = new Redis(process.env.REDIS_URL || 'redis://localhost:6379');
        this.windows = {
            SECOND: 1,
            MINUTE: 60,
            HOUR: 3600,
            DAY: 86400
        };
        
        this.limits = {
            // Limites por tipo de endpoint
            AUTH: {
                SECOND: 5,    // 5 tentativas por segundo
                MINUTE: 30,   // 30 tentativas por minuto
                HOUR: 100     // 100 tentativas por hora
            },
            API: {
                SECOND: 10,
                MINUTE: 100,
                HOUR: 1000,
                DAY: 10000
            },
            PUBLIC: {
                SECOND: 20,
                MINUTE: 200,
                HOUR: 5000
            },
            CRITICAL: {
                SECOND: 2,
                MINUTE: 10,
                HOUR: 50
            }
        };
    }

    generateKey(identifier, window, endpointType) {
        const roundedTime = Math.floor(Date.now() / 1000 / window) * window;
        return `rate_limit:${endpointType}:${identifier}:${window}:${roundedTime}`;
    }

    async checkLimit(identifier, endpointType = 'API') {
        const limits = this.limits[endpointType] || this.limits.API;
        const results = {};
        
        for (const [windowName, windowDuration] of Object.entries(this.windows)) {
            if (!limits[windowName]) continue;
            
            const key = this.generateKey(identifier, windowDuration, endpointType);
            const current = await this.redis.incr(key);
            
            if (current === 1) {
                await this.redis.expire(key, windowDuration);
            }
            
            results[windowName] = {
                current,
                limit: limits[windowName],
                remaining: Math.max(0, limits[windowName] - current),
                window: windowDuration
            };
            
            if (current > limits[windowName]) {
                return {
                    allowed: false,
                    exceededWindow: windowName,
                    results,
                    retryAfter: await this.calculateRetryAfter(key, windowDuration)
                };
            }
        }
        
        return {
            allowed: true,
            results
        };
    }

    async calculateRetryAfter(key, windowDuration) {
        const ttl = await this.redis.ttl(key);
        return ttl > 0 ? ttl : windowDuration;
    }

    async middleware(req, res, next) {
        // Identificador pode ser IP, userId, ou combinação
        let identifier = req.ip || req.connection.remoteAddress;
        
        // Para endpoints autenticados, usar userId também
        if (req.user && req.user.id) {
            identifier = `${identifier}:${req.user.id}`;
        }
        
        // Determinar tipo de endpoint baseado na rota
        let endpointType = 'API';
        const path = req.path;
        
        if (path.includes('/auth/') || path.includes('/login') || path.includes('/register')) {
            endpointType = 'AUTH';
        } else if (path.includes('/admin/') || path.includes('/config/')) {
            endpointType = 'CRITICAL';
        } else if (path.includes('/public/') || path === '/') {
            endpointType = 'PUBLIC';
        }
        
        try {
            const result = await this.checkLimit(identifier, endpointType);
            
            if (!result.allowed) {
                // Adiciona headers de rate limiting
                res.setHeader('X-RateLimit-Limit', result.results[result.exceededWindow].limit);
                res.setHeader('X-RateLimit-Remaining', 0);
                res.setHeader('X-RateLimit-Reset', Math.floor(Date.now() / 1000) + result.retryAfter);
                res.setHeader('Retry-After', result.retryAfter);
                
                return res.status(429).json({
                    error: 'Muitas requisições',
                    code: 'RATE_LIMIT_EXCEEDED',
                    retryAfter: result.retryAfter,
                    message: `Limite excedido. Tente novamente em ${result.retryAfter} segundos.`
                });
            }
            
            // Adiciona headers de rate limiting mesmo quando permitido
            const minuteLimit = result.results.MINUTE;
            if (minuteLimit) {
                res.setHeader('X-RateLimit-Limit', minuteLimit.limit);
                res.setHeader('X-RateLimit-Remaining', minuteLimit.remaining);
                res.setHeader('X-RateLimit-Reset', Math.floor(Date.now() / 1000) + minuteLimit.window);
            }
            
            next();
        } catch (error) {
            console.error('[Rate Limiter Error]', error);
            next(); // Em caso de erro, permite a requisição
        }
    }

    // Rate limiting adaptativo baseado em comportamento
    async adaptiveLimit(identifier, behaviorScore) {
        // behaviorScore: 0-100, onde 100 é comportamento normal, 0 é suspeito
        const baseLimits = this.limits.API;
        const multiplier = behaviorScore / 100;
        
        return {
            SECOND: Math.floor(baseLimits.SECOND * multiplier),
            MINUTE: Math.floor(baseLimits.MINUTE * multiplier),
            HOUR: Math.floor(baseLimits.HOUR * multiplier)
        };
    }

    // Limpeza de registros antigos
    async cleanupOldRecords() {
        const keys = await this.redis.keys('rate_limit:*');
        const now = Math.floor(Date.now() / 1000);
        
        for (const key of keys) {
            const ttl = await this.redis.ttl(key);
            if (ttl < 0) {
                await this.redis.del(key);
            }
        }
    }
}

module.exports = RateLimiter;
```

```javascript
        cost: this.costRates[name]
      }));
  }

  calculateCost(modelName, inputTokens, outputTokens) {
    const costRate = this.costRates[modelName];
    if (!costRate) {
      console.warn(`No cost rate for model ${modelName}`);
      return 0;
    }
    
    const inputCost = (inputTokens / 1000) * costRate.input;
    const outputCost = (outputTokens / 1000) * costRate.output;
    
    return parseFloat((inputCost + outputCost).toFixed(6));
  }
}
```

class LLMGateway {
  constructor() {
    this.cache = new InferenceCache();
    this.fallbackChain = this._initializeFallbackChain();
    this.metrics = {
      calls: {},
      latencies: {},
      errors: {},
      fallbacks: {}

  _initializeMetrics() {
    const models = providerConfig.getAvailableModels();
    models.forEach(model => {
      this.metrics.calls[model.name] = 0;
      this.metrics.latencies[model.name] = [];
      this.metrics.errors[model.name] = 0;
      this.metrics.fallbacks[model.name] = 0;
    });
  }

  async call({
    model,
    messages,
    temperature = 0.7,
    maxTokens = null,
    feature = 'unknown',
    useCache = true,
    cacheKey = null
  }) {
    const callId = uuidv4();
    const startTime = Date.now();
    
    // Log da chamada
    logger.info('LLM Call', {
      callId,
      model,
      feature,
      messageCount: messages.length,
      temperature,
      timestamp: new Date().toISOString()
    });

    try {
      // Verificar cache primeiro
      if (useCache) {
        const cacheKeyFinal = cacheKey || this._generateCacheKey(model, messages, temperature);
        const cached = await this.cache.get(cacheKeyFinal);
        
        if (cached) {
          logger.info('Cache hit', { callId, model, feature });
          this._recordMetrics(model, Date.now() - startTime, false, false);
          
          return {
            ...cached,
            cached: true,
            callId,
            modelUsed: model,
            fallbackUsed: false
          };
        }
      }

      // Chamada principal
      const result = await this._callModel(model, messages, temperature, maxTokens);
      const endTime = Date.now();
      const latency = endTime - startTime;

      // Calcular custo
      const inputTokens = this._estimateTokens(messages);
      const outputTokens = this._estimateTokens([{ content: result.content }]);
      const cost = providerConfig.calculateCost(model, inputTokens, outputTokens);

      // Armazenar no cache
      if (useCache) {
        const cacheKeyFinal = cacheKey || this._generateCacheKey(model, messages, temperature);
        await this.cache.set(cacheKeyFinal, {
          content: result.content,
          inputTokens,
          outputTokens,
          cost,
          timestamp: new Date().toISOString()
        });
      }

      // Registrar métricas
      this._recordMetrics(model, latency, false, false);

      // Log completo
      logger.info('LLM Call Complete', {
        callId,
        model,
        feature,
        latency,
        inputTokens,
        outputTokens,
        cost: `$${cost.toFixed(6)}`,
        success: true
      });

      return {
        content: result.content,
        inputTokens,
        outputTokens,
        cost,
        latency,
        callId,
        modelUsed: model,
        fallbackUsed: false,
        cached: false
      };

    } catch (error) {
      logger.error('LLM Call Failed', {
        callId,
        model,
        feature,
        error: error.message,
        timestamp: new Date().toISOString()
      });

      // Tentar fallback
      return await this._handleFallback(
        callId,
        model,
        messages,
        temperature,
        maxTokens,
        feature,
        startTime,
        error
      );
    }
  }

  async _callModel(model, messages, temperature, maxTokens) {
    const { client, config } = providerConfig.getProvider(model);
    const provider = config.provider;
    
    const finalMaxTokens = maxTokens || config.maxTokens;

    switch (provider) {
      case 'openai':
        const response = await client.chat.completions.create({
          model,
          messages,
          temperature,
          max_tokens: finalMaxTokens
        });
        
        return {
          content: response.choices[0].message.content,
          finishReason: response.choices[0].finish_reason
        };

      case 'anthropic':
        const anthropicResponse = await client.messages.create({
          model,
          max_tokens: finalMaxTokens,
          temperature,
          messages: messages.map(msg => ({
            role: msg.role === 'assistant' ? 'assistant' : 'user',
            content: msg.content
          }))
        });
        
        return {
          content: anthropicResponse.content[0].text,
          finishReason: 'end_turn'
        };

      case 'google':
        const googleModel = client.getGenerativeModel({ model });
        const googleResponse = await googleModel.generateContent({
          contents: messages.map(msg => ({
            role: msg.role,
            parts: [{ text: msg.content }]
          })),
          generationConfig: {
            temperature,
            maxOutputTokens: finalMaxTokens
          }
        });
        
        return {
          content: googleResponse.response.text(),
          finishReason: 'STOP'
        };

      default:
        throw new Error(`Unsupported provider: ${provider}`);

## Solução de Problemas Comuns

### Problema: Erro de conexão com banco de dados
**Solução:**
```bash
# Verifique se o PostgreSQL está rodando
sudo systemctl status postgresql

# Teste a conexão manualmente
psql -U postgres -h localhost
```

### Problema: Porta 3000 já em uso
**Solução:**
```bash
# Altere a porta no .env
PORT=3001

# Ou libere a porta
sudo lsof -ti:3000 | xargs kill -9
```

### Problema: Erro de permissão
**Solução:**
```bash
# Dê permissões adequadas
chmod -R 755 ./src
chmod 644 .env
```
