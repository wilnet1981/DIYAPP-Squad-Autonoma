Architecture Decision Record - ADR-001: Microsserviços com Event-Driven Architecture
Data: 2024-01-15
Status: Aceita
Autores: Tech Lead + Especialista Infra

CONTEXTO:
O DIYAPP V11 está enfrentando problemas de escalabilidade devido à arquitetura monolítica.
Necessidade de deploy independente, escalabilidade granular e resiliência a falhas.
Forças: Time distribuído, necessidade de deploy contínuo, complexidade crescente.

DECISÃO:
Adotar arquitetura de microsserviços com comunicação via eventos (Event-Driven Architecture).
Cada serviço será independente, com seu próprio banco de dados e ciclo de vida.
Comunicação síncrona via REST para operações críticas, assíncrona via mensageria para background.

OPÇÕES CONSIDERADAS:
- Opção A: Monolito modularizado — Prós: Simples, transações ACID. Contras: Escalabilidade limitada, deploy único.
- Opção B: Microsserviços RESTful — Prós: Independência, deploy contínuo. Contras: Acoplamento temporal, cascata de falhas.
- Opção C: Event-Driven Microservices — Prós: Desacoplamento, resiliência, escalabilidade. Contras: Complexidade eventual consistency.
- Opção escolhida: C — Justificativa: Melhor para sistemas autônomos, permite evolução independente.

CONSEQUÊNCIAS:
Positivas: Deploy independente, escalabilidade granular, tolerância a falhas, evolução tecnológica por serviço.
Negativas: Complexidade distribuída, consistência eventual, overhead operacional.
Riscos: Debug distribuído difícil, latência de rede, gerenciamento de transações distribuídas.
Mitigação: Observabilidade robusta, circuit breakers, sagas para transações.

REVISÃO: 2024-04-15

# src/shared/lib-logging/logger.py
import logging
import json
import sys
from datetime import datetime
from typing import Dict, Any, Optional
import uuid

class StructuredLogger:
    """Logger estruturado para microsserviços"""
    
    def __init__(self, service_name: str, log_level: str = "INFO"):
        self.service_name = service_name
        self.logger = logging.getLogger(service_name)
        self.logger.setLevel(getattr(logging, log_level))
        
        # Console handler com formato JSON
        handler = logging.StreamHandler(sys.stdout)
        formatter = logging.Formatter(
            '%(message)s'  # Logs já são JSON
        )
        handler.setFormatter(formatter)
        self.logger.addHandler(handler)
        
    def _create_log_record(
        self,
        level: str,
        message: str,
        extra: Optional[Dict[str, Any]] = None,
        error: Optional[Exception] = None
    ) -> Dict[str, Any]:
        """Cria registro de log estruturado"""
        record = {
            "timestamp": datetime.utcnow().isoformat() + "Z",
            "level": level,
            "service": self.service_name,
            "message": message,
            "trace_id": getattr(self, 'trace_id', None),
            "span_id": getattr(self, 'span_id', None),
            "user_id": getattr(self, 'user_id', None),
            "request_id": getattr(self, 'request_id', None),
            "environment": getattr(self, 'environment', "production"),
            "extra": extra or {}
        }
        
        if error:
            record["error"] = {
                "type": error.__class__.__name__,
                "message": str(error),
                "stack_trace": self._get_stack_trace(error)
            }
            
        return record
    
    def _get_stack_trace(self, error: Exception) -> str:
        """Extrai stack trace do erro"""
        import traceback
        return ''.join(traceback.format_exception(
            type(error), error, error.__traceback__
        ))
    
    def info(self, message: str, **kwargs):
        record = self._create_log_record("INFO", message, kwargs)
        self.logger.info(json.dumps(record))
    
    def error(self, message: str, error: Optional[Exception] = None, **kwargs):
        record = self._create_log_record("ERROR", message, kwargs, error)
        self.logger.error(json.dumps(record))
    
    def warn(self, message: str, **kwargs):
        record = self._create_log_record("WARN", message, kwargs)
        self.logger.warning(json.dumps(record))
    
    def debug(self, message: str, **kwargs):
        record = self._create_log_record("DEBUG", message, kwargs)
        self.logger.debug(json.dumps(record))
    
    def set_context(self, **context):
        """Define contexto para logs subsequentes"""
        for key, value in context.items():
            setattr(self, key, value)

```typescript
import { Routes, Route } from 'react-router-dom'
import { useDispatch } from 'react-redux'
import { setTheme } from './store/modules/ui'
import MainLayout from './layouts/MainLayout'
import LoadingSpinner from './components/ui/LoadingSpinner/LoadingSpinner'

// Lazy loading para páginas
const DashboardPage = React.lazy(() => import('./pages/DashboardPage'))
const AnalyticsPage = React.lazy(() => import('./pages/AnalyticsPage'))
const SettingsPage = React.lazy(() => import('./pages/SettingsPage'))

const App: React.FC = () => {
  const dispatch = useDispatch()

  // Inicializar tema do sistema
  useEffect(() => {
    const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches
    const savedTheme = localStorage.getItem('theme') as 'light' | 'dark' | null
    
    const theme = savedTheme || (prefersDark ? 'dark' : 'light')
    dispatch(setTheme(theme))
    document.documentElement.setAttribute('data-theme', theme)
    
    // Monitorar mudanças de tema do sistema
    const mediaQuery = window.matchMedia('(prefers-color-scheme: dark)')
    const handleChange = (e: MediaQueryListEvent) => {
      if (!savedTheme) {
        const newTheme = e.matches ? 'dark' : 'light'
        dispatch(setTheme(newTheme))
        document.documentElement.setAttribute('data-theme', newTheme)
      }
    }
    
    mediaQuery.addEventListener('change', handleChange)
    return () => mediaQuery.removeEventListener('change', handleChange)
  }, [dispatch])

  return (
    <Suspense fallback={<LoadingSpinner fullScreen />}>
      <Routes>
        <Route path="/" element={<MainLayout />}>
          <Route index element={<DashboardPage />} />
          <Route path="analytics" element={<AnalyticsPage />} />
          <Route path="settings" element={<SettingsPage />} />
        </Route>
      </Routes>
    </Suspense>
  )
}

export default App
```

from datetime import datetime, timedelta

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")


def verify_password(plain_password: str, hashed_password: str) -> bool:
    """Verifica senha com bcrypt."""
    return pwd_context.verify(plain_password, hashed_password)


def get_password_hash(password: str) -> str:
    """Gera hash bcrypt da senha."""
    return pwd_context.hash(password)


def create_access_token(data: Dict[str, Any], expires_delta: Optional[timedelta] = None) -> str:
    """Cria JWT access token."""
    to_encode = data.copy()
    if expires_delta:
        expire = datetime.utcnow() + expires_delta
    else:
        expire = datetime.utcnow() + timedelta(minutes=settings.access_token_expire_minutes)
    
    to_encode.update({"exp": expire, "type": "access"})
    encoded_jwt = jwt.encode(to_encode, settings.jwt_secret_key, algorithm=settings.jwt_algorithm)
    return encoded_jwt


def create_refresh_token(data: Dict[str, Any]) -> str:
    """Cria JWT refresh token."""
    to_encode = data.copy()
    expire = datetime.utcnow() + timedelta(days=settings.refresh_token_expire_days)
    to_encode.update({"exp": expire, "type": "refresh"})
    encoded_jwt = jwt.encode(to_encode, settings.jwt_secret_key, algorithm=settings.jwt_algorithm)
    return encoded_jwt


def verify_token(token: str, token_type: str = "access") -> Optional[Dict[str, Any]]:
    """Verifica e decodifica JWT token."""
    try:
        payload = jwt.decode(token, settings.jwt_secret_key, algorithms=[settings.jwt_algorithm])
        
        # Verifica tipo do token
        if payload.get("type") != token_type:
            return None
            
        # Verifica expiração
        if datetime.utcnow() > datetime.fromtimestamp(payload.get("exp")):
            return None
            
        return payload
    except JWTError:
        return None


def generate_verification_token() -> str:
    """Gera token para verificação de email."""
    return secrets.token_urlsafe(32)


def generate_password_reset_token() -> str:
    """Gera token para reset de senha."""
    return secrets.token_urlsafe(32)

def transform_to_staging(**context):
    """
    Transformação dos dados brutos para camada staging
    Limpeza, padronização e enriquecimento
    """
    ch_hook = ClickHouseHook(clickhouse_conn_id='clickhouse_default')
    
    transform_query = """
    INSERT INTO diyapp_analytics.stg_ux_events
    SELECT 
        event_id,
        session_id,
        NULLIF(user_id, '') as user_id,
        event_type,
        event_name,
        page_url,

        extract(page_url, '^https?://[^/]+([^?#]*)') as page_path,
        -- Extrai domínio
        extract(page_url, '^https?://([^/]+)') as page_domain,
        NULLIF(element_id, '') as element_id,

        JSONExtractKeysAndValues(metadata, 'String', 'String') as metadata,

            lower(device_info) LIKE '%mobile%' OR lower(device_info) LIKE '%android%' OR lower(device_info) LIKE '%iphone%', 
            'mobile',
            lower(device_info) LIKE '%tablet%' OR lower(device_info) LIKE '%ipad%', 
            'tablet',
            'desktop'
        ) as device_type,
        
        -- Normalização do browser
        CASE 
            WHEN lower(browser) LIKE '%chrome%' THEN 'chrome'
            WHEN lower(browser) LIKE '%firefox%' THEN 'firefox'
            WHEN lower(browser) LIKE '%safari%' AND lower(browser) NOT LIKE '%chrome%' THEN 'safari'
            WHEN lower(browser) LIKE '%edge%' THEN 'edge'
            ELSE 'other'
        END as browser_family,
        
        -- Normalização do OS
        CASE 
            WHEN lower(os) LIKE '%windows%' THEN 'windows'
            WHEN lower(os) LIKE '%mac%' OR lower(os) LIKE '%darwin%' THEN 'macos'
            WHEN lower(os) LIKE '%linux%' THEN 'linux'
            WHEN lower(os) LIKE '%android%' THEN 'android'
            WHEN lower(os) LIKE '%ios%' OR lower(os) LIKE '%iphone%' THEN 'ios'
            ELSE 'other'
        END as os_family,
        
        -- Extrai resolução da tela
        CAST(extract(screen_resolution, '^([0-9]+)') as Int16) as screen_width,
        CAST(extract(screen_resolution, 'x([0-9]+)$') as Int16) as screen_height,
        
        UPPER(NULLIF(country_code, '')) as country_code,

    WHERE received_at >= NOW() - INTERVAL 10 MINUTE
      AND received_at < NOW() - INTERVAL 5 MINUTE  -- Processa janela de 5 minutos
      AND timestamp >= NOW() - INTERVAL 7 DAY  -- Limite de retenção para processamento

```javascript
                    /exec(\s|\+)+(s|x)p\w+/i,
                    /insert(\s|\+)+into/i,
                    /select(\s|\+)+from/i,
                    /update(\s|\+)+set/i,
                    /delete(\s|\+)+from/i,
                    /drop(\s|\+)+table/i
                ],
                action: 'block',
                severity: 'high'
            },
            xss: {
                patterns: [
                    /<script\b[^>]*>([\s\S]*?)<\/script>/i,
                    /javascript:/i,
                    /on\w+\s*=/i,
                    /<iframe\b[^>]*>/i,

                    /<applet\b[^>]*>/i,
                    /<meta\b[^>]*>/i,
                    /<link\b[^>]*>/i,

                ],

                severity: 'high'

                    /\/proc\/self/,

                    /\$\(/,
                    /`/,
                    /\|\|/,
                    /&&/
                ],
                action: 'block',
                severity: 'critical'
            }
        };
    }

    scanRequest(req) {
        const threats = [];
        const requestData = {
            url: req.url,
            method: req.method,
            headers: JSON.stringify(req.headers),
            body: req.body ? JSON.stringify(req.body) : '',
            query: JSON.stringify(req.query),
            params: JSON.stringify(req.params),
            ip: req.ip || req.connection.remoteAddress
        };

        // Check all rules
        for (const [ruleName, rule] of Object.entries(this.rules)) {
            for (const pattern of rule.patterns) {
                for (const [key, value] of Object.entries(requestData)) {
                    if (typeof value === 'string' && pattern.test(value)) {
                        threats.push({
                            rule: ruleName,
                            severity: rule.severity,
                            action: rule.action,
                            detectedIn: key,
                            value: value.substring(0, 100),
                            timestamp: new Date().toISOString(),
                            requestId: this.generateRequestId(req)
                        });
                        break;
                    }
                }
            }
        }

        return threats;
    }

    generateRequestId(req) {
        const data = `${req.ip}-${req.method}-${req.url}-${Date.now()}`;
        return createHash('sha256').update(data).digest('hex').substring(0, 16);
    }

    shouldBlock(threats) {
        return threats.some(t => 
            t.severity === 'critical' || 
            (t.severity === 'high' && t.action === 'block')
        );
    }
}
```

```javascript
const WAFRules = require('./rules');
const RateLimiter = require('./rate-limiter');

class WAFMiddleware {
    constructor(config = {}) {
        this.wafRules = new WAFRules();
        this.rateLimiter = new RateLimiter(config.rateLimit || {
            windowMs: 15 * 60 * 1000, // 15 minutes
            max: 100 // limit each IP to 100 requests per windowMs
        });
        this.logger = config.logger || console;
        this.blockedIPs = new Map();
        this.learningMode = config.learningMode || false;
    }

    async middleware(req, res, next) {
        const ip = req.ip || req.connection.remoteAddress;
        
        // Check if IP is blocked
        if (this.isIPBlocked(ip)) {
            this.logSecurityEvent('IP_BLOCKED', { ip, reason: 'manual_block' });
            return res.status(403).json({
                error: 'Access denied',
                code: 'IP_BLOCKED'
            });
        }

        // Rate limiting
        try {
            await this.rateLimiter.check(ip);
        } catch (rateLimitError) {
            this.logSecurityEvent('RATE_LIMIT_EXCEEDED', { 
                ip, 
                path: req.path,
                method: req.method 
            });
            return res.status(429).json({
                error: 'Too many requests',
                retryAfter: Math.ceil(rateLimitError.resetTime / 1000)
            });
        }

        // WAF scanning
        const threats = this.wafRules.scanRequest(req);
        
        if (threats.length > 0) {
            this.logSecurityEvent('WAF_THREAT_DETECTED', {
                ip,
                threats,
                requestId: this.wafRules.generateRequestId(req),
                path: req.path,
                method: req.method
            });

            // Block if not in learning mode
            if (!this.learningMode && this.wafRules.shouldBlock(threats)) {
                this.blockIP(ip, 'waf_threat', 3600000); // Block for 1 hour
                return res.status(403).json({
                    error: 'Request blocked by security rules',
                    code: 'WAF_BLOCK',
                    requestId: this.wafRules.generateRequestId(req)
                });
            }
        }

        // Add security headers
        this.addSecurityHeaders(res);

        // Add request ID for tracing
        req.requestId = this.wafRules.generateRequestId(req);
        res.setHeader('X-Request-ID', req.requestId);

        next();
    }

    addSecurityHeaders(res) {
        res.setHeader('X-Content-Type-Options', 'nosniff');
        res.setHeader('X-Frame-Options', 'DENY');
        res.setHeader('X-XSS-Protection', '1; mode=block');
        res.setHeader('Referrer-Policy', 'strict-origin-when-cross-origin');
        res.setHeader('Content-Security-Policy', 
            "default-src 'self'; script-src 'self' 'unsafe-inline' 'unsafe-eval'; style-src 'self' 'unsafe-inline';"
        );
    }

    blockIP(ip, reason, durationMs = 3600000) {
        const blockUntil = Date.now() + durationMs;
        this.blockedIPs.set(ip, { blockUntil, reason });
        
        // Auto-unblock after duration
        setTimeout(() => {
            this.blockedIPs.delete(ip);
        }, durationMs);

        this.logSecurityEvent('IP_BLOCKED', { ip, reason, blockUntil });
    }

    isIPBlocked(ip) {
        const blockInfo = this.blockedIPs.get(ip);
        if (!blockInfo) return false;
        
        if (Date.now() > blockInfo.blockUntil) {
            this.blockedIPs.delete(ip);
            return false;
        }
        
        return true;
    }

    logSecurityEvent(eventType, data) {
        const logEntry = {
            timestamp: new Date().toISOString(),
            event: eventType,
            data,
            source: 'waf-middleware'
        };
        
        this.logger.info(JSON.stringify(logEntry));
        
        // Send to audit system if available
        if (global.auditLogger) {
            global.auditLogger.logSecurityEvent(logEntry);
        }
    }
}

module.exports = WAFMiddleware;
```

```javascript
class RateLimiter {
    constructor(config = {}) {
        this.windowMs = config.windowMs || 15 * 60 * 1000;
        this.maxRequests = config.max || 100;
        this.store = new Map();
        this.cleanupInterval = setInterval(() => this.cleanup(), this.windowMs);
    }

    async check(ip) {
        const now = Date.now();
        const windowStart = now - this.windowMs;
        
        let requests = this.store.get(ip) || [];
        
        // Remove old requests
        requests = requests.filter(timestamp => timestamp > windowStart);
        
        if (requests.length >= this.maxRequests) {
            const oldestRequest = Math.min(...requests);
            const resetTime = oldestRequest + this.windowMs;
            
            throw {
                type: 'RATE_LIMIT_EXCEEDED',
                resetTime,
                remaining: 0,
                limit: this.maxRequests
            };
        }
        
        // Add current request
        requests.push(now);
        this.store.set(ip, requests);
        
        return {
            remaining: this.maxRequests - requests.length,
            limit: this.maxRequests,
            reset: windowStart + this.windowMs
        };
    }

    cleanup() {
        const now = Date.now();
        const windowStart = now - this.windowMs;
        
        for (const [ip, requests] of this.store.entries()) {
            const validRequests = requests.filter(timestamp => timestamp > windowStart);
            if (validRequests.length === 0) {
                this.store.delete(ip);
            } else {
                this.store.set(ip, validRequests);
            }
        }
    }

    getStats(ip) {
        const requests = this.store.get(ip) || [];
        const now = Date.now();
        const windowStart = now - this.windowMs;
        
        const recentRequests = requests.filter(timestamp => timestamp > windowStart);
        
        return {
            ip,
            currentRequests: recentRequests.length,
            maxRequests: this.maxRequests,
            windowMs: this.windowMs,
            resetTime: windowStart + this.windowMs
        };
    }

    destroy() {
        clearInterval(this.cleanupInterval);
        this.store.clear();
    }
}

module.exports = RateLimiter;
```

```javascript
            headers: {

                ...(this.config.namespace && { 'X-Vault-Namespace': this.config.namespace })
            }
        });
        
        this.cache = new Map();
        this.cacheTTL = config.cacheTTL || 30000; // 30 seconds
    }

    async initialize() {
        try {
            const response = await this.client.get('/v1/sys/health');
            return {
                initialized: response.data.initialized,
                sealed: response.data.sealed,
                standby: response.data.standby,
                version: response.data.version
            };
        } catch (error) {
            throw new Error(`Vault initialization failed: ${error.message}`);
        }
    }

    async readSecret(path, version = null) {
        const cacheKey = `secret:${path}:${version || 'latest'}`;
        
        // Check cache
        if (this.cache.has(cacheKey)) {
            const cached = this.cache.get(cacheKey);
            if (Date.now() - cached.timestamp < this.cacheTTL) {
                return cached.data;
            }
        }
        
        try {
            const url = `/v1/${this.config.enginePath}/data/${path}`;
            const params = version ? { version } : {};
            
            const response = await this.client.get(url, { params });
            const secretData = response.data.data.data;
            
            // Cache the result
            this.cache.set(cacheKey, {
                data: secretData,
                timestamp: Date.now()
            });
            
            return secretData;
        } catch (error) {
            if (error.response && error.response.status === 404) {
                throw new Error(`Secret not found: ${path}`);
            }
            throw error;
        }
    }

    async writeSecret(path, data, options = {}) {
        try {
            const url = `/v1/${this.config.enginePath}/data/${path}`;
            const payload = {
                data,
                options: {
                    cas: options.cas || 0
                }
            };
            
            const response = await this.client.post(url, payload);
            
            // Invalidate cache for this path
            this.invalidateCache(path);
            
            return {
                version: response.data.data.version,
                createdTime: response.data.data.created_time,
                metadata: response.data.data.metadata
            };
        } catch (error) {
            throw new Error(`Failed to write secret: ${error.message}`);
        }
    }

    async listSecrets(path = '') {
        try {
            const url = `/v1/${this.config.enginePath}/metadata/${path}`;
            const response = await this.client('LIST', url);
            return response.data.data.keys || [];
        } catch (error) {
            if (error.response && error.response.status === 404) {
                return [];
            }
            throw error;
        }
    }

    async deleteSecret(path, versions = []) {
        try {
            if (versions.length > 0) {
                // Delete specific versions
                const url = `/v1/${this.config.enginePath}/delete/${path}`;
                await this.client.post(url, { versions });
            } else {
                // Delete all versions
                const url = `/v1/${this.config.enginePath}/metadata/${path}`;
                await this.client.delete(url);
            }
            
            // Invalidate cache
            this.invalidateCache(path);
            
            return true;
        } catch (error) {
            throw new Error(`Failed to delete secret: ${error.message}`);
        }
    }

    async generateDatabaseCredentials(role, ttl = '1h') {
        try {
            const url = `/v1/database/creds/${role}`;
            const response = await this.client.get(url);
            
            return {
                username: response.data.data.username,
                password: response.data.data.password,
                leaseId: response.data.lease_id,
                leaseDuration: response.data.lease_duration,
                renewable: response.data.renewable
            };
        } catch (error) {
            throw new Error(`Failed to generate database credentials: ${error.message}`);
        }
    }

    async encryptTransit(keyName, plaintext) {
        try {
            const url = `/v1/transit/encrypt/${keyName}`;
            const response = await this.client.post(url, {
                plaintext: Buffer.from(plaintext).toString('base64')
            });
            
            return response.data.data.ciphertext;
        } catch (error) {
            throw new Error(`Encryption failed: ${error.message}`);
        }
    }

    async decryptTransit(keyName, ciphertext) {
        try {
            const url = `/v1/transit/decrypt/${keyName}`;
            const response = await this.client.post(url, { ciphertext });
            
            return Buffer.from(response.data.data.plaintext, 'base64').toString();
        } catch (error) {
            throw new Error(`Decryption failed: ${error.message}`);
        }
    }

    invalidateCache(path) {
        for (const key of this.cache.keys()) {
            if (key.startsWith(`secret:${path}`)) {
                this.cache.delete(key);
            }
        }
    }

    clearCache() {
        this.cache.clear();
    }
}
```

## Decisões de Design

### ADR-001: Event-Driven sobre Request-Response
**Contexto:** Necessidade de desacoplamento entre módulos
**Decisão:** Usar Redis Streams como event bus
**Consequências:**
- ✅ Alta resiliência (eventos persistidos)
- ✅ Escalabilidade independente
- ⚠️ Complexidade aumentada no debugging
- ⚠️ Latência adicional (~10-50ms)

### ADR-002: Circuit Breaker Pattern
**Contexto:** Falhas em APIs externas afetavam todo o sistema
**Decisão:** Implementar circuit breakers em todas chamadas externas
**Configuração:**
```javascript

  timeout: 30000,

  resetTimeout: 60000,
  rollingCountTimeout: 10000,
  rollingCountBuckets: 10
}
```

### ADR-003: Configuração Centralizada
**Contexto:** Configs hardcoded dificultavam operações
**Decisão:** Usar Consul para configuração dinâmica
**Benefícios:**
- Mudanças de config sem redeploy
- Rollback instantâneo de configs
- Versionamento de configurações
