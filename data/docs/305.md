# DIYAPP Evolution - V12 Core - Arquitetura e Stack

## ADR-001: Arquitetura V12 - Microsserviços vs Backendless

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista LLM

### CONTEXTO:
O DIYAPP precisa evoluir para V12 com foco em:
1. Escalabilidade para 100k+ usuários ativos
2. 100% de autonomia e estabilidade
3. Capacidade de refatoração paralela (Modo Hive)
4. Manutenibilidade com múltiplas squads
5. Integração robusta com múltiplos provedores LLM
6. Interface unificada para gestão de agentes autônomos

**Forças em jogo:**
- Complexidade crescente do sistema
- Necessidade de deploy independente de componentes
- Time-to-market para novas features
- Custo de infraestrutura e operação
- Habilidade de testar componentes isoladamente

### DECISÃO:
Adotar arquitetura de microsserviços com API Gateway como orquestrador central, mantendo certos serviços serverless para operações específicas.

**Stack principal:**
- **API Gateway:** Node.js + Express + TypeScript
- **Serviço de Usuários:** Node.js + Express + TypeScript + PostgreSQL
- **Serviço de Tarefas:** Python + FastAPI + PostgreSQL + Redis
- **Serviço de LLM Orchestration:** Python + FastAPI + Redis
- **Frontend/Dashboard:** React + TypeScript + Vite

### OPÇÕES CONSIDERADAS:

**Opção A: Microsserviços completos (escolhida)**
- Prós:
  - Desacoplamento total entre serviços
  - Escalabilidade granular
  - Tecnologia adequada por domínio
  - Refatoração paralela facilitada
  - Testes isolados
- Contras:
  - Complexidade operacional maior
  - Latência de comunicação entre serviços
  - Necessidade de orquestração

**Opção B: Backendless (Supabase/Firebase)**
- Prós:
  - Time-to-market rápido
  - Custo inicial baixo
  - Operação simplificada
- Contras:
  - Vendor lock-in significativo
  - Escalabilidade limitada pelo provedor
  - Dificuldade em refatoração paralela
  - Controle limitado sobre performance

**Opção C: Monolito modular**
- Prós:
  - Simplicidade de deploy
  - Comunicação direta entre módulos
  - Debug facilitado
- Contras:
  - Escalabilidade limitada
  - Acoplamento técnico
  - Dificuldade em refatoração paralela

### CONSEQUÊNCIAS:

**Positivas:**
1. Cada squad pode trabalhar em serviços independentes
2. Escala horizontal por serviço conforme demanda
3. Tecnologia adequada para cada domínio (Python para IA, Node.js para web)
4. Isolamento de falhas - problema em um serviço não derruba o sistema
5. Deploy independente - atualizações sem downtime total

**Negativas:**
1. Complexidade aumentada em comunicação entre serviços
2. Necessidade de monitoramento distribuído
3. Overhead de rede para chamadas entre serviços
4. Gerenciamento de transações distribuídas mais complexo

**Riscos:**
1. Latência acumulada em cadeias de chamadas
   - Mitigação: Implementar circuit breakers e timeouts agressivos
2. Consistência de dados entre serviços
   - Mitigação: Event-driven architecture com mensageria
3. Debug em produção mais complexo
   - Mitigação: Implementar correlation IDs e logging centralizado

### 3. Service Discovery
```javascript
// api-gateway/src/utils/service-discovery.js
const CircuitBreaker = require('opossum');

class ServiceDiscovery {
  constructor() {
    this.services = new Map();
    this.healthCheckInterval = 30000; // 30 segundos
  }
  
  async registerService(name, url) {
    const breaker = new CircuitBreaker(async () => {
      const response = await fetch(`${url}/health`, { timeout: 5000 });
      if (!response.ok) throw new Error('Service unhealthy');
      return response.json();
    }, {
      timeout: 6000,
      errorThresholdPercentage: 50,
      resetTimeout: 30000
    });
    
    this.services.set(name, { url, breaker, healthy: true });
    
    // Health check periódico
    setInterval(() => this.checkHealth(name), this.healthCheckInterval);
  }
  
  async getService(name) {
    const service = this.services.get(name);
    if (!service || !service.healthy) {
      throw new Error(`Service ${name} unavailable`);
    }
    return service;
  }
}
```

## Implementação do API Gateway

```javascript
// api-gateway/src/index.js
const express = require('express');
const cors = require('cors');
const helmet = require('helmet');
const rateLimit = require('express-rate-limit');
const { createProxyMiddleware } = require('http-proxy-middleware');
const { ServiceDiscovery } = require('./utils/service-discovery');
const logger = require('./utils/logger');

const app = express();
const PORT = process.env.PORT || 3000;

// Middleware de segurança
app.use(helmet());
app.use(cors({
  origin: process.env.FRONTEND_URL || 'http://localhost:5173',
  credentials: true
}));

// Rate limiting
const limiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 minutos
  max: 100 // limite por IP
});
app.use('/api/', limiter);

// Logging middleware
app.use((req, res, next) => {
  req.correlationId = req.headers['x-correlation-id'] || require('crypto').randomUUID();
  logger.info(`${req.method} ${req.url}`, { correlationId: req.correlationId });
  next();
});

// Health check
app.get('/health', (req, res) => {
  res.json({
    status: 'healthy',
    timestamp: new Date().toISOString(),
    version: 'v12.0.0',
    services: Array.from(serviceDiscovery.services.keys())
  });
});

// Dashboard
app.get('/', (req, res) => {
  res.sendFile(path.join(__dirname, '../public/index.html'));
});

// Service discovery
const serviceDiscovery = new ServiceDiscovery();

// Registrar serviços
serviceDiscovery.registerService('users', process.env.USER_SERVICE_URL);
serviceDiscovery.registerService('tasks', process.env.TASK_SERVICE_URL);
serviceDiscovery.registerService('llm', process.env.LLM_SERVICE_URL);

// Proxy middleware dinâmico
app.use('/api/:service', async (req, res, next) => {
  try {
    const serviceName = req.params.service;
    const service = await serviceDiscovery.getService(serviceName);
    
    const proxy = createProxyMiddleware({
      target: service.url,
      changeOrigin: true,
      pathRewrite: {
        [`^/api/${serviceName}`]: ''
      },
      onProxyReq: (proxyReq, req) => {
        proxyReq.setHeader('X-Correlation-ID', req.correlationId);
        proxyReq.setHeader('X-User-ID', req.user?.id || 'anonymous');
      },
      onError: (err, req, res) => {
        logger.error(`Proxy error for ${serviceName}: ${err.message}`);
        res.status(503).json({
          error: 'Service temporarily unavailable',
          service: serviceName,
          correlationId: req.correlationId
        });
      }
    });
    
    proxy(req, res, next);
  } catch (error) {
    res.status(503).json({
      error: 'Service unavailable',
      message: error.message,
      correlationId: req.correlationId
    });
  }
});

// Error handling
app.use((err, req, res, next) => {
  logger.error('Unhandled error', {
    error: err.message,
    stack: err.stack,
    correlationId: req.correlationId
  });
  
  res.status(err.status || 500).json({
    error: 'Internal server error',
    correlationId: req.correlationId,
    timestamp: new Date().toISOString()
  });
});

app.listen(PORT, () => {
  logger.info(`API Gateway running on port ${PORT}`);
  logger.info(`Registered services: ${Array.from(serviceDiscovery.services.keys()).join(', ')}`);
});
```

// src/services/user.service.ts - Lógica de negócio
import bcrypt from 'bcrypt';
import jwt from 'jsonwebtoken';
import { PrismaClient } from '@prisma/client';
import { 
  RegisterInput, 
  LoginInput, 
  UserResponse, 
  AuthResponse 
} from '../types/user.types';
import { AppError } from '../utils/errors';

const prisma = new PrismaClient();
const JWT_SECRET = process.env.JWT_SECRET || 'your-secret-key-change-in-production';
const JWT_EXPIRES_IN = parseInt(process.env.JWT_EXPIRES_IN || '86400'); // 24 horas

export class UserService {
  // Circuit breaker state
  private static circuitBreakerState = {
    isOpen: false,
    failureCount: 0,
    lastFailureTime: 0,
    resetTimeout: 30000 // 30 segundos
  };

  private static checkCircuitBreaker(): void {
    if (UserService.circuitBreakerState.isOpen) {
      const now = Date.now();
      if (now - UserService.circuitBreakerState.lastFailureTime > UserService.circuitBreakerState.resetTimeout) {
        // Reset circuit breaker
        UserService.circuitBreakerState.isOpen = false;
        UserService.circuitBreakerState.failureCount = 0;
      } else {
        throw new AppError('Service temporarily unavailable', 503);
      }
    }
  }

  private static recordFailure(): void {
    UserService.circuitBreakerState.failureCount++;
    UserService.circuitBreakerState.lastFailureTime = Date.now();
    
    if (UserService.circuitBreakerState.failureCount >= 5) {
      UserService.circuitBreakerState.isOpen = true;
    }
  }

  private static recordSuccess(): void {
    UserService.circuitBreakerState.failureCount = 0;
  }

  static async register(input: RegisterInput): Promise<AuthResponse> {
    try {
      UserService.checkCircuitBreaker();

      // Verificar se usuário já existe
      const existingUser = await prisma.user.findUnique({
        where: { email: input.email }
      });

      if (existingUser) {
        throw new AppError('User with this email already exists', 409);
      }

      // Hash da senha
      const saltRounds = 10;
      const passwordHash = await bcrypt.hash(input.password, saltRounds);

      // Criar usuário
      const user = await prisma.user.create({
        data: {
          email: input.email,
          password: passwordHash,
          name: input.name
        }
      });

      // Gerar token JWT
      const token = jwt.sign(
        { userId: user.id, email: user.email },
        JWT_SECRET,
        { expiresIn: JWT_EXPIRES_IN }
      );

      UserService.recordSuccess();

      return {
        user: {
          id: user.id,
          email: user.email,
          name: user.name || undefined,
          createdAt: user.createdAt
        },
        token,
        expiresIn: JWT_EXPIRES_IN
      };
    } catch (error) {
      if (error instanceof AppError) {
        throw error;
      }
      
      UserService.recordFailure();
      
      // Log detalhado interno
      console.error(JSON.stringify({
        level: 'error',
        message: 'Registration failed',
        error: error instanceof Error ? error.message : 'Unknown error',
        stack: error instanceof Error ? error.stack : undefined,
        timestamp: new Date().toISOString()
      }));
      
      throw new AppError('Registration failed', 500);
    }
  }

  static async login(input: LoginInput): Promise<AuthResponse> {
    try {
      UserService.checkCircuitBreaker();

      // Buscar usuário
      const user = await prisma.user.findUnique({
        where: { email: input.email }
      });

      if (!user) {
        // Mesma mensagem para evitar user enumeration
        throw new AppError('Invalid credentials', 401);
      }

      // Verificar senha
      const passwordValid = await bcrypt.compare(input.password, user.password);
      
      if (!passwordValid) {
        throw new AppError('Invalid credentials', 401);
      }

      // Gerar token JWT
      const token = jwt.sign(
        { userId: user.id, email: user.email },
        JWT_SECRET,
        { expiresIn: JWT_EXPIRES_IN }
      );

      UserService.recordSuccess();

      return {
        user: {
          id: user.id,
          email: user.email,
          name: user.name || undefined,
          createdAt: user.createdAt
        },
        token,
        expiresIn: JWT_EXPIRES_IN
      };
    } catch (error) {
      if (error instanceof AppError) {
        throw error;
      }
      
      UserService.recordFailure();
      
      console.error(JSON.stringify({
        level: 'error',
        message: 'Login failed',
        error: error instanceof Error ? error.message : 'Unknown error',
        timestamp: new Date().toISOString()
      }));
      
      throw new AppError('Login failed', 500);
    }
  }

  static async getCurrentUser(userId: string): Promise<UserResponse> {
    try {
      UserService.checkCircuitBreaker();

      const user = await prisma.user.findUnique({
        where: { id: userId },
        select: {
          id: true,
          email: true,
          name: true,
          createdAt: true
        }
      });

      if (!user) {
        throw new AppError('User not found', 404);
      }

      UserService.recordSuccess();

      return user;
    } catch (error) {
      if (error instanceof AppError) {
        throw error;
      }
      
      UserService.recordFailure();
      
      console.error(JSON.stringify({
        level: 'error',
        message: 'Get user failed',
        error: error instanceof Error ? error.message : 'Unknown error',
        timestamp: new Date().toISOString()
      }));
      
      throw new AppError('Failed to retrieve user', 500);
    }
  }
}

import { Request, Response, NextFunction } from 'express';
import jwt from 'jsonwebtoken';
import { JwtPayload } from '../types/user.types';
import { AppError } from '../utils/errors';

const JWT_SECRET = process.env.JWT_SECRET || 'your-secret-key-change-in-production';

declare global {
  namespace Express {
    interface Request {
      user?: {
        userId: string;
        email: string;
      };
    }
  }
}

export const authenticate = (req: Request, res: Response, next: NextFunction): void => {
  try {
    const authHeader = req.headers.authorization;
    
    if (!authHeader || !authHeader.startsWith('Bearer ')) {
      throw new AppError('No token provided', 401);
    }

    const token = authHeader.substring(7); // Remove "Bearer "
    
    // Verificar token
    const decoded = jwt.verify(token, JWT_SECRET) as JwtPayload;
    
    // Adicionar usuário ao request
    req.user = {
      userId: decoded.userId,
      email: decoded.email
    };

    next();
  } catch (error) {
    if (error instanceof jwt.JsonWebTokenError) {
      next(new AppError('Invalid token', 401));
    } else if (error instanceof jwt.TokenExpiredError) {
      next(new AppError('Token expired', 401));
    } else {
      next(error);
    }
  }
};

export const authorize = (...roles: string[]) => {
  return (req: Request, res: Response, next: NextFunction): void => {
    if (!req.user) {
      throw new AppError('Authentication required', 401);
    }
    
    // Aqui pode ser extendido para verificar roles específicas
    // Por enquanto, apenas verifica se está autenticado
    next();
  };
};

DECLARE

-- Trigger para registrar histórico de tasks
CREATE OR REPLACE FUNCTION record_task_history()
RETURNS TRIGGER AS $$
DECLARE
    changed_fields TEXT[];
    field_name TEXT;
    old_val TEXT;
    new_val TEXT;
BEGIN
    changed_fields := ARRAY[]::TEXT[];
    
    -- Verifica campos alterados
    IF TG_OP = 'INSERT' THEN
        INSERT INTO task_history (task_id, changed_by, field_name, old_value, new_value, change_type)
        VALUES (NEW.id, COALESCE(NEW.assigned_to, NEW.created_by), 'task', NULL, 'Created', 'create');
    ELSIF TG_OP = 'UPDATE' THEN
        -- Status change
        IF OLD.status IS DISTINCT FROM NEW.status THEN
            INSERT INTO task_history (task_id, changed_by, field_name, old_value, new_value, change_type)
            VALUES (NEW.id, COALESCE(NEW.assigned_to, NEW.created_by), 'status', OLD.status, NEW.status, 'status_change');
        END IF;
        
        -- Assignment change
        IF OLD.assigned_to IS DISTINCT FROM NEW.assigned_to THEN
            INSERT INTO task_history (task_id, changed_by, field_name, old_value, new_value, change_type)
            VALUES (NEW.id, COALESCE(NEW.assigned_to, NEW.created_by), 'assigned_to', 
                   OLD.assigned_to::TEXT, NEW.assigned_to::TEXT, 'assignment');
        END IF;
        
        -- Progress change
        IF OLD.progress IS DISTINCT FROM NEW.progress THEN
            INSERT INTO task_history (task_id, changed_by, field_name, old_value, new_value, change_type)
            VALUES (NEW.id, COALESCE(NEW.assigned_to, NEW.created_by), 'progress', 
                   OLD.progress::TEXT, NEW.progress::TEXT, 'update');
        END IF;
    END IF;
    
    RETURN NEW;
END;
$$ language 'plpgsql';

CREATE TRIGGER track_task_changes
    AFTER INSERT OR UPDATE ON tasks
    FOR EACH ROW
    EXECUTE FUNCTION record_task_history();

// src/middleware/auth.js
const jwt = require('jsonwebtoken');
const { promisify } = require('util');

/**
 * Middleware de autenticação JWT com validação robusta
 * Implementa STRIDE: Spoofing protection, Tampering detection
 */
class AuthMiddleware {
  constructor() {
    this.verifyToken = promisify(jwt.verify);
    this.blacklist = new Set(); // Em produção, usar Redis
  }

  /**
   * Valida token JWT com múltiplas verificações
   */
  async validateToken(token) {
    if (!token) {
      throw new Error('Token não fornecido');
    }

    // Verifica se token está na blacklist (logout)
    if (this.blacklist.has(token)) {
      throw new Error('Token revogado');
    }

    try {
      const decoded = await this.verifyToken(token, process.env.JWT_SECRET);
      
      // Verificações adicionais de segurança
      if (!decoded.sub || !decoded.iat) {
        throw new Error('Token inválido: campos obrigatórios ausentes');
      }

      // Verifica expiração
      const now = Math.floor(Date.now() / 1000);
      if (decoded.exp && decoded.exp < now) {
        throw new Error('Token expirado');
      }

      // Verifica issued at (prevenção de replay attacks)
      if (decoded.iat > now + 60) { // 60 segundos de tolerância
        throw new Error('Token emitido no futuro');
      }

      return decoded;
    } catch (error) {
      if (error.name === 'JsonWebTokenError') {
        throw new Error('Token inválido');
      }
      if (error.name === 'TokenExpiredError') {
        throw new Error('Token expirado');
      }
      throw error;
    }
  }

  /**
   * Middleware principal de autenticação
   */
  authenticate() {
    return async (req, res, next) => {
      try {
        // Extrai token do header Authorization
        const authHeader = req.headers.authorization;
        
        if (!authHeader || !authHeader.startsWith('Bearer ')) {
          return res.status(401).json({
            error: 'Acesso não autorizado',
            message: 'Token Bearer não fornecido'
          });
        }

        const token = authHeader.substring(7); // Remove "Bearer "
        
        // Rate limiting básico por IP (prevenção DoS)
        const clientIp = req.ip || req.connection.remoteAddress;
        const rateLimitKey = `auth_attempt:${clientIp}`;
        // Em produção, implementar com Redis
        
        // Valida token
        const decoded = await this.validateToken(token);
        
        // Injeta dados do usuário no request
        req.user = {
          id: decoded.sub,
          email: decoded.email,
          roles: decoded.roles || ['user'],
          permissions: decoded.permissions || []
        };

        // Log de auditoria (STRIDE: Repudiation)
        this.logAuthSuccess(req.user.id, clientIp, req.method, req.originalUrl);

        next();
      } catch (error) {
        // Log de tentativa falha (sem expor detalhes internos)
        const clientIp = req.ip || req.connection.remoteAddress;

        });

   */
  authorize(requiredRoles = [], requiredPermissions = []) {
    return (req, res, next) => {
      if (!req.user) {
        return res.status(401).json({
          error: 'Não autenticado'
        });
      }

      // Verifica roles
      if (requiredRoles.length > 0) {
        const hasRole = requiredRoles.some(role => 
          req.user.roles.includes(role)
        );
        
        if (!hasRole) {

            error: 'Acesso negado',
            message: 'Permissões insuficientes (role)'

          req.user.permissions.includes(permission)
        );
        
        if (!hasPermission) {

        .matches(/[!@#$%^&*(),.?":{}|<>]/)
        .withMessage('Senha deve conter pelo menos um caractere especial'),

      // Sanitização para números
      sanitizeNumber: (field) => body(field)
        .isInt()
        .withMessage('Deve ser um número inteiro')
        .toInt(),

      // Sanitização para booleanos
      sanitizeBoolean: (field) => body(field)
        .isBoolean()
        .withMessage('Deve ser verdadeiro ou falso')
        .toBoolean(),

      // Sanitização para URLs
      sanitizeURL: (field) => body(field)
        .trim()
        .isURL({
          protocols: ['http', 'https'],
          require_protocol: true,
          require_valid_protocol: true
        })
        .withMessage('URL inválida'),

      // Sanitização para conteúdo HTML (prevenção XSS)
      sanitizeHTML: (field) => body(field)
        .customSanitizer(value => {

  validateRequest() {
    return (req, res, next) => {

      if (!errors.isEmpty()) {
        // Log de tentativa de tampering
        console.warn(`[VALIDATION_FAILURE] IP: ${req.ip}, Path: ${req.path}, Errors: ${JSON.stringify(errors.array())}`);
        
        return res.status(400).json({
          error: 'Validação falhou',
          messages: errors.array().map(err => ({
            field: err.param,
            message: err.msg,
            value: err.value
          }))
        });
      }
      
      next();
    };
  }

  /**
   * Prevenção de MIME sniffing e outros headers de segurança
   */
  getSecurityHeaders() {
    return helmet(this.helmetConfig);
  }

  /**
   * Sanitização de outputs (prevenção Info Disclosure)
   */
  sanitizeOutput(data, fieldsToMask = []) {
    if (!data || typeof data !== 'object') return data;
    
    const sanitized = Array.isArray(data) ? [] : {};
    
    for (const [key, value] of Object.entries(data)) {
      if (fieldsToMask.includes(key) && value) {
        // Mascaramento de dados sensíveis
        if (typeof value === 'string' && value.length > 4) {
          sanitized[key] = value.substring(0, 2) + '***' + value.substring(value.length - 2);
        } else {
          sanitized[key] = '***';
        }
      } else if (key === 'password' || key.includes('secret') || key.includes('token')) {
        // Nunca retornar senhas ou tokens
        sanitized[key] = undefined;
      } else if (value && typeof value === 'object') {
        // Recursão para objetos aninhados
        sanitized[key] = this.sanitizeOutput(value, fieldsToMask);
      } else {
        sanitized[key] = value;
      }
    }
    
    return sanitized;
  }

  /**
   * Middleware para sanitização automática de responses
   */
  responseSanitizer(fieldsToMask = []) {
    return (req, res, next) => {
      const originalJson = res.json;
      
      res.json = function(data) {
        // Aplica sanitização apenas para sucesso (2xx)
        if (res.statusCode >= 200 && res.statusCode < 300) {
          const sanitizedData = this.sanitizeOutput(data, fieldsToMask);
          return originalJson.call(this, sanitizedData);
        }
        
        return originalJson.call(this, data);
      }.bind(this);
      
      next();
    };
  }

      }

        {

        },
        process.env.JWT_SECRET,
        {
          algorithm: this.algorithm,
          expiresIn: this.accessTokenExpiry
        }
      );

      return newAccessToken;
    } catch (error) {
      throw new Error(`Falha ao renovar token: ${error.message}`);
    }
  }

   * Gera secret seguro para JWT
   */
  static generateSecret() {
    return crypto.randomBytes(64).toString('hex');
  }

  /**
   * Valida força do secret
   */
  static validateSecret(secret) {
    if (!secret || secret.length < 32) {
      throw new Error('JWT secret deve ter pelo menos 32 caracteres');

    // Verifica complexidade
    const hasUpperCase = /[A-Z]/.test(secret);

    const hasNumbers = /\d/.test(secret);

    if (!(hasUpperCase && hasLowerCase && hasNumbers && hasSpecial)) {
      throw new Error('JWT secret deve conter maiúsculas, minúsculas, números e caracteres especiais');
    }

  }

# File: src/controllers/tag_suggestions.py
"""
Controller para sugestão automática de tags usando IA generativa
Integração com OpenAI GPT-4/3.5 para análise de conteúdo de tarefas
"""

import os
import json
import logging
from typing import List, Dict, Optional
from datetime import datetime
import openai
from flask import Blueprint, request, jsonify, current_app
import sqlite3
from dataclasses import dataclass
from enum import Enum

# Configuração de logging
logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)

# Blueprint para rotas de sugestão de tags
tag_suggestions_bp = Blueprint('tag_suggestions', __name__)

class TagCategory(Enum):
    """Categorias de tags para melhor organização"""
    PRIORITY = "priority"
    CONTEXT = "context"
    SKILL = "skill"
    TOOL = "tool"
    TIME = "time"
    CUSTOM = "custom"

@dataclass
class TagSuggestion:
    """Estrutura para sugestões de tags"""
    tag: str
    category: TagCategory
    confidence: float
    reason: str

class TagSuggester:
    """Classe principal para sugestão de tags usando IA"""
    
    def __init__(self, api_key: Optional[str] = None):
        """Inicializa o sugestor de tags com OpenAI"""
        self.api_key = api_key or os.getenv('OPENAI_API_KEY')
        if self.api_key:
            openai.api_key = self.api_key
        else:
            logger.warning("OPENAI_API_KEY não configurada - modo fallback ativado")
        
        # Tags padrão para fallback
        self.fallback_tags = {
            TagCategory.PRIORITY: ["urgente", "importante", "baixa-prioridade"],
            TagCategory.CONTEXT: ["trabalho", "pessoal", "estudo", "saúde"],
            TagCategory.SKILL: ["programação", "design", "análise", "escrita"],
            TagCategory.TOOL: ["excel", "vscode", "figma", "photoshop"],
            TagCategory.TIME: ["rápido", "longo-prazo", "15min", "1h"]
        }
        
        # Prompt template otimizado
        self.prompt_template = """Analise a seguinte tarefa e sugira tags relevantes.

TAREFA: {task_title}
DESCRIÇÃO: {task_description}

INSTRUÇÕES:
1. Sugira 3-5 tags que ajudariam a categorizar e encontrar esta tarefa
2. Classifique cada tag em uma categoria: priority, context, skill, tool, time, ou custom
3. Para cada tag, forneça uma breve justificativa
4. Formato de resposta JSON:
{{
    "suggestions": [
        {{
            "tag": "nome-da-tag",
            "category": "categoria",
            "confidence": 0.95,
            "reason": "explicação breve"
        }}
    ]
}}

Exemplo de tags comuns:
- priority: urgente, importante, baixa-prioridade
- context: trabalho, pessoal, estudo, família
- skill: programação, design, análise, marketing
- tool: excel, vscode, figma, photoshop
- time: rápido, longo-prazo, 15min, 1h
- custom: tags específicas do contexto

Tags sugeridas:"""
    
    def suggest_tags(self, task_title: str, task_description: str = "") -> List[TagSuggestion]:
        """
        Sugere tags para uma tarefa usando OpenAI ou fallback
        
        Args:
            task_title: Título da tarefa
            task_description: Descrição da tarefa (opcional)
            
        Returns:
            Lista de sugestões de tags
        """
        try:
            # Se não tiver API key, usa fallback
            if not self.api_key:
                return self._get_fallback_suggestions(task_title)
            
            # Prepara o prompt
            prompt = self.prompt_template.format(
                task_title=task_title[:200],  # Limita tamanho
                task_description=task_description[:500] if task_description else "(sem descrição)"
            )
            
            # Chama OpenAI API
            response = openai.ChatCompletion.create(
                model="gpt-3.5-turbo",  # Pode ser alterado para gpt-4 se disponível
                messages=[
                    {"role": "system", "content": "Você é um assistente especializado em organização e produtividade."},
                    {"role": "user", "content": prompt}
                ],
                temperature=0.7,
                max_tokens=500,
                response_format={"type": "json_object"}
            )
            
            # Processa a resposta
            content = response.choices[0].message.content
            suggestions_data = json.loads(content)
            
            # Converte para objetos TagSuggestion
            suggestions = []
            for item in suggestions_data.get("suggestions", []):
                try:
                    suggestion = TagSuggestion(
                        tag=item["tag"].lower().replace(" ", "-"),
                        category=TagCategory(item["category"]),
                        confidence=float(item.get("confidence", 0.8)),
                        reason=item["reason"]
                    )
                    suggestions.append(suggestion)
                except (KeyError, ValueError) as e:
                    logger.warning(f"Erro ao processar sugestão: {e}")
                    continue
            
            # Ordena por confiança
            suggestions.sort(key=lambda x: x.confidence, reverse=True)
            
            # Limita a 5 sugestões
            return suggestions[:5]
            
        except Exception as e:
            logger.error(f"Erro ao sugerir tags: {e}")
            return self._get_fallback_suggestions(task_title)
    
    def _get_fallback_suggestions(self, task_title: str) -> List[TagSuggestion]:
        """Sugestões de fallback baseadas em palavras-chave"""
        suggestions = []
        task_lower = task_title.lower()
        
        # Mapeamento de palavras-chave para tags
        keyword_mapping = {
            "urgent": ("urgente", TagCategory.PRIORITY, 0.9),
            "important": ("importante", TagCategory.PRIORITY, 0.8),
            "bug": ("bug", TagCategory.PRIORITY, 0.85),
            "fix": ("correção", TagCategory.PRIORITY, 0.8),
            "work": ("trabalho", TagCategory.CONTEXT, 0.7),
            "personal": ("pessoal", TagCategory.CONTEXT, 0.7),
            "study": ("estudo", TagCategory.CONTEXT, 0.7),
            "code": ("programação", TagCategory.SKILL, 0.8),
            "design": ("design", TagCategory.SKILL, 0.8),
            "write": ("escrita", TagCategory.SKILL, 0.7),
            "quick": ("rápido", TagCategory.TIME, 0.6),
            "meeting": ("reunião", TagCategory.CONTEXT, 0.75),
        }
        
        # Procura palavras-chave no título
        for keyword, (tag, category, confidence) in keyword_mapping.items():
            if keyword in task_lower:
                suggestions.append(TagSuggestion(
                    tag=tag,
                    category=category,
                    confidence=confidence,
                    reason=f"Palavra-chave '{keyword}' encontrada no título"
                ))
        
        # Adiciona algumas tags padrão se não encontrou muitas
        if len(suggestions) < 3:
            default_tags = [
                TagSuggestion("importante", TagCategory.PRIORITY, 0.5, "Tag padrão de prioridade"),
                TagSuggestion("trabalho", TagCategory.CONTEXT, 0.5, "Tag padrão de contexto"),
                TagSuggestion("15min", TagCategory.TIME, 0.4, "Estimativa de tempo padrão"),
            ]
            suggestions.extend(default_tags[:3 - len(suggestions)])
        
        return suggestions[:5]

@tag_suggestions_bp.route('/tasks/suggest-tags', methods=['POST'])
def suggest_tags():
    """
    Endpoint para sugestão de tags usando IA
    
    Request JSON:
    {
        "title": "Título da tarefa",
        "description": "Descrição da tarefa (opcional)",
        "existing_tags": ["tag1", "tag2"] (opcional)
    }
    
    Response JSON:
    {
        "success": true,
        "suggestions": [
            {
                "tag": "nome-tag",
                "category": "priority|context|skill|tool|time|custom",
                "confidence": 0.95,
                "reason": "Explicação da sugestão"
            }
        ],
        "timestamp": "2024-01-15T10:30:00Z"
    }
    """
    try:
        data = request.get_json()
        
        if not data or 'title' not in data:
            return jsonify({
                "success": False,
                "error": "Campo 'title' é obrigatório"
            }), 400
        
        task_title = data['title']
        task_description = data.get('description', '')
        existing_tags = set(data.get('existing_tags', []))
        
        # Obtém sugestões
        suggestions = tag_suggester.suggest_tags(task_title, task_description)
        
        # Filtra tags que já existem
        filtered_suggestions = [
            s for s in suggestions 
            if s.tag not in existing_tags
        ]
        
        # Converte para dicionário
        suggestions_dict = [
            {
                "tag": s.tag,
                "category": s.category.value,
                "confidence": s.confidence,
                "reason": s.reason
            }
            for s in filtered_suggestions
        ]
        
        # Log da sugestão (para análise de eficácia)
        logger.info(f"Tags sugeridas para tarefa: '{task_title[:50]}...' - {len(suggestions_dict)} sugestões")
        
        return jsonify({
            "success": True,
            "suggestions": suggestions_dict,
            "timestamp": datetime.utcnow().isoformat() + "Z",
            "model_used": "gpt-3.5-turbo" if tag_suggester.api_key else "fallback"
        })
        
    except Exception as e:
        logger.error(f"Erro no endpoint suggest-tags: {e}")
        return jsonify({
            "success": False,
            "error": "Erro interno ao processar sugestão de tags",
            "timestamp": datetime.utcnow().isoformat() + "Z"
        }), 500

    const safeUser = { ...this.dataValues };
    
    // Campos sensíveis que devem ser anonimizados em logs
    const sensitiveFields = [
      'cpf', 'cnpj', 'rg', 'passport', 
      'credit_card_number', 'bank_account',
      'phone', 'email', 'address', 'birth_date'
    ];
    
    sensitiveFields.forEach(field => {
      if (safeUser[field]) {
        // Anonimização: mantém apenas os últimos 4 caracteres
        const value = String(safeUser[field]);
        if (value.length > 4) {
          safeUser[field] = '***' + value.slice(-4);
        } else {
          safeUser[field] = '***';
        }
      }
    });
    
    // Hash do ID para logs (reversível apenas com chave)
    if (safeUser.id) {
      safeUser.log_id = crypto.createHash('sha256')
        .update(process.env.LOG_HASH_SALT + safeUser.id)
        .digest('hex')
        .substring(0, 16);
      delete safeUser.id;

  // Endpoint para exclusão de dados (GDPR Right to Erasure / LGPD Direito à Exclusão)
  static async deleteUserData(req, res) {
    try {
      const { userId, reason, adminId } = req.body;
      
      // Validações
      if (!userId || !reason) {
        return res.status(400).json({
          error: 'LGPD_VALIDATION_ERROR',
          message: 'userId e reason são obrigatórios'
        });
      }
      
      // Buscar usuário
      const user = await User.findByPk(userId);
      if (!user) {
        return res.status(404).json({
          error: 'USER_NOT_FOUND',
          message: 'Usuário não encontrado'
        });
      }
      
      // Verificar se já está excluído
      if (user.deleted_at) {
        return res.status(400).json({
          error: 'ALREADY_DELETED',
          message: 'Usuário já foi excluído'
        });
      }
      
      // Executar exclusão com conformidade
      await user.softDeleteWithCompliance(reason, adminId);
      
      // Registrar log
      await ComplianceLog.create({
        user_id: userId,
        action: 'DATA_DELETION',
        details: {
          reason,
          admin_id: adminId,
          timestamp: new Date().toISOString(),
          compliance_article: 'LGPD Art. 16, GDPR Art. 17'
        },
        ip_address: req.ip,
        user_agent: req.get('User-Agent')
      });
