# DIYAPP Evolution - V12 Core - Arquitetura de Microsserviços

## 1. ADR-001: Arquitetura de Microsserviços V12

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista Backend

### CONTEXTO:
A V11 do DIYAPP atingiu limites de escalabilidade devido à arquitetura monolítica. Necessitamos de:
1. Deploy independente por funcionalidade
2. Isolamento de falhas
3. Escala granular por demanda
4. Autonomia das squads para evolução de serviços

### DECISÃO:
Adotar arquitetura de microsserviços com os seguintes princípios:
- **12-Factor App** para todos os serviços
- **Service Mesh** (Istio) para comunicação e observabilidade
- **Domain-Driven Design** para bounded contexts
- **Database per Service** com eventos para consistência eventual
- **API Gateway** como ponto único de entrada

### OPÇÕES CONSIDERADAS:
- **Opção A:** Monólito modularizado - Prós: Simplicidade inicial, transações ACID. Contras: Escalabilidade limitada, deploy único.
- **Opção B:** Microsserviços completos - Prós: Escala independente, resiliência, deploy contínuo. Contras: Complexidade operacional, consistência eventual.
- **Opção C:** Arquitetura em camadas - Prós: Separação de responsabilidades. Contras: Acoplamento ainda presente.

**Opção escolhida:** B - Justificativa: Alinhamento com meta de 100% autonomia e necessidade de escalabilidade exponencial.

### CONSEQUÊNCIAS:
**Positivas:**
- Deploy independente por squad
- Escala granular por demanda
- Isolamento de falhas
- Stack tecnológica adequada por domínio

**Negativas:**
- Complexidade distribuída
- Consistência eventual
- Overhead de comunicação
- Monitoramento distribuído necessário

**Riscos:**
- Latência em chamadas entre serviços
- Debug distribuído complexo
- Mitigação: Service Mesh com tracing distribuído, circuit breakers, timeouts configuráveis

## 2. ADR-002: Padrões de Comunicação entre Serviços

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra

### CONTEXTO:
Necessidade de comunicação eficiente, tipada e observável entre microsserviços. Requisitos:
- Baixa latência para chamadas síncronas
- Desacoplamento para operações assíncronas
- Contratos explícitos entre serviços
- Observabilidade completa

### DECISÃO:
Adotar comunicação híbrida:
1. **gRPC** para chamadas síncronas (service-to-service)
2. **Apache Kafka** para eventos de domínio assíncronos
3. **REST/GraphQL** apenas para APIs públicas via Gateway

### OPÇÕES CONSIDERADAS:
- **Opção A:** REST apenas - Prós: Simplicidade, HTTP padrão. Contras: Sem contrato forte, overhead JSON, sem streaming.
- **Opção B:** gRPC apenas - Prós: Performance, contrato forte, streaming. Contras: Complexidade para eventos assíncronos.
- **Opção C:** Kafka apenas - Prós: Desacoplamento total, replay. Contras: Latência para respostas síncronas.

**Opção escolhida:** Combinação gRPC + Kafka - Justificativa: Balanceamento entre performance síncrona e desacoplamento assíncrono.

### CONSEQUÊNCIAS:
**Positivas:**
- Performance otimizada para cada caso
- Contratos Protocol Buffers para gRPC
- Event sourcing habilitado
- Schema Registry para evolução de contratos

**Negativas:**
- Duas stacks de comunicação para manter
- Necessidade de serialização/deserialização eficiente

**Riscos:**
- Complexidade de debugging em duas camadas
- Mitigação: Tracing distribuído com correlation IDs em ambos os canais

		zap.String("service_id", req.ServiceId))
	
	// Coleta métricas históricas
	metrics, err := e.metricsClient.GetHistoricalMetrics(ctx, req.ServiceId, 5*time.Minute)
	if err != nil {
		return nil, fmt.Errorf("falha ao coletar métricas: %w", err)
	}
	
	// Analisa sintomas
	analysis := e.analyzeSymptoms(req.Symptoms, metrics)
	
	// Identifica causa raiz usando regras e ML
	rootCause, confidence := e.identifyRootCause(analysis)
	
	// Sugere remédios baseados no conhecimento
	remedies := e.suggestRemedies(rootCause, analysis)
	
	// Filtra remédios por risco
	filteredRemedies := e.filterByRisk(remedies, e.config.RiskThreshold)
	
	// Calcula tempo estimado de recuperação
	recoveryTime := e.estimateRecoveryTime(filteredRemedies)
	
	return &protos.DiagnosisResponse{
		DiagnosisId:           uuid.New().String(),
		RootCause:             rootCause,
		Confidence:            confidence,
		SuggestedRemedies:     filteredRemedies,
		EstimatedRecoveryTime: int64(recoveryTime.Seconds()),
	}, nil
}

// ExecuteRemedy executa um remédio específico
func (e *AutoCureEngine) ExecuteRemedy(ctx context.Context, req *protos.RemediesRequest) (*protos.RemediesResponse, error) {
	executionID := uuid.New().String()
	
	e.logger.Info("Executando remédio",
		zap.String("execution_id", executionID),
		zap.String("remedy_id", req.RemedyId),
		zap.Bool("dry_run", req.DryRun))
	
	// Valida autorização
	if !e.validateAuthorization(req.AuthorizationToken) {
		return nil, fmt.Errorf("autorização inválida")
	}
	
	// Recupera diagnóstico
	diagnosis, err := e.getDiagnosis(ctx, req.DiagnosisId)
	if err != nil {
		return nil, fmt.Errorf("diagnóstico não encontrado: %w", err)
	}
	
	// Encontra o remédio
	var remedy *protos.Remedy
	for _, r := range diagnosis.SuggestedRemedies {
		if r.Id == req.RemedyId {
			remedy = r
			break
		}
	}
	
	if remedy == nil {
		return nil, fmt.Errorf("remédio não encontrado no diagnóstico")
	}
	
	// Verifica pré-requisitos
	if err := e.checkPrerequisites(ctx, remedy); err != nil {
		return nil, fmt.Errorf("pré-requisitos não atendidos: %w", err)
	}
	
	// Se for dry run, apenas simula
	if req.DryRun {
		return e.simulateRemedy(ctx, executionID, remedy)
	}
	
	// Executa o remédio
	executor, exists := e.remedyRegistry[remedy.Type.String()]
	if !exists {
		return nil, fmt.Errorf("executor não encontrado para tipo: %s", remedy.Type)
	}
	
	startTime := time.Now().Unix()
	results, err := executor.Execute(ctx, remedy, diagnosis)
	
	response := &protos.RemediesResponse{
		ExecutionId: executionID,
		StartedAt:   startTime,
		Results:     results,
	}
	
	if err != nil {
		e.logger.Error("Falha na execução do remédio",
			zap.Error(err),
			zap.String("execution_id", executionID))
		
		response.Status = protos.RemediesResponse_FAILED
		response.Message = err.Error()
		
		// Tenta rollback automático
		if remedy.RiskScore > 0.7 {
			e.autoRollback(ctx, executionID, remedy, err)
			response.Status = protos.RemediesResponse_ROLLED_BACK
		}
	} else {
		response.Status = protos.RemediesResponse_COMPLETED
		response.CompletedAt = time.Now().Unix()
		response.Message = "Remédio aplicado com sucesso"
		
		// Publica evento de cura
		e.publishCureEvent(ctx, executionID, remedy, diagnosis)
	}
	
	// Registra execução no histórico
	e.recordExecution(ctx, executionID, remedy, diagnosis, response)
	
	return response, nil
}

logger = logging.getLogger(__name__)

# ============================================================================
# MODELOS DE DADOS
# ============================================================================

class HealthStatus(Enum):
    HEALTHY = "healthy"
    DEGRADED = "degraded"
    UNHEALTHY = "unhealthy"
    UNKNOWN = "unknown"

class ServiceType(Enum):
    API = "api"
    DATABASE = "database"
    CACHE = "cache"
    EXTERNAL = "external"
    LLM = "llm"

@dataclass
class ServiceHealth:
    name: str
    service_type: ServiceType
    status: HealthStatus
    latency_ms: Optional[float] = None
    error_rate: Optional[float] = None
    last_check: Optional[datetime] = None
    details: Dict[str, Any] = None
    
    def to_dict(self):
        return {
            **asdict(self),
            "service_type": self.service_type.value,
            "status": self.status.value,
            "last_check": self.last_check.isoformat() if self.last_check else None
        }

@dataclass
class SystemMetrics:
    cpu_percent: float
    memory_percent: float
    disk_usage_percent: float
    active_connections: int
    request_rate: float
    timestamp: datetime
    
    def to_dict(self):
        return {
            **asdict(self),
            "timestamp": self.timestamp.isoformat()
        }

# ============================================================================
# REPOSITORY LAYER - ACESSO A DADOS
# ============================================================================

class HealthRepository:
    def __init__(self, db_path: str = DB_PATH):
        self.db_path = db_path
        self.init_db()
    
    @contextmanager
    def get_connection(self):
        """Context manager para conexões com banco de dados"""
        conn = sqlite3.connect(self.db_path)
        conn.row_factory = sqlite3.Row
        try:
            yield conn
            conn.commit()
        except Exception as e:
            conn.rollback()
            raise e
        finally:
            conn.close()
    
    def init_db(self):
        """Inicializa o banco de dados com tabelas necessárias"""
        with self.get_connection() as conn:
            # Tabela de métricas de serviço
            conn.execute('''
                CREATE TABLE IF NOT EXISTS service_metrics (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    service_name TEXT NOT NULL,
                    service_type TEXT NOT NULL,
                    status TEXT NOT NULL,
                    latency_ms REAL,
                    error_rate REAL,
                    timestamp DATETIME DEFAULT CURRENT_TIMESTAMP,
                    details TEXT
                )
            ''')
            
            # Tabela de métricas do sistema
            conn.execute('''
                CREATE TABLE IF NOT EXISTS system_metrics (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    cpu_percent REAL NOT NULL,
                    memory_percent REAL NOT NULL,
                    disk_usage_percent REAL NOT NULL,
                    active_connections INTEGER NOT NULL,
                    request_rate REAL NOT NULL,
                    timestamp DATETIME DEFAULT CURRENT_TIMESTAMP
                )
            ''')
            
            # Tabela de incidentes
            conn.execute('''
                CREATE TABLE IF NOT EXISTS incidents (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    service_name TEXT NOT NULL,
                    previous_status TEXT NOT NULL,
                    new_status TEXT NOT NULL,
                    reason TEXT,
                    timestamp DATETIME DEFAULT CURRENT_TIMESTAMP,
                    resolved_at DATETIME
                )
            ''')
            
            # Índices para performance
            conn.execute('CREATE INDEX IF NOT EXISTS idx_service_metrics_timestamp ON service_metrics(timestamp)')
            conn.execute('CREATE INDEX IF NOT EXISTS idx_system_metrics_timestamp ON system_metrics(timestamp)')
    
    def save_service_metric(self, service_health: ServiceHealth):
        """Salva métrica de saúde de serviço no banco"""
        with self.get_connection() as conn:
            conn.execute('''
                INSERT INTO service_metrics 
                (service_name, service_type, status, latency_ms, error_rate, details)
                VALUES (?, ?, ?, ?, ?, ?)
            ''', (
                service_health.name,
                service_health.service_type.value,
                service_health.status.value,
                service_health.latency_ms,
                service_health.error_rate,
                json.dumps(service_health.details) if service_health.details else None
            ))
    
    def save_system_metric(self, system_metrics: SystemMetrics):
        """Salva métrica do sistema no banco"""
        with self.get_connection() as conn:
            conn.execute('''
                INSERT INTO system_metrics 
                (cpu_percent, memory_percent, disk_usage_percent, active_connections, request_rate)
                VALUES (?, ?, ?, ?, ?)
            ''', (
                system_metrics.cpu_percent,
                system_metrics.memory_percent,
                system_metrics.disk_usage_percent,
                system_metrics.active_connections,
                system_metrics.request_rate
            ))
    
    def record_incident(self, service_name: str, previous_status: HealthStatus, 
                       new_status: HealthStatus, reason: str = None):
        """Registra um incidente de mudança de status"""
        with self.get_connection() as conn:
            conn.execute('''
                INSERT INTO incidents 
                (service_name, previous_status, new_status, reason)
                VALUES (?, ?, ?, ?)
            ''', (
                service_name,
                previous_status.value,
                new_status.value,
                reason
            ))
    
    def get_service_health_history(self, service_name: str, hours: int = 24) -> List[Dict]:
        """Recupera histórico de saúde de um serviço"""
        with self.get_connection() as conn:
            cursor = conn.execute('''
                SELECT * FROM service_metrics 
                WHERE service_name = ? 
                AND timestamp > datetime('now', ?)
                ORDER BY timestamp DESC
            ''', (service_name, f'-{hours} hours'))
            
            return [dict(row) for row in cursor.fetchall()]
    
    def get_system_metrics_history(self, hours: int = 24) -> List[Dict]:
        """Recupera histórico de métricas do sistema"""
        with self.get_connection() as conn:
            cursor = conn.execute('''
                SELECT * FROM system_metrics 
                WHERE timestamp > datetime('now', ?)
                ORDER BY timestamp DESC
            ''', (f'-{hours} hours',))
            
            return [dict(row) for row in cursor.fetchall()]

class HealthService:
    def __init__(self):
        self.repository = HealthRepository()
        self.services = self._initialize_services()
        self.request_counter = 0
        self.request_lock = threading.Lock()
        self.last_request_time = datetime.now()
        
        # Inicia worker de coleta periódica
        self._start_health_worker()
    
    def _initialize_services(self) -> Dict[str, ServiceHealth]:
        """Inicializa serviços monitorados"""
        return {
            "main-api": ServiceHealth(
                name="main-api",
                service_type=ServiceType.API,
                status=HealthStatus.UNKNOWN,
                last_check=None
            ),
            "user-db": ServiceHealth(
                name="user-db",
                service_type=ServiceType.DATABASE,
                status=HealthStatus.UNKNOWN,
                last_check=None
            ),
            "redis-cache": ServiceHealth(
                name="redis-cache",
                service_type=ServiceType.CACHE,
                status=HealthStatus.UNKNOWN,
                last_check=None
            ),
            "payment-gateway": ServiceHealth(
                name="payment-gateway",
                service_type=ServiceType.EXTERNAL,
                status=HealthStatus.UNKNOWN,
                last_check=None
            ),
            "llm-service": ServiceHealth(
                name="llm-service",
                service_type=ServiceType.LLM,
                status=HealthStatus.UNKNOWN,
                last_check=None
            )
        }
    
    def _start_health_worker(self):
        """Inicia worker que coleta métricas periodicamente"""
        def health_check_worker():
            schedule.every(30).seconds.do(self._perform_health_checks)
            schedule.every(5).minutes.do(self._collect_system_metrics)
            
            while True:
                schedule.run_pending()
                time.sleep(1)
        
        thread = threading.Thread(target=health_check_worker, daemon=True)
        thread.start()
    
    @circuit(failure_threshold=3, recovery_timeout=60)
    def _check_api_health(self, service_name: str, endpoint: str) -> ServiceHealth:
        """Verifica saúde de uma API"""
        import requests
        
        start_time = time.time()
        try:
            response = requests.get(
                endpoint, 
                timeout=2,  # Timeout explícito
                headers={'User-Agent': 'Health-Core/1.0'}
            )
            latency_ms = (time.time() - start_time) * 1000
            
            if response.status_code == 200:
                return ServiceHealth(
                    name=service_name,
                    service_type=ServiceType.API,
                    status=HealthStatus.HEALTHY,
                    latency_ms=latency_ms,
                    error_rate=0.0,
                    last_check=datetime.now(),
                    details={"status_code": response.status_code, "response_time": f"{latency_ms:.2f}ms"}
                )
            else:
                return ServiceHealth(
                    name=service_name,
                    service_type=ServiceType.API,
                    status=HealthStatus.DEGRADED,
                    latency_ms=latency_ms,
                    error_rate=100.0,
                    last_check=datetime.now(),
                    details={"status_code": response.status_code, "reason": "non_200_response"}
                )
        
        except requests.exceptions.Timeout:
            return ServiceHealth(

# DIYAPP Evolution - V12 Core - Arquitetura de Microsserviços

## Arquitetura de Microsserviços V12 - Documento Técnico

### ADR-001: Arquitetura de Microsserviços V12
**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista Backend

**CONTEXTO:**
DIYAPP V11 opera como monólito modular, limitando escalabilidade horizontal e deploy independente. Necessitamos evoluir para arquitetura de microsserviços que suporte:
- Deploy independente por funcionalidade
- Escalabilidade granular
- Resiliência através de circuit breakers
- Observabilidade distribuída
- 100% de autonomia operacional

**DECISÃO:**
Implementar arquitetura baseada em 8 microsserviços core, comunicação via gRPC para síncrono e Apache Kafka para assíncrono, com Core de Autocura como orquestrador central.

**OPÇÕES CONSIDERADAS:**
- **Opção A:** Monólito modular com módulos carregáveis
  - Prós: Simplicidade de deploy, debugging facilitado
  - Contras: Escalabilidade limitada, acoplamento técnico, downtime total em falhas
- **Opção B:** Microsserviços completos com Service Mesh (Istio)
  - Prós: Observabilidade avançada, balanceamento inteligente
  - Contras: Complexidade operacional, overhead de performance, curva de aprendizado
- **Opção C:** Microsserviços híbridos (gRPC + Eventos) com autocura
  - Prós: Balanceamento entre autonomia e simplicidade, resiliência nativa
  - Contras: Necessidade de orquestração de eventos

**Opção escolhida: C** — Justificativa: Atende requisitos de autonomia com complexidade controlada. Core de Autocura reduz necessidade de intervenção humana.

**CONSEQUÊNCIAS:**
**Positivas:**
- Deploy independente por serviço
- Escalabilidade granular (ex: processamento de IA escala separadamente)
- Isolamento de falhas
- Evolução tecnológica independente por serviço

**Negativas:**
- Complexidade de debugging distribuído
- Latência de rede entre serviços
- Gerenciamento de consistência eventual

**Riscos:**
- Cascata de falhas sem circuit breakers adequados
- Perda de mensagens em eventos
- Mitigação: Implementar dead letter queues e retry com backoff exponencial

DesignTokens.validateAccessibility = function() {
  const issues = [];
  
  // Check color contrast ratios (WCAG AA)
  const contrastChecks = [
    { foreground: this.colors.text.primary, background: this.colors.background.base, minRatio: 4.5 },
    { foreground: this.colors.text.secondary, background: this.colors.background.base, minRatio: 4.5 },
    { foreground: this.colors.semantic.success.base, background: this.colors.background.base, minRatio: 4.5 },
    { foreground: this.colors.semantic.error.base, background: this.colors.background.base, minRatio: 4.5 },
  ];
  
  // Simple contrast calculation (simplified)
  function getLuminance(r, g, b) {
    const [rs, gs, bs] = [r, g, b].map(c => {
      c = c / 255;
      return c <= 0.03928 ? c / 12.92 : Math.pow((c + 0.055) / 1.055, 2.4);
    });
    return 0.2126 * rs + 0.7152 * gs + 0.0722 * bs;
  }
  
  function hexToRgb(hex) {
    const result = /^#?([a-f\d]{2})([a-f\d]{2})([a-f\d]{2})$/i.exec(hex);
    return result ? {
      r: parseInt(result[1], 16),
      g: parseInt(result[2], 16),
      b: parseInt(result[3], 16)
    } : null;
  }
  
  contrastChecks.forEach(check => {
    const fg = hexToRgb(check.foreground);
    const bg = hexToRgb(check.background);
    
    if (fg && bg) {
      const luminance1 = getLuminance(fg.r, fg.g, fg.b);
      const luminance2 = getLuminance(bg.r, bg.g, bg.b);
      const brightest = Math.max(luminance1, luminance2);
      const darkest = Math.min(luminance1, luminance2);
      const ratio = (brightest + 0.05) / (darkest + 0.05);
      
      if (ratio < check.minRatio) {
        issues.push(`Contrast ratio ${ratio.toFixed(2)}:1 for ${check.foreground} on ${check.background} (min ${check.minRatio}:1 required)`);
      }
    }
  });
  
  return {
    valid: issues.length === 0,
    issues,
    timestamp: new Date().toISOString()
  };
};

class DatabaseService:
    """Database service with connection pooling"""
    
    def __init__(self):
        self.engine = engine
        self.SessionLocal = SessionLocal
    
    def get_db(self):
        """Dependency for FastAPI"""
        db = self.SessionLocal()
        try:
            yield db
        finally:
            db.close()
    
    async def check_connection(self) -> Tuple[bool, float, Optional[str]]:
        """Check database connection with timeout"""
        start_time = time.time()
        try:
            # Use asyncio to run sync code with timeout
            loop = asyncio.get_event_loop()
            db = self.SessionLocal()
            result = await loop.run_in_executor(
                None, 
                lambda: db.execute("SELECT 1").fetchone()
            )
            db.close()
            latency = (time.time() - start_time) * 1000
            
            if result and result[0] == 1:
                return True, latency, None
            else:
                return False, latency, "Invalid response from database"
        except Exception as e:
            latency = (time.time() - start_time) * 1000
            return False, latency, str(e)

class CircuitBreakerService:
    """Circuit breaker service with persistence"""
    
    def __init__(self, db_session: Session):
        self.db = db_session
    
    @circuit(
        failure_threshold=config.CB_FAILURE_THRESHOLD,
        recovery_timeout=config.CB_RECOVERY_TIMEOUT,
        expected_exception=config.CB_EXPECTED_EXCEPTION
    )
    async def call_with_circuit(
        self, 
        service_name: str,
        callable_func,
        *args,
        **kwargs
    ) -> Any:
        """Call a function with circuit breaker protection"""
        try:
            result = await callable_func(*args, **kwargs)
            self._record_success(service_name)
            return result
        except Exception as e:
            self._record_failure(service_name, str(e))
            raise
    
    def _record_success(self, service_name: str):
        """Record successful call"""
        circuit_state = self.db.query(CircuitState).filter(
            CircuitState.service_name == service_name
        ).first()
        
        if circuit_state:
            circuit_state.state = 'closed'
            circuit_state.failure_count = 0
            circuit_state.success_count += 1
            circuit_state.updated_at = datetime.utcnow()
        else:
            circuit_state = CircuitState(
                service_name=service_name,
                state='closed',
                success_count=1
            )
            self.db.add(circuit_state)
        
        self.db.commit()
        CIRCUIT_STATE.labels(service_name=service_name).set(0)  # 0 = closed
    
    def _record_failure(self, service_name: str, error: str):
        """Record failed call"""
        circuit_state = self.db.query(CircuitState).filter(
            CircuitState.service_name == service_name
        ).first()
        
        if circuit_state:
            circuit_state.failure_count += 1
            circuit_state.last_failure = datetime.utcnow()
            
            if circuit_state.failure_count >= config.CB_FAILURE_THRESHOLD:
                circuit_state.state = 'open'
                circuit_state.opened_at = datetime.utcnow()
                CIRCUIT_STATE.labels(service_name=service_name).set(1)  # 1 = open
        else:
            circuit_state = CircuitState(
                service_name=service_name,
                state='open' if 1 >= config.CB_FAILURE_THRESHOLD else 'closed',
                failure_count=1,
                last_failure=datetime.utcnow(),
                opened_at=datetime.utcnow() if 1 >= config.CB_FAILURE_THRESHOLD else None
            )
            self.db.add(circuit_state)
            state_value = 1 if 1 >= config.CB_FAILURE_THRESHOLD else 0
            CIRCUIT_STATE.labels(service_name=service_name).set(state_value)
        
        self.db.commit()
    
    def get_state(self, service_name: str) -> Optional[CircuitState]:
        """Get current circuit state"""
        return self.db.query(CircuitState).filter(
            CircuitState.service_name == service_name
        ).first()

    async def perform_health_check(
        self, 
        check_type: str,
        include_dependencies: bool = True
    ) -> HealthCheckResponse:
        """Perform comprehensive health check"""
        start_time = time.time()
        
        checks = {}
        overall_status = "healthy"
        
        # System checks
        system_checks = await self._check_system_resources()
        checks.update(system_checks)
        
        # Database check
        db_healthy, db_latency, db_error = await self.db_service.check_connection()
        checks["database"] = {
            "status": "healthy" if db_healthy else "unhealthy",
            "latency_ms": db_latency,
            "error": db_error
        }
        
        if not db_healthy:
            overall_status = "degraded" if overall_status == "healthy" else "unhealthy"
        
        # Dependency checks
        if include_dependencies:
            dependency_checks = await self._check_dependencies()
            checks["dependencies"] = dependency_checks
            
            unhealthy_deps = [
                dep for dep in dependency_checks 
                if dep.get("status") != "healthy"
            ]
            
            if unhealthy_deps:
                if overall_status == "healthy":
                    overall_status = "degraded"
                if len(unhealthy_deps) > 2:  # More than 2 critical deps down
                    overall_status = "unhealthy"
        
        response_time = (time.time() - start_time) * 1000
        
        # Audit log
        self._audit_health_check(
            check_type=check_type,
            status=overall_status,
            response_time_ms=response_time,
            checks=checks
        )
        
        # Update metrics
        status_value = 1.0 if overall_status == "healthy" else 0.5 if overall_status == "degraded" else 0.0
        HEALTH_CHECK_STATUS.labels(check_type=check_type).set(status_value)
        
        return HealthCheckResponse(
            status=overall_status,
            service=config.SERVICE_NAME,
            version=config.VERSION,
            timestamp=datetime.utcnow(),
            response_time_ms=response_time,
            checks=checks
        )

        """Check system resources"""

                "usage_percent": cpu_percent,
                "threshold": config.CPU_THRESHOLD
            }
            
            # Memory
            memory = psutil.virtual_memory()
            checks["memory"] = {
                "status": "healthy" if memory.percent < config.MEMORY_THRESHOLD else "degraded",
                "usage_percent": memory.percent,
                "available_gb": memory.available / (1024**3),
                "threshold": config.MEMORY_THRESHOLD
            }
            
            # Disk
            disk = psutil.disk_usage('/')

class MutualTLSManager:
    """mTLS implementation for inter-service authentication"""
    
    def __init__(self, cert_dir: str = "./certs"):
        self.cert_dir = cert_dir
        os.makedirs(cert_dir, exist_ok=True)
        self.logger = logging.getLogger("mTLS")
    
    def generate_service_certificate(self, service_name: str) -> Dict:
        """Generate RSA key pair and self-signed certificate for a service"""
        
        # Generate private key
        private_key = rsa.generate_private_key(
            public_exponent=65537,
            key_size=2048
        )
        
        # Generate public key
        public_key = private_key.public_key()
        
        # Save private key
        private_pem = private_key.private_bytes(
            encoding=serialization.Encoding.PEM,
            format=serialization.PrivateFormat.PKCS8,
            encryption_algorithm=serialization.NoEncryption()
        )
        
        # Save public key
        public_pem = public_key.public_bytes(
            encoding=serialization.Encoding.PEM,
            format=serialization.PublicFormat.SubjectPublicKeyInfo
        )
        
        # Create service identity
        service_id = f"svc-{service_name}-{uuid.uuid4().hex[:8]}"
        
        # Save to files
        private_key_path = os.path.join(self.cert_dir, f"{service_id}.key")
        public_key_path = os.path.join(self.cert_dir, f"{service_id}.pub")
        
        with open(private_key_path, 'wb') as f:
            f.write(private_pem)
        
        with open(public_key_path, 'wb') as f:
            f.write(public_pem)
        
        # Create certificate metadata
        cert_metadata = {
            "service_id": service_id,
            "service_name": service_name,
            "private_key_path": private_key_path,
            "public_key_path": public_key_path,
            "issued_at": datetime.datetime.utcnow().isoformat(),
            "expires_at": (datetime.datetime.utcnow() + datetime.timedelta(days=365)).isoformat()
        }
        
        self.logger.info(f"Generated certificate for service: {service_name} ({service_id})")
        return cert_metadata
    
    def verify_service_certificate(self, public_key_path: str, signature: str, data: str) -> bool:
        """Verify signature using service's public key"""
        try:
            with open(public_key_path, 'rb') as f:
                public_key = serialization.load_pem_public_key(f.read())
            
            # Verify signature
            public_key.verify(
                base64.b64decode(signature),
                data.encode('utf-8'),
                padding.PSS(
                    mgf=padding.MGF1(hashes.SHA256()),
                    salt_length=padding.PSS.MAX_LENGTH
                ),
                hashes.SHA256()
            )
            return True
        except Exception as e:
            self.logger.error(f"Certificate verification failed: {e}")
            return False
    
    def create_service_token(self, service_id: str, private_key_path: str, audience: str) -> str:
        """Create JWT token for service authentication"""
        
        with open(private_key_path, 'rb') as f:
            private_key = serialization.load_pem_private_key(f.read(), password=None)
        
        # Create JWT payload
        payload = {
            "iss": "diyapp-auth-service",
            "sub": service_id,
            "aud": audience,
            "iat": datetime.datetime.utcnow().timestamp(),
            "exp": (datetime.datetime.utcnow() + datetime.timedelta(minutes=15)).timestamp(),
            "jti": uuid.uuid4().hex,
            "scope": "service:internal"
        }
        
        # Sign with private key
        token = jwt.encode(payload, private_key, algorithm="RS256")
        return token
    
    def validate_service_token(self, token: str, public_key_path: str) -> Dict:
        """Validate service JWT token"""
        try:
            with open(public_key_path, 'rb') as f:
                public_key = serialization.load_pem_public_key(f.read())
            
            payload = jwt.decode(
                token,
                public_key,
                algorithms=["RS256"],
                options={"verify_aud": False}
            )
            
            # Additional validation
            if payload.get("scope") != "service:internal":
                raise ValueError("Invalid token scope")
            
            return payload
        except jwt.ExpiredSignatureError:
            self.logger.error("Service token expired")
            raise
        except Exception as e:
            self.logger.error(f"Token validation failed: {e}")
            raise

      await new Promise(resolve => setTimeout(resolve, 100));

    // Aguarda todas as promises
    await Promise.allSettled(promises);

    });

## 4. Agente de Detecção de Anomalias

**src/agents/anomaly_detector.py**
```python
import numpy as np
import pandas as pd
from datetime import datetime, timedelta
from typing import Dict, List, Optional, Tuple
from sqlalchemy.orm import Session
import logging
from sklearn.ensemble import IsolationForest
from scipy import stats

from src.models.anomaly import Anomaly, AnomalyType, AnomalySeverity
from src.models.metric import LLMMetric
from src.database.queries import get_metrics_window
from src.utils.config import load_thresholds
from src.utils.logger import get_logger

logger = get_logger(__name__)

class AnomalyDetector:
    def __init__(self, db_session: Session):
        self.db = db_session
        self.thresholds = load_thresholds()
        
    def detect_cost_anomalies(self, hours: int = 24) -> List[Anomaly]:
        """Detecta anomalias de custo nas últimas N horas"""
        anomalies = []
        
        # 1. Análise por feature
        features_cost = self._get_cost_by_feature(hours)
        for feature, data in features_cost.items():
            if self._is_cost_spike(data):
                anomaly = self._create_cost_anomaly(feature, data)
                anomalies.append(anomaly)
        
        # 2. Análise por provedor
        providers_cost = self._get_cost_by_provider(hours)
        for provider, data in providers_cost.items():
            if self._is_cost_spike(data):
                anomaly = self._create_cost_anomaly(
                    f"provider:{provider}", data, provider=provider
                )
                anomalies.append(anomaly)
        
        # 3. Detecção de outliers usando Isolation Forest
        token_anomalies = self._detect_token_outliers(hours)
        anomalies.extend(token_anomalies)
        
        return anomalies
    
    def detect_quality_anomalies(self) -> List[Anomaly]:
        """Detecta degradação de qualidade comparando com baseline"""
        anomalies = []
        
        # Buscar resultados mais recentes do golden dataset
        recent_results = self._get_recent_golden_results()
        
        for result in recent_results:
            if abs(result.percentage_change) >= self.thresholds['quality']['degradation_threshold']:
                anomaly = Anomaly(
                    anomaly_type=AnomalyType.QUALITY_DEGRADATION,
                    severity=self._calculate_quality_severity(result.percentage_change),
                    feature=result.feature,
                    model=result.model,
                    current_value=result.overall_score,
                    baseline_value=result.baseline_overall_score,
                    percentage_change=result.percentage_change,
                    metadata={
                        "accuracy_change": result.accuracy_score - (result.baseline_overall_score * 0.25),
                        "relevance_change": result.relevance_score - (result.baseline_overall_score * 0.25),
                        "coherence_change": result.coherence_score - (result.baseline_overall_score * 0.25),
                        "safety_change": result.safety_score - (result.baseline_overall_score * 0.25),
                        "test_cases": result.test_cases_count,
                        "passed_cases": result.passed_cases,
                        "failed_cases": result.failed_cases
                    }
                )
                anomalies.append(anomaly)
        
        return anomalies
    
    def detect_latency_anomalies(self, hours: int = 1) -> List[Anomaly]:
        """Detecta anomalias de latência"""
        anomalies = []
        
        metrics = get_metrics_window(self.db, hours=hours)
        if metrics.empty:
            return anomalies
        
        # Agrupar por provedor e feature
        grouped = metrics.groupby(['provider', 'feature'])
        
        for (provider, feature), group in grouped:
            p95_latency = group['latency_ms'].quantile(0.95)
            
            if p95_latency > self.thresholds['latency']['p95_slo_ms']:

                        "p99_latency": group['latency_ms'].quantile(0.99),
                        "request_count": len(group),
                        "error_rate": (1 - group['success'].mean()) * 100
                    }
                )
                anomalies.append(anomaly)
        
        return anomalies
