# DIYAPP Evolution - V12 Core - Arquitetura de Microsserviços

## ADR-001: Arquitetura de Microsserviços V12

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista Backend

### CONTEXTO:
O DIYAPP evoluiu de uma aplicação monolítica para uma arquitetura de microsserviços para permitir:
- Escalabilidade independente por domínio de negócio
- Deploy contínuo sem downtime
- Autonomia das squads
- Resiliência através de isolamento de falhas

### DECISÃO:
Adotar arquitetura de microsserviços baseada em domínios (DDD) com os seguintes princípios:
1. Cada microsserviço é responsável por um domínio de negócio específico
2. Comunicação via eventos assíncronos (Kafka) para consistência eventual
3. API Gateway único para entrada de requisições HTTP
4. Banco de dados por serviço (Database-per-service)
5. Observabilidade centralizada (logs, métricas, traces)

### OPÇÕES CONSIDERADAS:
- **Opção A**: Monolito modularizado - Prós: Simplicidade operacional, consistência forte. Contras: Acoplamento alto, escalabilidade limitada.
- **Opção B**: Microsserviços completos - Prós: Autonomia total, escalabilidade granular. Contras: Complexidade operacional, consistência eventual.
- **Opção C**: Arquitetura hexagonal com módulos - Prós: Balanceamento entre simplicidade e autonomia. Contras: Compromisso que pode não atender todos os requisitos.

**Opção escolhida: B** - Justificativa: Requisitos de autonomia da squad e escalabilidade independente justificam a complexidade adicional.

### CONSEQUÊNCIAS:
**Positivas:**
- Deploy independente por serviço
- Escala seletiva baseada em carga
- Tecnologias diferentes por domínio
- Falhas isoladas por serviço

**Negativas:**
- Complexidade de operação aumentada
- Consistência eventual em transações distribuídas
- Overhead de comunicação entre serviços
- Monitoramento distribuído necessário

**Riscos:**
- Latência aumentada em chamadas entre serviços
- Debug distribuído complexo
- Gerenciamento de dados distribuídos

## Performance Monitoring Dashboard

```typescript
// services/monitoring/src/dashboard/metrics-collector.ts
import { Counter, Gauge, Histogram, Registry } from 'prom-client';
import { Kafka, Producer } from 'kafkajs';
import { logger } from '../utils/logger';

export interface PerformanceMetrics {
  service: string;
  endpoint: string;
  method: string;
  statusCode: number;
  duration: number;
  timestamp: Date;
  userId?: string;
  correlationId: string;
}

export class MetricsCollector {
  private registry: Registry;
  private requestDuration: Histogram<string>;
  private requestCounter: Counter<string>;
  private errorCounter: Counter<string>;
  private activeRequests: Gauge<string>;
  private kafkaProducer: Producer;

  constructor(serviceName: string) {
    this.registry = new Registry();
    
    // Initialize metrics
    this.requestDuration = new Histogram({
      name: 'http_request_duration_seconds',
      help: 'Duration of HTTP requests in seconds',
      labelNames: ['service', 'method', 'route', 'status_code'],
      buckets: [0.01, 0.05, 0.1, 0.2, 0.5, 1, 2, 5],
      registers: [this.registry],
    });

    this.requestCounter = new Counter({
      name: 'http_requests_total',
      help: 'Total HTTP requests',
      labelNames: ['service', 'method', 'route', 'status_code'],
      registers: [this.registry],
    });

    this.errorCounter = new Counter({
      name: 'http_errors_total',
      help: 'Total HTTP errors',
      labelNames: ['service', 'method', 'route', 'error_type'],
      registers: [this.registry],
    });

    this.activeRequests = new Gauge({
      name: 'http_active_requests',
      help: 'Active HTTP requests',
      labelNames: ['service'],
      registers: [this.registry],
    });

    // Initialize Kafka for metrics streaming
    const kafka = new Kafka({
      clientId: `metrics-${serviceName}`,
      brokers: [process.env.KAFKA_BROKERS || 'localhost:9092'],
    });

    this.kafkaProducer = kafka.producer();
  }

  async start(): Promise<void> {
    await this.kafkaProducer.connect();
    logger.info('Metrics collector started');
  }

  async recordRequest(metrics: PerformanceMetrics): Promise<void> {
    const labels = {
      service: metrics.service,
      method: metrics.method,
      route: metrics.endpoint,
      status_code: metrics.statusCode.toString(),
    };

    // Record to Prometheus
    this.requestDuration.observe(labels, metrics.duration / 1000);
    this.requestCounter.inc(labels);
    
    if (metrics.statusCode >= 400) {
      this.errorCounter.inc({
        ...labels,
        error_type: this.getErrorType(metrics.statusCode),
      });
    }

    // Stream to Kafka for real-time analysis
    await this.kafkaProducer.send({
      topic: 'performance-metrics',
      messages: [{
        key: metrics.correlationId,
        value: JSON.stringify({
          ...metrics,
          duration_ms: metrics.duration,
          p95_target: 200,
          within_sla: metrics.duration <= 200,
        }),
      }],
    });

    // Alert if P95 threshold is breached
    if (metrics.duration > 200) {
      await this.triggerAlert(metrics);
    }
  }

  private getErrorType(statusCode: number): string {
    if (statusCode >= 500) return 'server_error';
    if (statusCode >= 400) return 'client_error';
    return 'unknown';
  }

  private async triggerAlert(metrics: PerformanceMetrics): Promise<void> {
    const alertMessage = {
      type: 'PERFORMANCE_ALERT',
      severity: metrics.duration > 500 ? 'CRITICAL' : 'WARNING',
      service: metrics.service,
      endpoint: metrics.endpoint,
      duration: metrics.duration,
      threshold: 200,
      timestamp: new Date().toISOString(),
      correlationId: metrics.correlationId,
    };

    await this.kafkaProducer.send({
      topic: 'alerts',
      messages: [{ value: JSON.stringify(alertMessage) }],
    });

    logger.warn('Performance alert triggered', alertMessage);
  }

  async getMetrics(): Promise<string> {
    return await this.registry.metrics();
  }

  async getP95Latency(service: string, endpoint?: string): Promise<number> {
    // Query Prometheus for P95 latency
    // This would typically query an external Prometheus server
    const query = endpoint 
      ? `histogram_quantile(0.95, sum(rate(http_request_duration_seconds_bucket{service="${service}",route="${endpoint}"}[5m])) by (le))`
      : `histogram_quantile(0.95, sum(rate(http_request_duration_seconds_bucket{service="${service}"}[5m])) by (le))`;
    
    // In production, this would make an HTTP request to Prometheus
    return this.queryPrometheus(query);
  }

  private async queryPrometheus(query: string): Promise<number> {
    // Mock implementation - in production, use prometheus API
    const response = await fetch(`${process.env.PROMETHEUS_URL}/api/v1/query?query=${encodeURIComponent(query)}`);
    const data = await response.json();
    return parseFloat(data.data.result[0]?.value[1] || '0');
  }
}

// Express middleware for automatic metrics collection
export function metricsMiddleware(serviceName: string) {
  const collector = new MetricsCollector(serviceName);
  
  return async (req: any, res: any, next: any) => {
    const start = Date.now();
    const correlationId = req.headers['x-correlation-id'] || crypto.randomUUID();
    
    // Store correlation ID for logging
    req.correlationId = correlationId;
    
    // Track active requests
    collector.activeRequests.inc({ service: serviceName });
    
    // Hook into response finish
    res.on('finish', async () => {
      const duration = Date.now() - start;
      
      await collector.recordRequest({
        service: serviceName,
        endpoint: req.route?.path || req.path,
        method: req.method,
        statusCode: res.statusCode,
        duration,
        timestamp: new Date(),
        userId: req.user?.id,
        correlationId,
      });
      
      collector.activeRequests.dec({ service: serviceName });
    });
    
    next();
  };
}
```

```javascript
// Validador de contraste WCAG
export function checkContrast(color1, color2) {
  // Implementação simplificada - em produção usar biblioteca como chroma-js
  const luminance = (hex) => {
    const rgb = parseInt(hex.slice(1), 16);
    const r = (rgb >> 16) & 0xff;
    const g = (rgb >> 8) & 0xff;
    const b = (rgb >> 0) & 0xff;
    const [rs, gs, bs] = [r, g, b].map(c => {
      c = c / 255;
      return c <= 0.03928 ? c / 12.92 : Math.pow((c + 0.055) / 1.055, 2.4);
    });
    return 0.2126 * rs + 0.7152 * gs + 0.0722 * bs;
  };
  
  const l1 = luminance(color1) + 0.05;
  const l2 = luminance(color2) + 0.05;
```

import redis
from redis.exceptions import RedisError, ConnectionError, TimeoutError
from typing import Any, Optional, Union
import json
import pickle
from functools import wraps
import time
import logging
from circuitbreaker import circuit

logger = logging.getLogger(__name__)

class CircuitBreakerException(Exception):
    """Exceção quando circuit breaker está aberto"""
    pass

class RedisCache:
    """Cliente Redis com circuit breaker, retry e fallback"""
    
    def __init__(self, settings: Settings):
        self.settings = settings
        self.client: Optional[redis.Redis] = None
        self._connect()
        self.circuit_failure_threshold = 5
        self.circuit_recovery_timeout = 60
        self.circuit_state = "CLOSED"
        self.circuit_failures = 0
        self.circuit_last_failure_time = 0
        
    def _connect(self):
        """Estabelece conexão com Redis"""
        try:
            self.client = redis.from_url(
                self.settings.redis_url,
                socket_timeout=self.settings.redis_connection_timeout,
                socket_connect_timeout=self.settings.redis_connection_timeout,
                decode_responses=True,
                health_check_interval=30
            )
            # Testa conexão
            self.client.ping()
            logger.info("Redis conectado com sucesso")
            self.circuit_state = "CLOSED"
            self.circuit_failures = 0
        except (ConnectionError, TimeoutError) as e:
            logger.error(f"Falha na conexão Redis: {e}")
            self.circuit_state = "OPEN"
            self.circuit_last_failure_time = time.time()
            self.client = None
    
    @circuit(failure_threshold=5, recovery_timeout=60)
    def get(self, key: str) -> Optional[Any]:
        """Obtém valor do cache com circuit breaker"""
        if self.circuit_state == "OPEN":
            if time.time() - self.circuit_last_failure_time > self.circuit_recovery_timeout:
                self.circuit_state = "HALF_OPEN"
                self._connect()
            else:
                raise CircuitBreakerException("Circuit breaker aberto para Redis")
        
        if not self.client:
            self._connect()
            if not self.client:
                return None
        
        try:
            value = self.client.get(key)
            if value:
                try:
                    return json.loads(value)
                except json.JSONDecodeError:
                    return pickle.loads(value.encode('latin1'))
            return None
        except (RedisError, ConnectionError, TimeoutError) as e:
            self._handle_redis_error(e)
            return None
    
    @circuit(failure_threshold=5, recovery_timeout=60)
    def set(self, key: str, value: Any, ttl: Optional[int] = None) -> bool:
        """Define valor no cache com circuit breaker"""
        if self.circuit_state == "OPEN":
            raise CircuitBreakerException("Circuit breaker aberto para Redis")
        
        if not self.client:
            self._connect()
            if not self.client:
                return False
        
        try:
            if isinstance(value, (dict, list, tuple)):
                serialized = json.dumps(value)
            else:
                serialized = pickle.dumps(value).decode('latin1')
            
            ttl = ttl or self.settings.redis_cache_ttl
            result = self.client.setex(key, ttl, serialized)
            
            if self.circuit_state == "HALF_OPEN":
                self.circuit_state = "CLOSED"
                self.circuit_failures = 0
            
            return bool(result)
        except (RedisError, ConnectionError, TimeoutError) as e:
            self._handle_redis_error(e)
            return False
    
    def delete(self, key: str) -> bool:
        """Remove valor do cache"""
        if not self.client:
            return False
        
        try:
            return bool(self.client.delete(key))
        except RedisError:
            return False
    
    def clear_pattern(self, pattern: str) -> int:
        """Remove múltiplas chaves por pattern"""
        if not self.client:
            return 0
        
        try:
            keys = self.client.keys(pattern)
            if keys:
                return self.client.delete(*keys)
            return 0
        except RedisError:
            return 0
    
    def _handle_redis_error(self, error: Exception):
        """Trata erros do Redis e gerencia circuit breaker"""
        self.circuit_failures += 1
        self.circuit_last_failure_time = time.time()
        
        if self.circuit_failures >= self.circuit_failure_threshold:
            self.circuit_state = "OPEN"
            logger.error(f"Circuit breaker aberto para Redis após {self.circuit_failures} falhas")
        
        logger.error(f"Erro Redis: {error}")

def cache_response(ttl: int = 300, key_prefix: str = "cache"):
    """Decorator para cache de respostas de API"""
    def decorator(func):
        @wraps(func)
        async def wrapper(*args, **kwargs):
            # Gera chave única baseada nos argumentos
            cache_key = f"{key_prefix}:{func.__name__}:{str(args)}:{str(kwargs)}"
            
            # Tenta obter do cache
            cache = RedisCache(get_settings())
            cached = cache.get(cache_key)
            
            if cached is not None:
                logger.info(f"Cache hit para {cache_key}")
                return cached
            
            # Executa função se cache miss
            result = await func(*args, **kwargs)
            
            # Armazena no cache
            cache.set(cache_key, result, ttl)
            
            return result
        return wrapper
    return decorator

```typescript
// src/store/slices/aiSlice.ts
import { createSlice, createAsyncThunk, PayloadAction } from '@reduxjs/toolkit'
import { z } from 'zod'

const AIResponseSchema = z.object({
  id: z.string(),
  content: z.string(),
  model: z.string(),
  tokens: z.number(),
  timestamp: z.number()
})

export type AIResponse = z.infer<typeof AIResponseSchema>

interface AIState {
  responses: AIResponse[]
  streamingData: string | null
  isLoading: boolean
  error: string | null
  currentModel: string
  temperature: number
  maxTokens: number
}

const initialState: AIState = {
  responses: [],
  streamingData: null,
  isLoading: false,
  error: null,
  currentModel: 'gpt-4',
  temperature: 0.7,
  maxTokens: 2000
}

export const generateAIResponse = createAsyncThunk(
  'ai/generateResponse',
  async (prompt: string, { getState, dispatch }) => {
    const state = getState() as { ai: AIState }
    const { currentModel, temperature, maxTokens } = state.ai
    
    // Simulate streaming response
    return new Promise<string>((resolve) => {
      let accumulated = ''
      const words = prompt.split(' ').reverse()
      
      const stream = setInterval(() => {
        if (words.length > 0) {
          const word = words.pop() + ' '
          accumulated += word
          dispatch(streamChunk(word))
        } else {
          clearInterval(stream)
          resolve(accumulated)
        }
      }, 50)
    })
  }
)

const aiSlice = createSlice({
  name: 'ai',
  initialState,
  reducers: {
    streamChunk: (state, action: PayloadAction<string>) => {
      state.streamingData = (state.streamingData || '') + action.payload
    },
    clearStream: (state) => {
      state.streamingData = null
    },
    setModel: (state, action: PayloadAction<string>) => {
      state.currentModel = action.payload
    },
    setTemperature: (state, action: PayloadAction<number>) => {
      state.temperature = Math.max(0, Math.min(1, action.payload))
    },
    setMaxTokens: (state, action: PayloadAction<number>) => {
      state.maxTokens = action.payload
    },
    clearError: (state) => {
      state.error = null
    }
  },
  extraReducers: (builder) => {
    builder
      .addCase(generateAIResponse.pending, (state) => {
        state.isLoading = true
        state.error = null
        state.streamingData = ''
      })
      .addCase(generateAIResponse.fulfilled, (state, action) => {
        state.isLoading = false
        state.responses.unshift({
          id: Date.now().toString(),
          content: action.payload,
          model: state.currentModel,
          tokens: action.payload.length / 4,
          timestamp: Date.now()
        })
        state.streamingData = null
      })
      .addCase(generateAIResponse.rejected, (state, action) => {
        state.isLoading = false
        state.error = action.error.message || 'AI generation failed'
        state.streamingData = null
      })
  }
})

export const {
  streamChunk,
  clearStream,
  setModel,
  setTemperature,
  setMaxTokens,
  clearError
} = aiSlice.actions

export default aiSlice.reducer
```

```sql
        LAG(COUNT(DISTINCT user_id)) OVER (ORDER BY event_date) as previous_day_users

        ELSE NULL
```

```sql
        SUM(CASE WHEN event_name LIKE '%_success' THEN 1 ELSE 0 END) as success_events,
        SUM(CASE WHEN event_name LIKE '%_error' THEN 1 ELSE 0 END) as error_events
    FROM {{ ref('stg_app_events') }}
    WHERE feature_name IS NOT NULL

    SELECT
        event_date,
        COUNT(DISTINCT user_id) as total_daily_users
    FROM {{ ref('stg_app_events') }}
    WHERE event_date >= CURRENT_DATE - INTERVAL '30 days'
    GROUP BY 1
)

    fu.event_date,
    fu.feature_name,
    fu.unique_users,
    fu.unique_sessions,
    fu.total_events,
    fu.success_events,
    fu.error_events,
    tdu.total_daily_users,
    ROUND(fu.unique_users::DECIMAL / tdu.total_daily_users * 100, 2) as adoption_rate,
    CASE 
        WHEN fu.total_events > 0 
        THEN ROUND(fu.success_events::DECIMAL / fu.total_events * 100, 2)
        ELSE 0

    ROW_NUMBER() OVER (PARTITION BY fu.event_date ORDER BY fu.unique_users DESC) as daily_rank
```

```sql
WITH hourly_costs AS (
    SELECT
        DATE_TRUNC('hour', request_timestamp) as hour_start,
        feature_name,
        model_name,
        provider,
        COUNT(*) as total_calls,
        SUM(CASE WHEN success THEN 1 ELSE 0 END) as successful_calls,
        SUM(input_tokens) as total_input_tokens,
        SUM(output_tokens) as total_output_tokens,
        SUM(total_tokens) as total_tokens,
        SUM(cost_usd) as total_cost_usd,
        AVG(latency_ms) as avg_latency_ms,
        PERCENTILE_CONT(0.95) WITHIN GROUP (ORDER BY latency_ms) as p95_latency_ms
    FROM {{ ref('stg_llm_calls') }}
    WHERE request_timestamp >= CURRENT_DATE - INTERVAL '7 days'
    GROUP BY 1, 2, 3, 4
),

daily_aggregation AS (
    SELECT
        DATE(hour_start) as cost_date,
        feature_name,
        model_name,
        provider,
        SUM(total_calls) as daily_calls,
        SUM(successful_calls) as daily_successful_calls,
        SUM(total_input_tokens) as daily_input_tokens,
        SUM(total_output_tokens) as daily_output_tokens,
        SUM(total_tokens) as daily_tokens,
        SUM(total_cost_usd) as daily_cost_usd,
        AVG(avg_latency_ms) as daily_avg_latency,
        AVG(p95_latency_ms) as daily_p95_latency
    FROM hourly_costs
    GROUP BY 1, 2, 3, 4
)

SELECT
    cost_date,
    feature_name,
    model_name,
    provider,
    daily_calls,
    daily_successful_calls,
    daily_input_tokens,
    daily_output_tokens,
    daily_tokens,
    daily_cost_usd,
    ROUND(daily_cost_usd::DECIMAL / NULLIF(daily_calls, 0), 6) as cost_per_call,
    ROUND(daily_cost_usd::DECIMAL / NULLIF(daily_tokens, 0), 8) as cost_per_token,
    daily_avg_latency,
    daily_p95_latency,
    CASE 
        WHEN daily_calls > 0 
        THEN ROUND(daily_successful_calls::DECIMAL / daily_calls * 100, 2)
        ELSE 0 
    END as success_rate,
    SUM(daily_cost_usd) OVER (PARTITION BY cost_date, feature_name ORDER BY cost_date) as cumulative_daily_cost_by_feature
FROM daily_aggregation
ORDER BY cost_date DESC, daily_cost_usd DESC
```

DECISÃO:
Adotar arquitetura de microsserviços com os seguintes princípios:

2. Comunicação via HTTP/REST para síncrono, RabbitMQ para assíncrono
3. Banco de dados por serviço (exceto dados compartilhados essenciais)
4. API Gateway como ponto único de entrada
5. Service Discovery via Kubernetes Services

- Opção A: Monólito modularizado - Prós: Simples, menos overhead. 
  Contras: Acoplamento alto, deploy único, escala vertical apenas.
- Opção B: Microsserviços completos - Prós: Escala independente,

- Opção C: Arquitetura híbrida - Prós: Balanceamento.

```javascript
// Função para verificar contraste WCAG
export function checkContrast(color1, color2) {
  // Implementação simplificada - em produção usar biblioteca como chroma.js
  const luminance = (hex) => {
    const rgb = parseInt(hex.slice(1), 16);
    const r = (rgb >> 16) & 0xff;
    const g = (rgb >> 8) & 0xff;
    const b = (rgb >> 0) & 0xff;
    const [rs, gs, bs] = [r, g, b].map(c => {
      c = c / 255;
      return c <= 0.03928 ? c / 12.92 : Math.pow((c + 0.055) / 1.055, 2.4);
    });
    return 0.2126 * rs + 0.7152 * gs + 0.0722 * bs;
  };
  
  const l1 = luminance(color1);
  const l2 = luminance(color2);
  const contrast = (Math.max(l1, l2) + 0.05) / (Math.min(l1, l2) + 0.05);
  
  return {
    ratio: contrast.toFixed(2),
    aa: contrast >= 4.5,
    aaLarge: contrast >= 3,
    aaa: contrast >= 7
  };
}
```

        """Context manager for database sessions with automatic cleanup"""
        if not self._session_factory:
            raise RuntimeError("Database not connected. Call connect() first.")
        
        session: Session = self._session_factory()
        try:
            yield session
            session.commit()
        except SQLAlchemyError as e:

            raise
        finally:
            session.close()

        except Exception as e:
            logger.error(f"Database health check failed: {str(e)}")
            return False

        """Close all database connections"""
        if self._engine:
            self._engine.dispose()

class RedisCache:
    def __init__(self, settings: Optional[CacheSettings] = None):
        self.settings = settings or CacheSettings()
        self._pool: Optional[ConnectionPool] = None
        self._client: Optional[Redis] = None
        self.circuit_breaker = CircuitBreaker(
            name="redis_cache",
            failure_threshold=5,
            recovery_timeout=30
        )
        
    def connect(self):
        """Initialize Redis connection pool"""
        try:
            self._pool = ConnectionPool.from_url(
                self.settings.redis_url,
                max_connections=self.settings.max_connections,
                socket_connect_timeout=self.settings.connection_timeout,
                socket_timeout=self.settings.socket_timeout,
                health_check_interval=self.settings.health_check_interval,
                decode_responses=True
            )
            
            self._client = Redis(connection_pool=self._pool)
            
            # Test connection
            self._client.ping()
            
            logger.info("Redis cache connection established successfully")
            
        except (ConnectionError, TimeoutError) as e:
            logger.error(f"Redis connection failed: {str(e)}")
            raise
        except Exception as e:
            logger.error(f"Unexpected Redis error: {str(e)}")
            raise
    
    @circuit_breaker.wrap
    def get(self, key: str) -> Optional[Any]:
        """Get value from cache with circuit breaker protection"""
        if not self._client:
            raise RuntimeError("Cache not connected. Call connect() first.")
        
        try:
            value = self._client.get(key)
            if value:
                return json.loads(value)
            return None
        except RedisError as e:
            logger.error(f"Redis GET error for key {key}: {str(e)}")
            raise
    
    @circuit_breaker.wrap
    def set(self, key: str, value: Any, ttl: Optional[int] = None) -> bool:
        """Set value in cache with TTL"""
        if not self._client:
            raise RuntimeError("Cache not connected. Call connect() first.")
        
        try:
            serialized = json.dumps(value)
            expire = ttl or self.settings.default_ttl
            return self._client.setex(key, expire, serialized)
        except RedisError as e:
            logger.error(f"Redis SET error for key {key}: {str(e)}")
            raise
    
    @circuit_breaker.wrap

        """Delete key from cache"""
        if not self._client:

        try:

import React, { Component, ErrorInfo, ReactNode } from 'react';

interface ErrorBoundaryProps {
  children: ReactNode;
  fallback?: ReactNode;
  onError?: (error: Error, errorInfo: ErrorInfo) => void;
}

interface ErrorBoundaryState {
  hasError: boolean;
  error?: Error;
}

export class ErrorBoundary extends Component<ErrorBoundaryProps, ErrorBoundaryState> {
  constructor(props: ErrorBoundaryProps) {
    super(props);
    this.state = { hasError: false };
  }

  static getDerivedStateFromError(error: Error): ErrorBoundaryState {
    return { hasError: true, error };
  }

  componentDidCatch(error: Error, errorInfo: ErrorInfo): void {
    console.error('ErrorBoundary caught an error:', error, errorInfo);
    this.props.onError?.(error, errorInfo);
    
    // Log para serviço de monitoramento
    if (typeof window !== 'undefined' && (window as any).Sentry) {
      (window as any).Sentry.captureException(error);
    }
  }

  render(): ReactNode {
    if (this.state.hasError) {
      if (this.props.fallback) {
        return this.props.fallback;
      }

      return (
        <div 
          className="error-boundary-fallback"
          role="alert"
          aria-live="polite"
        >
          <h2>Something went wrong</h2>
          <p>Please try refreshing the page.</p>
          <button 
            onClick={() => window.location.reload()}
            className="retry-button"
          >
            Retry
          </button>
          <details>
            <summary>Error details</summary>
            <pre>{this.state.error?.toString()}</pre>
          </details>
        </div>
      );
    }

    return this.props.children;
  }
}

## 4. dbt/models/staging/stg_events.sql

```sql
{{
    config(
        materialized='view',
        schema='staging'
    )
}}

WITH raw_events AS (
    SELECT
        event_id,
        user_id,
        event_type,
        feature_name,
        session_id,
        timestamp::timestamp AS timestamp,
        properties::jsonb AS properties,
        ingested_at::timestamp AS ingested_at
    FROM {{ source('raw', 'events') }}
    WHERE timestamp >= '{{ var("start_date") }}'
),

cleaned_events AS (
    SELECT
        event_id,
        COALESCE(user_id, 'anonymous') AS user_id,
        LOWER(TRIM(event_type)) AS event_type,
        CASE 
            WHEN feature_name IS NULL THEN 'unknown'
            ELSE LOWER(TRIM(feature_name))
        END AS feature_name,
        session_id,
        timestamp,
        properties,
        ingested_at,
        DATE(timestamp) AS event_date
    FROM raw_events
)

SELECT * FROM cleaned_events
```

```sql

WITH raw_calls AS (
    SELECT
        call_id,
        request_id,
        user_id,
        model,
        provider,
        feature,
        prompt_tokens::integer AS prompt_tokens,
        completion_tokens::integer AS completion_tokens,

        timestamp::timestamp AS timestamp,
        success::boolean AS success,
        error_message,
        ingested_at::timestamp AS ingested_at
    FROM {{ source('raw', 'llm_calls') }}
    WHERE timestamp >= '{{ var("start_date") }}'
),

cost_calculated AS (
    SELECT
        *,
        CASE 
            WHEN model = 'gpt-4' THEN total_tokens * 0.03 / 1000
            WHEN model = 'gpt-3.5-turbo' THEN total_tokens * 0.002 / 1000
            WHEN model = 'claude-2' THEN total_tokens * 0.01102 / 1000
            WHEN model = 'llama-2-70b' THEN total_tokens * 0.00195 / 1000
            ELSE total_tokens * 0.002 / 1000  -- default
        END AS cost_usd
    FROM raw_calls
)

SELECT * FROM cost_calculated
```

## 6. dbt/models/marts/product/daily_active_users.sql

```sql
{{
    config(
        materialized='table',
        schema='product_marts',
        tags=['daily', 'product', 'kpi']
    )
}}

WITH daily_users AS (
    SELECT
        DATE(timestamp) AS date,
        user_id,
        COUNT(DISTINCT session_id) AS sessions,
        COUNT(*) AS events,
        COUNT(DISTINCT feature_name) AS features_used
    FROM {{ ref('stg_events') }}
    WHERE user_id != 'anonymous'
    GROUP BY 1, 2
),

daily_aggregates AS (
    SELECT
        date,
        COUNT(DISTINCT user_id) AS active_users,
        COUNT(DISTINCT CASE WHEN features_used >= 3 THEN user_id END) AS power_users,
        SUM(sessions) AS total_sessions,
        SUM(events) AS total_events,
        AVG(sessions) AS avg_sessions_per_user,
        AVG(events) AS avg_events_per_user
    FROM daily_users
    GROUP BY 1
),

rolling_metrics AS (
    SELECT
        date,
        active_users,
        power_users,
        total_sessions,
        total_events,
        avg_sessions_per_user,
        avg_events_per_user,
        AVG(active_users) OVER (ORDER BY date ROWS BETWEEN 6 PRECEDING AND CURRENT ROW) AS weekly_active_users_avg,
        AVG(power_users) OVER (ORDER BY date ROWS BETWEEN 6 PRECEDING AND CURRENT ROW) AS weekly_power_users_avg
    FROM daily_aggregates
)

SELECT 
    *,
    CASE 
        WHEN active_users < LAG(active_users, 7) OVER (ORDER BY date) * 0.9 
        THEN true 
        ELSE false 
    END AS needs_attention
FROM rolling_metrics
ORDER BY date DESC
```
