## ADR-001: Arquitetura de Microsserviços

```markdown
# ADR-001: Arquitetura de Microsserviços para DIYAPP V12

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra

## CONTEXTO:
DIYAPP V11 está enfrentando problemas de escalabilidade devido à arquitetura monolítica. 
Necessitamos de uma arquitetura que permita:
1. Desenvolvimento paralelo por múltiplas squads (Modo Hive)
2. Deploy independente de funcionalidades
3. Escalabilidade horizontal por componente
4. Resiliência a falhas
5. 100% de autonomia operacional

## DECISÃO:
Adotar arquitetura de microsserviços com os seguintes princípios:
- Cada microsserviço é responsável por um domínio de negócio específico
- Serviços são independentes em deploy e escalabilidade
- Comunicação via gRPC para chamadas síncronas e eventos para assíncronas
- API Gateway como ponto único de entrada
- Service Mesh (Istio) para observabilidade e resiliência

## OPÇÕES CONSIDERADAS:

### Opção A: Monolito Modularizado
- **Prós:** Simplicidade inicial, transações ACID fáceis
- **Contras:** Escalabilidade limitada, deploy único, acoplamento alto
- **Rejeitada:** Não atende requisitos de autonomia squad

### Opção B: Microsserviços Leves (REST)
- **Prós:** Simples implementação, ampla adoção
- **Contras:** Overhead de comunicação, versionamento complexo
- **Rejeitada:** Performance inferior ao gRPC para comunicação interna

### Opção C: Microsserviços com gRPC + Eventos (ESCOLHIDA)
- **Prós:** Alta performance, contrato forte com Protobuf, streaming bidirecional
- **Contras:** Curva de aprendizado, tooling menos maduro que REST
- **Justificativa:** Atende todos requisitos com melhor performance

## CONSEQUÊNCIAS:

### Positivas:
- Desenvolvimento paralelo por squads
- Deploy independente por serviço
- Escalabilidade granular
- Resiliência através de circuit breakers
- Tecnologia adequada por serviço

### Negativas:
- Complexidade distribuída
- Monitoramento mais complexo
- Transações distribuídas (Saga Pattern necessário)
- Latência de rede entre serviços

### Riscos:
1. **Service Mesh overhead:** Mitigar com resource limits adequados
2. **Eventual consistency:** Implementar compensação patterns
3. **Debug distribuído:** Implementar distributed tracing (Jaeger)

## REVISÃO: 2024-04-15
```

## ADR-002: Protocolos de Comunicação

```markdown
# ADR-002: Protocolos de Comunicação entre Microsserviços

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Backend

## CONTEXTO:
Necessidade de comunicação eficiente e confiável entre microsserviços.
Dois cenários principais:
1. Chamadas síncronas (request/response)
2. Comunicação assíncrona (event-driven)

## DECISÃO:
- **gRPC** para comunicação síncrona interna entre serviços
- **Apache Kafka** para eventos assíncronos e streaming
- **REST/GraphQL** apenas para APIs públicas via API Gateway

## OPÇÕES CONSIDERADAS:

### Opção A: REST para tudo
- **Prós:** Simplicidade, tooling maduro
- **Contras:** Overhead JSON, sem streaming nativo, versionamento complexo

### Opção B: gRPC + RabbitMQ
- **Prós:** Mensageria madura, fácil setup
- **Contras:** RabbitMQ não escala como Kafka para streaming

### Opção C: gRPC + Kafka (ESCOLHIDA)
- **Prós:** Alta performance, streaming nativo, escalabilidade comprovada
- **Contras:** Operacional mais complexo, curva de aprendizado

## CONSEQUÊNCIAS:

### Positivas:
- Performance 5-10x melhor que REST
- Contratos fortes com Protobuf
- Streaming bidirecional nativo
- Event sourcing habilitado

### Negativas:
- Necessidade de gRPC gateway para clientes HTTP
- Tooling menos maduro que REST
- Serialização binária (não human-readable)

### Riscos:
1. **gRPC learning curve:** Treinamento obrigatório para devs
2. **Kafka operacional:** Squad Infra dedicada para operação
```

const express = require('express');
const helmet = require('helmet');
const cors = require('cors');
const { createLogger, format, transports } = require('winston');
const { grpcServer } = require('./grpc/server');
const { kafkaProducer, kafkaConsumer } = require('./events/kafka');
const { healthCheck } = require('./health');
const { metricsMiddleware, startMetricsServer } = require('./metrics');
const config = require('./config');

// Logger centralizado
const logger = createLogger({
  level: config.logLevel,
  format: format.combine(
    format.timestamp(),
    format.errors({ stack: true }),
    format.json()
  ),
  transports: [
    new transports.Console(),
    new transports.File({ filename: 'logs/error.log', level: 'error' }),
    new transports.File({ filename: 'logs/combined.log' })
  ]
});

// Graceful shutdown
const shutdown = async (signal) => {
  logger.info(`Recebido ${signal}, iniciando shutdown gracioso...`);
  
  try {
    // Parar novos requests
    server.close();
    
    // Desconectar Kafka
    await kafkaProducer.disconnect();
    await kafkaConsumer.disconnect();
    
    // Parar gRPC server
    grpcServer.tryShutdown(() => {
      logger.info('gRPC server desligado');
    });
    
    logger.info('Shutdown completo');
    process.exit(0);
  } catch (error) {
    logger.error('Erro durante shutdown:', error);
    process.exit(1);
  }
};

process.on('SIGTERM', () => shutdown('SIGTERM'));
process.on('SIGINT', () => shutdown('SIGINT'));

// Inicialização
const app = express();

// Middlewares de segurança
app.use(helmet({
  contentSecurityPolicy: {
    directives: {
      defaultSrc: ["'self'"],
      styleSrc: ["'self'", "'unsafe-inline'"],
      scriptSrc: ["'self'", "'unsafe-inline'"]
    }
  }
}));
app.use(cors(config.corsOptions));
app.use(express.json({ limit: '10mb' }));
app.use(express.urlencoded({ extended: true }));

// Metrics e logging
app.use(metricsMiddleware);
app.use((req, res, next) => {
  logger.info(`${req.method} ${req.url}`, {
    ip: req.ip,
    userAgent: req.get('User-Agent')
  });
  next();
});

// Health checks
app.get('/health', healthCheck);
app.get('/health/live', (req, res) => res.status(200).json({ status: 'UP' }));
app.get('/health/ready', async (req, res) => {
  const checks = await healthCheck();
  res.status(checks.healthy ? 200 : 503).json(checks);
});

// Rotas da API REST (apenas para admin/internal)
app.use('/api/v1', require('./routes'));

// Error handling
app.use((err, req, res, next) => {
  logger.error('Erro não tratado:', {
    error: err.message,
    stack: err.stack,
    path: req.path,
    method: req.method
  });
  
  res.status(err.status || 500).json({
    error: config.isProduction ? 'Internal Server Error' : err.message,
    requestId: req.id
  });
});

// 404 handler
app.use((req, res) => {
  res.status(404).json({
    error: 'Endpoint não encontrado',
    path: req.path
  });
});

// Iniciar servidores
const start = async () => {
  try {
    // Iniciar métricas
    startMetricsServer(config.metricsPort);
    
    // Conectar Kafka
    await kafkaProducer.connect();
    await kafkaConsumer.connect();
    
    // Iniciar gRPC server
    grpcServer.start();
    
    // Iniciar HTTP server
    const server = app.listen(config.port, () => {
      logger.info(`Microsserviço ${config.serviceName} iniciado`, {
        port: config.port,
        env: config.nodeEnv,
        version: config.version
      });
    });
    
    return server;
  } catch (error) {
    logger.error('Falha ao iniciar serviço:', error);
    process.exit(1);
  }
};

// Auto-start se não estiver em teste
if (require.main === module) {
  start();
}

module.exports = { app, start, shutdown, logger };

const grpc = require('@grpc/grpc-js');
const protoLoader = require('@grpc/proto-loader');
const path = require('path');
const logger = require('../utils/logger');
const config = require('../config');

// Load proto file
const PROTO_PATH = path.join(__dirname, '../proto/service.proto');
const packageDefinition = protoLoader.loadSync(PROTO_PATH, {
  keepCase: true,
  longs: String,
  enums: String,
  defaults: true,
  oneofs: true
});

const proto = grpc.loadPackageDefinition(packageDefinition);

// Implementação do serviço gRPC
const serviceImplementation = {
  ProcessRequest: async (call, callback) => {
    try {
      const { requestId, data } = call.request;
      
      logger.info('Processando requisição gRPC', { requestId });
      
      // Business logic aqui
      const result = await processBusinessLogic(data);
      
      callback(null, {
        requestId,
        success: true,
        result: JSON.stringify(result),
        timestamp: new Date().toISOString()
      });
    } catch (error) {
      logger.error('Erro no gRPC:', error);
      callback({
        code: grpc.status.INTERNAL,
        message: error.message
      });
    }
  },
  
  StreamData: (call) => {
    logger.info('Iniciando stream gRPC');
    
    call.on('data', async (request) => {
      try {
        const result = await processStreamChunk(request);
        call.write({
          chunkId: request.chunkId,
          processed: true,
          result
        });
      } catch (error) {
        logger.error('Erro no stream:', error);
        call.write({
          chunkId: request.chunkId,
          error: error.message
        });
      }
    });
    
    call.on('end', () => {
      logger.info('Stream gRPC finalizado');
      call.end();
    });
  }
};

// Health check gRPC
const healthImplementation = {
  Check: (call, callback) => {
    callback(null, { status: 'SERVING' });
  }
};

// Criar e configurar servidor
const server = new grpc.Server();

server.addService(proto.ServiceName.service, serviceImplementation);
server.addService(grpc.health.v1.Health.service, healthImplementation);

// Interceptors para logging e métricas
const loggingInterceptor = (options, nextCall) => {
  return new grpc.InterceptingCall(nextCall(options), {
    start: (metadata, listener, next) => {
      const startTime = Date.now();
      const newListener = {
        onReceiveMessage: (message, next) => {
          const duration = Date.now() - startTime;
          logger.debug('gRPC request completed', {
            method: options.method_definition.path,
            duration,
            metadata: metadata.getMap()
          });
          next(message);
        }
      };
      next(metadata, newListener);
    }
  });
};

server.start = () => {
  server.bindAsync(
    `0.0.0.0:${config.grpcPort}`,
    grpc.ServerCredentials.createInsecure(),
    (error, port) => {
      if (error) {
        logger.error('Falha ao iniciar gRPC server:', error);
        return;
      }
      logger.info(`gRPC server rodando na porta ${port}`);
    }
  );
};

server.tryShutdown = (callback) => {
  const deadline = Date.now() + 10000; // 10 segundos
  server.tryShutdown(() => {
    logger.info('gRPC server desligado graciosamente');
    callback();
  });
  
  setTimeout(() => {
    server.forceShutdown();
    logger.warn('gRPC server forçado a desligar');
  }, deadline - Date.now());
};

module.exports = { grpcServer: server };
```

import { useQuery, useMutation, useQueryClient, UseQueryOptions, UseMutationOptions } from '@tanstack/react-query'
import { toast } from 'react-hot-toast'

const API_BASE_URL = import.meta.env.VITE_API_URL || 'http://localhost:3000/api'

interface ApiError {
  message: string
  code: string
  status: number
}

async function fetcher<T>(endpoint: string, options?: RequestInit): Promise<T> {
  const response = await fetch(`${API_BASE_URL}${endpoint}`, {
    headers: {
      'Content-Type': 'application/json',
      ...options?.headers,
    },
    ...options,
  })

  if (!response.ok) {
    const error: ApiError = await response.json().catch(() => ({
      message: response.statusText,
      code: 'UNKNOWN_ERROR',
      status: response.status,
    }))
    
    throw new Error(error.message || 'API request failed')
  }

  return response.json()
}

export function useApiQuery<T>(
  key: string[],
  endpoint: string,
  options?: Omit<UseQueryOptions<T, Error>, 'queryKey' | 'queryFn'>
) {
  return useQuery<T, Error>({
    queryKey: key,
    queryFn: () => fetcher<T>(endpoint),
    retry: (failureCount, error) => {
      if (error.message.includes('401') || error.message.includes('403')) {
        return false
      }
      return failureCount < 3
    },
    ...options,
  })
}

export function useApiMutation<T, V = unknown>(
  endpoint: string,
  method: 'POST' | 'PUT' | 'DELETE' | 'PATCH' = 'POST',
  options?: Omit<UseMutationOptions<T, Error, V>, 'mutationFn'> & {
    invalidateQueries?: string[][]
  }
) {
  const queryClient = useQueryClient()

  return useMutation<T, Error, V>({
    mutationFn: async (data) => {
      return fetcher<T>(endpoint, {
        method,
        body: JSON.stringify(data),
      })
    },
    onSuccess: (data, variables, context) => {
      if (options?.invalidateQueries) {
        options.invalidateQueries.forEach((queryKey) => {
          queryClient.invalidateQueries({ queryKey })
        })
      }
      
      if (options?.onSuccess) {
        options.onSuccess(data, variables, context)
      }
    },
    onError: (error) => {
      toast.error(error.message)
    },
    ...options,
  })
}

    }
  )

export function useAIStream() {
  const mutation = useApiMutation<AIStreamResponse, AICompletionRequest>(
    '/ai/stream',
    'POST'
  )

  const streamResponse = async (
    request: AICompletionRequest,
    onChunk: (chunk: string) => void,
    onComplete: () => void
  ) => {
    try {
      const response = await fetch(`${import.meta.env.VITE_API_URL}/ai/stream`, {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
        },
        body: JSON.stringify(request),
      })

      if (!response.ok) {
        throw new Error('Stream request failed')
      }

      const reader = response.body?.getReader()
      const decoder = new TextDecoder()

      if (!reader) {
        throw new Error('No reader available')
      }

      while (true) {
        const { done, value } = await reader.read()
        
        if (done) {
          onComplete()
          break
        }

        const chunk = decoder.decode(value)
        onChunk(chunk)
      }
    } catch (error) {
      console.error('Stream error:', error)
      throw error
    }
  }

  return { ...mutation, streamResponse }
}

  return useApiQuery<AIMessage[]>(

import { Pool, PoolConfig } from 'pg';
import { config } from './environment';
import { logger } from './logger';

class Database {
  private static instance: Database;
  private pool: Pool;
  private isConnected = false;

  private constructor() {
    const poolConfig: PoolConfig = {
      connectionString: config.get('DATABASE_URL'),
      max: config.get('DB_POOL_MAX'),
      min: config.get('DB_POOL_MIN'),
      idleTimeoutMillis: 30000,
      connectionTimeoutMillis: 5000,
    };

    this.pool = new Pool(poolConfig);

    this.pool.on('connect', () => {
      logger.info('Database connection established');
      this.isConnected = true;
    });

    this.pool.on('error', (err) => {
      logger.error('Unexpected database error', { error: err.message });
      this.isConnected = false;
    });

    this.pool.on('remove', () => {
      logger.info('Database connection removed');
      this.isConnected = false;
    });
  }

  public static getInstance(): Database {
    if (!Database.instance) {
      Database.instance = new Database();
    }
    return Database.instance;
  }

  public async query<T = any>(
    text: string,
    params?: any[],
    timeout?: number
  ): Promise<T[]> {
    const start = Date.now();
    const client = await this.pool.connect();
    
    try {
      const result = await client.query({
        text,
        values: params,
        rowMode: 'array',
        ...(timeout && { timeout }),
      });
      
      const duration = Date.now() - start;
      logger.debug('Database query executed', {
        query: text,
        duration,
        rowCount: result.rowCount,
      });
      
      return result.rows;
    } catch (error) {
      logger.error('Database query failed', {
        query: text,
        error: error instanceof Error ? error.message : 'Unknown error',
        params: this.maskSensitiveData(params),
      });
      throw error;
    } finally {
      client.release();
    }
  }

  public async transaction<T>(
    callback: (client: any) => Promise<T>
  ): Promise<T> {
    const client = await this.pool.connect();
    
    try {
      await client.query('BEGIN');
      const result = await callback(client);
      await client.query('COMMIT');
      return result;
    } catch (error) {
      await client.query('ROLLBACK');
      throw error;
    } finally {
      client.release();
    }
  }

  public async healthCheck(): Promise<boolean> {
    try {
      await this.query('SELECT 1');
      return true;
    } catch {
      return false;
    }
  }

  private maskSensitiveData(params?: any[]): any[] {
    if (!params) return [];
    
    return params.map(param => {
      if (typeof param === 'string') {
        // Mask emails, tokens, etc.
        if (param.includes('@')) {
          return '[MASKED_EMAIL]';
        }
        if (param.length > 20) {
          return '[MASKED_LONG_STRING]';
        }
      }
      return param;
    });
  }

  public async close(): Promise<void> {
    await this.pool.end();
    this.isConnected = false;
  }
}

import Redis from 'ioredis';
import { config } from './environment';
import { logger } from './logger';

class RedisClient {
  private static instance: RedisClient;
  private client: Redis;
  private subscriber: Redis;
  private isConnected = false;

  private constructor() {
    const redisOptions = {
      maxRetriesPerRequest: 3,
      retryStrategy: (times: number) => {
        const delay = Math.min(times * 50, 2000);
        return delay;
      },
      reconnectOnError: (err: Error) => {
        const targetError = 'READONLY';
        if (err.message.includes(targetError)) {
          return true;
        }
        return false;
      },
    };

    this.client = new Redis(config.get('REDIS_URL'), redisOptions);
    this.subscriber = new Redis(config.get('REDIS_URL'), redisOptions);

    this.client.on('connect', () => {
      logger.info('Redis connected');
      this.isConnected = true;
    });

    this.client.on('error', (err) => {
      logger.error('Redis error', { error: err.message });
      this.isConnected = false;
    });

    this.client.on('close', () => {
      logger.warn('Redis connection closed');
      this.isConnected = false;
    });
  }

  public static getInstance(): RedisClient {
    if (!RedisClient.instance) {
      RedisClient.instance = new RedisClient();
    }
    return RedisClient.instance;
  }

  public async set(
    key: string,
    value: any,
    ttl?: number
  ): Promise<void> {
    const start = Date.now();
    try {
      const serialized = typeof value === 'string' ? value : JSON.stringify(value);
      
      if (ttl) {
        await this.client.setex(key, ttl, serialized);
      } else {
        await this.client.set(key, serialized);
      }
      
      const duration = Date.now() - start;
      logger.debug('Redis set operation', { key, ttl, duration });
    } catch (error) {
      logger.error('Redis set failed', {
        key,
        error: error instanceof Error ? error.message : 'Unknown error',
      });
      throw error;
    }
  }

  public async get<T = any>(key: string): Promise<T | null> {
    const start = Date.now();
    try {
      const value = await this.client.get(key);
      const duration = Date.now() - start;
      
      logger.debug('Redis get operation', { key, duration, hit: value !== null });
      
      if (!value) return null;
      
      try {
        return JSON.parse(value) as T;
      } catch {
        return value as T;
      }
    } catch (error) {
      logger.error('Redis get failed', {
        key,
        error: error instanceof Error ? error.message : 'Unknown error',
      });
      throw error;
    }
  }

  public async del(key: string): Promise<void> {
    await this.client.del(key);
  }

  public async exists(key: string): Promise<boolean> {
    const result = await this.client.exists(key);
    return result === 1;
  }

  public async incr(key: string): Promise<number> {
    return await this.client.incr(key);
  }

  public async expire(key: string, seconds: number): Promise<void> {
    await this.client.expire(key, seconds);
  }

  public async healthCheck(): Promise<boolean> {
    try {
      await this.client.ping();
      return true;
    } catch {
      return false;
    }
  }

  public getClient(): Redis {
    return this.client;
  }

  public getSubscriber(): Redis {
    return this.subscriber;
  }

  public async close(): Promise<void> {
    await this.client.quit();
    await this.subscriber.quit();
    this.isConnected = false;
  }
}

export const redis = RedisClient.getInstance();

// src/security/auth/oauth2.1.js
const crypto = require('crypto');
const jwt = require('jsonwebtoken');
const { createHash, randomBytes } = crypto;

class OAuth21Provider {
  constructor() {
    this.clients = new Map();
    this.authorizationCodes = new Map();
    this.accessTokens = new Map();
    this.refreshTokens = new Map();
    this.pkceVerifiers = new Map();
  }

  // Implementação PKCE (RFC 7636) - OBRIGATÓRIO OAuth2.1
  generatePKCE() {
    const codeVerifier = randomBytes(32).toString('base64url');
    const codeChallenge = createHash('sha256')
      .update(codeVerifier)
      .digest('base64url');
    
    return { codeVerifier, codeChallenge };
  }

  // Fluxo Authorization Code com PKCE
  async authorizeRequest(params) {
    const {
      client_id,
      redirect_uri,
      response_type,
      scope,
      state,
      code_challenge,
      code_challenge_method = 'S256'
    } = params;

    // 1. Validação do cliente
    const client = this.validateClient(client_id, redirect_uri);
    if (!client) {
      throw new SecurityError('Invalid client or redirect URI');
    }

    // 2. Validação PKCE
    if (!code_challenge || !['plain', 'S256'].includes(code_challenge_method)) {
      throw new SecurityError('PKCE code challenge required');
    }

    // 3. Validação de scopes
    const validScopes = this.validateScopes(scope, client.allowed_scopes);
    
    // 4. Geração de código de autorização
    const authCode = randomBytes(32).toString('hex');
    const expiresAt = Date.now() + (10 * 60 * 1000); // 10 minutos
    
    this.authorizationCodes.set(authCode, {
      client_id,
      redirect_uri,
      scope: validScopes,
      user_id: params.user_id,
      code_challenge,
      code_challenge_method,
      expiresAt
    });

    // 5. Store PKCE verifier se necessário
    if (params.code_verifier) {
      this.pkceVerifiers.set(authCode, params.code_verifier);
    }

    return {
      code: authCode,
      state,
      expires_in: 600
    };
  }

  // Token endpoint com validação PKCE
  async exchangeCodeForToken(params) {
    const {
      code,
      client_id,
      client_secret,
      redirect_uri,
      code_verifier
    } = params;

    // 1. Validação básica
    const authCodeData = this.authorizationCodes.get(code);
    if (!authCodeData) {
      throw new SecurityError('Invalid authorization code');
    }

    if (authCodeData.expiresAt < Date.now()) {
      this.authorizationCodes.delete(code);
      throw new SecurityError('Authorization code expired');
    }

    // 2. Validação PKCE
    if (authCodeData.code_challenge_method === 'S256') {
      const calculatedChallenge = createHash('sha256')
        .update(code_verifier)
        .digest('base64url');
      
      if (calculatedChallenge !== authCodeData.code_challenge) {
        throw new SecurityError('PKCE verification failed');
      }
    }

    // 3. Geração de tokens
    const accessToken = this.generateAccessToken({
      client_id: authCodeData.client_id,
      user_id: authCodeData.user_id,
      scope: authCodeData.scope
    });

    const refreshToken = this.generateRefreshToken({
      client_id: authCodeData.client_id,
      user_id: authCodeData.user_id
    });

    // 4. Cleanup
    this.authorizationCodes.delete(code);
    this.pkceVerifiers.delete(code);

    return {
      access_token: accessToken,
      token_type: 'Bearer',
      expires_in: 900, // 15 minutos
      refresh_token: refreshToken,
      scope: authCodeData.scope.join(' ')
    };
  }

  generateAccessToken(payload) {
    // JWT com algoritmo seguro (ES256/RS256)
    return jwt.sign(
      {
        ...payload,
        typ: 'at+jwt', // RFC 9068 - JWT Access Token
        iat: Math.floor(Date.now() / 1000),
        exp: Math.floor(Date.now() / 1000) + 900 // 15 minutos
      },
      process.env.JWT_PRIVATE_KEY,
      { 
        algorithm: 'RS256',
        issuer: process.env.OAUTH_ISSUER,
        audience: process.env.OAUTH_AUDIENCE
      }
    );
  }

  generateRefreshToken(payload) {
    // Refresh token opaco
    const token = randomBytes(64).toString('hex');
    const expiresAt = Date.now() + (7 * 24 * 60 * 60 * 1000); // 7 dias
    
    this.refreshTokens.set(token, {
      ...payload,
      expiresAt,
      used: false
    });

    return token;
  }

  validateClient(clientId, redirectUri) {
    const client = this.clients.get(clientId);
    if (!client) return null;
    
    // Validação exata de redirect_uri (evita open redirect)
    if (!client.redirect_uris.includes(redirectUri)) {
      return null;
    }
    
    return client;
  }

  validateScopes(requestedScopes, allowedScopes) {
    const requested = requestedScopes ? requestedScopes.split(' ') : [];
    const valid = requested.filter(scope => allowedScopes.includes(scope));
    
    if (valid.length === 0) {
      throw new SecurityError('No valid scopes requested');
    }
    
    return valid;
  }
}

            if self.model_type == "lstm":
                X_reshaped = self._create_sequences(X_scaled, sequence_length=10)
                reconstructions = self.model.predict(X_reshaped, verbose=0)
                # Pegar apenas a última reconstrução de cada sequência
                reconstructions = reconstructions[:, -1, :]
                X_scaled = X_scaled[9:]  # Ajustar para corresponder às sequências
            else:
                reconstructions = self.model.predict(X_scaled, verbose=0)
            
            # Calcular erro de reconstrução
            mse = np.mean(np.power(X_scaled - reconstructions, 2), axis=1)
            anomaly_scores = mse
            anomaly_flags = (mse > np.percentile(mse, self.threshold * 100)).astype(int)
        
        return anomaly_scores, anomaly_flags
    
    def evaluate(self, X_scaled: np.ndarray) -> Dict:
        """Avalia o modelo"""
        if self.model_type == "isolation_forest":
            scores = self.model.decision_function(X_scaled)
            return {
                "mean_score": float(np.mean(scores)),
                "std_score": float(np.std(scores)),
                "contamination": float(np.mean(scores < 0))
            }
        else:
            return {"model_type": self.model_type, "is_trained": self.is_trained}
    
    def save(self, path: str):
        """Salva o modelo"""
        model_data = {
            "model_type": self.model_type,
            "scaler": self.scaler,
            "feature_names": self.feature_names,
            "is_trained": self.is_trained,
            "threshold": self.threshold
        }
        
        if self.model_type == "isolation_forest":
            model_data["model"] = self.model
        else:
            # Para modelos Keras, salvar separadamente
            self.model.save(f"{path}_keras.h5")
            
        joblib.dump(model_data, f"{path}.joblib")
        logger.info(f"Model saved to {path}")
    
    def load(self, path: str):
        """Carrega o modelo"""
        model_data = joblib.load(f"{path}.joblib")
        
        self.model_type = model_data["model_type"]
        self.scaler = model_data["scaler"]
        self.feature_names = model_data["feature_names"]
        self.is_trained = model_data["is_trained"]
        self.threshold = model_data.get("threshold", 0.95)
        
        if self.model_type == "isolation_forest":
            self.model = model_data["model"]
        else:
            from tensorflow.keras.models import load_model
            self.model = load_model(f"{path}_keras.h5")
        
        logger.info(f"Model loaded from {path}")
    
    def _build_autoencoder(self, input_dim: int) -> Model:
        """Constrói autoencoder para detecção de anomalias"""
        input_layer = Input(shape=(input_dim,))
        
        # Encoder
        encoded = Dense(64, activation='relu')(input_layer)
        encoded = Dense(32, activation='relu')(encoded)
        encoded = Dense(16, activation='relu')(encoded)
        
        # Decoder
        decoded = Dense(32, activation='relu')(encoded)
        decoded = Dense(64, activation='relu')(decoded)
        decoded = Dense(input_dim, activation='linear')(decoded)
        
        autoencoder = Model(input_layer, decoded)
        autoencoder.compile(optimizer='adam', loss='mse')
        
        return autoencoder
    
    def _build_lstm_autoencoder(self, sequence_length: int, n_features: int) -> Model:
        """Constrói LSTM autoencoder para séries temporais"""
        inputs = Input(shape=(sequence_length, n_features))
        
        # Encoder LSTM
        encoded = LSTM(32, activation='relu', return_sequences=True)(inputs)
        encoded = LSTM(16, activation='relu', return_sequences=False)(encoded)
        encoded = RepeatVector(sequence_length)(encoded)
        
        # Decoder LSTM
        decoded = LSTM(16, activation='relu', return_sequences=True)(encoded)
        decoded = LSTM(32, activation='relu', return_sequences=True)(decoded)
        decoded = TimeDistributed(Dense(n_features))(decoded)
        
        autoencoder = Model(inputs, decoded)
        autoencoder.compile(optimizer='adam', loss='mse')
        
        return autoencoder
    
    def _create_sequences(self, data: np.ndarray, sequence_length: int) -> np.ndarray:
        """Cria sequências para LSTM"""
        sequences = []
        for i in range(len(data) - sequence_length + 1):
            sequences.append(data[i:i + sequence_length])
        return np.array(sequences)

## 1. Arquitetura - Visão Geral (docs/architecture/v12-architecture-overview.md)

```markdown
# DIYAPP V12 - Arquitetura de Microserviços

**Data de criação:** 2024-01-15  
**Última atualização:** 2024-01-15  
**Owner:** Tech Lead  
**Status:** Ativo

## Visão Geral

A V12 do DIYAPP migra de uma arquitetura monolítica para microserviços, permitindo:
- Escalabilidade independente por componente
- Deploy contínuo sem downtime
- Resiliência através de circuit breakers
- Observabilidade aprimorada

## Componentes Principais

### 1. API Gateway
- **Tecnologia:** NGINX + Lua
- **Função:** Roteamento, rate limiting, autenticação
- **Porta:** 443 (HTTPS)

### 2. Serviço de Autenticação
- **Tecnologia:** Node.js + JWT
- **Função:** Gerenciamento de sessões, OAuth 2.0
- **Banco:** Redis (sessões)

### 3. Serviço de Usuários
- **Tecnologia:** Python FastAPI
- **Função:** CRUD de usuários, perfis
- **Banco:** PostgreSQL

### 4. Serviço de Conteúdo
- **Tecnologia:** Go
- **Função:** Upload, processamento de mídia
- **Armazenamento:** S3/MinIO

### 5. Serviço de Analytics (Premium)
- **Tecnologia:** Python + Pandas
- **Função:** Processamento de dados em tempo real
- **Fila:** RabbitMQ/Kafka

### 6. Serviço de Notificações
- **Tecnologia:** Node.js
- **Função:** Email, push, WhatsApp
- **Integração:** Twilio, SendGrid

## Comunicação entre Serviços

```
Usuário → API Gateway → [Serviços]
                    ↓
              Message Bus (RabbitMQ)
                    ↓
           [Serviços Assíncronos]
```

## Decisões de Design

1. **API First:** Todos os serviços expõem APIs REST/GraphQL documentadas
2. **Event-Driven:** Comunicação assíncrona para operações não críticas
3. **Health Checks:** Cada serviço expõe endpoint /health
4. **Config Externa:** Configurações via Consul/Vault
5. **Logs Estruturados:** JSON format para aggregator

## Diagrama de Contexto

![System Context Diagram](system-context-diagram.png)

## Métricas de Saúde

- **Disponibilidade:** 99.95% (SLA)
- **Latência P95:** < 200ms
- **Throughput:** 1000 req/seg por serviço
- **Recovery Time Objective:** 5 minutos
```

# Checklist de Implantação - V12

**Data:** 2024-01-15  
**Owner:** SRE Team  
**Aplicável para:** Produção, Staging

## Pré-requisitos

### Infraestrutura
- [ ] Kubernetes cluster v1.24+ (3 nós mínimo)
- [ ] PostgreSQL 14+ (HA configurado)
- [ ] Redis 7+ (cluster mode)
- [ ] MinIO/S3 bucket configurado
- [ ] Load Balancer externo
- [ ] Certificado SSL válido (Let's Encrypt)

### Configurações
- [ ] Variáveis de ambiente definidas no Vault
- [ ] Secrets gerenciados (database, API keys)
- [ ] DNS apontando para load balancer
- [ ] Backup schedule configurado

## Passo a Passo - Implantação

### 1. Build das Imagens
```bash
# Build todos os serviços
./scripts/build-all.sh --version v12.0.0

# Push para registry
docker push registry.diyapp.com/api-gateway:v12.0.0
docker push registry.diyapp.com/auth-service:v12.0.0
# ... todos os serviços
```

### 2. Atualização do Kubernetes
```bash
# Aplicar configurações
kubectl apply -f k8s/namespaces.yaml
kubectl apply -f k8s/configmaps/
kubectl apply -f k8s/secrets/

# Deploy serviços (ordem crítica)
kubectl apply -f k8s/database/
kubectl apply -f k8s/redis/
kubectl apply -f k8s/message-queue/

# Deploy serviços da aplicação
kubectl apply -f k8s/services/auth/
kubectl apply -f k8s/services/users/
kubectl apply -f k8s/services/content/
kubectl apply -f k8s/services/analytics/
kubectl apply -f k8s/services/notifications/

# API Gateway (último)
kubectl apply -f k8s/services/api-gateway/
```

### 3. Verificação Pós-Deploy
```bash
# Verificar pods
kubectl get pods -n diyapp-prod

# Verificar health checks
curl https://api.diyapp.com/v12/health

# Smoke tests
./scripts/smoke-tests.sh --env production

# Verificar métricas
kubectl port-forward svc/prometheus 9090 -n monitoring
```

### 4. Migração de Dados (se necessário)
```bash
# Backup pré-migração
pg_dump diyapp_v11 > backup_v11.sql

# Executar migração
kubectl apply -f k8s/jobs/migration-v12.yaml

# Verificar migração
kubectl logs job/migration-v12 -n diyapp-prod
```

## Rollback Procedure

### Rollback Automático (5 minutos)
Se health checks falharem por mais de 5 minutos:
```bash
# Reverter para versão anterior
kubectl rollout undo deployment/api-gateway -n diyapp-prod
kubectl rollout undo deployment/auth-service -n diyapp-prod
# ... todos os serviços
```

### Rollback Manual
```bash
# 1. Parar tráfego no load balancer
# 2. Reverter imagens para v11.5.0
kubectl set image deployment/api-gateway api-gateway=registry.diyapp.com/api-gateway:v11.5.0
# ... todos os serviços

# 3. Restaurar database se necessário
psql diyapp_v12 < backup_v11.sql
```

# Runbook: Resposta a Incidentes

**Severidade:** P1-P4  
**Owner:** SRE On-Call  
**Última revisão:** 2024-01-15

## Classificação de Severidade

### P1 - Crítico
- **Impacto:** Sistema completamente indisponível
- **Exemplo:** API Gateway offline, Database down
- **RTO:** 15 minutos
- **Comunicação:** Imediata para todos os stakeholders

### P2 - Alto
- **Impacto:** Funcionalidade principal degradada
- **Exemplo:** Upload de arquivos falhando, Login lento
- **RTO:** 1 hora
- **Comunicação:** Equipe interna + atualizações a cada 30min

### P3 - Médio
- **Impacto:** Funcionalidade não crítica afetada
- **Exemplo:** Relatórios premium não gerando
- **RTO:** 4 horas
- **Comunicação:** Atualização diária

### P4 - Baixo
- **Impacto:** Problema cosmético ou menor
- **Exemplo:** Typo na interface, cor incorreta
- **RTO:** 24 horas
- **Comunicação:** Ticket normal

## Fluxo de Resposta

### Minuto 0-5: Detecção e Triagem


### [Sessão Paralela: Inovação]
# DIYAPP Evolution - V12 Core - Feature Killer: "AI Workflow Copilot"

## Análise de Sinais e Hipótese de Inovação

**SINAL IDENTIFICADO**: 
- Competidores (Zapier, Make) lançando features de "AI Assistant" para criação de workflows
- Usuários reportando complexidade na criação de workflows visuais (tickets de suporte)
- Novo modelo GPT-4o com capacidade multimodal (texto + imagem) lançado recentemente
- Tendência de mercado: democratização de automação para usuários não-técnicos

**HIPÓTESE**:
Acreditamos que **usuários não-técnicos de automação** têm o problema de **dificuldade em traduzir processos manuais em workflows visuais** e que um **assistente por IA generativa que converte descrições textuais em workflows funcionais** vai resultar em **redução de 40% no tempo para criar um workflow básico** (métrica: tempo desde descrição até workflow funcional).

const schedule = require('node-schedule');
const db = require('../database');

class AutoDeletionSystem {
    constructor() {
        this.initSchedules();
    }
    
    initSchedules() {
        // Executa diariamente às 02:00
        schedule.scheduleJob('0 2 * * *', async () => {
            await this.deleteInactiveUsers();
            await this.rotateLogs();
            await this.cleanBackups();
        });
    }
    
    async deleteInactiveUsers() {
        const twoYearsAgo = new Date();
        twoYearsAgo.setFullYear(twoYearsAgo.getFullYear() - 2);
        
        const result = await db.query(`
            DELETE FROM users 
            WHERE last_login < $1 
            AND account_status = 'inactive'
            RETURNING user_id, email
        `, [twoYearsAgo]);
        
        if (result.rows.length > 0) {
            await this.logDeletion('users', result.rows);
        }
    }
    
    async rotateLogs() {
        const sixMonthsAgo = new Date();
        sixMonthsAgo.setMonth(sixMonthsAgo.getMonth() - 6);
        
        await db.query(`
            DELETE FROM access_logs 
            WHERE timestamp < $1
        `, [sixMonthsAgo]);
    }
    
    async logDeletion(table, records) {
        await db.query(`
            INSERT INTO compliance_logs 
            (action, table_name, records_count, metadata) 
            VALUES ($1, $2, $3, $4)
        `, ['auto_deletion', table, records.length, JSON.stringify(records)]);
    }
}
