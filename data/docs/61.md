## 2. Architecture Decision Record (ADR-001)

**docs/architecture/ADR-001-service-boundaries.md**:
```markdown
# ADR-001: Boundaries de Microsserviços Baseados em Domínio

## Status
Aceito

## Data
2024-01-15

## Autores
Tech Lead V11 Core

## Contexto
O DIYAPP evoluiu de uma aplicação monolítica para uma plataforma distribuída com múltiplas funcionalidades. Precisamos definir boundaries claros entre serviços para permitir:
- Desenvolvimento paralelo (Modo Hive)
- Escalabilidade independente
- Manutenção autônoma
- Deploy contínuo sem downtime

## Decisão
Dividir a aplicação em 6 microsserviços baseados em bounded contexts do domínio:

1. **Auth Service**: Autenticação, autorização, gestão de usuários
2. **Project Service**: Gestão de projetos, templates, configurações
3. **Task Service**: Execução de tarefas, filas, workers
4. **Notification Service**: Notificações em tempo real, email, push
5. **Event Bus**: Barramento de eventos assíncronos
6. **API Gateway**: Ponto único de entrada, roteamento, rate limiting

## Opções Consideradas

### Opção A: Serviços por Funcionalidade
- Prós: Alinhamento direto com features do produto
- Contras: Acoplamento alto entre serviços, duplicação de lógica

### Opção B: Serviços por Camada Técnica
- Prós: Especialização técnica clara
- Contras: Fragilidade em mudanças de domínio

### Opção C: Serviços por Domínio (ESCOLHIDA)
- Prós: Alinhamento com negócio, baixo acoplamento, alta coesão
- Contras: Complexidade inicial maior

## Consequências
### Positivas
- Cada squad pode operar independentemente
- Escalabilidade granular
- Tecnologias específicas por domínio

### Negativas
- Overhead de comunicação entre serviços
- Complexidade de deploy distribuído
- Necessidade de observabilidade robusta

### Riscos
- Latência em chamadas entre serviços
- Consistência eventual
- Mitigação: Circuit breakers, retry policies, sagas

## Revisão
Reavaliar em 6 meses ou quando throughput atingir 10k req/seg
```

- Cada serviço tem seu próprio banco
- Não compartilhe tabelas entre serviços
- Use migrações versionadas
- Backup automático diário

- Structured logging em JSON
- Níveis: error, warn, info, debug
- Métricas: Prometheus
- Tracing: Jaeger
- Alertas: Grafana

## 6. Segurança
- Autenticação via JWT
- Autorização baseada em roles
- Validação de input em todas as APIs
- Rate limiting por usuário/IP
- Scan de dependências semanal

- Containerização com Docker
- Orquestração: Kubernetes
- Blue-green deployment
- Rollback automático em falhas >5%
- Health checks obrigatórios

## 6. Configuração do Event Bus

**packages/event-bus/src/index.ts**:
```typescript
import { EventEmitter } from 'events';
import { BaseEvent, EventTypes } from '@diyapp/shared/contracts/events';
import { logger } from '@diyapp/shared/logger';

export class EventBus {
  private emitter: EventEmitter;
  private subscriptions: Map<string, Function[]>;
  
  constructor() {
    this.emitter = new EventEmitter();
    this.subscriptions = new Map();
    this.setupErrorHandling();
  }
  
  async publish(event: BaseEvent): Promise<void> {
    try {
      logger.info(`Publishing event: ${event.eventType}`, {
        eventId: event.eventId,
        aggregateId: event.aggregateId,
        source: event.metadata.source,
      });
      
      // Emit para subscribers locais
      this.emitter.emit(event.eventType, event);
      
      // Emit para subscribers remotos (via Redis/RabbitMQ em produção)
      await this.publishToExternalBus(event);
      
      logger.debug(`Event published successfully: ${event.eventId}`);
    } catch (error) {
      logger.error(`Failed to publish event: ${event.eventId}`, { error });
      throw error;
    }
  }
  
  subscribe(eventType: EventTypes, handler: (event: BaseEvent) => Promise<void>): void {
    if (!this.subscriptions.has(eventType)) {
      this.subscriptions.set(eventType, []);
    }
    
    const wrappedHandler = async (event: BaseEvent) => {
      try {
        await handler(event);
      } catch (error) {
        logger.error(`Error in event handler for ${eventType}`, {
          eventId: event.eventId,
          error: error.message,
        });
        // Não rethrow - evitar quebrar outros handlers
      }
    };
    
    this.subscriptions.get(eventType)!.push(wrappedHandler);
    this.emitter.on(eventType, wrappedHandler);
    
    logger.debug(`Subscribed to event: ${eventType}`);
  }
  
  private async publishToExternalBus(event: BaseEvent): Promise<void> {
    // Em produção, implementar com Redis Streams ou RabbitMQ
    // Para desenvolvimento local, apenas log
    logger.debug(`External bus publish (simulated): ${event.eventType}`);
  }
  
  private setupErrorHandling(): void {
    this.emitter.on('error', (error) => {
      logger.error('Event bus error:', { error });
    });
  }
  
  async shutdown(): Promise<void> {
    this.emitter.removeAllListeners();
    this.subscriptions.clear();
    logger.info('Event bus shutdown complete');
  }
}

// Singleton instance
export const eventBus = new EventBus();
```

## 2. Ponto de Entrada Principal (src/index.js)

```javascript
import express from 'express';
import helmet from 'helmet';
import cors from 'cors';
import compression from 'compression';
import { createServer } from 'http';
import { initializeObservability } from './core/observability/index.js';
import { initializeSecurity } from './core/security/index.js';
import { initializeResilience } from './core/resilience/index.js';
import { initializeDatabase } from './data/database.js';
import { initializeRESTAPI } from './api/rest/index.js';
import { initializeGraphQL } from './api/graphql/index.js';
import { initializeAIProviders } from './external/ai/providers.js';
import { errorHandler, notFoundHandler } from './api/middleware/errorHandler.js';
import { requestLogger, correlationId } from './api/middleware/logging.js';
import { rateLimiter } from './api/middleware/rateLimiter.js';
import { metricsMiddleware } from './api/middleware/metrics.js';
import { config } from './shared/config.js';
import logger from './shared/logger.js';

const app = express();
const server = createServer(app);

async function bootstrap() {
  try {
    // 1. Observabilidade primeiro (antes de qualquer coisa)
    initializeObservability();
    
    // 2. Middleware de segurança
    initializeSecurity(app);
    
    // 3. Middleware básico
    app.use(compression());
    app.use(cors(config.cors));
    app.use(express.json({ limit: '10mb' }));
    app.use(express.urlencoded({ extended: true }));
    
    // 4. Middleware customizado
    app.use(correlationId);
    app.use(requestLogger);
    app.use(metricsMiddleware);
    app.use(rateLimiter);
    
    // 5. Inicializar sistemas
    await initializeDatabase();
    await initializeResilience();
    await initializeAIProviders();
    
    // 6. APIs
    initializeRESTAPI(app);
    initializeGraphQL(app);
    
    // 7. Health checks e status
    app.get('/health', (req, res) => {
      res.status(200).json({
        status: 'healthy',
        timestamp: new Date().toISOString(),
        uptime: process.uptime(),
        version: config.version
      });
    });
    
    app.get('/metrics', (req, res) => {
      res.set('Content-Type', 'text/plain');
      res.end(require('prom-client').register.metrics());
    });
    
    // 8. Documentação
    if (config.env !== 'production') {
      const { setupSwagger } = await import('./docs/swagger.js');
      setupSwagger(app);
    }
    
    // 9. Error handlers (devem ser os últimos)
    app.use(notFoundHandler);
    app.use(errorHandler);
    
    // 10. Iniciar servidor
    const PORT = config.port || 3000;
    server.listen(PORT, () => {
      logger.info(`Server running on port ${PORT}`, {
        env: config.env,
        node_version: process.version,
        pid: process.pid
      });
    });
    
    // 11. Graceful shutdown
    const shutdown = async (signal) => {
      logger.info(`Received ${signal}, starting graceful shutdown`);
      
      server.close(async () => {
        logger.info('HTTP server closed');
        
        // Fechar conexões de banco
        const { closeDatabase } = await import('./data/database.js');
        await closeDatabase();
        
        // Fechar conexões Redis
        const { closeRedis } = await import('./shared/cache.js');
        await closeRedis();
        
        logger.info('Graceful shutdown completed');
        process.exit(0);
      });
      
      // Force shutdown after 30 seconds
      setTimeout(() => {
        logger.error('Could not close connections in time, forcefully shutting down');
        process.exit(1);
      }, 30000);
    };
    
    process.on('SIGTERM', () => shutdown('SIGTERM'));
    process.on('SIGINT', () => shutdown('SIGINT'));
    
  } catch (error) {
    logger.error('Failed to bootstrap application', { error: error.message, stack: error.stack });
    process.exit(1);
  }
}

bootstrap();
```

```javascript
import client from 'prom-client';

const collectDefaultMetrics = client.collectDefaultMetrics;
collectDefaultMetrics({ register: client.register });

const httpRequestDurationMicroseconds = new client.Histogram({
  name: 'http_request_duration_ms',
  help: 'Duration of HTTP requests in ms',
  labelNames: ['method', 'route', 'status_code'],
  buckets: [50, 100, 200, 300, 500, 1000, 2000, 5000],
});

const httpRequestsTotal = new client.Counter({
  name: 'http_requests_total',
  help: 'Total number of HTTP requests',
  labelNames: ['method', 'route', 'status_code'],
});

const httpRequestErrors = new client.Counter({
  name: 'http_request_errors_total',
  help: 'Total number of HTTP request errors',
  labelNames: ['method', 'route', 'error_type'],
});

export const metricsMiddleware = (req, res, next) => {
  const start = Date.now();
  const originalEnd = res.end;
  
  res.end = function (...args) {
    const duration = Date.now() - start;
    const route = req.route?.path || req.path;
    
    httpRequestDurationMicroseconds
      .labels(req.method, route, res.statusCode)
      .observe(duration);
    
    httpRequestsTotal
      .labels(req.method, route, res.statusCode)
      .inc();
    
    if (res.statusCode >= 400) {
      const errorType = res.statusCode >= 500 ? '5xx' : '4xx';
      httpRequestErrors
        .labels(req.method, route, errorType)
        .inc();
    }
    
    originalEnd.apply(res, args);
  };
  
  next();
};

export const register = client.register;
```

```sql
{{
    config(
        materialized='incremental',
        unique_key='event_id',
        incremental_strategy='merge',
        merge_update_columns=['event_data', 'processed_at']
    )
}}

WITH source AS (
    SELECT
        event_id,
        user_id,
        session_id,
        event_type,
        event_name,
        event_data::jsonb AS event_data,
        device_info::jsonb AS device_info,
        app_version,
        timestamp AS event_timestamp,
        ingested_at,
        '{{ invocation_id }}' AS batch_id
    FROM {{ source('raw', 'telemetry_events') }}
    {% if is_incremental() %}
    WHERE ingested_at > (SELECT MAX(processed_at) FROM {{ this }})
    {% endif %}
),

cleaned AS (
    SELECT
        event_id,
        -- Validação e limpeza de user_id
        CASE 
            WHEN user_id ~ '^[0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12}$' 
            THEN user_id
            ELSE NULL 
        END AS user_id,
        
        -- Validação de session_id
        CASE 
            WHEN session_id ~ '^[A-Z0-9]{32}$' 
            THEN session_id
            ELSE MD5(COALESCE(session_id, 'unknown') || event_timestamp::text)
        END AS session_id,
        
        -- Categorização de event_type
        CASE event_type
            WHEN 'page_view' THEN 'navigation'
            WHEN 'button_click' THEN 'interaction'
            WHEN 'api_call' THEN 'api'
            WHEN 'error' THEN 'error'
            ELSE 'other'
        END AS event_category,
        
        event_name,
        
        -- Extração de campos estruturados do event_data
```

```sql
    SELECT
        date,
        COUNT(DISTINCT user_id) AS daily_active_users,
        COUNT(DISTINCT CASE WHEN session_count >= 3 THEN user_id END) AS engaged_users,
        AVG(session_count) AS avg_sessions_per_user,
        AVG(event_count) AS avg_events_per_user,
        SUM(event_count) AS total_events,

        COUNT(DISTINCT CASE

        END) AS weekend_users

        avg_events_per_user,

        weekend_users,
        AVG(daily_active_users) OVER (
            ORDER BY date ROWS BETWEEN 6 PRECEDING AND CURRENT ROW

        AVG(daily_active_users) OVER (

        ) AS mau

    ROUND(avg_events_per_user, 2) AS avg_events_per_user,
```

```javascript
// automated-security-scanner.js
const fs = require('fs');
const path = require('path');
const crypto = require('crypto');
const { execSync } = require('child_process');

class SecurityScanner {
  constructor() {
    this.findings = [];
    this.criticalPatterns = [
      /password.*=.*['"][^'"]*['"]/gi,
      /token.*=.*['"][^'"]*['"]/gi,
      /secret.*=.*['"][^'"]*['"]/gi,
      /eval\(/gi,
      /innerHTML.*=.*user/gi,
      /localStorage\.setItem.*password/gi
    ];
  }

  scanDirectory(dir) {
    const files = this.getFiles(dir);
    
    files.forEach(file => {
      if (file.endsWith('.js') || file.endsWith('.ts') || file.endsWith('.jsx') || file.endsWith('.tsx')) {
        this.scanFile(file);
      }
    });
    
    return this.generateReport();
  }

  scanFile(filePath) {
    try {
      const content = fs.readFileSync(filePath, 'utf8');
      const lines = content.split('\n');
      
      lines.forEach((line, index) => {
        this.criticalPatterns.forEach(pattern => {
          if (pattern.test(line)) {
            this.findings.push({
              file: filePath,
              line: index + 1,
              code: line.trim(),
              severity: 'CRITICAL',
              pattern: pattern.toString()
            });
          }
        });
        
        // Check for SQL injection patterns
        if (line.includes('query(') && line.includes('${') && !line.includes('parameterized')) {
          this.findings.push({
            file: filePath,
            line: index + 1,
            code: line.trim(),
            severity: 'HIGH',
            issue: 'Potential SQL Injection'
          });
        }
      });
    } catch (error) {
      console.error(`Error scanning ${filePath}:`, error.message);
    }
  }

  getFiles(dir) {
    let results = [];
    const list = fs.readdirSync(dir);
    
    list.forEach(file => {
      const filePath = path.join(dir, file);
      const stat = fs.statSync(filePath);
      
      if (stat && stat.isDirectory() && !filePath.includes('node_modules') && !filePath.includes('.git')) {
        results = results.concat(this.getFiles(filePath));
      } else if (filePath.match(/\.(js|ts|jsx|tsx|json)$/)) {
        results.push(filePath);
      }
    });
    
    return results;
  }

  generateReport() {
    const report = {
      timestamp: new Date().toISOString(),
      totalFilesScanned: this.findings.length,
      criticalCount: this.findings.filter(f => f.severity === 'CRITICAL').length,
      highCount: this.findings.filter(f => f.severity === 'HIGH').length,
      findings: this.findings,
      summary: `Found ${this.findings.length} security issues`
    };
    
    fs.writeFileSync('security-scan-report.json', JSON.stringify(report, null, 2));
    return report;
  }
}
```

### 2.1 Configuração do Servidor de Autenticação

```javascript
// auth-server/oauth2-server.js
const express = require('express');
const crypto = require('crypto');
const jwt = require('jsonwebtoken');
const { v4: uuidv4 } = require('uuid');
const rateLimit = require('express-rate-limit');

class OAuth2Server {
  constructor() {
    this.app = express();
    this.app.use(express.json());
    this.setupSecurityHeaders();
    this.setupRateLimiting();
    this.initialize();
  }

  setupSecurityHeaders() {
    this.app.use((req, res, next) => {
      res.setHeader('X-Content-Type-Options', 'nosniff');
      res.setHeader('X-Frame-Options', 'DENY');
      res.setHeader('X-XSS-Protection', '1; mode=block');
      res.setHeader('Strict-Transport-Security', 'max-age=31536000; includeSubDomains');
      res.setHeader('Content-Security-Policy', "default-src 'self'; script-src 'self' 'unsafe-inline'; style-src 'self' 'unsafe-inline'");
      next();
    });
  }

  setupRateLimiting() {
    const authLimiter = rateLimit({
      windowMs: 15 * 60 * 1000, // 15 minutes
      max: 5, // 5 attempts per window
      message: 'Too many authentication attempts, please try again later',
      standardHeaders: true,
      legacyHeaders: false
    });

    this.app.use('/oauth/token', authLimiter);
    this.app.use('/oauth/authorize', authLimiter);
  }

  initialize() {
    // In-memory storage for demo (use Redis/DB in production)
    this.authorizationCodes = new Map();
    this.accessTokens = new Map();
    this.refreshTokens = new Map();
    this.pkceCodes = new Map();

    this.setupRoutes();
  }

  setupRoutes() {
    // Authorization endpoint
    this.app.get('/oauth/authorize', this.handleAuthorization.bind(this));
    
    // Token endpoint
    this.app.post('/oauth/token', this.handleTokenRequest.bind(this));
    
    // Token introspection
    this.app.post('/oauth/introspect', this.handleTokenIntrospection.bind(this));
    
    // Token revocation
    this.app.post('/oauth/revoke', this.handleTokenRevocation.bind(this));
    
    // User info endpoint
    this.app.get('/oauth/userinfo', this.authenticateToken.bind(this), this.handleUserInfo.bind(this));
  }

  async handleAuthorization(req, res) {
    try {
      const { client_id, redirect_uri, response_type, scope, state, code_challenge, code_challenge_method } = req.query;
      
      // Validate client
      if (!this.validateClient(client_id, redirect_uri)) {
        return res.status(400).json({ error: 'invalid_client' });
      }

      // PKCE validation
      if (!code_challenge || !code_challenge_method) {
        return res.status(400).json({ error: 'invalid_request', error_description: 'PKCE required' });
      }

      if (code_challenge_method !== 'S256') {
        return res.status(400).json({ error: 'invalid_request', error_description: 'Only S256 code challenge method supported' });
      }

      // Store PKCE code challenge
      const authRequestId = uuidv4();
      this.pkceCodes.set(authRequestId, {
        code_challenge,
        code_challenge_method,
        client_id,
        redirect_uri,
        scope,
        expires: Date.now() + 600000 // 10 minutes
      });

      // Redirect to login page (in production, this would be a proper login UI)
      const loginUrl = `/login?auth_request_id=${authRequestId}&redirect_uri=${encodeURIComponent(redirect_uri)}`;
      res.redirect(loginUrl);
    } catch (error) {
      console.error('Authorization error:', error);
      res.status(500).json({ error: 'server_error' });
    }
  }

  async handleTokenRequest(req, res) {
    try {
      const { grant_type, code, redirect_uri, client_id, code_verifier } = req.body;

      // Validate grant type
      if (grant_type !== 'authorization_code') {
        return res.status(400).json({ error: 'unsupported_grant_type' });
      }

      // Find authorization code
      const authCode = this.authorizationCodes.get(code);
      if (!authCode || authCode.expires < Date.now()) {
        return res.status(400).json({ error: 'invalid_grant' });
      }

      // PKCE verification
      const pkceData = this.pkceCodes.get(authCode.pkceId);
      if (!pkceData) {
        return res.status(400).json({ error: 'invalid_grant' });
      }

      const calculatedChallenge = crypto
        .createHash('sha256')
        .update(code_verifier)
        .digest('base64')
        .replace(/\+/g, '-')
        .replace(/\//g, '_')
        .replace(/=/g, '');

      if (calculatedChallenge !== pkceData.code_challenge) {
        return res.status(400).json({ error: 'invalid_grant', error_description: 'PKCE verification failed' });
      }

      // Generate tokens
      const accessToken = this.generateAccessToken(authCode.userId, authCode.scope);
      const refreshToken = this.generateRefreshToken(authCode.userId, authCode.scope);

      // Cleanup
      this.authorizationCodes.delete(code);
      this.pkceCodes.delete(authCode.pkceId);

      res.json({
        access_token: accessToken.token,
        token_type: 'Bearer',
        expires_in: 900, // 15 minutes
        refresh_token: refreshToken.token,
        scope: authCode.scope
      });
```
