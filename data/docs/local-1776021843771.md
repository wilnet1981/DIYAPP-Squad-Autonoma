# DIYAPP Evolution - V13 Core - Arquitetura de Microsserviços

## 1. ADR-001: Arquitetura de Microsserviços V13

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra

### CONTEXTO:
A V12 do DIYAPP evoluiu para uma arquitetura monolítica que está limitando:
- Velocidade de deploy independente por feature
- Escalabilidade horizontal específica por componente
- Isolamento de falhas entre módulos
- Adoção de diferentes stacks tecnológicas por domínio

### DECISÃO:
Adotar arquitetura de microsserviços com os seguintes princípios:
1. **Domínio por serviço**: Cada serviço encapsula um domínio de negócio completo
2. **Autonomia total**: Cada serviço tem seu próprio banco de dados e pode ser deployado independentemente
3. **Comunicação assíncrona por padrão**: Eventos via Apache Kafka para desacoplamento
4. **gRPC para comunicação síncrona crítica**: Onde latência mínima é essencial
5. **API Gateway unificado**: Para clientes externos e frontend

### OPÇÕES CONSIDERADAS:
- **Opção A**: Arquitetura monolítica modularizada - Prós: Simplicidade operacional, debugging fácil. Contras: Acoplamento forte, deploy único.
- **Opção B**: Microsserviços com REST/HTTP - Prós: Familiaridade, ferramentas maduras. Contras: Overhead de serialização, sem streaming nativo.
- **Opção C**: Microsserviços com gRPC + Eventos (escolhida) - Prós: Performance, contratos fortes, streaming bidirecional. Contras: Curva de aprendizado, tooling menos maduro.

### CONSEQUÊNCIAS:
**Positivas:**
- Deploy independente por serviço
- Escalabilidade granular
- Isolamento de falhas
- Poliglotismo tecnológico

**Negativas:**
- Complexidade operacional aumentada
- Debugging distribuído necessário
- Latência de rede entre serviços
- Gerenciamento de transações distribuídas

**Riscos:**
- Falhas em cascata (mitigação: circuit breakers)
- Inconsistência de dados (mitigação: padrão Saga)
- Monitoramento complexo (mitigação: OpenTelemetry + Jaeger)

## 5. Protocol Buffers Definition Template

```proto
// src/proto/user_service.proto
syntax = "proto3";

package diyapp.user.v1;

import "google/protobuf/timestamp.proto";
import "google/protobuf/empty.proto";

service UserService {
  rpc GetUser(GetUserRequest) returns (UserResponse);
  rpc CreateUser(CreateUserRequest) returns (UserResponse);
  rpc UpdateUser(UpdateUserRequest) returns (UserResponse);
  rpc ListUsers(ListUsersRequest) returns (stream UserResponse);
}

message GetUserRequest {
  string user_id = 1;
}

message CreateUserRequest {
  string email = 1;
  string name = 2;
  map<string, string> metadata = 3;
}

message UpdateUserRequest {
  string user_id = 1;
  optional string email = 2;
  optional string name = 3;
  map<string, string> metadata = 4;
}

message ListUsersRequest {
  int32 page = 1;
  int32 limit = 2;
  string filter = 3;
}

message UserResponse {
  string user_id = 1;
  string email = 2;
  string name = 3;
  google.protobuf.Timestamp created_at = 4;
  google.protobuf.Timestamp updated_at = 5;
  UserStatus status = 6;
  map<string, string> metadata = 7;
}

enum UserStatus {
  USER_STATUS_UNSPECIFIED = 0;
  USER_STATUS_ACTIVE = 1;
  USER_STATUS_INACTIVE = 2;
  USER_STATUS_SUSPENDED = 3;
}
```

# src/core/circuit_breaker.py
import time
from typing import Callable, Any, Optional
from functools import wraps
import logging

logger = logging.getLogger(__name__)

class CircuitBreaker:
    def __init__(
        self,
        name: str,
        failure_threshold: int = 5,
        recovery_timeout: int = 30,
        max_retries: int = 3
    ):
        self.name = name
        self.failure_threshold = failure_threshold
        self.recovery_timeout = recovery_timeout
        self.max_retries = max_retries
        self.failure_count = 0
        self.last_failure_time = 0
        self.state = "CLOSED"  # CLOSED, OPEN, HALF_OPEN
        
    def can_execute(self) -> bool:
        if self.state == "OPEN":
            if time.time() - self.last_failure_time > self.recovery_timeout:
                self.state = "HALF_OPEN"
                logger.warning(f"Circuit breaker {self.name} moving to HALF_OPEN")
                return True
            return False
        return True
    
    def on_success(self):
        if self.state == "HALF_OPEN":
            self.state = "CLOSED"
            self.failure_count = 0
            logger.info(f"Circuit breaker {self.name} reset to CLOSED")
    
    def on_failure(self):
        self.failure_count += 1
        self.last_failure_time = time.time()
        
        if self.failure_count >= self.failure_threshold:
            self.state = "OPEN"
            logger.error(f"Circuit breaker {self.name} OPENED after {self.failure_count} failures")
    
    def __call__(self, func: Callable) -> Callable:
        @wraps(func)
        def wrapper(*args, **kwargs) -> Any:
            if not self.can_execute():
                raise CircuitBreakerOpenError(
                    f"Circuit breaker {self.name} is OPEN. Service unavailable."
                )
            
            retries = 0
            while retries <= self.max_retries:
                try:
                    result = func(*args, **kwargs)
                    self.on_success()
                    return result
                except Exception as e:
                    retries += 1
                    if retries > self.max_retries:
                        self.on_failure()
                        raise
                    
                    # Exponential backoff with jitter
                    delay = (2 ** retries) + (0.1 * retries)
                    time.sleep(delay)
            
            self.on_failure()
            raise Exception(f"Max retries ({self.max_retries}) exceeded for {self.name}")
        
        return wrapper

class CircuitBreakerOpenError(Exception):
    pass

class JSONFormatter(logging.Formatter):
    def format(self, record: logging.LogRecord) -> str:
        log_data: Dict[str, Any] = {
            "timestamp": datetime.utcnow().isoformat() + "Z",
            "level": record.levelname,
            "logger": record.name,
            "message": record.getMessage(),
            "correlation_id": getattr(record, 'correlation_id', None),
            "user_id": getattr(record, 'user_id', None),
            "endpoint": getattr(record, 'endpoint', None),
            "duration_ms": getattr(record, 'duration_ms', None),
            "status": getattr(record, 'status', None),
        }
        
        # Add exception info if present
        if record.exc_info:
            log_data["exception"] = self.formatException(record.exc_info)
        
        # Mask sensitive data
        self._mask_sensitive_data(log_data)
        
        return json.dumps(log_data)
    
    def _mask_sensitive_data(self, log_data: Dict[str, Any]):
        sensitive_fields = ['password', 'token', 'secret', 'authorization', 'credit_card', 'cpf', 'email']
        
        def mask_dict(d: Dict[str, Any]):
            for key, value in d.items():
                if isinstance(value, dict):
                    mask_dict(value)
                elif isinstance(value, str):
                    for field in sensitive_fields:
                        if field in key.lower():
                            d[key] = "***MASKED***"
                            break
        
        mask_dict(log_data)

from passlib.context import CryptContext
from datetime import datetime, timedelta
from typing import Optional
import jwt
from jwt.exceptions import InvalidTokenError

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")

def verify_password(plain_password: str, hashed_password: str) -> bool:
    return pwd_context.verify(plain_password, hashed_password)

def get_password_hash(password: str) -> str:
    return pwd_context.hash(password)

def create_access_token(
    data: dict,
    secret_key: str,
    algorithm: str,
    expires_delta: Optional[timedelta] = None
) -> str:
    to_encode = data.copy()
    if expires_delta:
        expire = datetime.utcnow() + expires_delta
    else:
        expire = datetime.utcnow() + timedelta(minutes=15)
    
    to_encode.update({"exp": expire, "type": "access"})
    encoded_jwt = jwt.encode(to_encode, secret_key, algorithm=algorithm)
    return encoded_jwt

def create_refresh_token(
    data: dict,
    secret_key: str,
    algorithm: str,
    expires_delta: Optional[timedelta] = None
) -> str:
    to_encode = data.copy()
    if expires_delta:
        expire = datetime.utcnow() + expires_delta
    else:
        expire = datetime.utcnow() + timedelta(days=7)
    
    to_encode.update({"exp": expire, "type": "refresh"})
    encoded_jwt = jwt.encode(to_encode, secret_key, algorithm=algorithm)
    return encoded_jwt

def verify_token(token: str, secret_key: str, algorithm: str) -> dict:
    try:
        payload = jwt.decode(token, secret_key, algorithms=[algorithm])
        return payload
    except InvalidTokenError:
        raise

            session.commit()
        except Exception:
            session.rollback()
            raise
        finally:
            session.close()

        Base.metadata.create_all(bind=self.engine)

        self.settings = settings
    
    @auth_circuit_breaker
    def authenticate_user(
        self,
        db: Session,
        email: str,
        password: str
    ) -> Optional[User]:
        """Authenticate user with email and password"""
        user = db.query(User).filter(User.email == email).first()
        if not user:
            return None
        if not user.hashed_password:
            return None
        if not verify_password(password, user.hashed_password):
            return None
        if not user.is_active:
            return None
        return user
    
    @auth_circuit_breaker
    def create_user(self, db: Session, user_create: UserCreate) -> User:
        """Create new user with hashed password"""
        # Check if user already exists
        existing_user = db.query(User).filter(User.email == user_create.email).first()
        if existing_user:
            raise ValueError("User with this email already exists")
        
        # Create user
        hashed_password = get_password_hash(user_create.password)
        db_user = User(
            email=user_create.email,
            hashed_password=hashed_password,
            full_name=user_create.full_name,
            is_active=True,
            is_verified=False
        )
        
        db.add(db_user)
        db.commit()
        db.refresh(db_user)

```typescript
import { createSlice, createAsyncThunk, PayloadAction } from '@reduxjs/toolkit';
import { api } from '@app/api';

interface AuthState {
  user: User | null;
  token: string | null;
  isLoading: boolean;
  error: string | null;
}

interface User {
  id: string;
  email: string;
  name: string;
  avatar?: string;
  role: 'user' | 'admin';
}

interface LoginCredentials {
  email: string;
  password: string;
}

interface RegisterData extends LoginCredentials {
  name: string;
}

const initialState: AuthState = {
  user: null,
  token: localStorage.getItem('token'),
  isLoading: false,
  error: null,
};

export const login = createAsyncThunk(
  'auth/login',
  async (credentials: LoginCredentials, { rejectWithValue }) => {
    try {
      const response = await fetch('/api/auth/login', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(credentials),
      });

      if (!response.ok) {
        const error = await response.json();
        return rejectWithValue(error);
      }

      const data = await response.json();
      localStorage.setItem('token', data.token);
      return data;
    } catch (error) {
      return rejectWithValue({
        message: 'Network error. Please check your connection.',
      });
    }
  }
);

export const register = createAsyncThunk(
  'auth/register',
  async (userData: RegisterData, { rejectWithValue }) => {
    try {
      const response = await fetch('/api/auth/register', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(userData),
      });

      if (!response.ok) {
        const error = await response.json();
        return rejectWithValue(error);
      }

      const data = await response.json();
      localStorage.setItem('token', data.token);
      return data;
    } catch (error) {
      return rejectWithValue({
        message: 'Network error. Please check your connection.',
      });
    }
  }
);

export const logout = createAsyncThunk('auth/logout', async () => {
  localStorage.removeItem('token');
});

const authSlice = createSlice({
  name: 'auth',
  initialState,
  reducers: {
    setUser: (state, action: PayloadAction<User>) => {
      state.user = action.payload;
    },
    clearError: (state) => {
      state.error = null;
    },
  },
  extraReducers: (builder) => {
    builder
      // Login
      .addCase(login.pending, (state) => {
        state.isLoading = true;
        state.error = null;
      })
      .addCase(login.fulfilled, (state, action) => {
        state.isLoading = false;
        state.user = action.payload.user;
        state.token = action.payload.token;
      })
      .addCase(login.rejected, (state, action) => {
        state.isLoading = false;
        state.error = action.payload as string;
      })
      // Register
      .addCase(register.pending, (state) => {
        state.isLoading = true;
        state.error = null;
      })
      .addCase(register.fulfilled, (state, action) => {
        state.isLoading = false;
        state.user = action.payload.user;
        state.token = action.payload.token;
      })
      .addCase(register.rejected, (state, action) => {
        state.isLoading = false;
        state.error = action.payload as string;
      })
      // Logout
      .addCase(logout.fulfilled, (state) => {
        state.user = null;
        state.token = null;
      });
  },
});

export const { setUser, clearError } = authSlice.actions;
export default authSlice.reducer;
```

# DIYAPP Evolution - V13 Core - Arquitetura de Microsserviços

## 1. ADR-001: Arquitetura de Microsserviços V13

**ADR-001: Adoção de Arquitetura de Microsserviços com Comunicação Híbrida gRPC/Eventos**

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead + Especialista Infra + Especialista Backend

**CONTEXTO:**
A V12 do DIYAPP atingiu limites de escalabilidade devido à arquitetura monolítica. O sistema precisa:
- Escalar componentes independentemente
- Permitir deploy contínuo sem downtime
- Suportar times distribuídos trabalhando em paralelo
- Manter 100% de estabilidade autônoma
- Facilitar adoção de novas tecnologias por componente

**DECISÃO:**
Adotar arquitetura de microsserviços com comunicação híbrida:
1. Comunicação síncrona: gRPC para chamadas diretas entre serviços
2. Comunicação assíncrona: Apache Kafka para eventos de domínio
3. API Gateway: Kong para roteamento e gerenciamento de APIs
4. Service Mesh: Linkerd para comunicação entre serviços
5. Banco de dados por serviço (Database-per-service)

**OPÇÕES CONSIDERADAS:**
- **Opção A (Monólito Modularizado):** Manter arquitetura monolítica com módulos bem definidos
  - Prós: Simplicidade de deploy, debugging mais fácil
  - Contras: Escalabilidade limitada, acoplamento alto, deploy único
- **Opção B (Microsserviços REST):** Microsserviços com comunicação REST/HTTP
  - Prós: Padrão amplamente adotado, fácil entendimento
  - Contras: Overhead de serialização JSON, sem contrato forte, versionamento complexo
- **Opção C (Microsserviços gRPC + Eventos):** Microsserviços com gRPC para síncrono e Kafka para assíncrono
  - Prós: Performance superior, contrato forte com Protobuf, streaming bidirecional
  - Contras: Curva de aprendizado, tooling menos maduro que REST

**Opção escolhida: C** - Justificativa: Performance crítica para operações autônomas, necessidade de contratos fortes para evitar breaking changes, suporte nativo a streaming para operações em tempo real.

**CONSEQUÊNCIAS:**
**Positivas:**
- Escalabilidade independente por serviço
- Deploy contínuo sem downtime
- Isolamento de falhas
- Polyglot persistence (cada serviço escolhe seu banco ideal)
- Times podem trabalhar em paralelo no Modo Hive

**Negativas:**
- Complexidade operacional aumentada
- Necessidade de orquestração (Kubernetes)
- Latência de rede entre serviços
- Gerenciamento distribuído de transações

**Riscos:**
1. **Distributed monolith:** Mitigação: Definir boundaries de domínio claros, evitar acoplamento
2. **Cascading failures:** Mitigação: Implementar circuit breakers, timeouts, retries
3. **Data consistency:** Mitigação: Event sourcing, sagas, compensação

**REVISÃO:** 2024-04-15 (3 meses após implementação)

@contextmanager
def get_db():
    """Context manager for database sessions with automatic cleanup."""
    db = SessionLocal()
    correlation_id = structlog.contextvars.get_contextvars().get("correlation_id", "unknown")
    
    try:
        logger.info("database.session_started", correlation_id=correlation_id)
        yield db
        db.commit()
        logger.info("database.session_committed", correlation_id=correlation_id)
    except Exception as e:
        db.rollback()
        logger.error(
            "database.session_rollback",
            correlation_id=correlation_id,
            error=str(e),
            error_type=type(e).__name__
        )
        raise
    finally:
        db.close()
        logger.info("database.session_closed", correlation_id=correlation_id)

from circuitbreaker import circuit
import time
import structlog
from typing import Callable, Any
from functools import wraps

logger = structlog.get_logger()

class CircuitBreaker:
    """Enhanced circuit breaker with metrics and fallback support."""
    
    def __init__(
        self,
        failure_threshold: int = 5,
        recovery_timeout: int = 60,
        expected_exception: tuple = (Exception,),
        name: str = "default"
    ):
        self.failure_threshold = failure_threshold
        self.recovery_timeout = recovery_timeout
        self.expected_exception = expected_exception
        self.name = name
        self.metrics = {
            "total_calls": 0,
            "successful_calls": 0,
            "failed_calls": 0,
            "circuit_opened": 0,
            "fallback_used": 0
        }
    
    def __call__(self, fallback_func: Callable = None):
        """Decorator for circuit breaker with fallback."""
        
        def decorator(func: Callable):
            @wraps(func)
            def wrapper(*args, **kwargs):
                self.metrics["total_calls"] += 1
                correlation_id = structlog.contextvars.get_contextvars().get("correlation_id", "unknown")
                
                try:
                    # Apply circuit breaker
                    @circuit(
                        failure_threshold=self.failure_threshold,
                        recovery_timeout=self.recovery_timeout,
                        expected_exception=self.expected_exception,
                        name=self.name
                    )
                    def protected_func():
                        return func(*args, **kwargs)
                    
                    result = protected_func()
                    self.metrics["successful_calls"] += 1
                    logger.info(
                        "circuit_breaker.success",
                        circuit_name=self.name,
                        correlation_id=correlation_id
                    )
                    return result
                    
                except Exception as e:
                    self.metrics["failed_calls"] += 1
                    logger.error(
                        "circuit_breaker.failure",
                        circuit_name=self.name,
                        correlation_id=correlation_id,
                        error=str(e),
                        error_type=type(e).__name__
                    )
                    
                    # Use fallback if provided
                    if fallback_func:
                        self.metrics["fallback_used"] += 1
                        logger.warning(
                            "circuit_breaker.fallback_activated",
                            circuit_name=self.name,
                            correlation_id=correlation_id
                        )
                        return fallback_func(*args, **kwargs)
                    
                    raise
            
            return wrapper
        return decorator
    
    def get_metrics(self) -> dict:
        """Get circuit breaker metrics."""
        return {
            **self.metrics,
            "success_rate": (
                self.metrics["successful_calls"] / self.metrics["total_calls"]
                if self.metrics["total_calls"] > 0 else 0
            ),
            "name": self.name,
            "timestamp": time.time()
        }

        return pwd_context.verify(plain_password, hashed_password)
    except Exception as e:
        logger.error("security.password_verification_failed", error=str(e))
        return False

def get_password_hash(password: str) -> str:
    """Generate password hash."""
    return pwd_context.hash(password)

def create_access_token(
    data: Dict[str, Any],
    expires_delta: Optional[timedelta] = None
) -> str:
    """Create JWT access token."""
    to_encode = data.copy()
    
    if expires_delta:
        expire = datetime.utcnow() + expires_delta
    else:
        expire = datetime.utcnow() + timedelta(minutes=settings.ACCESS_TOKEN_EXPIRE_MINUTES)
    
    to_encode.update({"exp": expire, "type": "access"})
    encoded_jwt = jwt.encode(to_encode, settings.SECRET_KEY, algorithm=settings.ALGORITHM)
    
    logger.info("security.access_token_created", user_id=data.get("sub"))
    return encoded_jwt

def create_refresh_token(
    data: Dict[str, Any],
    expires_delta: Optional[timedelta] = None
) -> str:
    """Create JWT refresh token."""
    to_encode = data.copy()
    
    if expires_delta:
        expire = datetime.utcnow() + expires_delta
    else:
        expire = datetime.utcnow() + timedelta(days=settings.REFRESH_TOKEN_EXPIRE_DAYS)
    
    to_encode.update({"exp": expire, "type": "refresh"})
    encoded_jwt = jwt.encode(to_encode, settings.SECRET_KEY, algorithm=settings.ALGORITHM)
    
    logger.info("security.refresh_token_created", user_id=data.get("sub"))
    return encoded_jwt

def verify_token(token: str) -> Optional[Dict[str, Any]]:
    """Verify JWT token and return payload."""
    try:
        payload = jwt.decode(token, settings.SECRET_KEY, algorithms=[settings.ALGORITHM])
        return payload
    except JWTError as e:
        logger.error("security.token_verification_failed", error=str(e))
        return None

def validate_token_type(payload: Dict[str, Any], expected_type: str) -> bool:
    """Validate token type."""
    token_type = payload.get("type")
    if token_type != expected_type:
        logger.warning(
            "security.invalid_token_type",
            expected=expected_type,
            actual=token_type
        )
        return False
    return True

```python
from datetime import datetime, timedelta
from airflow import DAG
from airflow.operators.python import PythonOperator
from airflow.operators.bash import BashOperator
from airflow.providers.postgres.operators.postgres import PostgresOperator
from airflow.utils.task_group import TaskGroup
from src.data_pipeline.extractors import extract_events, extract_llm_calls
from src.data_pipeline.transformers import transform_events, transform_llm_calls
from src.data_pipeline.loaders import load_to_staging
from src.monitoring.alerts import send_pipeline_alert

default_args = {
    'owner': 'data_engineer',
    'depends_on_past': False,
    'start_date': datetime(2024, 1, 1),
    'email_on_failure': True,
    'email_on_retry': False,
    'retries': 3,
    'retry_delay': timedelta(minutes=5),
    'on_failure_callback': lambda context: send_pipeline_alert(
        pipeline_name=context['dag'].dag_id,
        error=str(context['exception']),
        execution_date=context['execution_date']
    )
}

with DAG(
    'etl_main_pipeline',
    default_args=default_args,
    description='ETL principal para dados DIYAPP',
    schedule_interval='@hourly',
    catchup=False,
    max_active_runs=1,
    tags=['etl', 'data_warehouse']
) as dag:

    # Task para notificar início do pipeline
    start_notification = PythonOperator(
        task_id='start_notification',
        python_callable=lambda: print(f"Pipeline iniciado em {datetime.now()}")
    )

    with TaskGroup('extraction_group') as extraction_group:
        # Extração de eventos do produto
        extract_product_events = PythonOperator(
            task_id='extract_product_events',
            python_callable=extract_events,
            op_kwargs={
                'source': 'product_api',
                'start_time': '{{ data_interval_start }}',
                'end_time': '{{ data_interval_end }}'
            }
        )

        # Extração de chamadas LLM
        extract_llm_metrics = PythonOperator(
            task_id='extract_llm_metrics',
            python_callable=extract_llm_calls,
            op_kwargs={
                'source': 'llm_gateway',
                'start_time': '{{ data_interval_start }}',
                'end_time': '{{ data_interval_end }}'
            }
        )

        # Extração de dados de usuários
        extract_user_data = PythonOperator(
            task_id='extract_user_data',
            python_callable=extract_events,
            op_kwargs={
                'source': 'user_service',
                'start_time': '{{ data_interval_start }}',
                'end_time': '{{ data_interval_end }}'
            }
        )

    with TaskGroup('transformation_group') as transformation_group:
        # Transformação de eventos
        transform_events_task = PythonOperator(
            task_id='transform_events',
            python_callable=transform_events,
            op_kwargs={
                'input_data': '{{ ti.xcom_pull(task_ids="extraction_group.extract_product_events") }}'
            }
        )

        # Transformação de métricas LLM
        transform_llm_task = PythonOperator(
            task_id='transform_llm_metrics',
            python_callable=transform_llm_calls,
            op_kwargs={
                'input_data': '{{ ti.xcom_pull(task_ids="extraction_group.extract_llm_metrics") }}'
            }
        )

    with TaskGroup('loading_group') as loading_group:
        # Carregamento para staging
        load_events_staging = PythonOperator(
            task_id='load_events_staging',
            python_callable=load_to_staging,
            op_kwargs={
                'table_name': 'stg_events',
                'data': '{{ ti.xcom_pull(task_ids="transformation_group.transform_events") }}'
            }
        )

        load_llm_staging = PythonOperator(
            task_id='load_llm_staging',
            python_callable=load_to_staging,
            op_kwargs={
                'table_name': 'stg_llm_calls',
                'data': '{{ ti.xcom_pull(task_ids="transformation_group.transform_llm_metrics") }}'
            }
        )

    # Execução do dbt para transformações
    run_dbt_staging = BashOperator(
        task_id='run_dbt_staging',
        bash_command='cd /opt/airflow/dbt && dbt run --models staging',
        env={
            'DBT_PROFILES_DIR': '/opt/airflow/dbt',
            'PGPASSWORD': 'diyapp123'
        }
    )

    run_dbt_marts = BashOperator(
        task_id='run_dbt_marts',
        bash_command='cd /opt/airflow/dbt && dbt run --models marts',
        env={
            'DBT_PROFILES_DIR': '/opt/airflow/dbt',
            'PGPASSWORD': 'diyapp123'
        }
    )

    # Testes de qualidade de dados
    run_data_tests = BashOperator(
        task_id='run_data_tests',
        bash_command='cd /opt/airflow/dbt && dbt test',
        env={
            'DBT_PROFILES_DIR': '/opt/airflow/dbt',
            'PGPASSWORD': 'diyapp123'
        }
    )

    # Atualização do cache Redis
    update_cache = PythonOperator(
        task_id='update_cache',
        python_callable='''from src.cache_manager.redis_client import CacheManager
cache = CacheManager()
cache.refresh_aggregated_metrics()''',
    )

    # Task para notificar sucesso
    success_notification = PythonOperator(
        task_id='success_notification',
        python_callable=lambda: print(f"Pipeline concluído com sucesso em {datetime.now()}")
    )

    # Definir dependências
    start_notification >> extraction_group
    extraction_group >> transformation_group
    transformation_group >> loading_group
    loading_group >> run_dbt_staging
    run_dbt_staging >> run_dbt_marts
    run_dbt_marts >> run_data_tests
    run_data_tests >> update_cache
    update_cache >> success_notification
```

class DIYAppWAF:
    """WAF com regras baseadas em STRIDE framework"""
    
    def __init__(self, config_path: str = "config/waf_rules.yaml"):
        self.rules = self._load_rules(config_path)
        self.blocked_ips = {}
        self.request_log = []
        self.logger = logging.getLogger("waf")
        
        # Thresholds para rate limiting
        self.rate_limits = {
            "auth": {"max_attempts": 5, "window_minutes": 15},
            "api": {"max_requests": 100, "window_minutes": 1},

    def _load_rules(self, config_path: str) -> Dict:
        """Carrega regras WAF do arquivo de configuração"""
        default_rules = {
            "sql_injection": {
                "enabled": True,
                "patterns": [
                    r"(?i)(union\s+select|select\s+.*from|insert\s+into|delete\s+from|drop\s+table|--|\/\*)",
                    r"(?i)(sleep\(|benchmark\(|waitfor\s+delay)",
                    r"(?i)(exec\(|xp_cmdshell|sp_configure)"
                ],
                "action": "block"
            },
            "xss": {
                "enabled": True,
                "patterns": [
                    r"<script[^>]*>.*?</script>",
                    r"javascript:",
                    r"on\w+\s*=",
                    r"alert\(|confirm\(|prompt\("
                ],
                "action": "sanitize"
            },
            "path_traversal": {
                "enabled": True,
                "patterns": [
                    r"\.\./|\.\.\\",
                    r"etc/passwd",
                    r"windows/win\.ini"
                ],
                "action": "block"
            },
            "command_injection": {
                "enabled": True,
                "patterns": [
                    r"[;&|`]\s*(ls|cat|rm|wget|curl|nc|python|perl)",
                    r"\$\{.*\}",
                    r"\(.*\)"
                ],
                "action": "block"
            },

                "enabled": True,
                "patterns": [
                    r"(?i)(ignore previous|system prompt|as an ai|you are now|roleplay)",
                    r"(?i)(disregard|override|break character|forget instructions)",
                    r"```system.*?```"
                ],
                "action": "sanitize"
            }
        }
        
        try:
            if os.path.exists(config_path):
                with open(config_path, 'r') as f:
                    return yaml.safe_load(f)
            else:
                # Cria arquivo de configuração padrão
                os.makedirs(os.path.dirname(config_path), exist_ok=True)
                with open(config_path, 'w') as f:
                    yaml.dump(default_rules, f, default_flow_style=False)
                return default_rules
        except Exception as e:
            self.logger.error(f"Erro ao carregar regras WAF: {e}")
            return default_rules
    
    def inspect_request(self, request_data: Dict) -> Tuple[bool, str, Dict]:
        """
        Inspeciona requisição contra regras WAF
        
        Args:
            request_data: Dict com method, path, headers, params, body
            
        Returns:
            Tuple (is_allowed, reason, details)
        """
        client_ip = request_data.get('client_ip', '0.0.0.0')
        path = request_data.get('path', '')
        method = request_data.get('method', 'GET')
        params = request_data.get('params', {})
        headers = request_data.get('headers', {})
        body = request_data.get('body', '')
        
        # Verifica IP bloqueado
        if self._is_ip_blocked(client_ip):
            return False, "IP_BLOCKED", {"ip": client_ip, "reason": "Rate limit exceeded"}
        
        # Rate limiting por endpoint
        if not self._check_rate_limit(client_ip, path):
            self._block_ip(client_ip, "Rate limit exceeded")
            return False, "RATE_LIMIT_EXCEEDED", {"ip": client_ip, "path": path}
        
        # Constrói string para análise
        analysis_string = f"{method} {path} {json.dumps(params)} {json.dumps(headers)} {body}"
        
        violations = []
        
        # Aplica regras
        for rule_name, rule_config in self.rules.items():
            if not rule_config.get('enabled', True):
                continue
                
            for pattern in rule_config.get('patterns', []):
                import re
                if re.search(pattern, analysis_string, re.IGNORECASE | re.DOTALL):
                    violations.append({
                        "rule": rule_name,
                        "pattern": pattern,
                        "action": rule_config.get('action', 'block')
                    })
                    
                    if rule_config.get('action') == 'block':
                        self.logger.warning(f"WAF Block: {rule_name} - IP: {client_ip} - Path: {path}")
                        self._block_ip(client_ip, f"WAF rule violation: {rule_name}")
                        return False, "WAF_VIOLATION", {
                            "ip": client_ip,
                            "rule": rule_name,
                            "violations": violations
                        }
        
        # Log da requisição (mantém apenas últimas 1000)
        self.request_log.append({
            "timestamp": datetime.utcnow().isoformat(),
            "ip": client_ip,
            "path": path,
            "method": method,
            "violations": violations
        })
        
        if len(self.request_log) > 1000:
            self.request_log = self.request_log[-1000:]
        
        return True, "ALLOWED", {"violations": violations}
    
    def _is_ip_blocked(self, ip: str) -> bool:
        """Verifica se IP está bloqueado"""
        if ip in self.blocked_ips:
            block_until = self.blocked_ips[ip]
            if datetime.utcnow() < block_until:
                return True
            else:
                del self.blocked_ips[ip]
        return False
    
    def _block_ip(self, ip: str, reason: str, minutes: int = 30):
        """Bloqueia IP temporariamente"""
        self.blocked_ips[ip] = datetime.utcnow() + timedelta(minutes=minutes)
        self.logger.warning(f"IP {ip} bloqueado por {minutes} minutos. Razão: {reason}")
    
    def _check_rate_limit(self, ip: str, path: str) -> bool:

        endpoint_type = self._get_endpoint_type(path)
        limits = self.rate_limits.get(endpoint_type, {})
        
        if not limits:
            return True
        
        window_start = datetime.utcnow() - timedelta(minutes=limits.get('window_minutes', 1))
        recent_requests = [
            r for r in self.request_log 
            if r['ip'] == ip and 
            r['path'].startswith(path.split('?')[0]) and
            datetime.fromisoformat(r['timestamp']) > window_start
        ]
        
        max_allowed = limits.get('max_requests', limits.get('max_attempts', 100))
        
        return len(recent_requests) < max_allowed
    
    def _get_endpoint_type(self, path: str) -> str:
        """Classifica endpoint para rate limiting"""
        if '/auth/' in path or '/login' in path or '/register' in path:
            return 'auth'
        elif '/upload' in path or '/file' in path:
            return 'uploads'
        else:
            return 'api'
    
    def generate_waf_report(self) -> Dict:

        return {

            "recent_violations": [
                r for r in self.request_log[-100:] if r['violations']
            ],
            "active_rules": {k: v['enabled'] for k, v in self.rules.items()}
        }

class DIYAppVault:
    """Sistema de gerenciamento de secrets compatível com HashiCorp Vault"""
    
    def __init__(self, vault_path: str = "data/vault.db", master_key_path: str = "config/master.key"):
        self.vault_path = vault_path
        self.master_key_path = master_key_path
        self.master_key = self._load_or_create_master_key()
        self.logger = logging.getLogger("vault")
        
        # Inicializa banco de dados
        self._init_db()
    
    def _load_or_create_master_key(self) -> bytes:
        """Carrega ou cria master key para criptografia"""
        os.makedirs(os.path.dirname(self.master_key_path), exist_ok=True)
        
        if os.path.exists(self.master_key_path):
            with open(self.master_key_path, 'rb') as f:
                return f.read()
        else:
            # Gera nova master key
            key = Fernet.generate_key()
            with open(self.master_key_path, 'wb') as f:
                f.write(key)
            
            # Define permissões restritivas (apenas owner pode ler)
            os.chmod(self.master_key_path, 0o600)
            
            self.logger.info("Nova master key gerada")
            return key
    
    def _init_db(self):
        """Inicializa banco de dados do vault"""
        conn = sqlite3.connect(self.vault_path)
        cursor = conn.cursor()
        
        # Tabela de secrets
        cursor.execute('''
            CREATE TABLE IF NOT EXISTS secrets (
                id TEXT PRIMARY KEY,
                path TEXT NOT NULL,
                data TEXT NOT NULL,
                version INTEGER DEFAULT 1,
                created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
                updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
                created_by TEXT,
                metadata TEXT,
                UNIQUE(path, version)
            )
        ''')
        
        # Tabela de políticas de acesso
        cursor.execute('''
            CREATE TABLE IF NOT EXISTS policies (
                id TEXT PRIMARY KEY,
                name TEXT NOT NULL,
                rules TEXT NOT NULL,
                created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
            )
        ''')
        
        # Tabela de tokens
        cursor.execute('''
            CREATE TABLE IF NOT EXISTS tokens (
                token TEXT PRIMARY KEY,
                policy_id TEXT,
                expires_at TIMESTAMP,
                created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
                last_used TIMESTAMP,
                FOREIGN KEY (policy_id) REFERENCES policies (id)
            )
        ''')
        
        # Tabela de auditoria
        cursor.execute('''
            CREATE TABLE IF NOT EXISTS audit_log (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                timestamp TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
                action TEXT NOT NULL,
                path TEXT,
                token TEXT,
                client_ip TEXT,
                success BOOLEAN
            )
        ''')
        
        conn.commit()
        conn.close()
    
    def _encrypt_data(self, data: str) -> str:
        """Criptografa dados usando Fernet"""
        fernet = Fernet(self.master_key)
        encrypted = fernet.encrypt(data.encode())
        return base64.b64encode(encrypted).decode()
    
    def _decrypt_data(self, encrypted_data: str) -> str:
        """Descriptografa dados"""
        fernet = Fernet(self.master_key)
        decrypted = fernet.decrypt(base64.b64decode(encrypted_data))
        return decrypted.decode()
    
    def store_secret(self, path: str, data: Dict, token: str, created_by: str = "system") -> bool:
        """Armazena secret no vault"""
        if not self._validate_token(token, path, "write"):
            self._log_audit("write", path, token, False)
            return False
        
        conn = sqlite3.connect(self.vault_path)
        cursor = conn.cursor()
        
        try:
            # Obtém próxima versão
            cursor.execute(
                "SELECT MAX(version) FROM secrets WHERE path = ?",
                (path,)
            )
            result = cursor.fetchone()
            version = (result[0] or 0) + 1
            
            # Criptografa dados
            encrypted_data = self._encrypt_data(json.dumps(data))
            secret_id = hashlib.sha256(f"{path}:{version}".encode()).hexdigest()[:32]
            
            cursor.execute('''
                INSERT INTO secrets (id, path, data, version, created_by)
                VALUES (?, ?, ?, ?, ?)
            ''', (secret_id, path, encrypted_data, version, created_by))
            
            conn.commit()
            
            self._log_audit("write", path, token, True)
            self.logger.info(f"Secret armazenado: {path} v{version}")
            return True
            
        except Exception as e:
            self.logger.error(f"Erro ao armazenar secret: {e}")
            return False
        finally:
            conn.close()
    
    def get_secret(self, path: str, token: str, version: Optional[int] = None) -> Optional[Dict]:
        """Recupera secret do vault"""
        if not self._validate_token(token, path, "read"):
            self._log_audit("read", path, token, False)
            return None
        
        conn = sqlite3.connect(self.vault_path)
        cursor = conn.cursor()
        
        try:
            if version:
                cursor.execute(
                    "SELECT data FROM secrets WHERE path = ? AND version = ?",
                    (path, version)
                )
            else:
                cursor.execute(
                    "SELECT data FROM secrets WHERE path = ? ORDER BY version DESC LIMIT 1",
                    (path,)
                )
            
            result = cursor.fetchone()
            
            if result:
                decrypted_data = self._decrypt_data(result[0])
                self._log_audit("read", path, token, True)
                return json.loads(decrypted_data)
            else:
                return None
                
        except Exception as e:
            self.logger.error(f"Erro ao recuperar secret: {e}")
            return None
        finally:
            conn.close()
    
    def list_secrets(self, token: str, prefix: str = "") -> List[str]:
        """Lista secrets disponíveis"""
        if not self._validate_token(token, prefix, "list"):
            return []
        
        conn = sqlite3.connect(self.vault_path)
        cursor = conn.cursor()
        
        try:
            cursor.execute(
                "SELECT DISTINCT path FROM secrets WHERE path LIKE ?",
                (f"{prefix}%",)
            )
            results = cursor.fetchall()
            return [r[0] for r in results]
        finally:
            conn.close()
    
    def create_token(self, policy_id: str, ttl_hours: int = 24) -> Optional[str]:
        """Cria novo token de acesso"""
        token = base64.b64encode(os.urandom(32)).decode()[:32]
        expires_at = datetime.utcnow() + timedelta(hours=ttl_hours)
        
        conn = sqlite3.connect(self.vault_path)
        cursor = conn.cursor()
        
        try:
            cursor.execute('''
                INSERT INTO tokens (token, policy_id, expires_at)
                VALUES (?, ?, ?)
            ''', (token, policy_id, expires_at.isoformat()))
            
            conn.commit()
            return token
        except Exception as e:
            self.logger.error(f"Erro ao criar token: {e}")
            return None
        finally:
            conn.close()
    
    def _validate_token(self, token: str, path: str, action: str) -> bool:
        """Valida token e verifica permissões"""
        conn = sqlite3.connect(self.vault_path)
        cursor = conn.cursor()
        
        try:
            # Verifica token
            cursor.execute('''
                SELECT t.expires_at, t.policy_id, p.rules 
                FROM tokens t
                LEFT JOIN policies p ON t.policy_id = p.id
                WHERE t.token = ?
            ''', (token,))
            
            result = cursor.fetchone()
            
            if not result:
                return False
            
            expires_at, policy_id, rules_json = result
            
            # Verifica expiração

                return False

                "UPDATE tokens SET last_used = ? WHERE token = ?",
                (datetime.utcnow().isoformat(), token)
            )
            conn.commit()

            if not rules_json:
                return True

                if rule['path'] == path or path.startswith(rule['path']):
                    if action in rule['capabilities']:
                        return True
            
            return False
            
        except Exception as e:
            self.logger.error(f"Erro ao validar token: {e}")
