# DIYAPP Evolution - V12 Core - Documento de Arquitetura

## 1. Visão Geral da Arquitetura V12

### 1.1 Princípios Fundamentais
- **Autonomia Total**: Cada componente deve ser independente e resiliente
- **Zero Downtime**: Deploy contínuo com rollback automático
- **Observabilidade Nativa**: Métricas, logs e traces em todos os níveis
- **Segurança por Design**: Autenticação, autorização e criptografia integradas

### 1.2 Stack Tecnológica

#### Frontend
- **Framework**: React 18 com TypeScript
- **Estado**: Zustand (leve e escalável)
- **Roteamento**: React Router v6
- **UI/UX**: Tailwind CSS + Headless UI
- **Testes**: Vitest + React Testing Library + Playwright (E2E)

#### Backend
- **Runtime**: Node.js 20 LTS
- **Framework**: Fastify (alta performance, baixa latência)
- **ORM**: Prisma (type-safe, migrações automáticas)
- **Autenticação**: JWT + Refresh Tokens com rotatória
- **Validação**: Zod (schema validation)

#### Banco de Dados
- **Primário**: PostgreSQL 15 (com replicação)
- **Cache**: Redis 7 (sessões e cache distribuído)
- **Fila**: BullMQ (Redis-based, confiável)

#### Infraestrutura
- **Containerização**: Docker + Docker Compose
- **Orquestração**: Kubernetes (para produção)
- **CI/CD**: GitHub Actions
- **Monitoramento**: Prometheus + Grafana + Loki
- **Logs**: Structured JSON logging

```typescript
// components/features/task/TaskList.tsx
import { useTaskStore } from '@/stores/taskStore';
import { TaskCard } from './TaskCard';
import { LoadingState } from '@/components/ui/LoadingState';
import { ErrorState } from '@/components/ui/ErrorState';

interface TaskListProps {
  projectId: string;
  onTaskSelect?: (taskId: string) => void;
}

export function TaskList({ projectId, onTaskSelect }: TaskListProps) {
  const { tasks, isLoading, error } = useTaskStore(projectId);
  
  if (isLoading) return <LoadingState />;
  if (error) return <ErrorState error={error} />;
  
  return (
    <div className="space-y-4">
      {tasks.map((task) => (
        <TaskCard
          key={task.id}
          task={task}
          onClick={() => onTaskSelect?.(task.id)}
        />
      ))}
    </div>
  );
}
```

### 3.4 Estrutura de Módulos Backend
```typescript
// modules/tasks/task.controller.ts
import { FastifyRequest, FastifyReply } from 'fastify';
import { taskService } from './task.service';
import { createTaskSchema, updateTaskSchema } from './task.schema';
import { AppError } from '@/core/errors/AppError';

export class TaskController {
  async createTask(request: FastifyRequest, reply: FastifyReply) {
    try {
      const validatedData = createTaskSchema.parse(request.body);
      const userId = request.user.id;
      
      const task = await taskService.createTask(userId, validatedData);
      
      return reply.code(201).send({
        success: true,
        data: task,
        message: 'Task created successfully'
      });
    } catch (error) {
      if (error instanceof AppError) {
        throw error;
      }
      throw new AppError('Failed to create task', 500);
    }
  }
  
  async getTasks(request: FastifyRequest, reply: FastifyReply) {
    const { projectId } = request.query as { projectId?: string };
    const userId = request.user.id;
    
    const tasks = await taskService.getUserTasks(userId, projectId);
    
    return reply.send({
      success: true,
      data: tasks
    });
  }
}
```

```yaml
        uses: azure/k8s-deploy@v4
        with:
          namespace: diyapp-staging
          manifests: |
            infrastructure/k8s/staging/
          images: |
            ${{ secrets.DOCKER_REGISTRY }}/diyapp-frontend:${{ github.sha }}
            ${{ secrets.DOCKER_REGISTRY }}/diyapp-backend:${{ github.sha }}
          
      - name: Run smoke tests
        run: |
          npm run test:smoke -- --url ${{ secrets.STAGING_URL }}

    runs-on: ubuntu-latest

          manifests: infrastructure/k8s/production/
          images: ${{ secrets.DOCKER_REGISTRY }}/diyapp-backend:${{ github.sha }}
```

```typescript
    level: (label) => ({ level: label }),
    bindings: () => ({ 
      pid: process.pid,
      hostname: process.env.HOSTNAME 
    })
  },
  timestamp: () => `,"time":"${new Date().toISOString()}"`
});

export default logger;

// Uso nos módulos
import logger from '@/core/logger/logger';

export class TaskService {
  async createTask(userId: string, data: CreateTaskDto) {
    logger.info({ userId, taskData: data }, 'Creating task');
    
    try {
      const task = await this.taskRepository.create(userId, data);
      logger.info({ taskId: task.id }, 'Task created successfully');
      return task;
    } catch (error) {
      logger.error({ error, userId }, 'Failed to create task');
      throw error;
    }
  }
}
```

```javascript
class Database {
  constructor() {
    this.pool = null;
    this.init();
  }

  init() {
    try {
      this.pool = new Pool({
        host: process.env.POSTGRES_HOST,
        port: parseInt(process.env.POSTGRES_PORT),
        database: process.env.POSTGRES_DB,
        user: process.env.POSTGRES_USER,
        password: process.env.POSTGRES_PASSWORD,
        max: 20,
        idleTimeoutMillis: 30000,
        connectionTimeoutMillis: 5000,
      });

      // Test connection
      this.pool.on('connect', () => {
        console.log('Database connected successfully');
      });

      this.pool.on('error', (err) => {
        console.error('Unexpected database error:', err);
        process.exit(-1);
      });

    } catch (error) {
      console.error('Database initialization failed:', error);
      throw error;
    }
  }

  async query(text, params) {
    const start = Date.now();
    try {
      const result = await this.pool.query(text, params);
      const duration = Date.now() - start;
      
      // Log slow queries
      if (duration > 100) {
        console.warn(`Slow query detected: ${duration}ms`, { text });
      }
      
      return result;
    } catch (error) {
      console.error('Database query error:', {
        query: text,
        params: params ? params.map(p => typeof p === 'string' ? p.substring(0, 100) : p) : [],
        error: error.message
      });
      throw error;
    }
  }

  async getClient() {
    const client = await this.pool.connect();
    const query = client.query;
    const release = client.release;
    
    // Set a timeout of 5 seconds
    const timeout = setTimeout(() => {
      console.error('Client has been checked out for more than 5 seconds');
    }, 5000);
    
    client.release = () => {
      clearTimeout(timeout);
      client.query = query;
      client.release = release;
      return release.apply(client);
    };
    
    return client;
  }
}

module.exports = new Database();
```

```javascript
async function runMigrations() {
  const client = await database.getClient();
  
  try {
    await client.query('BEGIN');
    
    // Create migrations table if not exists
    await client.query(`
      CREATE TABLE IF NOT EXISTS migrations (
        id SERIAL PRIMARY KEY,
        name VARCHAR(255) UNIQUE NOT NULL,
        executed_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
      )
    `);
    
    // Get executed migrations
    const executedResult = await client.query('SELECT name FROM migrations ORDER BY id');
    const executedMigrations = new Set(executedResult.rows.map(row => row.name));
    
    // Read migration files
    const migrationsDir = path.join(__dirname);
    const files = await fs.readdir(migrationsDir);
    const migrationFiles = files
      .filter(file => file.endsWith('.sql') && file !== 'run-migrations.js')
      .sort();
    
    // Execute pending migrations
    for (const file of migrationFiles) {
      if (!executedMigrations.has(file)) {
        console.log(`Executing migration: ${file}`);
        
        const migrationPath = path.join(migrationsDir, file);
        const migrationSQL = await fs.readFile(migrationPath, 'utf8');
        
        await client.query(migrationSQL);
        await client.query('INSERT INTO migrations (name) VALUES ($1)', [file]);
        
        console.log(`✓ Migration ${file} executed successfully`);
      }
    }
    
    await client.query('COMMIT');
    console.log('All migrations completed successfully');
    
  } catch (error) {
    await client.query('ROLLBACK');
    console.error('Migration failed:', error);
    throw error;
  } finally {
    client.release();
  }
}
```

```javascript
        // Verify user still exists and is active

        if (!user) {

            error: 'User not found',
            code: 'USER_NOT_FOUND'
          });
        }
        
        next();
      } catch (jwtError) {

          error: 'Invalid or expired token',
          code: 'INVALID_TOKEN'
```

```javascript
#!/usr/bin/env node

const http = require('http');

const options = {
  hostname: 'localhost',
  port: process.env.PORT || 3000,
  path: '/health',
  method: 'GET',
  timeout: 5000,
};

const req = http.request(options, (res) => {
  if (res.statusCode === 200) {
    let data = '';
    res.on('data', (chunk) => {
      data += chunk;
    });
    res.on('end', () => {
      try {
        const health = JSON.parse(data);
        if (health.status === 'healthy' && health.timestamp) {
          console.log('Health check passed');
          process.exit(0);
        } else {
          console.error('Health check failed: Invalid response');
          process.exit(1);
        }
      } catch (error) {
        console.error('Health check failed: JSON parse error', error);
        process.exit(1);
      }
    });
  } else {
    console.error(`Health check failed: Status ${res.statusCode}`);
    process.exit(1);
  }
});

req.on('error', (error) => {
  console.error('Health check failed: Connection error', error.message);
  process.exit(1);
});

req.on('timeout', () => {
  console.error('Health check failed: Timeout');
  req.destroy();
  process.exit(1);
});

req.end();
```

      })
    };
  }

```javascript
    app.use(this.helmetConfig);

    app.use(cors(securityConfig.cors));

    app.use('/api/auth/login', this.rateLimiters.auth);

    app.use(express.json({ limit: '10mb' }));
    app.use(express.urlencoded({ extended: true, limit: '10mb' }));

    app.use(this._securityLogger());
  }

  _securityLogger() {
    return (req, res, next) => {
      const startTime = Date.now();
      
      res.on('finish', () => {
        const duration = Date.now() - startTime;
        const securityEvent = {
          timestamp: new Date().toISOString(),
          method: req.method,
          url: req.url,
          statusCode: res.statusCode,
          duration,
          userAgent: req.get('User-Agent'),
          ip: req.ip,

        };
```

```javascript
// auth/auth.service.js
const bcrypt = require('bcrypt');
const jwt = require('jsonwebtoken');
const crypto = require('crypto');
const { promisify } = require('util');
const securityConfig = require('../security/config');

class AuthService {
  constructor() {
    this.bcryptCompare = promisify(bcrypt.compare);
    this.bcryptHash = promisify(bcrypt.hash);
    this.commonPasswords = new Set([
      'password', '123456', 'qwerty', 'admin', 'welcome'
    ]);
  }

  async validatePassword(password) {
    // Password policy enforcement
    if (password.length < securityConfig.password.minLength) {
      throw new Error(`Password must be at least ${securityConfig.password.minLength} characters`);
    }
    
    if (securityConfig.password.requireSpecialChar && !/[!@#$%^&*(),.?":{}|<>]/.test(password)) {
      throw new Error('Password must contain at least one special character');
    }
    
    if (securityConfig.password.requireNumbers && !/\d/.test(password)) {
      throw new Error('Password must contain at least one number');
    }
    
    if (securityConfig.password.requireMixedCase && !/(?=.*[a-z])(?=.*[A-Z])/.test(password)) {
      throw new Error('Password must contain both uppercase and lowercase letters');
    }
    
    if (securityConfig.password.blockCommonPasswords && this.commonPasswords.has(password.toLowerCase())) {
      throw new Error('Password is too common, please choose a stronger one');
    }
    
    return true;
  }

  async hashPassword(password) {
    await this.validatePassword(password);
    return this.bcryptHash(password, securityConfig.password.bcryptRounds);
  }

  async verifyPassword(password, hash) {
    return this.bcryptCompare(password, hash);
  }

  generateTokens(user) {
    const accessToken = jwt.sign(
      {
        sub: user.id,
        email: user.email,
        role: user.role,
        type: 'access'
      },
      process.env.JWT_SECRET || securityConfig.jwt.privateKeyPath,
      { 
        algorithm: securityConfig.jwt.algorithm,
        expiresIn: securityConfig.jwt.accessTokenExpiry
      }
    );

    const refreshToken = crypto.randomBytes(40).toString('hex');
    const refreshTokenHash = crypto
      .createHash('sha256')
      .update(refreshToken)
      .digest('hex');

    return {
      accessToken,
      refreshToken,
      refreshTokenHash,
      accessTokenExpiry: new Date(Date.now() + 15 * 60 * 1000), // 15 minutes
      refreshTokenExpiry: new Date(Date.now() + 7 * 24 * 60 * 60 * 1000) // 7 days
    };
  }

  async verifyToken(token) {
    try {
      return jwt.verify(
        token,
        process.env.JWT_SECRET || securityConfig.jwt.publicKeyPath,
        { algorithms: [securityConfig.jwt.algorithm] }
      );
    } catch (error) {
      if (error.name === 'TokenExpiredError') {
        throw new Error('Token expired');
      }
      throw new Error('Invalid token');
    }
  }

  async invalidateToken(token) {
```

## 🏗️ Arquitetura

### Visão Geral

DIYAPP V12 Core é uma aplicação Node.js baseada em Express com arquitetura modular. O sistema é dividido em:

1. **Camada de Apresentação**: Frontend HTML/CSS/JS + APIs REST
2. **Camada de Aplicação**: Controladores + Serviços
3. **Camada de Domínio**: Modelos + Lógica de Negócio
4. **Camada de Infraestrutura**: Banco de Dados + Serviços Externos

### Fluxo de Requisição

```
Cliente → Middleware → Roteador → Controlador → Serviço → Modelo → Banco de Dados
```

### Componentes Principais

- **Express.js**: Framework web
- **PostgreSQL/SQLite**: Banco de dados
- **Winston**: Sistema de logging
- **Jest**: Framework de testes
- **Docker**: Containerização

## 🗄️ Banco de Dados

### PostgreSQL (Padrão)

```sql
-- Estrutura básica das tabelas
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    email VARCHAR(255) UNIQUE NOT NULL,
    name VARCHAR(255),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE sessions (
    id SERIAL PRIMARY KEY,
    user_id INTEGER REFERENCES users(id),
    token TEXT NOT NULL,
    expires_at TIMESTAMP NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### SQLite (Alternativa)

Para desenvolvimento ou ambientes leves, use SQLite:

```bash
# No .env
DB_TYPE=sqlite
SQLITE_PATH=./data/database.sqlite
```

## 🚨 Troubleshooting

### Problemas Comuns

#### 1. Erro de Conexão com Banco de Dados
```bash
# Verifique se o PostgreSQL está rodando
sudo service postgresql status

# Verifique credenciais no .env
# Teste conexão manual
psql -h localhost -U postgres -d diyapp_v12
```

#### 2. Porta 3000 em Uso
```bash
# Encontre o processo
lsof -i :3000
# Mate o processo
kill -9 <PID>
```

#### 3. Erros de Dependência
```bash
# Limpe cache do npm
npm cache clean --force
# Reinstale dependências
rm -rf node_modules package-lock.json
npm install
```

#### 4. Docker não Inicia
```bash
# Verifique logs
docker-compose logs app
# Rebuild
docker-compose down
docker-compose build --no-cache
docker-compose up -d
```

### Logs de Depuração

```bash
# Modo verbose
npm run dev --verbose

# Logs da aplicação
tail -f logs/app.log

docker-compose logs -f app
```
