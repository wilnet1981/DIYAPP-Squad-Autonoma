## 2. ADR: Decisão Arquitetural para V13

```markdown
# ADR-001: Transição para Arquitetura de Microserviços

**Data:** 2024-01-15
**Status:** Aceita
**Autores:** Tech Lead, Especialista Infra, Especialista Backend

## CONTEXTO:
A arquitetura monolítica atual (V12) apresenta limitações significativas:
1. **Acoplamento forte**: Mudanças em um módulo exigem redeploy completo
2. **Escalabilidade limitada**: Não é possível escalar componentes individualmente
3. **Complexidade crescente**: Base de código com 45k+ linhas dificulta manutenção
4. **Resiliência fraca**: Falha em um componente pode derrubar todo o sistema

Forças em jogo:
- Necessidade de deploy independente por squad
- Requisitos de SLA 99.95% para serviços críticos
- Time-to-market para novas features precisa ser reduzido em 40%
- Orçamento limitado para infraestrutura

## DECISÃO:
Adotar arquitetura baseada em microserviços com os seguintes princípios:

### Princípios Fundamentais:
1. **Domain-Driven Design**: Cada serviço corresponde a um bounded context
2. **Independent Deployability**: Cada serviço pode ser implantado independentemente
3. **Decentralized Data Management**: Cada serviço gerencia seu próprio banco de dados
4. **Resilience by Design**: Circuit breakers, retries, fallbacks implementados

### Stack Tecnológica:
- **API Gateway**: Kong (open-source, extensível)
- **Service Mesh**: Istio (para observabilidade e segurança)
- **Message Broker**: RabbitMQ (para comunicação assíncrona)
- **Service Discovery**: Consul
- **Container Orchestration**: Kubernetes
- **Monitoring**: Prometheus + Grafana + Jaeger

## OPÇÕES CONSIDERADAS:

### Opção A: Monolito Modularizado (Incremental)
- **Prós**: Menor risco, curva de aprendizado suave, reutilização de código
- **Contras**: Não resolve problemas fundamentais de acoplamento, escalabilidade limitada

### Opção B: Microserviços Completos (Revolucionário)
- **Prós**: Máxima flexibilidade, escalabilidade independente, resiliência
- **Contras**: Complexidade operacional alta, custo inicial elevado, necessidade de nova expertise

### Opção C: Arquitetura Híbrida (Strangler Pattern)
- **Prós**: Migração gradual, risco controlado, permite aprendizado incremental
- **Contras**: Overhead de manutenção durante transição, complexidade temporária

**Opção escolhida: C** — Justificativa:
- Balanceia benefícios de microserviços com gerenciamento de risco
- Permite migração sem downtime
- Time pode aprender gradualmente
- Custo controlado por fases

## CONSEQUÊNCIAS:

### Positivas:
1. **Escalabilidade**: Componentes podem escalar independentemente
2. **Resiliência**: Falhas isoladas não derrubam sistema inteiro
3. **Velocidade**: Squads podem desenvolver e deployar independentemente
4. **Tecnologia**: Flexibilidade para usar stack adequada por domínio

### Negativas (Trade-offs aceitos):
1. **Complexidade**: Sistema distribuído é inerentemente mais complexo
2. **Latência**: Comunicação entre serviços adiciona overhead
3. **Operações**: Requer expertise em DevOps e SRE
4. **Debugging**: Tracing distribuído necessário

### Riscos e Mitigações:
1. **Risco**: Complexidade operacional excessiva
   **Mitigação**: Investir em automação desde o início (GitOps, CI/CD)

2. **Risco**: Performance degradada por comunicação entre serviços
   **Mitigação**: Implementar cache agressivo, usar gRPC para comunicação interna

3. **Risco**: Inconsistência de dados
   **Mitigação**: Adotar Event Sourcing e Sagas para transações distribuídas

## REVISÃO:
Reavaliar decisão em 6 meses (2024-07-15) com métricas:
- Lead time para changes
- Deployment frequency
- Mean time to recovery (MTTR)
- Availability por serviço
```

/**
 * Estratégia de Cache Multi-camada para V13
 * L1: In-memory (Node.js) → L2: Redis Cluster → L3: Database
 */
class MultiLayerCacheStrategy {
  constructor() {
    this.config = {
      layers: {
        L1: {
          enabled: true,
          ttl: 60, // seconds
          maxSize: 1000,
          strategy: 'LRU'
        },
        L2: {
          enabled: true,
          ttl: 300, // seconds
          cluster: true,
          nodes: 3,
          persistence: 'AOF every sec'
        },
        L3: {
          enabled: true, // Database as last resort
          queryOptimization: true,
          materializedViews: true
        }
      },
      invalidation: {
        strategy: 'Write-Through + TTL',
        eventDriven: true,
        broadcastInvalidation: true
      }
    };
  }

  async get(key, options = {}) {
    const { bypassL1 = false, bypassL2 = false } = options;
    
    // 1. Try L1 Cache (In-memory)
    if (!bypassL1) {
      const l1Value = await this._getFromL1(key);
      if (l1Value !== null) {
        this._recordHit('L1');
        return l1Value;
      }
    }

    // 2. Try L2 Cache (Redis)
    if (!bypassL2) {
      const l2Value = await this._getFromL2(key);
      if (l2Value !== null) {
        // Populate L1 cache
        await this._setL1(key, l2Value);
        this._recordHit('L2');
        return l2Value;
      }
    }

    // 3. Fetch from L3 (Database)
    const l3Value = await this._getFromDatabase(key);
    if (l3Value !== null) {
      // Populate both caches
      await Promise.all([
        this._setL1(key, l3Value),
        this._setL2(key, l3Value)
      ]);
      this._recordHit('L3');
    }

    return l3Value;
  }

  async set(key, value, options = {}) {
    const { ttl, layers = ['L1', 'L2'] } = options;
    
    // Write-Through strategy
    const promises = [];
    
    if (layers.includes('L1')) {
      promises.push(this._setL1(key, value, ttl));
    }
    
    if (layers.includes('L2')) {
      promises.push(this._setL2(key, value, ttl));
    }
    
    // Always write to database
    promises.push(this._setDatabase(key, value));
    
    await Promise.all(promises);
    
    // Publish invalidation event for other instances
    if (this.config.invalidation.eventDriven) {
      await this._publishInvalidation(key);
    }
  }

  // Cache Patterns Implementation
  getCachePatterns() {
    return {
      cacheAside: {
        description: 'Lazy loading pattern',
        useCases: ['User profiles', 'Product catalog'],
        implementation: this._implementCacheAside.bind(this)
      },
      writeThrough: {
        description: 'Write to cache and DB simultaneously',
        useCases: ['Session data', 'Configuration'],
        implementation: this._implementWriteThrough.bind(this)
      },
      writeBehind: {
        description: 'Write to cache, batch DB writes',
        useCases: ['Analytics data', 'Audit logs'],
        implementation: this._implementWriteBehind.bind(this)
      },
      readThrough: {
        description: 'Cache handles DB read on miss',
        useCases: ['Reference data', 'Localization'],
        implementation: this._implementReadThrough.bind(this)
      }
    };
  }

  // Cache Invalidation Strategies
  getInvalidationStrategies() {
    return {
      timeBased: {
        ttl: 300,
        sliding: true,
        description: 'Automatic invalidation after TTL'
      },
      eventBased: {
        events: ['USER_UPDATED', 'PRODUCT_MODIFIED'],
        description: 'Invalidate on domain events'
      },
      manual: {
        endpoints: ['/api/cache/invalidate'],
        description: 'Manual invalidation API'
      },
      patternBased: {
        patterns: ['user:*', 'product:*:details'],
        description: 'Invalidate by key pattern'
      }
    };
  }

  // Monitoring and Metrics
  getMetricsDashboard() {
    return {
      hitRates: {
        L1: '85%',
        L2: '12%',
        L3: '3%'
      },
      latency: {
        L1: '0.2ms',
        L2: '2ms',
        L3: '50ms'
      },
      memoryUsage: {
        L1: '250MB',
        L2: '2GB',
        efficiency: '92%'
      },
      invalidation: {
        eventsPerSecond: 45,
        accuracy: '99.8%'
      }
    };
  }
}

module.exports = MultiLayerCacheStrategy;

    } catch (error: any) {
      return rejectWithValue(error.message || 'Login failed');

);

  'auth/logout',
  async (_, { rejectWithValue }) => {

      return rejectWithValue(error.message || 'Logout failed');
    }
  }
);

const authSlice = createSlice({
  name: 'auth',
  initialState,
  reducers: {
    clearError: (state) => {
      state.error = null;
    },
    setToken: (state, action: PayloadAction<string>) => {
      state.token = action.payload;
    },
  },
  extraReducers: (builder) => {
    builder
      // Login
      .addCase(login.pending, (state) => {
        state.isLoading = true;
        state.error = null;

        state.isLoading = false;
        state.user = action.payload.user;
        state.token = action.payload.token;
        state.lastLogin = new Date().toISOString();
      })
      .addCase(login.rejected, (state, action) => {
        state.isLoading = false;
        state.error = action.payload as string;
      })
      // Logout
      .addCase(logout.fulfilled, (state) => {
        state.user = null;
        state.token = null;
        state.lastLogin = null;
      });
  },
});

const metrics = require('./observability/metrics');

  async initializeObservability() {
    // Inicializa tracing distribuído
    initializeTracing();
    
    // Middleware de logging estruturado
    this.app.use(pinoHttp({
      logger,
      genReqId: (req) => req.headers['x-correlation-id'] || require('crypto').randomUUID(),

        req: (req) => ({
          method: req.method,
          url: req.url,

        }),
        res: (res) => ({
          statusCode: res.statusCode

  async initializeGraphQL() {
    this.apolloServer = new ApolloServer({
      typeDefs: graphqlSchema,
      resolvers: graphqlResolvers,
      context: ({ req }) => ({
        req,
        correlationId: req.correlationId,
        userId: req.user?.id,
        redis: this.redisClient,
        circuitBreakers: this.circuitBreakers
      }),
      plugins: [{
        async requestDidStart(requestContext) {
          const startTime = Date.now();
          const { request, context } = requestContext;
          
          return {
            async willSendResponse(responseContext) {
              const duration = Date.now() - startTime;
              const operationName = request.operationName || 'anonymous';
              
              logger.info({
                type: 'graphql',
                correlationId: context.correlationId,
                operationName,
                query: request.query?.substring(0, 200), // Log parcial por segurança
                variables: request.variables ? this.maskSensitiveVariables(request.variables) : {},
                duration,
                userId: context.userId
              }, 'GraphQL request completed');

              // Métrica de latência
              metrics.graphqlRequestDuration
                .labels(operationName, responseContext.response.http?.status.toString() || '200')
                .observe(duration / 1000);
            },
            async didEncounterErrors(errorContext) {
              errorContext.errors.forEach(error => {
                logger.error({
                  type: 'graphql_error',
                  correlationId: context.correlationId,
                  error: error.message,
                  path: error.path,
                  userId: context.userId
                }, 'GraphQL error');
              });
            }
          };
        }
      }],
      formatError: (error) => {
        // Nunca expor stack trace em produção
        const formattedError = {
          message: error.message,
          locations: error.locations,
          path: error.path
        };

        if (config.env === 'development') {
          formattedError.extensions = error.extensions;
        }

        return formattedError;
      }
    });

    await this.apolloServer.start();
    this.apolloServer.applyMiddleware({ 
      app: this.app,
      path: '/graphql',
      cors: false // Já tratado pelo middleware CORS
    });

    logger.info('GraphQL server initialized');
  }

  maskSensitiveVariables(variables) {
    const masked = { ...variables };
    const sensitiveFields = ['password', 'token', 'secret', 'creditCard', 'cvv'];
    
    sensitiveFields.forEach(field => {
      if (masked[field]) {
        masked[field] = '***';
      }
    });

    return masked;
  }

  initializeErrorHandling() {
    // 404 handler
    this.app.use((req, res) => {
      res.status(404).json({
        error: 'Not Found',
        message: `Cannot ${req.method} ${req.url}`,
        correlationId: req.correlationId
      });
    });

    // Global error handler
    this.app.use((error, req, res, next) => {
      logger.error({
        correlationId: req.correlationId,
        error: error.message,
        stack: error.stack,
        url: req.url,
        method: req.method,
        userId: req.user?.id
      }, 'Unhandled error');

      // Incrementa métrica de erro
      metrics.errorCounter.inc();

      res.status(error.status || 500).json({
        error: 'Internal Server Error',
        message: config.env === 'production' ? 'An unexpected error occurred' : error.message,
        correlationId: req.correlationId,
        ...(config.env === 'development' && { stack: error.stack })
      });
    });
  }

  async start() {
    try {
      await this.initializeGraphQL();
      
      this.server = this.app.listen(config.port, () => {
        logger.info({
          port: config.port,
          env: config.env,
          nodeVersion: process.version
        }, `DIYAPP V13 Core started successfully`);
        
        // Log de rotas disponíveis
        this.logAvailableRoutes();
      });

      // Graceful shutdown
      this.setupGracefulShutdown();

    } catch (error) {
      logger.error({ error }, 'Failed to start application');
      process.exit(1);
    }
  }

  logAvailableRoutes() {
    const routes = [];
    this.app._router.stack.forEach(middleware => {
      if (middleware.route) {
        routes.push({
          path: middleware.route.path,
          methods: Object.keys(middleware.route.methods)
        });
      } else if (middleware.name === 'router') {
        middleware.handle.stack.forEach(handler => {
          if (handler.route) {
            routes.push({
              path: handler.route.path,
              methods: Object.keys(handler.route.methods)
            });
          }
        });
      }
    });

    logger.info({ routes }, 'Available routes');
  }

  setupGracefulShutdown() {
    const shutdown = async (signal) => {
      logger.info(`Received ${signal}, starting graceful shutdown`);
      
      // 1. Parar de aceitar novas conexões
      this.server.close(() => {
        logger.info('HTTP server closed');
      });

      // 2. Fechar conexões com infraestrutura
      const shutdownPromises = [];
      
      if (this.redisClient) {
        shutdownPromises.push(this.redisClient.quit().then(() => {
          logger.info('Redis connection closed');
        }));
      }

      if (this.rabbitMQChannel) {
        shutdownPromises.push(this.rabbitMQChannel.close().then(() => {
          logger.info('RabbitMQ channel closed');
        }));
      }

      if (this.apolloServer) {
        shutdownPromises.push(this.apolloServer.stop().then(() => {
          logger.info('GraphQL server stopped');
        }));
      }

      // 3. Aguardar tudo fechar com timeout
      Promise.race([
        Promise.all(shutdownPromises),
        new Promise(resolve => setTimeout(resolve, 10000))
      ]).then(() => {
        logger.info('Graceful shutdown completed');
        process.exit(0);
      });
    };

    process.on('SIGTERM', () => shutdown('SIGTERM'));
    process.on('SIGINT', () => shutdown('SIGINT'));
  }
}

// Inicialização
if (require.main === module) {
  const app = new App();
  app.start().catch(error => {
    console.error('Failed to start application:', error);
    process.exit(1);
  });
}

module.exports = App;

            aws_secret_access_key=config['storage']['secret_key'],
            config=Config(signature_version='s3v4')
        )
        
        # Inicializar buckets
        self._init_buckets()
    
    def _init_buckets(self):
        """Inicializa buckets no data lake"""
        buckets = self.config['storage']['buckets'].values()
        for bucket in buckets:
            try:
                self.s3_client.create_bucket(Bucket=bucket)
                logger.info(f"Bucket {bucket} criado/inicializado")
            except Exception as e:
                logger.debug(f"Bucket {bucket} já existe: {e}")
    
    def ingest_app_event(self, event_data: Dict[str, Any]) -> bool:
        """Ingere evento de aplicação em tempo real"""
        try:
            # Validar schema
            schema = self.schema_registry.get_schema('diyapp_app_events')
            if not self._validate_event(event_data, schema):
                logger.error(f"Evento inválido: {event_data}")
                return False
            
            # Adicionar metadados
            enriched_event = self._enrich_event(event_data)
            
            # Publicar no Kafka para processamento em tempo real
            self._publish_to_kafka('diyapp.app.events', enriched_event)
            
            # Armazenar no data lake (camada raw)
            self._store_in_data_lake('raw', 'app_events', enriched_event)
            
            logger.info(f"Evento ingerido: {enriched_event['event_id']}")
            return True
            
        except Exception as e:
            logger.error(f"Erro ao ingerir evento: {e}")
            return False
    
    def ingest_llm_call(self, llm_data: Dict[str, Any]) -> bool:
        """Ingere chamada LLM com dados de custo e performance"""
        try:
            schema = self.schema_registry.get_schema('diyapp_llm_calls')
            if not self._validate_event(llm_data, schema):
                return False
            
            # Calcular custo baseado em tokens
            llm_data['cost_usd'] = self._calculate_llm_cost(llm_data)
            
            # Publicar no Kafka
            self._publish_to_kafka('diyapp.llm.calls', llm_data)
            
            # Armazenar no data lake
            self._store_in_data_lake('raw', 'llm_calls', llm_data)
            
            logger.info(f"Chamada LLM ingerida: {llm_data['call_id']}")
            return True
            
        except Exception as e:
            logger.error(f"Erro ao ingerir chamada LLM: {e}")
            return False
    
    def _validate_event(self, event_data: Dict[str, Any], schema: Dict) -> bool:
        """Valida evento contra schema Avro"""
        required_fields = schema.get('required', [])
        
        for field in required_fields:
            if field not in event_data:
                logger.error(f"Campo obrigatório faltando: {field}")
                return False
        
        # Validar tipos de dados
        for field, value in event_data.items():
            field_schema = schema['properties'].get(field)
            if field_schema and not self._validate_type(value, field_schema.get('type')):
                logger.error(f"Tipo inválido para campo {field}: {type(value)}")
                return False
        
        return True
    
    def _validate_type(self, value: Any, expected_type: str) -> bool:
        """Valida tipo de dado"""
        type_mapping = {
            'string': str,
            'integer': int,
            'number': (int, float),
            'boolean': bool,
            'array': list,
            'object': dict
        }
        
        if expected_type not in type_mapping:
            return True
        
        expected = type_mapping[expected_type]
        return isinstance(value, expected)
    
    def _enrich_event(self, event_data: Dict[str, Any]) -> Dict[str, Any]:
        """Enriquece evento com metadados"""
        enriched = event_data.copy()
        enriched['ingestion_timestamp'] = datetime.utcnow().isoformat()
        enriched['event_id'] = f"evt_{datetime.utcnow().timestamp()}_{hash(str(event_data))}"
        enriched['data_version'] = 'v1.0'
        
        # Adicionar contexto de sessão se disponível
        if 'session_id' not in enriched:
            enriched['session_id'] = f"sess_{hash(str(event_data))}"
        
        return enriched
    
    def _calculate_llm_cost(self, llm_data: Dict[str, Any]) -> float:
        """Calcula custo de chamada LLM baseado em provedor e tokens"""
        provider = llm_data.get('provider', 'openai')

  async scanDirectory(dirPath) {
    const files = await this.getFiles(dirPath);
    
    for (const file of files) {
      if (file.endsWith('.js') || file.endsWith('.ts') || 
          file.endsWith('.jsx') || file.endsWith('.tsx') ||
          file.endsWith('.py') || file.endsWith('.java') ||
          file.endsWith('.php') || file.endsWith('.go')) {
        await this.analyzeFile(file);
      }
    }
    
    return this.generateReport();
  }

  async getFiles(dir) {
    const entries = await fs.promises.readdir(dir, { withFileTypes: true });
    const files = [];
    
    for (const entry of entries) {
      const fullPath = path.join(dir, entry.name);
      
      if (entry.isDirectory() && !entry.name.includes('node_modules') && 
          !entry.name.includes('.git')) {
        files.push(...await this.getFiles(fullPath));
      } else if (entry.isFile()) {
        files.push(fullPath);
      }
    }
    
    return files;
  }

  async analyzeFile(filePath) {
    try {
      const content = await fs.promises.readFile(filePath, 'utf8');
      const lines = content.split('\n');
      
      for (const [vulnType, patterns] of Object.entries(this.patterns)) {
        patterns.forEach((pattern, index) => {
          const matches = content.match(pattern);
          if (matches) {
            this.addVulnerability(vulnType, filePath, matches, lines);
          }
        });
      }
      
      // Check for missing input validation
      if (content.includes('req.body') || content.includes('req.query') || 
          content.includes('req.params')) {
        const hasValidation = content.includes('validate') || 
                             content.includes('sanitize') ||
                             content.includes('validator') ||
                             content.includes('joi') ||
                             content.includes('zod');
        
        if (!hasValidation) {
          this.vulnerabilities.high.push({
            file: filePath,
            type: 'Missing Input Validation',
            description: 'No input validation/sanitization found for user inputs',
            line: this.findLineNumber(content, 'req.')
          });
        }
      }
      
    } catch (error) {
      console.error(`Error analyzing ${filePath}:`, error.message);
    }
  }

    const severityMap = {
      sqlInjection: 'critical',
      xss: 'high',

      authBypass: 'critical',
      fileInclusion: 'high'

      type: type.charAt(0).toUpperCase() + type.slice(1),
      matches: matches.slice(0, 3),

    });
  }

  findLineNumber(content, search) {

    for (let i = 0; i < lines.length; i++) {

    }

                  ip_address: Optional[str] = None, user_agent: Optional[str] = None) -> str:
        """

            action: Descrição da ação
            user_id: ID do usuário que realizou a ação

            cursor.execute('''

                ORDER BY block_number DESC LIMIT 1

    def verify_integrity(self) -> Dict:
        """
        Verifica a integridade de todos os logs usando a hash chain
        
        Returns:
            Dict com resultado da verificação
        """
        with self.lock:
            conn = sqlite3.connect(self.db_path)
            cursor = conn.cursor()
            
            cursor.execute('''
                SELECT h.block_number, h.log_id, h.previous_hash, h.current_hash,
                       l.timestamp, l.event_type, l.action
                FROM hash_chain h
                LEFT JOIN audit_logs l ON h.log_id = l.id
                ORDER BY h.block_number
            ''')
            
            rows = cursor.fetchall()
            conn.close()
            
            if not rows:
                return {"valid": False, "error": "No blocks found"}
            
            # Verifica cada bloco
            previous_hash = None
            corrupted_blocks = []
            
            for row in rows:
                block_num, log_id, stored_prev_hash, stored_current_hash, timestamp, event_type, action = row
                
                # Para o bloco gênese
                if log_id == 'GENESIS':
                    expected_hash = hashlib.sha256(b'GENESIS_BLOCK').hexdigest()
                    if stored_current_hash != expected_hash:
                        corrupted_blocks.append({
                            "block": block_num,
                            "reason": "Genesis block hash mismatch"
                        })
                    previous_hash = stored_current_hash
                    continue
                
                # Verifica se o hash anterior corresponde
                if stored_prev_hash != previous_hash:
                    corrupted_blocks.append({
                        "block": block_num,
                        "reason": "Previous hash mismatch"
                    })
                
                # Recalcula o hash atual
                cursor.execute('''
                    SELECT id, timestamp, event_type, user_id, action, 
                           data_before, data_after, ip_address, user_agent, previous_hash
                    FROM audit_logs WHERE id = ?
                ''', (log_id,))
                
                log_row = cursor.fetchone()
                if log_row:
                    log_data = {
                        "id": log_row[0],
                        "timestamp": log_row[1],
                        "event_type": log_row[2],
                        "user_id": log_row[3],
                        "action": log_row[4],
                        "data_before": json.loads(log_row[5]) if log_row[5] else None,
                        "data_after": json.loads(log_row[6]) if log_row[6] else None,
                        "ip_address": log_row[7],
                        "user_agent": log_row[8],
                        "previous_hash": log_row[9]
                    }
                    
                    calculated_hash = self._calculate_hash(log_data)
                    if calculated_hash != stored_current_hash:
                        corrupted_blocks.append({
                            "block": block_num,
                            "reason": "Current hash mismatch",
                            "log_id": log_id
                        })
                
                previous_hash = stored_current_hash
            
            return {
                "valid": len(corrupted_blocks) == 0,
                "total_blocks": len(rows),
                "corrupted_blocks": corrupted_blocks,
                "last_hash": previous_hash
            }
