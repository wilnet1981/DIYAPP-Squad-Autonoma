# WhatsApp Assistant - Vanusa Gràndo Mentoria

## Visão Geral
Este projeto é um assistente automatizado para WhatsApp que visa melhorar a experiência do cliente da Vanusa Gràndo Mentoria ao oferecer respostas rápidas e inteligentes sobre seus produtos e serviços.

## Estrutura Inicial
1. **index.html**: Primeiro ponto de entrada para o painel de controle do assistente.
2. **bot.js**: Contém a lógica inicial do chatbot para simulação de conversas.
3. **questions.json**: Armazena perguntas comuns e respostas simuladas.

## Como Executar
1. Abra o `index.html` em um navegador moderno.
2. Clique no botão 'Start Chat Simulation' para iniciar o chatbot.
3. Digite uma pergunta para ver como o assistente responde atualmente.
