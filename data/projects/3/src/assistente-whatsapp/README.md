# Assistente WhatsApp - Vanusa Gràndo Mentoria

Este projeto visa desenvolver um assistente automatizado para atender e qualificar leads via WhatsApp.

## Estrutura do Projeto

- **src/integrations**: Integrações com APIs externas, como a plataforma LearnWorlds.
- **src/flows**: Scripts que definem os fluxos para leads e clientes.
- **src/utils**: Funções utilitárias que suportam o projeto.

## Como Iniciar

1. Clone o repositório.
2. Navegue até o diretório do projeto.
3. Utilize o comando `node index.js` para iniciar o painel.
